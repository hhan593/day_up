```js

    const topColor = new THREE.Color(0xd5deff);
    const bottomColor = new THREE.Color(0xffffff);
    const offset = 1200;
    const exponent = 0.6;
    const material = new THREE.MeshStandardMaterial({
      side: THREE.BackSide,
      onBeforeCompile: (shader) => {
        // 顶部颜色
        shader.uniforms.topColor = { value: topColor };
        // 底部颜色
        shader.uniforms.bottomColor = { value: bottomColor };
        // 控制颜色偏移值
        shader.uniforms.offset = { value: offset };
        // 模糊程度
        shader.uniforms.exponent = { value: exponent };
        // 定义了一个varying变量，用于在顶点着色器和片段着色器之间传递世界坐标位置
        shader.vertexShader = `
        varying vec3 vWorldPosition;
        void main() {
          vec4 worldPosition = modelMatrix * vec4(position, 1.0);
          vWorldPosition = worldPosition.xyz;
          gl_Position = projectionMatrix * modelViewMatrix * vec4(position, 1.0);
        }
      `;
        shader.fragmentShader = `
        uniform vec3 topColor;
        uniform vec3 bottomColor;
        uniform float offset;
        uniform float exponent;
        varying vec3 vWorldPosition;
        void main() {
          float h = clamp((vWorldPosition.y + offset) / (30.0 * offset), 0.0, 1.0);
          gl_FragColor = vec4(mix(bottomColor, topColor, pow(h, exponent)), 1.0);
        }
      `;
      },
    });
    const sky = new THREE.Mesh(
      new THREE.SphereGeometry(6000, 32, 32),
      material
    );
    this.scene.add(sky);
  }
```

我们在材质中定义 `onBeforeCompile` 来执行 WebGL 编译前的操作，这个时候我们可以获取 shader，通过 shader 我们可以定义常量 uniform 等以及定义 vertexShader 顶点着色器、fragmentShader 片元着色器（简单理解为颜色）。 

注意这里字符串的语言是着色器 shader 的语言 glsl，接下来我们分别开始讲解顶点和片元。

## 渐变天空——顶点着色器

```js
// 模型矩阵 * 模型坐标 = 世界坐标，也就是物体本身存在屏幕的绝对坐标
vec4 worldPosition = modelMatrix * vec4(position, 1.0);
// 世界坐标 * 模型视图矩阵 * 模型坐标 = 最终坐标
gl_Position = projectionMatrix * modelViewMatrix * vec4(position, 1.0);
```

投影矩阵是一种透视变换矩阵，用于将三维场景投影到二维屏幕上。它定义了视锥体（视野体）的形状和大小，包括近平面、远平面、左平面、右平面、顶平面和底平面。通过将物体的三维坐标乘以投影矩阵，可以将其转换为裁剪空间坐标，然后进行透视除法和视口变换，最终得到屏幕上的二维坐标。 

模型视图矩阵是一种组合矩阵，由模型矩阵和视图矩阵相乘得到。模型矩阵用于将物体的局部坐标变换到世界坐标系中，包括平移、旋转和缩放等变换。视图矩阵用于将场景从世界坐标系转换到相机（观察者）的坐标系中，包括相机的位置和方向等信息。通过将物体的局部坐标乘以模型视图矩阵，可以将其转换为相机坐标系中的坐标。

投影矩阵说白了就是将无透视的物体改成透视状态。模型视图就是人的视角状态下的物体。在后续的 WebGL 章节中会深入讲解。

## 渐变天空——片元着色器

```js
float h = clamp((vWorldPosition.y + offset) / (30.0 * offset), 0.0, 1.0);
gl_FragColor = vec4(mix(bottomColor, topColor, pow(h, exponent)), 1.0);
```

`clamp`、`mix` 是 glsl 内置封装的方法，`clamp` 我们可以用来限制值，`mix` 我们用于计算颜色的混合值。

根据世界坐标位置计算高度值 h，将其限制在 0.0 到 1.0 之间。这里的计算根据偏移量和指数来控制颜色的渐变。

`gl_FragColor` 是最终的输出颜色的全局变量，据高度值 h，通过混合底部颜色和顶部颜色，并使用指数值来调整混合程度，最终得到片段的颜色。

```js
export function generateLine(
  starts = [0, 0, 0],
  ends = [0, 0, 0],
  color = 0x000000
) {
  let geometry = new THREE.BufferGeometry().setFromPoints([
    new THREE.Vector3(...starts),
    new THREE.Vector3(...ends),
  ]);
  let material = new THREE.LineBasicMaterial({
    color: color,
  });
  return new THREE.Line(geometry, material);
}
```

```js
export function generateGrid(scene, w, h, xNum, yNum) {
  const lines = [];
  const lineGroup = new THREE.Group();
  lineGroup.position.set(-w / 2, 0, h / 2);
  scene.add(lineGroup);

  const xSize = w / xNum;
  const ySize = h / yNum;

  for (let i = 0; i < yNum + 1; i++) {
    let size = i * ySize;
    const line = generateLine([0, 0, -size], [w, 0, -size], 0xcccccc);
    lines.push(line);
  }

  for (let i = 0; i < xNum + 1; i++) {
    let size = i * xSize;
    const line = generateLine([size, 0, 0], [size, 0, -h], 0xcccccc);
    lines.push(line);
  }
  lineGroup.add(...lines);

  return lineGroup;
}
```

```js
      font: textOption.font,
```

```js
  constructor() {
    this.on();
  }

  on() {
    window.addEventListener("mousemove", this.onMouseMove.bind(this));
  }

  destory() {
    window.removeEventListener("mousemove", this.onMouseMove.bind(this));
  }

  onMouseMove(event) {
    this.mouse.x = (event.clientX / window.innerWidth) * 2 - 1;
    this.mouse.y = -(event.clientY / window.innerHeight) * 2 + 1;
  }

  // 射线投射
  raycast(scene, camera) {
    this.raycaster.setFromCamera(this.mouse, camera);

    // 检测射线与物体的交点
    const intersects = this.raycaster.intersectObjects(scene.children);

    // 处理交点结果
    if (intersects.length > 0) {
      const firstIntersectedObject = intersects[0].object;
      if (firstIntersectedObject.name === "cube") {
        return firstIntersectedObject;
      }
    }
  }
}
```
