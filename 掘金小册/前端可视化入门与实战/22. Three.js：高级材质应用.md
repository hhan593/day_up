## HDR 高动态范围成像

**HDR（High Dynamic Range）是一种图像格式，也称为 HDRI，它用于捕捉和显示更广泛亮度范围的图像**。HDR 图像可以包含比标准图像格式（如 JPEG）更高的 `亮度和颜色信息`，以提供更真实和逼真的光照和颜色效果。

HDR 图像使用`浮点数格式存储像素值，可以捕捉到非常明亮和非常暗的细节`。这使得它们能够更好地模拟真实世界中的光照条件，包括高光和阴影细节。相比之下，标准图像格式（如 JPEG）通常使用 8 位或 16 位整数格式，无法捕捉到同样广泛的亮度范围。

在计算机图形学中，HDR 图像常用于创建环境贴图、光照贴图和高质量的渲染效果。它们可以用于`模拟真实世界中的光照和反射`，提供更逼真的视觉效果。

## PBR 原理

**PBR，或者用更通俗一些的称呼是指基于物理的渲染（Physically Based Rendering），它指的是一些在不同程度上都基于与现实世界的物理原理更相符的基本理论所构成的渲染技术的集合**。

正因为基于物理的渲染便是为了使用一种更符合物理学规律的方式来模拟光线，因此这种渲染方式与我们原来的 Phong 或者 Blinn-Phong 光照算法相比，总体上看起来要更真实一些。

判断是否是 PBR 材质，必须满足以下三个条件：

1. **基于微平面**（Microfacet）的表面模型：PBR 材质使用基于微平面的模型来描述物体表面的微小凹凸细节。这意味着材质考虑了表面的微观结构，并使用统计学上的微平面理论来计算光线的反射和散射。微平面模型能够更准确地模拟真实世界中的光线与粗糙表面的相互作用。 
2. **能量守恒**：PBR 材质满足能量守恒的原则，即光线的总能量在反射和散射过程中保持不变。这意味着材质的光反射和散射行为符合能量守恒的物理规律，使得渲染结果更加真实和可信。
 
3. **应用基于物理的 BRDF**（双向反射分布函数）：PBR 材质使用基于物理的 BRDF 来描述光线在表面上的反射行为。BRDF 是一个函数，它根据入射光线的方向和出射光线的方向计算出光线的反射率。PBR 材质使用 BRDF 来模拟光线在不同角度和表面特性下的反射和散射，以达到更真实的渲染效果。



## 通道贴图

通道贴图可以让我们实现更高上限、更强的真实效果，对于大部分的渲染师来说会直接使用建模软件生成材质球，并附加各种通道贴图。

<p align=center><img src="https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/7c1cba74e1de4045afc7691adc64624b~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=788&h=429&s=293337&e=png&b=f6f1f1" alt="image.png"  /></p>

通道贴图（Channel Map）是一种技术，用于将单个纹理图像中的不同颜色通道分离出来，以控制材质的颜色、粗糙度、金属度、法线等属性。 

通道贴图的常见用途包括： 

1. **Albedo**（反射率）通道贴图：也称为漫反射贴图，用于控制材质的基本颜色。它决定了物体表面的反射率，即光线被表面吸收或反射的程度。 
2. **Normal**（法线）通道贴图：用于模拟物体表面的细微凹凸细节。法线贴图通过改变光照计算中的法线方向，以产生视觉上的凹凸效果，从而增加物体的表面细节。 
3. **Metallic**（金属度）通道贴图：用于控制材质的金属度属性。金属度贴图通常是灰度图像，较亮的像素表示较金属的表面，较暗的像素表示较非金属的表面。 
4. **Roughness**（粗糙度）通道贴图：用于控制材质的表面粗糙度。粗糙度贴图通常是灰度图像，较亮的像素表示较光滑的表面，较暗的像素表示较粗糙的表面。
5. **AO**（环境光遮蔽）通道贴图：用于模拟环境光在物体表面的遮蔽效果。AO 贴图通常是灰度图像，较亮的像素表示受到更多环境光的照射，较暗的像素表示受到较少环境光的照射。一般用于控制场景的边缘角落的光线遮挡和散射，产生更加柔和和自然的阴影效果。
6. **Bump**（凹凸贴图）：bumpMap 使用灰度值来表示表面的高度变化。灰度值越高，表面越凸起；灰度值越低，表面越平坦。bumpMap 通过改变光线的入射角度来模拟凹凸细节，但不会实际改变法线的方向。相比于 normalMap，bumpMap 的效果可能略显粗糙，不如法线贴图能够提供更精确的细节。

这里需要注意下，`normalMap` 和 `bumpMap` 都是用于模拟物体表面细节的贴图，normalMap 使用 RGB 通道来存储法线信息，能够提供更精确和真实的凹凸细节；而bumpMap 使用灰度值来表示高度变化，效果相对较粗糙。在实际应用中，normalMap 通常能够提供更好的视觉效果和真实感，但需要更高的计算成本。而 bumpMap 则可以在一些情况下提供较为简单的凹凸效果。另外，当同时配置了 `normalMap` 和 `bumpMap` 时，通常只有 `normalMap` 会生效，而 `bumpMap` 则不会生效。

## 清漆材质

清漆通常是一种透明的涂层，用于保护物体表面、增加光泽度并提供一种光滑的外观。常用于模拟车漆、涂层、玻璃、塑料等物体的外观。

```js
// 清漆材质
let material = new THREE.MeshPhysicalMaterial({
  // 光泽强度
  clearcoat: 1,
  // 光泽的粗糙度
  clearcoatRoughness: 0,
  // 金属度
  metalness: 1,
  // 粗糙度
  roughness: 0.5,
  color: 0x0000ff,
});

let mesh = new THREE.Mesh(geometry, material);
mesh.position.x = -1;
mesh.position.y = 1;
this.group.add(mesh);
```

很明显，材质球已经反射了环境贴图，接着我们来解析下每个参数的作用。

1. `clearcoat` （光泽强度）：该参数控制着清漆层的光泽强度。较高的值会增加物体表面的反射光线，使其看起来更加光滑和有光泽。 
2. `clearcoatRoughness` （光泽的粗糙度）：该参数控制着清漆层的粗糙度。较低的值会使清漆层更加光滑，而较高的值会使其表面更加粗糙。 
3. `metalness` （金属度）：该参数控制着物体表面的金属感。较高的值会使物体看起来更像金属，而较低的值则会使其看起来更像非金属材质，如塑料或木材。 
4. `roughness` （粗糙度）：该参数控制着物体表面的粗糙度。较低的值会使物体表面更加光滑，而较高的值则会使其表面更加粗糙。 
5. `color` （颜色）：该参数定义了物体的颜色。你可以使用 16 进制的颜色值来指定颜色。例如，`0x0000ff` 表示蓝色。

接下来我们使用 FlakesTexture 纹理用于模拟物体表面上的颗粒效果，例如金属表面上的碎屑、雪花或细小的颗粒。它可以为材质添加微小的凹凸细节，增加物体的真实感和细节。

同时我们设置 `clearcoat` 和 `clearcoatRoughness` 来实现清漆效果。

```js
import { FlakesTexture } from "three/examples/jsm/textures/FlakesTexture";
const normalMap = new THREE.CanvasTexture(new FlakesTexture());
// 重复方式为 RepeatWrapping
normalMap.wrapS = THREE.RepeatWrapping;
normalMap.wrapT = THREE.RepeatWrapping;
// 设置纹理在水平和垂直方向上的重复次数
normalMap.repeat.x = 10;
normalMap.repeat.y = 6;
// 设置纹理的各向异性过滤的程度。较高的值（例如16）将提高纹理的细节和质量，但可能会增加计算和性能开销
normalMap.anisotropy = 16;

let material = new THREE.MeshPhysicalMaterial({
  // 光泽强度
  clearcoat: 1,
  // 光泽的粗糙度
  clearcoatRoughness: 0,
  // 金属度
  metalness: 1,
  // 粗糙度
  roughness: 0.5,
  color: 0x0000ff,
  // 法线贴图
  normalMap: normalMap,
  // 法线贴图尺寸
  normalScale: new THREE.Vector2(0.15, 0.15),
});
```

## 自发光材质

自发光材质的主要特点是具有自发光颜色和强度。通过设置自发光颜色，可以定义物体发出的光的颜色。而自发光强度则控制了发出的光的强度或亮度级别。 

自发光材质通常用于模拟发光物体，如发光灯、荧光物体或发光字等。通过给这些物体应用自发光材质，可以使它们在场景中发出光线，从而产生逼真的发光效果。

```js
// 自发光材质
material = new THREE.MeshPhysicalMaterial({
  emissive: 0xffffff, // 设置自发光的颜色
  emissiveIntensity: 10, // 设置自发光颜色的强度
});
```

但是需要注意的是，自发光本身是不会产生光源的，我们可以通过方向光或点光源来模拟。

```js
  color: new THREE.Color(1, 1, 1),
});
// 为了模拟折射，我们要设置玻璃厚度
material.thickness = 0.5;
```

玻璃的反射率通常是非常低的，因为大部分光线会透过玻璃而非被反射。对于普通的透明玻璃，可以将反射率设置为一个很小的值，通常在 0.02 到 0.08 之间。这样的设置可以模拟出玻璃的透明性和反射性。


<p align=center><img src="https://p9-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/0aab1ae45d38457abbae6ba84a901dd3~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=278&h=275&s=163818&e=png&b=cce2ed" alt="image.png"  /></p>

但如果是玻璃弹珠那种厚重的质感，材质反射率高的时候，我们将反射率修改为 1 时，光线不会完全透过玻璃，很明显折射效果就出来了。

<p align=center><img src="https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/e568995a57be45f2a422f6e40c3540c3~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=305&h=313&s=194860&e=png&b=cbe0e7" alt="image.png"  /></p>

另外，我们需要知道折射率的概念，折射率是光线从一种介质（例如空气）进入另一种介质（例如水或玻璃）时，光线会折射（弯曲），光线的传播速度在两种介质之间的比率。折射率通常用符号 n 表示。我们知道玻璃的折射率是 1.7。

除此之外，我们还需要记住常见物体材质的折射率还有这些：

- 真空：1.000
- 空气/大气：1.0003
- 水：1.333
- 冰：1.31
- 玻璃：1.5-1.9（具体取决于玻璃的类型）
- 亚克力：1.49-1.52
- 蓝宝石：1.76
- 钻石：2.42
- 红宝石：1.76-1.77

这些都是基于可见光波长的近似值。请注意，折射率值可能因为材料的纯度、温度和光波长的变化而有所不同。

同时我们可以通过 sheenColor 来修改光泽颜色，specularColor 来修改高光颜色。对于有色玻璃，我们可以直接修改 color。

```js
color: new THREE.Color(1, 0, 1),
```

## 草地材质

草地材质的特征就是无高光、反射很弱、凹凸感强。我们只要导入对应的通道贴图即可。

<p align=center><img src="https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/3e38c5334efa4b52a272d00e664f7971~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=436&h=425&s=429619&e=png&b=9ea35e" alt="image.png"  /></p>

```js
material = new THREE.MeshPhysicalMaterial({
      map: colorGreenMap,
      bumpMap: bumpGreenMap,
      normalMap: normalGreenMap,
      aoMap: aoGreenMap,
      normalScale: new THREE.Vector2(5, 5),
      envMapIntensity: 0.5,
      // displacementMap: dispGreenMap,
      // displacementBias: 0.2,
      // displacementScale: 2,
    });
```

对于草地材质，这里要了解下`displacementMap`。

`displacementMap`是一种灰度贴图，其中不同的灰度值表示了不同的位移量，用于在渲染过程中对物体的几何形状进行位移或变形。它通常用于模拟物体表面的细节或凹凸效果。

需要注意的是， `displacementMap` 贴图的效果在渲染时会受到物体的几何细分程度的影响。更高的几何细分级别可以提供更精细的位移效果，而较低的细分级别可能会导致位移效果不够明显。 

`displacementMap` 贴图常用于模拟`地形、细节纹理、水波纹`等效果，以增加物体的真实感和细节。

## 砖墙材质

砖墙材质相比草地，会反射强度高一点，凹凸感更强。

我们通过加载颜色通道贴图渲染，为了模拟凹凸效果，我们使用了法线贴图。为了让边缘的光影更加自然，我们设置了 AO 贴图。为了模拟物体的不同表面位置的粗糙程度，我们设置了粗糙度贴图，并将 roughness 设置为 0.5，保留一定的反射效果。

```js
material = new THREE.MeshPhysicalMaterial({
  map: colorBrickMap,
  normalMap: normalBrickMap,
  aoMap: aoBrickMap,
  envMapIntensity: 0.5,
  roughnessMap: roughnessBrickMap,
  roughness: 0.5,
});
```

<p align=center><img src="https://p1-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/ca422cf3cf734720b7a3caa17761368b~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=613&h=554&s=698521&e=png&b=9f988e" alt="image.png"  /></p>

这里有一个问题，对于加载大尺寸的纹理很可能导致纹理比例不对，所以我们可以先将纹理缩放。

```js
let maps = [colorBrickMap, bumpBrickMap, normalBrickMap, aoBrickMap];
maps.forEach((map) => {
  map.wrapS = THREE.RepeatWrapping;
  map.wrapT = THREE.RepeatWrapping;
  map.repeat.set(0.5, 0.5);
});
```

## 水体材质

<p align=center><img src="https://p6-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/22eb42167ad2482481d59229493cd27e~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=966&h=773&s=1032148&e=png&b=5e6d71" alt="image.png"  /></p>

```js
import { Water } from "three/examples/jsm/objects/Water2";

//...
// 水体材质

// 法线贴图
const waterMap = textureLoader.load("textures/normal/water.jpg");
const waterOptions = {
  color: "#abcdef", // 水体颜色
  scale: 0.1, // 波浪大小, 值越小越明显
  flowDirection: new THREE.Vector2(1, 1), // 水流方向
  textureWidth: 1024, // 纹理宽度
  textureHeight: 1024, // 纹理高度
  normalMap0: waterMap,
  normalMapUrl0: waterMap,
  envMap: envMap, // 反射天空盒的立方体纹理
  receiveShadow: true, // 是否接收阴影
  distortionScale: 3.7, // 扭曲效果的大小
};

// 创建 Water 对象
const water = new Water(geometry, waterOptions);
water.position.x = -3;
water.position.y = -1;
this.group.add(water);
```

实现水体材质非常简单，在 three 中已经封装了 Water 对象，通过控制 waterOptions 实现水体的变化，如果想要有不同的光照反射带来凹凸效果，可以尝试不同的法线贴图。
