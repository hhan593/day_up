Star 类具有轨道半径、半径、轨道中心位置、时间等属性。在构造函数中，我们为每个星星生成随机的属性，并将其添加到 `stars` 数组中。

我们来看下 `maxOrbit` 方法：

```js
function maxOrbit(x, y) {
    let max = Math.max(x, y),
        diameter = Math.round(Math.sqrt(max * max + max * max));
    return diameter / 2;
}
```

`maxOrbit` 函数接收两个参数 `x` 和 `y`，分别代表宽度和高度。函数首先找到 `x` 和 `y` 中的较大值，并将其赋值给变量 `max` 。然后，通过使用勾股定理计算出以 `max` 为边长的正方形的对角线长度（直径）。最后，将直径除以 2，得到的结果即为最大轨道半径。


接着我们看下 `random` 方法：

```js
function random(min, max) {
    if (arguments.length < 2) {
        max = min;
        min = 0;
    }

    if (min > max) {
        let hold = max;
        max = min;
        min = hold;
    }

    return Math.floor(Math.random() * (max - min + 1)) + min;
}
```

如果只提供一个参数，那么该参数将被视为最大值，最小值默认为 0。如果提供了两个参数，第一个参数将被视为最小值，第二个参数将被视为最大值。很明显我们限制了 `[0, maxOrbit(w, h)]`，也就是 0 到浏览器视口的尺寸限制。最后我们得到的是随机的轨道半径。

## 曲线动效

在实现曲线动画效果前，我们先讲解下具体在 Canvas 中 **`曲线绘制`**。

**Canvas 的曲线绘制是指使用贝塞尔曲线来创建平滑的曲线路径。贝塞尔曲线是通过控制点来定义曲线形状的数学曲线。**


在 Canvas 中，有两种类型的贝塞尔曲线可供使用。

1. **二次贝塞尔曲线（Quadratic Bezier Curve）**：二次贝塞尔曲线由起点（start point）、一个控制点（control point）和终点（end point）组成。通过调整控制点的位置，可以改变曲线的形状。二次贝塞尔曲线我们会用 `quadraticCurveTo()` 方法来绘制，因为它只有一个控制点，创建曲线的形状更简单。

```js
// 移动到起始点
ctx.moveTo(50, 150);  
// 前2个参数代表一个控制点的坐标，后2个参数代表结束点坐标。 
ctx.quadraticCurveTo(150, 100, 250, 150);
```

2. **三次贝塞尔曲线（Cubic Bezier Curve）**：三次贝塞尔曲线由起点、两个控制点和终点组成。通过调整两个控制点的位置，可以更加灵活地改变曲线的形状。三次贝塞尔曲线我们会用`bezierCurveTo()` 来控制两个操作点。

```js
// 移动到起始点
ctx.moveTo(50, 50); 
// 前4个参数代表两个控制点的坐标，后2个参数代表结束点坐标。
ctx.bezierCurveTo(100, 20, 200, 150, 250, 50); 
```

3. **通过正余弦函数计算曲线**：通过 `x += 1` 递增 x 的值，然后通过正弦函数 `Math.sin(frequency * x + yOffset)` 计算出 y 的值。其中， `frequency` 表示波浪的频率， `amplitude` 表示波浪的振幅， `yOffset` 表示波浪的垂直偏移量。通过不断计算并连接这些点，就可以绘制出一条波浪曲线。调整 `frequency` 、 `amplitude` 和 `yOffset` 的值，可以改变曲线的形状和运动效果。

```js
class Wave {
    canvas = null
    ctx = null

    amplitude = 50
    frequency = 0.02
    speed = 0.1
    yOffset = 0
    constructor() {
        this.initCanvas()
        this.animateWave();
    }
    initCanvas() {
        this.canvas = document.getElementById("myCanvas");
        this.ctx = this.canvas.getContext("2d");
    }
    drawWave() {
        const { ctx, canvas, amplitude, frequency, speed, yOffset } = this
        ctx.clearRect(0, 0, canvas.width, canvas.height);
        ctx.beginPath();

        var x = 0;
        var y = canvas.height / 2;

        ctx.moveTo(x, y);

        while (x < canvas.width) {
            x += 1;
            y = canvas.height / 2 + amplitude * Math.sin(frequency * x + yOffset);
            ctx.lineTo(x, y);
        }

        ctx.strokeStyle = "blue";
        ctx.lineWidth = 20;
        ctx.stroke();
    }
    animateWave = () => {
        this.yOffset += this.speed;
        this.drawWave();
        requestAnimationFrame(this.animateWave);
    }
}
const wave = new Wave()
```

首先，实现它我们需要考虑的变量有哪些：一个是曲线的数量，一个是控制曲线控制点变化速率的变量，一个是曲线集合的混合值，其实就是控制图上聚集点位置的变量。

```js
const config = {
    // 曲线数
    curvesNum: 1,
    // 波浪速度
    waveSpeed: .53,
    // 混合的波浪数量
    wavesToBlend: 10,  //waves count to blend 
}
```

接下来我们定义曲线类：

```js
class WaveNoise {
    constructor() {
        // 随机角度集合
        this.wavesSet = [];
    }
    // 生成随机角度值
    addWaves(requiredWaves) {
        for (let i = 0; i < requiredWaves; ++i) {
            let randomAngle = Math.random() * 360;
            this.wavesSet.push(randomAngle);
        }
    }
    // 得到波浪的混合值
    getWave() {
        let blendedWave = 0;
        for (let e of this.wavesSet) {
            blendedWave += Math.sin(e / 180 * Math.PI); //convert degree to radian and get sine
        }
        return (blendedWave / this.wavesSet.length + 1) / 2;
    }
    // 更新波浪的角度
    update() {
        this.wavesSet.forEach((e, i) => {
            let r = (i + 1) * config.waveSpeed;
            this.wavesSet[i] = (e + r) % 360;
        });
    }
}
```

1. **addWaves 生成控制点**：根据指定的波浪数量（`requiredWaves`），生成随机角度值，并将这些角度添加到波浪集合中。通过循环，生成指定数量的随机角度（0 到 360 之间）并存储在 wavesSet 数组中。 

2. **getWave 获得混合值**：计算波浪集合中所有角度的正弦值，并将这些正弦值相加。通过将角度值转换为弧度值（`e / 180 * Math.PI`）并计算正弦值（Math.sin），得到每个角度对应的波浪值。最后，将所有波浪值相加并除以波浪集合的长度，再加 1，最终得到波浪的混合值（范围在 0 到 1 之间）。 

3. **update 修改角度**：遍历波浪集合中的每个角度，根据指定的波浪速度（`config.waveSpeed`）更新角度的值。通过将当前角度与波浪速度的乘积加到角度上，并使用模运算（%）将角度限制在 0 到 360 之间。这样可以实现波浪的旋转效果。 

通过使用 WaveNoise 类，可以生成指定数量的随机角度曲线，并根据需要获取波浪的混合值和更新波浪的角度。

通过使用 WaveNoise 类，可以生成指定数量的随机角度值。通过角度值我们在动画类中可以生成控制点。

接着我们定义 Animation 类：


```js
    class Animation {
        constructor() {
            this.cnv = null;
            this.ctx = null;
            this.size = { w: 0, h: 0, cx: 0, cy: 0 };
            this.controls = [];
            this.controlsNum = 3;
        }
        init() {
            this.createCanvas();
            // 控制点我们可以在渲染前提前获取
            this.createControls();
            this.updateAnimation();
        }
        createCanvas() {
            this.cnv = document.createElement("canvas");
            this.ctx = this.cnv.getContext('2d');
            this.setCanvasSize();
            document.body.appendChild(this.cnv);
            window.addEventListener(`resize`, () => this.setCanvasSize());
        }
        setCanvasSize() {
            this.size.w = this.cnv.width = window.innerWidth;
            this.size.h = this.cnv.height = window.innerHeight;
            this.size.cx = this.size.w / 2;
            this.size.cy = this.size.h / 2;
        }
        // 生成控制点集合
        createControls() {
            for (let i = 0; i < (config.curvesNum * 4); ++i) {
                let control = new WaveNoise();
                control.addWaves(config.wavesToBlend);
                this.controls.push(control);
            }
        }
        // 更新控制点集合中的每个控制点
        updateControls() {
            this.controls.forEach(e => e.update());
        }
        // 根据控制点生成真正的坐标
        updateCurves() {
            let radius = 300;
            let c = this.controls;
            for (let i = 0; i < config.curvesNum; ++i) {

                let angle = (360 / (config.curvesNum * 2) * i) / 180 * Math.PI;
                let cos = Math.cos(angle);
                let sin = Math.sin(angle);
                // 开始点和结束点
                let _startX = cos * radius;
                let _startY = sin * radius;
                let _endX = -_startX;
                let _endY = -_startY;

                // 生成控制点
                let _controlX1 = c[i].getWave()  + this.size.cx;
                let _controlY1 = c[i * 2].getWave()  + this.size.cy;
                let _controlX2 = c[i * 3].getWave()  + this.size.cx;
                let _controlY2 = c[i * 4].getWave();
                let curveParam = {
                    startX: _startX + this.size.cx,
                    startY: _startY + this.size.cy,
                    controlX1: _controlX1,
                    controlY1: _controlY1,
                    controlX2: _controlX2,
                    controlY2: _controlY2 * motionRadius + this.size.cy,
                    endX: _endX + this.size.cx,
                    endY: _endY + this.size.cy,
                    alpha: _controlY2
                }

                this.drawCurve(curveParam);
            }
        }
        // 绘制三次贝塞尔曲线
        drawCurve({ startX, startY, controlX1, controlY1, controlX2, controlY2, endX, endY, alpha }) {
            this.ctx.strokeStyle = `rgba(255, 255, 255, ${alpha})`;
            this.ctx.beginPath();
            this.ctx.moveTo(startX, startY);
            this.ctx.bezierCurveTo(controlX1, controlY1, controlX2, controlY2, endX, endY);
            this.ctx.stroke();
        }
        updateCanvas() {
            this.ctx.fillStyle = `rgb(22, 22, 25)`;
            this.ctx.fillRect(0, 0, this.size.w, this.size.h);
        }
        updateAnimation() {
            // 首先要更新画布
            this.updateCanvas();
            // 其次才是重新渲染曲线和控制点
            this.updateCurves();
            this.updateControls();

            window.requestAnimationFrame(() => this.updateAnimation());
        }
    }

    window.onload = () => {
        new Animation().init();
    }
```

对于上文代码，我们要注意的是：一定要区分开更新操作和初次操作，防止不必要的多余渲染。所以，我们这里定义了 init 方法初始化画布、获取控制点集合、最后执行动画渲染。在 `updateAnimation`中，我们先初始化画布、更新第一次的控制点生成曲线，为了动态效果，我们需要再次执行 updateControls 来修改控制点的角度值。

这块代码核心在于 `updateCurves`，动态的地方在于混合值和角度生成的控制点，与之对应的就是`getWave()` 和 `controls`。由于角度变化会影响生成的控制点，意味着曲线会变化。`为了让每条曲线都有趋向性，也就是能有明显的交集`，我们通过 `getWave`来获取混合的坐标值。


## 总结

最后，我们总结下本文所学的知识点：

1. **星空粒子特效**：对于数以千计数量级的粒子特效来说，我们要区分静态和动态渲染，将静态的粒子渲染做离屏加载，这样能极大地提升性能。
