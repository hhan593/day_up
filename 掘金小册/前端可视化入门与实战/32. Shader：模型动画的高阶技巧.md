我们先来学习下 mix 插值函数。


## mix 插值函数

在 GLSL 中， `mix` 函数用于在两个值之间进行线性插值。它接收三个参数：`mix(x, y, a)` ，其中 `x` 和 `y` 是要插值的两个值，`a` 是插值因子。

插值因子 `a` 通常是一个范围在 0 到 1 之间的值。当 `a` 为 0 时， `mix` 函数返回 `x` 的值；当 `a` 为 1 时，返回 `y` 的值；当 `a` 在 0 到 1 之间时，返回 `x` 和 `y` 之间的线性插值结果。

 `mix` 函数在图形编程中常用于颜色的混合、纹理的采样、动态参数的插值等场景。它提供了一种方便的方式来在不同的值之间进行平滑的过渡。
 
所以，这里`smoothstep`用于计算插值因子，将边缘范围限制在 -1 到 1，返回值是 0 到 1 的插值因子。然后通过 `mix` 来映射到 1 到 3 的范围。

## 模型阵列

```js
// 立方体3d SDF
float triangleCylinder(vec3 p,float width,float height){
    float dist=max(abs(p.x)-width,max(abs(p.y)-width,abs(p.z)-height-3.));
    return(dist);
}

// 用于对三维向量进行旋转
vec3 erot(vec3 p,vec3 ax,float ro){
    return mix(dot(p,ax)*ax,p,cos(ro))+sin(ro)*cross(ax,p);
}

// 计算一个二维向量的边界
vec2 edge(vec2 p){
    vec2 p2=abs(p);
    if(p2.x>p2.y)return vec2((p.x<0.)?-1.:1.,0.);
    else return vec2(0.,(p.y<0.)?-1.:1.);
}

// 计算场景形状
float scene(vec3 p){
    // 每个柱子中心位置
    vec2 center=floor(p.xy)+.5;
    // 相邻柱子中心位置
    vec2 neighbour=center+edge(p.xy-center);
    // 随机高度
    float height=sin(center.y+center.x+iTime)*.4;
    float width=.45;
    
    // 生成相接的两个 SDF 造型
    float me=triangleCylinder(p-vec3(center,0),width,height)-.03;
    float next=triangleCylinder(p-vec3(neighbour,0),width,3.)-.03;
    
    // 保留最低的柱体形状，形成连续的表面
    return min(me,next);
}

// 计算法线
vec3 norm(vec3 p){
    mat3 k=mat3(p,p,p)-mat3(.01);
    return normalize(scene(p)-vec3(scene(k[0]),scene(k[1]),scene(k[2])));
}

// 主入口
void mainImage(out vec4 fragColor,in vec2 fragCoord)
{
    vec2 uv=(fragCoord-.5*iResolution.xy)/iResolution.y;
    
    // 相机方向
    vec3 cam=normalize(vec3(1.5,uv));
    // 相机初始位置
    vec3 init=vec3(-10,0,0);
    
    float yrot=.5;
    float zrot=iTime*.2;
    
    // 旋转
    cam=erot(cam,vec3(0,1,0),yrot);
    init=erot(init,vec3(0,1,0),yrot);
    cam=erot(cam,vec3(0,0,1),zrot);
    init=erot(init,vec3(0,0,1),zrot);
    init.z+=1.;
    
    vec3 p=init;
    bool hit=false;
    // 光线追踪的算法
    for(int i=0;i<500&&!hit;i++){
        float dist=scene(p);
        // 判断碰撞
        hit=(dist*dist)<1e-6;
        // 得到向量
        p+=dist*cam;
        // 超出边界不显示
        if(distance(p,init)>50.)break;
    }
    // 得到法线，用于计算光照
    vec3 n=norm(p);
    // 得到反射向量 r
    vec3 r=reflect(cam,n);
    float col=length(sin(r*2.)*.5+.5)/sqrt(3.);
    col=col*.1+pow(col,3.);
    
    // 控制渲染背景与物体
    fragColor=hit?vec4(col):vec4(.8);
    fragColor=sqrt(fragColor);
}

```

阵列的核心在 `scene`，参数 p 是光线追踪算法中的当前位置向量。它代表了相机射出的光线在场景中的位置。在每次迭代中，我们将当前位置 `p` 与场景中的柱体进行碰撞检测，通过调用 `scene(p)` 函数来获取与柱体的距离。如果距离小于某个阈值（这里是 1e-6），则认为光线与柱体相交，设置 `hit` 为 `true` 。

然后，我们根据距离和光线方向更新当前位置 `p` ，将其移动到与柱体相交的点上。如果当前位置 `p` 与相机初始位置 `init` 之间的距离超过了一个阈值（这里是 50.0），则退出迭代。

我们可以通过 `texture` 函数从纹理 `iChannel0` 中对三个不同的坐标进行采样操作，分别是 `p.xz` 、 `p.yz` 和 `p.xy` 。这样就可以获取到三个采样点对应的颜色值，分别存储在 `colXZ` 、 `colYZ` 和 `colXY` 中。

然后，对法向量 `n` 取绝对值，确保其每个分量都为非负数。 最后，通过线性插值的方式将三个采样点的颜色值与法向量 `n` 的对应分量进行加权叠加，得到最终的颜色值 `imgCol` 。这里的加权叠加是为了根据法向量的不同分量值，在纹理中获取对应的颜色值。

## 总结

- 熟练使用 shader 的最好的方法就是理解图形与函数的关系，比如三角函数我们可以用来做周期性、随机性、曲线、旋转等；
- 在做阶梯造型的时候我们可以使用 smoothstep；
- 做范围差值可以用 mix；
- 扭曲可以用旋转矩阵；
- 镜像可以通过 abs 限定范围后将各分量减一后实现中心为原点；
- 阵列核心在于 SDF 中分别计算两个不同的高度的形体达到阶梯效果，最后在光线进行中不断补全形体。
