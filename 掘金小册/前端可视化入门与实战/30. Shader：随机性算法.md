
在游戏中，地面、天空、材质等都存在随机性算法的影子，比如火焰特效、《我的世界》中随机生成的网格地形、水纹的噪声材质等。学会并理解这些算法的原理可以帮助我们更轻松实现更真实的场景。

本文我们将用 glsl 手写实现这些随机性算法，另外我们同样可以使用 Javascript 结合 Canvas 实现二维的随机性算法应用。



## 伪随机

使用伪随机算法我们可以实现电视屏幕的雪花材质、花岗岩材质等效果。

比如下面的代码：

```js
#ifdef GL_ES
precision mediump float;
#endif

uniform vec2 u_resolution;
uniform vec2 u_mouse;
uniform float u_time;

float random(vec2 st) {
  return fract(sin(dot(st.xy,vec2(12.,100.)))*1000.);
}

void main(){
    vec2 st=gl_FragCoord.xy/u_resolution.xy;

    float rnd=random(st);

    gl_FragColor=vec4(vec3(rnd),1.);
}
```

> 这段代码中使用的随机数生成函数 `random` 被称为伪随机数生成函数。它被称为伪随机数是因为它实际上是通过确定性的数学计算生成的，而不是真正的随机性。

算法的主要步骤如下：

1.  `dot(st.xy, vec2(12., 100.))` ：将输入的 2D 向量 `st` 与固定的 2D 向量 `vec2(12., 100.)` 进行点乘运算。点乘运算将两个向量的对应分量相乘，并将结果相加。这一步骤的目的是将输入向量与一个随机化的向量进行关联。

2.  `sin(dot(st.xy, vec2(12., 100.)))` ：将上一步得到的结果应用正弦函数，将其映射到 -1 到 1 之间。正弦函数将输入的角度映射到一个周期性的正弦波曲线上。

3.  `fract(sin(dot(st.xy, vec2(12., 100.))))` ：将上一步得到的结果应用 `fract` 函数，取其小数部分。这一步骤的目的是将结果限制在 0 到 1 之间，得到一个连续的随机数。

4.  `fract(sin(dot(st.xy, vec2(12., 100.))) * 1000.)` ：将上一步得到的结果乘以 1000，以增加随机性的范围和精度。这样可以得到更大范围的随机数。

最终，函数返回的结果是一个在 0 到 1000 之间的随机数。这个算法使用了输入的 2D 向量 `st` 来生成随机数，因此可以根据不同的输入值来获取不同的随机数。

## Noise 噪声

GLSL 中的噪声函数是用于生成各种类型噪声的函数，常用于模拟自然场景、纹理生成、粒子效果等。噪声函数可以生成随机但有一定连续性和规律性的数值。

GLSL 中常用的噪声函数有以下几种：

1. **柏林噪声**（Perlin Noise）：最早由 Ken Perlin 在 1983 年提出的噪声算法，用于生成连续且无规律的噪声模式。

2. **Simplex 噪声**（Simplex Noise）：由 Ken Perlin 在 2001 年提出的改进版噪声算法，相对于 Perlin Noise 具有更高的性能和更好的质量。Perlin Noise 的优化版。

3. **细胞噪声**（Cellular Noise）：由 Steven Worley 在 1996 年提出的噪声算法，用于生成具有细胞结构的噪声模式。

这些噪声函数可以通过调整输入参数、组合函数、缩放和平移来获得不同的噪声效果。可以根据具体的需求和场景来选择合适的噪声函数，并结合其他图形技术来实现更复杂的效果。

我们先来看看这个案例柏林噪声的 DEMO。

### 柏林噪声 Perlin Noise

```js
#ifdef GL_ES
precision mediump float;
#endif

uniform vec2 u_resolution;
uniform vec2 u_mouse;
uniform float u_time;

float random(vec2 st){
    return fract(sin(dot(st.xy,vec2(12.9898,78.233)))*43758.5453);
}

float noise(vec2 st){
    vec2 i=floor(st);
    vec2 f=fract(st);
    
    // 四个角点的随机数
    float a=random(i);
    float b=random(i+vec2(1.,0.));
    float c=random(i+vec2(0.,1.));
    float d=random(i+vec2(1.,1.));
    
    // 双线性插值
    vec2 u=f*f*(3.-2.*f);
    return mix(a,b,u.x)+(c-a)*u.y*(1.-u.x)+(d-b)*u.x*u.y;
}

void main(){
    vec2 st=gl_FragCoord.xy/u_resolution.xy;
    float n=noise(st*10.);
    
    gl_FragColor=vec4(vec3(n),1.);
}
```

`float noise(vec2 st)` 是一个自定义噪声函数，用于生成 Perlin 噪声。它接受一个二维向量作为输入，并将其分为整数部分和小数部分。然后，根据整数部分的坐标，在**梯度向量场**中找到**四个随机梯度向量**，并使用**双线性插值**，也就是假设有一个二维网格，其中四个顶点具有已知的值。要估计网格内部的某个位置的值，首先在水平方向上对相邻的两个顶点进行线性插值，得到一个中间值。然后，在垂直方向上对相邻的两个中间值进行线性插值，计算出最终的噪声值。

> **梯度向量场**（Gradient Vector Field）是一个在**二维或三维空间**中定义的向量场。它在每个点上定义了一个向量，该向量指向函数在该点处的最大变化率方向。 
> 
> 在数学和物理学中，梯度通常表示函数在给定点上的变化率和方向。对于标量函数，梯度是一个向量，指向函数在给定点上的最大变化率方向。梯度向量场则是将这个概念扩展到整个空间的函数。



### Simplex 噪声

Simplex 噪声是对 Perlin 噪声的改进，最简单的理解在于将**四边形的计算转变为三角形的计算**。其主要优化包括以下几点：

1. **更好的质量**：Simplex 噪声在生成的噪声模式中具有更好的连续性和平滑性。相对于 Perlin 噪声，Simplex 噪声在生成的噪声中没有明显的方向性偏差，减少了生成图案中的伪像和棱角。

2. **更高的性能**：Simplex 噪声相对于 Perlin 噪声具有更高的计算效率。Simplex 噪声的计算过程中涉及的操作更简单，减少了计算量，因此在实时渲染和计算密集型应用中更具优势。

3. **更少的参数**：Simplex 噪声相对于 Perlin 噪声来说，使用的参数更少。它只需要一个输入向量作为参数，而 Perlin 噪声需要更多的参数（如梯度表、插值方法等）来控制生成的噪声。

总的来说，Simplex 噪声在质量、性能和参数使用方面相对于 Perlin 噪声进行了优化。它在生成连续且无规律的噪声模式方面更加出色，并且在实时渲染和计算密集型应用中更加高效。


```js
#ifdef GL_ES
precision mediump float;
#endif
uniform vec2 u_resolution;
uniform vec2 u_mouse;
uniform float u_time;

vec2 skew(vec2 st){
    vec2 r=vec2(0.);
    r.x=1.1547*st.x;
    r.y=st.y+.5*r.x;
    return r;
}

vec3 simplexGrid(vec2 st){
    vec3 xyz=vec3(0.);
    vec2 p=fract(skew(st));
    if(p.x>p.y){
        xyz.xy=1.-vec2(p.x,p.y-p.x);
        xyz.z=p.y;
    }else{
        xyz.yz=1.-vec2(p.x-p.y,p.y);
        xyz.x=p.x;
    }
    return fract(xyz);
}

void main(){
    vec2 st=gl_FragCoord.xy/u_resolution.xy;
    vec2 repeat=vec2(5.,5.);// 设置重复的次数
    st*=repeat;
    vec3 color=simplexGrid(st);
    gl_FragColor=vec4(color,1.);
}
```

### 细胞噪声（Cellular Noise）

细胞噪声（Cellular Noise），也被称为 **Worley 噪声或 Voronoi 噪声**，是一种计算机生成的噪声模式。它是通过将空间划分为规则的**网格单元**，**并计算每个点到最近的网格点的距离来生成的**。

在细胞噪声中，每个网格单元都有一个随机的**特征点**（也称为种子点）。对于给定的空间位置，计算该位置到最近的特征点的距离。这些距离可以表示为欧几里得距离、曼哈顿距离或其他度量方式。最终，通过对距离进行某种操作（例如取最小值、最大值、平均值等），可以得到最终的噪声值。

细胞噪声在计算机图形学中被广泛应用，特别是在生成自然纹理、模拟自然形态和产生随机模式等方面。它可以用于创建山脉、云层、地形、纹理等效果，以及在渲染和游戏开发中模拟自然的纹理和形态。细胞噪声的特点是具有**明显的细节和结构**，可以产生各种有趣的视觉效果。

```js
#ifdef GL_ES
precision mediump float;
#endif

uniform vec2 u_resolution;
uniform vec2 u_mouse;
uniform float u_time;

void main(){
    vec2 st=gl_FragCoord.xy/u_resolution.xy;
    st.x*=u_resolution.x/u_resolution.y;
    
    vec3 color=vec3(.0,.15,0.);
    vec2 point[5];
    point[0]=vec2(.83,.75);
    point[1]=vec2(.60,.07);
    point[2]=vec2(.28,.64);
    point[3]=vec2(.31,.26);
    point[4]=u_mouse/u_resolution;
    
    float m_dist=1.;
    
    for(int i=0;i<5;i++){
        float dist=distance(st,point[i]);
        
        m_dist=min(m_dist,dist);
    }
    
    color+=m_dist;
    gl_FragColor=vec4(color,1.);
}
```

定义了一个包含 5 个点坐标的数组 `point`，这些点将用于计算当前像素与这些点之间的距离。其中，前 4 个点是预定义的，最后一个点是根据鼠标位置动态计算的。然后，通过循环遍历这些点，计算当前像素与每个点之间的距离，并将最小距离存储在 `m_dist` 变量中。最后，将 `m_dist` 添加到 `color` 变量中，并将结果作为最终的颜色输出。

## FBM 分形布朗运动（Fractal Brownian Motion）

分形布朗运动是一种分形噪声生成技术，用于创建具有自相似结构的复杂纹理、地形等效果。FBM 的主要特点是运动路径具有分形结构，即无论放大多少倍，都会看到类似的形状。

FBM 的原理基于**多层次的叠加噪声，通过调整这些噪声层级的频率、振幅和权重，生成具有分形特性的复杂纹理**。

FBM 的诞生主要为了使用不同类型的噪声、控制噪声的变化使其产生丰富的细节特征，我们就可以模拟更真实的地面、云彩、洋流、烟雾等。

```js
precision mediump float;

uniform float u_time;
uniform vec2 u_resolution;

float random(vec2 st){
    return fract(sin(dot(st.xy,vec2(12.9898,78.233)))*43758.5453);
}

// 柏林噪声
float noise(vec2 st){
    vec2 i=floor(st);
    vec2 f=fract(st);
    
    float a=random(i);
    float b=random(i+vec2(1.,0.));
    float c=random(i+vec2(0.,1.));
    float d=random(i+vec2(1.,1.));
    
    vec2 u=f*f*(3.-2.*f);
    return mix(a,b,u.x)+(c-a)*u.y*(1.-u.x)+(d-b)*u.x*u.y;
}

float fbm(vec2 st){
    float value=0.;
    float amplitude=.5;
    float frequency=0.;
    
    for(int i=0;i<5;i++){
        value+=amplitude*noise(st);
        st*=2.;
        amplitude*=.5;
    }
    
    return value;
}

void main(){
    vec2 st=gl_FragCoord.xy/u_resolution.xy;
    st*=8.;// 控制分形的细节程度
    
    float fractalValue=fbm(st);
    
    vec3 color=vec3(fractalValue);
    
    gl_FragColor=vec4(color,1.);
}
```

### Domain Warping 实现动态效果

使用 Domain Warping（域扭曲）可以在 fBm 函数中实现动态效果。域扭曲是一种技术，通过对输入坐标进行扭曲来改变生成的噪声图案。以下是一个示例代码，演示了如何在片段着色器中使用域扭曲来实现动态效果：

```js
//...
vec2 warp(vec2 st){
    float time=u_time*1.1;
    float strength=.5;
    
    vec2 offset=vec2(
        fbm(st+vec2(cos(time),sin(time))*strength),
        fbm(st+vec2(cos(time+1.),sin(time+1.))*strength)
    );
    
    return st+offset;
}

void main(){
    vec2 st=gl_FragCoord.xy/u_resolution.xy;
    st*=8.;
    
    st=warp(st);
    
    float fractalValue=fbm(st);
    
    vec3 color=vec3(fractalValue);
    
    gl_FragColor=vec4(vec3(.1,.9,.99)*color,1.);
}
```

在代码中我添加了一个 `warp` 函数来实现域扭曲。在 `warp` 函数中，我使用两个嵌套的 fBm 函数来计算偏移量，并将其添加到输入坐标上。通过调整时间变量和扭曲强度，你可以控制扭曲的速度和强度。

## 总结

- **伪随机**：可以简单理解为存在某种规律的随机性算法。
- **Perlin 噪声**：原理基于梯度的插值。它使用梯度向量在每个点上的权重来插值，以产生平滑而连续的噪声。
- **Simplex 噪声**：对 Perlin 的优化，具体表现将四边形的计算转变为三角形的计算。
- **细胞噪声**：通过将空间划分为规则的网格单元，并计算网格内的种子点到最近的种子点的距离来生成的结构。
- **FBM 分形布朗运动**：多层次的叠加噪声，通过调整这些噪声层级的频率、振幅和权重，生成具有分形特性的复杂纹理。
