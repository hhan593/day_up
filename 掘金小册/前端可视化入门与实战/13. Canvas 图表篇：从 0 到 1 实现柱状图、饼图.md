```js
       // 绘制线段
        function drawLine(ctx, startX, startY, endX, endY, color) {
            // 存储状态
            ctx.save();
            ctx.strokeStyle = color;
            ctx.lineWidth = 0.2
            ctx.beginPath();
            ctx.moveTo(startX, startY);
            ctx.lineTo(endX, endY);
            ctx.stroke();
            // 回退到刚绘制的状态，防止样式等影响其他绘图
            ctx.restore();
        }
        // 绘制矩形
        function drawBar(
            ctx,
            upperLeftCornerX,
            upperLeftCornerY,
            width,
            height,
            color
        ) {
            ctx.save();
            ctx.fillStyle = color;
            ctx.fillRect(upperLeftCornerX, upperLeftCornerY, width, height);
            ctx.restore();
        }

```

`drawLine` 函数用于绘制线段，接收一个 2D 绘图上下文对象 `ctx` ，起始点的 x 和 y 坐标，终点的 x 和 y 坐标，以及线段的颜色。在函数内部，首先通过 `ctx.save()` 保存当前的绘图状态，然后设置线段的样式，包括颜色和线宽。接着使用 `ctx.beginPath()` 开始绘制路径，使用 `ctx.moveTo()` 和 `ctx.lineTo()` 指定起始点和终点，最后使用 `ctx.stroke()` 进行绘制。最后通过 `ctx.restore()` 回退到刚绘制的状态，以防止样式等影响其他绘图。 

`drawBar` 函数用于绘制矩形，接收一个 2D 绘图上下文对象 `ctx` ，矩形左上角的 x 和 y 坐标，矩形的宽度和高度，以及矩形的颜色。在函数内部，首先通过 `ctx.save()` 保存当前的绘图状态，然后设置矩形的填充颜色。接着使用 `ctx.fillRect()` 绘制矩形，指定左上角的坐标、宽度和高度。最后通过 `ctx.restore()` 回退到刚绘制的状态，以防止样式等影响其他绘图。

drawGridLines() {
    let gridValue = 0;
    while (gridValue <= this.maxValue) {
        // 限制范围计算实际的 Y 坐标
        const gridY =
            this.canvasActualHeight * (1 - gridValue / this.maxValue) +
            this.options.padding;
        drawLine(

            0,
            gridY,
            this.canvas.width,
            gridY,

        );

        // 绘制Y轴文本

        this.ctx.restore();

### 绘制柱子


```js
drawBars() {
    // 用于计算 x 坐标
    let barIndex = 0;
    // 总数
    const numberOfBars = Object.keys(this.options.data).length;
    // bar 尺寸
    const barSize = this.canvasActualWidth / numberOfBars;
    const values = Object.values(this.options.data);
    for (const val of values) {
        // 计算高度百分比
        const barHeight = Math.round((this.canvasActualHeight * val) / this.maxValue);
        const x = this.options.padding + barIndex * barSize
        const y = this.canvas.height - barHeight - this.options.padding
        if (this.barInfo.length < numberOfBars) {
            this.barInfo.push({ x, y, width: barSize, height: barHeight })
        }

        // 绘制
        drawBar(
            this.ctx,
            x,
            y,
            barSize,
            barHeight,
            this.colors[barIndex]
        );
        barIndex++;
    }
}
```

在 `drawBars` 方法中，我们使用循环遍历柱状图的数据，通过计算得到每个柱子的高度和位置。具体解析如下：

-  `barIndex` 用于计算柱子的 x 坐标，初始值为 0。
-  `numberOfBars` 表示柱状图的总数，通过获取 `this.options.data` 的键的数量得到。
-  `barSize` 表示每个柱子的尺寸，通过将画布的实际宽度除以柱状图的总数得到。
-  `values` 是柱状图数据中的值的数组，通过获取 `this.options.data` 的值得到。

在循环中，我们根据柱子的索引和尺寸计算出柱子的 x 坐标 `x` ，以及柱子的 y 坐标 `y`。`y` 坐标的计算是将柱子的高度从画布底部开始向上计算，并考虑到边距。

如果柱子信息数组 `this.barInfo` 的长度小于总数，我们将柱子的位置信息（x、y、宽度、高度）添加到数组中，方便我们后续计算鼠标位置是否悬浮在当前柱子上。

### 柱子 hover 效果

如果我们想实现动态的动画效果，往往需要使用 `requestAnimationFrame`。`requestAnimationFrame` 主要用于创建动画效果或需要在下一次浏览器重绘之前执行的操作。 

对于静态绘制的 Canvas 图表，例如柱状图、折线图等，通常可以在一次绘制中完成，不需要使用 `requestAnimationFrame` 。比如 hover 效果，我们完全可以直接当悬浮对应柱子的时候，重新执行 `drawBars` 绘制，但这样同时会存在重叠样式问题。

接下来，我们将原有的静态绘制改为动态绘制：

```js
draw() {
    this.ctx.clearRect(0, 0, this.canvas.width, this.canvas.height);
    // dom 渲染是否是第一次
    if (this.firstRender) {
        this.drawTitle();
        this.drawLabels()
        this.drawLegend();
        this.firstRender = false
    }
    this.drawGridLines();
    this.drawBars();
    requestAnimationFrame(this.draw.bind(this))
}
```

对于 DOM 我们不需要重复绘制，对于 Canvas 来说，我们一定要在每次绘制前用 `clearRect` 清理画布。

接下来我们只要知道鼠标的位置，根据矩形边距的判定方法来识别是否触发 hover 效果。

### 实现步骤小结

步骤稍微有点多，所以我们也简单总结下这整体的绘制步骤：

**`生成 canvas 上下文 -> 准备阶段（封装基础图元方法等工具类） -> 自定义配置阶段 -> 定义图表类 -> 绘制静态效果 -> 绘制动态效果`**

通过掌握基础的步骤，可以让我们减少不必要的错误。

为了更好的解耦，在准备阶段我们甚至可以拓展事件类、图元类、比例尺类等，然后将所有这些类挂载在**注册类**上。使用注册类其实就是一个注册中心统一方便管理，我们可以处理注册、卸载、通用逻辑封装等。

在图表类上，我们只要根据传参将注册类实例化后得到需要用到的**工具类**。这样**图表类**我们只需要处理挂载工具类、配置、渲染，而其他的图表类型我们可以轻松复用之前的工具类。

### 绘制扇形

绘制扇形如下：

```js
drawSlices() {
    let colorIndex = 0;
    let startAngle = -Math.PI / 2;

    for (let categ in this.options.data) {
        let val = this.options.data[categ];
        let sliceAngle = (2 * Math.PI * val) / this.totalValue;

        drawPieSlice(
            this.ctx,
            this.canvas.width / 2,
            this.canvas.height / 2,
            this.radius,
            startAngle,
            startAngle + sliceAngle,
            this.colors[colorIndex % this.colors.length]
        );
        startAngle += sliceAngle;
        colorIndex++;
    }
}
```

首先初始化了颜色索引  `colorIndex`  和起始角度  `startAngle`。在循环中，获取当前属性的值 `val` ，然后计算出当前扇形的角度  `sliceAngle` ，即该属性值在总数值中所占的比例对应的角度。接下来调用  `drawPieSlice`  函数来绘制扇形。

然后，更新  `startAngle`  的值，使其加上当前扇形的角度，以便绘制下一个扇形。同时，增加  `colorIndex`  的值，以便获取下一个颜色。



### 绘制引导线

绘制引导线，核心在于计算各个扇形弧度的中心点，然后再偏移一个更大的圆形，这样我们能获得圆的边缘的中心点坐标、偏移圆的中心点坐标，然后我们计算标注在图表的左右位置，得到同为水平 x 轴的引线末端的坐标，最后根据左右位置做文本的偏移。


```js
drawGuideLine() {
    let index = 0;
    let startAngle = -Math.PI / 2;
    for (let k in this.options.data) {
        let val = this.options.data[k];
        const angle = (val / 100) * 360;
        let sliceAngle = (angle * Math.PI) / 180;
        const midAngle = startAngle + sliceAngle / 2;
        const x = this.canvas.width / 2 + Math.cos(midAngle) * this.radius;
        const y = this.canvas.height / 2 + Math.sin(midAngle) * this.radius;
        startAngle += sliceAngle;

        // 绘制弧度中心位置，演示使用
        this.ctx.beginPath();
        this.ctx.fillColor = 'red'
        this.ctx.arc(x, y, 10, 0, 2 * Math.PI)
        this.ctx.fill();
        this.ctx.stroke();
        this.ctx.closePath();
        
        index++;
    }
}
```

通过将角度乘以 `Math.PI` 并除以 180。这样，得到了当前扇形的弧度。

### 如何 hover 修改鼠标样式

如何 hover 修改鼠标样式？我们只要计算 client 坐标与圆形的距离是否小于半径即可。

接下来编写一个判断坐标和圆形是否碰撞的方法：

```js
function checkCollision(x, y, circleX, circleY, radius) {
  const distance = Math.sqrt(Math.pow(circleX - x, 2) + Math.pow(circleY - y, 2));
  if (distance <= radius) {
    // 发生碰撞
    return true;
  }
  // 未发生碰撞
  return false;
}
```

处理鼠标移动的逻辑：

```js
onMouseMove(e) {
    const { clientX, clientY } = e
    const cx = clientX - this.canvas.offsetLeft
    const cy = clientY - this.canvas.offsetTop

    const centerX = myCanvas.width / 2;
    const centerY = myCanvas.height / 2;
    // 计算鼠标样式
    const isHover = checkCollision(cx, cy, centerX, centerY, this.radius)
    document.body.style.cursor = isHover ? 'pointer' : 'default'
}
```



### 弹性动画效果

接下来我们将要实现 hover 对应的扇形区域，会展示有速率变化、外阴影、扇形区域外扩的弹性动画效果。

首先通过 requestAnimation 来处理动态绘制：

接着我们需要定位鼠标在哪个扇形位置，记录为 hoverId，然后修改 radius 即可。

```js
   constructor(options) {
        //...
        
        // 速率
        this.v = 0.5
        // 半径
        this.r = this.radius
        // 未hover的id
        this.hoverId = -1
        // 当前hover的角度
        this.deg = 0
  }
  setV() {
        if (this.isHover) {
            this.r = Math.min(this.r + this.v, 140)
        } else {
            this.r = Math.max(this.r - this.v, this.radius)
        }
        return this.r
  }
  onMouseMove(e) {
        const { clientX, clientY } = e
        const cx = clientX - this.canvas.offsetLeft
        const cy = clientY - this.canvas.offsetTop

        const centerX = myCanvas.width / 2;
        const centerY = myCanvas.height / 2;
        // 计算鼠标样式
        this.isHover = checkCollision(cx, cy, centerX, centerY, this.radius)
        document.body.style.cursor = this.isHover ? 'pointer' : 'default'

        // 计算弧度
        const angle = Math.atan2(cy - centerY, cx - centerX);

        // 计算角度
        let deg = (angle * 180) / Math.PI + 90;
        if (deg < 0) {
            deg += 360;
        }

        // 计算角度得到 hoverId
        let index = 0;
        let oldAngle = 0
        for (let k in this.options.data) {
            let val = this.options.data[k];
            let newAngle = oldAngle + (val / 100) * 360; //每块的角度
            if (oldAngle < deg && deg < newAngle) {
                // 连续hover重置r来实现过渡
                if (this.hoverId !== index) {
                    this.r = this.radius
                }
                this.hoverId = index
            }
            oldAngle = newAngle
            index++;
        }
        this.deg = deg
    }
```


然后我们在绘制扇形的地方，根据 index 是否是 hoverId，根据 v 做速率变化控制 r 的动态半径的更新。

function drawPieSlice(
    ctx,
    centerX,
    centerY,
    radius,
    startAngle,
    endAngle,
    fillColor,
    shadow
) {
    ctx.save();
    ctx.fillStyle = fillColor;
    ctx.beginPath();
    ctx.moveTo(centerX, centerY);
    ctx.arc(centerX, centerY, radius, startAngle, endAngle);
    ctx.closePath();
    if (shadow) {
        ctx.shadowBlur = 20;
        ctx.shadowOffsetX = 0;
        ctx.shadowOffsetY = 0;
        ctx.shadowColor = "rgba(0, 0, 0, 0.5)";
    }
    ctx.fill();
    ctx.restore();
}
