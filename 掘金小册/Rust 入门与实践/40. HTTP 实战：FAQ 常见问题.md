# 编译指定 cargo 版本

1.  所有项目指定版本

```Bash
rustup default nightly # nightly 可以改成别的 
```

2.  单个项目指定版本

创建 rust-toolchain.toml

```Rust
[toolchain]
channel = "nightly"
```

# SSL 报错

1.  编译报错

```Bash
apt install libssl-dev
```

2.  自己正常运行，发给别人报错

静态编译，reqwest SSL 改为 vendored，Linux 可以使用 musl 编译

```Rust
reqwest = { version = "0.11", features = ["native-tls-vendored"] }
```

  


# 控制日志格式

Cargo.toml

```TOML
tracing = "0.1"
tracing-subscriber = { version = "0.3", features = ["fmt", "local-time"] }
time = { version = "0.3", features = ["macros", "local-offset"] }
```

初始化日志收集器

```Rust
    // 最简单的方式
    // tracing_subscriber::fmt::init();
    
    // 设置日志格式
    use tracing_subscriber::layer::SubscriberExt;
    use tracing_subscriber::util::SubscriberInitExt;
    tracing_subscriber::registry()
        .with(
            tracing_subscriber::fmt::layer()
                .with_target(true)
                .with_timer(tracing_subscriber::fmt::time::OffsetTime::new(
                    time::UtcOffset::__from_hms_unchecked(8, 0, 0),
                    time::macros::format_description!(
                        "[year]-[month]-[day] [hour]:[minute]:[second]"
                    ),
                )),
        )
        .init();
```

设置日志等级环境变量

```Bash
export RUST_LOG=info
```
