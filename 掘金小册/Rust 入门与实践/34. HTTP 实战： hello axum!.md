项目名：hello-axum

在这个章节中，会从零开始创建 axum 项目，编写最基本的 hello handler，在 http://localhost:3000/hello 被访问时，打印日志（包括 query 参数），并返回字符串 "hello"。

3.  修改 src/lib.rs，删除原有内容，编写 hello handler

> 代码中 `query` 的类型为 `Query<HashMap<String, String>>`，如果需要获取里面的值，可以使用 `let value = query.0.get("key");`。
>
> 如果参数改为 `Query(query): Query<HashMap<String, String>>`（模式匹配语法），query 类型为 `HashMap<String, String>`，可以直接 `let value = query.get("key");`，在后续章节的参数中，会经常用到这种写法。
>
> 如有疑问可以使用 IDE 进入 Query 源码查看 Query 定义、搜索“Rust 模式匹配 解构”。

```Rust
use axum::extract::Query;
use std::collections::HashMap;

pub async fn hello(query: Query<HashMap<String, String>>) -> &'static str {
    tracing::info!("hello, query: {query:?}"); // 日志输出参数，比如 /hello?a=1&b=2
    "hello"
}
```

5.  创建 src/bin/server.rs 文件

刚创建时 IDE 可能会提示 Files is not included in module tree, analysis is not available，编写 main 函数后，点击右上角小图标即可解决。

```Rust
use axum::routing::get;
use axum::{Router, Server};
use hello_axum::hello;
use std::net::SocketAddr;

#[tokio::main]
async fn main() {
    tracing_subscriber::fmt::init();

    let app = Router::new().route("/hello", get(hello));

    // run it
    let addr = SocketAddr::from(([0, 0, 0, 0], 3000));
    tracing::info!("listening on {addr}");
    Server::bind(&addr)
        .serve(app.into_make_service())
        .await
        .unwrap();
}
```
