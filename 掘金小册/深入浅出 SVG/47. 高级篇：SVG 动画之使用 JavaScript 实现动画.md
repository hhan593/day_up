实现上面这个效果，我们需要一段像下面这样的 HTML 结构：

我们使用内联 SVG 的方式来创建 Blob，因为这样更偏于 [CSS 和 JavaScript 来操作 SVG](https://juejin.cn/book/7341630791099383835/section/7351339840161447945)。你可能会注意到我使用了 CSS 自定义属性来设置线[性渐变（`<linearGradient>`）的起始和停止颜色](https://juejin.cn/book/7341630791099383835/section/7354948936039137289)，但它们还没有任何值。没关系，稍后我们会在 CSS 给其设置一个初始值，并使用 JavaScript 动态设置它们的值。

  


```CSS

    width: 90vmin;
    aspect-ratio: 1;

```

  


在我们这个示例中，Blob 形状是一个单一的 SVG `<path>` 。`<path>` 是一个强大的 SVG 元素，可以用来绘制各种形状。到目前为止，`<path>` 元素具有一个空的 `d` 属性，它的值是用来定义路径形状的数据，这个数据由一系列的命令符和坐标位置组合而成。和之前的示例有所不同，`d` 属性的值将由 JavaScript 动态创建，因为我们需要一个动态化的 Blob 形状。与此同时，我们将定义的线性渐变填充到该路径中。

  


接下来，我们可以不需要再触碰任何 SVG 标记，因为我们可以简单地操作我们的自定义属性并更新 `<path>` 数据属性。当然，这一切的工作都将交给 JavaScript 来完成。

  


为了创建我们的 Blob，我在这里使用了 [SplineJS](https://github.com/georgedoescode/splinejs#readme) 和 [Simplex-noise](https://github.com/jwagner/simplex-noise.js) 两个库：

  


-   SplineJs：用于通过一组点绘制平滑曲线
-   Simplex-noise：是一个 Simplex 噪声库，用于生成平滑的、自相似的随机值

  


在 CodePen，我们可以像下面这样来导入这两个 JavaScript 库：

  


```JavaScript
import { spline } from "https://cdn.skypack.dev/@georgedoescode/spline@1.0.1";
import SimplexNoise from "https://cdn.skypack.dev/simplex-noise@2.4.0";
```

  


当然，你也可以根据使用文档以及你自己的开发环境，选择适当的导入方式，这里就不做过多的阐述。

  


现在，我们可以开始使用 JavaScript 来创建我们的 Blob 形状了！下面是我们创建动画化 Blob 形状所需的几个关键步骤：

  


-   1️⃣：在圆的周长上创建六个等距点，你也可以根据需要设置更多的点，但这个点数越多，路径数据就越复杂，动画化时所需计算也更耗时，这有可能会影响到 Web 页面的性能。请谨慎选择
-   2️⃣：随着时间的推移，改变每个点的 `{x,y}` 值
-   3️⃣：使用 `spline()` 在每个点之间绘制平滑曲线
-   4️⃣：重复步骤 2️⃣ 和 3️⃣

  


正如上面步骤 1️⃣ 所述，我们需要在一个圆周上绘制六个等距点。为此，我们可以添加一个函数 `createPoints()` ：

  


```JavaScript
// 创建初始点数组
// numPoints: 圆周上的等距点的数量
// radius : 圆半径
const createPoints = (numPoints = 6, radius = 75) => {
    // 计算每个点之间的角度步长
    const angleStep = (Math.PI * 2) / numPoints;
  
    // 使用 Array.from 创建点数组
    return Array.from({ length: numPoints }, (_, i) => {
        const theta = (i + 1) * angleStep; // 当前点的角度
        const x = 100 + Math.cos(theta) * radius; // 计算 x 坐标
        const y = 100 + Math.sin(theta) * radius; // 计算 y 坐标
        
        // 返回包含坐标和噪声偏移的点对象
        return {
            x,
            y,
            originX: x, // 初始 x 坐标
            originY: y, // 初始 y 坐标
            noiseOffsetX: Math.random() * 1000, // 随机 x 噪声偏移
            noiseOffsetY: Math.random() * 1000 // 随机 y 噪声偏移
        };
    });
};
```

  


然后我们可以像这样初始化我们的 Blob 点：

  


```JavaScript
// 初始化点数组
const points = createPoints();
```

  


到目前为止，虽然在圆周上绘制了一些点，但我们仍然看不到任何东西。我们还需要继续往 JavaScript 中添加代码。

  


让我们使用 `requestAnimationFrame` 添加一个动画循环：

  


```JavaScript
// 动画函数
const animate = () => {
    // 更新 path 元素的 d 属性
    path.setAttribute("d", spline(points, 1, true));
    // 更新点位置和颜色
    updatePoints();
    updateColors();
    // 请求下一帧动画
    requestAnimationFrame(animate);
};
```

  


在 `animate()` 动画函数中，我们主要会做四件事情：

  


-   更新 `<path>` 元素的 `d` 属性。在这里使用了 `spline()` 函数，它会基于圆周上的等距点生成一个平滑的连续的曲线
-   通过 `updatePoints()` 函数更新点的位置
-   通过 `updateColors()` 函数更新点的颜色
-   使用 `requestAnimationFrame(animate)` 请求下一帧动画

  


因此，我们需要再两个函数，即 `updatePoints()` 和 `updateColors()` 。先来看 `updatePoints()` 函数：

  


```JavaScript
// 创建 SimplexNoise 实例
const simplex = new SimplexNoise();

// 辅助函数：计算噪声值
const noise = (x, y) => simplex.noise2D(x, y);

// 更新点的位置
const updatePoints = () => {
    points.forEach(point => {
        // 计算新的噪声值
        const nX = noise(point.noiseOffsetX, point.noiseOffsetX);
        const nY = noise(point.noiseOffsetY, point.noiseOffsetY);
        
        // 将噪声值映射到新的坐标范围
        point.x = map(nX, -1, 1, point.originX - 20, point.originX + 20);
        point.y = map(nY, -1, 1, point.originY - 20, point.originY + 20);
        
        // 更新噪声偏移值
        point.noiseOffsetX += noiseStep;
        point.noiseOffsetY += noiseStep;
    });
};
```

  


注意，我们在这里使用`SimplexNoise` 库来创建平滑的波动，改变 Blob 每个点的 `{x,y}` 值。首先初始化 `SimplexNoise` ，然后使用 `simplex.noise2D()` 动态调整每个 Blob 点的 `{ x, y }` 值。

  


接着创建 `updateColors()` 函数，来动态调整线性渐变的颜色：

  


```JavaScript
// 初始化颜色噪声偏移和噪声步长
let hueNoiseOffset = 0;
const noiseStep = 0.005;

// 辅助函数：将一个数从一个范围映射到另一个范围
const map = (n, start1, end1, start2, end2) => ((n - start1) / (end1 - start1)) * (end2 - start2) + start2;

// 更新颜色
const updateColors = () => {
    // 计算新的颜色噪声值
    const hueNoise = noise(hueNoiseOffset, hueNoiseOffset);
    const hue = map(hueNoise, -1, 1, 0, 360);
    // 设置 blob 元素的渐变颜色
    blobEle.style.setProperty("--startColor", `hsl(${hue} 100% 75%)`);
    blobEle.style.setProperty("--stopColor", `hsl(${hue + 60} 100% 75%)`);
    // 更新颜色噪声偏移值
    hueNoiseOffset += noiseStep / 6;
};
```

  


你现在调用 `animate()` 函数启动动画，就应该有一个流畅的动画化 Blob，它根据噪声变化形状并定期改变渐变颜色：

  


```JavaScript
// 启动动画
animate();
```

## 创建圆形进度条

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/6b48719bf2874d1694209e1abb80cb54~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1140&h=570&s=1517701&e=gif&f=16&b=e6ebf3)

  


圆形进度条是种视觉上吸引人的进度指示器，常用于显示任务完成的百分比。在现代 Web 应用程序中，圆形进度条广泛用于各种场景，如下载进度、文件上传进度、表单填写进度等。它不仅提供了进度的直观显示，还能提升用户体验。

  


通常情况之下，一个圆形进度条主要由三部分组成：

  


-   外圈：通常是一个完整的圆环，作为进度条的背景。我喜欢将其称为轨道（Track）
-   内圈：表示当前的进度（Current Progress），会随着进度的增加而填充圆环
-   中心文本：显示具体的百分比数值或其他相关信息，增强用户的理解

圆形进度条的实现基于圆周的部分填充。通过计算填充部分的角度和圆周率（`π`），可以精确的表示进度的百分比。常见的实现方法有使用 HTML5 的 `<canvas>` 元素、SVG 图形或纯 CSS。不过，在这里，我们主要探讨 SVG 和 JavaScript 是如何实现圆形进度条。

  


构建圆形进度条的 HTML 结构不会太复杂：

  


```HTML
<div class="progress--circle">
    <svg viewBox="0 0 100 100" class="progress__bar">
        <circle cx="50" cy="50" r="50" class="progress__track" fill="none" stroke="var(--process-color)" stroke-width="3" stroke-linecap="round" stroke-opacity=".3"/>
        <circle cx="50" cy="50" r="50" class="progress__current" fill="none" stroke="var(--process-color)" stroke-width="3" stroke-linecap="round" stroke-opacity="1"/>
    </svg>
    <div class="progress__text">
        <span class="percent__int">0.</span>
        <span class="percent__dec">00</span>
    </div>
</div>
```

  


我们使用了两个 `<circle>` 元素来绘制圆环，第一个是要填充的路径（即外圈），第二个是将被动画填充的路径（即内圈）。在这个示例中，中心文本被分为整数和小数，分别应用了两个 `<span>` ，并应用了不同的 `font-size` 。

  


在 `<circle>` 元素上设置 `stroke-width` 、`stroke-opacity` 和 `stroke-linecap` 属性，为圆环应用描边效果。另外，`cx` ，`cy` 和 `r` 定义了圆环的大小。另外，整个圆形进度条我采用了 CSS Grid 布局。为了使整个 HTML 结构变得更简洁和清晰，可以将 `<circle>` 元素的描边样式放到 CSS 中定义。

  


```CSS
@layer demo {
    .progress--circle {
        --process-color: #aaff00;
        width: 50vmin;
        aspect-ratio: 1;
        display: grid;
        place-content: center;
    
        > * {
            place-self: center;
            grid-area: 1 / 1 / -1 / -1;
        }
    
        .progress__bar {
            display: block;
            width: 100%;
            height: 100%;
        }
    
        .percent__int {
            font-size: 28px;
        }
        .percent__dec {
            font-size: 12px;
        }
        
        circle {
            stroke: var(--process-color);
            stroke-width: 3;
            stroke-linecap: round;
            fill: none;
        }
    }
}
```

不难发现，圆环有被裁切的情况。因为我们在代码中设置了 `r` 属性的值为 `50` （圆的半径），`viewBox="0 0 100 100"` （画布大小），加上圆的描边宽度为 `3` ，因此整个圆形尺寸大于画布尺寸，造成圆环被裁剪的现象。

  


针对这个现象，较比快速的方式是将 `svg` 元素的 `overflow` 设置为 `visible` ，但这并不是我们所要的效果，因为圆环尺寸还是会大于 SVG 画布。另外一种方式，就是调整 `r` 的大小，要使 `r` 加上 `stroke-width` 的和等于或小于 `viewBox` 宽度或高度的一半。

  


```CSS
.progress--circle {
    --process-color: #aaff00;
    --stroke-width: 3;
}
circle {
    stroke: var(--process-color);
    stroke-width: var(--stroke-width);
    stroke-linecap: round;
    fill: none;
    r: calc(50 - var(--stroke-width));
}
```

  


注意，上面代码中 `r` 的值为 `calc(50 - var(--stroke-width))` ，其中 `50` 是 `viewBox` 的 `width` 值的一半，在这个示例中，`viewBox` 的 `width` 值为 `100` 。我要提醒大家的是，你需要根据你的实际需求来调整该值。

  


有了这个基础之后，我们就可以开始为圆环进度条添加动画效果。这个效果很简单，只要用好了 `<circle>` 的 `stroke-dasharray` 和 `stroke-dashoffset` 即可，这也就是大家所说的 SVG 的线条动画，[我们在上一节课对这个动画效果做过详细的阐述](https://juejin.cn/book/7341630791099383835/section/7368318500093296667#heading-14)。为了加强大家的印象，在这里简单地回顾一下。

  


`stroke-dasharray` 定义了找边中的虚线间隔模式。它可以接受最多四个值：

  


-   当设置为单个整数时（`stroke-dasharray: 10`），虚线和间隙大小相同
-   对于两个值（`stroke-dasharray: 10 5`），第一个应用于虚线，第二个应用于间隙
-   第三和第四种形式（`stroke-dasharray: 10 5 2` 和 `stroke-dasharray: 10 5 2 3`）将生成不同大小的虚线和间隙

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/b60bce0c3b9c4eb0ab09b3ed677ff781~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1339&h=968&s=657672&e=gif&f=548&b=3e423c)

  


而 `stroke-dashoffset` 属性则可以定义虚线的起始偏移量。通过改变这个值，可以控制虚线图案的起始位置。它也被设置为从 `0` 到圆的周长长度（`2πr`）。另外，还可以使用 `getTotalLength()` 来快速获取整个路径的总长度（即圆周长度）。

  


为了产生填充效果，我们将 `stroke-dasharray` 设置为圆的周长长度，这样整个长度将被填满，没有间隙。我们还将它以相同的值偏移，使其“隐藏”。然后，`stroke-dashoffset` 将根据过渡持续时间更新为相应的百分比值，根据填充描边。

  


```CSS
.progress--circle {
    --process-color: #aaff00;
    --stroke-width: 3;
}
 
.progress__current {
    --initialStroke: 0;
    --transitionDuration: 0;
    stroke-dasharray: var(--initialStroke);
    stroke-dashoffset: var(--initialStroke);
    transition: stroke-dashoffset var(--transitionDuration) ease;
}
```

  


接下来，开始进入 JavaScript。我们使用 JavaScript 来设置初始值并更新变量。其中 `--transitionDuration` 设置为 `900ms` 。而 `--initialStroke` 则设置为圆环的圆周长度，在这里使用 `getTotalLength()` 来获取。

  


```JavaScript
const progressCircle = document.querySelector(".progress--circle");
const progressCurrent = document.querySelector(".progress__current");
const circumference = progressCurrent.getTotalLength();
const transitionDuration = 900;

progressCurrent.style.setProperty(
    "--transitionDuration",
    `${transitionDuration}ms`
);
progressCurrent.style.setProperty("--initialStroke", circumference);
```

  


然后定义一个 `update` 函数，并使用 `requestAnimationFrame` 来调度该函数以实现平滑的动画效果：

  


```JavaScript
let start = null;

const update = (timestamp) => {
    // 如果start未定义，则将其设置为当前时间戳
    if (!start) start = timestamp;
  
    // 计算从开始到当前的时间差
    let progressTime = timestamp - start;

    // 如果时间差大于或等于100毫秒，则执行动画更新
    if (progressTime >= 100) {
        progressCurrent.style.strokeDashoffset = 50;
    } else {
        // 否则，继续请求下一帧动画
        requestAnimationFrame(update);
    }
};

// 请求第一帧动画，传入update回调函数
requestAnimationFrame(update);
```

现在，让我们根据“中心文本”内容的“百分比值”来计算 `strokeDashoffset` 的值。首先，我们在 `.progress--circle` 元素上定义一个 `data-*` 属性，其中 `*` 可以是任意你喜欢的名称。在这里，我使用 `data-note` 来表示：

  


```HTML
<div class="progress--circle" data-note="7.50">
    <svg viewBox="0 0 100 100" class="progress__bar">
        <circle cx="50" cy="50" r="50" class="progress__track" stroke-opacity=".3" />
        <circle cx="50" cy="50" r="50" class="progress__current" stroke-opacity="1" />
    </svg>
    <div class="progress__text">
        <span class="percent__int">0.</span>
        <span class="percent__dec">00</span>
    </div>
</div>
```

  


然后，在 JavaScript 通过 `dataset: element.dataset.*` 检索 `data-*` 。我们使用 `parseFloat` 方法将把由 `progressCircle.dataset.note` 返回的字符串转换为浮点数。偏移量表示达到最高分还差的百分比。因此，对于 `data-note = 7.50`，我们有 `(10 - 7.50) / 10 = 0.25`，这意味着周长长度应该偏移其值的 `25%`：

  


```JavaScript
const progressCircle = document.querySelector(".progress--circle");
const progressCurrent = document.querySelector(".progress__current");
const circumference = progressCurrent.getTotalLength();
const transitionDuration = 900;

let note = parseFloat(progressCircle.dataset.note);
let offset = circumference * (10 - note) / 10;

progressCurrent.style.setProperty(
  "--transitionDuration",
  `${transitionDuration}ms`
);
progressCurrent.style.setProperty("--initialStroke", circumference);

let start = null;

const update = (timestamp) => {
  // 如果start未定义，则将其设置为当前时间戳
  if (!start) start = timestamp;
  
  // 计算从开始到当前的时间差
  let progressTime = timestamp - start;

  // 如果时间差大于或等于100毫秒，则执行动画更新
  if (progressTime >= 100) {
    progressCurrent.style.strokeDashoffset = offset;
  } else {
    // 否则，继续请求下一帧动画
    requestAnimationFrame(update);
  }
};

// 请求第一帧动画，传入update回调函数
requestAnimationFrame(update);
```

在继续之前，让我们将描边过渡提取到其自己的方法中：

  


```JavaScript
// 获取包含进度条圆圈的元素
const progressCircle = document.querySelector(".progress--circle");
// 获取进度条当前进度的元素
const progressCurrent = document.querySelector(".progress__current");

// 定义动画持续时间（单位：毫秒）
const transitionDuration = 900;

// 从 HTML data 属性中获取 note 的值，并转换为浮点数
let note = parseFloat(progressCircle.dataset.note);

// 定义一个函数，用于执行进度条的动画
const strokeTransition = (progressCurrent, note) => {
    // 获取进度条的周长
    let circumference = progressCurrent.getTotalLength();
    // 计算进度条的偏移量，根据 note 的值设置
    let offset = (circumference * (10 - note)) / 10;

    // 设置 CSS 变量，定义初始的进度条周长和动画持续时间
    progressCurrent.style.setProperty("--initialStroke", circumference);
    progressCurrent.style.setProperty("--transitionDuration",`${transitionDuration}ms`);

    let start = null;

    // 定义动画函数
    const animateStroke = (timestamp) => {
        // 如果 start 未定义，则将其设置为当前时间戳
        if (!start) start = timestamp;
        // 计算从开始到当前的时间差
        let elapsed = timestamp - start;
    
        // 如果时间差大于或等于 100 毫秒，则执行动画更新
        if (elapsed >= 100) {
            progressCurrent.style.strokeDashoffset = offset;
        } else {
            // 否则，继续请求下一帧动画
            requestAnimationFrame(animateStroke);
        }
    };

    // 请求第一帧动画，传入 animateStroke 回调函数
    requestAnimationFrame(animateStroke);
};

// 执行进度条动画
strokeTransition(progressCurrent, note);
```

  


接下来，我们要做的是完成从 `0.00` 到 `data-note` 指定值的过渡效果。首先使用 `split()` 方法来分离整数部分和小数部分的值，并将它们转换为数字，然后作为参数传递给 `increaseNumber()` 函数，同时还传递 `progressCircle` 元素和一个标志，指示它是整数还是小数。

  


```JavaScript
const displays = document.querySelectorAll('.note-display');
const transitionDuration = 900;

displays.forEach(display => {
  let note = parseFloat(display.dataset.note);
  let [int, dec] = display.dataset.note.split('.');
  [int, dec] = [Number(int), Number(dec)];

  strokeTransition(display, note);

  increaseNumber(display, int, 'int');
  increaseNumber(display, dec, 'dec');
});
```

  


在 `increaseNumber()` 函数中，根据 `className` 选择 `.percent__int` 或 `.percent__dec` 元素，并且确定输出是否包含小数点。我们将 `transitionDuration` 设置为 `900ms`。现在，要将一个数字从 `0` 增加到 `7`，例如，持续时间必须被音符数除以 `900 / 7 = 128.57ms`。这个结果表示每次增加迭代将花费的时间。

  


```JavaScript
const increaseNumber = (progress, number, className) => {
    // 获取要更新的元素
    let element = document.querySelector(`.percent__${className}`);
    // 根据 className 确定是否添加小数点
    let decPoint = className === 'int' ? '.' : '';
    // 计算每个数字递增的时间间隔
    let interval = transitionDuration / number;
    // 初始化计数器
    let counter = 0;
    // 记录动画开始时间
    let start = null;

    // 定义动画函数
    const animate = (timestamp) => {
        // 如果 start 未定义，则将其设置为当前时间戳
        if (!start) start = timestamp;
        // 计算从开始到当前的时间差
        let elapsed = timestamp - start;

        // 计算应该递增多少次
        let steps = Math.floor(elapsed / interval);

        // 如果已经达到或超过目标数字，则设置为目标数字，并结束动画
        if (counter + steps >= number) {
            element.textContent = number + decPoint;
        } else {
            // 否则，更新计数器，设置文本内容，并继续动画
            counter += steps;
            element.textContent = counter + decPoint;
            requestAnimationFrame(animate);
        }
    };

  // 请求第一帧动画，传入 animate 回调函数
  requestAnimationFrame(animate);
}
```

  


最后对整个效果的所需要的 JavaScript 做一些优化，最终的 JavaScript 代码如下：

  


```JavaScript
// 获取包含进度条圆圈的元素
const progressCircle = document.querySelector(".progress--circle");
// 获取进度条当前进度的元素
const progressCurrent = document.querySelector(".progress__current");

// 定义动画持续时间（单位：毫秒）
const transitionDuration = 900;

// 从 HTML data 属性中获取 note 的值，并转换为浮点数
let note = parseFloat(progressCircle.dataset.note);
// 将 note 的值分成整数和小数部分，并将其转换为数字
let [int, dec] = progressCircle.dataset.note.split(".").map(Number);

// 定义一个函数，用于执行进度条的动画
const strokeTransition = (progressElement, noteValue) => {
    // 获取进度条的周长
    const circumference = progressElement.getTotalLength();
    // 计算进度条的偏移量，根据 note 的值设置
    const offset = (circumference * (10 - noteValue)) / 10;

    // 设置进度条的初始样式
    progressElement.style.strokeDasharray = circumference;
    progressElement.style.strokeDashoffset = circumference;

    // 触发重绘
    progressElement.getBoundingClientRect();

    // 等待下一个渲染帧来重新启动动画
    requestAnimationFrame(() => {
        // 设置动画过渡效果
        progressElement.style.transition = `stroke-dashoffset ${transitionDuration}ms ease-in-out`;
        progressElement.style.strokeDashoffset = offset;
    });
};

// 定义一个函数，用于增加数字
const increaseNumber = (displayElement, number, className) => {
    const element = displayElement.querySelector(`.percent__${className}`);
    const decPoint = className === 'int' ? '.' : '';
    const interval = transitionDuration / number;
    let counter = 0;
    let start = null;

    // 定义动画函数
    const animate = (timestamp) => {
        if (!start) start = timestamp;
        const elapsed = timestamp - start;
        const steps = Math.floor(elapsed / interval);
    
        if (counter + steps >= number) {
            element.textContent = number + decPoint;
        } else {
            counter += steps;
            element.textContent = counter + decPoint;
            requestAnimationFrame(animate);
        }
    };
    requestAnimationFrame(animate);
};

// 执行进度条动画和数字增加动画
strokeTransition(progressCurrent, note);
increaseNumber(progressCircle, int, "int");
increaseNumber(progressCircle, dec, "dec");
```

## 创建数字变形倒计时效果

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/8f0ec500555a44ae880e80481462f0ec~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1600&h=1200&s=273403&e=gif&f=34&b=000304)

  


数字倒计时效果也是 Web 上常见的一种效果，通常用于展示页面上的倒计时或者动态数字变化。接下来，我们来看一个关于数字变形倒计时的效果。它将应用到 SVG 的路径和滤镜相关的特性，以及SVG 路径特性提供的辅助函数 `getTotalLength()` 和 `getPointAtLength()` 。

  


在开始之前，简单的介绍一下 `getTotalLength()` 和 `getPointAtLength()` 函数，因为这两个函数很重要。`getTotalLength()` 和 `getPointAtLength()`，它们都存在于 `SVGGeometryElements` 上。

  


-   `getTotalLength()` 返回路径的长度
-   `getPointAtLength()` 接受路径上的距离并返回 `x` 和 `y` 坐标

  


利用这两个函数，我们可以沿着路径的位置放置一些其他图形元素，例如 `<circle>` 。这对于创建一些简单的粒子动画效果，或者借助 WebGL 创建复杂的粒子动画效果非常有用。在之前的课程中，我们就介绍了 [WebGL 如何基于 SVG 路径来创建粒子动画效果](https://juejin.cn/book/7341630791099383835/section/7368317620015071282)，在整个效果制作当中，就离不开这两个函数。

实现上图这个效果，我们首先需要定义每个数字的路径数据。你可以使用诸如 Figma 这样的图形设计软件来绘制每个数字，并且导出相应的 SVG 代码。

  


```XML
<svg viewBox="0 0 256 256" class="gooey">
```

你可能发现了，示例中每个变形的数字是由很多个圆组成的，每个圆都被移动到数字路径的不同位置。这是我们整个效果中比较关键的一部分。简单地说，通过 SVG 路径特性提供的 `getTotalLength()` 和 `getPointAtLength()` 函数，我们可以根据每个数字路径的总长度来计算每个圆在路径上的位置，并且动态创建圆形 `<circle>` 。

  


在创建圆形的过程中，我们还借助 GSAP 库动态化变换每个数字。详细的代码如下：

  


```JavaScript
// 获取包含 gooey 效果的元素
const gooeyEle = document.querySelector('.gooey');

// 配置选项：圆圈数量和半径
const opts = {
    num: 31, // 圆圈数量
    radius: 20 // 圆圈半径
};

// 初始化函数
const init = () => {
    // 获取包含所有圆圈的元素
    const circles = document.querySelector('.circles');
    // SVG 命名空间，用于创建 SVG 元素
    const namespace = 'http://www.w3.org/2000/svg';
  
    // 创建多个圆圈元素并添加到 SVG 容器中
    for (let i = 0; i < opts.num; i++) {
        const circle = document.createElementNS(namespace, 'circle'); // 创建圆圈元素
        circle.classList.add('circle'); // 添加 class 名
        circle.setAttribute('r', opts.radius); // 设置圆圈半径
        circle.setAttribute('cx', 128); // 设置圆圈初始 x 坐标
        circle.setAttribute('cy', 128); // 设置圆圈初始 y 坐标
        circles.appendChild(circle); // 将圆圈添加到 SVG 容器中
    }
  
    // 初始化倒计时数字
    let n = 10;
    // 使用 setInterval 定时调用动画函数
    setInterval(() => {
        n = n - 1; // 倒计时减 1
        animate(n); // 执行动画
        if (n === 0) n = 10; // 倒计时到 0 后重置为 10
    }, 1200); // 每隔 1200 毫秒调用一次
};

// 调用初始化函数
init();

// 动画函数
const animate = (n) => {
    // 获取所有路径和圆圈元素
    const paths = document.querySelectorAll('.path');
    const circles = document.querySelectorAll('.circle');
    if (!paths[n]) return; // 如果没有对应路径则返回
    const length = paths[n].getTotalLength(); // 获取路径总长度
    const step = length / opts.num; // 计算每个圆圈之间的距离
  
    // 对每个圆圈进行动画设置
    for (let i = 0; i < opts.num; i++) {
        const { x, y } = paths[n].getPointAtLength(i * step); // 获取路径上指定位置的点
        const randomColor = `hsl(${Math.random() * 360}, 100%, 50%)`; // 生成随机 HSL 颜色
        
        // 使用 GSAP 库设置动画效果
        gsap.to(circles[i], {
          cx: x, // 设置圆圈新的 x 坐标
          cy: y, // 设置圆圈新的 y 坐标
          ease: 'power3.out', // 设置动画缓动效果
          fill: randomColor, // 设置圆圈填充颜色
          delay: i * 0.024 // 设置动画延迟时间
        });
    }
};

// 调用动画函数
animate();
```

### 第二步：创建第一个时间轴

```JavaScript
});
```

### 第五步：创建日夜交替动画

```JavaScript

    // 动画背景渐变，第一个停止点
    .to('#daytime-gradient stop:nth-of-type(1)', {
        duration: dayNightCycleTime, // 设置动画持续时间为总动画时间 dayNightCycleTime
        stopColor: '#060414',// 改变颜色到深夜的颜色
        ease: animationEase // 使用 animationEase 缓动函数
    }, 0)
    // 动画背景渐变，第二个停止点
    .to('#daytime-gradient stop:nth-of-type(2)', {
        duration: dayNightCycleTime,// 设置动画持续时间为总动画时间 dayNightCycleTime
        stopColor: '#001f3f',// 改变颜色到更深的颜色，以确保足够黑

    }, 0)

    .to('#nighttime-overlay', {

    }, 0)
    // 动画月亮，第一阶段
    .to('#moon', {
        duration: dayNightCycleTime / 2,// 设置动画持续时间为总动画时间的一半
        scale: 0.9,// 缩小月亮到90%大小
        attr: { cx: "174.5", cy: "202.5" },// 改变月亮的位置
        ease: animationEase // 使用 animationEase 缓动函数
    }, 0)
    // 动画月亮，第二阶段
    .to('#moon', {
        duration: dayNightCycleTime / 2,// 设置动画持续时间为总动画时间的一半
        scale: 0.9,// 缩小月亮到90%大小
        attr: { cx: "410.5", cy: "114.5" },// 改变月亮的位置
        ease: animationEase // 使用 animationEase 缓动函数
    }, 
    dayNightCycleTime / 2 // 在总动画时间的一半后开始
    ) 
    // 动画星星，第一阶段
    .to('#stars', {
        duration: dayNightCycleTime / 2, // 设置动画持续时间为总动画时间的一半
        opacity: 1 // 改变星星的透明度为完全显示
    },
     dayNightCycleTime / 2// 在总动画时间的一半后开始
    ) 
    // 从底部添加星星，第二阶段

        rotation: -15,// 逆时针旋转15度

    },
```

正如你所看到的，图中的小白点就是“雪花”，它们目前静止的。接下来，我们给雪花添加动画效果。

  


我们为雪花创建了两个单独的时间轴，以便能够在不同的持续时间内对它们进行动画。为了动画我们的雪花层，我们希望沿着垂直轴移动这两个层。通过不同的持续时间，我们将获得层以不同速度移动的外观。动画通过沿着垂直轴移动椭圆集合，直到第二个集合取代第一个集合来完成。然后我们重复动画。

  


```JavaScript
// 创建一个无限循环的时间轴动画实例，用于顶部雪花
const animationSnowTop = gsap.timeline({
    repeat: -1 // 设置动画重复次数为无限次
});

// 创建一个无限循环的时间轴动画实例，用于底部雪花
const animationSnowBottom = gsap.timeline({
    repeat: -1 // 设置动画重复次数为无限次
});

// 定义顶部雪花的动画
animationSnowTop.to(
    "#snow-top-layer", // 选择ID为"snow-top-layer"的元素
    { 
        duration: 7, // 动画持续时间为7秒
        attr: { 
            transform: "translate(24 -108)"  // 通过transform属性将雪花移动
        }, 
        ease: animationEase // 使用预定义的缓动效果
    }
);

// 定义底部雪花的动画
animationSnowBottom.to(
    "#snow-bottom-layer", // 选择ID为"snow-bottom-layer"的元素
    { 
        duration: 15, // 动画持续时间为15秒
        attr: { 
            transform: "translate(13 140)" // 通过transform属性将雪花移动
        }, 
        ease: animationEase // 使用预定义的缓动效果
    }
);
```

最后，我们把 GSAP 制作整个动画的效果再优化一下，将 `repeat`，`yoyo` 和 `ease` 提取到 `timelineConfig` 中，简化时间轴创建时的参数设置。以及合并相同的缓动配置和时间轴配置，减少代码重复。

  


```JavaScript
// 注册 GSAP 插件
gsap.registerPlugin(MorphSVGPlugin);

// 缓动
const animationEase = "none";

// 创建重复和缓动配置
const timelineConfig = {
    repeat: -1, 
    yoyo: true,
    ease: animationEase
};

// 创建水波时间轴和动画
const animationWater = gsap.timeline(timelineConfig);
animationWater.to("#water", {
    duration: 2,
    y: 12,
    morphSVG: "#water-2"
});

// 创建云时间轴和动画
const animationCloud = gsap.timeline(timelineConfig);
animationCloud.to("#cloud", {
    duration: 3,
    x: -2,
    y: 1,
    scale: 0.95,
    rotation: 1
});

// 总动画时间（秒）和动画停止时间，然后反向播放
const dayNightCycleTime = 15;
const animationDelay = dayNightCycleTime / 4;

// 创建昼夜转换时间轴和动画
const animationToNight = gsap.timeline({
    repeat: -1,
    yoyo: true,
    repeatDelay: animationDelay
});

animationToNight
    // 动画背景渐变
    .to('#daytime-gradient stop:nth-of-type(1)', {
        duration: dayNightCycleTime,
        stopColor: '#060414'
    }, 0)
    .to('#daytime-gradient stop:nth-of-type(2)', {
        duration: dayNightCycleTime,
        stopColor: '#001f3f' // 调整为更深的颜色以确保足够黑
    }, 0)
    // 动画夜间覆盖层
    .to('#nighttime-overlay', {
        duration: dayNightCycleTime,
        opacity: 1
    }, 0)
    // 动画太阳
    .to('#sun', {
        duration: dayNightCycleTime / 1.25,
        scale: 0.9,
        attr: { cx: "753", cy: "697" }
    }, 0)
    // 动画月亮
    .to('#moon', {
        duration: dayNightCycleTime / 2,
        scale: 0.9,
        attr: { cx: "174.5", cy: "202.5" }
    }, 0)
    .to('#moon', {
        duration: dayNightCycleTime / 2,
        scale: 0.9,
        attr: { cx: "410.5", cy: "114.5" }
    }, dayNightCycleTime / 2)
    // 动画星星
    .to('#stars', {
        duration: dayNightCycleTime / 2,
        opacity: 1
    }, dayNightCycleTime / 2)
    .from('#stars', {
        duration: dayNightCycleTime - (dayNightCycleTime / 4),
        y: 150,
        rotation: -15
    }, dayNightCycleTime / 4);

// 创建星星时间轴和动画
const animationStars = gsap.timeline(timelineConfig);
animationStars
    .to("#star-one", { duration: 0.5, opacity: 0.5 }, 0)
    .to("#star-two", { duration: 0.5, opacity: 0.5 }, 1)
    .to("#star-three", { duration: 0.5, opacity: 0.5 }, 0.5)
    .to("#star-four", { duration: 0.5, opacity: 0.5 }, 1.5);

// 创建雪花时间轴和动画
const animationSnowTop = gsap.timeline({ repeat: -1 });
const animationSnowBottom = gsap.timeline({ repeat: -1 });

animationSnowTop.to("#snow-top-layer", {
    duration: 7,
    attr: { transform: "translate(24 -108)" },
    ease: animationEase
});

animationSnowBottom.to("#snow-bottom-layer", {
    duration: 15,
    attr: { transform: "translate(13 140)" },
    ease: animationEase
});
```
