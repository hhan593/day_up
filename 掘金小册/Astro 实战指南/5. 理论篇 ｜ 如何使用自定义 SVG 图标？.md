## 1. 前言

如果我们要使用 SVG 图标，在 Astro 中该怎么引入使用呢？

简单来说，有三种方法，我们一一看下。

## 2. 第一种方法：img

Astro 无需配置即支持大多数静态资源，所以像图片比如 `.svg`、`.jpg`、`.png`是可以直接导入的，可是导入后该如何使用呢？让我们看个例子。

### 2.1. 设置样式

作为一个图标，稍微有些大，我们调整一下样式：

```jsx
---
import svgReference from '../icons/home.svg';
---

<img src={svgReference.src} alt="home page icon" />

<style>
  img {
    width: 40px;
    color: blue;
  }
</style>
```

![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/1be2b915dae64aeeac9a951f7e671ca3~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1014\&h=290\&s=20247\&e=png\&b=ffffff)

除了修改 SVG 本身的颜色代码，也可以通过 CSS3 的 `filter:drop-shadow` 滤镜实现：

```jsx

    width: 40px;

    transform: translate(-1000px);
  }
</style>
```

## 3. 第二种方法：Fragment

第二种方法是采用 `Fragment`：

```jsx
---
import svg from '../icons/home.svg?raw';
console.log(svg)
---

<Fragment set:html={svg} />
```

浏览器效果如下：

![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/38b015bd0992454085197a853eec9720~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2920\&h=1410\&s=652807\&e=png\&b=2b2b2b)

查看命令行的打印输出：

![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/d727b4b56a674a32936cffef93bd142b~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=3232\&h=738\&s=403112\&e=png\&b=1f1f1f)

导入的时候使用 `?raw`属于 Astro 的高级用例，此时会绕过 Astro 的自动 CSS 处理，直接从磁盘上读取，从命令行中可以看出，直接读取了 SVG 的代码。

然后通过 `<Fragment set:html={svg} />`将 SVG 元素直接写入了 HTML 中。

## 4. 第三种方法：使用 astro-icon

[astro-icon](https://www.astroicon.dev/getting-started/) 是一个社区处理图标的库。

首先安装 `astro-icon`：

```bash
npx astro add astro-icon
```

> 注：如果安装的时候遇到 `fetch failed`错误，可以尝试调整 VPN 的连接模式。如果总是失败，可以参考 astro-icon 的[《手动安装文档》](https://www.astroicon.dev/getting-started/#manual-install)。

然后添加图标，将 `.svg` 放到 `/src/icons/`目录，astro-icon 会自动读取该目录下的文件。

最后使用图标：

```jsx
---
import { Icon } from 'astro-icon/components'
---

<Icon name="home" />
```

此时会读取 `/src/icons/home.svg`文件，浏览器效果如下：

![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/4a3b671b5c3d4256b51fa901d66be853~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2276\&h=854\&s=274104\&e=png\&b=2b2b2b)

### 4.1. 设置样式

如果要设置样式，首先删除 SVG 图标里的 `fill="#231815"`代码，然后修改 `src/pages/index.astro`：

```jsx
---
import { Icon } from "astro-icon/components";
---

<Icon name="home" />

<style>
  [data-icon="home"] {
    font-size: 2rem;
    color: blue;
  }
</style>
```

浏览器效果如下：

![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/e049ac3d577d4022b442f3ee6c3d3f92~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2830\&h=990\&s=410333\&e=png\&b=2c2c2c)

如果你使用了 Tailwind CSS，也可以直接在 `<Icon>`上添加类：

```html
<Icon name="home" class="text-xl text-emerald-700" />
```
