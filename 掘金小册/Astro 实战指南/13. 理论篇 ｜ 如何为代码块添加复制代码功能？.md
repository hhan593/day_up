## 1. 前言

本篇我们讲解在 Astro 中，如何为代码块添加代码复制功能。

## 2. 第一种方法：直接写

为了方便演示，我们基于《理论篇 | 如何自定义 Markdown 样式？》这篇的代码进行修改。当然你也可以直接拷贝下面例子中的核心脚本代码用于自己的项目进行测试。

修改 `src/pages/blog/[...slug].astro`，代码如下：

```jsx
---
import { type CollectionEntry, getCollection } from "astro:content";
import Prose from "../../components/Prose.astro";

export async function getStaticPaths() {
  const posts = await getCollection("blog");
  return posts.map((post) => ({
    params: { slug: post.slug },
    props: { post },
  }));
}

interface Props {
  post: CollectionEntry<"blog">;
}

const { post } = Astro.props;
const { Content } = await post.render();
---

<html lang="zh-cn">
  <head>
    <meta charset="utf-8" />
    <link rel="icon" type="image/svg+xml" href="/favicon.svg" />
    <meta name="viewport" content="width=device-width" />
    <meta name="generator" content={Astro.generator} />
    <title>{post.data.title}</title>
  </head>
  <body>
    <Prose>
      <Content />
    </Prose>
  </body>
</html>

<style>
  @media (prefers-color-scheme: dark) {
    body {
      background-color: #0d1117;
    }
  }
</style>

<style is:global>
  .copy-code {
    position: absolute;
    top: 10px;
    right: 10px;
    color: #666;
  }
</style>

<script>
  let copyButtonLabel = "复制代码";
  let codeBlocks = Array.from(document.querySelectorAll("pre"));

  for (let codeBlock of codeBlocks) {
    let wrapper = document.createElement("div");
    wrapper.style.position = "relative";

    let copyButton = document.createElement("button");
    copyButton.className = "copy-code";
    copyButton.innerHTML = copyButtonLabel;

    codeBlock.setAttribute("tabindex", "0");
    codeBlock.appendChild(copyButton);
    codeBlock.parentNode!.insertBefore(wrapper, codeBlock);
    wrapper.appendChild(codeBlock);

    copyButton.addEventListener("click", async () => {
      await copyCode(codeBlock, copyButton);
    });
  }

  async function copyCode(block: HTMLPreElement, button: HTMLButtonElement) {
    let code = block.querySelector("code");
    let text = code!.innerText;

    await navigator.clipboard.writeText(text);

    button.innerText = "复制成功！";

    setTimeout(() => {
      button.innerText = copyButtonLabel;
    }, 700);
  }
</script>
```

实现的代码并不复杂，主要代码在脚本代码中，查询每一个 `pre`元素，然后创建复制按钮元素，并未复制按钮添加拷贝代码的事件。

## 3. 第二种方法：MDX 自定义组件

第二种方法其实跟第一种方法差不多，只不过利用的 MDX 的自定义组件功能。

先集成 `mdx`，运行：

```bash
npx astro add mdx
```

然后修改 `src/pages/blog/[...slug].astro`，代码为：

```diff
---
import { type CollectionEntry, getCollection } from "astro:content";
import Prose from "../../components/Prose.astro";
+import Pre from "../../components/Pre.astro";

export async function getStaticPaths() {
  const posts = await getCollection("blog");
  return posts.map((post) => ({
    params: { slug: post.slug },
    props: { post },
  }));
}

interface Props {
  post: CollectionEntry<"blog">;
}

const { post } = Astro.props;
const { Content } = await post.render();
---

<html lang="zh-cn">
  <head>
    <meta charset="utf-8" />
    <link rel="icon" type="image/svg+xml" href="/favicon.svg" />
    <meta name="viewport" content="width=device-width" />
    <meta name="generator" content={Astro.generator} />
    <title>{post.data.title}</title>
  </head>
  <body>
    <Prose>
      <Content
+        components={{
+          pre: Pre,
+        }}
      />
    </Prose>
  </body>
</html>

<style>
  @media (prefers-color-scheme: dark) {
    body {
      background-color: #0d1117;
    }
  }
</style>
```

接着将 `src/content/blog/markdown.md`重命名为 `src/content/blog/markdown.mdx`

最后新建 `src/components/Pre.astro`，代码为：

```jsx
---
const props = Astro.props;
---

<div class="relative group">
  <pre {...props}><slot /></pre>
  <button
    class="copy-code text-gray-600 absolute top-3 right-3 invisible group-hover:visible"
  >
    复制代码
  </button>
</div>

<script>
  const buttons = document.querySelectorAll("button.copy-code");

  buttons.forEach((button) => {
    const block = button.parentElement;
    button.addEventListener("click", () => {
      copyCode(block, button);
    });
  });

  async function copyCode(block: any, button: any) {
    let code = block.querySelector("code");
    let text = code!.innerText;

    await navigator.clipboard.writeText(text);

    button.innerText = "复制成功！";

    setTimeout(() => {
      button.innerText = "复制代码";
    }, 700);
  }
</script>
```

## 4. 第三种方法：借助 Expressive Code

第三种方法是直接使用现有的一些实现，Astro 中为代码添加样式和功能，最常见的应该就是使用 [Expressive Code](https://expressive-code.com/) 了：

```bash
npx astro add astro-expressive-code
```

安装完成后，可以看到立刻就有了改变：

![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/087f1a7d976d44eaa61809298f9e0a20~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1278\&h=287\&s=29954\&e=gif\&f=17\&b=fdfdfd)

但是样式我想自定义该怎么办？拷贝成功时提示的 "Copied" 文字该怎么修改？

> 这个时候就好想吐槽了……因为修改样式有点麻烦，而且文字修改起来的方法在文档里竟然还找不到……

如果你要修改样式，需要通过 `styleOverrides`[配置项](https://expressive-code.com/reference/configuration/#styleoverrides)，但不能直接修改，而是要使用 Expressive Code 提供的配置项，这个时候就需要查文档了……



而关于文字的修改，最终我是从 [GitHub Issues](How%20to%20modify%20<pre>%20tag%20·%20Issue%20#242%20·%20expressive-code/expressive-code) 里查到的……



修改 `astro.config.mjs`，代码如下：



```javascript
// @ts-check
import { defineConfig } from "astro/config";

import tailwind from "@astrojs/tailwind";

import expressiveCode, { pluginFramesTexts } from "astro-expressive-code";

pluginFramesTexts.overrideTexts(undefined, {
  copyButtonCopied: "复制成功",
});
// https://astro.build/config
export default defineConfig({
  integrations: [
    tailwind(),
    expressiveCode({
      styleOverrides: {
        codeBackground: ({ theme }) => {
          return theme.type == "dark" ? "#24292e" : "#e6eaef";
        },
        frames: {
          frameBoxShadowCssValue: "none",
          inlineButtonBorder: "none",
          inlineButtonBackgroundHoverOrFocusOpacity: "0",
        },
      },
    }),
  ],
  markdown: {
    shikiConfig: {
      themes: {
        light: "github-light",
        dark: "github-dark",
      },
    },
  },
});
```

此外，如果你想要 Hover 按钮一直显示，可以通过 plugin 插件显示，代码如下：


```javascript
// @ts-check
import { defineConfig } from "astro/config";

import tailwind from "@astrojs/tailwind";

import expressiveCode, { pluginFramesTexts } from "astro-expressive-code";

pluginFramesTexts.overrideTexts(undefined, {
  copyButtonCopied: "复制成功！",
});
// https://astro.build/config
export default defineConfig({
  integrations: [
    tailwind(),
    expressiveCode({
      styleOverrides: {
        codeBackground: ({ theme }) => {
          return theme.type == "dark" ? "#24292e" : "#e6eaef";
        },
        frames: {
          frameBoxShadowCssValue: "none",
          inlineButtonBorder: "none",
          inlineButtonBackgroundHoverOrFocusOpacity: "0",
        },
      },
      plugins: [
        {
          name: "Always Show Copy Button",
          baseStyles: `
            @media (hover: hover) {
              .copy button {
                opacity: 0.75;
              }
            }
          `,
        },
      ],
    }),
  ],
  markdown: {
    shikiConfig: {
      themes: {
        light: "github-light",
        dark: "github-dark",
      },
    },
  },
});
```



此时 Hover 按钮会一直显示：
