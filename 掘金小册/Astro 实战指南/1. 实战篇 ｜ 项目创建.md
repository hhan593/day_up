## 1. 前言

不同于我的前两本小册《Next.js 开发指南》和《Svelte 开发指南》，这两本小册分为基础篇、实战篇和原理篇，旨在系统地讲解一个框架的使用。

考虑到 Astro 的整体内容相对简单且中文文档齐全，我们直接从实战篇开始，做一个博客项目。

博客整体设计参考我的个人博客 [yayujs.com](https://yayujs.com)，支持响应式布局、暗黑模式，包含多种最佳实践和解决方案。网站看似简单，但细节很多。

每篇实战内容同时会配套多篇理论内容，详细讲解背后的原理与最佳实践，让你知其然知其所以然。

**简单来说，实战篇为主线内容，理论篇为拓展阅读。**

话不多说，那就让我们直接开始吧。

```bash

cd ./blog

# 添加 tailwind css
npx astro add tailwind
```

### 2.3. 编辑器配置

如果是 VSCode，Astro 有一个官方维护的[编辑器插件](https://marketplace.visualstudio.com/items?itemName=astro-build.astro-vscode)，可以：

1.  为 `.astro` 文件提供语法高亮
2.  为 `.astro` 文件提供 TypeScript 类型信息
3.  智能提示和代码补全

如果是其他编辑器，参考官方文档[《编辑器配置》](https://docs.astro.build/zh-cn/editor-setup/)。

### 2.4. 使用 Prettier

运行：

```bash
npm install -D prettier prettier-plugin-tailwindcss prettier-plugin-astro
```

项目根目录新建 `.prettierrc.json`，代码如下：

```json
{
  "plugins": ["prettier-plugin-astro", "prettier-plugin-tailwindcss"]
}
```

`Command + Shift + P` 打开编辑器命令行，选择 “Open User Settings”：

![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/a06873a0b2e545bb9c7c855d68e868b9~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2036\&h=286\&s=58352\&e=png\&b=1f3a58)

添加代码如下：

```json
{
  "editor.formatOnSave": true,
  "[astro]": {
    "editor.defaultFormatter": "astro-build.astro-vscode"
  }
}
```

一是开启保存时自动格式化，二是指定 astro 文件使用 astro 插件进行格式化。

## 5. 支持暗黑模式

接下来我们实现暗黑模式，关于实现暗黑模式的原理和实现，具体参考[《理论篇 | 如何支持深色模式？》](https://juejin.cn/book/7452253924608147467/section/7453021848851185683)。

```jsx
---

import ThemeIcon from "@/components/ThemeIcon.astro";

  title: string;
}

---

<!DOCTYPE html>
<html lang="zh-hans" class="bg-[var(--page-bg)]">
  <head>
    <meta charset="UTF-8" />
    <meta name="description" content="Astro description" />
    <meta name="viewport" content="width=device-width" />

    <link rel="manifest" href="/site.webmanifest" />
    <meta name="generator" content="{Astro.generator}" />
    <title>{title}</title>
    <script is:inline>
      const setTheme = () => {
        const theme = (() => {
          if (typeof localStorage !== "undefined" && localStorage.getItem("theme")) {
            return localStorage.getItem("theme");
          }
          if (window.matchMedia("(prefers-color-scheme: dark)").matches) {
            return "dark";

        } else {
          document.documentElement.setAttribute("data-theme", "dark");

      document.addEventListener("astro:after-swap", setTheme);

  <body>
    <div id="page" class="flex min-h-screen overflow-hidden rounded-md text-[var(--text)] md:mx-10 md:mt-14">
```
