## 使用 `clip-path` 创建圆角边缘

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/73ac1235cff04b2e93bc8a68d056087c~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2626&h=765&s=793795&e=jpg&b=fdfcfc)

  


我们先来看如何使用 CSS 的 `clip-path` 创建圆角边缘。你可能会问，`clip-path` 能裁剪出圆角边缘吗？是的，这是完全可以的。我们可以借助 `clip-path` 的 `ellipse()` 函数裁剪出圆角边缘形状，而且代码非常简单：

  


```CSS
.rounded-edge {
    clip-path: ellipse(85% 100% at top);
}
```

  


我用下图来帮助大家理解 `ellipse()` 如何裁剪出圆角边缘形状：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/6cc0e8d35dbc4897a39db17ad3f188e5~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1077&s=380880&e=jpg&b=ffffff)

  


如你所见，我们从未应用 `clip-path` 的矩形元素开始（上图最左侧）：

  


```CSS
.rounded-edge {
    height: 300px;
    aspect-ratio: 4 / 3;
    background: #376367;
}
```

  


接着，我们在这个元素上使用 `clip-path` 的 `ellipse()` 函数进行裁剪：

  


```CSS
.rounded-edge {
    height: 300px;
    aspect-ratio: 4 / 3;
    background: #376367;
    
    clip-path: ellipse(50% 100% at top);
}
```

  


裁剪路径是一个椭圆形状，其中心位于元素的顶部中心（`at top`），其水平半径是 `50%` ，垂直半径是 `100%` （上图中间所示）。这个椭圆超出了元素的边界，因此我们只能看到它的下半部分，而上半部分没有遮挡任何内容。

  


这里单独解释一下 `ellipse()` 函数中的 `at top` 。关键词 `at` 后面的值，比如 `top` 是用来椭圆圆心的位置。默认情况下，如果不显式设置值，椭圆圆心位于元素的正中央。它的使用有点类似于 `background-position` 。

  


我们再次回到示例中来。如果我们增加 `ellipse()` 的水平半径，比如 `85%` ，那么椭圆会变大，并且覆盖矩形元素更大的区域（上图右侧所示）。

  


```CSS
.rounded-edge {
    height: 300px;
    aspect-ratio: 4 / 3;
    background: #376367;
    
    clip-path: ellipse(85% 100% at top);
}
```

  


不难发现，我们只看到椭圆底部的一部分，这就是我们想要的圆角边缘！

  


需要知道的是，使用这种方法创建的圆角边缘形状是响应式的，因为我们使用了百分比值。其中 `ellipse()` 函数的水平半径是相对于元素的宽度计算，垂直半径则相对于元素的高度计算。当然，如果你想要创建一个固定尺寸的圆角边缘形状，你也可以使用固定单位值，例如像素值，创建一个固定尺寸的椭圆裁剪形状。而且，在某些特定场景中，使用固定值会得到更好的效果。

  


另外，除了可以调整 `ellipse()` 函数的半径值之外，还可以调整椭圆的圆心位置，例如 `at bottom` ，这样你就可以在矩形元素的顶部获得一个圆角边缘效果。这意味着，通过改变椭圆的圆心位置，你可以调整圆角边缘位置，轻松实现四个方向的变化：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/3d537679293d40c7b625b34dec69f0dd~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1807&s=691516&e=jpg&b=ffffff)

  


为了能更好的控制圆角边缘形状，我们可以使用 CSS 自定义属性来表示 `ellipse()` 函数的水平半径、垂直半径和圆心位置：

  


```CSS
.rounded-edge {
    --radius-x: 85%;  /* 水平半径 */
    --radius-y: 100%; /* 垂直半径 */
    --pos: top;       /* 圆心位置 */
    height: 300px;
    aspect-ratio: 4 / 3;
    background: #376367;
    
    clip-path: ellipse(var(--radius-x) var(--radius-y) at var(--pos));
}
```

  


现在，你只需要根据需要，调整这些变量的值就可以获得不同方向的圆角边缘形状：

  


```HTML
<div class="rounded-edge rounded-edge--top" style="--radius-x: 85%; --radius-y: 100%; --pos: bottom"></div>
<div class="rounded-edge rounded-edge--bottom" style="--radius-x: 85%; --radius-y: 100%; --pos: top"></div>
<div class="rounded-edge rounded-edge--left" style="--radius-x: 100%; --radius-y: 85%; --pos: right"></div>
<div class="rounded-edge rounded-edge--right" style="--radius-x: 100%; --radius-y: 85%; --pos: left"></div>
<div class="rounded-edge rounded-edge--center" style="--radius-x: 85%; --radius-y: 50%; --pos: center"></div>
<div class="rounded-edge rounded-edge--center" style="--radius-x: 50%; --radius-y: 85%; --pos: center"></div>
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/b8c57487d60d42d3be5a839d030506ba~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=2121&s=315419&e=jpg&b=ff5500)

  


> Demo 地址：https://codepen.io/airen/full/PoMEzKG

  


我们还可以使用百分比值来调整椭圆圆心的位置，这样就可以得到更具自定义的圆角边缘形状。例如：

  


```CSS
.rounded-edge {
    --radius-x: 85%;  /* 水平半径 */
    --radius-y: 100%; /* 垂直半径 */
    --pos-x: 30%;     /* 圆心水平位置 */
    --pos-y: 0%;      /* 圆心垂直位置 */
    --pos: var(--pos-x) var(--pos-y);    /* 圆心位置 */
    height: 300px;
    aspect-ratio: 4 / 3;
    background: #376367;
    
    clip-path: ellipse(var(--radius-x) var(--radius-y) at var(--pos));
}
```

  


除此之外，还可以通过 `top` 、`right` 、`bottom` 和 `left` 关键词与百分比值相互结合，明确告诉圆心距元素边缘的位置。例如：

  


```CSS
.rounded-edge {
    --pos-x: 30%;     /* 圆心水平位置 */
    --pos-y: 0%;      /* 圆心垂直位置 */
    --pos: left var(--pos-x) top var(--pos-y);  
}
```

  


上面代码中的圆心位置距顶部边缘 `--pos-y` ，距左侧边缘 `--pos-x` 。如果我们希望圆心距元素底部边缘 `--pos-y` ，则可以使用 `bottom var(--pos-y)` ；同时距元素右侧边缘 `--pos-x` ，则可以使用 `right var(--pos-x)` ：

  


```CSS
.rounded-edge {
    --pos-x: 30%;     /* 圆心水平位置 */
    --pos-y: 0%;      /* 圆心垂直位置 */
    --pos: right var(--pos-x) bottom var(--pos-y); 
}
```

  


现在，你可以调整相关参数的值，获得更多自定义的形状：

  


```CSS
.rounded-edge {
    --radius-x: 85%; /* 水平半径 */
    --radius-y: 100%; /* 垂直半径 */
    --pos-x: 50%;
    --pos-y: 0%;
    --d-x: left;
    --d-y: top;
    --pos: var(--d-x) var(--pos-x) var(--d-y) var(--pos-y); /* 圆心位置 */
    height: 50vmin;
    aspect-ratio: 3 / 2;
    background: #f50;

    clip-path: ellipse(var(--radius-x) var(--radius-y) at var(--pos));
}
```

> Demo 地址：https://codepen.io/airen/full/QWeaEOO

我们可以使用 [CSS Flexbox](https://juejin.cn/book/7161370789680250917/section/7161621092560273439) 或 [CSS Grid](https://juejin.cn/book/7161370789680250917/section/7161372229123440648) 对其进行布局，以及样式化整个效果。正如之前所述，可以使用 `clip-path` 的 `ellipse()` 函数给元素 `header` 的底部添加一个圆形边缘形状：

  


```CSS
header {
    width: clamp(300px, 80vw, 1140px);
    padding: 1.5rem 2rem 4.375rem;
    clip-path: ellipse(75% 100% at top);
}
```

## 使用 `mask` 创建弧形边缘

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/7c42ca4cdc5e43c6914d61deb40f819e~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=800&h=600&s=9147334&e=gif&f=218&b=f8f5f5)

  


通过之前的学习，我们可以快速使用 `ellipse()` 裁剪出带有圆角边缘的形状。如果你足够仔细的话，不难发现，`ellipse()` 裁剪出来的圆角边缘形状是向外凸起的。但我们很多时候需要一种与其相反的形状，即圆形边缘是向内凹进的，如上图中产品展示区所示的效果。我们常把这种效果称为“弧形边缘形状”。

  


不过，我们无法直接使用 `clip-path` 裁剪出这个形状。因为，到目前为止，CSS 并没有为 `clip-path` 属性提供一个类似的函数，允许我们直接裁剪出内凹的弧形形状（除 SVG 路径之外）。庆幸的是，CSS 提供了另一种方案可以让我们很容易创建弧形边缘形状，这个方案就是 [CSS 遮罩](https://juejin.cn/book/7223230325122400288/section/7259668885224456252#heading-9)（`mask`）。

  


使用 `mask` ，我们只需要一行代码就可以给元素添加一个弧形边缘形状：

  


```CSS
.curved-edge {
    mask: radial-gradient(60% 70px at bottom, #0000 100%, #000);
}
```

  


它的工作原理与 `clip-path` 是相似的，我们通过径向渐变创建的遮罩图来隐藏元素的部分区域。

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/44b2c3608b714df7a290d885c49431f6~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1280&h=696&s=75539&e=jpg&b=f6f5f5)

  


需要知道的是，使用 `mask` 有两种方式（带 Alpha 通道遮罩模式和高亮遮罩模式）可以对元素进行裁剪：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/ee4c850cf95146bdbeebabfdec4d1718~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1280&h=796&s=102778&e=jpg&b=565874)

  


如果你不想花时间了解它们之间的差异，只是想知道如何使用 `mask` 来隐藏元素的内容。只需要记得，使用 `mask` 时，渐变中的透明颜色（`#0000`）会隐藏元素，而不透明颜色（`#000`）则会显示元素。这采用的就是带有 Alpha 通道遮罩模式！

  


颜色的具体值并不重要，关键是颜色的透明度。所以你可以自由选择任何颜色语法。例如，使用 `rgb()` ：

  


```CSS
.curved-edge {
    mask: radial-gradient(60% 70px at bottom, rgb(0 0 0 / 0) 100%, rgb(0 0 0));
}
```

  


你也可以使用其他的颜色，不局限于黑色 `#000` ，例如 `#09aefe`:

  


```CSS
.curved-edge {
    mask: radial-gradient(60% 70px at bottom, #0000 100%, #09aefe);
}
```

  


再次强调一下，透明度很关键，它直接会影响遮罩的效果。[有关于这方面更详细的介绍，请点击这里获取](https://juejin.cn/book/7223230325122400288/section/7259668885224456252#heading-10)！

  


了解完 `mask` 的工作原理之后，再来解释一下径向渐变 `radial-gradient()` 。它与 `clip-path` 的 `ellipse()` 函数类似，`radial-gradient()` 也会创建一个椭圆形。例如：

  


```CSS
.box {
    width: 400px;
    aspect-ratio: 2 / 1;
    background: radial-gradient(50% 50% at center, #0000 100%, #09aefe) no-repeat;
}
```

  


上面代码中的 `radial-gradient()` 将会创建一个透明的椭圆，椭圆区域会露出其父元素的底色：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/a99d2ef2695247379151342298944654~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=770&s=137307&e=jpg&b=ffc0ca)

  


同样的，我们可以调整 `radial-gradient()` 的半径和圆心位置，获得不同的遮罩图：

  


```CSS
.box {
    width: 400px;
    aspect-ratio: 2 / 1;
    background: radial-gradient(60% 70px at bottom, #0000 100%, #000) no-repeat;
    outline: 1px dashed;
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/80a9014461e94adf80178b38edf60c20~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=770&s=121826&e=jpg&b=ffc0ca)

  


现在，我们只需要将 `radial-gradient()` 创建的渐变作为遮罩图，CSS 的 `mask` 就会隐藏椭圆内的部分，而保留个部部分。由于 `radial-gradient()` 创建的椭圆有部分超出了元素边界，从而获得了可见的曲线。如此一来，最终使矩形元素获得了所需要的弧形边缘形状：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/70723f58e0044cd49ca715f9376d9ab2~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=770&s=339814&e=jpg&b=ffffff)

  


```CSS
.curved-edge {
    width: 400px;
    aspect-ratio: 4 / 3;
    background: #376367;
    
    mask: radial-gradient(60% 70px at bottom, #0000 100%, #000);
}
```

  


你现在应该明白它背后的工作原理了吧！

  


与 `clip-path` 的 `ellipse()` 创建圆形边缘形状类似，我们也可以通过调整 `radial-gradient()` 的椭圆半径和圆心位置，创建不同方向的弧形边缘形状：

  


```CSS
.curved-edge {
    --c: 70px; /* 控制弧形曲率，控制椭圆垂直半径 */
    --radius: 60%; /* 控制椭圆水平半径*/
    --pos: bottom; /* 控制圆心位置 */
    height: 300px;
    aspect-ratio: 4 / 3;

    mask: radial-gradient(var(--radius) var(--c) at var(--pos),#0000 calc(100% - 2px),#000);
    background: #f50;
}
```

  


根据需要调整 `--c` 、`--radius` 和 `--pos` 变量的值，就可以获得不得方向的弧形边缘形状：

  


```CSS
.curved-edge {
    --c: 70px;
    --radius: 60%; 
    --pos: top; 
    aspect-ratio: 4 / 3;
    background: #f50;

    mask: radial-gradient(
        var(--radius) var(--c) at var(--pos),
        #0000 calc(100% - 2px),
        #000
    );

    &:nth-child(2) {
        --pos: bottom;
    }

    &:nth-child(3) {
        --pos: center;
        mask: radial-gradient(
            var(--radius) var(--c) at var(--pos) var(--c),
            #0000 calc(100% - 2px),
            #000
        ) 0 calc(-1 * var(--c));
    }

    &:nth-child(4) {
        --pos: right;
        mask: radial-gradient(
            var(--c) var(--radius) at var(--pos),
            #0000 calc(100% - 1px),
            #000
        );
    }

    &:nth-child(5) {
        --pos: left;
        mask: radial-gradient(
            var(--c) var(--radius) at var(--pos),
            #0000 calc(100% - 1px),
            #000
        );
    }

    &:nth-child(6) {
        --pos: center;
        mask: radial-gradient(
            var(--c) var(--radius) at var(--c) var(--pos),
            #0000 calc(100% - 2px),
            #000
        ) calc(-1 * var(--c)) 0;
    }
}
```

## 如何结合这两种 CSS 形状

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/503f69ab26244cf585258dbe955939ff~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=800&h=600&s=5434380&e=gif&f=182&b=e7eaf7)

  


请仔细观察上图的效果，页面的第二个区域同时具有之前所介绍的两种形状：元素的顶部具有圆角边缘形状，同时元素的底部具有弧形边缘形状。

  


你可能已经猜到了，结合之前创建圆角边缘和弧形边缘的代码，我们就可以创建所要的效果：

  


```CSS
.curved-rectangle {
    /* 底部的弧形边缘 */
    mask: radial-gradient(60% 70px at bottom, #0000 100%, #000);
    /* 顶部的圆角边缘 */
    clip-path: ellipse(80% 100% at bottom);
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/929aafce415e42d6bf746899530b31cc~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1026&s=510718&e=jpg&b=ffffff)

  


你可能已经发现了，这两条弧线并不匹配，这使得整个形状看起来有些不协调。但我们可以通过调整来修复这个问题。我们需要确保两部分代码创建的椭圆形状相同，即设置相同的半径值：

  


```CSS
.curved-header {
    /* 顶部的弧形边缘 */
    mask: radial-gradient(80% 100% at bottom, #0000 100%, #000);
    /* 底部的圆角边缘 */
    clip-path: ellipse(80% 100% at bottom);
}
```

  


注意，`radial-gradient()` 和 `ellipse()` 两个函数都声明了相同的值 `80% 100% at bottom` 。但如果直接使用这段代码，什么都不会显示出来。别忘了，`clip-path` 会隐藏椭圆的外部区域，而 `mask` 则会隐藏椭圆内部区域。因此，如果两个椭圆形状完全相同（大小和位置），结果就是元素的所有内容都会被隐藏。

  


为了解决这个问题，我们需要对 `mask` 的 `radial-gradient()` 绘制的椭圆形位置进行调整，将其往下移：

  


```CSS
.curved-rectangle {
    /* 顶部的弧形边缘 */
    mask: radial-gradient(80% 100% at 50% 178%, #0000 100%, #000);
    /* 底部的圆角边缘 */
    clip-path: ellipse(80% 100% at bottom);
}
```

  


现在，两条弧线也对齐了，整个形状也更完美了。

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/970e69874d9f45de991530fe8da76147~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1108&s=113445&e=jpg&b=050505)

  


> Demo 地址：https://codepen.io/airen/full/YzmYryL

  


注意，你可能会问，`radial-gradient()` 中的 `178%` 是如何得来的。这需要一些数学知识，我们可以借助[三角函数](https://juejin.cn/book/7223230325122400288/section/7242216512176521277)来计算出这个值：

  


```CSS
.curved-rectangle {
    --c: 80; /* 控制弧度 */
    mask: radial-gradient(
        calc(var(--c) * 1%) 100% at 50% calc(100% + 100% * cos(asin(50 / var(--c)))),
        #0000 calc(100% - 1px),#000);
    clip-path: ellipse(calc(var(--c) * 1%) 100% at bottom);
}
```

  


它将获得相同与上一个示例相同的效果。相比之下，这段 CSS 变得更为复杂了。你需要对相关的几何知识有一定的了解。当然，如果你不想花太多心思重新学习相关的几何知识，你只需要通过调整 `--c` 变量的值即可，因为它可以控制弧形的曲率。

  


同样的，我们调整椭圆位置，就可以获得其他的版本：

  


```CSS
.curved-rectangle {
    width: 400px;
    aspect-ratio: 4 / 3;
    background: #f50;
    
    --c: 80; /* 控制弧度 */

    &:nth-child(1) {
        mask: radial-gradient(
            calc(var(--c) * 1%) 100% at 50% calc(-100% * cos(asin(50 / var(--c)))),
            #0000 calc(100% - 1px),
            #000
        );
        clip-path: ellipse(calc(var(--c) * 1%) 100% at top);
    }

    &:nth-child(2) {
        mask: radial-gradient(
            calc(var(--c) * 1%) 100% at 50% calc(100% + 100% * cos(asin(50 / var(--c)))),
            #0000 calc(100% - 1px),
            #000
        );
        clip-path: ellipse(calc(var(--c) * 1%) 100% at bottom);
    }

    &:nth-child(3) {
        mask: radial-gradient(
            100% calc(var(--c) * 1%) at calc(-100% * cos(asin(50 / var(--c)))),
            #0000 calc(100% - 1px),
            #000
        );
        clip-path: ellipse(100% calc(var(--c) * 1%) at left);
    }

    &:nth-child(4) {
        mask: radial-gradient(
            100% calc(var(--c) * 1%) at calc(100% + 100% * cos(asin(50 / var(--c)))),
            #0000 calc(100% - 1px),
            #000
        );
        clip-path: ellipse(100% calc(var(--c) * 1%) at right);
    }
  }
```

### 总结

  


你掌握了吗？

  


简单地说，使用现代 CSS 创建这两种形状不再是难事。简简单单的两行代码就可以搞定。你可以使用 `clip-path` 的 `ellipse()` 函数创建一个圆角边缘（弧形向外凸）：

  


```CSS
.rounded-edge {
    clip-path: ellipse(85% 100% at top);
}
```

  


你可以使用 `mask` 创建一个弧形边缘（弧度向内凹）：

  


```CSS
.curved-edge {
    mask: radial-gradient(60% 70px at top, #0000 100%, #000);
}
```

  


通过将两者结合，你可以得到一个弧形矩形：

  


```CSS
.curved-rectangle {
    mask: radial-gradient(80% 100% at 50% -78%, #0000 100%, #000);
    clip-path: ellipse(80% 100% at top);
}

/* 或者 */
.curved-rectangle {
    --c: 80;
    mask: radial-gradient(calc(var(--c) * 1%) 100% at 50% calc(-100% * cos(asin(50 / var(--c)))),#0000 calc(100% - 1px),#000);
    clip-path: ellipse(calc(var(--c) * 1%) 100% at top);
}
```

  


你也可以像课程中所展示的示例那样，通过 CSS 自定义属性来控制相关的参数，如此一来，你只需要调整 CSS 自定义属性的值，就可以获得不同形状的变体。
