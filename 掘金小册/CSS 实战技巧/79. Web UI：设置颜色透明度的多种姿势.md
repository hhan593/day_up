## 设计中的透明度

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/994c26a631464329bcea4db60ebff6a6~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1792&h=1024&s=340208&e=webp&b=7b8d97)

  


在开始进入 CSS 之前，我们先来看看 Web 设计师通常是如何给一个 UI 对象设置透明度。我们以 Figma 设计软件为例。

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/f93e77357b8f4c0c98dbad64bb1f1607~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1363&s=1254760&e=jpg&b=8557a3)

  


第一种方式是调整 UI 元素对象的 `opacity` 参数。从 Figma 中直接复制该对象的 CSS：

  


```CSS
.element {
    background: #FEBA0B;
    opacity: 0.5;
}
```

  


不难发现，它是通过 CSS 的 `opacity` 属性来调整元素的透明度。稍微熟悉 CSS 的 `opacity` 属性的开发者都应该知道，`opacity` 属性将会影响元素的以及它的后代元素的所有 UI 的透明度。

  


第二种方式是调整 UI 对象填充颜色的透明度。复制出来的 CSS 如下：

  


```CSS
.element {
    background: rgba(254, 186, 11, 0.5);
}
```

  


它仅是调整了 `rgb(254 186 11)` 颜色的透明通道（即 `0.5`），从而获得一个带有透明度的颜色变体。

  


从这一点而言，如果我们只需要调整元素的 UI 颜色透明度，就不应该使用 `opacity` 属性，更应该使用带有透明度的颜色值。

  


然而，很多时候我们获取的原始颜色的值可能是一个描述颜色的关键词，比如 `red` ，也可能是一个十六进制的颜色值，比如 `#09acef` 。那么，要调整这些颜色的透明度，还是比较麻烦的。庆幸的是，CSS 的一些新特性可以改变这一现象，比如 CSS 的相对颜色语法和 `color-mix()` 函数。接下来，我们一起来看看，在 CSS 中有哪些方式可以帮助我们快速有效的创建带有透明度的颜色变体！

  


## CSS 如何为现有颜色添加透明度

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/94dd4b65636343cbae60ba6c4bf15bf7~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1792&h=1024&s=294842&e=webp&b=7e8386)

  


假设你的品牌颜色是 `dodgerblue` （道奇蓝），即 `#1e90ff` ：

  


```CSS
:root {
    --brand-color: dodgerblue; /* 👉 #1e90ff */
}
```

  


现在，你需要一个带有透明度的版本。

  


### 方法一：使用八位十六进制（Hex）代码

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/a7ca8db757824cfeaf0c2f7ce5acf70b~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1792&h=1024&s=438124&e=webp&b=bdd0d2)

  


通常情况下，我们会使用六位十六进制（Hex）代码设置颜色，这是不带透明度的。现在，你可以使用八位十六进制（Hex）代码来添加透明度，范围从 `00`（完全透明）到 `FF`（完全不透明）。比如：

  


```CSS
:root {
    --brand-color: dodgerblue; /* 👉 #1e90ff */
    --brand-color-5: #1e90ff0d; /* 👉 Alpha = 5% */
    --brand-color-10: #1e90ff1a; /* 👉 Alpha = 10% */
    --brand-color-50: #1e90ff7f; /* 👉 Alpha = 50% */
}
```

  


正如你所看到的，要记住透明度对应的两个额外字符可能有些麻烦。当然，你可以通过 JavaScript 编写一个转换函数或者借助其他可视化工具来转换：

  


```JavaScript
const root = document.documentElement;
const range = document.querySelector('input[type="range"]');
const output = document.querySelector("output");
const box = document.querySelector(".box--variant");

const updateColor = () => {
    const { value } = range;
    const alpha255 = Math.floor(value * 255 / 100);
    const alphaHex = alpha255.toString(16).padStart(2, "0").toUpperCase();
    const color = `#1E90FF${alphaHex}`;

    root.style.setProperty("--opacity-variant", color);
    box.textContent = color;
    output.textContent = `${value}%`;
};

range.addEventListener("input", updateColor);
updateColor();
```

虽然这种方法可行，但不便于 Web 开发者使用与维护。因此，在实际使用过程中，不建议采用这种方式。

  


### 方法二：使用颜色函数

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/a0e6aed36f56423781db27c2dc0d12f3~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1280&h=720&s=88547&e=png&b=fffdfd)

  


在 CSS 中，除了使用关键词和十六进制（Hex）描述颜色之外，还可以使用颜色函数来描述，比如 `rgb()` 、`lab()` 、`hwb()` 、`hsl()` 、`lch()` 、`oklch()` 、`oklab()` 和 `color()` 等：

  


```CSS
:root {
    --c-keyword: dodgerblue;
    --c-hex: #1e90ff;
    --c-rgb: rgb(30 144 255);
    --c-hsl: hsl(209.6deg 100% 55.88%);
    --c-hwb: hwb(209.6deg 11.76% 0%);
    --c-lch: lch(58 64.79 270.8);
    --c-oklch: oklch(0.65 0.19 253.11);
    --c-lab: lab(58 0.84 -64.71);
    --c-oklab: oklab(0.65 -0.06 -0.18);
    --c-color: color(srgb 0.03 0.57 0.99);
}
```

  


通过这些颜色函数定义的颜色，我们更容易调整它的透明度：

  


```CSS
:root {
    --c-rgb: rgb(30 144 255 / .5);
    --c-hsl: hsl(209.6deg 100% 55.88% / .5);
    --c-hwb: hwb(209.6deg 11.76% 0% / .5);
    --c-lch: lch(58 64.79 270.8 / .5);
    --c-oklch: oklch(0.65 0.19 253.11 / .5);
    --c-lab: lab(58 0.84 -64.71 / .5);
    --c-oklab: oklab(0.65 -0.06 -0.18 / .5);
    --c-color: color(srgb 0.03 0.57 0.99 / .5);
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/0eac29bdf3a2471e95633b088c3c91dc~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1346&h=896&s=17655835&e=gif&f=210&b=316ca6)

  


> Demo 地址：https://codepen.io/airen/full/xbxZrob

  


这种方式虽然比八位十六进制（Hex8）设置颜色透明度要方便得多，但它也有一个缺陷，需要知道颜色在所在颜色空间的通道值。以 `rgb()` 函数为例，你需要知道 `dodgerblue` 颜色在 RGB 颜色空间中 `r` （红色）、`g` （绿色）和 `b` （蓝色）通道的值。如果你使用的拾色器不能自动转换的话，同样需要借助其他的工具来转换。就以我个人习惯来说，我通常会借助 Chrome 浏览器开发者的调试工具来获得相应的颜色通道值：

### 方法三：CSS 自定义属性

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/64da26927ee645ecbfcf2f2351055efb~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2400&h=1350&s=595801&e=png&b=9070e7)

  


我们还可以利用 CSS 自定义属性独有的特性为现有颜色添加透明度，但前提是与使用 CSS 函数为颜色添加透明度相似，我们需要知道颜色所在颜色空间的通道值。假设我们知道 `dodgerblue` 颜色在 `hsl` 颜色空间中的每个通道的值：

  


-   `h = 209.6deg`
-   `s = 100%`
-   `l = 55.88%`

  


那么我们可以通过四个自定义属性来定义 `dodgerblue` 颜色：

  


```CSS
:root {
    --h: 209.6deg;
    --s: 100%;
    --l: 55.88%;
    --a: 1;
    
    --brand-color: hsl(var(--h) var(--s) var(--l) / var(--a)); /* 等同于 hsl(209.6deg 100% 55.88% / 1)*/
}
```

  


这个时候，如果我们要调整颜色的透明度，只需要调整自定义属性 `--a` 的值。例如：

  


```CSS
:root {
    --h: 209.6deg;
    --s: 100%;
    --l: 55.88%;
    --a: 1;
    
    --brand-color: hsl(var(--h) var(--s) var(--l) / var(--a)); /* 等同于 hsl(209.6deg 100% 55.88% / 1)*/
}

.element {
    background-color: var(--brand-color);
    
    &:hover {
        --a: .5;
    }
}
```

  


上面代码中，当用户鼠标悬浮在 `.element` 元素上时，它的背景颜色是 `hsl(209.6deg 100% 55.88% / .5)` 。

  


我们再来看一个简单的示例，根据元素在 DOM 中的索引值（内联定义了 `--i`）来调整背景颜色的透明：

  


```CSS
:root {
    --h: 209.6deg;
    --s: 100%;
    --l: 55.88%;
    --a-ratio: 10%;
}
  
.box {
    background-color: hsl(var(--h) var(--s) var(--l) / calc(var(--i) * var(--a-ratio)));
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/971e8b13cbd442999c98b444b27abf47~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2756&h=1790&s=413564&e=jpg&b=468bda)

  


> Demo 地址：https://codepen.io/airen/full/vEYLWWL

  


这种方式在某些场景下是非常的便利，但需要定义多个 CSS 自定义属性。这意味着，如果颜色数量多的话，自定义属性的数量也会成指数级别增加，不易于代码的管理和维护！

  


### 方法四：使用 CSS 相对颜色语法

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/51feee54024549318b4db9885f66a61e~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1024&h=1024&s=186596&e=webp&b=426c81)

  


虽然 CSS 颜色函数和 CSS 自定义属性为现有颜色添加透明度要比八位十六进制更灵活和便利一些，但大家还是更希望有一些更灵活的方法。比如说，不需要通过工具或代码，直接使用 CSS 就可以基于关键词颜色或十六进制颜色添加透明度。有点类似于 CSS 处理器（比如 Sass）中的 `rgba()` 函数：

  


```SCSS
$brand-color: dodgerblue;

.element {
    color: rgba($your_color, .5); /* 等同于 rgb(30 144 255 / .5)*/
}
```

  


庆幸的是，现代 CSS 中的 CSS 相对颜色语法和 `color-mix()` 函数具有类似的功能。我们先来看 CSS 的相对颜色语法。我们可以像下面这样为现有颜色添加透明度：

  


```CSS
:root {
    --brand-color: dodgerblue;
    
    --o-rgb: rgb(from var(--brand-color) r g b / .5);
    --o-hsl: hsl(from var(--brand-color) h s l / .5);
    --o-hwb: hwb(from var(--brand-color) h w b / .5);
    --o-lch: lch(from var(--brand-color) l c h / .5);
    --c-oklch: oklch(from var(--brand-color) l c h / .5);
    --c-lab: lab(from var(--brand-color) l a b / .5);
    --c-oklab: oklab(from var(--brand-color) l a b / .5);
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/23684d56b17e43a780f93acd44071631~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2780&h=1454&s=382356&e=jpg&b=4b79ad)

  


> Demo 地址：https://codepen.io/airen/full/ogNbpXz

  


上面这个种方式是在现有颜色的基础上指定具体的透明度。其实，使用 CSS 相对颜色语法，我们还可以通过 `alpha` 参数给现有颜色指定相对的透明度。例如：

  


```CSS
:root {
    --brand-color: dodgerblue;
}

.element {
    background-color: oklch(from var(--brand-color) l c h / calc(alpha / var(--i)));
}
```

  


其中 `--i` 是元素在 DOM 中的索引值，假设它从 `1` 开始。

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/33a2baa0332c41e882593f0f30f91c6d~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2756&h=1374&s=372171&e=jpg&b=526d7f)

  


> Demo 地址：https://codepen.io/airen/full/xbxZpqV

  


使用相对颜色语法，即便初始颜色带有一定的透明度，我们也可以通过 `alpha` 参数在原透明度的基础上指定一个相对透明度。例如：

  


```CSS
:root {
    --brand-color: oklch(0.65 0.19 253.11 / .025);
}

.element {
    background-color: oklch(from var(--brand-color) l c h / calc(alpha * var(--i)));
}
```

  


新颜色的透明度将是 `--i` 与 `0.5` 的乘积：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/4e882252662640fbb97b53a9ca137308~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2774&h=1002&s=196508&e=jpg&b=ffffff)

  


> Demo 地址：https://codepen.io/airen/full/dPyGJZR

  


正如你所看到的，使用 CSS 相对颜色语法，我们除了不要知道颜色所在空间的通道值之外，还可以使用 `calc()` 动态调整颜色的透明度。

  


### 方法五：使用 `color-mix()` 混合透明度

```CSS
}
```

  


上面的代码相当于 `75%` 的 `dodgerblue` 颜色与 `25%` 的 `transparent` 混合，等同于给 `dodgerblue` 颜色添加了 `25%` 的透明度：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/95e8556c967246b99f73b571ddfb9b95~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=3260&h=2864&s=721347&e=png&b=3185d8)

  


> Demo 地址：https://codepen.io/airen/full/gbOPvME

  


需要知道的是，`transparent` 的计算值是 `rgb(0 0 0 0)` (即完全透明的黑色)，这意味着，原颜色 `dodgerblue` 与完全透明的黑色进行混合，这可能在某些设备上会造成颜色偏差。为了避免这个现象，可以使用完全透明的原颜色来替代 `transparent` 。比如：

  


```CSS
:root {
    --brand-color: dodgerblue; /* #1e90ff */
}

.element {
    background: color-mix(in oklch, var(--brand-color), #1e90ff00 50%);
}
```

  


`#1e90ff00` 等同于完全透明的 `dodgerblue` 颜色（`#1e90ff`）。这样，混合的是透明的道奇蓝，而不是透明的黑色，避免颜色漂移问题。

  


另外需要注意的是，使用 `color-mix()` 函数时，颜色空间会决定最终颜色的效果。比如，如果你在 `color-mix()` 使用的是长路径（`longer hue`），它可能会与原颜色有很大的差异。比如：

  


```CSS
:root {
    --brand-color: dodgerblue;
    /* 短路径*/
    --c1: color-mix(in hsl, var(--brand-color), #1e90ff00 50%);
    --c2: color-mix(in hwb, var(--brand-color), #1e90ff00 50%);
    --c3: color-mix(in lch, var(--brand-color), #1e90ff00 50%);
    --c4: color-mix(in oklch, var(--brand-color), #1e90ff00 50%);
    
    /* 长路径 */
    --c1_l: color-mix(in hsl longer hue, var(--brand-color), #1e90ff00 50%);
    --c2_l: color-mix(in hwb longer hue, var(--brand-color), #1e90ff00 50%);
    --c3_l: color-mix(in lch longer hue, var(--brand-color), #1e90ff00 50%);
    --c4_l: color-mix(in oklch longer hue, var(--brand-color), #1e90ff00 50%);
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/4fb4ed485c31480aa37caab17aa46b50~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2774&h=1406&s=380527&e=jpg&b=4b79ad)

  


> Demo 地址：https://codepen.io/airen/full/zxYrRPP

  


这是因为，颜色空间决定了两种颜色混合的插值算法。

  


## 小结

  


通过这节课的学习，我想你已经掌握了为现有颜色添加透明度的多种方法，其中有传统的八位十六进制编码（Hex8）和各种 CSS 颜色函数，以及 CSS 自定义属性。这几种方法相比于 CSS 相对颜色语法和 `color-mix()` 要更鸡肋一些。使用 CSS 相对颜色语法和 `color-mix()` 给现有颜色添加透明度带来更多的灵活性和便利性。利用这些新特性，可以帮助我们更好的给颜色设置透明度。
