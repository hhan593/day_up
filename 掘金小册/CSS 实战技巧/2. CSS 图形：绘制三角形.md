## 创建三角形形状

  


使用 `clip-path` 属性的 `polygon()` 函数创建三角形非常简单，你只需要知道三个点的坐标即可：

  


```CSS
.triangle {
    clip-path: polygon(x1 y1, x2 y2, x3 y3);
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/7923adcdb865499cb3fde0a3760fbdf2~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1262&h=772&s=4397939&e=gif&f=328&b=f9fcf5)

  


实际上，我们不会使用随机点，而是考虑像角点（元素框的角顶点）或边的中间点这样简单的点。例如，下面的代码将创建一个等腰直角三角形：

  


```CSS
.triangle {
    width: 50vmin;
    aspect-ratio: 1;
    
    clip-path: polygon(0 0, 100% 100%, 0 100%);
    
    &:nth-child(2) {
        clip-path: polygon(0 0, 100% 0, 100% 100%);
    }
    
    &:nth-child(3) {
        clip-path: polygon(0 0, 100% 0, 0 100%);
    }
    
    &:nth-child(4) {
        clip-path: polygon(100% 0, 100% 100%, 0 100%);
    }
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/98518c90b7244b818229d175dcd316cf~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=922&s=204909&e=jpg&b=7b669e)

这里有两个三角形的例子：

## 创建等腰三角形

  


在几何学中，等腰三角形是指至少有两边长度相等的三角形，这两条相等的边称为腰：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/fbf2b53568d84a9284289dd2b4c110b0~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1824&s=201898&e=jpg&b=fcfcfc)

  


使用 `clip-path` 的 `polygon()` 函数可以轻易创建出等腰三角形：

  


```CSS
/* 等腰直角三角形 */
.triangle--isosceles {
    clip-path: polygon(0 0, 100% 100%, 0 100%);
}

/* 等要三角形 */
.triangle--isosceles {
    clip-path: polygon(50% 0, 100% 100%, 0 100%);
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/bc640b80046b4bc18eb21e9a86a50e91~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1360&s=971700&e=jpg&b=fafbf6)

  


我们只需要知道元素框的四个角的坐标和四条边中间点坐标，就可以快速创建等腰三角形。当然，这并不是千篇一律的，例如，等于直角三角形，只有元素是正方形的时候，可以按上图的方式快速获取三角形的三个点的坐标。如果元素是非正方形（比如，是一个矩形），则需要应用三角函数，才能计算出正确的坐标点。

  


简单的分析一下。使用下图来描述等腰三角形三个顶点的坐标：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/970e4aa2411543ad8f10252b828ae935~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=3276&h=2314&s=1053670&e=jpg&b=fbfcf7)

  


等腰三角形通过顶角的中线具有对称性。底边等于元素的宽度 `width` （使用自定义属性 `--w` 来表示），等腰三角形中线等于元素的高度 `height` （使用自定义属性 `--h` 来表示）。中线会将一个等腰三角形分成两个对称的直角三角形。

  


刚才提到过，如果基于一个正方形，我们绘制一个等腰直接三角形，只需要知道元素的顶点和边的中间点即可。如果是非正方形，则需要做一些简单的计算：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/fadb0c3b6ef4409da1002b5eb48fe384~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=3276&h=2314&s=754489&e=jpg&b=fbfcf7)

  


使用 CSS 来描述，大致如下：

  


```CSS
.triangle {
    --w: 50;
    --h: 25;
    width: calc(var(--w) * 1vmin);
    height: calc(var(--h) * 1vmin);
    clip-path: polygon(0 0, calc(var(--h) / var(--w) * 100%) 100%, 0 100%);
    
    &:nth-child(2) {
        --h: 50;
    }
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/f8ff19065b034ecaa6e8c4146700c272~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1056&s=225063&e=jpg&b=7b669e)

  


> Demo 地址：https://codepen.io/airen/full/MWMbqYb

  


注意 `calc()` 函数的计算，相同单位值的乘法（`calc(10vmin * 2vmin)`）或除法（` calc(10vmin  ``/ 2vmin``)`）都是无效计算。因此，在示例中，我们的 `--w` 和 `--h` 自定义属性的值都没有带单位，然后通过 `calc()` 计算时，换算成所需要的单位值。

  


另外，有的时候，我们所需要的等腰三角形的顶点可能不会在元素的边上，它有可能距离元素边有一定的距离，如下图所示：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/d2035ccbf2cd46ce84fbd0c6a6bbb262~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=3276&h=2314&s=1194456&e=jpg&b=fbfcf7)

  


```CSS
.triangle {
    --w: 300;
    --h: 300;
    --s: 30;
    
    width: calc(var(--w) * 1px);
    height: calc(var(--h) * 1px);
    clip-path: polygon(50% calc(var(--s) / var(--h) * 100%), 100% 100%, 0 100%);
    
    &:nth-child(2) {
        clip-path: polygon(0 0, calc(100% - var(--s) / var(--w) * 100%) 50%, 0 100%)
    }
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/d3921fa92279432999041274bedb5c96~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1167&s=257264&e=jpg&b=7b669e)

  


> Demo 地址：https://codepen.io/airen/full/MWMbPeB

  


再次强调一下，`calc()` 函数的使用，需要确保它的计算生效！

  


## 创建等边三角形

  


[维基百科是这样描述等边三角形的](https://en.wikipedia.org/wiki/Equilateral_triangle)：“等边三角形（Equilateral Triangle）是一个具有相等边长和相等角度的三角形”。它的特征如下：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/0687874088c3484f86d8bb5885a9e233~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1824&s=325084&e=jpg&b=fcfcfc)

  


-   **相等的边长：** `a = b = c`
-   **相等的角度：** `α = β = γ`
-   **相等的高：** `ha = hb = hc`

  


这些特性意味着等边三角形的三个边长相等，三个内角都等于 `60deg`，三个高（从顶点垂直到对边的距离）也相等（`a * sqrt(3) / 2`）。你可能会好奇，这个高怎么就是 `a * sqrt(3) / 2` ？其实很简单，我们通过勾股定理或[三角函数](https://juejin.cn/book/7223230325122400288/section/7242216512176521277)就可以计算出来：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/b4a2098ab5b74ab193b4751643ef4b8c~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=3677&h=2898&s=1426686&e=jpg&b=fbfcf7)

下图展示的是 **[三角函数](https://juejin.cn/book/7223230325122400288/section/7242216512176521277)** 的计算公式：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/7280d25d66b145508bfae7eb6d59677b~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1280&h=983&s=91201&e=jpg&b=2c2b47)

  


通过勾股定理或三角函数，我们可以计算出等边三形的高 `h'` ，可以获得构建等边三角形的三个顶点坐标。使用 CSS 代码来描述的话，大致如下：

  


```CSS
.triangle--equilateral {
    --w: 50;
    --a: 60deg;
    --h: calc(var(--w) * sqrt(3) / 2); /* 或者 calc(sin(var(--a)) * var(--w)) */
    width: calc(var(--w) * 1vmin);
    aspect-ratio: 1;
    clip-path: polygon(50% calc((var(--w) - var(--h)) / var(--w) * 100%), 100% 100%, 0 100%);
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/02f2d4d4e8fd4e0f93681638bbaaf427~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1167&s=272832&e=jpg&b=7b669e)

  


> Demo 地址：https://codepen.io/airen/full/PorbyrQ

  


我们还可以将上面的代码再优化一下，将计算出来的高 `h'` 作为元素的高（`height`），这样使 `polygon()` 的坐标值更简单明了：

  


```CSS
.triangle--equilateral {
    --w: 50vmin;
    --a: 60deg;
    --h: calc(var(--w) * sqrt(3) / 2); /* 或者 calc(sin(var(--a)) * var(--w)) */
    width: var(--w);
    height: var(--h);
    clip-path: polygon(50% 0, 100% 100%, 0 100%);
}
```

  


或者根据计算出来的 `h'` 重置 [`aspect-ratio` 属性](https://juejin.cn/book/7223230325122400288/section/7259316040151236666)的值：

  


```CSS
.triangle {
    --w: 50vmin;
    --a: 60deg;
    width: var(--w);
    aspect-ratio: calc(1 / sqrt(3) * 2); /* 或 calc( 1 / sin(var(--a))) 或 calc(1 / cos(30deg))*/
    clip-path: polygon(50% 0, 100% 100%, 0 100%);
}
```

## 创建边框三角形

  


  


使用 `clip-path` 的 `polygon()` 函数创建一个简单的三角形很容易，只需要知道三角形的三个顶点坐标即可，但想要创建类似下图这样的边框三角形，挑战性还是蛮高的。

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/6d8b29b4f0aa4391861d3c07393df4e4~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=777&s=246158&e=jpg&b=ffffff)

  


我们仍然使用 `clip-path` 的 `polygon()` 函数来创建边框三角形，只不过需要更多的点。

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/2dd7a4285be54b60a295ec61de4ff974~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1236&s=490883&e=jpg&b=fbfcf7)

  


注意，上图左侧顶部的空隙是我特意留出的，以便大家更易于理解。如上图所示，绘制边框三角形，我们需要八点的坐标。你可能会认为只要使用六个点就可以绘制出所需要的边框三角形，因为我们绘制一个三角形只需要三个点，绘制一个边框三角形就只要六个点（边框外三个点，边框内三个点），但事实并非如此。

  


也就是说，绘制一个边框三角形，我们需要八个点，不过我们只要六个点的坐标，因为顶部有四个点重叠，如上图中所示，`P1` 和 `P4` 重叠，它们两坐标相同，`P5` 和 `P8` 重叠，它们两坐标相同。如果用代码来描述的话，大致如下：

  


```CSS
.triangle--border {
    clip-path: polygon(
        /* 外部坐标点 */
        P1, P2, P3, P1, /* p4 与 P1 重叠，P4 = P1 */
        
        /* 内部坐标点 */
        P5, P6, P7, P5 /* P8 与 P5 重叠， P8 = p5 */   
    )
}
```

  


把上面代码中 `P1 ~ P8` 替换为真实的坐标值：

  


```CSS
.triangle {
    width: 50vmin;
    aspect-ratio: 1;
    clip-path: polygon(
        /* 外部点 */
        50% 0, 100% 100%, 0 100%, 50% 0,
        /* 内部点 */
        50% 10%, 10% 90%, 90% 90%, 50% 10%
    );
}
```

正如你所看到的，上面代码创建的边框三角形是有瑕疵的，每条边的粗细程度不一致。换句话说，使用 `clip-path` 的 `polygon()` 创建边框三角形，其最困难的部分是找到内部点的正确坐标，以便在所有边上获得一致的厚度。这就需要应用一些几何知识，通过一些公式来计算出内部点的正确位置。我们以等边三角形为例：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/d958ed211eb84e6193bea8d977faf3e8~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=3095&h=2065&s=746897&e=jpg&b=fbfcf7)

  


这有一个小技巧，为了便于计算，我们把 `P1P5` 两点之间的距离定义为 `b` （即 `--b` ，使用 [CSS 自定义属性](https://juejin.cn/book/7223230325122400288/section/7249357815410589733)来定义该值）。然后，根据[三角函数](https://juejin.cn/book/7223230325122400288/section/7242216512176521277)我们可以计算出直角三角形的 `a` 和 `c` 边长，相应的就能得到 `P6` 和 `P7` 两个点的确切值：

  


```CSS
.triangle--border {

    --w: 50vmin;
    --p1: 50% 0;
    --p2: 100% 100%;
    --p3: 0 100%;
    --p5: 50% var(--b);
    --p6: calc(var(--b) * cos(30deg))  calc(100% - var(--b) * sin(30deg));
    --p7: calc(100% - var(--b) * cos(30deg)) calc(100% - var(--b) * sin(30deg));
    
    width: var(--w);
    aspect-ratio: 1 / cos(30deg);
    clip-path: polygon(
        /* 外部的点 */
        var(--p1), var(--p2), var(--p3), var(--p1), /* p4 = p1 */
        var(--p5), var(--p6), var(--p7), var(--p5)
    );
}
```

## 创建圆角三角形

  


有的时候，我们在还原 Web UI 时，难免会碰到带有圆角的三角形形状，例如：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/e7d429cdeaba4126b382e0427d01f973~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=597&s=316446&e=jpg&b=f3c78d)

  


以上图中左侧的徽标（卡片右上角黄色三角形）为例。对于大部分 Web 开发者而言，还原这种 UI 还是有点棘手的，因为三角形的所有角都是圆角，除了右上角和左下角，它们是从边缘裁剪出来的。为了实现这个效果，[我们可以利用上一节课中提到的 `polygon()` 技巧](https://juejin.cn/book/7395119173656903715/section/7395172754053890063)，它的坐标值可以超出 `[0%, 100%]` 的范围。正如下图所示，我们可以将 `polygon()` 的坐标超过 `100%` ，使其只裁剪一半的圆角：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/88b517f8e8c9493baf69c26e40e89f1a~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1334&s=232442&e=jpg&b=ffffff)

  


左侧展示的是 `polygon()` 坐标在 `[0%,100%]` 范围内，这是一种错误的做法；右侧展示的是 `polygon()` 坐标超过了 `[0%,100%]` 范围，是正确的裁剪方式。注意，浅蓝色遮盖的部分表示被裁剪掉的部分。如果用代码来表示的话，大致如下：

  


```CSS
/* 错误的做法 */
.element {
    clip-path: polygon(0 0, 100% 0, 100% 0, 0% 100%);
}

/* 正确的做法  */
.element {
    clip-path: polygon(0 0, 100% 0, 104% 0, 0% 104%);
}
```

  


我们把正确的代码用于实际案例中：

  


```CSS
.badge {
    clip-path: polygon(0 0, 104% 0, 0 104%);
    display: grid;
    width: 100px;
    aspect-ratio: 1;
    background: #FCB216;
    border-radius: 16px;
    position: absolute;
    left: -6px;
    top: -6px;
}
```

你可能已经发现了，上面示例中所呈现的圆角三角形是借助于 `border-radius` 实现的，而且刚好利用了元素框的角。这意味着，上面的方法只适用于特定的场景中，并不是一种通用的解决方案。如果 `clip-path` 裁剪的三角形刚好不在元素框的角上，那么上面的方法就无法创建出圆角三角形。例如下图这样的效果：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/1954f179452048fcace6bb41d71addf6~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1175&s=1502779&e=jpg&b=e3e9f2)

  


对于上图中的圆角三角形，仅使用 `clip-path` 是无法实现的，但 [`mask`](https://juejin.cn/book/7223230325122400288/section/7259668885224456252#heading-9) 结合[径向渐变](https://juejin.cn/book/7199571709102391328/section/7199845781149810727)（`radial-gradient()`）就可以做到。下图说明 `mask` 、径向渐变和 `clip-path` 实现圆角三角形的整个过程：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/5ff4798edef746c1b89dcf58e35364be~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1417&s=272563&e=jpg&b=ebf0f6)

  


简单解释一下：

  


-   1️⃣：使用 `clip-path: polygon()` 裁剪出一个三角形，例如 `clip-path: polygon(50% 0, 100% 100%, 0 100%)`
-   2️⃣：使用 `radial-gradient()` 创建三个圆，每个圆放置在三角形的一个角落（如上图中的红色小圆圈）。我们需要三个渐变和一个变量来控制它的半径
-   3️⃣：使用另一个渐变填充中间区域（如上图中蓝色区域所示）
-   4️⃣：将红色和蓝色区域视为元素的可见区域，这样就得到了一个圆角三角形

  


看上去似乎很简单，实际上并没有你所想的那么容易，其中最困难的地方是渐变配置。可以说，它比创建边框三角形还更具挑战性，因为它依赖于三角形的性质，每种类型的三角形都有其实现方式。

  


以下是上图的代码：

  


```CSS
.triangle {
    --r: 20px; /* 圆角半径 */

    width: 50vmin;
    aspect-ratio: 1 / cos(30deg);
    --_g: calc(tan(60deg) * var(--r)) bottom var(--r), #000 98%, #0000 101%;
    mask:
        /* 用于填充中间区域的锥形渐变,对应上图是蓝色区域 */ 
        conic-gradient(from -30deg at 50% calc(200% - 3 * var(--r) / 2), #000 60deg, #0000 0) 0 100% / 100% calc(100% - 3 * var(--r) / 2) no-repeat,
        /* 3个径向渐变用于3个圆，对应上图中红色圆圈 */
        radial-gradient(var(--r) at 50% calc(2 * var(--r)), #000 98%, #0000 101% ),
        radial-gradient(var(--r) at left var(--_g)),
        radial-gradient(var(--r) at right var(--_g));
        
    /* 裁剪三角形 */
    clip-path: polygon(50% 0, 100% 100%, 0 100%);
    background: #fcb216;
}
```

现在我们已经有了形状，[让我们为它们添加动画吧](https://s.juejin.cn/ds/iMbtB4fN/)！无论是边框三角形还是圆角三角形，都依赖一个 CSS 自定义属性来控制边框的厚度（`--b`）或圆角半径（`--r`）。我们可以对它们（`--b` 或 `--r`）进行动画化，从而实现一些有趣的效果。

  


为此，我们需要依赖[ CSS 的 `@property` 来注册 CSS 自定义属性](https://juejin.cn/book/7223230325122400288/section/7258870477462962236)（也就是常说的 CSS 变量），并使其能够进行动画：

  


```CSS
@property --r {
  syntax: "<length>";
  initial-value: 0px;
  inherits: false;
}

@property --b {
  syntax: "<length>";
  initial-value: 0px;
  inherits: false;
}
```

随着越来越多的[现代 Web 布局技术](https://s.juejin.cn/ds/iMghaTK8/)的涌现，我们不再局限于规则的矩形框布局，越来越多的[创意布局](https://juejin.cn/book/7161370789680250917/section/7161624198429802504)用在 Web上。其中不乏三角形的网格布局。例如下面这个图片墙的布局效果：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/a8dd68ca6d354fd3a9240c6c0c182696~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1656&h=548&s=16446715&e=gif&f=55&b=0f0f0f)

  


我们简单的来看看这个图片墙布局是如何实现的：

  


```HTML
<div class="container">
    <div><img src="https://picsum.photos/800/800?random=1" alt=""></div>
    <div><img src="https://picsum.photos/800/800?random=2" alt=""></div>
    <!-- 省略其他的图片 -->
</div>    
```

  


在这里我们使用 [CSS Grid 进行布局](https://juejin.cn/book/7161370789680250917/section/7161372229123440648)：

  


```CSS
:root {
    --grid-size: calc(4.5rem + 2vw);
    --grid-size-y: calc(var(--grid-size) * 0.8660254);
    --grid-column-gap-mult: 0.023;
    --num-cols: 3;
    
    @media only screen and (width >= 570px) {
        --num-cols: 5;
    }
    @media only screen and (width >= 810px) {
        --num-cols: 7;
    }
    @media only screen and (width >= 1120px) {
        --num-cols: 9;
    }
    @media only screen and (width >= 1340px) {
        --num-cols: 11;
    }
    @media only screen and (width >= 1600px) {
        --num-cols: 13;
    }
}

.container {
    width: calc(100% - var(--grid-size) * 2);
    margin: calc(var(--grid-size) * 0.46) auto;
    display: grid;
    grid-auto-rows: var(--grid-size-y);
    grid-template-columns: repeat(var(--num-cols), var(--grid-size));
    gap: var(--grid-size-y) calc(var(--grid-size) * var(--grid-column-gap-mult));
    justify-content: center;
    font-size: calc(var(--grid-size) * 0.15);
    position: relative;
}
```

接着使用 `clip-path` 的 `polygon()` 将 `div` 裁剪成三角形。这里的三角形并没有特殊之处，但双数和奇数的方向刚好相反：

  


```CSS
div:nth-child(odd) {
    clip-path: polygon(0 0, 100% 0, 50% 98%, 0 0);
}

div:nth-child(even) {
    clip-path: polygon(50% 2%, 100% 100%, 0 100%, 50% 2%);
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/f6ffc881b9564eae80f1aecca059b985~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=932&s=723788&e=jpg&b=070707)

  


经过 `clip-path` 裁剪之后，我们得到所需要的三角形，但它并没有填满网格单元格，或者说相邻图片之间过于稀松。我们可以通过 [CSS 变换的 `scale` 来放大元素](https://juejin.cn/book/7223230325122400288/section/7259668493158023205)，使其达到我们所期望的效果：

  


```CSS
.container > div {
    scale: 1.93;
}
```

很简单，对吧！在我们这个示例中，还使用了 [CSS 媒体查询](https://juejin.cn/book/7223230325122400288/section/7257368158451793935)，在不同的断点下设置不同的列数（`--num-cols`），使整个布局具有响应式效果。另外，还可以给图片添加一些简单的交互效果，例如悬浮效果。你最终看到一个像下图这样的效果：
