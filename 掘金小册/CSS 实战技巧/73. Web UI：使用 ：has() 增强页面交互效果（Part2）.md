## 高亮目标元素

  


不知道大家是否还有印象，我曾在《[CSS 布局：防止锚点链接滚动到固定或粘性头部后面](https://juejin.cn/book/7395119173656903715/section/7445248285679304713#heading-1)》课程中使用 CSS 的 `:has()` 为锚点链接的目标元素设置高亮样式。在过去，当某个元素被 `:target` 选择器选中时，是无法为它的父元素应用样式。但现在，你可以使用 `:has()` 选择器来实现这一点：

  


```
section:has(h2:target) {
    outline-color: #ffeb3b;
    color: color-contrast(var(--c) vs #fff, #000 to AAA);
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/e2f45f032feb40af9f52aafc14382c9a~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1118&h=566&s=12297094&e=gif&f=250&b=849feb)

  


> Demo 地址：https://codepen.io/airen/full/gbYJxPR

  


请留意，示例中右下角的返回顶部按钮的显示和隐藏也是由 `:has()` 选择器来控制的：

  


```CSS
.back-to-top {
    opacity: 0;
    translate: calc(100% + 1rem);
    transition: all 0.5s;
}

body:has(h2:target) .back-to-top {
    opacity: 1;
    translate: 0px;
}
```

  


这个示例非常适用于长内容的页面，使用户能够清楚的知道自己在什么位置，并且能快速回到页面顶部！

  


## 带有悬浮按钮的卡片

  


当用户鼠标悬浮在按钮上时，为卡片添加一个悬浮效果。尝试将鼠标悬浮在按钮上，你会看到卡片会有一个高亮的轮廓效果。这一切都可以通过 `:has()` 选择器实现。

  


```CSS
.card {
    box-shadow: 0 0 0.125em rgb(0 0 0 / 0.125);
}

.card:has(button:hover){
    box-shadow: 0 0 0.125em #123dc4;
    outline: 1px solid #123dc4;
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/c5072a36e17e4f6bbe229ad5a4e0c9ee~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1248&h=638&s=5733352&e=gif&f=167&b=a8b4db)

  


> Demo 地址：https://codepen.io/airen/full/XJrwvqr

  


除此之外，我们还可以通过 `:focus-visible` 伪类，当按钮获得焦点时，卡片也有高亮样式：

  


```CSS
.card:has(button:is(:hover,:focus-visible)){
    box-shadow: 0 0 0.125em #123dc4;
    outline: 1px solid #123dc4;
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/8912927f72324315b4a9f2a0decf251b~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1522&h=696&s=9981823&e=gif&f=142&b=a9b5db)

  


> Demo 地址：https://codepen.io/airen/full/wBwLwbv

  


这个效果有点像 `:focus-within` ，它选择包含获得焦点的后代的元素。它允许我们做类似的事情：

  


```CSS
.card:focus-within {
    box-shadow: 0 0 0.125em #123dc4;
    outline: 1px solid #123dc4;
}
```

  


`:focus-within` 伪类比 `:has()` 存在的时间要长得多，因此它具有更好的浏览器支持。但有两个原因让我更倾向于在这种情况下使用 `:has`：

  


-   `:focus-within` 匹配的是 `:focus` 状态，而不是 `:focus-visible` 状态。这意味着即使用户通过鼠标点击按钮，焦点轮廓仍会显示
-   我并不想在任何后代元素聚焦时显示焦点轮廓，我只想在按钮聚焦时应用它（某些卡片包含可聚焦的链接）。如果我使用 `:focus-within`，用户就无法清楚地知道哪个互动子元素实际上获得了焦点！

  


最终，`:focus-within` 可以很有用，但它是一个相对粗糙的工具。使用 `:has()` 我们可以获得更精细的控制。

  


> 温馨提示，有关于 `:focus-visible` 和 `:focus-within` 更详细的介绍，请移步阅读《[CSS 焦点样式：`:focus-visible` 和 `:focus-within`](https://juejin.cn/book/7223230325122400288/section/7226969870444232742)》！

  


我们还可以利用 `:has()` 选择器对卡片做进一步的改善。比如，这是一些展示商品的卡片，当某商品无货时，按钮处于禁用状态，我们完全可以使用 `:has()` 选择器来隐藏被禁用（`disabled`）的按钮，并且给用户展示出“缺货”的提示信息：

  


```HTML
<div class="cards">
    <div class="card">
        <img src="https://picsum.photos/id/163/800/600" alt="">
        <h3>Afternoon Hammock</h3>
        <p>Kiwis (the fruit) are native to China and were originally called Chinese gooseberries.</p>
        <small>5 items left</small>
        <button>Add to cart</button>
    </div>
    <div class="card">
        <img src="https://picsum.photos/id/643/800/600" alt="">
        <h3>Strawberry Ice Cream</h3>
        <p>Limes are a citrus fruit. They are valued for the acidity of their juice and are often used in Mexican, Vietnamese, and Thai dishes. </p>
         <button disabled>Add to cart</button>
    </div>
</div>      
```

  


```CSS
.card:has(button:is(:hover, :focus-visible)) {
    box-shadow: 0 0 0.125em #123dc4;
    outline: 1px solid #123dc4;
}

.card:has(button[disabled]) {
    grid-template-rows: auto;
    
    button {
        display: none;
    }

    &::after {
        content: "0 item left";
        border-radius: 0.25rem;
        display: inline-flex;
        justify-self: start;
        font-size: clamp(0.4rem, 5.5cqw, 0.625rem);
        letter-spacing: 0.15em;
        line-height: 1;
        margin: 0;
        padding: clamp(0.125rem, 2cqw, 0.25rem);
        font-variant: small-caps;
        background: oklch(96% 0.05 var(--hue, 334.61));
        border: 1px solid oklch(93% 0.1 var(--hue, 334.61));
        color: oklch(35% 0.104 var(--hue, 334.61));
    }
  
    &:last-of-type {
        position: relative;
      
        &::before {
            content: "Not available";
            position: absolute;
            top: 0;
            left: 0;
            right: 0;
            border-radius: 0.25em .25em 0 0;
            padding: .5em;
            text-align: center;
            background: #ff0000c2;
            color: #fff;
        }
    }
}
```

## 使用数据属性和 `:has()` 实现双重悬停

  


通过结合数据属性和 `:has()` 选择器，我们可以创建一个非常有趣的效果。你可以通过悬停在一个项目上来触发另一个项目，而且它们不需要在同一个父元素下。只需要为两个或更多元素添加数据属性，然后使用 `:has()` 来知道何时悬浮在某个项目上，并触发具有相同数据属性的其他项目：

  


```HTML
<main>
    <ul class="category">
        <li data-category="naps">Naps</li>
        <li data-category="computers">Computers</li>
        <li data-category="snacks">Snacks</li>
    </ul>
    <div class="cards">
        <div class="card" data-category="naps"></div>
        <div class="card" data-category="computers">Computers</div>
        <div class="card" data-category="snacks">Snacks</div>
    </div>
</main>
```

  


```CSS
main:has(li:hover) .card {
    background: oklch(96% 0.05 var(--hue));
    outline: 1px solid oklch(93% 0.1 var(--hue));
    color: oklch(35% 0.104 var(--hue));
}

main:has([data-category="naps"]:hover) [data-category="naps"] {
    --hue: 334.61;
}

main:has([data-category="computers"]:hover) [data-category="computers"] {
    --hue: 91.24;
}

main:has([data-category="snacks"]:hover) [data-category="snacks"] {
    --hue: 233.35;
}
```

## 使用 `:has()` 选择器实现过滤器

  


使用 CSS 的 `:has()` 选择器，我们可以检查输入状态，例如复选框（`checkbox`）、单选按钮（`radio`）或下拉选择框（`<select>`）。将 `:has()` 与输入状态结合使用时，可以实现一些有趣的应用场景，比如过滤器功能。

  


```HTML
<main>
    <div class="controls">
        <label for="naps">
            <input type="checkbox" value="naps" name="category"  id="naps" class="sr-only" checked />
            Naps
        </label>
        <label for="computers">
            <input type="checkbox" value="computers" name="category"  id="computers" class="sr-only" />
            Computers
        </label>
        <label for="snacks">
            <input type="checkbox" value="snacks" name="category"  id="snacks" class="sr-only" />
            Snacks
        </label>
    </div>
    <div class="cards">
        <div class="card" data-category="naps"></div>
        <div class="card" data-category="computers">Computers</div>
        <div class="card" data-category="snacks">Snacks</div>
    </div>
</main>
```

  


```CSS
.card {
    display: none;
}

main:has([value="snacks"]:checked) [data-category="snacks"],
main:has([value="naps"]:checked) [data-category="naps"],
main:has([value="computers"]:checked) [data-category="computers"] {
    display: grid;
}
```

你也可以将复选框换成下拉选择框（`<select>`），有兴趣的同学可以一试！

  


另外，`:has()` 选择器还可以根据表单控件的其他状态来调整 UI 样式或交互方式，例如下面这个示例：

  


```CSS
.username:not(:has(:user-valid)) + .password {
    height: 0;
    opacity: 0;
    overflow: clip;
}
```
