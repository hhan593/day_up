在设计一个用户界面时，时常会在两个部分之间添加一条分隔线。例如 CodePen 的登录界面，在桌面端时是一条竖线，在移动端时，这条线将变成水平线：

在实现这个效果之前，我们先把重心放在分隔线的制作上。

  


接下来，我将向大家展示如何基于一个 `<hr>` 元素来创建一些具有创意的线条效果。

  


```
<hr />
```

  


在 CSS 中，创建线条的方法有很多，我们可以[基于 `border`](https://juejin.cn/book/7199571709102391328/section/7199845356845629472) 和 `box-shadow 创建`一些普通的线条，可以通过 [CSS 渐变](https://juejin.cn/book/7199571709102391328/section/7199845781149810727)和 [`clip-path` 创建一些复杂的线条](https://juejin.cn/book/7223230325122400288/section/7259668885224456252)，例如带有渐变颜色的线条、纹理线条，锯齿线条和波浪线等。我们还可以使用 [SVG 的 `<path>` 来创建一些曲线线条](https://juejin.cn/book/7341630791099383835/section/7368317966821425190)。

  


```CSS
hr {
    --b: round(0.125em, 1px);
    --c: #3f4344;
    display: block;
    width: 80vw;
    height: var(--b);
    border: none 0;

    &:nth-child(1) {
        background: var(--c);
        box-shadow: 0 var(--b) 0 0 #5f6364;
    }

    &:nth-child(2) {
        height: calc(2 * var(--b));
        background: linear-gradient(to right, #0000,color-mix(in srgb, var(--c), yellow 60%),#0000);
    }

    &:nth-child(3) {
        overflow: hidden;
        height: 25px;

        &::after {
            content: "";
            display: block;
            margin: -25px auto 0;
            width: 100%;
            height: inherit;
            border-radius: 125px / 12px;
            box-shadow: 0 0 8px color-mix(in hsl, var(--c), #fff 60%);
        }
    }

    &:nth-child(4) {
        position: relative;

        &::before,
        &::after {
            content: "";
            position: absolute;
            inset: 0;
        }

        &::before {
            background: linear-gradient(90deg,#474c4d 0%,#474c4d 50%,#0000 50%,#0000 100%);
            background-size: 15px;
            background-position: center;
            z-index: 1;
            animation: bar 120s linear infinite;
        }
        
        &::after {
            background: linear-gradient(to right,#62efab 5%,#f2ea7d 15%,#f2ea7d 25%,#ff8797 35%,#ff8797 45%,#e1a4f4 55%,#e1a4f4 65%,#82fff4 75%,#82fff4 85%,#62efab 95%);
            background-size: 200%;
            background-position: 0%;
            animation: bar 15s linear infinite;
        }
    }
}

@keyframes bar {
    0% {
        background-position: 0%;
    }
    100% {
        background-position: 200%;
    }
}
```

这个示例中的代码都非常简单：

  


-   第一条：利用颜色的差，使用 `border` 和 `box-shadow` 制作了一个带立体感的线条
-   第二条：使用渐变，从透明过渡到某个颜色，然后再过渡到透明
-   第三条：使用 `border-radius` 和 `box-shadow` 制作一条两端带尖角的阴影线
-   第四条：使用渐变和 `animation` 绘制具有动画效果的七彩虚线

  


其实，我们只需要稍微利用好 CSS 的相关特性，并借助伪元素 `::before` 和（或 `::after`）还可创建出其他形状的线条。

        --s: 1rem;

        position: absolute;

        border: var(--b) solid var(--c);

        top: 50%;
        translate: -50% -50%;
    }
    
    &:nth-child(2)::before {
        top: 1px;

    }
    
    &:nth-child(3)::before {
        border-top: none;
        border-bottom: none; 
    }
    
    &:nth-child(4)::before {

在这里，我只是想通过这几个实际案例告诉大家，我们可以使用现代 CSS 特性来制作各种花式线条，而这些线条都可以用作分隔符。例如下图这个效果：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/83bb91aab60942d085273b6c7bb6ea8a~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1908&s=260138&e=jpg&b=ffffff)

  


我想，你应该可以使用 CSS 制作出上图中的线条分隔符的效果，但我接下来要向大家展示的是，应该如何使线条分隔更具响应式。

  


我们先来看一下 HTML 代码：

  


```HTML
<div class="section">
    <div class="section__item section__item--start">
        <!-- Section Content -->
    </div>
    <div class="section__item section__item--end">
        <!-- Section Content -->
    </div>
</div>
```

  


在 `.section` 元素中包括了两个 `.section__item` 元素。它们之间将要添加一条分隔线。我在这里[使用 CSS Flexbox 完成基本的布局](https://juejin.cn/book/7161370789680250917/section/7161621092560273439)：

  


```CSS
.section {
    display: flex;
    gap: 4rem;
    justify-content: center;
    flex-wrap: wrap;
}
```

  


两个子元素之间的间距是 `4rem` ，并且它们[垂直居中](https://juejin.cn/book/7161370789680250917/section/7161623670622781471)。现在的效果如下图所示：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/afc57a5c9d99499ca61211909c4c856e~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=954&s=319351&e=jpg&b=fdfafa)

  


为了减少使用额外的 HTML 标签元素，我将使用 `.section` 的伪元素 `::before` 来创建线条分隔符：

  


```CSS
.section::before {
    --b: round(.125em, 1px);
    content: "";
    width: var(--b);
    background: linear-gradient(var(--a, 0deg),#0000,currentColor,#0000);
}
```

  


我做了自定义属性 `--b` 来控制分隔线的宽度，并且使用 `linear-gradient` 给伪元素填充了一个渐变色。注意，我们并没有给伪元素设置 `height` ，主要原因是，我们并不知道 `.section__item` 容器的高度是多少（它们是由内容来决定的）。再加上，我们在 `.section` 显式设置了 `align-items: center` ，使元素垂直居中。此时，浏览器渲染出来的伪元素的高度是 `0` ，宽度则是 `--b` ，所以在浏览器中看不到这条渐变的分隔线。

  


我们可以在 `::before` 上显式设置 `align-self: stretch` ，使它的高度拉伸到与 `.section` 容器高度一样：

  


```CSS
.section::before {
    --b: round(.125em, 1px);
    content: "";
    width: var(--b);
    background: linear-gradient(var(--a, 0deg),#0000,currentColor,#0000);
    align-self: stretch;
}
```

  


现在它的效果如下：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/2c6563d41cc04fa789bdadc7a37a8ff3~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=954&s=326028&e=jpg&b=fdfafa)

正如你所看到的，现在伪元素绘制的渐变分隔线与容器等高了，但它并没有位于两个子元素的中间。接下来，我们需要重新对 Flex 项目进行排序，使分隔线位于两个 `.section__item` 之间：

  


```CSS
.section__item--start {
    order: -1;
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/c881a62c606d4361b3b36e78e598bba9~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1017&s=349639&e=jpg&b=fdf9f9)

  


现在，在桌面端（宽屏）下，整个效果还是不错的，但当你调整浏览器视窗宽度时，它的效果并不理想：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/6d3328a2932a4adcbda71ea803280073~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1068&h=522&s=10133328&e=gif&f=251&b=fdfafa)

  


我们期待的效果是，在窄屏时，分隔线需要变成水平的，同样位于两者之间。

  


我们可以考虑使用 [CSS 媒体查询 `@media` ](https://juejin.cn/book/7223230325122400288/section/7257368158451793935)来实现这个效果。通常情况之下，都是移动端先行。因此，默认情况之下，将 `flex-direction` 设置为 `column` ，在较大屏幕上再将其重置为 `row` ：

  


```CSS
.section {
    /* 省略其他 CSS */
    flex-direction: column;
    
    @media only screen and (width >= 768px) {
        flex-direction: row;
    }
}
```

  


与此同时，我们的分隔线也需要做一些调整，默认它是条水平线，然后在宽屏的时候才是垂直线：

  


```CSS
.section {
    /* 省略其他 CSS */
    
    flex-direction: column;
    
    &::before {
        height: var(--b);
        background: linear-gradient(var(--a, 90deg),#0000,currentColor,#0000);
    }
    
    @media only screen and (width >= 768px) {
        flex-direction: row;
      
        &::before {
            --a: 0deg;
            height: unset;
            width: var(--b);
        }
    }
}
```

  


注意，默认分隔线设置的是 `height` ，因为它是一条水平线，宽度等于容器的宽度，高度是 `--b` ，与此同时，线性渐变 `linear-gradient` 的渐变方向是 `90deg` （相当于 `to right`）。在宽屏下，例如浏览器视窗宽度大于或等于 `768px` 时，水平线变成了一条垂直线，需要重置 `width` 属性的值，在这里使用了 `unset` ，它会执行与 `initial` 或 `inherit` 相同的操作，与 `width: auto` 等效，因为 `width` 的初始值为 `auto`，并且是一个非继承属性。这是 CSS 重置值的一个小技巧，如果你对这方面的知识感兴趣，请移步阅读《[CSS 显式默认值：`inherit`，`initial`，`unset` 和 `revert`](https://juejin.cn/book/7223230325122400288/section/7232094160071688227)》！

  


除了重新设置伪元素的 `width` 和 `height` 属性的值之外，还需要记得调整 `linear-gradient` 渐变方向，在这里我们需要将 `--a` 调整为 `0deg` ，因为渐变的方法要变成从上往下或从下往上。

  


现在，整个效果就完美了：

完整的 CSS 代码如下：

  


```CSS
.section {
    display: flex;
    gap: 4rem;
    justify-content: center;
    align-items: center;
    flex-wrap: wrap;
    color: #333;
    flex-direction: column;

    &::before {
        --b: round(0.125em, 1px);
        content: "";
        height: var(--b);
        background: linear-gradient(var(--a, 90deg), #0000, currentColor, #0000);
        align-self: stretch;
    }

    @media only screen and (width >= 768px) {
        flex-direction: row;

        &::before {
            --a: 0deg;
            height: unset;
            width: var(--b); 
        }
    }

    .section__item--start {
        order: -1;
    }
}
```

  


这个示例向大家演示了如何创建一个响应式的线条分隔符，至于线条的外观，你可以根据自己的需求进行调整。

  


在这里需要提出来的是，使用现代 Web 布局技术（比如 CSS Flexbox）和 CSS 媒体查询 `@media` ，我们可以轻易实现具有响应式的线条分隔符。这种方案应用于页面级（或者说宏观上的布局）是不错的选择。但线条分隔符也常用于组件级别（微观上的布局），那么使用 CSS 的媒体查询就有点不恰当了。在这种情景之下，我更推荐使用 CSS 的容器查询。

  


以上面示例为基础，假设上面的效果是一个组件，它可能被放置在一个较小的容器，也可有可能会被放置在一个较大的容易。我们期望的是，被放置在小容器时，分隔符是一条水平线，反之则是条竖线。你可能已经猜到了，我们可以[使用 CSS 容器查询中的尺寸查询来实现这个效果](https://juejin.cn/book/7223230325122400288/section/7259668032165773368)，但它的 HTML 需要做一些调整，我们需要将整个 `.section` 放置在一个容器中，例如：

  


```HTML
<div class="container">
    <div class="section">
        <div class="section__item section__item--start">
            <!-- Section Content -->
        </div>
        <div class="section__item section__item--end">
            <!-- Section Content -->
        </div>
    </div>
</div>
```

  


然后在 CSS 中，将 `.container` 元素的 `container-type` 设置为 `inline-size` ，这样我们就可以通过 `@container` 规则来查询 `.container` 元素的内联轴尺寸（`inline-size` ，当书写模式是 `ltr` 或 `rtl` 时，它相当于 `width`）。`@container` 的使用方式与 `@media` 是相似的，我们可以像下面这样来调整 CSS ：

  


```CSS
.container {
    container-type: inline-size;
}

.section {
    display: flex;
    gap: 2rem;
    justify-content: center;
    align-items: center;
    flex-wrap: wrap;
    flex-direction: column;

    &::before {
        --b: round(0.125em, 1px);
        content: "";
        height: var(--b);
        background: linear-gradient(var(--a, 90deg), #0000, currentColor, #0000);
        align-self: stretch;
    }

    @container  (width >= 768px) {
        flex-direction: row;
        gap: 4rem;

        &::before {
            --a: 0deg;
            height: unset;
            width: var(--b);
            
        }
    }

    .section__item--start {
        order: -1;
    }
}
```

  


这个时候，`.section` 会根据它的容器尺寸来调整布局：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/76c1fc3581a04c5fb459f4fda27af93c~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=868&s=501750&e=jpg&b=0399aa)

  


> Demo 地址：https://codepen.io/airen/full/bGPmpbQ

  


正如你所看到，位于侧边栏的，分隔符是水平的；位于主内容栏，分隔符是垂直的。更为有意思的是，当你将浏览器视窗宽度调小，主内容栏宽度变时小，放置在内部的组件也会随着容器宽度响应，其分隔符也会由竖线变成水平线：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/a03574f79998472c912d13453155ab1a~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1374&h=582&s=13500952&e=gif&f=188&b=028ca1)

  


很神奇吧！这就是 [CSS 容器查询](https://juejin.cn/book/7223230325122400288/section/7259668032165773368)和[现代 Web 布局技术](https://s.juejin.cn/ds/ihaCFpnC/)的奇妙之处，我们不需要使用太多的 CSS ，就可以构建出响应性极强的效果。

  


## 创建标题分隔符

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/26155c778d274596b565ba453c32dc48~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2152&h=713&s=255003&e=jpg&b=000000)

  


标题分隔符是 Web 应用上的另一种常见分隔符，它们常用于在不同区域、章节或部分的标题中。标题分隔符有助于增强页面布局的清晰度、帮助用户快速理解和区分不同的内容模块或信息层次。例如下图中的“开学好物”和“潮玩手办”：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/f3573eac25634db08c117e9d1e6db5c2~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1170&h=2532&s=384382&e=jpg&b=fbf2e5)

  


标题分隔符与线条分隔类似，它会以各种不同的形式在 Web 上展现，但最为常见的是标题两侧有一些装饰物，比如下图这种效果：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/35d17ef1fd4441bba528b77719665dab~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1108&s=1288705&e=jpg&b=1a7979)

  


上图是一种非常常见的标题分隔符。接下来，我们就以这个为例，向大家展示如何使用 CSS 构建具有响应式的标题分隔符。

  


我们依旧从 HTML 结构开始。你可能需要一个类似下面这样的 HTML 结构：

  


```HTML
<h3 class="divider">Title</h3>
```

  


如果标题分隔符中的文本带有额外的样式，你可能需要使用一个元素将文本内容包裹起来，例如：

  


```HTML
<h3 class="divider"><span>Title</span></h3>
```

  


标题文本内容两侧的修饰符，我们可以考虑使用伪元素 `::before` 和 `::after` 来创建：

  


```CSS
.divider {
    --b: round(.125em, 1px);
    --gap: 1rem;
    font-size: clamp(1rem, 1.848vw + 0.522rem, 2rem);
    
    &::before,
    &::after {
        content: "";
        display: block;
        height: var(--b);
        background: linear-gradient(var(--a, 90deg), #0000, currentColor);
    }
    
    &::after {
        --a: -90deg;
    }
}
```

  


现在你看到的效果如下：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/a41687c4005a490abd3a15901ff1274c~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1046&s=87381&e=jpg&b=ffffff)

  


伪元素 `::before` 和 `::after` 分别创建了两条带有渐变效果的线条。注意，它们的渐变方向刚好相反。

  


接下来，我们要通过 Web 布局技术将线条和文本内容排成一行，并且垂直居中。实现这个布局效果有多种方式，我们先来看 [CSS Flexbox](https://juejin.cn/book/7161370789680250917/section/7161621092560273439)：

  


```CSS
 .divider {
     display: flex;
     align-items: center;
     gap: var(--gap);
     
     &::before,
     &::after {
         flex: 1 1 0%;
         min-width: 0;
     }
 }
```

  


这里的关键是，伪元素 `::before` 和 `::after` 上设置的 `flex: 1 1 0%` ，它会使伪元素均分 Flexbox 容器的剩余空间，并且两者保持一样的宽度。

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/cc1e3632e8f04687b959226ca8397cab~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1106&h=564&s=1666670&e=gif&f=213&b=ffffff)

  


完整的 CSS 代码如下：

  


```CSS
.divider {
    --b: round(.125em, 1px);
    --gap: 1rem;
    font-size: clamp(1rem, 1.848vw + 0.522rem, 2rem);
    display: flex;
    align-items: center;
    gap: var(--gap);
    
    &::before,
    &::after {
        content: "";
        display: block;
        height: var(--b);
        background: linear-gradient(var(--a, 90deg), #0000, currentColor);
        flex: 1 1 0%;
        min-width: 0;
    }
    
    &::after {
        --a: -90deg;
    }
}
```

  


温馨提示：Flexbox 布局中的 `flex` 属性非常强大，它能直接决定 Flex 项目的尺寸计算，但这个计算的过程是非常复杂的。如果你感兴趣的话，个人强烈建议你花一些时间阅读下面这些内容，通过这些内容的学习，你将能彻底的掌握 Flexbox 布局的核心：

  


-   [Flexbox 布局中的 `flex` 属性的基础运用](https://juejin.cn/book/7161370789680250917/section/7161623755972673550)
-   [Flexbox 中的计算：通过扩展因子比例来扩展 Flex 项目](https://juejin.cn/book/7161370789680250917/section/7161623797794078750)
-   [Flexbox 中的计算：通过收缩因子比例收缩 Flex 项目](https://juejin.cn/book/7161370789680250917/section/7164357320367931399)
-   [Flexbox 布局中的 `flex-basis`：谁能决定 Flex 项目的大小？](https://juejin.cn/book/7161370789680250917/section/7161623717074698247)

  


除了 CSS Flexbox 布局之外，你也可以考虑使[用 CSS Grid 布局](https://juejin.cn/book/7161370789680250917/section/7161623971073359902)，[使用 `grid-template-columns` 创建一个一行三列的网格](https://juejin.cn/book/7161370789680250917/section/7161623971073359902#heading-1)，并且第一列（伪元素`::before`）和最后一列（伪元素`::after`）的列宽设置为 `minmax(0,1fr)`，第二列列宽设置为 `auto` ，即可根据文本内容来决定该列列宽：

  


```CSS
.divider {
    --b: round(.125em, 1px);
    --gap: 1rem;
    font-size: clamp(1rem, 1.848vw + 0.522rem, 2rem);
    display: grid;
    grid-template-columns: minmax(0, 1fr) auto minmax(0, 1fr);
    place-content: center;
    gap: var(--gap);
    
    &::before,
    &::after {
        content: "";
        display: block;
        height: var(--b);
        background: linear-gradient(var(--a, 90deg), #0000, currentColor);
    }
    
    &::after {
        --a: -90deg;
    }
}
```

  


这里的关键第一列和第三列列宽是 `minmax(0, 1fr)` ，它有点类似于 CSS Flexbox 中的 `flex: 1` 。意思是说，将网格容器的剩余空间（除去文本内容宽度的空间）分成两等份，第一列和第三例各占一份。如果你对这方面的知识感兴趣的话，可以移步阅读《[Grid 布局中的计算](https://juejin.cn/book/7161370789680250917/section/7161624007702216735)》和《[可用于 Grid 布局中的函数](https://juejin.cn/book/7161370789680250917/section/7161624041885958151)》。

  


虽然这是两种完全不同的 Web 布局技术，但最终的结果是一致的：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/9ddae772717b421d96aa06696625fdcf~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1116&h=560&s=1996281&e=gif&f=236&b=ffffff)

  


你可能已经注意到了，当容器变窄之后，可能会引起标题文本断行，在这里我们可以添加两行 CSS 代码，使整个效果变得更完美一些：

  


```CSS
.divider {
    text-align: center;
    text-wrap: balance;
}
```

  


`text-align` 易于理解，`center` 值使文本水平居中。[`text-wrap:balance` 是 CSS 中一个排版技巧，它能使文本换行排版时更均衡](https://juejin.cn/book/7223230325122400288/section/7249196160626196515)。

  


  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/d09ba08811944642afec3257582633ae~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1084&h=562&s=2761803&e=gif&f=247&b=ffffff)

  


  


> Demo 地址：https://codepen.io/airen/full/mdZzOza

  


需要知道的是，当 Flexbox 或 Grid 容器没有足够多的空间时，文本两侧的线条的宽度会逐步收缩至 `0` ，最终不可见。

  


这个示例展示的标题分隔符，文本自身并没有太多额外的装饰效果。但是，有的时候你可能还需要给文本添加一些装饰效果。例如：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/bc008c439e4f4a1e9d0786c0bee6db4d~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=380&s=179388&e=jpg&b=ffffff)

  


实现上图所展示的标题分隔符，那么标题的文本内容使用一个标签元素来包裹，更易于控制它的 UI 效果：

  


```HTML
<h3 class="divider"><span>Title Divider</span></h3>
```

  


标题文本的背景样式，我们可以使用 CSS 的 `clip-path` 来实现，两侧的线条，可以像之前一样，使用伪元素 `::before` 和 `::after` 。整个布局你可以根据你自己的喜好来选择，比如 Flexbox 或 Grid。

  


```CSS
.divider {
    --b: round(0.125em, 1px);
    --s: round(1em, 1px);
    --gap: 1rem;
    --primary: #ff5722;
    --bg: var(--primary);
    --contrast-target: AA;
    --color-set: #001219, #2196f3, #9c27b0, #ff9800, #9e9e9e, #607d8b, #fff, #bbb;
    --color: color-contrast(var(--bg) vs var(--color-set) to var(--contrast-target));
    --shape: var(--s) 0, calc(100% - var(--s)) 0, 100% 50%, calc(100% - var(--s)) 100%, var(--s) 100%, 0 50%;
    font-size: clamp(1rem, 1.848vw + 0.522rem, 2rem);
    display: flex;
    align-items: center;
    gap: var(--gap);
    text-align: center;
    text-wrap: balance;

    span {
        display: grid;
        place-content: center;
        background: var(--bg);
        color: var(--color);
        padding: 1em calc(1em + var(--s));
        clip-path: polygon(var(--shape));
    }

    &::before,
    &::after {
        content: "";
        display: block;
        height: var(--b);
        background: linear-gradient(90deg, var(--bg), var(--bg));
        flex: 1 1 0%;
        min-width: 0;
    }

    &:nth-of-type(2) {
        display: grid;
        place-content: center;
        grid-template-columns: minmax(0, 1fr) auto minmax(0, 1fr);
        --gap: 0;
        --shape: var(--s) 0, 100% 0, calc(100% - var(--s)) 100%, 0 100%;
        --primary: #9c27b0;
      
        &::before {
            align-self: end;
            margin-right: -2px;
        }

        &::after {
            align-self: start;
            margin-left: -2px;
        }
    }
}
```

  


有关一布局的代码这里就不解释了。我在 `<span>` 元素上使用 `clip-path` 属性的 `polygon()` 分别裁剪了一个[丝带形状](https://juejin.cn/book/7395119173656903715/section/7399654560878788658)和一个[平行四边形](https://juejin.cn/book/7395119173656903715/section/7400975207551139875)，并且它们的切口大小都由 `--s` 变量来控制。另外需要知道的是，为了提高标题文本颜色（`color`）和它的背景颜色（`background`）的对比度能达到 Web 可访问性的 `AA` 级别，这里使用了[ CSS 的 `color-contrast()` 函数](https://juejin.cn/book/7223230325122400288/section/7239644346935410720)，使文本颜色能够根据 `background` 的变化做出响应式的调整。

  


你现在看到的效果如下：

在这个示例中，前两个标题分隔符效果（丝带形状）是由[锥形渐变 `conic-gradient`](https://juejin.cn/book/7223230325122400288/section/7259668771856941111) 和 `border-image` 创建的，其中锥形渐变绘制了中间的丝带效果，而 `border-image` 处理了两端的线条效果。后三个标题分割符效果（平行四边形）是由 `clip-path` 和 `border-image` 创建的，`clip-path` 用于裁剪形状，包括上下两条线，而 `border-image` 在这里可以被视作给元素填充背景。

  


## 创建区域分隔符

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/843b69e35b5a45c48f7f460b2f77283a~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2040&h=1900&s=1457340&e=jpg&b=f7eeed)

  


相对而言，创建区域分隔符要比线条分隔符和标题分隔符复杂一些，因为它除了涉及到创建形状之外，还直接与 Web 布局有关。换句话说，如果处理不好，它可能会影响到整个 Web 布局的效果。那么接下来，我将通过一些实际的案例，向大家展示如何使用 CSS 来创建区域分隔符。

  


创建区域分隔符重点会涵盖两个方面：创建图形和Web布局。我们先从分隔符的图形聊起。

  


### 创建一个倾斜的区域分隔符

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/a2e82da9ea484775b1d7d597eb4700c6~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1852&h=1120&s=717698&e=jpg&b=ed7437)

  


倾斜的区域分隔符算是最常见和最简单的区域分隔符之一了，它常常会出现在“对角布局”当中（如上图所示）。

  


倾斜的区域分隔符通常有两个元素组成，并通过一个倾斜的间隙分隔开来。

  


```HTML
<div class="section section--top">
    <!-- Content -->
</div>
<div class="section section--bottom">
    <!-- Content -->
</div>
```

  


为了实现这个效果，我们需要从每个元素中切除一部分，下图展示了它的实现过程：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/ad5573febee4410d9da18158ae11ca42~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2040&h=1622&s=407118&e=jpg&b=ffffff)

  


我将使用 CSS Grid 对两个元素进行布局，并且给它们设置一些基础样式，使其看起来叠放在一起：

  


```CSS
body {
    display: grid;
    grid-template-rows:repeat(2, minmax(0, 1fr));
}

.section {
    /* 控制颜色 */
    --primary: #2660F3;
    --bg: var(--primary);
    --contrast-target: AA;
    --color-set: #001219, #2196f3, #9c27b0, #ff9800, #9e9e9e, #607d8b, #fff, #bbb;
    --color: color-contrast(var(--bg) vs var(--color-set) to var(--contrast-target));

    font-size: clamp(1rem, 1.848vw + 0.522rem, 2rem);
    display: grid;
    place-content: center;
    color: var(--color);
    background: var(--bg);
    
    &:nth-child(2) {
        --primary: #E12F2F;
    }
}
```

  


你将看到的效果如下：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/f6f1841d269d4a43ad6f099c061dd51f~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1061&s=236137&e=jpg&b=2663f2)

  


接着，我们使用 `clip-path` 从顶部元素的底部切除一部分：

  


```CSS
.section {
    --s: round(3em, 1px); /* 控制被裁剪的高度 */
}

.section--top {
    clip-path: polygon(0 0, 100% 0, 100% 100%, 0 calc(100% - var(--s)));
}
```

  


上面代码，给 `polygon()` 函数设置了四个点，其中左下角的点向上移了 `--s` ，以创建切割效果。你可能已经注意到了，我们使用了 `calc(100% - var(--s))` 而不是 `100%` 。

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/e3c2594a7a794f1d859a88f7cb6da8f2~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1061&s=261880&e=jpg&b=e23030)

  


以同样的方式，对第二个元素进行裁剪，不同的是我们将右下角向下移动了 `--s` ：

  


```CSS
.section--bottom {
    clip-path: polygon(0 0, 100% var(--s), 100% 100%, 0 100%);
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/cfbfbff3cf054640b7e0454a49568442~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1061&s=272951&e=jpg&b=2663f2)

  


最后，我们给第二个元素添加一个负的 `margin-top` ，以减少两个元素之间的间距。我引入了一个新的变量 `--gap` 来控制两元素之间的间距：

  


```CSS
.section {
    --s: round(3em, 1px); /* 控制被裁剪的高度 */
    --gap: 1rem; /* 控制间距 */
}

.section--bottom {
    margin-top: calc(var(--gap) - var(--s));
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/874e710179b34df68d9aea538c2c2f5a~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1061&s=269048&e=jpg&b=e23030)

  


看上去效果似乎还不错，但我们还需要考虑另一个问题，那就需要在元素被裁的方向设置一个内距，否则元素的内容很容易被 `clip-path` 裁剪掉：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/8620ab23cbe740de9311822e81864a21~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1061&s=518244&e=jpg&b=e23030)

  


只需要像下面这样设置 `padding` 就可以避免上图这种现象出现：

  


```CSS
.section--top {
    padding-bottom: var(--s);
}

.section--bottom {
    padding-top: var(--s);
}
```

  


要是元素原本就设置了一定内距的话，我们需要在原来的基础上增加 `--s` ：

  


```CSS
.section {
    --p: round(1em, 1px); /* 控制内距 */
    padding: var(--p);
}

.section--top {
    padding-bottom: calc(var(--p) + var(--s));
}

.section--bottom {
    padding-top: calc(var(--p) + var(--s));
}
```

  


这样，你就创建了一个完美的倾斜的区域分隔符：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/1066378d98fe43dba1ff509a76d3672b~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1154&h=894&s=10761028&e=gif&f=234&b=de2727)

  


> Demo 地址：https://codepen.io/airen/full/dyBgmmL

  


上面示例对应的 CSS 代码如下：

  


```CSS
.section {
    --s: round(3em, 1px); /* 控制被裁剪的高度 */
    --gap: 1rem;          /* 控制间距 */
    --p: round(1em, 1px); /* 控制内距 */
    
    /* 控制颜色 */
    --primary: #2660f3;
    --bg: var(--primary);
    --contrast-target: AA;
    --color-set: #001219, #2196f3, #9c27b0, #ff9800, #9e9e9e, #607d8b, #fff, #bbb;
    --color: color-contrast(var(--bg) vs var(--color-set) to var(--contrast-target));
    
    display: grid;
    place-content: center;
    
    color: var(--color);
    background: var(--bg);
    padding: var(--p);
}

.section--top {
    clip-path: polygon(0 0, 100% 0, 100% 100%, 0 calc(100% - var(--s)));
    padding-bottom: calc(var(--p) + var(--s));
}

.section--bottom {
    --primary: #e12f2f;
    clip-path: polygon(0 0, 100% var(--s), 100% 100%, 0 100%);
    margin-top: calc(var(--gap) - var(--s));
    padding-top: calc(var(--p) + var(--s));
}
```

  


### 创建全宽箭头区域分隔符

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/5ff6c85cc7f248bf830a8b765a5eb5b1~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1852&h=1120&s=747190&e=jpg&b=ed6f39)

  


这个区域分隔符与之前的倾斜分隔符非常相似，我们只需要调整 `clip-path` 的 `polygon()` 坐标点即可。不同的是，我们通过三角形的高度 `--s` ：

  


```CSS
.section {
    --s: round(6em, 1px); /* 控制三角形高度 */
    --gap: 1rem; /* 控制间距 */
    --p: round(1em, 1px); /* 控制内距 */
    
    padding: var(--p);
    clip-path: polygon(var(--shape));
}

.section--top {
    --shape: 0 0, 100% 0, 100% calc(100% - var(--s)), 50% 100%, 0 calc(100% - var(--s));
    padding-bottom: calc(var(--p) + var(--s));
}

.section--bottom {
    --primary: #e12f2f;
    --shape: 0 0, 50% var(--s), 100% 0, 100% 100%, 0 100%;
    margin-top: calc(var(--gap) - var(--s));
    padding-top: calc(var(--p) + var(--s));
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/aed2189f7f5445449fa3e7b4b9224dba~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=988&h=892&s=10785124&e=gif&f=199&b=de2727)

  


> Demo 地址：https://codepen.io/airen/full/JjQmZqp

  


你可能已经注意到了，上面示例中的三角形箭头的位置始终位于元素的水平中心。我们可以通过引入一个 CSS 变量来替代这个 `50%` ，那么你就可以随意调整箭头的位置：

  


```CSS
.section {
    --s: round(6em, 1px); /* 控制三角形高度 */
    --gap: 1rem; /* 控制间距 */
    --p: round(1em, 1px); /* 控制内距 */
    --posX: 50%; /* 控制箭头位置 */
    
    padding: var(--p);
    clip-path: polygon(var(--shape));
}

.section--top {
    --shape: 0 0, 100% 0, 100% calc(100% - var(--s)), var(--posX) 100%, 0 calc(100% - var(--s));
    padding-bottom: calc(var(--p) + var(--s));
}

.section--bottom {
    --primary: #e12f2f;
    --shape: 0 0, var(--posX) var(--s), 100% 0, 100% 100%, 0 100%;
    margin-top: calc(var(--gap) - var(--s));
    padding-top: calc(var(--p) + var(--s));
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/7fce890065da42f588682a38e4cd8f2c~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=986&h=806&s=1222136&e=gif&f=143&b=2152f1)

  


> Demo 地址：https://codepen.io/airen/full/MWMPBjm

  


你可能已经注意到了，当 `--posX` 的值为 `0` 或 `100%` 时，全宽箭头区域分隔符就变成了倾斜的区域分隔符。

  


### 创建箭头区域分隔符

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/8c4136babdef44aaabdf0c152e0eda2e~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1852&h=1120&s=763842&e=jpg&b=ed6f39)

  


全宽箭头区域分隔符，它有一个最大的特色，那就是箭头（三角形）的宽度与容器的宽度相等。但是，有的时候，我们需要的箭头区域分隔符并不是全宽的。

  


创建箭头区域分隔符与创建全宽箭头区域分隔符原理是相似的。只不过 `clip-path` 的 `polygon()` 需要更多的数学计算来确定各点的坐标。对于这个区域分隔符，我将使用 `--s` （三角形高度）和 `--a`(三角形顶角的角度)来计算三角形的宽度 ：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/ddf132b4a6444e63af32235796dc209e~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1852&h=1120&s=791689&e=jpg&b=f66a8f)

  


我们通过 CSS 的三角函数来计算：

  


```CSS
.section {
    --s: round(6em, 1px); /* 控制三角形的高度 */
    --a: 90deg;           /* 控制三角形顶角的角度 */
    
    --gap: 1rem; /* 控制间距 */
    --p: round(1em, 1px); /* 控制内距 */
    
    clip-path: polygon(var(--shape));
}

.section--top {
    --shape: 0 0,100% 0,100% calc(100% - var(--s)),calc(50% + tan(var(--a)/2)*var(--s)) calc(100% - var(--s)),50% 100%,calc(50% - tan(var(--a)/2)*var(--s)) calc(100% - var(--s)),0 calc(100% - var(--s));
    padding-bottom: calc(var(--p) + var(--s));
}

.section--bottom {
    --primary: #e12f2f;
    --shape: 0 0,calc(50% - tan((180deg - var(--a))/4)*var(--gap) - tan(var(--a)/2)*var(--s)) 0,50% calc(var(--s) + (1/sin(var(--a)/2) - 1)*var(--gap)),calc(50% + tan((180deg - var(--a))/4)*var(--gap) + tan(var(--a)/2)*var(--s)) 0,100% 0,100% 100%,0 100%;
    margin-top: calc(var(--gap) - var(--s));
    padding-top: calc(var(--p) + var(--s));
}
```

  


温馨提示，`--shape` 中的坐标点有点复杂，它与制作带有边框的提示框是相似的。如果你对这个公式感兴趣，可以阅读《[CSS 图形：创建工具提示和语音气泡](https://juejin.cn/book/7395119173656903715/section/7408966599622787083)》一文。

  


你现在看到的效果如下：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/a88b8e1cd9c74a81969fbf57493c44eb~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1061&s=263845&e=jpg&b=e23030)

  


与之前的全宽箭头区域分隔类似，你可以通过引入 `--posX` 变量来调整箭头（三角形）的位置：

  


```CSS
.section {
    --s: round(6em, 1px); /* 控制三角形的高度 */
    --a: 90deg;           /* 控制三角形顶角的角度 */
    
    --gap: 1rem; /* 控制间距 */
    --p: round(1em, 1px); /* 控制内距 */
    --posX: 50%; /* 控制箭头位置 */
    
    display: grid;
    place-content: center;

    color: var(--color);
    background: var(--bg);
    padding: var(--p);
    clip-path: polygon(var(--shape));
}

.section--top {
    --shape: 0 0,100% 0,100% calc(100% - var(--s)),calc(var(--posX) + tan(var(--a)/2)*var(--s)) calc(100% - var(--s)),var(--posX) 100%,calc(var(--posX) - tan(var(--a)/2)*var(--s)) calc(100% - var(--s)),0 calc(100% - var(--s));
    padding-bottom: calc(var(--p) + var(--s));
}

.section--bottom {
    --primary: #e12f2f;
    --shape: 0 0,calc(var(--posX) - tan((180deg - var(--a))/4)*var(--gap) - tan(var(--a)/2)*var(--s)) 0,var(--posX) calc(var(--s) + (1/sin(var(--a)/2) - 1)*var(--gap)),calc(var(--posX) + tan((180deg - var(--a))/4)*var(--gap) + tan(var(--a)/2)*var(--s)) 0,100% 0,100% 100%,0 100%;
    margin-top: calc(var(--gap) - var(--s));
    padding-top: calc(var(--p) + var(--s));
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/58f8da7c6c7a4647a1668bf8e5b922cb~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1124&h=802&s=1491714&e=gif&f=202&b=de2727)

  


> Demo 地址：https://codepen.io/airen/full/qBzJyQv

  


到目前为止，我们已经使用相同的技术制作了三种不同的区域分隔符。每次我们通过调整 `clip-path` 值来考虑新的形状。你可以使用相同的技术创建任何区域分隔符，组合是无限的。唯一的限制就是你的想象力。

  


### 创建圆角区域分隔符

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/d216371eb9594ffcb0bd13d8a29e1d07~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1061&s=121667&e=jpg&b=275ef5)

  


这个区域分隔符，我们需要使用 `mask` 来制作，而不是 `clip-path` 。`mask` 和 `clip-path` 的区别在于，`mask` 依赖于图像（遮罩图）来裁剪或隐藏元素的部分区域。有关于这方面更详细的介绍，请移步阅读《[CSS Clipping 和 Masking](https://juejin.cn/book/7223230325122400288/section/7259668885224456252)》一文！

  


需要知道的是，遮罩图也可以是 CSS 渐变绘制的图像。对于我们即将要创建的圆角区域分隔符，它需要的遮罩图如下图所示：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/d7f25cec281544e2b64b684446359db6~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2040&h=1401&s=351431&e=jpg&b=ffffff)

  


在 `.section--top` 元素上使用两个渐变：

  


-   一个 `linear-gradient()` 创建顶部的矩形形状，在底部留出空间；
-   另一个 `radial-gradient()` 创建底部的圆形

  


两者结合将为第一个元素提供最终的形状。`.section--bottom` 元素只需要一个 `radial-gradient()` 在顶部创建一个圆孔，以完成拼图效果。具体的 CSS 代码如下：

  


```CSS
.section {
    --s: round(6em, 1px); /* 控制三角形的高度 */
    --a: 90deg; /* 控制三角形顶角的角度 */

    --gap: 1rem; /* 控制间距 */
    --p: round(1em, 1px); /* 控制内距 */
    
    /* 控制颜色 */
    --primary: #2660f3;
    --bg: var(--primary);
    --contrast-target: AA;
    --color-set: #001219, #2196f3, #9c27b0, #ff9800, #9e9e9e, #607d8b, #fff, #bbb;
    --color: color-contrast(var(--bg) vs var(--color-set) to var(--contrast-target));
    display: grid;
    place-content: center;

    color: var(--color);
    background: var(--bg);
    padding: var(--p);
    mask: var(--m);
}

.section--top {
    --m: 
        linear-gradient(#000 0 0) top/100% calc(100% - var(--s)) no-repeat,
        radial-gradient(farthest-side, #000 98%, #0000) bottom/calc(2 * var(--s)) calc(2 * var(--s)) no-repeat;
    padding-bottom: calc(var(--p) + var(--s));
}

.section--bottom {
    --primary: #e12f2f;
    --m: radial-gradient(calc(var(--gap) + var(--s)) at 50% calc(-1 * var(--gap)),#0000 98%, #000);
    margin-top: calc(var(--gap) - var(--s));
    padding-top: calc(var(--p) + var(--s));
}
```

  


你将看到的效果如下：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/a76591f4085d42a2b39978d275832b83~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1128&h=900&s=11863517&e=gif&f=253&b=de2727)

使用相似的方法，可以创建出[锯齿形状](https://juejin.cn/book/7395119173656903715/section/7403197889905688585#heading-0)、[凹槽形状](https://juejin.cn/book/7395119173656903715/section/7403197889905688585#heading-1)、[凸起形状](https://juejin.cn/book/7395119173656903715/section/7403197889905688585#heading-2)和[水波纹形状](https://juejin.cn/book/7395119173656903715/section/7403197889905688585#heading-3) 等区域分隔符：

  


```CSS
.section {
    --s: 40px;  /* 锯齿大小 */
    --a: 90deg; /* 锯齿角度*/
    --w: calc(var(--s) * tan(var(--a) / 2) * 2); /* 锯齿背景图宽度或高度 */
    --g: #0000, var(--bg) 1deg calc(var(--a) - 1deg), #0000 var(--a); /* 渐变颜色过渡 */
    
    --gap: 1rem; /* 控制间距 */
    --p: round(1em, 1px); /* 控制内距 */
    /* 控制颜色 */
    --primary: #2660f3;
    --bg: var(--primary);
    --contrast-target: AA;
    --color-set: #001219, #2196f3, #9c27b0, #ff9800, #9e9e9e, #607d8b, #fff,#bbb;
    --color: color-contrast( var(--bg) vs var(--color-set) to var(--contrast-target));
    display: grid;
    place-content: center;

    color: var(--color);
    background: var(--bg);
    padding: var(--p);
    padding-block: calc(var(--p) + var(--w));
    mask: var(--_g);
    z-index: var(--index);
}

/* 锯齿形状 */
.zig-zag  {
    --_g: conic-gradient(from calc(var(--a) / -2) at bottom, var(--g)) 50%/var(--w) 100% repeat-x;
    padding-block: calc(var(--p) + var(--w) / 2);
}

/* 凹槽形状 */
.scooped {
    --primary: #e12f2f;    
    --_g: radial-gradient(var(--s) at bottom, #0000 98%, var(--bg)) 50% / var(--w) 100%;
    margin-top: calc(-1 * var(--w) / 2);
      
}  

/* 凸起形状 */
.scalloped {
    --primary: #8BC34A;
    --gl: linear-gradient(to top, #0000 var(--s), var(--bg) 0);
    --gr: radial-gradient(circle var(--s), var(--bg) 98%, #0000) bottom / var(--w) calc(2 * var(--s));
    --_g: var(--gl), var(--gr);
    margin-top: calc(-1 * var(--w) / 2);
}
  
/* 水波纹形状 */  
.wavy {
    --primary: #e9a51e;
    /* 凹槽形状*/
    --g1: radial-gradient(var(--s) at 75% 100%, #0000 98%, var(--bg)) 50% calc(100% - var(--s)) / calc(4 * var(--s)) 100% repeat-x;
    /* 凸起形状 */
    --g2: radial-gradient(var(--s) at 25% 50%, var(--bg) 99%, #0000 101%) bottom/calc(4 * var(--s)) calc(2 * var(--s)) repeat-x;    
    --_g: var(--g1), var(--g2);
    margin-top: calc(-1 * var(--w) / 2);
}
```

## 使用 SVG 创建分隔符

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/430b925ac0a34bf0a565950d12b76e32~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1280&h=561&s=528175&e=png&a=1&b=fdfcfc)

  


  


前面我们通过多个案例向大家展示了如何使用现代 CSS 来构建线条分隔符、标题分隔符和区域分隔符。仔细观察，这些分隔符还是有一定的规律性的，比如锯齿区域分隔符。

  


不过，很多时候，Web 设计师会为了创意和美感，提供一些不规则的布局，其中就包括分隔符，例如曲线分隔符，水波纹区域分隔符等。然而，Web 开发者通常情况下习惯了规则的矩形布局的效果，对于 Web 设计师提供的不规则布局，难免会带有一种恐惧与排斥的心理。

  


事实上，**[现代 Web 布局](https://s.juejin.cn/ds/iF7oyPL2/)** 技术构建这些[不规则的布局](https://juejin.cn/book/7161370789680250917/section/7161625566062313503)，例如[类似杂志类创意布局](https://juejin.cn/book/7161370789680250917/section/7161624198429802504)，[不规则图形的布局](https://juejin.cn/book/7161370789680250917/section/7161625566062313503)等，不再是问题。而且[现代 CSS 提供了很多新特性](https://s.juejin.cn/ds/iF7oB1a1/)，使这件事情变得更容易。要是你将 [SVG 相关的特性](https://s.juejin.cn/ds/ih9kRSqD/)以及一些工具（比如 [Shapedivider](https://www.shapedivider.app/) ）结合起来，构建这些不规则的布局你可以手到擒来，包括我们今天聊的区域分隔符。

## 案例：创建对角布局

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/b97cdeeed9c04e63987193a2132cd5c4~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=2095&s=2394089&e=jpg&b=fdfcfc)

  


类似上图这种对角分割的布局已经流行好多年了，这是 Web 设计师用来打破那些乏味的直角矩形布局的一种常用工具。不难发现，这种布局（对角布局）算得上是区域分隔的典型案例之一，它非常类似之前所介绍的倾斜区域分隔符。

  


在 CSS 中，构建这种布局的方法有很多种。我所知道的方法至少有四种：

  


-   [使用 `clip-path` 隐藏部分区域](https://juejin.cn/book/7223230325122400288/section/7259668885224456252#heading-1)，类似于倾斜的区域分隔符
-   [使用 `mask` 隐藏部分区域](https://juejin.cn/book/7223230325122400288/section/7259668885224456252#heading-9)，可以使用两个锥形渐变 `conic-gradient` 和一个线性渐变 `linear-gradient` 绘制遮罩图
-   [使用 SVG 的路径（`<path>`）创建三角形形状](https://juejin.cn/book/7341630791099383835/section/7368317966821425190)，它可以作为 [SVG 的剪切](https://juejin.cn/book/7341630791099383835/section/7362748816769941540)（`<clipPath>`）或 [SVG 的遮罩](https://juejin.cn/book/7341630791099383835/section/7366549423813296165)（`<mask>`），然后结合[ CSS 的 `clip-path` 和（或 ）`mask` 隐藏部分区域](https://juejin.cn/book/7223230325122400288/section/7259668885224456252)
-   [使用 CSS 变换](https://juejin.cn/book/7288940354408022074/section/7295240572736897064#heading-7)，通过 `skew()` 函数对元素进行倾斜操作

  


接下来，我将向大家展示 CSS 变换的方法，因为之前的内容都是关于 `clip-path` 、`mask` 和 SVG 的，所以我选择了 CSS 变换来完成对角布局的效果。

  


我们从 HTML 结构开始，它由两个容器组成：

  


```HTML
<div class="diagonal">
    <div class="content"></div>
</div>
```

  


外层的 `.diagonal` 容器通常与浏览器视窗宽度相等，而内层 `.content` 包含实际的内容。通常会给 `.content` 设置一个最大宽度（`max-width`），并使其水平居中。注意，[实现水平居中方案有很多种](https://juejin.cn/book/7161370789680250917/section/7161623855054716935#heading-0)，我在这里使用了 `margin-inline: auto` ，其中 [`margin-inline` 是一个逻辑属性](https://juejin.cn/book/7223230325122400288/section/7254820664442355773)，书写模式为 `ltr` 或 `rtl` 时，表示水平方向居中：

  


```CSS
.diagonal {
    --p: 1.25rem; /* 控制内距 */
    width: min(100%, 100dvw);
    display: grid;
    
    .content {
        width: min(100% - 2 * var(--p), 60ch);
        margin-inline: auto;
    } 
}
```

  


注意，这里使用了 [CSS 的比较函数 `min()` 和 `max()`](https://juejin.cn/book/7223230325122400288/section/7241401565653762108) ，其中 `min(100%, 100dvw)` 可以用于[避免出现水平滚动条](https://juejin.cn/book/7199571709102391328/section/7213705145954074679#heading-14)；后者 `min(100% - 2 * var(--p), 60ch)` 等同于 `width: calc(100% - 2 * var(--p))` 和 `max-width: 60ch` 。其中 `dvw` 和 `ch` 都是 [CSS 中的相对单位](https://juejin.cn/book/7223230325122400288/section/7249357892611440700)。

  


现在，你可以给外层容器 `.diagonal` 设置一些颜色，使其看起来好看一些：

  


```CSS
:root {
    --angle: 0deg;
}

.diagonal {
    --p: 1.25rem; /* 控制内距 */
    
    --primary: #003;
    --bg: var(--primary);
    --contrast-target: AA;
    --color-set: #001219, #2196f3, #9c27b0, #ff9800, #9e9e9e, #607d8b, #fff,#bbb;
    --color: color-contrast(var(--bg) vs var(--color-set) to var(--contrast-target));
    background: linear-gradient(calc(var(--angle) - 45deg), var(--bg), color-mix(in srgb, var(--bg), currentColor 30%));
    color: var(--color);
}
```

  


注意，这里使用了 [`color-mix()` 函数](https://juejin.cn/book/7223230325122400288/section/7237288025221234744)，将 `--bg` 和 `30%` 的 `currentColor` （即 `--color`）进行混合调色，这样做的好处是，只需调整 `--bg` （即 `--primary`）就可以为其他区域设置不同的渐变颜色。另外，之前已经提到过了[`color-contrast()` 函数](https://juejin.cn/book/7223230325122400288/section/7239644346935410720)，它主要用于增强 `color` 和 `background` 的颜色对比度，利于提高文本的可读性。

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/2cda2c4b5db3423da2a244d89431433e~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1308&s=1062566&e=jpg&b=44993a)

  


接下来，我们将使用 `skewY()` 函数来沿着 `Y` 轴倾斜 `.diagonal` 元素，并且倾斜的角度是 `--angle` ：

  


```CSS
:root {
    --angle: 0deg;
}

.diagonal {
    /* 省略其他 CSS */
    transform: skewY(var(--angle));
        
    .content {
        transform: skewY(calc(-1 * var(--angle)));
    }
}
```

  


需要知道的是，当你对 `.diagonal` 元素进行倾斜 `skewY()` 操作时，它的子元素 `.content` 也交会继承这个变换。为了使内容 `.content` 不受父容器的变换影响，你需要对 `.content` 进行反转变换操作。在这个示例中，`skewY()` 需要向 `.diagonal` 元素相反的方向倾斜相同的角度。因此，上面代码中的 `.content` 的 `skewY()` 函数的值为 `calc(-1 * var(--angle))` 。这个时候你看到的效果如下：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/c28b0a4b6b7a4ab491e011d82d91cac5~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1202&h=898&s=9331171&e=gif&f=152&b=011043)

  


你可能已经注意到了，现在内容并没有完全被 `.diagonal` 容器包裹。如果你想 `.content` 中的内容不会溢出对角线（变换后的形状），那么需要添加一些内距（`padding`）。这个内距的正确值可以通过[三角函数](https://juejin.cn/book/7223230325122400288/section/7242216512176521277)计算出来：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/2c5fc74b9c5a41bfbd0b77f487a582e6~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1308&s=420779&e=jpg&b=070771)

  


我们知道 `.content` 的宽度 `--w` ，然后 `--angle` 是我们用来倾斜元素的角度。我们知道所有的部分形成了一个直角三角形。利用这些信息，我们可以使用下面的[三角函数公式](https://juejin.cn/book/7223230325122400288/section/7242216512176521277)计算出内距 `--p` ：

  


```CSS
--p = tan(--angle) * --w / 2
```

  


将这些变量应用到 CSS 中：

  


```CSS
:root {
    --angle: 0deg;
}

.diagonal{
    --w: 60ch;   /* max-width */
    --_p: 2rem;  /* 初始内距 */
    --_angle: abs(var(--angle)); /* 角度的绝对值 */
    --p: calc(tan(var(--_angle)) * var(--w) / 2); /* 上下内距*/
    
    transform: skewY(var(--angle));
    transform-origin: 50% 0;
    backface-visibility: hidden;
    
    .content {
        width: min(100% - 2 * var(--_p), var(--w));
        padding: calc(var(--p) + var(--_p)) var(--_p);
        transform: skewY(calc(-1 * var(--angle)));
        transform-origin: 50% 0;
        backface-visibility: hidden;
    }
}
```

  


注意，我们的内距 `padding` 不能是负值，所以在这里使用 CSS 的 `abs()` 函数，将应用于 `tan()` 函数中的角度值始终是一个正值，避免 `padding` 产生负值。另外，还给元素设置了一个初始内距，避免 `--angle` 值为 `0` 时，元素没有任何内距。

如果你想在内部元素上添加其他过渡效果（例如，淡入淡出的动画效果），你必须首先考虑添加反转变换。其他变换将叠回在第一个变换上。这可能会使你在创建动画效果时有点棘手。幸运的是，我们可以给 `.diagonal` 元素添加一个伪元素 `::before` 或 `::after` ，并且使其与容器具有一个相同的尺寸，然后对其进行倾斜变换操作。

  


```CSS
:root {
    --angle: 0deg;
}

.diagonal {
    --w: 60ch;
    --_p: 2rem;
    --_angle: abs(var(--angle));
    --p: calc(tan(var(--_angle)) * var(--w) / 2);
    --primary: #003;
    --bg: var(--primary);
    --contrast-target: AA;
    --color-set: #001219, #2196f3, #9c27b0, #ff9800, #9e9e9e, #607d8b, #fff,#bbb;
    --color: color-contrast(var(--bg) vs var(--color-set) to var(--contrast-target));
    
    color: var(--color);
    position: relative;
    width: min(100%, 100dvw);
    display: grid;
    
    
    &::after {
        content: "";
        position: absolute;
        inset: 0;
        z-index: -1;
        
        background: linear-gradient(calc(var(--angle) - var(--index) * 45deg),var(--bg),color-mix(in srgb, var(--bg), currentColor 30%));
        
        transform: skewY(var(--angle));
        transform-origin: 50% 0;
        backface-visibility: hidden;
        transition: transform 0.2s;
    }
    
    .content {
        width: min(100% - 2 * var(--_p), var(--w));
        margin-inline: auto;
        padding: calc(var(--p) + var(--_p)) var(--_p);
      
        display: grid;
        place-content: start center;
        gap: 1rem;
    }
}
```

  


整个变换操作都在伪元素 `::after` 上进行，此时也不再需要考虑对 `.content` 做反向变换操作之类的，而且在对内容做动画处理时，也无需考虑有没有初设的变换。注意，[我给伪元素 `::after` 的 `z-index` 设置了 `-1` 值](https://juejin.cn/book/7199571709102391328/section/7204402059225858052)，避免伪元素的背景遮盖住了 `.content` 。除了给伪元素的 `z-index` 设置负值之外，还有另外一种方式，那就是给 `.content` 元素设置一个 `position: relative` ，并且将 `z-index` 设置为 `2` ，记得要比伪元素的 `z-index` 属性的值更大，否则在视觉上会造成内容丢失不可见。

  


## 小结

  


现在，你随时随地都能获得一个炫酷的分隔符，更重要的是你知道如何使用现代 CSS 和 SVG 来创建它们的秘诀。

  


需要注意的是，在我们课程中演示的案例，分隔符形状的朝向基本上都是朝下的。但这并不意味着，课程中所介绍创建分隔符的技巧不适用于其他方向。我这样做是故意的，目的是让你尝试理解需要更新哪些数值才能获得相反的方向。我相信你一定可以做到，因为这并不难，而且你会在实践中学到很多。在这个过程中，你将会对 CSS 的 `clip-path` 、`mask` 、渐变、三角函数等特性会有更进一步的理解。
