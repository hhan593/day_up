![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/735837b90111480a98c70acb200e9e98~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1400&h=788&s=107973&e=png&b=e3ba73)

  


虚线（Dashed）是 Web 设计中的一个重要视觉元素。它通常用于视觉分隔、强调和引导用户的注意力。例如，虚线可以用来在导航栏中突出显示当前页面的链接，分隔不同的内容块，或者作为装饰性边框来增加页面的层次感。

在给元素设置虚线样式时，大家首先会想到的是关键词 `dashed` ，例如虚线边框（`border-style: dashed`）、虚线轮廓（`outline-style: dashed`）和文本下划线或删除线使用虚线来装饰（`text-decoration-style: dashed`）。不幸的是，默认情况下，`dashed` 除了在不同的系统和客户端中渲染出来的结果有差异之外，还无法控制虚线的单段长度与间隙。

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/10a105ec4a5a4e0391aa8c83e244c956~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=3582&h=2188&s=304029&e=jpg&b=fdfdfd)

  


除此之外，Web 设计师为了表达自己的创意或者达到某种诉求，会设计一些具有个性化的虚线效果。在此情况之下，默认的 `dashed` 也无法满足需求：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/4b4eef3f1aee441ebee885d46a37eaa3~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1239&h=1164&s=792388&e=jpg&b=f6f5f5)

  


庆幸的是，我们可以使用一些 CSS 技巧，例如渐变、遮罩等，可以帮助你实现高可定制性的虚线效果，而且还可以通过动画等技术，进一步增强虚线的视觉效果，使其更具吸引力和互动性。

## 虚线的关键术语

  


不知道你是否有这样的印象，在使用诸如 Figma 图形设计软件给对象设置虚线时，可以通过 “Dash” 和 “Gap” 两个参数来定义自己想要的虚线效果：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/8ea784702c174afcb19a0f45a1403dd2~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1172&h=662&s=1253304&e=gif&f=331&b=f5f5f5)

  


其中“Dash”参数用来控制虚线的单段长度，而 “Gap” 参数则用来控制虚线段之间的间距。与此同时，我们把虚线的单段长度和间距的组合又称为“虚线数组”（Dansh Array）。

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/05b8f9a5510a4cc588871c9eced9f4eb~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1280&h=586&s=61809&e=jpg&b=faf9f9)

  


在 SVG 的世界中，[我们可以使用 `stroke-dasharray` 属性来控制虚线的单段长度和间距](https://juejin.cn/book/7341630791099383835/section/7349188496181887017#heading-5)。通过在 `stroke-dasharray` 中设置不同的数值，可以创建各种类型的虚线效果，例如间断虚线、点状线和复杂的虚线模式：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/70c865e47031483d91a45a15ef710e67~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1339&h=968&s=657672&e=gif&f=548&b=3e423c)

  


另外，SVG 中绘制虚线时还提供了一个 `stroke-dashoffset` 属性用于控制虚线的起始位置：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/0c6fcc3039a64ff4a091cacbb332c8cc~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=880&h=980&s=1979157&e=gif&f=218&b=fbfafa)

  


虽然 `stroke-dasharray` 和 `stroke-dashoffset` 两个属性是 SVG 中用于描边的重要属性，但通过它们我们能更容易，更清晰的了解虚线的特性。你如你想更深入的了解它们，请阅读《[SVG 描边和填充](https://juejin.cn/book/7341630791099383835/section/7349188496181887017)》！

  


另外，`stroke-dasharray` 和 `stroke-dashoffset` 两属性可以应用于 CSS 中，但它只能用于设置 [SVG 图形元素](https://juejin.cn/book/7341630791099383835/section/7345813971552698406)和[文本元素](https://juejin.cn/book/7341630791099383835/section/7346773005114507304)的描边样式，并无法控制 HTML 元素的虚线样式。与此同时，在 CSS 中也没有与它们类似的属性可用于定制虚线样式。庆幸的是，我们可以使用 CSS 渐变来模拟虚线中的 “Dash”（单段长度）和“Gap”（间距），从而实现可定制化的虚线效果。

  


## 使用 `linear-gradient` 绘制虚线

  


CSS 的线性渐变 `linear-gradient` 有一个最基本的特性，该特性允许我们绘制一个类似斑马线的背景效果：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/a7901158e5954d4f98797da20ecefc8d~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1258&h=684&s=988676&e=gif&f=162&b=dadee4)

  


> URL：https://gradient.style/

  


仔细观察，它是不是与虚线的 “Dash” 和 “Ga” 非常相似。我们可以把上图中黑色部分当作是 “Dash”，白色部分当作是“Gap”，它们组成了绘制虚线的最小单元，即 `Dash + Gap` 。

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/46bd6b2f065c4369906a27fb81dd73b4~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=880&s=397692&e=jpg&b=376366)

  


在 CSS 中，我们可以将这几个参数的值用 [CSS 的自定义属性](https://juejin.cn/book/7223230325122400288/section/7249357815410589733)存储起来：

  


```CSS
.dashed--line {
    --dash: 40px; /* 虚线单段长度 */
    --gap: 40px;  /* 虚线段之间的间距 */
    --size: calc(var(--dash) + var(--gap)); /* 虚线最小单元尺寸 */
    --b: 10px; /* 虚线厚度 */
}
```

  


现在，我们将这些变量放入 `linear-gradient` 中，并且最终以背景的形式来呈现：

  


```CSS
.dashed--line {
    --dash: 40px; /* 虚线单段长度 */
    --gap: 40px;  /* 虚线段之间的间距 */
    --size: calc(var(--dash) + var(--gap)); /* 虚线最小单元尺寸 */
    --b: 10px; /* 虚线厚度 */
    --bg: #09face; /* 虚线颜色 */
    background: linear-gradient(to right, var(--bg) 0, var(--bg) var(--dash),#0000 var(--dash), #0000 var(--size)) left center / var(--size) var(--b) repeat-x; 
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/3bc999255b774c34aab636dab3cfe6c7~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1442&h=568&s=5048102&e=gif&f=413&b=31565c)

  


> Demo 地址：https://codepen.io/airen/full/qBzxjNm

  


你可以尝试着调整上面示例中的参数值，查看虚线样式的变化。

  


-   当 `--dash` 和 `--gap` 值相等时，相当于模拟了 `stroke-dasharray: var(--dash)` 的效果，例如 `stroke-dasharray: 40px`
-   当 `--dash` 和 `--gap` 值不相等时，相当于模拟了 `stroke-dasharray: var(--dash) var(--gap)` 的效果，例如 `stroke-dasharray: 40px 20px`

  


注意，示例中的渐变，我们还可以像下面这样进行优化：

  


```CSS
.dashed--line {
    --dash: 40px; /* 虚线单段长度 */
    --gap: 40px;  /* 虚线段之间的间距 */
    --size: calc(var(--dash) + var(--gap)); /* 虚线最小单元尺寸 */
    --b: 10px; /* 虚线厚度 */
    --bg: #09face; /* 虚线颜色 */
    background: linear-gradient(to right, var(--bg) 0, var(--bg) var(--dash),#0000 0) left center / var(--size) var(--b) repeat-x; 
}
```

  


需要知道的是：

  


-   虚线的样式是由 `linear-gradient` 控制，相当于创建了一张背景图（`background-image`）
-   虚线的大小是由 `background-size` 控制。如果虚线是水平线，其大小是 `var(--size) var(--b)`，其中 `--size` 相当于背景图片的宽度，`--b` 相当于背景图片的高度；如果虚线是垂直的，其大小则是 `var(--b) var(--size)` ，其中 `--b` 相当于背景图片的宽度，`--size` 相当于背景图片的高度
-   `background-repeat` 的值将决定虚线是水平线还是垂直线，当渐变水平方向平铺（`repeat-x`）时，绘制的是水平线；当渐变垂直方向平铺（`repeat-y`）时，绘制的是垂直线。注意，平铺方向调整时，它的 `background-size` 也要随之调整
-   `background-position` 则可以控制虚线的位置，例如，当你需要实现类似一个 `border-bottom` 的效果时，它的值为 `left bottom` 。按同样的逻辑，我们可以模拟出任何边框的虚线效果。注意，调整`background-repeat` 和 `background-size` 属性的值

  


上面这个示例，它是由 `--dash` 和 `--gap` 来决定虚线最小单元尺寸的。实际上我们也可以反过来，例如：

  


```CSS
.dashed--line {
    --dash: 40px; /* 虚线单段长度 */
    --size: 80px;  /* 虚线最小单元尺寸 */
    --gap: calc(var(--size) - var(--dash)); /* 虚线最小单元尺寸 */
    --b: 10px; /* 虚线厚度 */
    --bg: #09face; /* 虚线颜色 */
    background: linear-gradient(to right, var(--bg) 0, var(--bg) var(--dash),#0000 0) left center / var(--size) var(--b) repeat-x; 
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/c4b6d807d0514fa7bb57bbc535bcf8be~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1566&h=472&s=4669804&e=gif&f=459&b=2e575b)

  


> Demo 地址：https://codepen.io/airen/full/WNqMOEq

  


注意，使用这种方式的时候，当 `--gap` 计算出来的值小于或等于 `0` 时，绘制出来的是一条实线，而不是虚线。如果你希望始终绘制一条虚线，则第一种方式更为妥当！

  


如果你可能习惯了[使用百分比来设置渐变颜色在渐变轴上的停止点](https://juejin.cn/book/7199571709102391328/section/7199845781149810727#heading-8)，那么你也可以像下面这样来定义你的渐变：

  


```CSS
.dashed--line {
    --dash: 40%; /* 虚线单段长度 */
    --gap: calc((100% - var(--dash)) * var(--size));  /* 虚线段之间的间距 */
    --size: 80px; /* 虚线最小单元尺寸 */
    --b: 10px; /* 虚线厚度 */
    --bg: #09face; /* 虚线颜色 */
    background: linear-gradient(to right, var(--bg) 0, var(--bg) var(--dash),#0000 0) left center / var(--size) var(--b) repeat-x;
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/6804ff132b484ba29940160c284bcbf5~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1498&h=482&s=2783981&e=gif&f=278&b=2e575b)

  


> Demo 地址：https://codepen.io/airen/full/WNqMERx

  


上面示例所展示的只是模拟 `stroke-dasharray` 最简单的效果，你可以按照类似的方式来模拟更复杂的虚线效果。只不过，需要使用多个 `--dash` 和 `--gap` 的值。与此同时，`linear-gradient` 的渐变颜色停止点也需要做相应的调整。比如，接下来使用 `linear-gradient` 实现一个类似 `stroke-dasharray: 40px 20px 80px 100px` 的虚线效果：

  


```CSS
.dashed--line {
    --dash-1: 40px; /* 第一段虚线长度 */
    --gap-1: 20px; /* 第一个间距 */
    --dash-2: 80px; /* 第二段虚线长度 */
    --gap-2: 100px; /* 第二个间距 */
    --size: calc(var(--dash-1) + var(--gap-1) + var(--dash-2) + var(--gap-2)); /* 虚线最小单元尺寸 */
    --b: 10px; /* 虚线厚度 */
    --bg: #09face; /* 虚线颜色 */
    
    background: linear-gradient(
        to right,
        var(--bg) 0, var(--bg) var(--dash-1), 
        #0000 var(--dash-1), #0000 calc(var(--dash-1) + var(--gap-1)),
        var(--bg) calc(var(--dash-1) + var(--gap-1)), var(--bg) calc(var(--dash-1) + var(--gap-1) + var(--dash-2)),
        #0000 0
    ) left center / var(--size) var(--b) repeat-x;
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/c9fd47b102d1452fadcb3015164e29a4~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1468&h=508&s=3778406&e=gif&f=421&b=2e575b)

  


> Demo 地址：https://codepen.io/airen/full/OJeQjjy

  


依此类推，你可以创建出更复杂的虚线效果。

  


到目前为止，我们仅仅展示了 `linear-gradient` 模拟 `stroke-dasharray` 的效果，如果你希望定制的虚线具备与 `stroke-dashoffset` 类似的功能呢？为了模拟出类似的效果，我们需要新增一个 `--offset` 的变量:

```CSS
.dashed--line {

}
```

```CSS
    --gv: /var(--b) var(--size) linear-gradient(to bottom,var(--bg) 0, var(--bg) var(--dash),#0000 0) repeat-y;

        0 0 var(--gh),    /* border-top */
        100% 0 var(--gv), /* border-right */
        0 100% var(--gh), /* border-bottom */
        0 0 var(--gv);    /* border-left */
}
```

## 使用 `repeating-linear-gradient` 绘制虚线

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/cad2488956a144f4b8c49ca9c88972e6~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1200&h=675&s=97342&e=jpg&b=c8edfd)

## 创建倾斜虚线

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/8006083c9ee7491788aff94eef883271~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1470&h=980&s=61001&e=jpg&b=ffffff)

  


有的时候，我们可能需要定制类似上图这样的带有倾斜角度的虚线效果（Slanted Dashed）。对于这种类型虚线，我们同样可以使用 `repeating-linear-gradient` 来实现。

  


稍微了解 CSS 渐变的同学就知道，可以通过角度来控制渐变的方向。例如，我们之前的示例中的 `to right` 相当于 `90deg` 。换句话说，当我们调整渐变线的角度值时，就可以很容易获得带有倾斜角度的虚线。

  


我们可以在之前的示例基础上新增一个 `--a` 变量，用来替代渐变中的 `to right` 之类的关键词，例如：

  


```CSS
.dashed--line {
    --dash: 40px; /* 虚线单段长度 */
    --size: 80px; /* 虚线最小单元尺寸 */
    --b: 10px; /* 虚线厚度 */
    --bg: #09face; /* 虚线颜色 */
    --a: -45deg;
    
    background: repeating-linear-gradient(calc(var(--a) + 90deg), var(--bg) 0 var(--dash), #0000 var(--dash)  var(--size)) 0 0 /  100% var(--b) repeat-x;
}
```

## 创建圆形虚线边框

```CSS
.danshed--line {
```

## 创建带圆角的虚线

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/cb53a4005a724bfdad67596119196d25~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=994&s=81631&e=jpg&b=f1f1f1)

  


虚线除了之前展示的直角、斜角虚线之外，有的时候还需要给元素设置带有圆角的虚线边框。你可能已经猜到怎么绘制了。每段虚线的圆角效果，我们可以使用两个径向渐变来绘制，中间部分使用一个线性渐变。最终的 CSS 代码大致如下：

  


```CSS
.dashed--line {
    --s: 50px; /* 虚线长度 */
    --b: 10px; /* 控制虚线厚度*/
    --r: calc(var(--b) / 2); /* 虚线圆角半径 */
    --bg: #09face;

    background: bottom/calc(2 * var(--s)) var(--b) space no-repeat;
    --_g: calc(var(--s) / 2) top 50%, var(--bg) 90%, #0000 105%;
    background-image: 
        radial-gradient(var(--r) at left var(--_g)),  /* 左边圆点 */
        radial-gradient(var(--r) at right var(--_g)), /* 右边圆点 */
        linear-gradient(90deg, #0000 calc(var(--s) / 2),var(--bg) 0 calc(3 * var(--s) / 2),#0000 0
      );
}
```

## 创建渐变虚线

  


不难发现，之前所创建的各种类型虚线，基本上都是使用 CSS 渐变绘制的，并用于 `background` 属性，允许于元素的背景图像。有意思的是，我们可以直接使用 `mask` 来替代 `background` ，与此同时，你只需给元素设置渐变，你就绘制了一条渐变虚线：

  


```CSS
.dashed--line {
    --s: 50px; /* 虚线长度 */
    --b: 10px; /* 控制虚线厚度*/
    --r: calc(var(--b) / 2); /* 虚线圆角半径 */
    --bg: #09face;

    background: linear-gradient(to right,var(--bg), color-mix(in oklab, var(--bg) 20%, #000 35%), hsl(from var(--bg) 150 s 50%));
    mask: bottom/calc(2 * var(--s)) var(--b) space no-repeat;
    --_g: calc(var(--s) / 2) top 50%, var(--bg) 90%, #0000 105%;
    mask-image: 
        radial-gradient(var(--r) at left var(--_g)),
        radial-gradient(var(--r) at right var(--_g)),
        linear-gradient(90deg,#0000 calc(var(--s) / 2),var(--bg) 0 calc(3 * var(--s) / 2),#0000 0);
}
```

```CSS
.dashed--line {

```

  


很简单吧！你只需要掌握了之前的内容，实现这样的效果将不会有任何问题。

我们将要实现的虚线有两种，一种是水平虚线，另一种是垂直虚线。我们使用最简单的 `linear-gradient` 来创建它们，并且使用 CSS 自定义属性来存储它们：

  


```CSS
:root {
    --dashH: linear-gradient(90deg, currentColor 25%, #0000 25% 75%, currentColor 75%) 0 100% / 8px 1px repeat-x;
    --dashV: linear-gradient(180deg, currentColor 25%, #0000 25% 75%, currentColor 75%) 0 0 / 1px 8px repeat-y;
}
```

  


注意，代码中的 `currentColor` 关键词，它使得我们虚线的颜色可以继续元素的 `color` 值，这样保证了虚线的颜色与文本颜色保持一致！

## 小结

  


在这节课中，我们主要探讨了如何使用 CSS 的渐变和遮罩来制作各种不同的虚线。简单的总结一下：

  


-   `linear-gradient` 和 `repeating-linear-gradient` 比较适合定制水平或垂直方向的虚线，它可以模拟出 SVG 的 `stroke-dasharray` 和 `stroke-dashoffset` 绘制的虚线
-   调整渐变线的角度，可以制作出倾斜虚线
-   `conic-gradient` 和 `repeating-conic-gradient` 比较适合定制圆形虚线
-   引入 `radial-gradient` 可以定制带有圆角的虚线效果
-   结合 `mask` ，可以实现渐变虚线效果

  


虽然这些技巧可以帮助你定制一些酷炫的虚线效果，但相比于 SVG 制作的虚线，还是略显鸡肋。即使如此，这些技巧能实现大部分虚线效果。大家在使用的时候，应该根据实际出发，选择更为适合的方案。
