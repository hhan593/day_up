## 暴力破解方式

我们先从一个简单的计时器开始——每秒更新的倒计时器，并且只使用 CSS 来实现。它的底层结构包含了从 `0` 到 `9` 的所有数字。然后使用 CSS 动画把这些数字轮换起来，模仿倒计时器的效果。它的 HTML 结构看起来像下面这样：

  


```HTML
<div class="countdown--container">
    <ul class="countdown">
        <li>10</li>
        <li>9</li>
        <li>8</li>
        <li>7</li>
        <li>6</li>
        <li>5</li>
        <li>4</li>
        <li>3</li>
        <li>2</li>
        <li>1</li>
    </ul>
</div>
```

  


把所有数字都写在标记里，比如 `.countdown--container` 容器中包含一个无序列表（`ul`），其中的列表项（`li`）代表倒计时从 `10` 到 `1` 的数字，这些数字会依次显示，随着计时器的倒计时更新。

  


接下来，我们使用 CSS 给它添加一些基本样式：

  


```CSS
.countdown--container {
    display: grid;
    place-content: center;
}

.countdown {

        .375em .375em .75em var(--c-shadow);

    color: white;
    font-size: clamp(3.125rem, 3.464vw + 2.229rem, 5rem);
    font-weight: 900;
    font-family: "Montserrat", "Inter", "Roboto", "Open Sans", "Helvetica Neue", sans-serif;

    font-feature-settings: "tnum";

    li {
        grid-area: 1 / 1 / -1 / -1;
        text-align: center;
    }
}
```

  


在这里，[使用 CSS Grid 布局，将所有数字叠加在一起](https://juejin.cn/book/7395119173656903715/section/7438887593464627235)：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/18ffee99766c426395fd03ad62ed9534~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2286&h=1238&s=154250&e=png&b=121122)

  


接下来，需要给列表项添加动画效果，使整个效果看起来像一个“真实”的倒计时定时器：

  


```CSS
@keyframes countdownAnimation {
  0%,
  10% {
      opacity: 1;
  }
  11%,
  100% {
      opacity: 0;
  }
}
.countdown {
    --steps: 10;
    --duration: calc(var(--steps) * 1s);
    --delay: calc((var(--i) - 1) * 1s);
    
    /* CSS... */
    
    li {
        /* CSS ... */
        opacity: 0;
        animation: countdownAnimation var(--duration) steps(var(--steps)) var(--delay) forwards;
    }
}
```

  


我们首先将所有数字（列表项）的 `opacity` 设置为 `0` ，把它们隐藏起来不可见，然后使用 `@keyframes` 和 `animation` 属性来创建倒计时效果。通过 `animation-delay` 属性，为每个列表项设置不同的延迟，确保它们依次显示。在这里，我们通过 CSS 的变量来计算动画的持续时间（`animation-duration`）、延迟时间（`animation-delay`）以及动画缓动函数 `steps()` 的步进值：

  


```CSS
li {
    --steps: 10; /* 步长，li 的总数 */
    --duration: calc(var(--steps) * 1s); /* 动画持续时间 */
    --delay: calc((var(--i) - 1) * 1s);  /* 动画延迟时间 */
}
```

  


代码中的 `--i` 是以内联的方式在 `li` 元素上定义的，索引值从 `1` 开始：

  


```HTML
<div class="countdown--container">
    <ul class="countdown">
        <li style="--i: 1;">10</li>
        <li style="--i: 2;">9</li>
        <li style="--i: 3;">8</li>
        <li style="--i: 4;">7</li>
        <li style="--i: 5;">6</li>
        <li style="--i: 6;">5</li>
        <li style="--i: 7;">4</li>
        <li style="--i: 8;">3</li>
        <li style="--i: 9;">2</li>
        <li style="--i: 10;">1</li>
    </ul>
</div>
```

  


另外，帧动画 `countdownAnimation` 将控制每个列表项的可见性，让它们显示一小段时间后再隐藏。与此同时，动画缓动函数 `steps()` 会让动画跳着变而不是平滑过度，它能保证每一步持续时间相同。你可以将 `steps()` 函数视为在指定间隔内使用线性缓动（`linear`）拍摄动画的快照，并在显示下一个快照之间显示该快照。

  


最后使用 `animation-fill-mode` 来控制动画前后样式是否保留，这里用的是 `forwards`，意思是动画结束后，保持最后一帧的样式。

  


> 温馨提示，CSS 动画中的 `steps()` 函数和 `animation-fill-mode` 涵盖了许多与动画相关的细节，如果你想更深入的了解这方面知识的话，请移步阅读《[使用 `steps()` 函数创建分段动画](https://juejin.cn/book/7288940354408022074/section/7301594361152667698)》和《[深度解析 CSS 动画的填充模式](https://juejin.cn/book/7288940354408022074/section/7307284837554585638)》！

```CSS
.countdown--container {
```

这个思路的核心是：**通过垂直滚动这些数字块来实现数字从** **`9`** **到** **`0`** **的动画，并且在任意时间点只显示其中一个数字**。这样看起来就像数字在倒计时一样。

依旧使用 CSS Grid 创建两列网格，把秒位（`.digit--seconds`）和毫秒位（`.digit--milliseconds`）排成一行，在视觉上呈现出两位数的形式。另外，在这里我们将容器 `.countdown` 的高度设置为 `1lh` ，并且将列宽设置为 `1ch` ，这样偏于后续数字向上位移。注意 `lh` 和 `ch` 都是 [CSS 的字体相对单位](https://juejin.cn/book/7223230325122400288/section/7249357892611440700)，其中 `lh` 相对于行高（`line-height`）计算，`ch` 相对于阿位伯数字 `0` 宽度计算。

为了使数字往上移（上图中橙色和黄色框起来的数字），我们可以使用 `translate` 属性或 `transform` 的 `translateY()` 。因为在这个示例中，数字只需要在 `Y` 轴上移动，比如：

  


```CSS
.milliseconds {
    translate: 0 -1lh;
    /* 或者 */
    transform: translateY(-1lh); 
    
    /* 或者 */
    transform: translate(0, -1lh);
}
```

  


注意，使用 `translate` 属性或 `translate()` 函数时，必须同时指定 `X` 轴的值，因为它们单独写一个值会默认应用到 `X` 轴上：

  


```CSS
.milliseconds {
    translate: -1lh;
    /* 或者 */
    transform: translate(-1lh); 
    
    /* 等价于 */
    translate: -1lh 0;
    /* 或者 */
    transform: translate(-1lh, 0);
}
```

```CSS
}
```

  


注意，我们不能将动画应用在每个数字上，而是要将它们应用于数字的容器上。在这里单独解释一下，我们将 `animtion-timing-function` 设置为 `step-end` ，它等价于 `steps(1, jump-end)` ，其中 `steps()` 函数会让动画跳着变而不是平滑过渡，每一步持续相同时间，`jump-end` 会使元素一步到位，然后停在最终状态。这样数字不会平滑滚动，而是一帧一帧切换。

  


需要知道的是，缓动函数对于动画的效果非常重要，如果你想更深入的了解和掌握这方面的知识，那么下面这几节课值得你花点时间阅读：

-   [使用 `cubic-bezier()` 函数创建高级动画](https://juejin.cn/book/7288940354408022074/section/7298995488580337714)
-   [使用 `steps()` 函数创建分段动画](https://juejin.cn/book/7288940354408022074/section/7301594361152667698)
-   [使用 `linear()` 函数创建令人惊叹的动画曲线](https://juejin.cn/book/7288940354408022074/section/7301665456824254515)

## 现代方式：`counter()` + `@property`

  


上一个示例中，我们通过动画化伪元素的 `content` 属性的值，实现了一个 CSS 倒计时。但它有一个明显的缺陷，如果我们要从一个较大的值开始倒计时，`@keyframes` 中的定义会变得非常的复杂和冗长。

  


庆幸的是，我们可以通过 [CSS 的计数器](https://juejin.cn/book/7223230325122400288/section/7259668400379527205)（`counter-reset`或`counter-increment` 和 `counter()`）、[自定义属性 `@propert`](https://juejin.cn/book/7223230325122400288/section/7258870477462962236) 和 `calc()` 对它进行优化。

  


首先，我们使用 `@property` 来创建一个自定义属性（比如 `--n`），它将保存一个 `<integer>` 类型的值：

  


```CSS
@property --n {
    syntax: "<integer>";
    inherits: true;
    initial-value: 0;
}
```

  


同时使用 `@keyframes` 定义一个动画，它可以在递减数字的同时跟踪秒数：

  


```CSS
@keyframes countdown {
    from {
        --n: 15;
    }
    to {
        --n: 0;
    }
}
```

  


接着通过 `animation` 属性来激活：

  


```CSS
.countdown::after {
    animation: 15s linear countdown forwards;
}
```

  


当我们为一个特定的值类型（例如 `<integer>` 、`<percentage>` 或 `<color>`）注册一个自定义属性时，浏览器会知道该属性是为该特定类型的值而创建的。有了这个信息，浏览器可以在未来更新自定义属性的值，甚至在动画过程中更新自定义属性的值。

  


这就是为什么我们的 `--n` 属性可以在 `@keyframes` 中从 `15` 递减到 `0` ，而且动画的持续时间设置为 `15s` ，这基本上是在 `15s` 内从 `15` 倒计时到 `0` 。因此，计时器就诞生了。

  


但要在页面上呈现这个倒时计效果，还差一步。我们需要将数字传递到伪元素的 `content` 中。然而，`content` 并不能直接接受数字，因此，我们需要通过 `counter()` 函数将数字 `--n` 传递给它。

  


```CSS
.countdown::after {
    content: counter(var(--n));
    animation: 15s linear countdown forwards;
}
```

  


在 CSS 中，计数器可以通过 `counter-reset` 或 `counter-increment` 来设置。我在这里使用 `counter-reset` 。由于`counter-reset` 目前不能正确处理 CSS 变量或自定义属性中的整数值，但可以接受 `calc()` 函数，因此 ` calc()` 函数将作为桥梁，将 CSS 变量 `--n` 传递给 `counter-reset`：

  


```CSS
.countdown::after {
    counter-reset: n calc(0 + var(--n));
    content: counter(n);
    animation: 15s linear countdown forwards;
}
```

  


上面的代码将可动画化的数字 `--n` 先传给 `calc()` ，`calc()` 函数的结果再传递给 `counter()` 函数，`counter()` 函数的结果赋值给伪元素的 `content` 属性，并在页面上渲染出 `--n` 。

  


剩下的事情浏览器会自动处理。它知道 `--n` 是一个整数，并且它会在动画过程中将该整数从 `15` 递减到 `0` 。由于整数用作 `content` 的值，浏览器会在更新时将整数显示在页面上。动画结束时，`animation-fill-mode: forwards` 会防止计时器立即恢复到初始的 `--n` 值（`0`）。最终的效果如下

## 案例：创建时钟计时器

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/0882d814e5b5460bad8720287562ae14~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1266&h=642&s=163033&e=gif&f=66&b=205976)

  


时钟是计时器的另一种表现形式，我曾在《[使用 `steps()` 函数创建分段动画](https://juejin.cn/book/7288940354408022074/section/7301594361152667698)》课程中向大家展示了[ CSS 如何创建一个时钟](https://juejin.cn/book/7288940354408022074/section/7301594361152667698#heading-10)。今天，在这里我将向大家展示一种更现代的方式——结合 CSS 的锥形渐变、遮罩、路径动画、三角函数和自定义属性创建一个时钟。

  


我们需要一个像下面这样的 HTML 结构：

  


```HTML
<div class="clock">
    <div class="clock__panel">
        <div class="clock__number">1</div>
        <div class="clock__number">2</div>
        <div class="clock__number">3</div>
        <div class="clock__number">4</div>
        <div class="clock__number">5</div>
        <div class="clock__number">6</div>
        <div class="clock__number">7</div>
        <div class="clock__number">8</div>
        <div class="clock__number">9</div>
        <div class="clock__number">10</div>
        <div class="clock__number">11</div>
        <div class="clock__number">12</div>
    </div>
    <div class="clock__hour clock__hand"></div>
    <div class="clock__minute clock__hand"></div>
    <div class="clock__seconds clock__hand"></div>
</div>
```

  


首先我们使用锥形渐变（`repeating-conic-gradient()`）和遮罩（`mask`）来创建钟表盘的刻度：

  


```CSS
.clock {
    --w: 50vmin; /* 表盘大小 */
    --b: 1rem;   /* 刻度长度 */
    display: grid;
    place-content: center;
    padding: 3rem;
    
    &::after {
        content: "";
        grid-area: 1 / 1 / -1 / -1;
        width: var(--w);
        aspect-ratio: 1;
        border-radius: 50%;
        background: 
            repeating-conic-gradient(from -.5deg,#333 0 1deg, #0000 0deg 30deg),           
            repeating-conic-gradient(from -.5deg,#999 0 1deg, #0000 0deg 6deg);
        mask: radial-gradient(#0000 calc((var(--w) - var(--b)) / 2), #000 0);
    }
}
```

  


简单解释一下上面的代码。

  


我们使用两个重复锥形渐变来绘制表盘中的小时刻度和分刻度，在一个表盘中，小时刻度线有 `12` 条，每相邻两条线的间距是 `30deg` （即 `360deg / 12 = 30deg`），且每条线的厚度是 `1deg` 。为了使它能够居中，锥形渐变的开始角度为 `-0.5deg` ：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/10829aa3be03413681d96f6672184ccf~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2286&h=1366&s=101805&e=png&b=ffffff)

  


分钟的刻度也是同样的道理，因为共有 `60` 个刻度，所以最小角度是 `6deg`（即`360deg / 60 = 6deg`），我们使用另一个重复锥形渐变来绘制：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/0a0c4e6421da4094b5cf43565659821d~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2286&h=1366&s=178939&e=png&b=ffffff)

  


为了便于大家区分小时和分钟的刻度，我特意暂时将分钟的宽度设置为 `red` 。通常情况下，我们会将分钟的刻度线设置的比小时刻度线更淡一点。

  


然后，我们使用一个径向渐变绘制的圆作为遮罩图，应用于 `mask` ，从圆的中心进行镂空，径向渐变绘制的圆的大小为 `calc((var(--w) - var(--b)) / 2)` ，其中 `--b` 变量设置的是刻度线的长度：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/bbb6ed36300649b7a037df829d4ce9a2~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2286&h=1366&s=92105&e=png&b=ffffff)

  


这样我们就绘制出了钟表的小时和分钟刻度线。但是不太完美，它们的长度都一样，如果我们能将小时的刻度稍微留长一点点，就更完美一些。这样一来，我们需要新增另一个伪元素，其中一个用于绘制小时刻度线，另一个绘制分钟刻度线：

  


```CSS
.clock {
    --w: 50vmin;
    --b_h: 1.7rem;
    --b_m: .85rem;
    
    display: grid;
    place-content: center;
    padding: 3rem;
    
    &::before,
    &::after {
        content: "";
        grid-area: 1 / 1 / -1 / -1;
        width: var(--w);
        aspect-ratio: 1;
        border-radius: 50%;
    }
    
    &::after {
        background: repeating-conic-gradient(from -.5deg,#333 0 1deg, #0000 0deg 30deg);       
        mask: radial-gradient(#0000 calc((var(--w) - var(--b_h)) / 2), #000 0);
    }
    
    &::before {
        background: repeating-conic-gradient(from -.5deg,red 0 1deg, transparent 0deg 6deg);
        mask: radial-gradient(#0000 calc((var(--w) - var(--b_m)) / 2), #000 0);
    }
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/c7b4e735a25548a8a4dc2d152b316a78~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2286&h=1366&s=92069&e=png&b=ffffff)

  


为了能够更好的控制刻度线的颜色，我们引入变量 `--c` ，并将其值设置为 `currentColor` （默认为 `color` 的值），然后通过 `color-mix()` 函数基于 `--c` 来调整分钟刻度线的颜色：

  


```CSS
.clock {
    --c: currentColor;
    /* CSS... */
    &::after {
        background:  repeating-conic-gradient(from -.5deg,var(--c) 0 1deg, #0000 0deg 30deg);       
        mask: radial-gradient(#0000 calc((var(--w) - var(--b_h)) / 2), #000 0);
    }
    
    &::before {
        background: repeating-conic-gradient(from -.5deg,color-mix(in oklch, var(--c), #fff 50%) 0 1deg, transparent 0deg 6deg);
        mask: radial-gradient(#0000 calc((var(--w) - var(--b_m)) / 2), #000 0);
    }
}
```

  


分钟的刻度线颜色是 `50%` 的 `--c` 与 `50%` 的 `#fff` 混合而成，始终会要比小时刻度线颜色更淡：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/3607b4c03d4f4c02a11a6ac9869e6ea2~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1136&h=672&s=261488&e=gif&f=146&b=ffffff)

  


绘制完刻度线之后，我们需要将小时的数字放到对应的位置。我们可以结合 [CSS 的三角函数 `sin()` 和 `cos()`](https://juejin.cn/book/7223230325122400288/section/7242216512176521277) 以及[最新的 `sibling-count()` 和 `sibling-index()` 函数](https://juejin.cn/book/7395119173656903715/section/7487008847370928167)为这些数字精确定位。我们将引入新的变量：

  


-   `--a` 用于 `sin()` 和 `cos()` 函数的角度值，它的值等于 ` calc((360deg / sibling-count()) * (sibling-index() - 1))  `，其中 `sibling-count()` 能获得所有数字的总数
-   `--r` 相当于直角三角形的斜边长，通过 `sin()` 和 `cos()` 函数计算出每个数字的 `x` 和 `y` 坐标，与此同时将它们分别存储在 `--x` 和 `--y` 变量中

  


获得 `x` 和 `y` 坐标之后，就可以使用 `translate` 对其进行位移，最终实现精确定位：

  


```CSS
.clock__number {
      --count: 12; /* 等同于 sibling-count() */
      --a: calc((360deg / var(--count)) * (var(--i) - 1)); /* 等同于 calc((360deg / sibling-count()) * (sibling-index() - 1))*/
      --r: calc((var(--w) - 4ch) / 2);
      --x: calc(var(--r) * cos(var(--a)));
      --y: calc(var(--r) * sin(var(--a)));
      translate: var(--x) var(--y);
      
      width: 4ch;
      text-align: center;
}
```

> 温馨提示，最近写课程中，发现使用 `sibling-count()` 和 `sibling-index()` 函数很容易使 Chrome 崩溃！因此在这个示例的代码中，我采用了 CSS 自定义属性替代它们。

现在，我们已经把表盘绘制好了。就差最后一步，给时针、分针和秒针添加动画效果。这个动画效果非常简单，就是绕着圆心旋转 `360deg` 。同样地，我们使用 `@property` 定义一个变量 `--angle` ，它的值类型是 `<angle>` ，初始值为 `0` ：

  


```CSS
@property --angle {
    syntax: "<angle>";
    inherits: true;
    initial-value: 0deg;
}
```

  


然后使用 `@keyframes` 定义一个旋转动画：

  


```CSS
@keyframes rotate {
    to {
        --angle 360deg;
    }
}
```

  


给所有针设置 `rotate: var(--angle)` ：

  


```CSS
.clock__hand {
    rotate: var(--angle);
}
```

  


不同的是，时针、分针、秒针的周期不一样，时针转一圈是 `12` 小时、分针是 `60` 分钟、秒针是 `60` 秒，各自需要换算成秒数(CSS 单位只支持**秒**和**毫秒**)，为了方便测试，这里将速度调快了 `60s → 6s` ：

  


```CSS
.clock__hour {
    animation: rotate calc(var(--step) * 60 * 12) infinite;
}
  
.clock__minute {
    animation: rotate calc(var(--step) * 60) infinite;
}
  
.clock__seconds {
    animation: rotate var(--step) infinite;
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/d36f6f3382784f9e9b062eef9e3ad462~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1132&h=674&s=683178&e=gif&f=180&b=ffffff)

  


这效果看上去怪怪的，对不对。我们需要调整一下 `animation-timing-function` :

  


```CSS
.clock__hour {
    animation: rotate calc(var(--step) * 60 * 12) linear infinite;
}
  
.clock__minute {
    animation: rotate calc(var(--step) * 60) steps(60, jump-end)  infinite;
}
  
.clock__seconds {
    animation: rotate var(--step) steps(60 ,jump-start) infinite;
}
```

  


使用 `steps()` 将允许我们创建模仿真实时钟上指针运动的动画。在使用 `steps()` 时会涉及一些简单的数学计算。比如：

  


-   秒针旋转一圈（`360deg`）刚好用时 `60s` ，这意味着秒针在 `60` 步内旋转 `360deg` ，并在 `60s` 秒内完成动画。即 `steps()` 函数的步数是 `60` ，动画持续时间（`animation-duration`）是 `60s`
-   一分钟等于 `60` 秒，而一秒需要 `60s` 才转完一圈，这意味着分针在 `60` 步内旋转 `360deg` ，需要用时 `3600s` 。即 `steps()` 函数的步数同样是 `60` ，只是动画持续时间要调到 `3600s`
-   时针和分针与秒针有所不同，我们希望时针恒速旋转一圈，只是根据时间计算出其持续时间，即 `43200s` （对应 `12` 小时）

  


我们在定义秒针和分针的动画缓动函数时，可能会考虑使用 `steps(60)` ，它相当于 `steps(60, jump-end)` 或 `steps(60, end)` 。这使得时钟的秒针都会随时钟上的每个标记步进。如果不跳过结束状态，我们将得到一个动画，其中开始和结束将位于顶部（`0deg`），因此我们的时钟在顶部分停留两秒，并在之间进行其余的 `58s` ，这样就不自然了。也就是说，当你考虑时钟上的秒针时，给 `steps()` 函数指定一个 `jump-start` 参数，可以使稍秒跳过其结束状态，使得秒针的动画更自然。

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/e3af3a009dde4d2887a932d7b1daef3d~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1134&h=670&s=171505&e=gif&f=93)

  


为了使这个时钟与真实时间结合起来，我们可以使用 JavaScript 初始化时间。需要注意的是，在获取的时候加上偏移量，比如 `12:30`，分针其实是 `12.5` ，以此类推。具体的代码很简单：

  


```JavaScript
const d = new Date()
const h = d.getHours();
const m = d.getMinutes();
const s = d.getSeconds();
const clock = document.querySelector('.clock');
clock.style.setProperty('--ds', s);
clock.style.setProperty('--dm', m + s/60);
clock.style.setProperty('--dh', h + m/60 + s/3600);
```

  


最后，通过 `animation-delay`来指定动画的起始位置：

  


```CSS
.clock__hour {
    animation: rotate calc(var(--step) * 60 * 12) linear infinite;
    animation-delay: calc( -1 * var(--step) * var(--dh) * 60);
}
  
.clock__minute {
    animation: rotate calc(var(--step) * 60) steps(60, jump-end)  infinite;
    animation-delay: calc( -1 * var(--step) * var(--dm));
}
  
.clock__seconds {
    animation: rotate var(--step) steps(60 ,jump-start) infinite;
    animation-delay: calc( -1 * var(--step) * var(--ds) / 60 );
}
```

## 小结

  


纯 CSS 计时器轻量且易于理解，非常适合简单的倒计时场景，且脚本要求最少。然而，它可能难以处理更复杂的动画和交互功能。另一方面，JavaScript计时器提供了更大的控制和灵活性，允许实现更动态的交互，但代价是更高的脚本开销和更高的复杂性。

  


最终，选择哪种方法取决于你项目的具体需求和你愿意接受的权衡。
