## 准备工作

  


假设，你需要创建的标志区域有下图所示这些 Logo 图像：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/7e2e30f5008a4b31bb91c5bf6d498d48~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1052&s=287709&e=jpg&b=607d8b)

  


不难发现，上图中展示的 Logo 图像，大小不一，形状不一。我们的目标是要使这些 Logo 图像在 Web 上有一个完美的视觉展示。

  


首先，我们按照下面的方式来创建 HTML 结构：

  


```HTML
<ul class="brands">
    <li class="brands__item">
        <a href="#">
            <img class="logo" src="https://s3-us-west-2.amazonaws.com/s.cdpn.io/827904/151792556147.jpg" alt="" />
```

首先，我们[使用 CSS Grid 来创建基础布局](https://juejin.cn/book/7161370789680250917/section/7161372229123440648)：

  


```CSS
:root {
    --w: 1024px;
}

.brands {
    margin-inline: max(1rem,50% - var(--w, 100%)/2); 
    display: grid;
    gap: 1rem;
    grid-template-columns: repeat(auto-fit, minmax(200px, 1fr));
    
    .brands__item {
        background: #eee;
    }
    
    img {
        display: block;
        max-width: 100%;
    }
}
```

正如你所见，每个 Logo 图像都会占据其父容器的完整宽度，但由于 Logo 图像尺寸大小不一致，导致最终效果并不如人意。

  


## 常规解决方案：统一宽高

  


常规的解决方案，Web 开发者会给 Logo 图像设置统一的宽度或（和）高度。先来看统一的宽度，比如给 Logo 图像设置宽度为 `100px` ：

  


```CSS
.brands__item {
    img {
        display: block;
        max-width: 100%;
        width: 100px;
    }
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/4d1250139ff0472eaccb4ce918ad494e~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=676&s=291297&e=jpg&b=f2f2f2)

  


很明显，Logo 图像宽度是保持一致了，但高度并一致。你可能也尝试着将所有 Logo 图像统一高度：

  


```CSS
.brands__item {
    img {
        display: block;
        max-width: 100%;
        height: 100px;
    }
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/897e7d0d9e5943dba9efbdb46f1081a9~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=886&s=453575&e=jpg&b=f1f1f1)

  


是的，现在 Logo 图像的高度是统一了，但新的问题也随之出现了，宽度较大的 Logo 图像变形了。比如“Nodejs”的 Logo 被拉伸了，因为它的高度相对于其他 Logo 图像来说较小，而我们提高了 `height` 值，对它来说太大了，导致了拉伸问题。

  


你可能想到了使用 `object-fit` 来解决这个拉伸问题：

  


```CSS
.brands__item {
    img {
        display: block;
        max-width: 100%;
        height: 100px;
        object-fit: contain;
    }
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/cc201f8b95e746b7a2b748d94ef4d672~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=886&s=429219&e=jpg&b=f1f1f1)

  


虽然高度统一了，但整体效果并不理想。这表明仅设置宽度或高度并不能很好地解决问题。接下来，可以尝试的方法是同时设置宽度和高度，但这也意味着 Logo 图像会被拉伸或压缩。为了避免这个问题出现，我们需要设置 `object-fit` 属性的值为 `contain` ：

  


```CSS
.brands__item {
    img {
        display: block;
        width: 150px;
        height: 100px;
        object-fit: contain;
    }
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/7769125a37f5497ca1256907d362d3ef~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=931&s=361613&e=jpg&b=f0f0f0)

  


`object-fit: contain` 的优点在于，无论宽度或高度如何，Logo 图像都会被限制在它们的范围内，同时保持其固有宽高比，不会被扭曲。

  


现在，Logo 图像的展示整体看上去要美观很多。从上图的效果上来看，不能发现，由于灰色的背景存在，我们可以发现部分 Logo 图像可能是 JPG 图片，它带有一个白色背景。在一些特殊场景中，可能无法获得透明 Logo 图像，又希望最终的结果中去除 Logo 图像的白色背景。那么，[我们可以使用 CSS 的混合模式 `mix-blend-mode` 来解决这个问题](https://juejin.cn/book/7223230325122400288/section/7259669043622690853#heading-23)：

  


```CSS
.brands__item {
    img {
        display: block;
        width: 150px;
        height: 100px;
        object-fit: contain;
        
        &:is([src$=".jpg"],[src$=".png"]) {
            mix-blend-mode: multiply;
        }
    }
}
```

  


上面的代码中，使用 CSS 属性选择器，仅对 `.jpg` 和 `.png` 格式的图片应用 `mix-blend-mode` 。这个时候，Logo 展示的效果如下图所示：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/e22f1bc691f341b3a0691245c8530924~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1184&h=522&s=3964837&e=gif&f=173&b=4f7b94)

  


> Demo 地址：https://codepen.io/airen/full/MYgJYag

  


这是一种传统的解决方案，从最终的结果你也发现了，我们调整 Logo 图像容器的背景色，它也能有一个完美的展示。

  


如果你的诉求不高，那么上面这个效果应该可以满足你的大部分需求。事实上，我们还可以做得更好。接下来，我们来探讨另一种解决方案——根据宽高比调整 Logo 图像！

  


## 根据宽高比调整 Logo 图像

  


为了打造更和谐美观的 Logo 展示，我们可以根据每个 Logo 的宽高比调整其高度。如果一个 Logo 宽度明显大于高度（宽高比大于 `1`），我们会相应地降低其高度。Logo 图像越宽，高度就越小。通过这种方式，可以让所有 Logo 图像在视觉上更加均衡。

  


要实现这这一点，我们需要知道每个 Logo 图像的原始宽度和高度。如果使用内容管理系统（CMS），通常可以轻松获取 Logo 图像的尺寸；如果是自己制作的 Logo 图像，那就更简单了。我们通过内联的方式，使用 `--w` 和 `--h` 两个 CSS 变量来存储 Logo 图像的原始宽度和高度。需要注意的是，这里一定要使用**无单位值**，因为在 CSS 的计算中，一旦添加了单位，就无法将其移除：

  


```HTML
<ul class="brands">
    <li class="brands__item" style="--w: 650; --h: 439;">
        <a href="#">
            <img class="logo" src="https://s3-us-west-2.amazonaws.com/s.cdpn.io/827904/151792556147.jpg" alt="" />
        </a>
    </li>
    <!-- 省略其他 Li -->
</ul>  
```

  


接下来就是 CSS 的魔力了：

  


```CSS
.brands {
    --base-height: 3rem; /* Logo 图像的基础高度 */
    --scale-factor-horizontal: .1; /* Logo 图像水平方向缩放系数 */
    
    .brands__item {
        background: #b8e20cbf;
        padding: 1rem 2rem;
        display: grid;
        place-content: center;

        a {
            --base-ratio: calc(var(--w) / var(--h)); /* Logo 图像的固有宽高比 */
            --factor-horizontal: min(
                var(--scale-factor-horizontal) * -1 * var(--base-ratio) + var(--scale-factor-horizontal) + 1,
                1
             );
             aspect-ratio: var(--base-ratio);
             height: max(
                var(--base-height) / 2,
                var(--base-height) * var(--factor-horizontal)
            );
        }
        
        img {
            display: block;
            width: 100%;
            height: auto; 
        }
    }
}
```

  


简单解释一下上面的代码。

  


我们在 `.brands` 上定义了两个变量：`--base-height` 和 `--scale-factor-horizontal`。这些变量将用于每个标志的高度计算：

  


```CSS
.brands {
    --base-height: 3rem; /* Logo 图像的基础高度 * /     --scale-factor-horizontal: 0.1; /* Logo 图像的横向缩放系数 * /
}
```

  


接着计算每个 Logo 图像的宽高比。我们通过 `--w` 和 `--h` 计算其宽高比，并将结果存储在 `--base-ratio` 变量中：

  


```CSS
.brands  {
    --base-height: 3rem; /* Logo 图像的基础高度 * /     --scale-factor-horizontal: 0.1; /* Logo 图像的横向缩放系数 * /
    
    a {
         --base-ratio: calc(var(--w) / var(--h)); /* 宽高比计算 */
         aspect-ratio: var(--base-ratio); /* 设置宽高比 */
    }    
}
```

  


然后就进入这个方案的核心部分，我们需要计算一个缩放因子，根据每个 Logo 图像的宽高比调整其高度。这可以确保较宽的 Logo 图像按比例缩小，从而在整个 Logo 图像排列中保持视觉上的和谐。我们使用线性插值来计算缩放因子，当宽高比为 `1` 时返回 `1` 。对应的公式如下：

  


```CSS
a {
    --factor-horizontal: min(
        (-1 * var(--scale-factor-horizontal) * var(--base-ratio)) + var(--scale-factor-horizontal) + 1,
        1
    );
}
```

  


我们可以插入一些示例数据，获得对应的缩放因子计算结果：

  


-   `--base-ratio` 为 `1` ，计算的结果将是 `1` ，表示宽高比为 `1` 的 Logo 图像保持原始高度
-   `--base-ratio` 为 `2` ，计算的结果将是 `0.9` ，表示宽高比为 `2` 的 Logo 图像的高度为宽高比为 `1` 的 Logo 图像高度的 `90%`
-   `--base-ratio` 为 `3` ，计算的结果将是 `0.8` ，表示宽高比为 `3` 的 Logo 图像的高度为宽度比为 `1` 的 Logo 图像高度的 `80%`
-   依此类推

  


通过这一公式，我们动态调整了宽高比较大的 Logo 图像，使其高度缩减，与其他 Logo 图像保持视觉平衡。例如：

  


-   一个宽高比为 `2` 的 Logo 图像，其高度会缩小到 `90%`，避免看起来过大
-   宽高比越大，缩小的比例越高，从而维持整个标志列的一致性和美观

  


最后，我们通过将基础高度（`--base-height`）乘以计算得出的缩放因子（`--factor-horizontal`）来设置每个 Logo 图像的高度。为了确保即使是最宽的 Logo 图像高度也不会变得过小，我们使用 `max()` 函数来限制其高度不低于基础高度的一半：

  


```CSS
.brands  {
    --base-height: 3rem; /* Logo 图像的基础高度 * /     --scale-factor-horizontal: 0.1; /* Logo 图像的横向缩放系数 * /
    
    a {
         --base-ratio: calc(var(--w) / var(--h)); /* 宽高比计算 */
         --factor-horizontal: min(
             var(--scale-factor-horizontal) * -1 * var(--base-ratio) + var(--scale-factor-horizontal) + 1,
             1
         );
         aspect-ratio: var(--base-ratio);
         height: max(
             var(--base-height) / 2,
             var(--base-height) * var(--factor-horizontal)
         );
    }    
}
```

  


注意，我们仍然将 `img` 宽高设置为 `100%` ，高度为 `auto` 。最终得到的效果如下图所示：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/0300691c677345cba33867097b681b7b~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=931&s=374294&e=jpg&b=607d8b)

  


> Demo 地址：https://codepen.io/airen/full/jENybMz

  


你可能已经注意到了，上面的示例我们只处理了 Logo 图像宽高比大于或等于 `1` 的情况。但有时，Logo 图像的宽高比会小于 `1` 。这意味着，竖向的 Logo 图像（宽小于高）的宽高比介于 `0 ~ 1` 之间，而横向的 Logo 图像（宽大于高）的宽高比介绍 `1` 和无穷大之间。如果按照相同的缩放因子对它们进行缩放，调整幅度不足以优化它们的大小。这意味着，我们之前针对横向 Logo 图像使用的插值方法在竖向 Logo 图像上效果不明显。

  


为了解决这个问题，我们引入一个专门针对竖向 Logo 图像的缩放因子 `--factor-vertical` 。并且在最终的高度计算中，我们不再使用 `max()` 函数，而且改用 `clamp()` 函数。`clamp()` 函数允许我们同时设置最小值和最大值，以确保竖向 Logo 图像不会过大，掩盖其他 Logo 图像的效果。最终的 CSS 代码如下：

  


```CSS
:root {
    --w: 1024px;
}

.brands {
    --base-height: 3rem;
    --scale-factor-horizontal: 0.1;
    --scale-factor-vertical: 0.6;
    --logo-min-size: 0.5;
    --logo-max-size: 1.25;

    margin-inline: max(1rem, 50% - var(--w, 100%) / 2);
    display: grid;
    gap: 1rem;
    grid-template-columns: repeat(auto-fit, minmax(200px, 1fr));

    .brands__item {
        background: #b8e20cbf;
        padding: 1rem 2rem;
        display: grid;
        place-content: center;

        a {
            --base-ratio: calc(var(--width) / var(--height));
            --factor-horizontal: min(
                var(--scale-factor-horizontal) * -1 * var(--base-ratio) + var(--scale-factor-horizontal) + 1,
                1
            );
            --factor-vertical: max(
                var(--scale-factor-vertical) * -1 * var(--base-ratio) + var(--scale-factor-vertical) + 1,
                1
            );

            aspect-ratio: var(--base-ratio);
            height: clamp(
                var(--base-height) * var(--logo-min-size),
                var(--base-height) * var(--factor-horizontal) * var(--factor-vertical),
                var(--base-height) * var(--logo-max-size)
            );
        }
    }

    img {
        display: block;
        width: 100%;
        height: auto;

        &:is([src$=".jpg"], [src$=".png"]) {
            mix-blend-mode: multiply;
        }
    }
}
```

## 根据相同面积调整 Logo 图像

  


这种方案的核心之处是将所有 Logo 图像按比例缩放，使其占据的面积与一个正方形（宽高比为 `1`）的 Logo 图像相同。换句话说，每个 Logo 图像的宽度和高度的积（即面积）在整个 Logo 列表中保持一致。我们可以按照下面三步来调整 Logo 图像的大小:

  


-   1️⃣**计算正方形的面积：** 它等于基础高度的（`--base-height`）的平方（即 `pow(var(--base-height), 2)`），宽乘以高，且宽等于高
-   2️⃣**找到缩放因子**：我们通过将正方形的面积除以每个 Logo 图像的实际面积（宽乘以高）来确定缩放因子。由于我们要同时缩放宽度和高度，因此需要对这个比值开平方根，其计算公式见下图
-   3️⃣缩放 Logo 图像：我们将每个 Logo 图像的高度乘以这个缩放因子，以调整 Logo 图像的大小，使其占据与正方形 Logo 相同的面积

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/4cf30431b4534ab8bfb3f433bed5f833~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1408&h=462&s=6891&e=avif&b=ffffff)

  


这种方案的优点在于，**无论是竖向（宽小于高）或横向（宽大于高）的 Logo 图像，都可以无缝应用，确保所有 Logo 的视觉呈现一致**。

  


需要注意的是，这个方案可以满足大部分的情景，但在一些极端的情况之下会导致 Logo 图像在视觉上不协调，例如某些 Logo 图像的宽高比比较极端。为了避免这个现象，我们可以引入一个“强度变量”，用于控制缩放效果的强弱程度。具体的 CSS 代码如下所示：

  


```CSS
:root {
    --w: 1024px;
}

@property --captured-length {
    syntax: "<length>";
    initial-value: 0px;
    inherits: false;
}

.brands {
    --strength: 1;
    --base-height: 3rem;
    --logo-min-size-factor: 0.375;
    --logo-max-size-factor: 1.25;

    margin-inline: max(1rem, 50% - var(--w, 100%) / 2);
    display: grid;
    gap: 1rem;
    grid-template-columns: repeat(auto-fit, minmax(200px, 1fr));

    .brands__item {
        background: #b8e20cbf;
        display: grid;
      
        a {
            display: grid;
            place-content: center;
            padding: 1rem 2rem;
        }
    }

    img {
        display: block;
        --captured-length: var(--base-height);
        --area: pow(tan(atan2(var(--captured-length), 1px)), 2);
        --diff: sqrt(var(--area) / (var(--w) * var(--h)));
        --scaled-height: calc(1px * var(--h) * var(--diff));

        height: clamp(
            var(--base-height) * var(--logo-min-size-factor),
            var(--base-height) + (var(--scaled-height) - var(--base-height)) * var(--strength),
            var(--base-height) * var(--logo-max-size-factor)
        );

        &:is([src$=".jpg"], [src$=".png"]) {
            mix-blend-mode: multiply;
        }
    }
}
```

使用这种方法有一个细节需要注意，在 `<img>` 上不能显式设置 `width` 和 `height` 属性的值，因为我们在 CSS 中只调整了 `img` 的 `height` ，如果你设置了 `width` ，将会影响整体的效果。切记！
