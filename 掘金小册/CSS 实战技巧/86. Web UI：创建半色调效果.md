## 什么是半色调效果？

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/9a73ff3c10a34e30907dfdde6be68a0c~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1170&h=780&s=326450&e=jpg&b=fbf1e1)

  


[维基百科是这样描述半色调的](https://en.wikipedia.org/wiki/Halftone)：

  


> 半色调（Halftone）是一种图像复制技术，它通过使用点阵来模拟连续色调的图像。这些点的大小或间距可以变化，从而产生类似渐变的效果。

  


简单地说，半色调是指通过不同大小的点来再现图像中的明暗变化的技术。它最初是一种印刷图像的技术，目的是为了在当时其他印刷工艺过于昂贵的情况下，能够在期刊和其他出版物中使用。你可能在较旧的漫画书中见过这种风格，那时的颜色再现是有限的。

## CSS 中的半色调

  


刚才提到过，我们只需要使用三个 CSS 规则就可以创建出半色调效果，这三个规则分别是：

  


-   首先，使用 CSS 渐变创建的图案，比如 `radial-gradient()` 创建点状背景，`linear-gradient()` 创建的遮罩层（或映射图）
-   接着，使用 CSS 混合模式（比如 `background-blend-mode: multiply`）对背景层进行混合处理
-   最后，使用 CSS 滤镜（比如 `filter: contrast(16)` ）调整图案对比度

  


具体的代码如下：

  


```CSS
.halftone {
    background:
        radial-gradient(closest-side, #777, #fff) 0 / 1em 1em space,
        linear-gradient(90deg, #888, #fff);
    background-blend-mode: multiply;
    filter: contrast(16);
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/94171d9fd8ac439489c978df81620473~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2352&h=712&s=280130&e=png&b=fdfdfd)

### 创建图案

  


在 CSS 中使用径向渐变 `radial-gradient()` 创建点状背景非常简单:

  


```CSS
.halftone {
    background: radial-gradient(
        circle at center, 
        #000 0.25em, 
        transparent 0
   ) 0 0 / 1em 1em space;
}
```

  


在我们的示例中，使用的是：

  


```CSS
.halftone {
    background: radial-gradient(closest-side, #000, #fff) 0 0 / 1em 1em space;
    
    /* 等同于 */
    background: radial-gradient(circle closest-side, #000, #fff) 0 0 / 1em 1em space;
}
```

  


第一段代码绘制的圆点效果是从黑色到透明过渡而没有混合，这样就能得到一个带有平滑平缘的圆；第二段中，我们使用了关键词 `closest-side` ，并且没有显式指定渐变颜色停止点位置，它将获得的圆点是从黑色平滑过渡到透明：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/4aa47842f0644402ad4550fc6aef2295~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2466&h=988&s=710917&e=png&b=f4f4f4)

  


> Demo 地址：https://codepen.io/airen/full/raNGQXo

  


`closest-side` 的意思是渐变结束形状如果是圆形，与容器距离渐变中心点最近的一边相切，如果是椭圆，则与距离渐变中心点最近的垂直和水平边相切。我用下图来描述，可能会更易于理解一些：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/681050e4631041efaaf6055579fb5db5~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1280&h=1222&s=116145&e=jpg&b=fdfdfd)

  


另一点就是，`background-repeat` 的值使用了 `space` ，它可以确保在两个轴上都有整数数量的圆点。也可以将 `background-repeat` 设置为 `round` ，确保点不会被裁剪。下面是 `space` 和 `round` 的差异：

细心的你可能已经发现了，我们示例中的渐变颜色使用的是 `#777` ，而这里使用的是 `#000` ，关于这一点，稍后我们会介绍。

  


在创建图案的过程中，我们还使用了另一个渐变，我们把这一层称为图形映射层——它主要用来决定哪些地方的点更大，哪些地方的点更小：

  


```CSS
.halftone{
    background: linear-gradient(90deg, #000, #fff);
}
```

> Demo 地址：https://codepen.io/airen/full/jEOGXaQ

  


到目前为止，两个背景层叠加在一起，并不能使圆点的大小不一：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/be8bd88678a74f3e88256c82fa7cedd2~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2466&h=988&s=1058234&e=png&b=e5e5e5)

  


> Demo 地址：https://codepen.io/airen/full/MYWERWP

  


温馨提示，这里我们应用了多背景的特性，[需要注意每个背景层的顺序](https://juejin.cn/book/7199571709102391328/section/7199845838103773195#heading-8)。在我们这个示例中，圆点背景层（`radial-gradient()`）在顶上，线性渐变（`linear-gradient()`）在底部。它们出现的先后顺序将会影响到最终的效果。

  


在我们继续往下之前，先仔细看一下这些渐变。两层中的每一层都从黑色到白色，中间是灰色（就是亮度为 `50%` 的灰色）或者说是 `50%` 灰。

  


请注意，对于任何灰色，无论它位于黑色和白色之间的哪个位置，饱和度始终是 `0%` ，并且与色相无关，所以我们使用 `0` 。唯一变化的值是亮度，它从黑色（`0%`）到白色（`100%`）。也就是说，沿着渐变线从 `0%` 到 `100%` 意味着沿着 HSL 双锥体的亮度轴从 `0%` 到 `100%` :

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/0d8590a097bc4b84b14e2262cdcaba85~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=720&h=620&s=21114&e=webp&b=22242a)

  


因此，通常情况下，任何 `p%` 的灰色可以写成 `rgb(p% p% p%)` 或 `hsl(0 0% p%)`。

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/ffbb1d8ba78e4ce4a33097d9efbe5f4e~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1210&h=496&s=2315221&e=gif&f=279&b=520a11)

### 将两背景层混合在一起

  


继续回到我们的背景，它由径向渐变绘制的圆点层和一个线性渐变绘制的映射层组成，由于圆点层位于映射层顶部，因此我们并不能看到线性渐变绘制的映射层。所以，接下来我们需要应 [CSS 的混合模式](https://juejin.cn/book/7223230325122400288/section/7259669043622690853#heading-23)，将这两个背景层混合在一起。

  


CSS 的混合模式有 `mix-blend-mode` 和 `background-blend-mode` ，在这里我们将使用 CSS 的 `background-blend-mode` 属性，因为我们要将两个背景层混合在一起。如果你对 CSS 的混合模式稍微有所了解的话，就应该知道混合模式有很多种类型。我们选择其中的正片叠底（即 `multiply`）。

  


正片叠底混合模式就像将多个透明图层叠放在一起。白色像素将变为透明，而黑色像素将保持为黑色。介于两者之间的颜色将会相乘其亮度（光）值。这意味着亮色会变得更亮，而暗色会变得更暗，通常会产生较暗的结果。

  


正片叠底混合模式（`multiply`）简单地将每个颜色通道的值相乘。由于这些值在 `0` 和 `1` 之间进行了归一化，因此其乘积始终比原始值更暗：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/502f12db1f0948cfbcc86b1b7f9988c4~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1280&h=519&s=28441&e=jpg&b=f4f4f6)

  


在这种模式下，计算每个颜色通道（红、绿、蓝）的新值，新值等于相应颜色通道的两个输入值相乘。例如，如果一个像素的背景颜色通道为 `0.5`（半亮度），前景颜色通道为 `0.8`（较亮），则计算结果为 `0.5 x 0.8 = 0.4`。这个过程针对每个颜色通道分别进行，得到新的 RGB 值。

  


这也意味着，创建半色调将用到的第二个 CSS 规则是:

  


```CSS
.halftone {
    background: 
         radial-gradient(closest-side, #000, #fff) 0 / 1em 1em space,
         linear-gradient(90deg, #000, #fff);
    background-blend-mode: multiply;
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/93018d03dc8c4d8ea31c5b80d337c922~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2466&h=988&s=1527570&e=png&b=272727)

  


> Demo 地址：https://codepen.io/airen/full/MYWEREx

  


在我们的例子中，两个渐变都从黑色到白色，中间的所有值都是灰色，它们的 RGB 通道值都相等。所以，如果在某个点，两个对应像素的值都是 `rgb(50% 50% 50%)` ，那么正片叠底混合模式的结果是 `0.25` 。

  


我们可以看到，正片叠底混合模式的结果总是至少和我们乘的两个像素中较暗的那个一样暗。这是因为 RGB 值在 `[0,1]` 区间内，正如之前提到的，乘以这些值总是会得到一个不超过两个值中较小者的结果。通道值较小，表示的灰色就越暗。

  


在混合图案层和映射层后，我们可以看到图案点在左侧变得更暗，因为那里映射更接近黑色：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/236a448b1cb044a98980d4b7dbc5f1de~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2466&h=988&s=1527570&e=png&b=272727)

  


你可以放大两个从映射层渐变线上的不同位置的点，第二个点比第一个点更亮（靠右）。下图中，深红色圆圈标记了每个点的 `50%` 灰色界限：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/532df17b99994713aa549241685317d6~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=720&h=358&s=26902&e=webp&b=6f6f6f)

  


对于较暗的点，`50%` 灰色界限是一个比较亮的点（更大的圆圈），每个深红色圆圈内的灰色比 `50%` 灰色更暗，而圈外的灰色则比 `50%` 灰色更亮。请先记住这一点！

  


### 使用滤镜调整对比度

  


在上面示例的基础上，我们新增第三个 CSS 规则，即[使用 `filter` 的 `contrast()` 函数来调整对比度](https://juejin.cn/book/7223230325122400288/section/7259669043622690853#heading-5)：

  


```CSS
.halftone {
    background: 
         radial-gradient(closest-side, #000, #fff) 0 / 1em 1em space,
         linear-gradient(90deg, #000, #fff);
    background-blend-mode: multiply;
    filter: contrast(16);
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/a4033743fbe24cbaa1cfa86d9d628309~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2466&h=988&s=351766&e=png&b=050505)

  


> Demo 地址：https://codepen.io/airen/full/bNGoJyQ

  


目前，你所看到的效果，并是不是我们所期望的半色调效果。你可能会感到好奇，为什么会这样呢？

  


`filter` 的 `contrast()` 函数取决于它的参数是否小于 `1` 。如果它的参数小于 `1` ，那么它将每个通道的值推向 `0.5` ，即 `[0,1]` 区间的中点。值为 `1` 表示没有变化，值为 `0` 表示该通道的值被推到 `0.5` 。这意味着 `filter: contrast(0)` 总是会给我们 `50%` 的灰，无论滤镜的输入是什么。

  


然而，如果 `contrast()` 函数的参数大于 `1` ，那么每个通道的值将被推向 `0` 或 `1` ，取决于哪个更接近。如果我们给 `contrast()` 函数传入一个足够大的值，所有通道的值要么变为 `0` ，要么变为 `1` ，这意味着我们只能得到八种可能的结果。

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/00a5459d9b87460495becf671633ae1e~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1536&h=303&s=81570&e=webp&b=fe7f00)

  


回到我们的示例中来，我们使用了 `filter: contrast(16)` ，它会将所有比 `50%` 灰更暗的灰色推向黑色，其他的则推向白色。

  


还记得之前提到过，如果点更暗，`50%` 灰的界限是更大的圆圈吗？那就是我们对比度的限制。在那个圆圈内，我们有比 `50%` 灰更暗的灰色，因此它们会被大的对比度值推到黑色。圆圈外的灰色则比 `50%` 灰更亮，因此它们会被大的对比度值推到白色。

  


由于点越暗，`50%` 灰的限制圆圈就越大，这意味着映射中较暗区域的半色调点会更大。因此，呈现给我们的效果就会像下面这样：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/00cf7d02dcc44a119896fd63abbf470c~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2466&h=988&s=351766&e=png&b=050505)

  


到目前为止，这个效果并不理想。由于映射层（线性渐变）的左半部分比 `50%` 灰更暗（RGB 通道值低于 `0.5`），将任何其他层与它进行正片叠底混合的结果至少也会是较暗的。这意味着在左半部分进行混合的结果是比 `50%` 灰更暗的灰色，因此大的对比度值将左部分的所有内容推到黑色。

  


解决这个问题的方法很简单，不让我们的渐变从黑色到白色，而是从中灰到白色。此外，为了获得最佳效果，映射最暗的部分应该比 `50%` 灰稍微亮一些，而图案可以稍微暗一些。这也就是我们最初的示例中，创建渐变时不使用黑色的主要原因：

  


```CSS
.halftone {
    background: 
        radial-gradient(closest-side, #777, #fff) 0 / 1em 1em space,
        linear-gradient(90deg, #888, #fff);
    background-blend-mode: multiply;
    filter: contrast(16);
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/9a2bc72da6984863b8b91fa8604aa91d~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2466&h=988&s=426559&e=png&b=fdfdfd)

  


> Demo 地址：https://codepen.io/airen/full/jEOaMwp

  


需要注意的一点是，`contrast()` 函数的值要足够大，以补偿我们点的模糊半径。因此，如果我们增加图案的大小（即圆点的大小），那么它的值也需要相应的增加：

  


```CSS
.halftone {
    background: 
        radial-gradient(closest-side, #777, #fff) 0 / 9em 9em space,
        linear-gradient(90deg, #888, #fff);
    background-blend-mode: multiply;
    filter: contrast(80);
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/211903409d8b4ff8b8671b73c4304f95~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2466&h=988&s=120089&e=png&b=ffffff)

上面我们探讨了 CSS 是如何基于一个元素，且仅用三个规则就能创建出半色调效果的全过程，以及它背后的工作原理和需要注意的相关事项。接下来，我将向大家展示另一种创建半色调效果的方法。这种方法有三个关键点：

  


-   使用 `radial-gradient()` 创建重复的背景（圆点）
-   使用一个遮罩层，即我们想要再现的图像或图案
-   对比度滤镜：`filter: contrast()` ，我们将提高 `radial-gradient` 背景的对比度，并将其与遮罩结合，这样就得到半色调效果

  


与之前方法不同之处是，我们需要将背景应用到伪元素上，这是因为我们将对元素本身应用 `filter: contrast()` 。与此同时，我们还需要给元素指定一个背景颜色，在这里我将它暂时设置为白色：

  


```CSS
.halftone {
    display: grid;
    filter: contrast(16);
    background: #fff; 
    
    &::after {
        content: "";
    }
}
```

  


在伪元素上使用 `radial-gradient()` 绘制圆点图案：

  


```CSS
.halftone {
    display: grid;
    filter: contrast(16);
    background: #fff; 
    
    &::after {
        content: "";
        background: 
            radial-gradient(
                circle at center,
                #000 3%,
                color-mix(in oklch, #000, transparent 100%) 65%
            ) 0 0 / 1em 1em,
            radial-gradient(
                circle at center,
                #000 3%,
                color-mix(in oklch, #000, transparent 100%) 65%
            ) 0.5em 0.5em / 1em 1em;
    }
}
```

接着使用 `linear-gradient` 创建一个遮罩层，应用在伪元素 `::after` 上：

  


```CSS
.halftone {
    display: grid;
    filter: contrast(16);
    background: #fff; 
    
    &::after {
        content: "";
        background: 
            radial-gradient(
                circle at center,
                #000 3%,
                color-mix(in oklch, #000, transparent 100%) 65%
            ) 0 0 / 1em 1em,
            radial-gradient(
                circle at center,
                #000 3%,
                color-mix(in oklch, #000, transparent 100%) 65%
            ) 0.5em 0.5em / 1em 1em;
            
        mask-image: linear-gradient(#000, color-mix(in oklch, #000, transparent 50%));        
    }
}
```

> Demo 地址：https://codepen.io/airen/full/EaxbWbY

  


注意，示例中我们使用了 `color-mix()` 来调整颜色的透明度，如果你对这方面不太了解的话，请移步阅读《[Web UI：你所不知道的 `color-mix()`](https://juejin.cn/book/7395119173656903715/section/7445261940961738761)》！

  


最后，需要再次强调一下，需要在元素自身（而不是伪元素）上应用对比度滤镜。其中背景颜色也非常重要，否则无法看到对比效果。

  


```CSS
.halftone {
    background: #fff;
    filter: contrast(16);
}
```

  


这里的目标是添加足够的对比度，使得我们的径向渐变绘制的圆点大小会根据渐性渐变的遮罩程序有所变化。

  


## 创建半色调效果的变体

  


之前，我们介绍了两种创建半色调效果的方法，虽然使用的技巧略有不同，但原理是相通的。到目前为止，我们所看到的都是最基础的半色调效果，但不意味着 CSS 不能创建出复杂的半色调效果。事实上，我们可以通过调整图案或映射层（或遮罩层）来获得不同的半色调效果。

  


为了更好的管理 CSS 代码，我们分别将图案和映射层（或遮罩）存储在 `--pattern` 和 `--mask` 两个变量中：

  


```CSS
.halftone {
    --pattern: radial-gradient(closest-side, #777, #fff) 0 / 1em 1em space;
    --mask: linear-gradient(in oklch to right, #888, #fff);
    
    background: var(--pattern), var(--mask);
    background-blend-mode: multiply;
    filter: contrast(16);
}
```

  


现在，我们可以通过改变 `--pattern` 或 `--mask` 使效果变得更有趣，以获得更酷炫的效果。

```JavaScript
let ticking = false;

ele.addEventListener('mousemove', (e) => {
    if (!ticking) {
        requestAnimationFrame(() => {
            const rect = ele.getBoundingClientRect(); // 获取元素的位置
            const x = e.clientX - rect.left; // 鼠标相对于元素左边缘的偏移
            const y = e.clientY - rect.top;  // 鼠标相对于元素上边缘的偏移
    
            ele.style.setProperty('--x', `${x}px`);
            ele.style.setProperty('--y', `${y}px`);
            ticking = false;
        });
        ticking = true;
    }
});
```

> Demo 地址：https://codepen.io/airen/full/xbxPvyb

  


更有趣的是，映射层（遮罩层）还可以是一张图片。直接使用任何随机图片作为 `--mask` 并不能很好的工作。我们需要将图片的饱和度降到 `0` ，并且对于这个特定的技巧，我们需要确保它的像素亮度大致在 `[50%,100%]` 区间内。

  


在 CSS 中我们可以将 `--pattern` 和 `--mask` 分别作为元素的伪元素，混合它们，并对伪元素的父元素应用 `contrast()` 滤镜。这样 `--mask` 伪元素也可以应用滤镜。然而，我们在这里寻找的是不涉及额外元素或伪元素的解决方案。

  


我们可以做的是让 `--mask` 通过多个混合的背景层的结果来实现。将背景颜色设置为任意灰（比如 `grey`），并使用亮度混合模式（`luminosity`）将其与 `--mask` 中的图片混合，可以得到一个结果，该结果的亮度是图片的亮度，饱和度是背景颜色的饱和度，而由于背景是灰色（饱和度为 `0`），因此色调变得无关紧要。具体的代码如下：

  


```CSS
.halftone {
    --url: url("https://picsum.photos/1024/768");
    --pattern: radial-gradient(closest-side, #777, #fff) 0 0 / var(--s, 1em)
      var(--s, 1em) space;
    --mask: var(--url) center / cover grey;
    background: var(--pattern), var(--mask);
    background-blend-mode: multiply , luminosity;

    filter: contrast(16);
}
```

  


上面代码中 `background-blend-mode` 应用了 `multiply` 和 `luminosity` ，其中 `multiply` 会将 `--pattern` 和 `--mask` 进行混合得到一个结果；`luminosity` 则将 `--mask` 中的图片和背景颜色进行混合得到一个结果：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/d3449d993d34410aa1e09353fcf03614~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1254&h=512&s=2660261&e=gif&f=199&b=060606)

  


> Demo 地址：https://codepen.io/airen/full/gbOXVqG

  


在我们的案例中，如果我们使用屏幕混合模式（`screen`）将上一步得到的去饱和图片与像 `#888` 这样的中灰色进行混合，那么结果总是至少和 `#888` 一样亮。仅在我们与纯黑像素混合时，结果才是 `#888` 。无论我们与比纯黑亮的像素混合，结果都会比 `#888` 亮。因此，基本上，我们得到了一个在最暗处为 `#888` 的 `--mask` ，就像我们的基础 `--mask` 一样（线性渐变创建的映射图）：

## 给半色调效果着色

  


到目前为止，我们看到的半色调效果都是黑白色的。其实，我们可以通过 `mix-blend-mode` 给它着色。我们可以在 `.halftone` 的父元素上设置一个更丰富的背景颜色，比如一个渐变颜色：

  


```CSS
main {
    background: linear-gradient(
        90deg,
        #75d9b4,
        #aa6b71,
        #f2896f,
        #ffb87f,
        #ffec72
    );
}
```

  


然后在 `.halftone` 上使用 `mix-blend-mode` ，将整个半色调与父元素进行屏幕混合（`screen`）处理，这样就可以给黑白半色调着色：

  


```CSS
.halftone {
    --pattern: radial-gradient(closest-side, #777, #fff) 0 0 / var(--s, 1em) var(--s, 1em) space;
    --mask: conic-gradient(
        from var(--a) at var(--x) var(--y),
        #fff,
        #888,
        #fff
    );
    background:  var(--pattern), var(--mask);
    background-blend-mode: multiply;
    mix-blend-mode:screen ;
    filter: contrast(16);
}
```

## 案例：半色调遮罩背景卡片

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/305f0529e6f343708cc72480f2f815cd~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=800&h=540&s=79898&e=webp&b=1a1c33)

  


上面这个效果来自于 [@Ana Tudor 提供的半色调效果的案例集合](https://codepen.io/thebabydino/full/GRNKqRv)。我们来看看如何通过前面所介绍的 CSS 技术给卡片添加半色调背景效果。

  


下图将是应用到卡片的背景图：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/8b59c76028ee4a1794033406999cadc1~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1400&h=1000&s=345369&e=jpg&b=2e2b28)

  


我们开始进入 CSS 的编写。

  


```CSS
.halftone {
    background: 
        linear-gradient(#f99a00, #ac1a1a),
        url(https://images.unsplash.com/photo-1611146479742-12b6cdacfbab?w=1400) 50% / cover;
    background-blend-mode: color;
    clip-path: inset(1px round 7px);
}
```

  


上面的代码将背景图应用到卡片元素 `.halftone` 中，在这个示例中，背景图上还覆盖了一层金色（`#f99a00`）到红色（`#ac1a1a`）的渐变，并且对这两个背景层进行了 `color` 模式混合。它将会对背景图进行调色。

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/004eb9a325d24e98a9f7cc71480d7765~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2358&h=988&s=3440182&e=png&b=e47520)

  


接下来，我们新增一个伪元素 `::before` ，并在这个伪元素上应用之前所介绍的半色调效果：

  


```CSS
.halftone {
    --s: 2em;
    --pattern: radial-gradient(#fff, #888) 0 0 / var(--s) var(--s);
    --mask: radial-gradient(circle at 0, #fff, #000);
    
    background: 
        linear-gradient(#f99a00, #ac1a1a),
        url(https://images.unsplash.com/photo-1611146479742-12b6cdacfbab?w=1400) 50% / cover;
    background-blend-mode: color;
    clip-path: inset(1px round 7px);
    
    &::before {
        content:"";
        grid-area: 1 / 1 / -1 / -1;
        background: var(--pattern), var(--mask);
        background-blend-mode: multiply;
        filter: contrast(50) blur(2px) contrast(4);
    }
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/87394e9129b447a689d569aab729058e~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2358&h=1538&s=689665&e=png&b=030303)

  


这个时候，半色调效果覆盖在卡片背景上。但它没有与背景图片融合在一起，我们可以使用 CSS 混合模式 `mix-blend-mode` 将它们进行混合，从而使半色调效果与背景融合在一起：

  


```CSS
.halftone {
    --s: 2em;
    --pattern: radial-gradient(#fff, #888) 0 0 / var(--s) var(--s);
    --mask: radial-gradient(circle at 0, #fff, #000);
    
    background: 
        linear-gradient(#f99a00, #ac1a1a),
        url(https://images.unsplash.com/photo-1611146479742-12b6cdacfbab?w=1400) 50% / cover;
    background-blend-mode: color;
    clip-path: inset(1px round 7px);
    
    &::before {
        content:"";
        grid-area: 1 / 1 / -1 / -1;
        background: var(--pattern), var(--mask);
        background-blend-mode: multiply;
        filter: contrast(50) blur(2px) contrast(4);
        mix-blend-mode: darken;
    }
}
```

  


[我们使用了 `darken` （变暗）混合模式](https://juejin.cn/book/7223230325122400288/section/7259669043622690853#heading-30)，这个时候你看到的效果如下：

最后，我们通过另一个伪元素 `::after` 给卡片添加内容。你也可以在元素内嵌套其他的元素，这里是为了追求在一个元素上创建半色调背景卡片效果。另外，在伪元素应用一个[变亮混合模式](https://juejin.cn/book/7223230325122400288/section/7259669043622690853#heading-31)（`lighten`）：

## 案例：创建复古印刷风格图片效果
