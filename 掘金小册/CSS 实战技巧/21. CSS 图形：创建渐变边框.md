## 创建渐变边框的基础

  


在开始介绍 CSS 创建渐变边框之前，我们先了解几个必要的基础知识。

  


通常情况之下，Web 开发者会[使用 CSS 的 `border` 属性给元素添加边框效果](https://juejin.cn/book/7199571709102391328/section/7199845356845629472)，然而在很多情景之下，仅仅使用 `border` 属性是无法满足边框样式的需求，例如我们要创建的渐变边框效果。这是因为，`border-color` 属性的值只能是 `<color>` 类型值，并不能使用 `<image>` 类型值。因此，Web 开发者会采用其他的一些方式来模拟边框的视觉效果，例如使用伪元素 `::before` 或 `::after` ，甚至是空的 HTML 元素来模拟边框视觉效果，尤其是在创建渐变边框的时候，这是被广泛使用的方法之一。

  


随着 CSS 的 `border-image` 属性越来越完善，Web 开发者逐渐将 `border` 和 `border-image` 相结合来创建渐变边框。即便如此，它们相互结合也无法满足所有的情景，例如带有圆角效果的渐变边框，这种方法就不适合了。

  


如果你对 CSS 熟悉的话，我们不应该遗忘 CSS 的 `background` 和盒模型的特性。首先，`border` 是 CSS 盒模型的一部分，并且 CSS 盒模型有一个“布局框限制”的特性，即盒模型的 `margin` 、`border` 、`padding` 和内容（`content`）分别限制在对应的布局框中：

  


-   `margin` 限制在 `margin-box` 框中
-   `border` 限制在 `border-box` 框中
-   `padding` 限制在 `padding-box` 框中
-   `content` 限制在 `content-box` 框中

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/fb2c8c9e5fad4c9583f894d7637b38f9~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1129&s=346507&e=jpg&b=fcf3f2)

  


简单的说，它们各种是不会溢出自己所在的“布局框”中。

  


与此同时，`background` 给元素设置背景色或背景图片（包括 CSS 渐变）时，可以通过 `background-origin` 指定元素背景的原点位置的背景相对区域；通过 `background-clip` 属性设置元素的背景（背景图片或颜色）是否延伸到边框（`border-box`）、内边框（`padding-box`）、内容框（`content-box`）下面。

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/01134597c93b4d2f85f6fed9480d7893~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1278&h=668&s=1867179&e=gif&f=307&b=040404)

  


> Demo 地址：https://codepen.io/airen/full/gONNPbv

  


正如你所看到的，我们可以将渐变限制在任意你想要的布局框中。

  


既然要创建渐变边框，那就要对 CSS 的渐变有所了解，我想大家对于渐变的基础知识应该不会有任何问题，至于它的细节和很多不为人知的一面，要是你感兴趣的话，[可以点击这里获取相关的信息](https://juejin.cn/book/7199571709102391328/section/7199845781149810727)。我们在这节课中，只会应用我们所熟悉的 CSS 渐变函数来创建渐变效果：

  


-   线性渐变：`linear-gradient()` （一次性线性渐变）和 `repeating-linear-gradient()` （重复性线性渐变）
-   径向渐变：`radial-gradient()` （一次性径向渐变）和 `repeating-radial-gradient()` （重复性径向渐变）
-   锥形渐变：`conic-gradient()` （一次性锥形渐变）和 `repeating-conic-gradient()` （重复性锥形渐变）

  


```CSS
.linear-gradient {
    background-image: linear-gradient(#ff8a00, #e52e71);
}

.repeating-linear-gradient {
    background-image: repeating-linear-gradient(
        to right,
        #ff8a00 10%,
        #e52e71 20%
    );
}

.radial-gradient {
    background-image: radial-gradient(#ff8a00, #e52e71);
}

.repeating-radial-gradient {
    background-image: repeating-radial-gradient(circle, #ff8a00 10%, #e52e71 20%);
}

.conic-gradient {
    background-image: conic-gradient(#ff8a00, #e52e71);
}

.repeating-conic-gradient {
    background-image: repeating-conic-gradient(#ff8a00 10%, #e52e71 20%);
}
```

> Demo 地址：https://codepen.io/airen/full/abaGdQK

  


了解完这些基础之后，我们开始来探讨制作渐变边框的几种常见方法！

  


## 创建渐变边框

  


在 CSS 中，创建渐变边框主要有四种常见的方法：

  


-   边框图片： `border-image`
-   布局框：`padding-box` 与 `border-box` 相结合
-   伪元素重叠
-   遮罩 `mask`

  


### 使用 `border-image` 创建渐变边框

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/edd8d36fee5a4424bd3a49a90081a373~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1200&h=630&s=233602&e=jpg&b=153168)

  


CSS 的 `border-image` 属性是那种你明明知道它存在，却可能从未真正使用过的属性之一。或者，也许你尝试过使用它，但发现其中的切片逻辑复杂又繁琐。然而，我要告诉你的是，`border-image` 不仅能够产生一些非常引人注目的 UI 设计，它还能实现一些我们通常会依赖其他属性或方法来完成的常见样式，例如渐变边框。

  


```CSS
.gradient--border {
    --b: round(.5em, 1px);
    --s: 40vw;
    --c: red;
    --a: 180deg;
    --g: conic-gradient(from var(--a) in oklch longer hue, var(--c) 0 0);
    
    background: color-mix(in srgb, var(--c), #000 60%);
    border: var(--b) solid #0000;
    border-image: var(--g) 1;
}
```

  


上面代码中，[使用了 `conic-gradient()` 函数创建了一个锥形渐变](https://juejin.cn/book/7223230325122400288/section/7259668771856941111)，它的效果看上去像下图这样：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/92f344515b2940e7be3047f3a9d92fee~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=866&s=196660&e=jpg&b=050505)

  


简单解释一下。`conic-gradient()` 中的 `from var(--a)` 指定了渐变的起始角度为 `--a` 。与平时创建渐变不同之处是，我在 `conic-gradient()` 函数中应用了[颜色插件的新功能](https://juejin.cn/book/7223230325122400288/section/7233227753909125178#heading-29)，**旨在避免两个颜色之间的** **[灰色死亡区](https://juejin.cn/book/7199571709102391328/section/7199845781149810727#heading-11)** **，并通过选择精确的停止位置（渐变颜色的停止位置）来缓解渐变，** 即 `in oklch longer hue` 。它包含了两层意思：

  


-   `in oklch` 指定了渐变的颜色空间为 [OKLCH](https://juejin.cn/book/7223230325122400288/section/7235205520154427429)（一种新的颜色模型，基于亮度 `L` 、色调 `H` 和色度 `C` 来定义颜色，它能更加准确反映人眼对颜色的感知）
-   `longer hue` 意味着在进行色相渐变时，选择更长的色相路径。这在渐变色彩过渡上会产生不同的视觉效果，尤其是在色相变化的范围较大时

  


`var(--c) 0 0` 比较好理解，`--c` 代表是渐变颜色，`0 0` 是渐变停靠点的位置和范围。注意，`conic-gradient()` 有点特殊，渐变轴是一个圆形，因此 `0%` 和 `100%` 位于圆周上相同的位置上。

  


在这个示例中，我们把 `conic-gradient()` 函数创建的锥形渐变作为边框区域的图像源（即 `border-image-source` 属性的值），并通过 `border-image-slice` 属性设置边框区域图像的分割数，用于匹配渐变和边框：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/d6e6a1f780f74755bdee00d88595931a~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1422&h=714&s=2780773&e=gif&f=229&b=fefefe)

  


> URL:https://mdn.github.io/css-examples/tools/border-image-generator/

  


注意，上面的代码中并没有显式设置 `border-image-width` 属性的值，但你会发现，我们渐变的边框厚度等于我们定义的 `border-width` ，即 `var(--b)` 。这意味着下面两段代码得到的效果将是等同的：

  


```CSS
.gradient--border {
    border: var(--b) solid #0000;
    border-image: var(--g) 1;
}

/* 等同于 */
.gradient--border {
    border-image: var(--g) 1 / var(--b);
}
```

  


这样你就给元素设置了一个渐变边框：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/3c0bbfddd3cf4e33a39a6309e2d169fa~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=866&s=151086&e=jpg&b=050505)

  


> Demo 地址：https://codepen.io/airen/full/KKjjgNL

  


有一点需要知道的是，同时使用 `border` 和 `border-image` 定义渐变边框时，需要注意两者的顺序，**`border-image`** **必须在** **`border`** **之后出现，否则** **`border`** **将会覆盖** **`border-image`** ，你将看到的 `border` 属性设置的边框效果。虽然规范上定义的是，`border-image` 应该替代你定义的任何常规边框（`border`），但因为技术上我们在 `border-image` 之后定义了 `border` 。因此，使用 `border-image` 时，属性的顺序非常重要，切记！

  


就我个人而言，我不喜欢同时使用 `border` 和 `border-image` ，因为这样可以避免覆盖我试图创建的边框图像，并且仅通过一个属性来控制边框样式。因此，如果你得到一个奇怪的结果，不要忘了检查是否在其他地方定义了 `border` 。

  


另外，当 `border-image` 和 `border-radius` 同时出现时，`border-radius` 也不会使边框变成圆角。即便 `border-radius` 有效，但在视觉上不会产生任何圆角效果。正因此，这也是 Web 开发者不太喜欢使用 `border-image` 原因之一。

  


我们可以利用 `clip-path` 中的一个小技巧使 `border-image` 创建的渐变边框带有圆角效果，但前提是圆角半径不能大于边框厚度。因为这个技巧无法使内容区域带有圆角。

  


```CSS
.gradient--border {
    --b: round(.5em, 1px);
    --r: var(--b);
    --s: 40vw;
    --c: red;
    --a: 180deg;
    --g: conic-gradient(from var(--a) in oklch longer hue, var(--c) 0 0);
    
    background: color-mix(in srgb, var(--c), #000 60%);
    border-image: var(--g) 1 / var(--b);
    clip-path: inset(0 round var(--r));
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/be0ff18d8e034800bf0d7b95fb56d8b4~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=652&s=181762&e=jpg&b=050505)

  


> Demo 地址：https://codepen.io/airen/full/KKjjNWg

  


现在，你可以通过调整 CSS 自定义属性的值，获得不同变体的渐变边框效果：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/c8067524e80c43ac9bb72f16daca766b~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1196&h=636&s=2580765&e=gif&f=281&b=040404)

  


> Demo 地址：https://codepen.io/airen/full/YzoopeM

  


注意，你可以将示例中的锥形渐变 `conic-gradient()` 替换成你喜欢的渐变函数。

  


### 使用 `background` 创建渐变边框

  


之前我们提到过，在 CSS 中，每个元素都是一个盒子，每个盒子主要由 `margin` 、`border` 、`padding` 和 `content` 组成，而且它们有自己对应的限制区域，即“布局框限制”。我们通常会用到其中的 `border-box` 、`padding-box` 和 `content-box` ：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/d9613c5b583442eb9deb91e647303618~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1196&h=834&s=7469259&e=gif&f=253&b=2c2c2c)

  


注意，`margin-box` 通常在[应用 `shape-outside` 和 `shape-inside` 构建不规则布局时会用到](https://juejin.cn/book/7161370789680250917/section/7161625566062313503#heading-17)，另外在使用 `clip-path` 裁剪图形时也会用到，比如[使用 `clip-path` 创建带圆角的多边形时](https://juejin.cn/book/7395119173656903715/section/7401099842379841545#heading-3)，我们就应用了 `margin-box` 。

  


这意味着，我们将会应用到两个背景，其中一个被限制在 `padding-box` （它可能是一个纯色，根据需要来决定），另外一个将被限制在 `border-box` ：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/8b5f4b974d0144b4838c471e1f0eac14~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=989&s=308786&e=jpg&b=2a2a2a)

  


在 CSS 中，我们可以通过 `background-origin` 和 `background-clip` 来控制每个背景的覆盖区域。具体的 CSS 代码如下所示：

  


```CSS
.gradient--border {
    --b: round(0.5em, 1px);
    --r: var(--b);
    --s: 40vw;
    --c: red;
    --a: 180deg;
     --bg: color-mix(in srgb, var(--c), #000 60%);
    --g1: linear-gradient(var(--bg) 0 100%);
    --g2: conic-gradient(from var(--a) in oklch longer hue, var(--c) 0 0);
    
    border-radius: var(--r);
    border: var(--b) solid #0000;
    background: var(--g1) padding-box, var(--g2) border-box;    
}
```

  


首先使用 `border` 设置了一个厚度为 `--b` 的透明实线边框，并且使用 `linear-gradient()` 创建了一个纯色渐变和 `conic-gradient()` 创建了一个锥形渐变。我将这两个渐变同时应用于 `background` ，其中 `--g1` （纯色的线性渐变）被限制在 `padding-box` 中，`--g2` 被限制在 `border-box` 中。

  


整个效果如下：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/09b725c282204c9eaf6fc88e75d5bd04~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1282&h=732&s=5443941&e=gif&f=386&b=070707)

  


> Demo 地址：https://codepen.io/airen/full/eYwwvep

  


这种方式创建的渐变边框与 `border-image` 创建的渐变边框最大的不同之处，你可以给元素设置任意你想要的 `border-radius` ，并且它能按照你的预期来控制元素的圆角效果：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/01330a89578c4a5abc90dc50a7af3720~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1294&h=634&s=1129883&e=gif&f=121&b=070707)

  


上图所示展示的效果是，`--r` 的值是 `--b` 的两倍，即 `--r: calc(2 * var(--b))` 。

  


使用这种方式创建渐变边框还有另一个好处，你可以使用 `padding` 来替代 `border` 。不过，你需要相应的调整 `background-origin` 和 `background-clip` 属性的值：

  


```CSS
.gradient--border {
    border-radius: var(--r);
    /* border: var(--b) solid #0000; */
    padding: var(--b);
    background: var(--g1) content-box content-box, var(--g2) padding-box padding-box;
}
```

  


呈现给你的视觉效果是一致的：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/a21c9c47b34542ec9cec758df86443c6~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1274&h=738&s=2182810&e=gif&f=250&b=070707)

  


> Demo 地址：https://codepen.io/airen/full/zYVVZXZ

  


使用 `background` 创建渐变边框时，有一个细节必须要知道，那就是要注意背景的顺序，它将会影响最终的结果：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/2ea604e0b66c455c840f77ab8a127f44~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=900&h=828&s=357228&e=jpg&b=242424)

  


最新资讯：[Safari TP 最新版本已经支持了 `background-clip: border-area`](https://webkit.org/blog/15798/release-notes-for-safari-technology-preview-202/) 。在将来，我们还可以直接使用 `background-clip: border-area` 来创建渐变边框：

  


```CSS
.gradient-border {
    border: var(--b) solid #0000;
    border-radius: var(--r);
    background: var(--g) border-box;
    background-clip: border-area; /* 新增的值 */
}
```

### 使用伪元素创建渐变边框

  


[在创建多边框效果的时候](https://juejin.cn/book/7395119173656903715/section/7411340113839390783)，我就曾提到过，大部分元素都具有 `::before` 和 `::after` 伪元素，这使得每个元素都具有多个背景和边框等：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/2e97f469c3cb4d5db70f87f91a4ed36e~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1111&s=275553&e=jpg&b=f5f5f5)

  


如此一来，我们可以借助伪元素 `::before` 和（或）`::after` 使元素具有多个背景，有点类似于在 `background` 属性上使用多背景：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/63a2cc8dd17b4927bd3821f72eccd1f7~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1118&h=704&s=5478711&e=gif&f=214&b=2c2c2c)

  


正如上图所示，我们可以为元素以及它的伪元素 `::before` 或 `::after` 都设置一个背景，并给它们之间设置相应的间距，然后按顺序叠加在一起，就可以轻易构建出我们所需要的渐变边框。注意，对于普通的渐变边框，我们只需要一个伪元素即可：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/8aa162f8e1b547acbb51e509dab9043b~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1162&h=700&s=4816037&e=gif&f=248&b=2c2c2c)

  


具体的 CSS 代码如下：

  


```CSS
.gradient--border {
    --b: round(0.5em, 1px);
    --r: round(.25em, 1px);
    --s: 40vw;
    --c: red;
    --a: 180deg;
     --bg: color-mix(in srgb, var(--c), #000 60%);
    --g1: linear-gradient(var(--bg) 0 100%);
    --g2: conic-gradient(from var(--a) in oklch longer hue, var(--c) 0 0);
    
    border-radius: var(--r);
    background: var(--g1);
    position: relative;
    
    &::after {
        content: "";
        position: absolute;
        inset: calc(-1 * var(--b));
        background: var(--g2);
        pointer-events: none;
        z-index: -1;
        border-radius: calc(var(--r) + var(--b));
    }
}
```

  


我们对 `::after` 进行绝对定位，并使用 `inset` 将沿着元素每一侧向外延伸 `--b` 的距离，这将在视觉上创建了一个厚度为 `--b` 的“边框”。然后将 `z-index` 属性的值设置为 `-1` ，将伪元素 `::after` 放置在元素内容的底部。

  


需要注意的是，我们需要给元素设置一个纯色或渐变色，并且不能带有任何透明度，否则将会漏出位于底部的伪元素 `::after` 。另外就是，你可能已经注意到了，元素和伪元素`::after` 都设置了 `border-radius` 属性的值，并且伪元素 `::after` 的 `border-radius` 属性的值是 `--r` 和 `--b` 的和，这样做可以使嵌套圆角半径保持协调，在我们这个示例中，不管是改变边框厚度 `--b` 还是调整元素半径 `--r` ，都能使嵌套圆角在视觉美观上保持最佳协调的状态。如果你对这方面知识感兴趣的话，[可以点击这里获取更多相关的信息](https://juejin.cn/book/7395119173656903715/section/7401099842379841545#heading-1)。

### 使用 `mask` 创建渐变边框

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/a03f024e310a4c488e279420bb2aaba4~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1280&h=796&s=102706&e=jpg&b=565874)

  


前面我们已经介绍了三种制作渐变边框的方法，通常情况之下，它们当中的任何一种都能满足你的需要，但在一些特定情况之下，例如制作一个类似胶囊类或圆形镂空的渐变边框，它们就显得有点棘手了：

  


-   `border-image` 制作渐变边框非常简单，但我们无法用这种方法实现胶囊形状或圆形形状，因为 `radial-gradient` 和 `border-image` 不能很好的配合，它们各自工作，但无法一起使用。即便可以通过 `clip-path: inset(0 round var(--r))` 获得一个小的圆角，但这个圆角的大小必须不超过边框宽度，否则会显得很奇怪
-   使用 `background` 创建渐变边框，虽然可以轻易实现胶囊或圆形形状的渐变边框，但内容框（`content-box`）必须是一个实色，否则将会漏出位于底部 `border-box` 框中的渐变。我们无法制作一个镂空的渐变边框
-   使用伪元素 `::before` 或 `::after` 创建渐变边框与 `background` 类似，我们需要给元素自己设置一个实色背景，否则将会漏出位于底部伪元素的渐变色。同样无法制作一个镂空的渐变边框

  


庆幸的是，我们可以将伪元素 `::before` 或 `::after` 与[ CSS 的遮罩 `mask`](https://juejin.cn/book/7223230325122400288/section/7259668885224456252#heading-9) 结合起来使用，这样就可以实现一个镂空的渐变边框。

  


CSS 的 `mask` 属性使用与 `background` 属性非常相似，它的 `mask-origin` 和 `mask-clip` 与 `background-origin` 和 `background-clip` 工作原理是相同的，只不过它们用于控制遮罩图的覆盖区域。

  


具体来说，使用 `mask-image` 属性定义两个遮罩层，它们可以是 `linear-gradient()` 创建的实色（即完全不透明的层，这意味着它的透明度值在任何地方都是 `1`）。并且将第一层覆盖整个 `border-box` ，第二层则限制在 `padding-box` ，这个可以通过 `mask-clip` 做到。

  


现在，底层在整个 `border-box` 上是完全不透明的，而上面的一层在 `padding-box` 区域是完全不透明的，在边框区域（即 `padding-box` 和 `border-box` 之间的区域）是完全透明的。[我们可以通过 `mask-composite` 属性设置为 `exclude` 对这两个遮罩层进行合成操作](https://juejin.cn/book/7223230325122400288/section/7259668885224456252#heading-16)。

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/94f3249df8e74d949776bcbbf299780d~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1139&s=349235&e=jpg&b=2b2b2b)

  


正如上图所示，最底层的遮罩图在整个 `border-box` 区域的透明度为 `1` ，第二层叠加在第一层上，在 `padding-box` 区域内的透明度为 `1` ，而在 `padding-box` 和 `border-box` 之间的边框区域透明度为 `0` 。

  


这个操作的名称（差集`exclude` ，等价于 `xor`，表示只显示前面的遮罩图像与后面的遮罩图像不相交的部分）在透明度为 `0` 或 `1` 的情况下非常形象，就像我们例子中那样——第一层的透明度在任何地方都是 `1`，而我们放在第一层上的第二层在 `padding-box` 内的透明度为 `1`，在边框区域透明度为 `0`。

  


在这种情况下，布尔逻辑的规则很直观——对两个相同的值进行 “差集” 运算（`xor` 或 `exclude`）会得到 `0`，而对两个不同的值进行“差集”运算会得到 `1`。

  


在 `padding-box` 区域，第一层和第二层的透明度都是 `1`，因此合成后的结果在这个区域的透明度为 `0`。然而，在边框区域（即 `padding-box` 之外，但在 `border-box` 之内），第一层的透明度为 `1`，而第二层的透明度为 `0`，因此合成后的结果在这个区域的透明度为 `1`。

  


这个互动演示通过允许你切换视图，展示了这两层遮罩分别在 3D 中的效果以及它们叠加后通过此操作合成的效果。

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/80856034eeb2404992c04cec4ce6a522~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1234&h=812&s=4578198&e=gif&f=132&b=2c2c2c)

  


对应的 CSS 代码如下：

  


```CSS
.mask {
    --g: linear-gradient(#fff, #fff);
    border: var(--b) solid #0000;
    mask: var(--g) padding-box, var(--g) border-box;
    mask-composite: exclude;
}
```

  


同样的，你也可以将 `border` 替换为 `padding` ，同时调整 `mask-clip` 属性的值：

  


```CSS
.mask {
    --g: linear-gradient(#fff, #fff);
    padding: var(--b);
    mask: var(--g) content-box, var(--g) padding-box;
    mask-composite: exclude;
}
```

  


注意，这里只是介绍了 `mask` 创建渐变边框的工作原理，如果你想更深入的了解 `mask` ，[可以点击这里获取更全面的信息](https://juejin.cn/book/7223230325122400288/section/7259668885224456252#heading-9)！

  


另外一点需要知道的是，使用 `mask` 创建渐变边框时，需要与一个带有渐变颜色的伪元素相互结合。因为上面这段 `mask` 只会显示边框区域（或内距区域），元素其他部分将会被隐藏。因此不能直接用于元素上，否则元素上其他内容将不可见。具体的 CSS 代码如下：

  


```CSS
.gradient--border {
    --b: round(0.5em, 1px);
    --r: round(5em, 1px);
    --s: 40vw;
    --c: red;
    --a: 180deg;
     
    --m: linear-gradient(var(--c) 0 100%);
    --g: conic-gradient(from var(--a) in oklch longer hue, var(--c) 0 0);
    
    position: relative;
    border-radius: calc(var(--r) - var(--b));

    &::after {
        content: "";
        position: absolute;
        inset: calc(-1 * var(--b));
        background: var(--g);
        pointer-events: none;
        z-index: -1;
        border-radius: var(--r);
        padding: var(--b);
        mask: var(--m) content-box, var(--m) padding-box;
        mask-composite: exclude;
    }
}
```

> Demo 地址：https://codepen.io/airen/full/ZEddMGN

  


注意，我在这个示例中选择了 `padding` 来模拟边框厚度，那是因为使用 `border` 的话，`mask` 在元素四个顶点处的渐变效果不佳：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/69b9b47358c343e181317acc861c928f~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=958&s=211978&e=jpg&b=050505)

  


我们也可以调整遮罩层和遮罩层合成操作，例如在 `content-box` 区域填充一个完全透明的遮罩层 `linear-gradient(#0000 0 100%)` ，同时在 `padding-box` 区域填充一个完全不透明的遮罩层 `linear-gradient(#fff 0 100%)` ，并将 `mask-composite` 设置为 `intersect` （交集），同样可以创建一个渐变边框的效果：

```CSS
        position: absolute;
        inset: calc(-1 * var(--b));

        pointer-events: none;
```

我想你现在已经掌握了 CSS 创建渐变边框效果的多种方式，接下来我们一起来看几个实际生产中的创建渐变边框的案例。

  


### 创建带有渐为边框的按钮

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/c83af2d1d9ba48758d0fb2c75f23fc65~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1109&s=1184996&e=jpg&b=080808)

  


首先以按钮为例。因为，在 Web 上按钮是最常见的 UI 组件之一，很多时候为了满足设计需要，你可能要创建一个带有渐变边框效果的按钮。不过，我们这个案例有一点点与众不同，按钮的文本也是带有渐变效果的。

  


通常情况之下，我们会使用 `<button>` 元素来创建按钮。在我们这个示例中，`<button>` 元素有一个 `data-icon` 属性，它的值是一个表情符，例如 `👻` 。它还在 `style` 属性中设置了一个自定义属性 `--slist` ，用于指定颜色列表：

  


```HTML
<button data-icon="👻" style="--slist: #ffda5f, #f9376b">boo!</button>
```

  


我们使用 CSS Grid 给按设置布局效果，要是你喜欢，也可以使用 CSS Flexbox 来布局，使文本和图标对齐得更加美观：

  


```CSS
button {
    display: inline-grid;
    place-content: center;
    width: clamp(15ch, 10vw, 30ch);
    place-self: center;
    gap: .5em;
    grid-auto-flow: column;
    font: inherit;
    text-transform: uppercase;
    white-space: nowrap;
    cursor: pointer;
    
    &::after {
        content: attr(data-icon);
    }
}
```

  


我们还可以为按钮设置 `border` 、`padding` 和 `border-radius` ，并使用 `--slist` 自定义属性来生成渐变，该渐变填充整个 `border-box` 区域：

  


```CSS
button {
    /* CSS ... */
    --b: round(0.25em, 1px);
    --r: calc(infinity * 1px);
    --grad: linear-gradient(to right bottom, var(--slist));
    
    border-radius: var(--r);
    padding: 1em 1.5em;
    border: solid var(--b) #0000;
    background: var(--grad) border-box;
}
```

  


这个时候，你看到的按钮效果如下：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/ddc544649c9a44d5a0872922a9d90200~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=911&s=504238&e=jpg&b=050505)

  


接着，使用 `mask` 来给按钮添加渐变边框效果，并且在这里使用 `mask-clip` 的 `text` 来创建渐变文本效果。由于 `mask-clip` 未得到 Firefox 浏览器支持（在 Chrome、Safari 和 Edge 表现完美），[我在这里使用 `@supports` 来做判断](https://juejin.cn/book/7223230325122400288/section/7258822669154877480)，只有支持 `mask-clip` 的浏览器才会应用相应的遮罩效果：

  


```CSS
button {
    /* CSS ... */
    @supports (mask-clip:text) {
        -webkit-text-fill-color: transparent;
        mask: var(--m), var(--m), var(--m);
        -webkit-mask-clip: text, padding-box, border-box;
        mask-composite: exclude;
    }
}
```

  


在支持 `mask-clip:text` 的浏览器（如 Chrome 、Safari 和 Edge）中你将得到一个真正的胶囊形状的渐变透明按钮。按钮的背景是透明的，可以透过按钮看到后面的背景图片，并且边框和内部的文本、图标呈现连续的柔和渐变。在 Firefox 中，按钮则呈现基础样式，带有渐变的背景。

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/e7eb299d7b274c4aad1ad802172a5fc2~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=539&s=441337&e=jpg&b=060606)

```CSS
button {

    --r: calc(infinity * 1px);
    --m: linear-gradient(#fff 0 0);
    --grad: linear-gradient(to right bottom, var(--slist));
    display: inline-grid;
    place-content: center;
    width: clamp(15ch, 10vw, 30ch);
    place-self: center;
    gap: 0.5em;
    grid-auto-flow: column;
    font: inherit;

    border: solid var(--b) #0000;
    background: var(--grad) border-box;
```

```CSS
    background-clip: text;
    position: relative;
    
    &::before {
        content: "";
        position: absolute;
        inset: calc(-1 * var(--b));
        pointer-events: none;

        border: inherit;
        border-radius: inherit;
        background: inherit;
        background-clip: border-box;
        mask: var(--m) padding-box exclude, var(--m);
    }
    
    &::after {
        content: attr(data-icon);
    }
}
```

  


这种方案能使所有现代浏览器获得一致的效果：

```CSS
button {
    --b: round(0.25em, 1px);
    --r: calc(infinity * 1px);
    --grad: linear-gradient(to right bottom, var(--slist));
    display: inline-grid;
    place-content: center;
    width: clamp(15ch, 10vw, 30ch);
    place-self: center;
    gap: 0.5em;
    grid-auto-flow: column;
    font: inherit;
    text-transform: uppercase;
    white-space: nowrap;
    cursor: pointer;
    padding: 1em 1.5em;
    border-radius: var(--r);
    border: solid var(--b) #0000;
    background: var(--grad) border-box;
    -webkit-text-fill-color: transparent;
    background: 
        var(--grad) border-box, 
        linear-gradient(hsl(0 0% 2%) 0 0), /* 按钮的填充颜色 */
        var(--grad) border-box;
    background-clip: text, padding-box, border-box;

    &::after {
        content: attr(data-icon);
    }
}
```

  


这种方案实际上给按钮的 `padding-box` 区域填充了一个完全不透明的实色，例如这个示例中的 `hsl(0 0% 2%)` ，一旦你设置一个带有透明度的颜色，就是漏出底部的渐变色。

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/35018392dbc64074b129fe2cb8dafba2~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=911&s=499013&e=jpg&b=050505)

  


> Demo 地址：https://codepen.io/airen/full/LYKKXzP

  


要是你创建的不是一个胶囊按钮的话，你还可以考虑 `border-image` 来实现：

  


```CSS
button {
    --b: round(0.25em, 1px);
    --r: var(--b);
    --grad: linear-gradient(to right bottom, var(--slist));
    display: inline-grid;
    place-content: center;
    width: clamp(15ch, 10vw, 30ch);
    place-self: center;
    gap: 0.5em;
    grid-auto-flow: column;
    font: inherit;
    text-transform: uppercase;
    white-space: nowrap;
    cursor: pointer;
    padding: 1em 1.5em;
    
    border: solid var(--b) #0000;
    background: var(--grad) border-box;
    -webkit-text-fill-color: transparent;
    background-clip: text;    
    border-image: var(--grad) 1;
    clip-path: inset(0 round var(--r));

    &::after {
        content: attr(data-icon);
    }
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/405c53f1fdfd47beb5bb8e7389b73290~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=911&s=413616&e=jpg&b=050505)

  


> Demo 地址：https://codepen.io/airen/full/abggQJM

  


简单小结一下。

  


第一个方法是分层并对三个完全不透明的遮罩层进行差集合成。第一个层限制在 `border-box` ，第二层在 `padding-box` ，两者差集合成使我们获得了边框区域的渐变。然后我们将这个中间结果与第三个完全不透明的层进行差集合成，它被限制在文本上。问题是， `mask-clip` 并没有得到所有主流浏览器的支持，因此它在 Firefox 浏览器中结果并不是我们所预期的效果。

  


第二个方法使用额外的伪元素，利用前两个遮罩层创建渐变边框，并在实际元素上使用相同的渐变裁剪到文本。这种方案在所有浏览器中都能得到一致的结果，但缺点是使用了额外的伪元素。

  


第三个方法使用 `border-image`，非常简单，但我们无法用这种方法实现按钮的药丸形状。即使可以通过 `clip-path: inset(0 round var(--r))` 获得一个小的圆角，但这个圆角的大小必须不超过边框宽度，否则会显得很奇怪。

  


在未来，我们可以使用更简单的一种方法，即 `background-clip: border-area` ：

  


```CSS
button {
    border: solid var(--b) #0000;
    --g: linear-gradient(90deg, #ffda5f, #f9376b) border-box;
    background:  var(--g) text,  var(--g) border-area;
    color: transparent
}
```

  


### 给用户头像添加渐变边框

  


我们可以使用锥形渐变背景和径向渐变蒙板，可以实现令人惊叹的部分渐变边框，为用户头像组件赋予独特的风格。

  


```HTML
<figure class="avatar">
    <img src="avatar.png" alt="avatar" />
</figure>
```

  


具体的 CSS 代码如下：

  


```CSS
@layer avatar {
    @keyframes spin {
        100% {
            transform: rotate(360deg);
        }
    }
    
    .avatar {
        --thickness: 3;
        --color1: #fd0e4c, #fe9000, #fff020, #3edf4b, #3363ff, #b102b7;
        --color2: #fd004c;
        --from: 30deg;
        --distance: 46;
    
        width: 80vmin;
        aspect-ratio: 1;
        border-radius: 50%;
        padding: calc(10px + var(--thickness) * 1px);
        position: relative;
        place-self: center;
    
        &::after {
            content: "";
            border-radius: 100%;
            display: block;
            position: absolute;
            inset:0;
            height: 100%;
            aspect-ratio: 1;
            
            background: conic-gradient(
                from var(--from),
                var(--color1),
                var(--color2) calc(var(--distance) * 1%),
                transparent 0
            );
    
            mask: radial-gradient(
                farthest-side,
                transparent calc(100% - calc(var(--thickness) * 1px)),
                #fff calc(100% - calc(var(--thickness) * 1px) + 1px)
            );
        }
    
        &.playing::after {
            animation: spin 2.5s infinite linear;
        }
    
        & img {
            border-radius: 50%;
        }
    }
}
```

  


简单的解释一下上面示例中的代码。首先，使用伪元素 `::after` 给用户头像添加了一个层，该层和图片容器 `.avatar` 一样大，然后伪元素 `::after` 中运用 `conic-gradient()` 函数，绘制了一个锥形渐变的背景，此时它会遮盖住用户头像。为了能让用户头像裸露出来，就需要使用 CSS 的 `mask` 属性。在这个示例中，使用了 CSS 的径向渐变绘制了一个遮罩图，并且它刚好比 `--thickness` 小。这样就从视觉上得到了一个锥形渐变的边框效果：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/6e0dcebbd0f64906a7fc778f5ca41d3a~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1280&h=1016&s=108326&e=jpg&b=555577)

  


你现在只需要调整变量 `--thickness` 和 `--distance` 就可以给用户头像创建不同的渐变边框效果：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/b7ef5167fc9c4d51ad79e726e1f3dfb8~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1088&h=606&s=11540918&e=gif&f=273&b=4c486b)

  


> Demo 地址：https://codepen.io/airen/full/GRbbwbE

  


稍微调整一下，[借助 CSS 的 `@property` 特性](https://juejin.cn/book/7223230325122400288/section/7258870477462962236)，我们可以实现类似下图这样的一个效果：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/a7406d07c4334a9485ed99c3e532dbb6~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1110&h=668&s=6830868&e=gif&f=177&b=030303)

具体的 CSS 代码如下：

  


```CSS

    syntax: "<percentage>";
    initial-value: 0%;
    inherits: true;
}

    initial-value: 0%;
    inherits: true;
}

.avatar {
    --b: round(0.5em, 1px);
    --g: 10px;
    --c: red;
    --a: 90deg;
    --s: 30vw;
    --h: 20%;
    
    width: var(--s);
    aspect-ratio: 1;
    object-fit: cover;
    margin-inline: auto;
    border-radius: 50%;
    padding: calc(var(--b) + var(--g));
    
    --grad: conic-gradient(from var(--a) in oklch longer hue, var(--c) 0 0);
    --m1: radial-gradient(
        farthest-side,
        #000 calc(100% - var(--b) - var(--g) - 1px),
        #0000 calc(100% - var(--b) - var(--g)) calc(100% - var(--b)),
        #000 calc(100% - var(--b) + 1px)
    );
    --m2: conic-gradient(
        from var(--a),
        #0000 calc(var(--p) - var(--h)),
        #000 0 var(--p),
        #0000,
        #000 0 calc(var(--p) + var(--h)),
        #0000 0
    );
    --m3: linear-gradient(#000 0 0);
    
    background: var(--grad);
    mask: var(--m1), var(--m2), var(--m3) content-box;
    mask-composite: intersect, add;
    transition: --h 0.5s linear, scale 0.5s;
    cursor: pointer;
    
    &:hover {
        --h: 100%;
        --p: 100%;
        scale: 1.1;
    }
}
```

### 基于图像源创建渐变边框

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/4f07f7176a7f4b69a6fff2ff1616172a~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1160&h=1232&s=221808&e=jpg&b=2c213a)

  


[@Ana Tudor 在 Twitter 上分享了一个非常先进的技巧](https://x.com/anatudor/status/1788211656306114701)，基于 `src()` 、`attr()` 和 `filter()` 等函数，给图片添加一个带有从图片颜色提取出的渐变边框效果。只可惜 `src()` 在任何地方都未得到支持，而且 `attr()` 目前只能作为 `content` 的值使用。

  


不过，[我们可以基于 `mask` 和 `border-image` 来实现一个类似的效果](https://codepen.io/thebabydino/full/VwqEapG)：

  


```HTML
<figure>
    <img src="https://picsum.photos/id/75/600/600" alt="" class="figure" style="--url:url('https://picsum.photos/id/75/600/600')" />
</figure>
```

  


注意，`<img>` 的 `style` 中定义了个名为 `--url` 的变量，它的值是 `url()` 引入的一张图像，它的值与 `img` 的 `src` 属性值相同。

  


```CSS
figure {
    --b: round(1em, 1px);
    display: grid;
    
    img,
    &::after {
        grid-area: 1/ 1;
        border: solid var(--b) hsla(0, 0%, 0%, 0.001);
    }

    img {
        display: block;
        box-sizing: border-box;
        border-image: var(--url) 10%;
        padding: var(--b);
        width: 100%;
        aspect-ratio: 1;
        object-fit: cover;
        clip-path: inset(0 round var(--b));
    }

    &::after {
        --full: conic-gradient(red 0 0);
        backdrop-filter: blur(99px);
        mask: var(--full) exclude, var(--full) padding-box;
        pointer-events: none;
        content: "";
        border-radius: var(--b);
    }
}
```

  


在 `img` 上使用 `border-image` 并将图像作为边框图像源 `border-image-source` 。注意，[在伪元素上我们使用了 `backdrop-filter` 给元素的背景应用模糊滤镜效果](https://juejin.cn/book/7223230325122400288/section/7259669043622690853#heading-48)。其他 CSS 没有特殊之处，最终效果如下图所示：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/d2cbb7035c4340d0a87cdd2fb0927750~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1135&s=1344784&e=jpg&b=080808)

  


> Demo 地址：https://codepen.io/airen/full/NWZZeVZ

  


## 小结

  


我想你现在可以使用多种不同的方式来创建渐变边框效果了。需要知道的是，虽然我们在这里主要阐述的是给元素添加渐变边框，但要知道的是，我们使用课程中所介绍的技巧可以给元素添加具有纹理的边框效果。你可以将示例中的单个渐变填充换成 CSS 渐变创建的纹理，或者 SVG 创建的纹理，元素就拥有了一个纹理效果的边框。如果你感兴趣，不妨一试！

  


另外，我们还可以借助 CSS 的 `@property` 将渐变中的参数定义为变量，那么你就可以通过 `transition` 或 `animation` 给边框添加酷炫的动画效果！我们小册后续的课程中还会专门向大家介绍，CSS 如何给元素边框添加有趣的动效。敬请期待！
