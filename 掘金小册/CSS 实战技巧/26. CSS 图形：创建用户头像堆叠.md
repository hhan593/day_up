## 传统方案：通过布局创建用户头像堆叠

  


我们先从传统方案开始！

  


创建堆叠用户头像所需要的 HTML 结构很简单，大致如下：

  


```HTML
<ul class="avatars">
    <li><img src="http://i.pravatar.cc/500?img=6" alt=""></li>
    <!-- 省略其他的 HTML -->
</ul>
```

  


[使用 CSS Flexbox 给组件进行布局](https://juejin.cn/book/7161370789680250917/section/7161621092560273439)：

  


```CSS
.avatars {
    --b: round(.5em, 1px);
    --c: #252F3D;
    --bg: #485673;
    --s: round(10em, 1px);
    list-style: none outside none;
    display: flex;
    background: var(--bg);
    border-radius: calc(infinity * 1px);
    padding: calc(var(--s) / 10);
    
    img {
        display: block;
        width: var(--s);
        aspect-ratio: 1;
        border-radius: 50%;
        border: var(--b) solid var(--c);
        object-fit: cover;
        object-position: center;
    }
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/dd61878085904992a4167bf1c98198b2~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1039&s=385413&e=jpg&b=050505)

  


不难发现，到目前为止，用户头像并没有堆叠在一起。我们可以使用 `:not()` 选择器和结构性选择器 `:first-child` 或 `:last-child` 对 `li` 进行偏移：

  


```CSS
li:not(:first-child) {
    margin-left: calc(var(--s) / -4);
}

/* 或者 */
li:not(:last-child) {
    margin-right: calc(var(--s) / -4);
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/5fcdecbb1f904937b96c1b967c0ee353~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1039&s=355227&e=jpg&b=050505)

  


> Demo 地址：https://codepen.io/airen/full/dyxMmgO

  


正如你所看到的，用户头像堆叠在一起了。不过，有的时候我们需要的堆叠效果是前面的用户头像在后面的用户头像上的上面。如下图所示：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/c8960efc408842bba3f5278eabab4627~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1039&s=587371&e=jpg&b=060606)

  


对于这种变体而言，我们只需要调整每个 `li` 的 `z-index` 属性即可。从第一个 `li` 开始，相邻 `li` 的 `z-index` 逐渐变小。在这里，我们可以借助 CSS 的自定义属性，简化 CSS 的代码：

  


```HTML
<ul class="avatars" style="--num: 10;">
    <li style="--index: 1;"><img src="http://i.pravatar.cc/500?img=6" alt=""></li>
    <li style="--index: 2;"><img src="http://i.pravatar.cc/500?img=16" alt=""></li>
    <!-- 省略其他的 li -->
</ul>  
```

  


上面代码在 `ul` 中定义了一个 `--num` 的变量，它的值是 `li` 列表项总数，并且在每个 `li` 中定义了 `--index` 变量，它的值对应了 `li` 的索引值，它可以从 `0` 开始，也可以从 `1` 开始，我在这里选择了 `1` 。这个时候，只需在 CSS 中像下面这样为 `li` 设置 `z-index` 属性的值即可：

  


```CSS
li {
    z-index: calc(var(--num) - var(--index));
      
    &:hover {
        z-index: calc(var(--num, 1) + 2);
    }
}
```

  


注意，鼠标悬浮在 `li` 上时会调整 `z-index` 的值，使当前的用户头像会位于最顶层，它的 `z-index` 的值要比其他 `li` 的 `z-index` 更大。你将看到的效果如下：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/2ca75db5c89f451db3470ec7b845951e~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1266&h=668&s=1445067&e=gif&f=131&b=040404)

  


如果有需要，你可以给 `li` 添加过渡动效，比如鼠标悬浮在 `li` 上时，它会变大一点点：

  


```CSS
li {
    z-index: calc(var(--num) - var(--index));
    transition: all .2s ease-in-out;
    cursor: pointer;
      
    &:hover {
        z-index: calc(var(--num, 1) + 2);
        scale: 1.1;
    }
}
```

  


这个时候，你看到的效果如下图所示：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/251ed40c575a4e96b81df1f90e3a6726~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1116&h=610&s=2122215&e=gif&f=179&b=040404)

  


> Demo 地址：https://codepen.io/airen/full/poMyZKq

  


上面的案例展示的是 [CSS Flexbox 布局](https://juejin.cn/book/7161370789680250917/section/7161621092560273439)，并通过负 `margin` 来偏移 `li` ，从而实现用户头像堆叠的效果。接下来，我向大家展示 [CSS Grid](https://juejin.cn/book/7161370789680250917/section/7161372229123440648) 实现用户头像堆叠的效果。

  


```HTML
<ul class="avatars" style="--num: 10">
    <li style="--index: 1;"><img src="http://i.pravatar.cc/500?img=6" alt=""></li>
    <li style="--index: 2;"><img src="http://i.pravatar.cc/500?img=16" alt=""></li>
    <!-- 省略其他 li -->
</ul>
```

  


HTML 结构与之前案例相似，同样使用 CSS 自定义属性定义了列表项的总数和索引值。不过，在这里我们将使用 CSS Grid 布局来构建布局。

  


```CSS
.avatars {
    display: grid;
    grid-template-columns: repeat(calc(var(--num) * 3), calc(var(--s) / 4)) calc(var(--s) / 4);
}
```

  


使用 `grid-template-columns` 属性定义了网格列数和列宽，它定义了一个 `31` 列的网格，并且每一列的列宽为 ` calc(var(--s) / 4)` 。然后使用 `grid-column` 来放置网格项目，每一个网格项目跨越四列，并且相邻网格项目的起始列间隔为 `3` ，因为每个相邻网格项目有一个列是相互重叠的：

  


```CSS
.avatars {
    display: grid;
    grid-template-columns: repeat(calc(var(--num) * 3), calc(var(--s) / 4)) calc(var(--s) / 4);
    
    li {
        grid-column: span 4;
        grid-column-start: calc(var(--index) + var(--index) * 2 - 2);
    }
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/563e4937ccf041c894f1aa59dc10239c~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1039&s=1102787&e=jpg&b=070707)

  


注意，我们使用了 `grid-column-start` 来控制每个网格项目的起始位置，它由对应的网格线名称（这里用的是网络线的索引数值）来控制。在这里，需要用到一些简单的数学计算，来计算每个网格项目的起始网络线的索引值，即 `calc(var(--index) + var(--index) * 2 - 2)` 。

  


你可能会好奇，为什么每个项目跨越四列呢？因为，在这个示例中，相邻两个网格项目重叠部分是用户头像宽度的四分之一，即 ` calc(var(--s) / 4)` 。这样做的主要原因是易于计算，你也可以根据自己的需求来调整网格列宽。

  


与 Flexbox 布局相比，使用网格布局，不需要在网格项目中使用负的 `margin` 对网格项目 `li` 进行偏移，也能使网格项目相互重叠，这是网格布局的主要特性之一。虽然采用的方案不同，但最终结果是相似的：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/361e77a88188448686c6f5b115b48314~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1116&h=610&s=2122215&e=gif&f=179&b=040404)

  


> Demo 地址：https://codepen.io/airen/full/KKOzxLz

  


相比之下，CSS Flexbox 的灵活性要优于 CSS Grid，因为它不需要做任何的数学计算；而网格项目你需要显式设置每个网格项目的起始位置。

  


## 现代方案：CSS 遮罩或 SVG 遮罩

  


不管你使用的是 CSS Flexbox 还是 CSS Grid ，它们都有一个共同之处，那就是相邻项目之间有四分之一宽度相互重叠。如果用户头像完全不透明，那么这两种方案构建的效果都能达到期望的效果；要是用户头像带有一定的透明度，那么它们所创建的效果就略有缺陷，会漏出用户头像的重叠部分：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/f05eac7f8ae349d381dc2b4753bdf7e5~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1039&s=531286&e=jpg&b=050505)

  


这样的效果可能不是我们期望的效果。不过，我们可以使用一些现代 CSS 特性来创建用户头像堆叠效果，比如 CSS 遮罩或 SVG 遮罩。我们先来看 CSS 遮罩的方案。

  


### CSS 遮罩创建用户头像堆叠

  


CSS 的遮罩 `mask` 可以隐藏元素的部分区域，因此我们可以将 CSS 的渐变和遮罩结合使用。我们需要绘制一个椭圆来实现裁剪效果：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/9394d1ce20e74ba997608c8816df83bf~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=2580&s=610586&e=jpg&b=ffffff)

  


我们有多种裁剪方法。第一种，使用一个径向渐变 `radial-gradient()` 和一个线性渐变 `linear-gradient()` 来创建一个遮罩层：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/ebdd012dde08482594c733bab3458527~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=2580&s=268347&e=jpg&b=ffffff)

  


```CSS
.mask {
    mask: 
        radial-gradient(circle var(--r) at calc(var(--r) / -2) center,#0000 var(--r),#000 0),
        linear-gradient(#0000, #0000);
}
```

  


我们把这个 `mask` 应用到 `li` 上（除第一个 `li`）：

  


```CSS
li:not(:first-child) {
    mask: 
        radial-gradient(circle var(--r) at calc(var(--r) / -2) center,#0000 var(--r),#000 0),
        linear-gradient(#0000, #0000);
}
```

  


这个时候，除第一个用户头像之外的其他用户头像，左侧都有一个椭圆形的缺口（被 `mask` 隐藏掉了）：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/6795bf6beb85486d8624decc7700f98b~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1039&s=375615&e=jpg&b=060606)

  


接下来，与之前的案例一样，使用负的 `margin`，将它们重叠在一起：

  


```CSS
li:not(:first-child) {
    mask: 
        radial-gradient(circle var(--r) at calc(var(--r) / -2) center,#0000 var(--r),#000 0),
        linear-gradient(#0000, #0000);
    margin-left: calc(var(--r) / -2);  
    border-radius: 50%;  
}
```

  


同样的，我们可以给用户头像添加悬浮状态的效果。例如，鼠标悬浮在 `li` 上，使用户头像不带镂空效果，即 `mask` 为 `none` （或 `unset`）。另外，给 `li` 设置一个 `border-radius: 50%` ，使其变成一个圆。

  


完整的 CSS 代码如下：

  


```CSS
.avatars {
    --b: round(0.5em, 1px);
    --c: #252f3d;
    --bg: #485673;
    --r: round(3em, 1px);
    --s: calc(var(--r) * 2);
    list-style: none outside none;
    display: flex;

    background: var(--bg);
    border-radius: calc(infinity * 1px);
    padding: calc(var(--s) / 10);

    img {
        display: block;
        width: var(--s);
        aspect-ratio: 1;
        border-radius: 50%;
        border: var(--b) solid var(--c);
        object-fit: cover;
        object-position: center;
    }

    li {
        transition: all 0.2s ease-in-out;
        cursor: pointer;
        border-radius: 50%;
        
        &:not(:first-child) {
            mask: 
                radial-gradient(circle var(--r) at calc(var(--r) / -2) center,#0000 var(--r),#000 0),
                linear-gradient(#0000, #0000);
            margin-left: calc(var(--r) / -2);
        }

        &:hover {
            z-index: 2;
            scale: 1.1;
            mask: unset;
        }
    }
}
```

除此之外，我们还可以使用两个 `radial-gradient()` 和一个 `linear-gradient()` 来绘制遮罩图。

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/59a9d39b93504478825778b172d2f2d5~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2944&h=3290&s=641934&e=jpg&b=ffffff)

  


如上图所示，首先使用两个径向渐变创建圆形遮罩图，并通过 `mask-composite` 属性对它们进行交集（ `intersect` ）操作，得到一个类似“叶子”的形状。然后将它与 `linear-gradient()` 创建的矩形进行减法操作（`subtract`）。最终得到我们所需要的形状。

  


```CSS
li:not(:first-child) {
    mask: 
        linear-gradient(#000, #000),
        radial-gradient(circle var(--r) at calc(var(--r) / -2) center,#000 var(--r),#0000 0),
        radial-gradient(circle var(--r) at calc(var(--r) / 2) center,#000 var(--r),#0000 0);
    mask-composite: subtract, intersect;
}
```

  


将 `mask` 应用到 `li` 上，你将获得同等效果的用户头像：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/da193393961f4e74a45fd075082d3585~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1180&h=604&s=1666202&e=gif&f=142&b=040404)

```CSS
.avatars {
    &:nth-child(1) li:not(:first-child) {

            linear-gradient(#0000, #0000);
    }

    &:nth-child(2) li:not(:last-child) {
        mask: 
            radial-gradient(circle var(--r) at  right calc(var(--r) / -2) top 50%,#0000 var(--r),#000 0),
            linear-gradient(#0000, #0000);
          
    }    
}
```

```CSS
    initial-value: 0%;
    inherits: true;

        drop-shadow(0 0 1px #000) 
        drop-shadow(0 0 1px #000)
        drop-shadow(0 0 #000) drop-shadow(0 0 #000);

        box-sizing: content-box;
        display: block;
        height: var(--s);
        aspect-ratio: 1;
        border-radius: 50%;

        margin-block: calc(-1 * var(--s));
        translate: 0 var(--t);
        transition: --h 0.6s, --t 0.6s;
        cursor: pointer;

        &:not(:last-child) {
            margin-right: calc(-1 * var(--o));
            mask: 
                radial-gradient(
                    50% 50% at calc(150% + var(--g) - var(--o)) calc(50% - var(--h)),
                    #0000 calc(100% + var(--g)),
                    #000 calc(101% + var(--g))
                ) padding-box;
        }
    }
    
    &:hover {
        clip-path: inset(-999px 0 0);
        transition: 0s;
    }
}

}

}

    }
}
```

### SVG 遮罩创建用户头像堆叠

  


在上一节课中，我们介绍了如何使用 `<mask>` 给用户头像创建镂空的效果，该方法同样可以用在这里。我们在 `<mask>` 中使用两个 `<circle>` 元素，一个填充白色（`fill="white"`），另一个填充黑色（`fill="black"`），并且调整黑色圆的 `cx` 属性的值，使得两个圆有一个错开的位置：

  


```XML
<svg role="none" class="avatar">
    <mask id="cut">
        <circle cx="100" cy="100" r="100" fill="white" />
        <circle fill="black" cx="-60" cy="100" r="100" />
    </mask>
    <g mask="url(#cut)">
        <image preserveAspectRatio="xMidYMid slice" x="0" y="0" height="100%" width="100%" href="http://i.pravatar.cc/200?img=6" />
        <circle fill="none" stroke="rgb(0 0 0 / .8)" stroke-width="6" cx="100" cy="100" r="100" />
    </g>
</svg>
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/48e7904cc22242d3982b99c7cbfadb0c~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1015&s=186046&e=jpg&b=a0bfb0)

  


在实际项目中，该组件可能会有多个变体。大多数情况下，这将与大小相关。为了解决这个问题，最好使用 CSS 变量来处理 `<circle>` 的 `cx`、`cy` 和 `r` 值。以下是用于处理头像和 `<mask>` 大小的 CSS：

  


```CSS
.avatar {
    --size: 100px; 
    width: var(--size);
    aspect-ratio: 1;
    
    circle {
        cx: calc(var(--size) / 2);
        cy: calc(var(--size) / 2);
        r:  calc(var(--size) / 2);
      
        &[fill="black"] {
            cx: calc(var(--size) / -4);
        }
    }
}
```

  


上面的代码很简单，这里我们主要来看看如何计算黑色 `<circle>` 的 `cx` 值：

  


```
cx: calc(var(--size) / -4)
```

  


`<circle>` 的 `cx` 和 `cy` 值是从圆心开始的。这意味着使用一半的值会完全隐藏图像。

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/221d1ced491b403794a9c5193c83ecb7~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1472&s=364672&e=jpg&b=f6f5f5)

  


为了便于可视化，上图中的紫色代表白色圆（我们希望显示的区域），而轮廓线代表黑色圆（我们希望隐藏的区域）。

  


当黑色圆的 `cx` 值为 `0` 时，它已隐藏了一半的图像。我们可以调整它并使用负值。该值可以根据剪切区域的大小来确定。你可以尝试着拖动下面示例中的 `cx` 滑块来调整黑色圆的 `cx` 值，查看相应的结果：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/08cf294079134f5eb7be72a1501f8306~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1184&h=584&s=1331698&e=gif&f=163&b=97b5a4)

  


> Demo 地址：https://codepen.io/airen/full/JjgKXEZ

  


通过上面的演示，我想你已经了解了 SVG 的 `<mask>` 如何控制用户头像镂空的效果。接下来，我们来看一个具体的示例——用户头像叠加。

  


它的 HTML 结构如下：

  


```HTML
<div class="avatars">
    <svg role="none" class="avatar">
        <mask id="cut1">
            <circle cx="100" cy="100" r="100" fill="white" />
        </mask>
        <g mask="url(#cut1)">
            <image preserveAspectRatio="xMidYMid slice" x="0" y="0" height="100%" width="100%" href="http://i.pravatar.cc/200?img=6" />
            <circle fill="none" stroke="#3F51B5" stroke-width="10" cx="100" cy="100" r="100" />
        </g>
    </svg>
    <svg role="none" class="avatar">
        <mask id="cut2">
            <circle cx="100" cy="100" r="100" fill="white" />
            <circle fill="black" cx="-60" cy="100" r="100" />
        </mask>
        <g mask="url(#cut2)">
            <image preserveAspectRatio="xMidYMid slice" x="0" y="0" height="100%" width="100%" href="http://i.pravatar.cc/200?img=8" />
            <circle fill="none" stroke="#3F51B5" stroke-width="10" cx="100" cy="100" r="100" />
        </g>
    </svg>
    <svg role="none" class="avatar">
        <mask id="cut3">
            <circle cx="100" cy="100" r="100" fill="white" />
            <circle fill="black" cx="-60" cy="100" r="100" />
        </mask>
        <g mask="url(#cut3)">
            <image preserveAspectRatio="xMidYMid slice" x="0" y="0" height="100%" width="100%" href="http://i.pravatar.cc/200?img=2" />
            <circle fill="none" stroke="#3F51B5" stroke-width="10" cx="100" cy="100" r="100" />
        </g>
    </svg>
    <!-- 省略其他 SVG -->
</div>  
```

  


请注意，上面的 HTML 中，第一个 `<svg>` 的 `<mask>` 只有一个白色的 `<circle>` ，因为我们希望第一个用户头像不需要有任何的裁剪。另外，在每个 `<svg>` 中都使用 `<mask>` 定义了遮罩，并且它们的 `id` 名不同，因为我们需要为每个用户头像引用 `<mask>` 的 `id` 值。这个代码会随着用户头像数量增加而变得冗余，但从灵活度的角度来看，这个冗余是值得接受的。

  


对于 CSS 就很简单的了：

  


```CSS
.avatars {
    display: flex;
    align-items: center;
    padding: 0.5em;
    border-radius: calc(infinity * 1px);
    background: #162b2c;
}

.avatar {
    --size: 100px;
    width: var(--size);
    aspect-ratio: 1;

    circle {
        cx: calc(var(--size) / 2);
        cy: calc(var(--size) / 2);
        r: calc(var(--size) / 2);

            cx: calc(var(--size) / -4);
        }
    }
    
    &:not(:first-child) {
        margin-left: calc(var(--size) / -4);
    }
}
```

  


和之前的代码没有什么太大的差异，只是定义了两个头像之间的负边距，这个负边距，可以根据自己的需要来进行调整。最终的结果如下：
