## 创建正多边形

  


在几何学习，多边形（Polygon）是指一种由多个直线段组成的封闭形状，其中每条线段称为多边形的边，线段的交点称为多边形的顶点。多边形的边和角都可以有不同的长度和大小，形成不同的形状。而我们要创建的正多边形是一种特殊的多边形：**等角（所有内角相等）又是等边（所有边长相等）的多边形**。例如，等边三角形，正方形，正五边形，正六边形等：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/2ce676c5710a440c832c95c89ac7d138~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=7448&h=3900&s=4049968&e=jpg&b=fcfcfc)

  


你可能会问？CSS 搞这么复杂干嘛，它又有何用？其实，在现代 Web 中，Web 上的 UI 元素不再局限于规则的矩形，它可能会用到不同类型的形状，而且一些优秀的 Web 设计师会向你提供一些具有[创意的布局](https://juejin.cn/book/7161370789680250917/section/7161624198429802504)，诸如杂志风格的布局，如下图所示：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/32c25820c9cf421fb3d4c2b56e52b4bd~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1813&s=1657720&e=jpg&b=f6f2f1)

  


面对上图中的布局效果，我们就不得不掌握 CSS 创建正多边形的技巧。

  


简单地说，在 CSS 中，可以使用 [`clip-path`](https://juejin.cn/book/7223230325122400288/section/7259668885224456252#heading-1) 结合 [CSS 的三角函数](https://juejin.cn/book/7223230325122400288/section/7242216512176521277)来创建任意需要的正多边形。诸如[上节课所介绍的等边三角形](https://juejin.cn/book/7395119173656903715/section/7397319555476881460#heading-2)以及接下来要探讨的正多边形。我们以正六边形为例，下图展示了一个正六边形的 `clip-path` 的 `polygon()` 函数所需的坐标点（即正六边形顶点坐标）：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/4c6f7d1184e8467c9aba831245b94118~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1514&s=387601&e=jpg&b=13181e)

  


首先我们考虑一个正方形元素：

  


```CSS
.hexagon {
    width: 50vmin;
    aspect-ratio: 1;
}
```

  


注意，这里使用 [`aspect-ratio` 属性](https://juejin.cn/book/7223230325122400288/section/7259316040151236666)，并且设置其值为 `1` ，表示 `.hexagon` 元素的宽高比为 `1` ，即宽和高相等，这样便于我们调整元素的宽和高任意一个都可以获得正方形。

  


然后，我们以 `.hexagon` 的中心为圆心，并且以正方形的宽或高的一半为圆的半径绘制一个圆。紧接着从圆心到圆周上任意一点绘制一条线段，同时将圆周上的该点连接到水平轴（`x` 轴），我们会得到一个直角三角形：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/52623e5a2f224750b50b7105de8bc086~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1514&s=267773&e=jpg&b=13181e)

如果我们将这条线每隔相等的间隔旋转六次，我们可以绘制出一个正六边形的点：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/baddb982574643358b08bb2b62235a43~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1514&s=405968&e=jpg&b=13181e)

  


从圆心到圆周画一条线段，得到一个三角形，其中斜边等于圆的半径（`r`）：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/ccacdd4e06104215a7c51909ac70dc21~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1514&s=384475&e=jpg&b=13181e)

  


我们可以通过将 `360deg` 除以多边形的顶点数量（`6`）来获得角度 `θ` ，即 `θ = 360deg ÷ 6 = 60deg` 。或将 `2π` （弧度 `rad`）除以多边形的顶点数量（`6`）来获得角度 `θ` ，即 `θ = 2π ÷ 6 ≈ 1.0472rad` 。这样我们就获得了直角三角形的斜边长度 `r` 和斜边与底边的夹角 `θ`。如此一来，可以根据三角函数计算出在圆周上顶点的坐标（假设为 `P1`）：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/716b74919db74725aaa591f427475720~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2464&h=1671&s=566638&e=jpg&b=13181e)

  


如果你不记得[三角函数的相关公式](https://juejin.cn/book/7223230325122400288/section/7242216512176521277)，请参阅下图：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/fe0ff820933a493fb54a8a7f9c66f2fb~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1280&h=983&s=91201&e=jpg&b=2c2b47)

  


```
sin(θ) = Opposite ÷ Hypotenuse = 对边 ÷ 斜边 = BC ÷ AB = a ÷ c
cos(θ) = Adjacent ÷ Hypotenuse = 邻边 ÷ 斜边 = AC ÷ AB = b ÷ c
tan(θ) = Opposite ÷ Adjacent = 对边 ÷ 邻边 = BC ÷ AC = a ÷ b = sin(θ) ÷ cos(θ)
```

  


如此一来，正多边形每个点（`polygon()` 点坐标）就可以通过下面的公式计算出来：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/6d63e1883f7b45cd8274f6fcd425d647~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2464&h=2024&s=1104002&e=jpg&b=13181e)

  


即：

  


```
Px = r + r * cos(angle + (i * 360deg) / sides) = 50% + 50% * cos(angle + (i * 360deg) / sides);
Py = r + r * sin(angle + (i * 360deg) / sides) = 50% + 50% * sin(angle + (i * 360deg) / sides);
```

  


简单的解释一下公式中的参数：

  


-   `r` ：正多边形所在圆的半径，它等于元素宽度或高度的一半，即 `w/2` 或 `h/2` ，也就是 `50%`
-   `sides` ：控制正多边形的边数
-   `angle` ：旋转角度，也就是之前所介绍的 `θ` ，它等于 `360deg` 除以边数（`sides`），或 `2π` 除以边数（`sides`）
-   `i` ：正多边形每个点的索引值，它可以从 `0` 开始计，也可以从 `1` 开始计，如果从 `1` 开始计，最后一个点所对应的角度值为超过 `360deg`

  


我们可以使用 CSS 自定义属性来定义 `sides` 和 `angle`：

  


-   `sides` 定义为 `--s`
-   `angle` 定义为 `--a` ，它等于 ` 360deg  / var(--s)`

  


如此一来，在 CSS 中，我们可以像下面这样来定义一个正六边形：

  


```CSS
.hexagon {
    --s: 6; /* 正多边形的边数*/
    --a: 360deg / var(--s); /* 旋转角度，即 θ */
    
    /* 计算每条边对应的角度: angle + (i * 360deg) / sides, i 从 0 开始索引 */
    --a1: var(--a) + (0 * 360deg) / var(--s);  /* 60deg */
    --a2: var(--a) + (1 * 360deg) / var(--s);  /* 120deg */
    --a3: var(--a) + (2 * 360deg) / var(--s);  /* 180deg */
    --a4: var(--a) + (3 * 360deg) / var(--s);  /* 240deg */
    --a5: var(--a) + (4 * 360deg) / var(--s);  /* 300deg */
    --a6: var(--a) + (5 * 360deg) / var(--s);  /* 360deg */
    
    /* 计算每个点的坐标
     * Px = 50% + 50% * cos(angle + (i * 360deg) / sides)
     * Py = 50% + 50% * sin(angle + (i * 360deg) / sides)
    */
    --p1x: calc(50% + 50% * cos(var(--a1)));
    --p1y: calc(50% + 50% * sin(var(--a1)));
  
    --p2x: calc(50% + 50% * cos(var(--a2)));
    --p2y: calc(50% + 50% * sin(var(--a2)));
    
    --p3x: calc(50% + 50% * cos(var(--a3)));
    --p3y: calc(50% + 50% * sin(var(--a3)));
  
    --p4x: calc(50% + 50% * cos(var(--a4)));
    --p4y: calc(50% + 50% * sin(var(--a4)));
  
    --p5x: calc(50% + 50% * cos(var(--a5)));
    --p5y: calc(50% + 50% * sin(var(--a5)));
  
    --p6x: calc(50% + 50% * cos(var(--a6)));
    --p6y: calc(50% + 50% * sin(var(--a6)));
    
    --p1: var(--p1x) var(--p1y); /* P1 点坐标 */
    --p2: var(--p2x) var(--p2y); /* P2 点坐标 */
    --p3: var(--p3x) var(--p3y); /* P3 点坐标 */
    --p4: var(--p4x) var(--p4y); /* P4 点坐标 */
    --p5: var(--p5x) var(--p5y); /* P5 点坐标 */
    --p6: var(--p6x) var(--p6y); /* P6 点坐标 */
    
    /* 定义 polygon() 坐标值 */
    --shape: var(--p1), var(--p2), var(--p3), var(--p4), var(--p5), var(--p6);
    
    /* 元素尺寸 */
    --w: 50vmin;
    
    width: var(--w);
    aspect-ratio: 1;
    clip-path: polygon(var(--shape));
}
```

我们接着来看边框正多边形的创建。

  


使用 `clip-path` 创建边框正多边形方式与创建正多边形相似，只不过我们需要更多的点来定义形状：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/ab2d3308c3df45a2818dd58c79155bf8~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2464&h=2649&s=1975386&e=jpg&b=13181e)

  


注意，我们把边框正多边形外外圆上的点与内圆上相对应的点之间的距离设置为边框的厚度 `b` （例如 `P1` 和 `P8` 连接在一起的距离），这样易于在三角形中计算。

  


正如上图所示，边框正多边形的点将分为外部点，在边框外所在圆上的点，它的半径是 `r` ，即元素宽度或高度的一半，即 `50%` ；在边框内所在圆上的点，它的半径是 `r - b` ，即 `50% - b` 。这样我们就可以根据下面的公式计算出绘制边框正多边形的所有点的坐标：

  


```
/* 外部点坐标计算公式 */
Px = r + r * cos(angle + (i * 360deg) / sides) = 50% + 50% * cos(angle + (i * 360deg) / sides);
Py = r + r * sin(angle + (i * 360deg) / sides) = 50% + 50% * sin(angle + (i * 360deg) / sides);

/* 内部点坐标计算公式 */
Px = r + (r - b) * cos(angle + (i * 360deg) / sides) = 50% + (50% - b) * cos(angle + (i * 360deg) / sides);
Py = r + (r - b) * sin(angle + (i * 360deg) / sides) = 50% + (50% - b) * sin(angle + (i * 360deg) / sides);
```

  


另外，需要知道的是，外部点起点与结束点位置相等（`P1 = P7`），内部点起点与结束点位置相等（`P8 = P14`）。[这个与绘制边框三角形是一样的](https://juejin.cn/book/7395119173656903715/section/7397319555476881460#heading-3)。

  


如果使用 CSS 代码来描述的话，大致如下：

  


```CSS
.hexagon {
    --s: 6; /* 正多边形的边数*/
    --a: 360deg / var(--s); /* 旋转角度，即 θ */
    --b: 30px; /* 边框厚度 */
    
    /* 计算每条边对应的角度: angle + (i * 360deg) / sides, i 从 0 开始索引 */
    --a1: var(--a) + (0 * 360deg) / var(--s);  /* 60deg */
    --a2: var(--a) + (1 * 360deg) / var(--s);  /* 120deg */
    --a3: var(--a) + (2 * 360deg) / var(--s);  /* 180deg */
```

当你掌握了 `clip-path` 如何创建正多边形（图中应用的是正六边形）时，构建类似上图这样的 [Web 布局](https://s.juejin.cn/ds/iMGhLPt8/)，难点已不是六边形的绘制，而是[网格布局](https://juejin.cn/book/7161370789680250917/section/7161624198429802504#heading-3)。

  


> 温馨提示，如果你从未接触过 [CSS Grid 布局](https://juejin.cn/book/7161370789680250917/section/7161372229123440648)，那么建议你花点时间阅读我的另一本小册《[现代 Web 布局](https://s.juejin.cn/ds/iMGhLPt8/)》中关于 [CSS Grid 布局](https://juejin.cn/book/7161370789680250917/section/7161372229123440648)相关的内容。不然，阅读接下来的代码你将会有一定的困难！

  


让我们来看看实现上图布局所需的 HTML 结构，然后再来讨论如何设置网格。这个 HTML 结构非常简单：

  


```HTML
<div class="hex-grid__list">
    <div class="hex-grid__item">
        <div class="hex-grid__content">
            <img src="https://picsum.photos/800/800?random=1" alt="">
        </div>
    </div>
    <!-- 省略其他项 -->
</div>
```

这里多嵌套了一个 `div.hex-grid__content` ，这是因为我们确实需要一个额外的 `<div>` 来实现我们想要的视觉效果，因为需要强制这些六边形的宽高比，当我们深入 CSS 时，你会看到这一点。如果你对在 CSS 中强制设置宽高比感兴趣，请移步阅读《[CSS 的宽高比：`aspect-ratio`](https://juejin.cn/book/7223230325122400288/section/7259316040151236666)》！

  


好了，让我们深入了解这个复杂的网格系统是如何工作的。就这个示例而言，我们并无法使用像 RAM 这样的自动布局技术（即[自动换行](https://juejin.cn/book/7199571709102391328/section/7199571708838150155)），因为互连的六边形会阻止任何类型的自动布局。不过，CSS 网格布局非常有趣，它们实际上并不在乎你告诉它们要去的空间是否已经有一个元素在占据。也就是说，如果你告诉浏览器，多个元素占据同一个网格单元，它们会很高兴地这样做，但[你必须明确告诉网格你需要重叠哪些元素](https://juejin.cn/book/7161370789680250917/section/7161623932439625758)，所以自动布局是不行的。弄清楚了这一点，接下来设置网格就容易得多。

  


首先，我们需要将 `.hex-grid__list` 变成一个网格，这一点非常容易，只需要将该元素的 `display` 设置为 `grid` 即可：

  


```CSS
.hex-grid__list {
    display: grid;
}
```

  


接下来，我们需要定义网格的行和列。相对而言，网格的行需要什么样子，其实很简单。因为示例的六边形形状的顶部和底部都是平的，再加上六边形沿 `y` 轴对称，所以网格的行不需要做任何的设置，所有这些都会在我们开始将各个网格项目放置到正确的位置时自动处理。

  


然而，要弄清楚网格列怎么设置就要困难得多。再看一下六边形，将每个网格项目（即每个六边形）放置在一个列中是不行的，我尝试着将其跨越三列，即六边形的中心占用一列，每一侧各占一列：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/b3ae97c49069459ea8f39d390d169da2~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1187&s=249290&e=jpg&b=050505)

  


正如你所看到的，六边形的两边正好是中心宽度的一半。这意味着，当我们为这个网格设置列时，列的样子应该是 `1-2-1` 。[我们可以使用 `fr` 单位](https://juejin.cn/book/7161370789680250917/section/7161624007702216735)，它可以缩放容器并使网格随之调整。一个非常简单的方法是直接写出我们所需要的列，在这个示例中，每行有五个六边形：

  


```CSS
.hex-grid__list {
    display: grid;
    grid-template-columns: 1fr 2fr 1fr 2fr 1fr 2fr 1fr 2fr 1fr 2fr 1fr;
}
```

  


我们可以使用 `repeat()` 函数来重写 `grid-template-columns` ，但问题是，由于我们的单个网格项目需要占用奇数列（`3` 列），我们的整个网格列的总数也应该是奇数。因此，我们可以使用 `repeat()` 函数，只重复 `1fr 2fr` ，然后在末尾添加一个 `1fr` 。另外，我还设置了一个 [CSS 自定义属性](https://juejin.cn/book/7223230325122400288/section/7249357815410589733)（`--amount`）来控制每行的六边形数量：

  


```CSS
.hex-grid__list {
    --amount: 5;
    display: grid;
    grid-template-columns: repeat(var(--amount), 1fr 2fr) 1fr;
}
```

  


我们还可以给网格列或行之间设置一个间距，例如：

  


```CSS
.hex-grid__list {
    --amount: 5;
    display: grid;
    grid-template-columns: repeat(var(--amount), 1fr 2fr) 1fr;
    gap: 0rem 5rem;
}
```

  


定义好网格系统之后，我们就可以开始将网格项目放置到相应的位置上。示例中的每个六边形（即网格项目）需要跨越三列两行，所以我们可以使用 `grid-column: 1 / span 3` （这会使每个网格项目定位到第一列并跨越三列）和 `grid-row: 1 / span 2` （这会使每个网格项目跨越两行）。

  


```CSS
.hex-grid__item {
    grid-column: 1 / span 3;
    grid-row: 1 / span 2;
    aspect-ratio: 1; 
}
```

  


注意，上面代码中的 `aspect-ratio` 属性设置为 `1` ，它会强制网格项目的宽和高相等。

  


仅仅这样是不够的，就上面的代码并没有使网格项目按照我们期望的那样放置，它们都堆叠在一起了。为了使六边形能按照我们期望的那样排列，我们将使用 `:nth-of-type()` 结构选择器来放置网格项目。我们的问题根源在于我们需要更新 `grid-column` 和 `grid-row` 规则的起始位置。

  


同样的，我们从 `grid-row` 开始，因为它更容易解释，并且列也能很好的堆叠在上面。我们将新增一个计数器变量 `--counter` ，然后每次移动到新行时递增它。也就是说，我们需要将行位置设置为我们的计算器的两倍，因为我们的六边形所在的每个“行”实际上是跨越了两个网格行。相应的 `grid-row` 值就变成下面这样：

  


```CSS
.hex-grid__item {
    grid-column: 1 / span 3;
    grid-row: calc(var(--counter) + var(--counter)) / span 2;
    aspect-ratio: 1;
}
```

  


还有一个问题是，找不到保持跟踪我们所在行的好方法，即使是使用 [CSS 计数器特性](https://juejin.cn/book/7223230325122400288/section/7259668400379527205)也是无法做到（它的值是字符串）。因此，只能回归到最原始的方式，使用一系列 `:nth-of-type(n + <amount>)` 选择器来调整 `--counter` 的值。

## 创建星爆形状

  


星爆形状（Starburst）是一种图形设计中常见的形状，通常由多个等距的线段从中心点向外辐射而形成。它看起来像是一颗星，或者是一个爆炸的效果，因此得名“星爆”。

  


在几何学中，星爆形状属于星形多边形的一种。星形多边形是通过修改常规多边形的一些顶点位置而生成的，通常是通过改变每隔一个顶点的半径来实现的。这使得星爆形状看起来像是多边形的一种变体，但具有更加动态和突出的视觉效果。

  


在 Web 设计中，星爆形状通常用于按钮、徽标和其他界面元素。例如下图所展示的就是使用星爆形状来装饰的徽标 UI 视觉效果：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/9e2574c9ce1349c6814907a69627508b~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=2276&s=1600117&e=jpg&b=f9f3f1)

  


以往的做法是，对多个同样大小的元素或伪元素进行旋转，从而拼凑出一个类似星爆的形状。但是，今天我们将使用现代的方法来创建星爆形状。即使用 `clip-path` 来创建你想要的星爆形状。下图展示了一个星爆形状所需要的点数，即 `clip-path` 的 `polygon()` 函数中的坐标点：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/8a1e081ed14347bcad94a21e04810125~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1304&s=339572&e=jpg&b=ffffff)

  


`polygon()` 中的点仍然放置在圆上，其中圆心位于元素的中心，即 `(50%， 50%)` 位置，但使用了不同的逻辑，星爆图形顶点交替放置，奇数顶点（顶点索引号）位于外圆上，该圆的半径是元素宽度或高度的一半，即 `50%` （类似于创建正多边形一样，我们基于一个正方形元素上裁剪星爆形状，它的宽和高相等）；偶数顶点（顶点索引号）位于内圆上。注意，内圆是外圆的它小版本，假设缩小的值是 `s` ，那么内圆的半径就是 `r - s` ：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/08dbefef39124b3088591b62bc59c435~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1304&s=558480&e=jpg&b=ffffff)

  


对应的 `clip-path` 点为：

  


```CSS

    --shape: p1, p2, p3, p4, p5, p6, p7, p8, p9, p10, p11, p12, p13, p14, p15, p16, p17, p18,p19,p20;
    clip-path: polygon(var(--shape));
    
    width: 50vmin;
    aspect-ratio: 1;

```

  


我们可以根据三角函数计算出每个点的坐标值。

  


```
/* 顶点位于外圆上，其坐标点的计算公式 */
Px = r + r * cos(angle + (i * 360deg) / spikes) = 50% + 50% * cos(angle + (i * 360deg) / spikes);
Py = r + r * sin(angle + (i * 360deg) / spikes) = 50% + 50% * sin(angle + (i * 360deg) / spikes);

/* 顶点位于内圆上，其坐标点的计算公式 */
Px = r + (r - size) * cos(angle + (i * 360deg + 180deg) / spikes) = 50% + (50% - size) * cos(angle + (i * 360deg + 180deg) / spikes);
Py = r + (r - size) * sin(angle + (i * 360deg + 180deg) / spikes) = 50% + (50% - size) * sin(angle + (i * 360deg + 180deg) / spikes);
```

  


上面公式中：

  


-   `r` 是外部圆的半径，它等于元素宽度或高度的一半
-   `r - size` 是内部圆的半径
-   `size` 是外部圆和内部圆之间的偏移距离
-   `spikes` 是星爆形状位于所在圆上的顶点数
-   `i` 是当前顶点的索引，从 `0` 开始
-   `angle` 是初始角度（可以是 `0`，也可以是某个特定的角度，通常上外圆相邻两点之间的夹角）

  


注意，在外部点和内部点之间，我们有一个等于 `180deg/spikes` 的偏移量和一个等于 `size` 的距离差异。
