## HTML 结构和基础样式

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/143cc1ab7f5347809969c28f49cb48e7~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1080&s=395299&e=jpg&b=db2159)

  


从根本上来说，选项卡的用户体验是：点击一个选项卡（Tab Label），该选项卡会高亮显示（Selection Indicators），然后对应的内容区域（Tab Panel）会随之更新：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/23450e7e0f0641f3ad094993fc67268a~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1168&h=502&s=1561158&e=gif&f=178&b=76306d)

  


它对应的 HTML 结构，通常像下面这样：

  


```HTML
<div class="tabs">
    <ul class="tabs__list">
        <li class="tab__label tab__label--active">
            <a href="#workstuff">Work Stuff</a>
        </li>
        <li class="tab__label">
            <a href="#food">Food</a>
        </li>
        <!-- 省略其他 -->    
    </ul>
    <section class="tabs__panels">
        <div id="workstuff" class="tab__panel tab__panel--active">
            <!-- Content -->
        </div>
        <div id="food" class="tab__panel">
            <!-- Content -->    
        </div>    
    </section>
</div>
```

  


以往，选项卡的交互效果是通过 JavaScript 来动态切换 `.tab__label--active` 和 `.tab__panel--active` 类名，来设置当前项，从而达到切换选项面板内容的目的。

  


现在，我们可以通过结构性内容：链接 `<a>` 和 `:target` 。我可以通过 `href` 和 `id` 属性在 `<a>` 和 `<div>` 元素之间建立连接 。每个链接的哈希值都会匹配一个对应的部分，让浏览器通过锚点来切换内容。也就是说，点击一个链接会自动聚焦到 `:target` 的选项卡的内容（Tab Panel），无面使用 JavaScript。在 CSS 中，它的实现过程与[使用 `:target` 选择器实现手风琴的交互效果是类似的](https://juejin.cn/book/7199571709102391328/section/7199845944760729632#heading-16)。

  


除此之外，我们还可以通过单选按钮与 [CSS 的 `:has()` 选择器](https://juejin.cn/book/7223230325122400288/section/7224404685799555111#heading-3)来实现选项卡的交互效果。注意，同样是不需要依赖任何 JavaScript。它对应的 HTML 结构会像下面这样：

  


```HTML
<div class="tabs">
    <ul class="tabs__list">
        <li class="tab__label tab__label--active">
            <label for="tab1">Work Stuff</label>
            <input type="radio" class="sr-only" checked name="tabset" aria-controls="workstuff" id="tab1" />
        </li>
        <li class="tab__label">
            <label for="tab2">Food</label>
            <input type="radio" class="sr-only"  name="tabset" aria-controls="food" id="tab2" />
        </li>
        <!-- 省略其他 -->    
    </ul>
    <section class="tabs__panels">
        <div id="workstuff" class="tab__panel">
            <!-- Content -->
        </div>
        <div id="food" class="tab__panel">
            <!-- Content -->    
        </div>    
    </section>
</div>
```

  


注意，`<label>` 的 `for` 属性的值与 `<input type="radio" />` 的 `id` 属性的值关连在一起，并且 `<input type="radio" />` 的 `aria-controls` 的值与选项卡面板的 `id` 关连在一起。选项卡的当前状态则由 `input` 的 `checked` 来控制。还有一个重点是，选项卡标签中的所有 `input` 的 `name` 要相同。

  


如无特殊说明，接下来的案例都将使用这个 HTML 结构。 接下来，我们给它添加一些基础样式，使其看起来好看一些，并且通过 CSS 的 `:has()` 选择器来完成选项卡的交互功能。

### 方案一：伪元素 + 变换

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/6dac62a841764bd48cf4e1725e061911~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=921&s=256939&e=jpg&b=ffffff)

  


这应该是最普通的方案了，分别在 `li` 元素的 `::before` 和 `::after` 元素上应用一个倾斜变换 `skew()` ，将它们与 `label` 元素拼接在一起，从而构成一个类似梯形的选项卡。

  


```CSS
.tabs__list {
    display: flex;
    align-items: center;
    margin-inline: 2lh;
    z-index: 1;
    margin-bottom: -1px;
}

.tab__label {
    --r: round(0.25em, 1px); /* 圆角半径 */
    --a: 25deg;              /* 倾斜角度 */
    --p: 0.5em;              /* 控制内距 */
    --h: calc(1lh + 1em);    /* 控制高度 */
    --w: calc(cos(var(--a)) * var(--h)); /* 根据倾斜角度和高度计算出伪元素宽度 */
    
    display: flex;
    font-size: clamp(1rem, 3vw + 1.25rem, 1.5rem);
    line-height: 1;
    z-index: var(--index);
    
    &::before,
    &::after {
        content: "";
        background: var(--primary);
        height: var(--h);
        width: var(--w);
        display: block;
        z-index: -1;
        backface-visibility: hidden;
    }

    &::before {
        transform: skew(calc(-1 * var(--a)));
        border-radius: var(--r) 0 0 0;
        margin-right: calc(-1 * var(--w) / 2);
    }
    
    &::after {
        transform: skew(var(--a));
        border-radius: 0 var(--r) 0 0;
        margin-left: calc(-1 * var(--w) / 2);
    }
    
    label {
        color: color-mix(in srgb, var(--primary), #000 40%);
        text-shadow: 1px 1px 0 rgb(255 255 255 / 0.25);
        border-radius: var(--r) var(--r) 0 0;
    }    
}

.tabs__panels {
    overflow: clip;
    border-radius: 1lh;
    padding: 0.8lh;
    overflow-clip-margin: content-box;
    z-index: 2;
}
```

  


简单的解释一下上面代码中的几个关键点：

  


-   `1lh` 与 `line-height` 大小相等，在这个示例中，它的值等于 `font-size` ，因为我们显式设置了 `line-height` 的值为 `1` 。由于 `font-size` 使用了 [`clamp()` 函数](https://juejin.cn/book/7223230325122400288/section/7241401565653762108)，它的大小限制在 `1rem ~ 1.5rem`
-   `--h` 的值在 `1lh` 的基础上增加了两倍的 `--p` 值，因为我们在 `label` 上设置了 `padding` 的值为 `var(--p)` ，因此整个 `li` 元素的高度是 `1lh + calc(2 * var(--p))`
-   通过 `--a` （倾斜角度）、高度 `--h` 和 [`cos()` 函数](https://juejin.cn/book/7223230325122400288/section/7242216512176521277)计算出伪元素的宽度
-   伪元素 `::before` 的倾斜角度与 `::after` 元素刚好相反，其中 `::before` 是逆时针方向倾斜，所以它的角度值是负值，`::after` 是顺时针方向倾斜。由于它们进行了倾斜变换操作，形状会变形，造成与 `label` 元素之间有一定的间隙，为了避免这个间隙产生，需要对它们做一些偏移的设置，因此 `::before` 元素有一个负的 `margin-right` ，`::after` 元素有一个负的 `margin-left`

![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/316d0d868cdf4f27870104d4ac21dd70~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=921&s=510086&e=jpg&b=060606)

  


你可能已经注意到了，默认情况之下，`li` 元素在 `z` 轴的层级是由其在 HTML 文档中出现的先后顺序决定的。因此，你所看到的效果是，后面相邻的 `li` 元素的 `z-index` 要更高，因此它在 `z` 轴的上面。这就是为什么第四个 `li` 在最顶层。如果你不希望出现这样的现象，[你可以通过调整 `z-index` 的值来改变它们的顺序](https://juejin.cn/book/7199571709102391328/section/7204402059225858052)。我在这里结合了 [CSS 自定义属性 `--index` 来控制它们的在 `z` 轴的层级](https://juejin.cn/book/7223230325122400288/section/7249357815410589733)。注意，`--index` 内联在 `li` 中，而且它的索引值从高到低。调整之后的效果如下：

选项卡面板内容的切换动画是由 [CSS 的 `@keyframes`](https://juejin.cn/book/7288940354408022074/section/7295617447058407474) 和 [`animation` 控制](https://juejin.cn/book/7288940354408022074/section/7292735608995184678#heading-15)，分别定义了一个移出（`slideOutLeft`）和移出（`slideInRight`）。默认情况之下，在所有内容面板 `.tab__panel` 元素上应用了 `slideOutLeft` 动画，使它位于容器之外。只有当前选项卡（被激活的，即 [`input` 的 `:checked` 为 `true`](https://juejin.cn/book/7223230325122400288/section/7224404685799555111#heading-3)）对应的内容面板才会应用 `slideInRight` 动画。

  


我们稍微调整一下 CSS ，就可以获得一个带有边框的选项卡效果：

### 方案二：CSS 3D 变换

  


你可能会感到好奇，[CSS 的 3D 变换](https://juejin.cn/book/7288940354408022074/section/7295240572736897064#heading-8)怎么实现类似梯形的形状呢？简单的说，我们利用 [CSS 3D 变换中的透视 `perspective()` 函数](https://juejin.cn/book/7288940354408022074/section/7295240572736897064#heading-8)，创建一个单一的消失点透视空间。然后对元素进行变换处理，比如元素沿着 `x` 轴旋转一定的角度。

  


```CSS
.box {
    --perspective: 120px;
    --rotateX: 20deg;
    transform: perspective(var(--perspective)) rotateX(var(--rotateX));
    transform-origin: center top;
    perspective-origin: center top;
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/ce8ccb78cbfc43c2a4a95c156e9c4975~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1142&h=496&s=1247125&e=gif&f=173&b=040404)

  


> Demo 地址：https://codepen.io/airen/full/OJeaaMG

  


正如你所看到的，在一个 3D 空间中，沿着 `X` 轴旋转，我们可以获得近似一个梯形的形状。我们可以通过种方式来创建我们所需要的选项卡 UI 效果。由于 `transform` 会直接影响到元素的所有后代元素，因此我将考虑在一个伪元素上（例如 `::before`）做 3D 变换，从而获得所需要的 UI 效果：

  


```CSS
.tabs {
    --a: 35deg;
    --b: round(0.125em, 1px);
    --r: round(0.25em, 1px);
}

.tabs__list {
    display: flex;
    align-items: center;
    margin-inline: calc(var(--r) + 2.5lh);
    z-index: 2;
    gap: calc(cos(var(--a)) * 1lh);
}

.tab__label {
    display: flex;
    font-size: clamp(1rem, 3vw + 1.25rem, 1.5rem);
    line-height: 1;
    z-index: var(--index);
    color: color-mix(in srgb, var(--primary), #000 40%);
    text-shadow: 1px 1px 0 rgb(255 255 255 / 0.25);
    position: relative;
    padding: 0.5em calc(cos(var(--a)) * 1lh) 0.5em;

    &::before {
        content: "";
        position: absolute;
        inset: 0 0 2px;
        background: var(--primary);
        border-radius: var(--r) var(--r) 0 0;
        border: var(--b) solid currentColor;
        border-bottom: none 0;
        z-index: -1;
        
        transform: perspective(var(--perspective, 120px)) rotateX(var(--a));
        transform-origin: center top;
        perspective-origin: center top;
    }

    label {
        display: inline-flex;
        align-items: center;
        justify-content: center;
        text-transform: uppercase;
        white-space: nowrap;
        cursor: pointer;
        position: relative;
        bottom: -0.125em;
    }
}
```

### 方案三：CSS 渐变

  


[CSS 的渐变](https://juejin.cn/book/7199571709102391328/section/7199845781149810727)是无所不能的，只要应用妥当，可以使用 CSS 绘制出任何你想要的图形。同样的，我们可以使用两个[锥形渐变 `conic-gradient`](https://juejin.cn/book/7223230325122400288/section/7259668771856941111) 和一个线性渐变 `linear-gradient` 拼接现一个梯形。使其形状看起来和我们所要创建的选项卡形状是相似的：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/2a080eceebc84d119417f2973e7465ba~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1242&s=212541&e=jpg&b=ffffff)

  


关键是控制 `conic-gradient` 绘制的三角形的大小，我们可以通过角度 `--a` 和高度 `--h` 来控制其底边的宽度 `--w` 。这个过程需要使用 [CSS 的三角函数](https://juejin.cn/book/7223230325122400288/section/7242216512176521277)，根据 `--a` 和 `--h` 精确计算出 `--w` 的值：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/bc3dd537a3754e9a936af153fffed106~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=2204&s=378876&e=jpg&b=ffffff)

  


其对应的 CSS 代码如下：

  


```CSS
.box {
    --h: 1lh;   /* 高度 */
    --a: 25deg; /* 倾斜角度 */
    --w: calc(tan(var(--a)) * var(--h)); /* 三角形宽度 */
    height: var(--h);
    border-left: var(--w) solid #0000;
    border-right: var(--w) solid #0000;
    background: 
        conic-gradient(from var(--a) at left bottom,lime 0 25%,#0000 0) 0 0 / var(--w) 100% no-repeat border-box,
        conic-gradient(from calc(360deg - var(--a)) at right bottom,#0000 25%,lime 0) 100% 0 / var(--w) 100% no-repeat border-box, 
        linear-gradient(orange, orange) padding-box;
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/26b0b72b3c2e493fbc16618e6d9ec0b8~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1138&h=510&s=1207264&e=gif&f=180&b=050505)

  


我们把绘制梯形的渐变应用选项卡中：

  


```CSS
.tabs {
    --p: 1em;
    --h:calc(var(--p) + 1lh);   /* 高度 */
    --a: 25deg; /* 倾斜角度 */
    --w: calc(tan(var(--a)) * var(--h)); /* 三角形宽度 */

    /* 省略其他 CSS */
}

.tabs__list {
    display: flex;
    align-items: center;
    margin-inline: calc(var(--w) + 1lh);
    z-index: 2;    
}

.tab__label {
    display: flex;
    align-items: center;
    font-size: clamp(1rem, 1.848vw + 0.522rem, 1.5rem);
    line-height: 1;
    z-index: var(--index);
    color: color-mix(in srgb, var(--primary), #000 40%);
    text-shadow: 1px 1px 0 rgb(255 255 255 / 0.25);
    position: relative;
    width: fit-content;
    padding: var(--p);
    height: var(--h);
    border-left: var(--w) solid #0000;
    border-right: var(--w) solid #0000;
    background: 
        conic-gradient(from var(--a) at left bottom,var(--primary) 0 25%,#0000 0) 0 0 / var(--w) 100% no-repeat border-box,
        conic-gradient(from calc(360deg - var(--a)) at right bottom,#0000 25%,var(--primary) 0) 100% 0 / var(--w) 100% no-repeat border-box, 
        linear-gradient(var(--primary), var(--primary)) padding-box;
    
    &:not(:last-child) {
      margin-right: calc(-1 * var(--w));
    }
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/f24a96f3e9ea432f8c3d61e81b277311~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1236&h=552&s=3187483&e=gif&f=173&b=8e9eb4)

  


> Demo 地址：https://codepen.io/airen/full/poXQGRB

  


该方案有几个明显的缺陷：

  


-   带有边框的选项卡会使代码变得更为复杂些，你可能需要在 `::before` 和 `::after` 伪元素上使用相似的渐变创建不同大小的梯形形状，模拟出一个带有边框的效果

-   这种方案不适用于创建带有渐变效果的选项卡

对应的 CSS 代码如下：

### 方案四：CSS 遮罩

  


在介绍第三个方案的时候，提到过，它不太适用于制作带有渐变效果的选项卡。但我们将渐变绘制的图形作为 `mask` 的遮罩图，这一切都不再是问题：

  


```CSS
.tab__label {
    /* 省略其他 CSS*/

        linear-gradient(var(--primary), var(--primary)) calc(var(--r) + var(--w)) 0 / calc(100% - 2 * (var(--r) + var(--w))) var(--r) no-repeat border-box,

        radial-gradient(circle at center, var(--primary) var(--r),#0000 0) right var(--w) top 0 / calc(2 * var(--r)) calc(2 * var(--r)) no-repeat border-box,
        conic-gradient(from var(--a) at left bottom,var(--primary) 0 25%,#0000 0) 0 0 / var(--w) 100% no-repeat border-box,
        conic-gradient(from calc(360deg - var(--a)) at right bottom,#0000 25%,var(--primary) 0) 100% 0 / var(--w) 100% no-repeat border-box,
        linear-gradient(var(--primary), var(--primary)) padding-box; 
}
```

### 方案五：CSS 图片边框

  


虽然 CSS 渐变非常强大，允许你创建出很人炫酷的图形形状，但它的缺陷是，多个渐变的衔接之处着实令人感到头痛。如果你对细节有极致的要求，那么接下来的方案可能更适合你。

  


我们可以使用 CSS 的 `border-image` 来创建选项卡的效果。它的前提是需要一张类似于 Chrome 浏览器选项的图像，这个图像可以是一张 PNG ，也可以是 SVG 。但我个人推荐你使用 SVG，因为 [SVG 路径 `<path>` 元素可以用来创建任意你想要的图形](https://juejin.cn/book/7341630791099383835/section/7368317966821425190)。比如：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/7610a64e9e1847b7a744bc6f623a4920~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=905&s=89708&e=jpg&b=000000)

  


```XML
<svg width="1980" height="680" viewBox="0 0 1980 680" fill="none" xmlns="http://www.w3.org/2000/svg">
    <path d="M243.509 38.6916C252.358 15.3993 274.681 0 299.598 0H1680.4C1705.32 0 1727.64 15.3994 1736.49 38.6916L1980 679.665H0L243.509 38.6916Z" fill="black"/>
</svg>
```

  


正如上面代码所示，要是你熟悉[ SVG 的 `<path>` 的高级命令](https://juejin.cn/book/7341630791099383835/section/7368317966821425190)，你可以直接通过这些命令创建出所需要的图形。另外，即使你不熟悉 SVG 的 `<path>` ，[你也可以使用诸如 Figma 这样的图形设计软件绘制所需的形状](https://juejin.cn/book/7341630791099383835/section/7343645889896022050)，然后导出 SVG 代码或 `.svg` 文件。

  


在 Web 开发过程中，[有多种不同的方式可以将 SVG 代码应用到 Web 上](https://juejin.cn/book/7341630791099383835/section/7344089098363076620)，在这里，我将采用的是 Data URI 的方式。[将上面图形对应的 SVG 代码通过在线生成器转换为 Data URI](https://yoksel.github.io/url-encoder/)，并且作为 `border-image` 图片资源：

  


```CSS
.tab__label {
    --g: url("data:image/svg+xml,%3Csvg width='2450' height='841' viewBox='0 0 2450 841' fill='none' xmlns='http://www.w3.org/2000/svg'%3E%3Cpath d='M304.801 38.6916C313.65 15.3994 335.973 0 360.89 0H2089.11C2114.03 0 2136.35 15.3994 2145.2 38.6916L2450 841H0L304.801 38.6916Z' fill='%23af4d98'/%3E%3C/svg%3E%0A");
}
```

  


请注意代码中的 `fill='%23af4d98'` ，它对应的是 `<path>` 中的 `fill` 属性的值，[用于设置路径的填充颜色](https://juejin.cn/book/7341630791099383835/section/7349188496181887017)：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/4dfc0afffae14afdb7ff59e6182b2f68~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=3330&h=1340&s=503220&e=jpg&b=ffffff)

简单的小结一下，在这里我们一共用了六种不同的方式来创建老式 Chrome 选项卡的 UI 效果：

  


-   伪元素 `::before` 和 `::after` 结合变换 `skewY()` 函数，将三个部分拼接出一个类似梯形的形状
-   使用 CSS 3D 变换中的透视函数 `perspective()` 和 `rotateX()` 函数使元素的伪元素 `::before` 沿 `X` 轴旋转，从而在视觉上获得一个近似梯形的形状
-   使用多个渐变绘制一个梯形形状
-   将渐变绘制的梯形形状作为遮罩图，并使用 CSS 的遮罩 `mask` 隐藏元素的部分区域
-   使用 SVG 绘制所需要的图形，并将其转换为 Data URI 格式，然后用于 `border-image` 或 `mask-border-image` ，从而得到选项卡的形状。

## 创建新式 Chrome 浏览器选项卡

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/3130a262a8ab44b58480d1968f4505bc~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1600&h=840&s=141405&e=jpg&b=fefefe)

  


Chrome 浏览器选项卡的新式 UI 与老式 UI 相比，它采用了一种称为“圆角外扩”的 UI 效果。这种效果通常也称为“内弧圆角”效果。实现这种 UI 效果难的并不是圆角顶部边缘，而是底部的内弧曲线，即内凹圆角。即便如此，现代 CSS 还是提供了多种不同的实现方式。我能想到的方案，就至少有五种。接下来，我将通过实际案例，向大家展示这种五种方案是如何实现新式 Chrome 选项卡的 UI 效果——内弧圆角选项卡！

  


### 方案一：伪元素拼接

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/e1d26ed03f9142ce9acef913bbe5cdf5~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1069&s=177829&e=jpg&b=ffffff)

  


刚才提到过了，实现“圆角外扩”的 UI 效果，最棘手的是底部内凹曲线。

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/f983abf401554a2084202003433e17cf~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1069&s=187949&e=jpg&b=ffffff)

  


仔细观察，不难发现，使用一个圆（或一个扇形，即四分之一的圆）和一个正方形拼接，就能实现内凹曲线（即内弧圆角）：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/cc5a72a1023c4e339d3b7970e54ba8ae~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1666&s=253373&e=jpg&b=ffffff)

  


比较传统的做法是，使用伪元素 `::before` 和 `::after` 分别绘制一个圆形（或扇形）和一个正方形，然后利用颜色，实现一种假象的内弧度圆角效果。

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/0e4aaac67b3144f098106f77f29db86f~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1527&s=678836&e=jpg&b=9aabbf)

  


正如你所看到的，你需要正确的为伪元素设置背景色。如果选项卡都是统一的纯色背景，这一切似乎都不是太大的问题，但如果每个选项卡都是独立的背景颜色，那么配置伪元素的颜色就是一件痛苦的事情。因为颜色过多，很容易配错。另外，要是颜色是渐变色，那么伪元素相互衔接处就有明显的色差，使整个外观失去美感。

  


在这里，我借助 CSS 自定义属性来帮助我们管理选项卡的颜色：

  


```HTML

        <div id="workstuff" class="tab__panel" style="--primary: #af4d98;"></div>
```

![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/5d81f9a889464356a7fc93fd65fa42a5~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1280&h=544&s=44147&e=jpg&b=fffefe)

  


我们首先在元素周周围添加一个边框（`border`），但不包括底部边框，并在左上角和右上角应用 `border-radius` ：

  


```CSS
.tab {
    --r: 40px; /* 半径大小 */

    border: var(--r) solid #0000; /* 透明黑色 */
    border-bottom: 0;
    border-radius: calc(2 * var(--r)) calc(2 * var(--r)) 0 0;
}
```

  


接下来，我们添加第一层遮罩。我们只想显示填充区域（`padding-box`），即上图中突出显示的红色区域。

  


```CSS
.tab {
    mask: linear-gradient(#000 0 0) padding-box;
}
```

  


让我们再添加两个径向渐变，来显示那些底部的弧形。

  


```CSS
.tab {
    mask:
        radial-gradient(100% 100% at 0 0, #0000 98%, #000) 0 100% / var(--r) var(--r),
        radial-gradient(100% 100% at 100% 0, #0000 98%, #000) 100% 100% / var(--r) var(--r),
        linear-gradient(#000 0 0) padding-box;
    mask-repeat: no-repeat;
}
```

  


显示用于创建形状内弧的径向渐变。

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/89f2be1c51b94d22946be788f83d45b0~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1280&h=544&s=31546&e=jpg&b=ffffff)

  


这是完整代码的样子：

  


```CSS
.tab {
    --r: 40px; /* 半径大小 */
    border: var(--r) solid #0000; /* 透明黑色 */
    border-bottom: 0;
    mask: 
        radial-gradient(100% 100% at 0 0, #0000 98%, #000) 0 100% / var(--r) var(--r),
        radial-gradient(100% 100% at 100% 0, #0000 98%, #000) 100% 100% / var(--r) var(--r),
        linear-gradient(#000 0 0) padding-box;
    mask-repeat: no-repeat;
    border-radius: calc(2 * var(--r)) calc(2 * var(--r)) 0 0;
}
```

  


像往常一样，控制形状只需要一个变量。让我们稍微关注一下 `border-radius`：

  


```CSS
.tab {
    border-radius: calc(2 * var(--r)) calc(2 * var(--r)) 0 0;
}
```

  


注意，形状的圆角顶部边缘等于半径（`--r`）值的两倍。如果你想知道为什么需要进行计算，这是因为我们有一个透明的边框在外面，需要将半径加倍。前面图中突出显示的蓝色区域的半径等于 `2 * rR`，而同一图中突出显示的红色区域的半径等于 `2 * R - R`，即 `R`。

```CSS

    border-radius: calc(2 * var(--r)) calc(2 * var(--r)) 0 0 / var(--r);
    mask: 
        radial-gradient(var(--r) at var(--r) 0,#0000 98%,#000 101%) calc(-1 * var(--r)) 100% / 100% var(--r) repeat-x,
        conic-gradient(#000 0 0) padding-box;
}
```
