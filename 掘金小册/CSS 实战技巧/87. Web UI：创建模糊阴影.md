![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/6d3bd3631d784079a2e3607b251e1bfa~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2620&h=1542&s=3882757&e=png&b=043b71)

  


在 CSS 中，我们通常会[使用 `box-shadow`](https://juejin.cn/book/7199571709102391328/section/7199844993455325216#heading-4) 或 [`filter` 的 `drop-shadow()`](https://juejin.cn/book/7199571709102391328/section/7199844993455325216#heading-5) 给元素框添加阴影或投影。通过这些属性可以非常便利地为元素框添加阴影或投影效果，但它们也有一些局限性，比如使用它们无法直接给元素框创建渐变阴影，或者模糊效果——元素的背景在其周围被模糊，并且模糊程度逐渐减弱。我暂且将这些阴影效果称之为**模糊阴影**或**磨砂玻璃阴影**。

  


庆幸的是，我们可以利用 CSS 的其他特性来创建渐变阴影（之前《[CSS 图形：创建渐变阴影和发光效果](https://juejin.cn/book/7395119173656903715/section/7417071449766101019)》）。同样地，我们可以利用 CSS 遮罩（`mask`）、渐变和滤镜（`backdrop-filter`）来实现模糊阴影，即模糊效果取代了阴影。接下来，我们将一步步解析如何利用遮罩、渐变和滤镜来实现这一效果。

  


为了能更好的理解接下来的内容，你需要对以下基础知识有所了解：

  


-   [CSS 遮罩](https://juejin.cn/book/7223230325122400288/section/7259668885224456252#heading-9)： `mask`
-   [CSS 渐变](https://juejin.cn/book/7395119173656903715/section/7429225312224804899)
-   [CSS 滤镜](https://juejin.cn/book/7223230325122400288/section/7259669043622690853#heading-48)：`backdrop-filter`
-   [CSS 自定义属性](https://juejin.cn/book/7223230325122400288/section/7249357815410589733)
-   [CSS 阴影](https://juejin.cn/book/7199571709102391328/section/7199844993455325216)

  


我们开始吧！

  


## 我们的目标

  


我们需要给一个元素框添加模糊阴影效果（或者说磨砂玻璃阴影效果），它并不是我们所说的盒子阴影或投影，只是用模糊效果取代了阴影——元素的背景在其周围被模糊，并且模糊程度逐渐减弱。

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/88c84f17576a428ea0ccce42ed10d5e7~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1026&h=574&s=15754960&e=gif&f=148&b=f8f4f3)

  


乍一看，它与 `box-shadow` 创建的阴影效果类似，但它围绕着元素逐渐模糊。使用 `box-shadow` 或 `filter` 的 `drop-shadow()` 是无法直接实现这种效果的。好在 CSS 非常强大且灵活，我们仍然可以通过组合多个 CSS 特性来实现这个效果，甚至用一些巧妙的技巧来“破解”它。接下来，我将向你展示我是如何一步步构建这个效果的。

  


## 从 HTML 标记开始

  


我们所说的这个模糊效果通常会出现在模态框、对话框、弹出框或工具提示框底部。我将通过元素的伪元素（比如 `::before` 或 `::after`）来承接模糊效果。这也意味着，创建这个效果所需要的 HTML 结构非常简单——一个 HTML 元素即可：

  


```HTML
<div class="blur"></div>
```

  


你可以给 `.blur` 元素设定任意尺寸，在其内部插入内容或子元素，甚至可以使用完全不同的 HTML 元素。但实际上，HTML 结构并不是我们实现这个特殊效果的关键部分。

  


```CSS
.blur {
    display: grid;
    place-self: center;
    place-content: center;
    background: #fff;
    border-radius: .25em;
    width: max(300px, 40vw);
    aspect-ratio: 21 / 9;
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/c615d3f62fc149928adee300883c9d42~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2358&h=1348&s=4290824&e=png&b=f0e4e1)

  


接下来，我们给 `.blur` 添加一个伪元素 `::after` ，伪元素 `::after` 将会占据 `.blur` 元素的整个区域，不仅要完全覆盖，还要向外溢出。这样可以定义可见区域，从而让模糊效果向外扩展。

  


```CSS
.blur {
    position: relative;
    
    &::after {
        content: "";
        position: absolute;
        inset: -100px;
        z-index: -1;
        background: orange; /* test */
    }
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/0a100bb5a68144afb9c5b76e2791ef80~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2358&h=1768&s=3712521&e=png&b=f0e6e4)

  


上图中的黄色区域就是 `::after` 元素，它向 `.blur` 元素四周溢出了 `100px` ，并且放置在 `.blur` 元素的底部（由 `z-index: -1` 控制）。需要知道的是，伪元素 `::after` 显式设置了 `z-index: -1` 有可能不会使它置于父元素的底部，要是这样的话，你需要排查是什么原因使其失效。如果你对这方面知识不太了解的话，请移步阅读《[`z-index` 失效与修复](https://juejin.cn/book/7199571709102391328/section/7204402059225858052)》！

  


另外，我们可以使用 `inset` 调整它的大小和位置，它有点类似 `box-shadow` 控制阴影方向的方式（即阴影的偏移）。在这里，我们可以将它存储在一个 [CSS 自定义属性](https://juejin.cn/book/7223230325122400288/section/7249357815410589733)中，例如 `--blur-size` :

  


```CSS
.blur {
    --blur-size: 100px;
    position: relative;
    
    &::after {
        content: "";
        position: absolute;
        inset: calc(-1 * var(--blur-size));
        z-index: -1;
        background: orange; /* test */
    }
}
```

  


注意，我们使用的是负的 `inset` 值，这样数值越小，模糊效果的扩展范围就越大。现在，通过调整 `--blur-size` 的值就可以控制模糊效果的扩展范围：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/d6499ec4c99b4b5b8207620263cc3ecf~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1190&h=776&s=8430449&e=gif&f=96&b=efe4de)

  


## 使用透明渐变进行遮罩

  


上一步，我们已经为模糊阴影效果打下了基础，虽然只有看到一个“橙色”背景，还看不到任何模糊阴影效果，但接下来才是关键。

  


如果你阅读过小册之前的《[CSS 图形：创建渐变阴影和发光效果](https://juejin.cn/book/7395119173656903715/section/7417071449766101019)》，就能很快联想到，接下来我们需要用到 **[CSS 的渐变](https://juejin.cn/book/7395119173656903715/section/7429225312224804899)**和**[遮罩](https://juejin.cn/book/7223230325122400288/section/7259668885224456252#heading-9)**。我们需要组合它们来创造出淡入淡出的视觉效果。

  


熟悉 CSS 渐变的同学应该知道，CSS 渐变本质上是会渲染成一张图片，它可以用于 CSS 遮罩（`mask-image`），来隐藏元素的部分区域，从而创建各种形状。需要知道的一点是，**渐变遮罩的核心是透明度**。透明区域可以让元素的部分内容逐渐消失或显现，创造出淡入淡出的视觉效果。这一特征非常适用于我们这个效果，因为我们希望**模糊效果在靠近元素时更强烈，远离时逐渐消失**。

  


我们将使用两个渐变，一个水平方向，一个垂直方向。这样可以模拟出一个大致的矩形形状，并让其边缘逐渐消失：

  


```CSS
.blur {
    --blur-size: 100px;
    --gradient-fade: var(--blur-size) calc(100% - var(--blur-size));
    position: relative;

    &::after {
        content: "";
        position: absolute;
        inset: calc(-1 * var(--blur-size));
        z-index: -1;
        background: orange;
        mask: 
            linear-gradient(
                to top,
                transparent 0%,
                black var(--gradient-fade),
                transparent 100%
            ),
            linear-gradient(
                to left,
                transparent 0%,
                black var(--gradient-fade),
                transparent 100%
            );
    }
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/5542645f89ff4be5bd45cc00280a77b1~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1194&h=780&s=12652098&e=gif&f=149&b=efe6e0)

  


正如你所看到的，橙色区域（伪元素 `::after`）的四个角落有了淡入淡出的效果。这就是渐变遮罩所起到的作用。

  


你可能会感到好奇？简单的解释一下。示例中的两个渐变其实绘制了类似下图这样的遮罩图：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/6d29908c99074242839c7a057c216acc~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2358&h=1530&s=3571471&e=png&b=0d0d0d)

  


```CSS
.blur {
    --blur-size: 100px;
    --gradient-fade: var(--blur-size) calc(100% - var(--blur-size));
    
    &::after {
        background: 
            linear-gradient(
                to top,
                transparent 0%,
                black var(--gradient-fade),
                transparent 100%
            ),
            linear-gradient(
                to left,
                transparent 0%,
                black var(--gradient-fade),
                transparent 100%
            ); 
    }
```

  


创建观察上面的代码。我们使用了两个 `linear-gradient()` ，分别创建了垂直和水平的线性渐变：

  


-   垂直渐变（`to top`）：从底部的透明（`transparent`）过渡到中间的黑色（`black`），然后在顶部再次变为透明（`transparent`）
-   水平渐变（`to left`）：从右侧的透明（`transparent`）过渡到中间的黑色（`black`），然后在左侧再次变为透明（`transparent`）

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/2eb1b94f48f44c6eb2a3246c2e5e1e37~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2440&h=827&s=1088611&e=jpg&b=080808)

  


将它们合在一起，就创建了一张遮罩图——黑色区域在中心部分融合，形成一个大致的矩形轮廓。它将用于 `mask` ，因为 `mask` 属性决定了哪些像素可见——黑色区域可见，透明区域不可见。有关于这方面更详细的介绍，请移步阅读《[CSS 的 Clipping 和 Masking](https://juejin.cn/book/7223230325122400288/section/7259668885224456252)》!

  


注意，黑色区域将由 `--blure-size` 决定，我们调整它的值时，渐变效果也会随之变化：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/8a400479a3c546c080dada20cc4c66da~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1192&h=774&s=16432908&e=gif&f=189&b=090909)

  


这样，我们的模糊效果就能按照预期逐渐消失，而不会在边缘形成突兀的裁剪效果。

  


接下来，我们使用 `mask-composite` 对遮罩进一步优化。[`mask-composite` 是 CSS 遮罩中的一部分](https://juejin.cn/book/7223230325122400288/section/7259668885224456252#heading-16)，它可以对遮罩图进行合成处理。简单地说，它可以逐像素控制遮罩层的合成方式，使得我们能够创建出更复杂的遮罩效果。在这里我们将 `mask-composite` 的值设置为 `intersect` 。它会保留多个遮罩的重叠区域，只有同时符合两个（水平和垂直）渐变的像素才会被渲染。这样就可以锁定一个矩形形状，让模糊效果严格限定在这个范围内。这个矩形可以应用于任何没有或只有轻微圆角（`border-radius`）的 DOM 元素，以保持理想的模糊边界：

  


```CSS
.blur {
    --blur-size: 100px;
    --gradient-fade: var(--blur-size) calc(100% - var(--blur-size));
    position: relative;

    &::after {
        content: "";
        position: absolute;
        inset: calc(-1 * var(--blur-size));
        z-index: -1;
        background: orange;
        mask: 
            linear-gradient(
                to top,
                transparent 0%,
                black var(--gradient-fade),
                transparent 100%
            ),
            linear-gradient(
                to left,
                transparent 0%,
                black var(--gradient-fade),
                transparent 100%
            );
        mask-composite: intersect;
    }
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/81945a1c1b5949f195a4a3a7f350a156~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1196&h=692&s=10707477&e=gif&f=79&b=151414)

  


现在，你能看到一个类似下图的效果：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/258309f1ae3247dbbeb06b0fa1471d90~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1192&h=686&s=13592934&e=gif&f=105&b=f1e4dd)

  


现在，我们获得了一个模糊的橙色阴影效果。非常神奇的是，如果你把最初的测试代码 `background: orange` 替换为其他颜色或者渐变色，就可以轻易获得一个彩色阴影的效果：

  


```CSS
.blur {
    --blur-size: 100px;
    --gradient-fade: var(--blur-size) calc(100% - var(--blur-size));
    position: relative;

    &::after {
        content: "";
        position: absolute;
        inset: calc(-1 * var(--blur-size));
        z-index: -1;
        background: conic-gradient(red, orange, yellow, green, cyan, blue, purple, red);
        mask: 
            linear-gradient(
                to top,
                transparent 0%,
                black var(--gradient-fade),
                transparent 100%
            ),
            linear-gradient(
                to left,
                transparent 0%,
                black var(--gradient-fade),
                transparent 100%
            );
        mask-composite: intersect;
    }
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/61cdbd9d483d45f983fbad77a7e0dcc9~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1200&h=690&s=14453250&e=gif&f=111&b=f1e3db)

  


这也算得上是[创建彩色阴影或渐变阴影](https://juejin.cn/book/7395119173656903715/section/7417071449766101019)的另一个小技巧吧！

  


反过来，如果伪元素 `::after` 不显式设置 `background` 或者将 `background` 设置为透明色（`transparent`），那么元素 `.blur` 不具有任何阴影效果（或者说阴影效果不明显）：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/e27cffa18fc7414d86f627d393d6c121~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1190&h=688&s=17806163&e=gif&f=148&b=f0e2da)

  


## 逐渐模糊背景

  


接下来，我们需要利用 CSS 的另一个特性——高级滤镜 `backdrop-filter` 。通过使用 `backdrop-filter` 的高斯模糊 `blur()` 来模糊元素后面的内容：

  


```CSS
.blur {
    --blur-size: 100px;
    --gradient-fade: var(--blur-size) calc(100% - var(--blur-size));
    position: relative;

    &::after {
        content: "";
        position: absolute;
        inset: calc(-1 * var(--blur-size));
        z-index: -1;
        mask: 
            linear-gradient(
                to top,
                transparent 0%,
                black var(--gradient-fade),
                transparent 100%
            ),
            linear-gradient(
                to left,
                transparent 0%,
                black var(--gradient-fade),
                transparent 100%
            );
        mask-composite: intersect;
        backdrop-filter: blur(10px);
    }
}
```

  


`backdrop-filter` 的 `blur()` 会使位于 `::after` 元素底部的内容变得模糊：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/c86225cd711d405d8974a024beac7e93~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1194&h=688&s=15612675&e=gif&f=137&b=eee0d9)

  


到这里，我们已经创建出所需要的模糊阴影的效果。同样的，我们可以将模糊半径存储到另一个 CSS 自定义属性中，比如 `--blur-strength` ，通过它来调整模糊半径：

  


```CSS
.blur {
    --blur-size: 100px;
    --gradient-fade: var(--blur-size) calc(100% - var(--blur-size));
    --blur-strength: 10px;
    position: relative;

    &::after {
        content: "";
        position: absolute;
        inset: calc(-1 * var(--blur-size));
        z-index: -1;
        mask: 
            linear-gradient(
                to top,
                transparent 0%,
                black var(--gradient-fade),
                transparent 100%
            ),
            linear-gradient(
                to left,
                transparent 0%,
                black var(--gradient-fade),
                transparent 100%
            );
        mask-composite: intersect;
        backdrop-filter: blur(var(--blur-strength));
    }
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/e62d70801198400094b228ac3b2960e9~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1194&h=690&s=19624667&e=gif&f=178&b=eddfd8)

  


### **增添一点阴影增强效果**

  


为了让模糊背景效果更有深度，可以给元素本身（而不是伪元素 `::after`）添加半透明的黑色阴影（`box-shadow`），这样可以增强模糊区域的视觉层次感：

  


```CSS
.blur {
    --blur-size: 100px;
    --gradient-fade: var(--blur-size) calc(100% - var(--blur-size));
    --blur-strength: 10px;
    --shadow-size: 50px;
    position: relative;
    box-shadow:  0 0 var(--shadow-size) #00000099;

    &::after {
        content: "";
        position: absolute;
        inset: calc(-1 * var(--blur-size));
        z-index: -1;
        mask: 
            linear-gradient(
                to top,
                transparent 0%,
                black var(--gradient-fade),
                transparent 100%
            ),
            linear-gradient(
                to left,
                transparent 0%,
                black var(--gradient-fade),
                transparent 100%
            );
        mask-composite: intersect;
        backdrop-filter: blur(var(--blur-strength));
    }
}
```
