现代 CSS 为我们提供了很多特性，比如 CSS 滤镜 `filter` 、`backdrop-filter` 和 `filter()` 以及 CSS 混合模式 `mix-blend-mode` 或 `background-blend-mode` 等。这些特性类似于设计软件（比如 Photoshop 或 Figma）中的滤镜和混合模式功能，允许我们直接对元素进行艺术效果处理。

  


虽然这些高级特性在处理艺术效果时给我们带来了很多的便利性与灵活性，但它们在处理一些艺术效果时还是没有设计软件那么强大，比如“**渐进式灰度**”效果。庆幸的是，我们可以通过 CSS 的一些技巧来实现它。接下来，我们就来看看 CSS 是如何实现渐进式灰度效果。

## 渐进式灰度简介

  


渐进式灰度是一种视觉效果，它让图像逐渐从彩色过渡到灰度。与传统的将图像直接转为灰度不同，渐进式灰度的特点是过渡过程是渐变的、动态的。

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/d4439a7c8b4e48138d92ad77dc16dbe7~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2440&h=1492&s=4609489&e=jpg&b=e5e1e1)

  


正如你所看到的，在渐进式灰度效果中，图像的灰度并不是立即出现的，而是通过渐变逐渐显现。例如，从下到上，图像的颜色逐渐变得更灰，直到完全灰度化。这样可以营造出图像逐步“失去颜色”的感觉。

  


在一些 UI 设计中，渐进式灰度可以作为悬停效果或焦点变化的背景效果，通过渐变的灰度变化，可以为界面或图像增加层次感，使得用户的视觉焦点逐步引导到某个区域。

  


接下来，我们来看看 CSS 是如何实现渐进式灰度效果！

  


## 创建渐进式灰度效果

  


我们先从简单的灰度效果开始！假设我们有一张色彩鲜艳的图片：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/a6462ea8b29645088c6269baa7489615~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1920&h=1024&s=275227&e=jpg&b=80390a)

  


> URL:https://picsum.photos/id/402/1920/1024

  


如果要将它转换为灰色的，大家首先会想到 [CSS 的 `filter` 滤镜中的 `grayscale()` 函数](https://juejin.cn/book/7223230325122400288/section/7259669043622690853#heading-6)。我们可以给 `grayscale()` 函数传递一个 `0 ~ 1` 的值，它将会调整图片的灰度。

  


```CSS
img {
    filter: grayscale(var(--grayscale, 1));
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/42a8ddb758b24623a23aae11f5612c6c~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1856&h=2178&s=4743946&e=png&b=faf7f7)

  


> Demo 地址：https://codepen.io/airen/full/qEBPZvM

  


正如你所看到的，虽然 `grayscale()` 函数可以将彩色图片变成完全灰色的图片，但它无法使图片具有渐进式灰度效果。

  


你可能会说，`filter: grayscale()` 行不通，[是不是可以考虑 `backdrop-filter: grayscale(1)`](https://juejin.cn/book/7223230325122400288/section/7259669043622690853#heading-48) 。我们来验证一下：

  


```CSS
figure {
    position: relative;
    
    &::after {
        content: "";
        position: absolute;
        inset: 0;
        background: color-mix(in oklch, gray, transparent 70%);
        z-index: 2;
        backdrop-filter: grayscale(var(--grayscale));
    }
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/932fb8e230fd4b208763e2877fce2bbe~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1856&h=2178&s=4656492&e=png&b=413d3d)

  


> Demo 地址：https://codepen.io/airen/full/bNGoBzV

  


事实告诉我们，最终的效果与 `filter: grayscale()` 相似。

  


与此同时，我又再想，是不是将伪元素的 `::after` 的背景换成渐变，就能实现渐进式灰度效果呢？于是乎，我立马调整了上面示例的代码：

  


```CSS
figure {
    position: relative;
    
    &::after {
        content: "";
        position: absolute;
        inset: 0;
        background: linear-gradient(
            to bottom,
            gray,
            color-mix(in oklch, gray, transparent 70%)
        );
        z-index: 2;
        backdrop-filter: grayscale(var(--grayscale));
    }
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/dec8069ee2684988881df133a1e90a41~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1856&h=2178&s=3626623&e=png&b=797371)

  


> Demo 地址：https://codepen.io/airen/full/gbOGLNY

  


此路行不通！这也意思味着，直接使用 CSS 滤镜（不管是 `filter` 还是 `backdrop-filter`）是无法实现渐进式灰度效果。

  


庆幸的是，CSS 的混合模式允许我们处理一些艺术效果。CSS 混合模式有两种方式：`mix-blend-mode` 和 `background-blend-mode` 。那么，我们是不是可以考虑一个渐变覆盖在图片上面，然后通过混合模式将它们融合在一起，从而实现我们所期望的渐进式灰度效果。

  


```CSS
figure {
    position: relative;
    
    &::after {
        content: "";
        position: absolute;
        inset: 0;
        background: linear-gradient(
            to bottom,
            gray,
            color-mix(in oklch, gray, transparent 70%)
        );
        z-index: 2;
        mix-blend-mode: color;
    }
}
```

  


我们在图片上覆盖了一层从灰色到透明色的渐变层，并且应用了颜色（`color`）混合模式。颜色混合模式（`color`）将基于前景颜色的色相（Hue）和饱和度（Saturation）以及背景颜色的亮度（Luminosity）创建一个新的颜色。

  


它的计算步骤如下：

  


-   从前景颜色中提取色相（Hue）和饱和度值（Saturation），保留前景颜色的亮度（Luminosity）值
-   从背景颜色中提取亮度值，保留背景颜色的色相和饱和度值
-   将前景颜色的色相和饱和度值应用到背景颜色的亮度值上，形成新的颜色

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/e5c8343fabf74fd59cf7403a7e9868e2~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1280&h=865&s=46410&e=jpg&b=f4f4f6)

  


这样一来，我们获得了一个带有渐进式灰度的图片效果：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/4c4b02c35b364e649b0257a5a0d0f087~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1856&h=1452&s=2766134&e=png&b=faf6f5)

  


> Demo 地址：https://codepen.io/airen/full/gbOGgbb

  


如果你接触过 CSS 的混合模式的话，应该知道，`mix-blend-mode` 主要用于元素与元素之间的混合，比如上面示例中，伪元素 `::after` 与 `img` 进行混合。如果你想使一个背景图片带有渐进式灰度效果，就需要应用多背景，并且使用 `background-blend-mode` 来替代 `mix-blend-mode` ：

  


```CSS
figure {
    background: linear-gradient(
        to bottom,
        gray,
        color-mix(in oklch, gray, transparent 70%)
    ),
    url("https://picsum.photos/id/402/800/600") center / cover no-repeat;

    background-blend-mode: color;
}
```

  


最终效果是相似的：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/9e781f1e7c49425e881d0530f1889f59~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1856&h=1246&s=2084232&e=png&b=fcfafa)

  


> Demo 地址：https://codepen.io/airen/full/XJWepNY

  


温馨提示，混合层的顺序将会直接影响到最终的效果。比如，如果我们将图片放在顶部，灰色到透明的渐变放在底部，同样应用 `background-blend-mode: color` 或 `mix-blend-mode: color` ，它将获得的效果如下：

  


```CSS
figure {
    background: 
        url("https://picsum.photos/id/402/800/600") center / cover no-repeat,
        linear-gradient(
            to bottom,
            gray,
            color-mix(in oklch, gray, transparent 70%)
        );
    background-blend-mode: color;
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/0a63b5001c8649e68f7397437576eee1~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1856&h=1246&s=2009133&e=png&b=ffffff)

  


> Demo 地址：https://codepen.io/airen/full/WbNZRjj

  


这已经不是我们想要的渐进式灰度效果了！因此，再次强调一下，灰色到透明的渐变需要覆盖在图片之下，然后混合模式设置为 `color` 才能获得我们所想要的渐进式灰度效果。

  


需要知道的是，虽然我们在这里主要探讨的是渐进式灰度效果，但是我们把渐变层换成别的颜色到透明的渐变，就能轻易获得其他颜色的渐进式效果：

  


```CSS
figure {
    background:
        linear-gradient(
            to bottom,
            var(--gray, gray),
            color-mix(in oklch, var(--gray, gray), transparent 70%)
        ),
        url("https://picsum.photos/id/402/800/600") center / cover no-repeat;

    background-blend-mode: color;
}
```

  


尝试调整示例中的渐变效果，你将获得不同颜色的渐进式效果：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/9c2b6e565f264a8db1b18a1450da305b~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1080&h=652&s=8718995&e=gif&f=112&b=fffdfd)

  


> Demo 地址：https://codepen.io/airen/full/vEYegJE

  


另外，我们还可以通过 CSS 的 `@property` 注册一个自定义属性，然后通过 `@keyframes` 来调整该属性的值，从而动画化灰色到透明的渐变。这样就可以获得一个动态化的渐进式灰度效果：

  


```CSS
@property --rotate {
    syntax: "<angle>";
    inherits: false;
    initial-value: 0deg;
}

@keyframes rotate {
    from {
        --rotate: 0deg;
    }
    to {
        --rotate: 720deg;
    }
}

figure {
    background: linear-gradient(
        var(--rotate),
        var(--gray, gray),
        color-mix(in oklch, var(--gray, gray), transparent 70%)
      ),
      url("https://picsum.photos/id/402/800/600") center / cover no-repeat;

    background-blend-mode: color;
    animation: rotate 1s linear infinite;
}
```
