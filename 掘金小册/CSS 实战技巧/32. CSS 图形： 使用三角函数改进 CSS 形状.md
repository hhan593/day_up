## 从 CSS 形状开始

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/715c89d05b41473ab66377015057e6f1~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1024&h=576&s=39924&e=jpg&b=eeecdf)

  


在《[现代 CSS 创建形状指南](https://juejin.cn/book/7395119173656903715/section/7395172754053890063)》一课中，我们一起探讨了使用 CSS 的 `clip-path` 属性创建形状的技巧。在 CSS 中，我们可以使用 `clip-path` 的几个关键函数来创建一些基本形状，例如：

  


-   `inset()` 创建矩形
-   `circle()` 创建圆形
-   `ellipse()` 创建椭圆形
-   `polygon()` 创建多边形

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/3964a8227152481586442400e0a7572a~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1128&h=506&s=3728635&e=gif&f=136&b=f5f5ee)

  


> URL: https://bennettfeely.com/clippy/

  


虽然有很多基本的形状函数，可以合理地创建 CSS 形状，但仍然有诸多的限制：

  


-   只能使用一个形状函数，因为 `clip-path` 属性不支持列表值（即多个形状函数）
-   椭圆只能是水平或垂直的，不能直接使用 `ellipse()` 创建对角线椭圆或旋转椭圆
-   不能轻松地创建带有曲线的形状，除单个圆、单个椭圆或单个圆角矩形
-   要构建任意多边形，可能需要大量的 JavaScript 或其他计算来获取正确的坐标值

  


## 使用三角函数改进 CSS 形状

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/b35a1726ddce433b870567e77051566a~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1660&h=584&s=88363&e=jpg&b=2c2c48)

  


[CSS 三角函数](https://juejin.cn/book/7223230325122400288/section/7242216512176521277)是[现代 CSS](https://s.juejin.cn/ds/iA1qvm3B/) 众多新特性之一。它使得 CSS 具备了更强大的几何计算的能力，使得我们在使用 CSS 创建图形、元素定位、动画等各方面拥有更多的可能性：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/3f5afdadd1924ec48b275ccaf12a430c~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2158&h=1658&s=843733&e=jpg&b=2c2c48)

  


例如下面这个示例，我尝试了两种方式来制作正五边形。第一种方式是没有使用任何三角函数，只是在 `polygon()` 函数中重复输入五个不同的坐标值，直到它在“视觉”上看上去接近一个正五边形。第二种方式是通过三角函数 `sin()` 、`cos()` 和一个角度值来计算绘制正五边形的坐标值。

  


```CSS
/* 未使用 CSS 三角函数 */
.pentagon {
    clip-path: polygon(50% 0%, 100% 38%, 82% 100%, 18% 100%, 0% 38%);
}

/* 使用 CSS 三角函数 */
.pentagon {
    --radius: 50%;
    --center: 50%;
    clip-path: polygon(
        calc(var(--center) + (sin(0turn) * var(--radius))) calc(var(--center) - (cos(0turn) * var(--radius))),
        calc(var(--center) + (sin(0.2turn) * var(--radius))) calc(var(--center) - (cos(0.2turn) * var(--radius))),
        calc(var(--center) + (sin(0.4turn) * var(--radius)))  calc(var(--center) - (cos(0.4turn) * var(--radius))),
        calc(var(--center) + (sin(0.6turn) * var(--radius)))  calc(var(--center) - (cos(0.6turn) * var(--radius))),
        calc(var(--center) + (sin(0.8turn) * var(--radius)))  calc(var(--center) - (cos(0.8turn) * var(--radius)))
    );
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/0549a4d1ac4c40ddb4e6b84acece20b7~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1215&s=695917&e=jpg&b=050505)

  


> Demo 地址：https://codepen.io/airen/full/MWNBvGE

  


非常明显，使用 CSS 三角函数计算出来的正五边形的坐标点都在同一个圆上。这种方式创建的五边形才是真正的正五边形。

  


接下来，我将以正多边形为例，一步一步向大家阐述 CSS 三角函数是如何辅助我们构建一个正多边形。

  


> 温馨提示，如果你只关注 CSS 如何创建一个正多边形，并不想了解其背后的三角函数，那么可以移步阅读小册之前的课程《[CSS 图形：创建正多边形和星爆形状](https://juejin.cn/book/7395119173656903715/section/7397319151691235328)》！

  


首先，我们要创建的正多边形是一种特殊的多边形：**等角（所有内角相等）又是等边（所有边长相等）的多边形**。例如，等边三角形，正方形，正五边形，正六边形等：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/21dde12a1e08403cb19bed17fb0ed9f0~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=7448&h=3900&s=4049968&e=jpg&b=fcfcfc)

  


从上图中，不难发现，我们要创建一个正多边形，可以在一个圆上等距离地放置点，并用直线将它们连接起来。比如，对于一个正五边形，我们可以在一个已知半径的圆上放置五个点。这五个点就是 `polygon()` 函数中的点坐标。

  


我们都知道，一个圆的周长是 `360` 度（即 `2πr` ），所以我们可以以 `72` 度（即 `360 / n` ，其中 `n` 是正多边形的边数，对于一个正五边形而言， `n=5`）的间隔放置正五边形的每个顶点：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/74a2594497c0470fa0170f27200aa984~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1215&s=569439&e=jpg&b=050505)

  


但是，在 `polygon()` 函数中的点的坐标是由 `x` 和 `y` 两个轴组成。那么，如何根据角度和圆的半径来获得在圆上点的坐标位置呢？

  


在传统的坐标系中，我们可以通过余弦 `cos()` 函数和圆的半径来确定点的 `x` 坐标； 同样地，使用正弦 `sin()` 函数和圆的半径可以确定点的 `y` 坐标：

  


```
posX = cos(α) × r
posY = sin(α) × r
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/0ce26e6f7e674166ad5525a7906918cc~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1178&h=636&s=2258800&e=gif&f=414&b=fbfbfb)

  


> Demo 地址：https://codepen.io/airen/full/gOQOGPN

  


我们通过一个简单的示例来模拟一下正五边形顶点在圆周上的位置：

  


```HTML
<div class="pentagon" style="--n: 5;">
    <div class="point" style="--i: 1"></div>
    <div class="point" style="--i: 2"></div>
    <div class="point" style="--i: 3"></div>
    <div class="point" style="--i: 4"></div>
    <div class="point" style="--i: 5"></div>
</div>
```

  


上面代码中的 `--n` 表示正多边形的顶点数，在这个示例中，`--n` 等于 `5` ，因为它是一个正五边形，`--i` 是正五边形的顶点索引值。

  


假设正五边形顶点所在圆的半径为 `--r` 等于 `30vmin` ，那么 `.pentagon` 元素的宽高都为 `2r` ：

  


```CSS
.pentagon {
    --r: 30vmin;
    --s: 1rem; /* 顶点尺寸 */
    --a: calc(360deg / var(--n));
    
    width: calc(2 * var(--r));
    aspect-ratio: 1;
    
    .point {
        --_a: calc(var(--i) * var(--a) - 90deg);
        position: absolute;
        width: var(--s);
        aspect-ratio: 1;
        border-radius: 50%;
        background: rgb(255 193 7);
        top: calc(50% - var(--s) / 2);
        left: calc(50% - var(--s) / 2);
        translate: calc(cos(var(--_a)) * var(--r)) calc(sin(var(--_a)) * var(--r));
        z-index: 2;
    }
}
```

  


注意，我们通过 `translate` 将 `.point` （模拟正五边形的顶点）移到半径为 `--r` 的圆上：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/2c98109237494c77a57400d606103f52~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1215&s=485813&e=jpg&b=050505)

  


> Demo 地址：https://codepen.io/airen/full/YzmjYZd

  


以其中一个点为例，它的坐标计算方式如下图所示：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/62ae2155dc224e83b11026394836128c~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1215&s=482107&e=jpg&b=050505)

  


如此一来，通过 CSS 三角函数，可以计算出构建正五边形每个顶点的坐标：

  


```
n = 5
α = 360 ÷ n = 360 ÷ 5 = 72
r = 30vmin

顶点 1️⃣ 👉  i = 1
β = i × α = 1 × 72 = 72
x = cos(β) × r = cos(72deg) × 30vmin 👉 clac(cos(72deg) * 30vmin)
y = sin(β) × r = sin(72deg) × 30vmin 👉 clac(sin(72deg) * 30vmin)

顶点 2️⃣ 👉  i = 2
β = i × α = 2 × 72 = 144
x = cos(β) × r = cos(144deg) × 30vmin 👉 clac(cos(144deg) * 30vmin)
y = sin(β) × r = sin(144deg) × 30vmin 👉 clac(sin(144deg) * 30vmin)


顶点 3️⃣ 👉  i = 3
β = i × α = 3 × 72 = 216
x = cos(β) × r = cos(216deg) × 30vmin 👉 clac(cos(216deg) * 30vmin)
y = sin(β) × r = sin(216deg) × 30vmin 👉 clac(sin(216deg) * 30vmin)

顶点 4️⃣ 👉  i = 4
β = i × α = 4 × 72 = 288
x = cos(β) × r = cos(288deg) × 30vmin 👉 clac(cos(288deg) * 30vmin)
y = sin(β) × r = sin(288deg) × 30vmin 👉 clac(sin(288deg) * 30vmin)


顶点 5️⃣ 👉  i = 5
β = i × α = 5 × 72 = 360
x = cos(β) × r = cos(360deg) × 30vmin 👉 clac(cos(360deg) * 30vmin)
y = sin(β) × r = sin(360deg) × 30vmin 👉 clac(sin(360deg) * 30vmin)
```

  


上面这些点是创建正五边形所需要的所有坐标点，也是创建正五边形最为关键部分。我们可以通过 CSS 的变量来存储这些信息：

  


```CSS
.pentagon {
    --r: 30vmin;
    --n: 5;
    --a: 360deg / var(--n);
    
    
    /* 顶点 i = 1 */
    --pos1-x: calc(cos(var(--a) * 1) * var(--r));
    --pos1-y: calc(sin(var(--a) * 1) * var(--r));
    --pos1: var(--pos1-x) var(--pos1-y);
    
    /* 顶点 i = 2 */
    --pos2-x: calc(cos(var(--a) * 2) * var(--r));
    --pos2-y: calc(sin(var(--a) * 2) * var(--r));
    --pos2: var(--pos2-x) var(--pos2-y);
    
    /* 顶点 i = 3 */
    --pos3-x: calc(cos(var(--a) * 3) * var(--r));
    --pos3-y: calc(sin(var(--a) * 3) * var(--r));
    --pos3: var(--pos3-x) var(--pos3-y);
    
    /* 顶点 i = 4 */
    --pos4-x: calc(cos(var(--a) * 4) * var(--r));
    --pos4-y: calc(sin(var(--a) * 4) * var(--r));
    --pos4: var(--pos4-x) var(--pos4-y);
    
    /* 顶点 i = 5 */
    --pos5-x: calc(cos(var(--a) * 5) * var(--r));
    --pos5-y: calc(sin(var(--a) * 5) * var(--r));
    --pos5: var(--pos5-x) var(--pos5-y);
}
```

  


我们只需要将它们组合在一起，并且存储到一个名为 `--shape` 的变量中，然后再应用到 `clip-path` 的 `polygon()` 函数中：

  


```CSS
.pentagon {
    /* 省略之前的代码 */
    --shape: var(--pos1),var(--pos2),var(--pos3),var(--pos4),var(--pos5);
    clip-path: polygon(var(--shape));
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/d6a7c4325ee24953b7451581bd667afc~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1215&s=253627&e=jpg&b=050505)

  


看到这样的结果，你一定会感到困惑，为什么正五边形的顶点并没有位于所期望的圆（绿色圆）上？造成这种现象是因为三角函数的坐标系是一个极坐标系。在极坐标系中，一个点的位置是通过半径 `r` （从原点到点的距离）和角度（通常相对于正 `x` 轴的旋转角度）来决定的。而 Web 中的元素是一个笛卡尔坐标系。这意味着，我们要将三角函数的极坐标转换笛卡尔坐标。在这个转换过程中，我们有两个主要因素需要考虑：

  


-   元素的`(0,0)` 点是左上角（在这里暂不考虑书写方式等因素）
-   `y` 轴的值向下为正

  


我们在使用 `clip-path` 的 `polygon()` 函数创建一个正五边形时，需要将正五边形位于元素的中心位置，因此需要将中心点偏移 `50%` 。根据我们所需的方向，将某些 `Y` 值乘以 `-1` 可能会很有用。在这种情况下，我们主要想让元素居中，那就可以使用以下方式来修正：

  


```CSS
.pentagon {
    --r: 30vmin;
    --n: 5;
    --a: 360deg / var(--n);
    --center: 50%;
    
    /* 顶点 i = 1 */
    --pos1-x: calc(var(--center) + cos(var(--a) * 1) * var(--r));
    --pos1-y: calc(var(--center) + sin(var(--a) * 1) * var(--r));
    --pos1: var(--pos1-x) var(--pos1-y);
    
    /* 顶点 i = 2 */
    --pos2-x: calc(var(--center) + cos(var(--a) * 2) * var(--r));
    --pos2-y: calc(var(--center) + sin(var(--a) * 2) * var(--r));
    --pos2: var(--pos2-x) var(--pos2-y);
    
    /* 顶点 i = 3 */
    --pos3-x: calc(var(--center) + cos(var(--a) * 3) * var(--r));
    --pos3-y: calc(var(--center) + sin(var(--a) * 3) * var(--r));
    --pos3: var(--pos3-x) var(--pos3-y);
    
    /* 顶点 i = 4 */
    --pos4-x: calc(var(--center) + cos(var(--a) * 4) * var(--r));
    --pos4-y: calc(var(--center) + sin(var(--a) * 4) * var(--r));
    --pos4: var(--pos4-x) var(--pos4-y);
    
    /* 顶点 i = 5 */
    --pos5-x: calc(cos(var(--a) * 5) * var(--r));
    --pos5-y: calc(sin(var(--a) * 5) * var(--r));
    --pos5: var(--pos5-x) var(--pos5-y);
}
```

更为有意思的是，我们通过[ `@property` 属性](https://juejin.cn/book/7223230325122400288/section/7258870477462962236)注册 `--positive` 自定义属性，[就可以在 `@keyframes` 中动画化 `--positive` 自定义属性的值](https://juejin.cn/book/7288940354408022074/section/7295617447058407474)，使得正五边形有一个类似旋转的动画效果。

  


```CSS
@property --positive {
    syntax: "<angle>";
    initial-value: 0deg;
    inherits: true;
}

@keyframes positive {
    from {
        --positive: 0deg;
    }
    to {
        --positive: 360deg;
    }
}

.pentagon {
    animation: positive 10000ms alternate infinite ease-in-out;
}
```

除此之外，使用 CSS 还有另一个重要优点，它支持任何合法的角度单位，比如度数 `deg` ，弧度 `rad` 、梯度 `grad` 和周长 `turn` 。它们之间是可以相互换算的：

  


```
360deg = 2π rad = 400grad = 1turn
```

  


如此一来， `--a` 的值可以是多种形式：

  


```CSS
 .pentagon {
     --n: 5;
     --deg: 360deg / var(--n);
     
     --rad: (2rad * pi) / var(--n);
     
     --turn: 1turn / var(--n);
     
     --grad: 400grad / var(--n);  
 }
```

  


注意，上面代码中的 `--deg` 、`--rad` 、`--turn` 和 `--grad` 代表的是相同的弧度值，只是表达方式不同。这意味着，`--a` 可以是当中的任意一个值。

  


既然我们知道如何使用三角函数沿着半径为 `--r` 的圆创建正五边形，那么我们就可以使用同样的原理来创建其他的正多边形。换句话说，我们可以将 `--n` 设置为任何大于 `2` 的值，因为最简单的多边形就是三角形。只不过，当 `--n` 的值越大，我们所需要的 CSS 变量就越多（它代表坐标点越多）。为了方便，我们可以借助 JavaScript 脚本的能力，根据 `--n` 来动态生成 `--shape` 的值：

  


```CSS
:root {
    --r: 50%;
    --size: 200px;
    --center: 50%;;
}

.hexagon {
    width: var(--size);
    aspect-ratio: 1;
    clip-path: var(--shape);
    background: linear-gradient(45deg, deeppink, orange);
}
```

  


```JavaScript
const root = document.documentElement;
const sizeInput = document.querySelector("#size");
const sizeOutput = document.querySelector("#size-output");
const slidesInput = document.querySelector("#slides");
const slidesOutput = document.querySelector("#slides-output");
const minSides = 3;


const updateSize = (e) => {
    const size = `${e.target.value}px`;
    root.style.setProperty("--size", size);
    sizeOutput.textContent = size;
};

sizeInput.addEventListener("input", updateSize);

const generateNSides = (n) => {
    return `polygon(evenodd, ${Array.from({ length: n }, (_, i) => 
        `calc(var(--center) + (cos(${(i / n) * 360}deg) * var(--r))) 
         calc(var(--center) + (sin(${(i / n) * 360}deg) * var(--r)))`
    ).join(",")})`;
};

const generateViaCss = () => {
    return Array.from({ length: 60 - minSides + 1 }, (_, i) => 
        generateNSides(i + minSides)
    );
};

const setupInteraction = () => {
    const finalPoints = generateViaCss();
    slidesInput.max = finalPoints.length + minSides - 1;

    const updateOutput = () => {
        const step = parseInt(slidesInput.value, 10);
        root.style.setProperty("--shape", finalPoints[step - minSides]);
        slidesOutput.textContent = step;
    };

    slidesInput.addEventListener("input", updateOutput);
    updateOutput();
};

setupInteraction();
```

从这个示例的效果中，不难发现，当 `--n` 的值越大时，`clip-path` 的 `polygon()` 创建出来的形状最终看起来像是圆形或椭圆形（当元素宽高比不同时，创建出来的形状看起来像一个椭圆）。通常情况下，当 `--n` 等于 `60` 时，整个形状看起来像一个圆，但这不是绝对的。因为和元素的大小有关系，元素尺寸越大时，所需要的边数（`--n`）就越多，反之则越少。

  


也就是说，我们可以借助三角函数，可以使用 `polygon()` 创建出近似曲线的形状，从而创造出更多有趣的可能性。

```CSS
.ribbon {
    --c: 25px; /* 控制凹口 */
    --a: 15deg; /* 控制旋转角度 */

    /* 控制丝带颜色 */
    --bg: #c73225; /* 主背景色 */
    --sc: color-mix(in srgb, var(--bg), #000 25%); /* 控制丝带颜色(伪元素的背景颜色) */
    --color: color-contrast(var(--bg) vs #fff,#aefeaf,#333); /* 控制文本颜色 */

    width: fit-content;
    font-size: 1.5rem;
    line-height: 2;
    padding-inline: calc(1lh + 0.5em);
    color: var(--color);
    position: relative;
    rotate: var(--a); /* 旋转整个元素 */
    background: linear-gradient(calc(90deg - var(--a)),#0000 calc(1lh * sin(var(--a))),var(--bg) 0 calc(100% - 1lh * sin(var(--a))),#0000 0);

    &::before,
    &::after {
        content: "";
        position: absolute;
        background: var(--sc);
        height: calc(1lh / cos(var(--a)));
        width: calc((100% - 1lh * tan(var(--a))) * cos(var(--a)));
        rotate: calc(-1 * var(--a));
        translate:0 0 -1px; /* 类似于 z-index: -1 */
    }

    &::before {
        left: 0;
        top: 0;
        transform-origin: top left;
        clip-path: polygon(0 0,100% 0,calc(100% - var(--c)) 50%,100% 100%,calc(1lh / sin(var(--a))) 100%);
    }

    &::after {
        right: 0;
        bottom: 0;
        transform-origin: bottom right;
        clip-path: polygon(0 0,calc(100% - 1lh/sin(var(--a))) 0,100% 100%,0 100%,var(--c) 50%);
    }
}
```
