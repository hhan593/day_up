## 关于 CSS 锚点定位

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/6232b65ccaa746e08c366d8e7aefc45a~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1400&h=785&s=1595403&e=png&b=477891)

  


随着 Chrome 125 引入 CSS 锚点定位，将一个元素相对于另一个元素进行定位变得前所未有的简单。这是管理复杂定位的好方法。例如，在还没有 CSS 锚点定位之前，如果我们想要在另一个元素上方放置一个元素（最常见的用例就是工具提示框或弹出框），我们需要将工具提示框的 `position` 属性设置为非 `static` ，然后使用其 `inset` 或 CSS 变换属性（例如 `translate` 或 `translate()`）将其精确地放置在我们想要的位置。这种方式虽然有效，但其位置很容易受到用户滚动、缩放或动画的影响，从而导致工具提示框溢出屏幕或出现在尴尬的位置。唯一的解决方法是使用 JavaScript 检测工具提示框是否超出边界，然后用 JavaScript 来纠正它的位置。

  


然后，CSS 锚点定位提供了一种简单的接口，只需要在 CSS 中声明要将元素连接到其他元素的哪一边，就可以轻松实现相对定位。它还允许我们设置备用位置，以避免前面提到的溢出问题。例如，我们可以将工具提示框元素设置在其锚点的上方，但当上方空间不足时，可以自动将其移到锚点下方。

  


简而言之，**CSS 锚点定位是相对另一个元素定位元素的好方法**。

  


在开始创建元素连接线之前，我们先简单的了解一下 CSS 锚点定位。为了让解释更加清晰，我们将使用特定术语来说明哪个元素连接到哪个：

  


-   锚点（Anchor）：这是定位其他元素的参考物，它常被称为“锚点”
-   目标（Target）：这是一个相对于一个或多个“锚点”进行定位的绝对定位元素，即目标元素

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/4ddb0c382d034a7eade5e211e8a031fd~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1024&s=260277&e=jpg&b=ffffff)

  


我们可以将这些元素想象为两个相邻的 `<div>` 元素：

  


```HTML
<div class="anchor">anchor</div>
<div class="target">target</div>
```

  


我在 `.anchor` 内部添加了一个锚点图标作为参考，这是可选的。接下来，我们添加一些基本样式来区分“锚点”元素和目标元素：

  


```CSS
body {
    display: grid;
    place-content: center;
}

.anchor {
    background: #09face;
    border-radius: 50%;
    color: #333;
    padding:1em;
    width: 88px;
    aspect-ratio: 1;
    
    display: grid;
    place-content: center;
    cursor: pointer;
    
    svg {
        display: block;
        width: 85%;
        aspect-ratio: 1;
        place-self: center;
    }
}
  
.target {
    padding: .5em;
    background: #f50;
    color: #fefefe;
    border-radius: .25em;
    width: fit-content;
    position: relative;
    max-width: 30ch;
    text-wrap: balance;
    text-align: center;
    
    &::after {
        content: "";
        position: absolute;
        left: 50%;
        translate: -50% 0;
        width: .875rem;
        aspect-ratio: 1;
        background: #f50;
        top: 100%;
        clip-path: polygon(0% 0%, 100% 0%, 50% 100%);
    }
}
```

  


目前你看到的效果如下图所示：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/2c5f228ebd2c42c29794c5ae670f027a~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1024&s=214756&e=jpg&b=060606)

  


现在，目标元素位于锚点元素的下方。接下来，我们需要使用 CSS 锚点定位的特性来调整目标元素的位置，使其符合我们所需要的效果。

  


到目前为止，有两种方式可以实现锚点定位。第一种是使用 `anchor()` 函数和 `@position-fallback` ，这是锚点定位最初有的一个实验性功能。后来，Webkit 推出了一个新的想法，采用类似网格系统的替代方案，这种方案提供了更简单且更易用。

  


首先来看第一种方法。我们先要使用 `anchor-name` 属性将 `.anchor` 元素定义为“锚点”：

  


```CSS
.anchor {
    anchor-name: --tooltip;
}
```

  


锚点名称应该以 `--` 作为前缀，[就像定义 CSS 自定义属性一样](https://juejin.cn/book/7223230325122400288/section/7249357815410589733)，除了这个要求之外，名称可以是任何你想要的。

  


现在，它需要一个与之锚点的目标元素，这个目标元素是通过 `position-anchor` 属性来设置的：

  


```CSS
.target {
    position-anchor: --tooltip;
}
```

  


还没有看到任何变化，这是因为我们还没有定位元素。为此，我们将 `.target` 元素的 `position` 设置为 `absolute` ，并引入 `anchor()` 函数来进行定位：

  


```CSS
.target {
    position-anchor: --tooltip;
    position: absolute;
    bottom: calc(anchor(top) + 1rem);
    justify-self: anchor-center;
}
```

  


上面代码的意思是，将 `.target` 目标元素放置在距离“锚点”元素顶部 `1rem` 处，并且水平居中。其中 `anchor()` 函数可以精确定位目标元素，`anchor-center` 值配合 `jusify-self` 属性使用，将目标元素水平居中。

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/0e45d2effac542e88002c16724811606~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1030&s=210311&e=jpg&b=050505)

  


> Demo 地址：https://codepen.io/airen/full/oNKYWMd

  


`anchor()` 函数除了可以使用 `top`、`right` 、`bottom` 和 `left` 关键词之外，还可以使用百分比，例如将 `top` 设置为 `anchor(20%)` 。它还可以使用 [CSS 逻辑属性和逻辑值](https://juejin.cn/book/7223230325122400288/section/7254820664442355773)，比如：

  


```CSS
.target {
    position-anchor: --anchor-name;
    position: absolute;
    inset-block-start: anchor(center);
    inset-inline-start: anchor(end);
}
```

  


这种方式可以根据书写模式来定位元素。

  


另外，`anchor()` 函数还可以用来告诉你的定位元素应该相对于哪个锚点来定位。换句话说，你可以在 `anchor()` 函数中同时定义 `anchor-name` 和要定位的边缘，确保元素之间的完美配合：

  


```CSS
.target {
    position: absolute;
    bottom: anchor(--anchor-name top);
    right: anchor(--anchor-name left);
}
```

  


它使我们可以将一个元素同时锚定到其他的两个元素上，即多锚点定位。例如：

  


```HTML
<div class="anchor"></div>
<div class="anchor"></div>
<div class="target"></div>
```

  


我们分别给 `.anchor` 分配一个锚点名，以便 `.target` 元素可以引用：

  


```CSS
.anchor {
    &:nth-child(1) {
        anchor-name: --anchor-element-one;
    }
    
    &:nth-child(2) {
        anchor-name: --anchor-element-two;
    }
}
```

  


现在，目标元素 `.target` 可以引用锚点一（`--anchor-element-one`）和锚点二（`--anchor-element-tow`）进行定位。使用 `anchor()` 函数使其将多个边缘连接到不同的锚点元素边缘：

  


```CSS
.target {
    position: absolute;
    top: anchor(--anchor-element-one bottom);
    left: anchor(--anchor-element-one right);
    right: anchor(--anchor-element-two left);
    bottom: anchor(--anchor-element-two top);
}
```

  


通过将目标元素连接到这两个锚点元素，它可以在窗口调整大小或锚点移动时动态调整。它会平滑地扩展和收缩，而不需要写一行 JavaScript。试试看，感受它如何无缝地适应变化！

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/2e5883cbfd3b44b0b58161ab72383c85~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1182&h=616&s=2149354&e=gif&f=193&b=030303)

  


> Demo 地址：https://codepen.io/airen/full/KKOapBw

  


有关于这种方法更详细的介绍请移步阅读《[CSS 锚点定位：探索下一代 Web 布局](https://juejin.cn/book/7223230325122400288/section/7259669151743279159)》一文。

  


接着，我们来看第二种使用方法，即 `position-area` 。同样在 `.anchor` 上使用 `anchor-name` 定义一个锚点名称：

  


```CSS
.anchor {
    anchor-name: --tooltip;
}
```

  


接下来，我们再次链接到我们的锚点，并在 `.target` 元素上添加一个新的属性 `position-area` ：

  


```CSS
.target {
    position-anchor: --tooltip;
    position: absolute;
    position-area: top  span-all;;
}
```

  


这将产生如下结果：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/c0f2b74e770c4ad3be6c837f5584a18b~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=923&s=199954&e=jpg&b=050505)

  


> Demo 地址：https://codepen.io/airen/full/vYogzmN

  


`position-area` 属性允许目标元素 `.target` 相对于一个 `3×3` 的网格进行定位，其中锚点元素 `.anchor` 元素位于网格的中心，类似下图所示：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/799c4aa3604f4501916df212a84af6ac~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1289&s=546748&e=jpg&b=050505)

  


这个网格使得精确定位成为可能，元素可以对齐到单个网格块，也可以跨越多个网格块（如 `2×2` 或 `2×3` 配置）。[你可以使用下面这个工具尝试不同的 position-area 位置设置](https://chrome.dev/anchor-tool/)，体验这一功能的强大之处：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/9e7e9b5558a747b89ebad046df959693~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1146&h=652&s=754506&e=gif&f=237&b=f4f4f4)

在这个示例中，`position-area` 设置为 `top` ，将目标元素 `.target` 放置在锚点网格的顶行。第二个参数 `span-all` 会告诉浏览器允许该元素跨越网格的其余两列。

  


需要知道的是，这里介绍的只是 CSS 锚点定位的基本功能，除此之外它还有其他一些强大功能。只不过，这些功能已经超出了这节课的范畴，因此不在这里做过多的阐述。要是大家对该特性感兴趣的话，不用着急，我们后续会有专门的课程来阐述这些特性，以及它在实际案例中的应用。

内联定义的 `--is` 和 `--to` 主要用于锚点函数 `anchor()` ：

  


```CSS
@layer anchor {

        &:not([open])::before {
            position: absolute;
            left: anchor(var(--is) right);
        }
    }

    .tree-item-label {
        anchor-name: var(--is);
    
        &::before,
        &::after {
            position: absolute;
            left: anchor(var(--to) right);
            right: anchor(var(--is) left);
        }
    
        &::before {
            top: calc(anchor(var(--to) top) + 0.5 * var(--lh));
            bottom: anchor(var(--is) center);
        }
    
        &::after {
            bottom: calc(anchor(var(--to) top) - 0.5 * var(--lh));
            top: anchor(var(--is) center);

}
```

## 小结

  


CSS 锚点定位极大地简化了复杂布局的实现。无论是连接线、树形菜单还是流程图，借助浏览器的自动计算能力，我们可以用更少的代码实现动态、响应式布局。同时，通过减少 JavaScript 的使用，提高了代码的可维护性和性能。这类应用展示了 CSS 未来发展的潜力，在提升开发效率的同时，也让前端工程师更加专注于设计和逻辑本身。
