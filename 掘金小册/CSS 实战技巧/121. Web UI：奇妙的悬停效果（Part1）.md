## 什么是悬停效果？

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/023ee269fa044fb1b2ebd041c2affd6d~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=996&h=792&s=16126048&e=gif&f=118&b=ebe8e6)

  


> URL:https://imagehover.io/

  


悬停效果是指当用户将鼠标悬停在 Web 页面元素上时所发生的视觉变化。这些变化可以非常细微，比如颜色或大小的改变，也可以是较为复杂的动画或过渡。无论哪种形式，悬停效果都能为用户界面或 Web 应用增加一层互动性，从而提升整体用户体验。

  


除此之外，悬停效果还能够为用户提供有用的反馈，增强他们对 Web 页面的理解。因此，让我们深入了解悬停效果的力量，看看它如何彻底改变你的 Web 应用的用户体验。

  


## 为什么要使用悬停效果

  


我总结了使用悬停效果的一些好处：

  


-   **更好的用户反馈**：悬停效果可以为用户提供即时反馈，让他们知道自己操作是有效的。这在填写表单或完成某些任务时尤为重要。例如，当用户将鼠标移到某个元素上时改变其颜色，或者弹出提示信息来显示内容
-   **增强视觉吸引力**：悬停效果能为 Web 页面增加层次感和空间感，使其更具视觉吸引力。这不仅有助于吸引用户，还能激励他们进一步探索网站内容
-   **突出重要内容**：通过悬停效果，你可以引导用户关注 Web 页面上的关键内容，比如号召性用语（CTA）按钮或推荐产品
-   **提供提示信息**：提示信息可以帮助用户更好地浏览网站，例如当鼠标悬停在图标上时，显示更详细的信息以解释该区域的用途。这也有助于减少页面上的文字杂乱
-   **创建动画效果**：悬停效果可以用来创建动画和过渡效果，让你的网站更具动感与趣味性

  


以上所列的仅是使用悬停效果的部分好处，其他一些就不一一列举了！

## 使用悬停效果的最佳实践

  


虽然悬停效果是增强 Web 体验的有效工具，但如果使用不当，也可能带来负面影响。以下是一些在设计时应遵循的建议：

  


-   **保持简洁**：设计师常犯的一个错误就是滥用悬停效果。虽然它们可以让页面更美观，但也可能导致用户分心或感到信息过载。Google 的一项研究指出，过多或响应迟缓的悬停效果会让用户感到沮丧。此外，页面上动画过多还可能影响加载速度
-   **保持一致性**：悬停效果在全站范围内应保持风格一致，这样用户才能形成预期，知道每种互动意味着什么
-   **考虑使用场景**：悬停效果应根据其使用场景来设计。例如，企业官网可能更适合使用细微的悬停变化，而创意类网站则可以使用更具视觉冲击力的效果
-   **兼容移动设备**：就像其他网页设计一样，悬停效果也需要在移动端做优化！由于越来越多用户通过手机浏览网页，因此你需要为触摸设备设计替代方案，比如点击或滑动手势，来实现类似的互动体验

  


这里着重强调一点，由于移动设备（如手机和平板电脑）没有像桌面电脑和笔记本电脑那样具备鼠标悬浮的交互操作，因此它们在你触摸元素时才会显示悬停状态。即使你不再触摸元素，它们仍然保持悬停状态。基于这个原因，我们在为 Web 元素添加悬停效果时，可以通过 **[CSS 媒体查询（`@media`）的 `hover` 查询条件](https://juejin.cn/book/7223230325122400288/section/7257368158451793935#heading-13)** 对设备进行判断，比如：

  


```CSS
 @media (hover: hover) {
    /* 有关于悬停相关的样式放在这里 */
}
```

  


有关于这方面更详细的介绍，请移步阅读《[CSS 媒体查询新特性：`@media`](https://juejin.cn/book/7223230325122400288/section/7257368158451793935#heading-13)》！

  


另外，有些悬停效果可能会部分用户群体（比如犯有晕动症的用户）带来不好的体验。庆幸的是，现在很多设备都提供了用户偏好设置，我们可以通过 **[CSS 媒体查询（`@meida`）的 `prefers-reduced-motion`](https://juejin.cn/book/7223230325122400288/section/7257368158451793935#heading-15)** 检测用户的系统是否被开启了动画减弱功能。或者说，我们在创建悬停效果时，应该尊重这部分用户媒体的偏好：

  


```CSS
 @media screen and (prefers-reduced-motion: reduce), (update: slow) { 
    * { 
        animation-duration: 0.001ms !important; 
        animation-iteration-count: 1 !important; 
        transition-duration: 0.001ms !important; 
    } 
} 
```

  


有关于这方面更详细的介绍，请移步阅读《[CSS 媒体查询新特性：`@media`](https://juejin.cn/book/7223230325122400288/section/7257368158451793935#heading-15)》!

  


## 如何实现悬停效果

  


到目前为止，给 Web 网站或应用中的元素添加悬停效果，主要有以下两种方式：

  


-   **使用 CSS**：CSS 是实现悬停效果最简单的方法。你可以通过 `:hover` 伪类来定义当用户悬停在元素上时应显示的样式，例如按钮颜色变化或文字样式改变
-   **使用 JavaScript**：如果你需要更复杂的动画或行为，可以用 JavaScript 实现悬停效果。JavaScript 提供了比 CSS 更强的控制力，比如显示弹出窗口或动态提示信息等复杂交互

### CSS 按钮过渡动画

  


CSS 过渡是为按钮增加活力的绝佳方式。它提供多种方法（比如放大、缩小、边框动画等）让你的按钮更加互动和吸引用户：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/0c390a1c84604ecb874ad73bb5103c24~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1334&h=612&s=640791&e=gif&f=165&b=251a29)

  


> Demo 地址：https://codepen.io/airen/full/gbbZoBQ

  


要创建悬停放大或缩小的动画，可以使用 `transition` 为元素的变化添加过渡效果。在 `:hover` 状态调整 `scale` 的值：

  


```CSS
button {
    transition: all 0.3s ease-in-out;
    
    /* 放大 */
    &.button--grow:hover {
        scale: 1.1;
    }
    /* 缩小 */
    &.button--shrink:hover {
        scale: 0.8;
    }
}
```

  


想要在悬停时实现填充动画，可以设置元素的颜色和背景色，并通过 `transition` 实现颜色的过渡：

  


```CSSS
button {
    transition: all 0.3s ease-in-out;
    
    &.button--fill {
        background: #fff;
        color: #4f7df3;
        
        &:hover {
            background: #4f7df3;
            color: #fff;
        }
    }
}
```

  


想要制作一个摆动效果，可以使用 `@keyframes` 定义动画，然后在 `:hover` 状态下应用这个动画：

  


```CSS
@keyframes swing {
    15% {
        translate: 5px;
    }
    30% {
        translate: -5px;
    }
    50% {
        translate: 3px;
    }
    65% {
        translate: -3px;
    }
    80% {
        translate: 2px;
    }
    100% {
        translate: 0;
    }
}

.button--swing:hover {
    animation: swing 1s ease 1;
}
```

  


边框动画相对而言要略为复杂一点。我们可以使用 `::before` 和 `::after` 创建位于按钮上下的边框线条，并通过 `:hover` 扩展它们的宽度：

  


```CSS
.button--bordered {
    border: none;
    outline: none;
    position: relative;
    
    &::before,
    &::after {
        border: 0 solid transparent;
        transition: all 0.3s;
        content: "";
        height: 0;
        position: absolute;
        width: 24px;
    }
    &::before {
        border-top: 2px solid #4f7df3;
        right: 0;
        top: -4px;
    }
    &::after {
        border-bottom: 2px solid #4f7df3;
        bottom: -4px;
        left: 0;
    }
    &:hover {
        &::before,
        &::after {
            width: 100%;
        }
    }
}
```

  


### 卡片悬停效果

  


卡片是现代 Web 设计中最常见的布局元素之一，下面是几种可以让你的网站更具互动性和吸引力的卡片悬停效果：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/3742238773d141c2a3fc9e5bf1ae75ff~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1334&h=664&s=9979612&e=gif&f=98&b=291929)

  


> Demo 地址：https://codepen.io/airen/full/yyyGvaV

  


上面示例中左侧是一个跟随鼠标位置轻微偏移卡片的效果。这个效果需要使用 CSS 变量配合 JavaScript 跟随鼠标位置，通过监听 `mousemove` 事件，获取鼠标与卡片中心的相对距离（使用 `Element.getBoundingClientRect()` 获取卡片的位置和尺寸）。然后使用这些值设置 CSS 变量，并通过 `transform` 属性让卡片随鼠标移动而轻微倾斜，同时使用 `transition` 使效果平滑：

  


```CSS
.card--shifting {
    transform: 
        rotateX(calc(10deg * var(--dx, 0)))
        rotateY(calc(10deg * var(--dy, 0)));
    transition: transform 0.2s ease-out;
}
```

  


```JavaScript
const card = document.querySelector('.card--shifting');
const { x, y, width, height } = card.getBoundingClientRect();
const cx = x + width / 2;
const cy = y + height / 2;

const handleMove = e => {
    const { pageX, pageY } = e;
    const dx = (cx - pageX) / (width / 2);
    const dy = (cy - pageY) / (height / 2);
    e.target.style.setProperty('--dx', dx);
    e.target.style.setProperty('--dy', dy);
};

card.addEventListener('mousemove', handleMove);
```

  


右侧是一个带有透视效果的卡片，只需使用 `transform` 结合 `perspective()` 和 `rotateY()` 函数即可。使用 `transition` 属性来平滑过渡旋转角度和透视深度。

  


```CSS
.card--perspective {
    transform: perspective(1500px) rotateY(15deg);
    transition: transform 1s ease 0s;
    
    &:hover {
        transform: perspective(3000px) rotateY(5deg);
    }
}
```

### 揭示式悬停效果

  


揭示式悬停效果是一种视觉上具有吸引力的悬停交互方式，它通过“遮挡→揭示”的过程来突出元素内容，常见于图片、卡片、按钮等 UI 组件。用户将鼠标悬停在元素上时，内容逐步或滑动地“显露”出来，给人一种有层次感、动画感的体验。

  


下面是一些常见的揭示式悬停效果。

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/a67d8c38a7b14bed9bae101f482265fa~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1332&h=658&s=1766112&e=gif&f=201&b=251928)

  


> Demo 地址：https://codepen.io/airen/full/dPPwdgL

```CSS
    transition: 0.5s;

    &:nth-child(1) {
        --pos: top;
        padding-bottom: var(--s);
    }
    
    &:nth-child(2) {
        --pos: right;
        --h: 45deg;
        padding-left: var(--s);
    }
    
    &:nth-child(3) {
        --pos: center;
        --h: 90deg;
        object-fit: none;
        padding: calc(var(--s) / 2);
    }
    
    &:nth-child(4) {
        --pos: bottom;
        --h: 135deg;
        padding-top: var(--s);
    }

    &:nth-child(5) {
        --pos: left;
        --h: 180deg;
        padding-right: var(--s);
    }

    &:nth-child(6) {
        --h: 225deg;
        padding-block: calc(var(--s) / 2);
    }
  
    &:nth-child(7) {
        --h: 270deg;
        padding-inline: calc(var(--s) / 2);
    }

    &:nth-child(8) {
        --pos: top left;
        --h: 315deg;
        object-fit: none;
        padding: 0 var(--s) var(--s) 0;
    }
    
    &:nth-child(9) {
        --pos: top right;
        --h: 60deg;
        object-fit: none;
         padding: 0 0 var(--s) var(--s);

    }

}
```

  


在上面这个示例的基础上，我们还可以搭配 `border-image` 实现一个镂空边框效果：

  


```CSS
img {
    --s: 200px;
    --b: 8px;
    --gap: 5px;
    --pos: center;
    width: var(--s);
    aspect-ratio: 1;
    box-sizing: border-box;
    object-fit: cover;
    object-position: var(--pos);
    border-image: linear-gradient(var(--c, #53777a) 0 0) 1 / var(--b) /
      calc(var(--b) + var(--gap));
    cursor: pointer;
    transition: 0.5s;

    &:nth-child(1) {
        --pos: top;
        padding-top: calc(2 * var(--s) / 3);
    }
    
    &:nth-child(2) {
        --c: #f90;
        --pos: bottom;
        padding-bottom: calc(2 * var(--s) / 3);
    }
    
    &:nth-child(3) {
        --c: #90acef;
        --pos: left;
        padding-right: calc(2 * var(--s) / 3);
    }
    
    &:nth-child(4) {
        --c: #aec90a;
        --pos: right;
        padding-left: calc(2 * var(--s) / 3);
    }

    &:hover {
        padding: 0;
    }
}
```

除此之外，我们还可以通过 [CSS 的剪切（`clip-path`）和遮罩（`mask`）来创建揭示式悬停效果](https://juejin.cn/book/7223230325122400288/section/7259668885224456252)。比如下面这个示例，是通过 CSS 的 `clip-path` 实现的：

  


```CSS
img {
    --s: 300px;

    width: var(--s);
    aspect-ratio: 1;
    border-radius: 50%;
    margin: calc(-0.14 * var(--s));
    outline: var(--s) solid #ffc48c;
    outline-offset: calc(var(--s) / -2);
    clip-path: inset(14.65% round 20px);
    transition: 0.7s;
```

### 图片缩放效果

  


鼠标悬停在图片上时时行缩放，这是一个很常见的效果。有趣的是，我们使用了一个非常巧妙的 CSS 技巧，用于实现“悬停时图片边缘展开填满整个容器”的动画效果，视觉上就像图片在容器“放大”一样。这个技巧的整体思路是：利用 `padding` 和 `clip-path` ，在鼠标悬停时将 `padding` 设置为 `0` ，让图片看起来撑满了盒子，离开时再回到收缩状态：

  


```CSS
img {
    --s: 300px;
    --d: 1.5em;
    
    width: var(--s);
    aspect-ratio: 1;
    box-sizing: border-box;
    cursor: pointer;
    padding: calc((1 - var(--hov, 0)) * var(--d));
    object-fit: cover;
    clip-path: inset(var(--d));
    transition: padding 0.4s;

    &:hover {
        --hov: 1;
    }
}
```

  


这里的核心是：

  


-   用 `padding` 控制内容尺寸感知，其中 `--d` 代表最大 `padding` 值，也就是图片在非悬停状态时从边缘缩进去的距离（等于容器留白）。注意，需要将 `box-sizing` 设置为 `border-box` ，确保 `padding` 会影响图片的内容区域，而不是单纯改变盒子大小
-   用 `clip-path` 固定显示范围，设置初始裁剪区域，与 `padding` 相同，从四个方向缩进 `--d` 。但要注意，这个值是固定的，不会跟着 `padding` 动态变化。这其实相当于一个“裁剪遮罩”，用于限制图片初始显示区域
-   用变量 `--hov` 做状态切换，这里的核心逻辑是，默认 `--hov` 的值为 `0` ，此时 `padding` 的值为 `var(--d)` ，即有空隙；悬停时 `--hov` 的值为 `1` ，所以 `padding` 的值为 `0` ，图片填满容器
-   为了保证图片在缩放或裁剪时保持比例不变，优雅填充内容，还需要将 `object-fit` 设置为 `cover`

在鼠标悬停时启动无限旋转的边框动画，在移出时平滑停止。

  


```CSS
@property --a {
    syntax: "<angle>";
    inherits: false;
    initial-value: 0deg;
}

@property --i {
    syntax: "<number>";
    inherits: false;
    initial-value: 0;
}

img {
    width: 300px;
    aspect-ratio: 1;
    border-radius: 30px;
    border: 5px solid #f50;
    padding: 10px;
    mask: 
        conic-gradient(#000 0 0) content-box,
        linear-gradient(
            calc(mod(var(--a), 180deg) * var(--i) + var(--_a, 45deg)),
            #0000 30%,
            #000 40% 60%,
            #0000 70%
        ) subtract,
        conic-gradient(#000 0 0) padding-box;
    transition: --i 0.5s, --a 0s 0.5s;
    cursor: pointer;

    &:nth-child(2) {
        --_a: -45deg;
    }

    &:hover {
        --i: 1;
        --a: 15turn;
        transition: --i 0s, --a 30s linear;
    }
}
```

  


其核心在于：

  


-   利用 `mask` 和 `conic-gradient` 来创建动感边框
-   使用 `@property` 注册变量，从而让 `--a` 和 `--i` 能参与平滑过渡
-   通过 `mod(var(--a), 180deg) * var(--i)` 控制遮罩光束显示、隐藏
-   动画开始后，`--a` 连续递增（`15turn`），动画持续进行
-   鼠标移出后，`--i` 变为 `0`，遮罩逐渐消失，但角度不立即归零，所以停止时非常平滑

### 炫酷的头像悬停效果

  


我们可以使用极简的代码，为你的头像添加一个很棒的“弹出”悬停效果：

  


```CSS
img {
    --s: 300px;
    --b: 5px;
    --c: #c02942;
    --f: 1;
    --_g: 50% / calc(100% / var(--f)) 100% no-repeat content-box;
    --_o: calc((1 / var(--f) - 1) * var(--s) / 2 - var(--b));
    
    width: var(--s);
    aspect-ratio: 1;
    box-sizing: content-box;
    padding-top: calc(var(--s) / 5);
    cursor: pointer;
    border-radius: 0 0 999px 999px;
    
    outline: var(--b) solid var(--c);
    outline-offset: var(--_o);
    background: radial-gradient(
        circle closest-side,
        #ecd078 calc(99% - var(--b)),
        var(--c) calc(100% - var(--b)) 99%,
        #0000
    ) var(--_g);
    mask: 
        linear-gradient(#000 0 0) no-repeat 50% calc(1px - var(--_o)) / calc(100% / var(--f) - 2 * var(--b) - 2px) 50%,
        radial-gradient(circle closest-side, #000 99%, #0000) var(--_g);
    scale: var(--f);
    transition: 0.5s;
    
    &:hover {
        --f: 1.35; 
    }
}
```

  


这个效果巧妙结合了：

  


-   `outline + outline-offset` → 营造滑动边框效果
-   `mask + gradient` → 控制图像区域的显隐
-   `scale` → 实现缩放放大

  


整体是一个高级但只使用单个元素（无额外 HTML 元素） 的头像悬停特效，非常适合展示头像、卡片或徽章类 UI：
