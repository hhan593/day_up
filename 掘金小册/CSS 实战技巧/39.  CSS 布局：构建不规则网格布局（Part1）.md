## 创建菱形网格

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/09799bd744f048e7b7f18696f4070347~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1440&h=1080&s=185764&e=jpg&b=e5f0f7)

  


我们从简单的菱形网格开始！为了不使问题复杂化，我们以四张图片为基础来创建菱形网格布局。

  


首先，我们使用 `grid-template-columns` 创建一个两列的网格：

  


```CSS
.grid {
    --size: 150px; /* 列宽 */
    --gap: 1rem;
    display: grid;
    grid-template-columns: repeat(2, var(--size));
    grid-auto-rows: var(--size);
    gap: var(--gap);
}
```

  


上面的代码会使网格有一个固定的列数（两列），但行数取决于网格项目的数量，而很多时候网格项目数量是未知的。因此，我们没有使用 `grid-template-rows` 来指定网格的行数，而是使用了 `grid-auto-rows` 属性定义了行轨道大小，具体行数由浏览器根据网格项目的数量来自动处理。

  


在我们这个示例中，网格项目（`<img>` 元素）的总数是 `4` ，因此上面的代码会创建一个 `2 x 2` 的网格（两行两列），并且行和列的尺寸相等，都是 `--size` 。同时，网格轨道之间的间距是 `--gap`:

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/e929ceb61af6447fb908e85ac6c42115~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1115&s=713874&e=jpg&b=030303)

  


如果你对 CSS Grid 比较熟悉的话，那么可以使用 `grid` 简写属性来简化上面的代码。它可以用一个声明定义整个网格布局，只不过这种语法不太直观，不易于理解。

  


```CSS
.grid {
    --size: 150px; /* 列宽 */
    --gap: 1rem;
    display: grid;
    grid: auto-flow var(--size) / repeat(2,var(--size));
    gap: var(--gap);
}
```

  


对于初学者来说，我建议还是分开书写比较合适，虽然代码量增加了，但直观易于理解，不容易出错。

  


接下来就是创建菱形网格的另一个关键点了。我们需要使用 `clip-path` 将图片裁剪成菱形：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/866a2c7e752941da98960118471437f8~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2536&h=1656&s=176213&e=jpg&b=fbfcf7)

  


> URL:https://bennettfeely.com/clippy/

  


```CSS
.grid {
    /* 省略其他 CSS */
    
    img {
        clip-path: polygon(50% 0, 100% 50%, 50% 100%, 0 50%);
    }
}
```

  


经过 `clip-path` 裁剪之后，你看到的效果如下图所示：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/5741b28f5a3647388cd58c7c78411744~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1115&s=559266&e=jpg&b=030303)

  


接着我们使用 `rotate` 对整个网格容器进行旋转，并且网格项目向反方向旋转相同的一个角度，使网格在形状上看上去是一个菱形：

  


```CSS
.grid {
    /* 省略其他 CSS */
    rotate: 45deg;
    
    img {
        /* 省略其他 CSS*/
        rotate: -45deg;
    }
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/f2bb641d250d4dabb665113940ef02f3~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1115&s=510317&e=jpg&b=020202)

  


离我们需要的目标效果已经非常接近了。最后要做的是，调整网格项目（图片）的大小，使它们之间无缝衔接。否则，就会像上图所展示的那样，网格项目之间太过于分散，看起来不像一个完整的网格。

  


这意味着，接下来调整网格项目尺寸是关键，那么如何调整网格项目的大小呢？我用下图来演示：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/282acc6fe4484e1db7340d7c207c48e8~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1115&s=561109&e=jpg&b=020202)

  


正如你所看到的。网格项目默认在橙色圆的边界内（它的宽高都是 `--size`），而我们要将网格项目尺寸调整到红色圆的边界。

  


仔细观察这两个圆，不难发现，橙色圆是网格单元格的内切圆，而红色圆是网格单元格的外接圆。根据我们以前所掌握到的几何知识，我们可以获知内切圆和外接圆的半径比是 `√2` 。也就是说，我们需要将网格项目（图片的尺寸）调整到原来的 `141%` ，即 `100%×√2 ≈ 141%` 。

  


```CSS
.grid {
    /* 省略其他 CSS */
    rotate: 45deg;
    
    img {
        /* 省略其他 CSS*/
        rotate: -45deg;
        width: calc(100% * sqrt(2));
        aspect-ratio: 1;
    }
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/7c40c2ae9ff54be58f0063e52eed210d~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1115&s=463149&e=jpg&b=020202)

  


调整完尺寸之后，所有图片在视觉上看上去都向下移动了一定的间距。不用着急，我们通过调整对齐方式来修复这个问题。在网格容器中上设置 `place-content` 为 `center` ，使网格轨道都居中对齐。最终的 CSS 代码如下：

  


```CSS
.grid {
    --size: 150px;
    --gap: 1rem;

    display: grid;
    grid-template-columns: repeat(2, var(--size));
    grid-auto-rows: var(--size);
    gap: var(--gap);
    place-items: center;
    outline: 1px dashed;
    rotate: 45deg;

    img {
        width: calc(100% * sqrt(2));
        aspect-ratio: 1;
        object-fit: cover;
        object-position: center;
        clip-path: polygon(50% 0, 100% 50%, 50% 100%, 0 50%);
        rotate: -45deg;
    }
}
```

  


你将看到的效果如下：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/403edc09bdd24d8883da38ece0642392~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1115&s=331812&e=jpg&b=010101)

  


> Demo 地址：https://codepen.io/airen/full/MWNMXaG

  


怎么样，完美吧！

  


创建一个菱形网格布局就是这么简单，关键是我们要学以致用。比如，使用我们所掌握的知识来创建下图这个布局效果：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/32bb7ad30ce7435598de47aaa3804c8c~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=800&h=936&s=145835&e=jpg&b=f9f3f2)

  


这个布局中，图片是相互交错的。与之前所介绍的菱形网格是有点类似的，但我们接下来的做法会略有不同。

  


首先，我们使用同样的方式来创建网格，但这里创建的是一个六列网格。这看起来有点反直觉，因为有些行只有两张或三张图片。但实际上，每张图片都会占据两列，而只有两张图片的行会向右“偏移”一列。

  


```CSS
.grid {
    --size: 100px;
    --gap: 5px;
    
    display: grid;
    grid-template-columns: repeat(6, var(--size));
    grid-auto-rows: var(--size);
    gap: var(--gap);
    place-items: start center;
    margin-bottom: var(--size);
}
```

  


上面代码与之前案例中的代码没有太大的差异，在这里设置 `place-items` 为 `start center` 是使网格项目水平方向居中，垂直方向靠顶部。`margin-bottom` 是为了解决最后一行图片溢出的情况，因为图片高度为两个单元格的高度，增加等于单元格高度的底部外边距来避免内容重叠。

  


接着设置图片样式，每张图片将占据四个网格单元格（行和列各两个）。然后，同样使用 `clip-path` 将它裁剪出我们所需要的菱形形状。

  


```CSS
.grid {
    /* 省略其他 CSS */
    
    img {
        width: 100%;
        aspect-ratio: 1;
        grid-column: auto / span 2;
        border-radius: 5px;
        clip-path: path("M90,10 C100,0 100,0 110,10 190,90 190,90 190,90 200,100 200,100 190,110 190,110 110,190 110,190 100,200 100,200 90,190 90,190 10,110 10,110 0,100 0,100 10,90Z"); 
        object-fit: cover;
        object-position: center;
    }
}
```

  


注意，在这里我们没有对整个网格做旋转处理，同时使用的是 `path()` 函数对图片进行裁剪，裁剪出所需要的具有旋转视觉效果的菱形图片。需要知道的是，使用 `polygon()` 裁剪出来的菱形是没有圆角效果的，[虽然之前课程有介绍如何使用 polygon() 裁剪出带有圆角的多边形形状](https://juejin.cn/book/7395119173656903715/section/7401099842379841545)，但它有一个明显的缺陷，不具响应式。

  


通过上面代码处理之后，我们得到一个像下图所示的效果：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/2b81ff70530c4761960db129f5aec531~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1115&s=796336&e=jpg&b=030303)

  


不难发现，到目前为止，所有图片排列成三行一列。而我们想要的效果是，从第四张图片起，每隔五张（第 `4` 、`9` 、 `14` ，依此类推）都会向右偏移一列。在 CSS 中，我们可以这样实现偏移：

  


```CSS
.grid {
    /* CSS ... */
    
    img {
        /* CSS ... */
        &:nth-child(5n - 1) {
            grid-column: 2 / span 2;
        }
    }
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/396e7d2aa3664a9391fc586725348134~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1115&s=933408&e=jpg&b=030303)

  


我们把调试的网格线隐藏，就将获得我们所需要的布局效果：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/de40fe72771b45eab9baf2b7d166a68d~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1266&s=1009182&e=jpg&b=000000)

  


需要知道的是，图片数量是可以根据你的需要来调整的。但我个人建议你使用的图片总数为 `5n` （以两张图结尾）或 `5n + 3` （以三张图结尾）。

  


如果需要的话，还可以给图片添加一些动画效果。例如：

  


```CSS
.grid {
    /* CSS ... */
    
    &:has(img:hover) img:not(:hover) {
        filter: brightness(0.5) contrast(0.5);
    }
    
    img {
        /* CSS ... */
        transition: clip-path 0.25s, filter 0.75s;

        &:hover,
        &:focus {
            clip-path: path("M0,0 C0,0 200,0 200,0 200,0 200,100 200,100 200,100 200,200 200,200 200,200 100,200 100,200 100,200 100,200 0,200 0,200 0,100 0,100 0,100 0,100 0,100Z");
            z-index: 1;
            transition: clip-path 0.25s, filter 0.25s;
        }
    }
}
```

  


这个时候整个效果如下所示：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/c985f9e4e6174b859fdc42bb9d04118b~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1236&h=762&s=7557223&e=gif&f=95&b=010101)

这次的网格布局，我们将提高一点难度，创建一个具有响应式的三角形网格布局。为了更好的控制网格，我们这里将引用更多的 CSS 自定义属性：

  


```CSS
.grid {
    --size: calc(4.5rem + 2vw);
    --size-row: calc(var(--size) * cos(30deg));
    --gap-mult: 0.023;
    --cols: 3;
    
    width: calc(100% - var(--size) * 2);
    margin: calc(var(--size) * 0.46) auto;
    
    display: grid;
    grid-auto-rows: var(--size-row);
    grid-template-columns: repeat(var(--cols), var(--size));
    gap: var(--size-row) calc(var(--size) * var(--gap-mult));
    justify-content: center;
    
    @media (width >= 570px) {
        --cols: 5;
    }
    
    @media (width >= 810px) {
        --cols: 7;
    }
    
    @media (width >= 1120px) {
        --cols: 9;
    }
    
    @media (width >= 1340px) {
        --cols: 11;
    }
    
    @media (width >= 1600px) {
        --cols: 13;
    }
    
    img {
        width: 100%;
        height: 100%;
        object-fit: cover;
        object-position: center;
    }
}
```

  


上面的代码中，通过 [CSS 的媒体查询 `@media` ](https://juejin.cn/book/7223230325122400288/section/7257368158451793935)在不同断点中调整了 `--cols` 的值（即网格列数），因此你调整浏览器视窗宽度时，可以看到网格列数的变化：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/df9883ce0d8244e2b340b8a3478be91c~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1674&h=780&s=17636101&e=gif&f=93&b=114247)

  


另外，由于我们呆会需要使用 `clip-path` 裁剪出一个等边三角形（腰长等于底边 `--size`）。因此，通过[三角函数](https://juejin.cn/book/7223230325122400288/section/7242216512176521277)计算出三角形的高，也就是网格行轨道的尺寸。

  


```CSS
.grid {
    --size: calc(4.5rem + 2vw); /* 列宽，对应的是三角形底边长度 */
    --size-row: calc(var(--size) * cos(30deg)); /* 行高，对应的是三角形的高 */
}
```

  


还有一点与之前案例的网格不同之处是，网格的行间距与列间距设置了不同的值，其中行间距与行高相等。至于为什么要这么做，我们稍后再介绍。

  


接下来，我们使用 `clip-path` 的 `polygon()` 函数将图片裁剪成三角形形状。这里有两种三角形，奇数三角形顶点朝下，偶数三角形顶点朝上。在 CSS 中，我们可以像下面这样来实现：

  


```CSS
.grid {
    /* CSS ... */
    
    img {
        /* CSS ... */
        
        /* 奇数 */
        &:nth-child(odd) {
            clip-path: polygon(0 0, 100% 0, 50% 98%, 0 0);
        }    
        
        /* 偶数 */
        &:nth-child(even) {
            clip-path: polygon(50% 2%, 100% 100%, 0 100%, 50% 2%);
        }
    }
}
```

  


经过 `clip-path` 裁剪之后，将会得到下图这样的效果：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/f4be8b54424249fc90ffff515a8adc3b~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=879&s=1658717&e=jpg&b=020202)

  


很容易发现，图片与图片之间的间距太大了，我们需要调整它们之间的间距，使其达到我们预期的效果。但是，我们不可能一个个通过 `margin` 来调整它们之间的间距。更简单的方式是，我们使用 CSS 变换的 `scale` 对图片进行放大，使图片之间的间距在视觉上变小，从而达到我们所期望的效果：

  


```CSS
.grid {
    /* CSS ... */
    
    img {
        /* CSS ... */
        scale: 1.93;
    }
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/cc86d42d301c4856b4bc953499a1dc57~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1200&s=2011567&e=jpg&b=060606)

  


你可能会问，`scale` 属性的值 `1.93` 是怎么来的。说实话，这是通过一步步调试验证出来的值。当然，如果你对几何学比较精通的话，或许可以根据三形缩放特征，通过几何公式获取更精确的缩放值。如果你找到了这个公式，请记得与我分享。有意思的是，我在调试这个缩放比例的过程中发现，如果我们希望图片之间的间距更大一点，那么只需要将 `scale` 的值调小一点，例如 `1.73` 会得到下图这个效果：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/d10a4c8e71c242ec9a3f0351cba547ec~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1200&s=1689015&e=jpg&b=040404)

  


最后再来解释一下，为什么行网格之间的间距要设置成与行高一样，或者说要设置得比较大。主要原因是，我们要通过放大图片来调整图片之间的间距，如果行与行之间的间距不够大，就会引起图片层叠的现象：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/fe32e7edd5d14a919c61b48f27eb2dac~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1136&h=662&s=18367202&e=gif&f=77&b=060606)

  


值得注意的是，当图片的 `scale` 值小到一定程度时，行高也要随之调小，反之也要调大，不然行与行之间要么有较大的间距，要么造成图片层叠。到目为止，这个过程是比较痛苦的，需要借助开发者工具不断的尝试，找到一个最佳值。

    margin: calc(var(--size) * 0.46) auto;

    }

    @media (width >= 810px) {
        --cols: 7;
    }

    @media (width >= 1120px) {
      --cols: 9;
    }

    }

    }

        height: 100%;

## 创建六边形网格

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/94bcb5a318b843cdb88e8bd41ca86edb~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1280&h=720&s=48148&e=jpg&b=161523)

  


或许你在 Web 很多地方看到类似上图的布局效果。这种布局有时也被称为“蜂窝”网格布局。在网上已经有很多关于如何制作这种网格布局的教程。如果你阅读过小册之前的《[CSS 图形：创建正多边形和星爆形状](https://juejin.cn/book/7395119173656903715/section/7397319151691235328)》课程，我想你已经知道如何制作这种网格布局。

  


接下来，我们来看一个特定的案例（如下图所示），我们可以采用一种更加简单的方法来实现：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/c0b819c1b1254ea080193e12e39e9404~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1130&s=427912&e=jpg&b=adbcf9)

  


首先，创建网格，但与之前创建网格不同，在这里不会使用 `grid-template-columns` 指定具体的网格列数，也不会使用 `grid-auto-rows` 指定网格行轨道的大小。我们只是将 `.grid` 容器定义为一个网格容器：

  


```CSS
.grid {
    --size: 200px;
    display: grid;
}
```

  


然后使用 `grid-area` 将所有网格项目（所有图片）放置在同一个网格区域上，让它们相互重叠：

  


```CSS
.grid {
    --size: 200px;
    display: grid;
    
    img {
        grid-area: 1 / 1 / -1 / -1;
        width: var(--size);
        aspect-ratio: 1;
        object-fit: cover;
        object-position: center;
    }
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/d2cc9d68a994422d80fa008fddc5df5c~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1130&s=488301&e=jpg&b=000000)

  


接着，使用《[CSS 图形：创建正多边形和星爆形状](https://juejin.cn/book/7395119173656903715/section/7397319151691235328)》课程所介绍的技巧，通过 `clip-path` 的 `polygon()` 函数将图片裁剪成正六边形形状：

  


```CSS
.grid {
    --size: 200px;
    display: grid;
    
    img {
        /* CSS... */
        --s: 6; /* 正多边形的边数*/
        --a: 360deg / var(--s); /* 旋转角度，即 θ */
    
        /* 计算每条边对应的角度: angle + (i * 360deg) / sides, i 从 0 开始索引 */
        --a1: var(--a) + (0 * 360deg) / var(--s); /* 60deg */
        --a2: var(--a) + (1 * 360deg) / var(--s); /* 120deg */
        --a3: var(--a) + (2 * 360deg) / var(--s); /* 180deg */
        --a4: var(--a) + (3 * 360deg) / var(--s); /* 240deg */
        --a5: var(--a) + (4 * 360deg) / var(--s); /* 300deg */
        --a6: var(--a) + (5 * 360deg) / var(--s); /* 360deg */
    
        /* 计算每个点的坐标
         * Px = 50% + 50% * cos(angle + (i * 360deg) / sides)
         * Py = 50% + 50% * sin(angle + (i * 360deg) / sides)
        */
        --p1x: calc(50% + 50% * cos(var(--a1)));
        --p1y: calc(50% + 50% * sin(var(--a1)));
    
        --p2x: calc(50% + 50% * cos(var(--a2)));
        --p2y: calc(50% + 50% * sin(var(--a2)));
    
        --p3x: calc(50% + 50% * cos(var(--a3)));
        --p3y: calc(50% + 50% * sin(var(--a3)));
    
        --p4x: calc(50% + 50% * cos(var(--a4)));
        --p4y: calc(50% + 50% * sin(var(--a4)));
    
        --p5x: calc(50% + 50% * cos(var(--a5)));
        --p5y: calc(50% + 50% * sin(var(--a5)));
    
        --p6x: calc(50% + 50% * cos(var(--a6)));
        --p6y: calc(50% + 50% * sin(var(--a6)));
    
        --p1: var(--p1x) var(--p1y); /* P1 点坐标 */
        --p2: var(--p2x) var(--p2y); /* P2 点坐标 */
        --p3: var(--p3x) var(--p3y); /* P3 点坐标 */
        --p4: var(--p4x) var(--p4y); /* P4 点坐标 */
        --p5: var(--p5x) var(--p5y); /* P5 点坐标 */
        --p6: var(--p6x) var(--p6y); /* P6 点坐标 */
    
        /* 定义 polygon() 坐标值 */
        --shape: var(--p1), var(--p2), var(--p3), var(--p4), var(--p5), var(--p6);
    
        clip-path: polygon(var(--shape));
    }
}
```

  


这个时候，你将会看到，所有图片都被裁剪成正六边形，并且完全重叠在一起：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/3b1b708bd2614e6dbb6647ff23a53832~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1130&s=359182&e=jpg&b=000000)

  


接下来，我们需要将图片定位到网格的正确位置上。不过，在这里不会采用绝对定位来对图片进行定位，而是通过 CSS 的 `translate` 将图片移动到正确的位置上。

  


在我们这个示例中，我们需要将其中一张图保持在网格容器的正中心，然后将剩余的图片围绕这个中心排列。我们可以通过三角函数计算出每张图片所需要位移的距离。下图将告诉大家如何计算出位移的距离：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/e4d86dbc2970441bb55f6ee462bbfbc0~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1924&s=1261047&e=jpg&b=000000)

  


如此一来，我们就可以计算出图片位移的距离了：

  


```CSS
.grid {
    /* CSS ... */
    
    img {
        /* CSS ... */
        --tx: calc(100% * sin(var(--_a)));
        --ty: calc(100% * cos(var(--_a)));
        translate: var(--tx) var(--ty);
      
        &:nth-child(1) {
            --_a:0deg;
        }
        
        &:nth-child(2) {
            --_a:60deg;
        }
      
        &:nth-child(3) {
            --_a:120deg;
        }
        
        &:nth-child(4) {
            --_a:180deg;
        }
        
        &:nth-child(5) {
            --_a:240deg;
        }
      
        &:nth-child(6) {
            --_a:300deg;
        }
    }
}
```

  


这个时候，你会看到图片根据需要移动到指定位置：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/bf3e2beef31049c4b31953bf1637b9e4~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1202&s=637647&e=jpg&b=010101)

  


你可能已经发现了，图片之间的间距可能超出了预期，甚至好奇，我们并没有设置间距，这个距离是哪来的呢？这个问题不难回答，请仔细回忆一下，当我们使用 `clip-path` 裁剪出正六边形时，正六边形的顶点都位于直接为 `--size` 的圆上。正六边形的每边距离元素框的边缘是会存在一定距离的：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/5a13b049f942416ea0ecd99c1a347940~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1526&s=588806&e=jpg&b=010101)

  


如果你不希望这个间距过于太大，可以像之前的三角形网格的案例一样，通过 `scale` 属性对图片进行放大。

  


完整的 CSS 代码如下：

  


```CSS
.grid {
    --size: 200px;
    display: grid;

    img {
        width: var(--size);
        aspect-ratio: 1;
        object-fit: cover;
        object-position: center;
        grid-area: 1 / 1 / -1 / -1;

        --s: 6; /* 正多边形的边数*/
        --a: 360deg / var(--s); /* 旋转角度，即 θ */

        /* 计算每条边对应的角度: angle + (i * 360deg) / sides, i 从 0 开始索引 */
        --a1: var(--a) + (0 * 360deg) / var(--s); /* 60deg */
        --a2: var(--a) + (1 * 360deg) / var(--s); /* 120deg */
        --a3: var(--a) + (2 * 360deg) / var(--s); /* 180deg */
        --a4: var(--a) + (3 * 360deg) / var(--s); /* 240deg */
        --a5: var(--a) + (4 * 360deg) / var(--s); /* 300deg */
        --a6: var(--a) + (5 * 360deg) / var(--s); /* 360deg */

        /* 计算每个点的坐标
         * Px = 50% + 50% * cos(angle + (i * 360deg) / sides)
         * Py = 50% + 50% * sin(angle + (i * 360deg) / sides)
       */
        --p1x: calc(50% + 50% * cos(var(--a1)));
        --p1y: calc(50% + 50% * sin(var(--a1)));

        --p2x: calc(50% + 50% * cos(var(--a2)));
        --p2y: calc(50% + 50% * sin(var(--a2)));

        --p3x: calc(50% + 50% * cos(var(--a3)));
        --p3y: calc(50% + 50% * sin(var(--a3)));

        --p4x: calc(50% + 50% * cos(var(--a4)));
        --p4y: calc(50% + 50% * sin(var(--a4)));

        --p5x: calc(50% + 50% * cos(var(--a5)));
        --p5y: calc(50% + 50% * sin(var(--a5)));

        --p6x: calc(50% + 50% * cos(var(--a6)));
        --p6y: calc(50% + 50% * sin(var(--a6)));

        --p1: var(--p1x) var(--p1y); /* P1 点坐标 */
        --p2: var(--p2x) var(--p2y); /* P2 点坐标 */
        --p3: var(--p3x) var(--p3y); /* P3 点坐标 */
        --p4: var(--p4x) var(--p4y); /* P4 点坐标 */
        --p5: var(--p5x) var(--p5y); /* P5 点坐标 */
        --p6: var(--p6x) var(--p6y); /* P6 点坐标 */

        /* 定义 polygon() 坐标值 */
        --shape: var(--p1), var(--p2), var(--p3), var(--p4), var(--p5), var(--p6);

        clip-path: polygon(var(--shape));
      
        --tx: calc(100% * sin(var(--_a)));
        --ty: calc(100% * cos(var(--_a)));
        translate: var(--tx) var(--ty);
        scale: 1.1;
      
        &:nth-child(1) {
            --_a:0deg;
        }
        
        &:nth-child(2) {
            --_a:60deg;
        }
      
        &:nth-child(3) {
            --_a:120deg;
        }
      
        &:nth-child(4) {
            --_a:180deg;
        }
      
        &:nth-child(5) {
            --_a:240deg;
        }
      
        &:nth-child(6) {
            --_a:300deg;
        }
    }
}
```
