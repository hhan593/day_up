![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/45e096d5f8bb4561832aee8e5191a67f~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1280&h=720&s=59521&e=jpg&b=f6f2f1)

  


众所周知，通过 CSS 的 `border` 属性可以给元素添加边框效果，并且内置了多种不同的边框样式（`border-style`），比如 `dotted` 、`dashed` 、`solid` 、`double` 、`groove` 、`ridge` 、`inset` 和 `outset` 等。不过，有的时候，这些内置的边框样式并不能满足 Web 设计的需求，比如双重边框的效果，虽然 `double` 在一定的情景下可以满足，但其缺陷也非常的明显。假设双重边框的粗细不同、颜色不同时，`double` 就无法满足。

  


庆幸的是，CSS 中有很多种方法可以用于绘制元素的边框，比如 `outline` 、`box-shadow` 、伪元素、渐变等。这些方法除了能模拟出简单的边框样式这外，还可以用于创建双重边框或者多重边框的效果。

  


接下来，我将与大家一起探讨 CSS 中创建多重边框的方式和技巧。希望这些技巧能在实际工作当中解决大家的痛点或帮助大家快速构建出符合设计需求的边框效果。

  


为了能更好的理解接下来的内容，你需要具备以下基础知识：

  


-   [CSS 的 `border`](https://juejin.cn/book/7199571709102391328/section/7199845356845629472)
-   [CSS 的渐变](https://juejin.cn/book/7199571709102391328/section/7199845781149810727)，例如 `linear-gradient` 和 `repeating-linear-gradient`
-   [CSS 的盒子阴影](https://juejin.cn/book/7199571709102391328/section/7199844993455325216)：`box-shadow`
-   [CSS 的轮廓](https://www.w3.org/TR/css-ui-4/#outline-props)：`outline`
-   [CSS 的边框图像](https://www.w3.org/TR/css-backgrounds-3/#border-images)：`border-image`

## 几个重要概念

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/7b780a6930de4049b2e1eb6d814e433a~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2400&h=1116&s=14432&e=webp&b=fff0e5)

  


在开始正式进入编写 CSS 代码之前，我们有几个概念需要了解一下。

  


首先，我们需要明确一点。在 CSS 中，只有 `border` 属性创建的边框才是真正的边框，而其他方式创建的“边框”不是真正的边框，它们只是在视觉上创建了“边框”。例如，`outline` （轮廓）和 `box-shadow` （盒子阴影）。

  


这三种创建边框方法的关键区别在于它们在元素上的位置，以及它们如何影响元素的尺寸。这个行为由 CSS 盒模型控制。

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/909c71aee65040b1a4c1ad359d936cfc~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1111&s=232352&e=jpg&b=222126)

  


-   **`border`**：精确地是元素的边界，位于内边距（`padding`）和外边距（`margin`）之间，它的宽度会影响计算后的元素尺寸
-   **`outline`**：与 `border` 相邻但位于其外侧，重叠 `box-shadow` 和外边距（`margin`），但不会影响元素的尺寸
-   **`box-shadow`**：默认情况下从 `border` 的边缘向外扩展，覆盖定义的方向上的空间，同样不会影响元素的尺寸

  


在 CSS 的世界中，每个元素都是一个盒子，每个盒子都具有不同布局框的限制范围：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/a17bb065e900435f84bfad36f7c8d3ab~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1111&s=299722&e=jpg&b=333333)

  


最外层的是边框框（`border-box`）。在它里面，距离边框一个边框厚度（`border-width`）的位置是内边距框（`padding-box`）。最后，在内边距框内，距离内边距（`padding`）限制一个内边距距离的位置是内容框（`content-box`）。我们的背景图片（例如渐变）可以通过 `background-clip` 和 `background-origin` 将背景限制在这几个布局框中：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/09cb51c9519941eaa880ea6434d930a0~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1196&h=834&s=7469259&e=gif&f=253&b=2c2c2c)

  


这也是我们通过渐变能创建双重或多重边框的主要原理，这也是创建渐变边框的原理。例如，我们叠加三个渐变：顶部的渐变限制在内容框，中间的渐变限制在内边距框，底部的渐变限制在边框框。这在视觉上，边框框和内边距框中的渐变就构建了一个简单的双重渐变边框，如果将渐变色换成单一色，那就是双重边框。

  


另外，在 CSS 中，大部分元素都具有伪元素 `::before` 和 `::after` 。这意味着，一个元素可以具备多个盒子框，即有多个背景和边框等。

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/34d64e369db242efb60b733b29f9ecae~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1111&s=275553&e=jpg&b=f5f5f5)

  


换句话说，我们可以在元素自身和它的伪元素上都同时应用渐变、`border` 、`outline` 和 `box-shadow` 等，给元素在视觉上创建边框。想象一下，它们同时组合在一起，是不是可以创建更多重的边框。

  


了解完之后，我们就可以开始使用 CSS 来创建多重边框了。我们先从简单的双重边框开始！

  


## 创建双重边框

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/b0ef471866104f53af125bb5f4909a82~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1680&h=840&s=128450&e=avif&b=2a2d30)

  


在 CSS 中创建双重边框的方法有很多种，我至少能想到七八种。

  


### 使用 `double` 创建双重边框

  


给一个元素设置双重边框最简单的方法是将 `border-style` 或 `outline-style` 属性的值设置为 `double` ，它会自动在两条边框线之间创建间距。：

  


```CSS
.box {
    --b: round(.5em, 1px);
    background: hsl(90 90% 30% / .5);
    
    &:nth-child(1) {
        border: var(--b) double currentColor;
    }
    
    &:nth-child(2) {
        outline: var(--b) double currentColor;
    }
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/1668ea2821a54b5f82b0971ab158826b~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1111&s=248842&e=jpg&b=050505)

  


> Demo 地址：https://codepen.io/airen/full/bGPyEgJ

  


使用 `double` 关键词创建双重边框时，它有一个小细节需要知道，`border-width` 或 `outline-width` 的值最好不要少于 `3px` ，否则双重边框的效果不明显。

  


从示例渲染的结果中不难发现，元素的背景颜色会延伸到 `double` 绘制的双重线之间的间距中。如果你不希望元素的背景延伸到边框的底下，那么需要将 `background` 限制在 `padding-box` 中：

  


```CSS
.box {
    border: var(--b) double currentColor;
    background-clip: padding-box;
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/af2729d4274047529bd1c17220c1e043~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1111&s=249452&e=jpg&b=050505)

  


`outline` 设置的双重边框默认会从元素的 `border` 外边缘向外延伸，如果你希望它向内延伸，我们需要将 `outline-offset` 属性的值设置为一个负值：

  


```CSS
.box {
    outline: var(--b) double currentColor;
    outline-offset: calc(-1* var(--b));
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/5d400c81552d488b828f6ee3b6f88750~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1111&s=247090&e=jpg&b=050505)

  


如果你只需要一个普通的双重边框效果，那么 `double` 关键词应该可以满足你的需求。只不过，它有一个致命的缺陷，两条边框线的颜色和厚度是相同的。换句话说，如果你希望两条边框线的颜色或厚度不相同，那么 `double` 关键词就不适合。

  


要是你希望元素具有不同样式风格的双重线，则可以将 `border` 和 `outline` 结合起来使用。例如：

  


```CSS
.box {
    --b: round(0.5em, 1px);
    --c1: #f36;
    --c2: color-mix(in srgb, var(--c1), #000 60%);
    
    border: calc(var(--b) / 2) solid var(--c1);
    outline: var(--b) solid var(--c2);
    
    &:nth-child(2) {
        outline-offset: calc(var(--b) / 2);
    }

    &:nth-child(3) {
        outline-offset: calc(-2 * var(--b));
    }
}
```

  


需要知道的是，`outline` 属性不会自动在轮廓和边框之间创建间距。要在轮廓和边框之间添加间距，我们需要使用 `outline-offset` ，正值向外延伸，轮廓在边框的外部，反之则向内延伸，轮廓在边框内部：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/bfe8fdccba30417f955970560d7892cc~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1129&s=257769&e=jpg&b=050505)

  


> Demo 地址：https://codepen.io/airen/full/QWXRdym

  


[CSS 的 `border`](https://juejin.cn/book/7199571709102391328/section/7199845356845629472) 和 `outline` 是 Web 开发中常用的两种样式特性。这两个属性都允许 Web 开发者对 HTML 元素进行视觉修改，但它们的功能不同。其主要区别如下：

  


-   `border` 是元素盒模型的一部分，而 `outline` 不是
-   `border` 会影响其他元素的布局，而 `outline` 不会
-   `border` 用于视觉上分隔组件（例如，分隔线），而 `outline` 用于提供视觉重点或表示聚焦，常用于焦点环样式设置
-   `outline` 不一定是矩形，但通常是
-   `border` 可以通过其子属性 `border-top` 、`border-right` 、`border-bottom` 和 `border-left` 为元素每一侧设置不同的边框样式，而 `outline` 到目前无法做到类似的效果

  


虽然 `border` 和 `outline` 两者有诸多差异，但它们有一个共同的特性，必须显式设置边框样式，即 `border-style` 和 `outline-style` ，否则边框不会渲染。

  


### 使用 `box-shadow` 创建双重边框

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/d5645f8961ed41a5953a2ed4e959ea3c~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1620&h=850&s=263862&e=jpg&b=1d2637)

  


经验丰富的 Web 开发者通常会[使用 `box-shadow` 来绘制各种视觉效果](https://juejin.cn/book/7199571709102391328/section/7199844993455325216#heading-4)，包括一些形状。在诸如 [CSS Art](https://www.cssartist.com/) 和 [A Single Div](https://a.singlediv.com/) 中展示的案例都不乏有 `box-shadow` 的身影：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/9cf41f2031614f58ae2cdb1ef1167a6a~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1640&h=814&s=10841281&e=gif&f=97&b=040404)

  


在很多场景之下，使用 `box-shadow` 给元素添加边框效果也是 Web 开发者常用方案之一，因为它与 `outline` 有点类似，不是元素盒模型的一部分，并且不会影响其他元素的布局。

  


虽然 `box-shadow` 属性有多个参数，但我们指定一个阴影颜色和阴影扩展（或收缩）半径 `spread` ，同时将其他参数设置为 `0` ，就可以创建一个不带任何偏移和模糊的阴影，即一个实色阴影。在视觉上，它看起来就像一个 `border-style` 或 `outline-style` 为 `solid` 创建的边框或轮廓。

  


```CSS
.shadow {
    box-shadow: 0 0 0 var(--r) var(--c);
}
```

  


与此同时，我们借助多阴影的特性，就能很容易地为元素设置一个双重边框：

  


```CSS
.box {
    --r: round(0.5em, 1px);
    --c1: #f36;
    --c2: color-mix(in srgb, var(--c1), #000 60%);
    box-shadow: 
        0 0 0 var(--r) var(--c1),
        0 0 0 calc(2 * var(--r)) var(--c2);
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/8a599cd23bfe4713bdcc39913fef99a7~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1129&s=172437&e=jpg&b=050505)

  


> Demo 地址：https://codepen.io/airen/full/jOjoByg

  


`box-shadow` 创建的阴影有内阴影（向内收缩）和外阴影（向外扩展）之分：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/a4008ca20f8346f58629d6e24fecc9f0~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1129&s=212525&e=jpg&b=fec300)

  


我们可以通过内外阴影相互结合来快速构建一个双重边框：

  


```CSS
.box {
    box-shadow: 
        inset 0 0 0 var(--r) var(--c1),
        0 0 0 var(--r) var(--c2);
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/187c61144a63404db32eca566775887a~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1129&s=254621&e=jpg&b=060606)

  


> Demo 地址：https://codepen.io/airen/full/NWZVpjV

  


你也可以只使用单个阴影与 `border` 或 `outline` 结合来创建双重边框：

  


```CSS
.box {
    border: var(--r) solid var(--c1);
    box-shadow: 0 0 0 var(--r) var(--c2);
}
    
.box {
    outline: var(--r) solid var(--c1);
    box-shadow: 0 0 0 calc(2 * var(--r)) var(--c2);
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/f63a8b253b304acda2f59ba20c9aa05f~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1129&s=313666&e=jpg&b=050505)

  


> Demo 地址：https://codepen.io/airen/full/XWLwMeN

  


### 基于布局框创建双重边框

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/8b01c9d628ae498bb7ce1c39bc20054e~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1129&s=346507&e=jpg&b=fcf3f2)

  


之前我们提到过，在 CSS 中，任何一个元素都是一个盒子框，而且每个盒子框在都有 `margin-box` 、`border-box` 、`padding-box` 和 `content-box` 等布局框，这几个布局框对应的就是 CSS 盒模型中 `margin` 、`border` 、`padding` 和 `content` 的限制区域。

  


元素的背景 `background` 默认是从 `border-box` 最外边缘开始，并填充整个元素。那么，我们可以借助该特性给元素设置双重边框。例如：

  


```CSS
.box {
    --b: round(0.5em, 1px);
    --c1: #f36;
    --c2: color-mix(in srgb, var(--c1), #000 60%);
    --c3: hsl(90 90% 30% / .5);
    
    border: var(--b) solid var(--c1);
    padding: var(--b);
    background:
        linear-gradient(var(--c3), var(--c3)) content-box,
        linear-gradient(var(--c2),var(--c2)) padding-box;
}
```

  


上面代码中，我们使用 `linear-gradient` 创建了两个纯色渐变，其中一个限制在 `content-box` ，另一个限制在 `padding-box` 。注意，这里使用的是 `background` 简写属性，它的 `background-clip` 的值为 `content-box, padding-box` 。这个时候，位于 `padding-box` 中的渐变色与 `border` 组合在一起，元素在视觉上具有双重边框的效果：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/b894ba7b56e541328bb4aa6bc5741c22~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=890&s=166721&e=jpg&b=050505)

  


> Demo 地址：https://codepen.io/airen/full/KKjLmwg

  


我猜你在编写 CSS 的时候，肯定用过[ CSS 渐变](https://juejin.cn/book/7199571709102391328/section/7199845781149810727)和 `background-*` 以及[多背景](https://juejin.cn/book/7199571709102391328/section/7199845838103773195#heading-8)等特性[制作过很多不同的纹理背景](https://css-pattern.com/)：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/c77d73ccc0a64114a80a3dd2473f0844~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1280&h=720&s=141705&e=jpg&b=141823)

  


在这里，我们也可以利用这些 CSS 特性来制作双重边框。例如：

  


```CSS
.box {
    --b: round(0.5em, 1px);
    --c1: #f36;
    --c2: color-mix(in srgb, var(--c1), #000 60%);
    --c3:hsl(90 90% 30% / 0.5);
    
    border: calc(2 * var(--b)) solid #0000;
    background:
        linear-gradient(to bottom, var(--c1) 0 50%, var(--c2) 50% 100%) left var(--b) top 0 / calc(100% - 2 * var(--b)) calc(2 * var(--b)) no-repeat border-box,
        linear-gradient(to top, var(--c1) 0 50%, var(--c2) 50% 100%) left var(--b) bottom 0 / calc(100% - 2 * var(--b)) calc(2 * var(--b))   no-repeat border-box,
        linear-gradient(to left, var(--c1) 0 50%, var(--c2) 50% 100%) right 0 top 0 / calc(2 * var(--b))  100% no-repeat border-box,
        linear-gradient(to right, var(--c1) 0 50%, var(--c2) 50% 100%) left 0 bottom 0 / calc(2 * var(--b))  100% no-repeat border-box,
        linear-gradient(var(--c3),var(--c3)) padding-box;
}
```

  


上面的代码使用五个线性渐变 `linear-gradient` ，其中四个 `linear-gradient` 分别绘制了元素的四条边的双重边框，它们都填充在 `border-box` 中，另一个 `linear-gradient` 线性渐变，填充在 `padding-box` 中：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/6cab969047aa4a1b8ad7f3dd06a5154d~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=2190&s=409618&e=jpg&b=fffefe)

### 使用伪元素创建双重边框

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/642de6865e0b4630939baa3ca493202b~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1129&s=201613&e=jpg&b=111111)

  


之前提到过，大部分元素都具有伪元素 `::before` 和 `::after` ，它们使得每个元素具备多个盒子框。这使得我们有更多控制双重边框的可能性。比如，元素和一个伪元素的 `border` 组合在一起构成一个双重边框：

  


```CSS
.box {
    border: var(--b) solid var(--c1);
    background: var(--c3);
    position: relative;
    
    &::before {
        content: "";
        position: absolute;
        inset: calc(-2 * var(--b));
        border: var(--b) solid var(--c2);
        z-index: -1;
        pointer-events: none;
    }
}
```

  


你也可以将元素的 `border` 和带有背景颜色的伪元素组合在一起构建一个双重边框：

  


```CSS
.box {
    border: var(--b) solid var(--c1);
    background: var(--c3);
    position: relative;
    
    &::before {
        content: "";
        position: absolute;
        inset: calc(-2 * var(--b));
        background-color: var(--c2);
        z-index: -1;
        pointer-events: none;
    }
}
```

  


上面这两种不同的方式使元素获得一致的双重边框效果：

-   元素的 `border` 与一个伪元素的 `border` 、`outline` 、`box-shadow` 和背景中的某一个相互结合
-   元素的 `outline` 与一个伪元素的 `border` 、`outline` 、`box-shadow` 和背景中的某一个相互结合
-   元素的 `box-shadow` 与一个伪元素的 `border` 、`outline` 、`box-shadow` 和背景中的某一个相互结合

## 创建多重边框

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/861c0bcd34304060a47a9ba249aa6da2~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1280&h=720&s=59521&e=jpg&b=f6f2f1)

  


既然你已经掌握了 CSS 创建双重边框的方法，那么你肯定知道如何创建多重边框。

  


### 使用 `box-shadow` 创建多重边框

  


给 `box-shadow` 设置多个阴影值是创建多重边框的最简单方法之一：

  


```CSS
.box {
    --b: round(0.5em, 1px);
    --c: #f36;
    
    background: hsl(from var(--c) 120 s 20% / .5);
    box-shadow:
        0 0 0 var(--b) color-mix(in srgb, var(--c), #000 0%),
        0 0 0 calc(2 * var(--b)) color-mix(in srgb, var(--c), #000 15%),
        0 0 0 calc(3 * var(--b)) color-mix(in srgb, var(--c), #000 30%),
        0 0 0 calc(4 * var(--b)) color-mix(in srgb, var(--c), #000 45%),
        0 0 0 calc(5 * var(--b)) color-mix(in srgb, var(--c), #000 60%),
        0 0 0 calc(6 * var(--b)) color-mix(in srgb, var(--c), #000 75%),
        0 0 0 calc(7 * var(--b)) color-mix(in srgb, var(--c), #000 90%);
}
```

  


上面代码中的阴影扩展半径始终是“边框厚度”倍数，它创建了一个七重边框效果：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/3c418d6c9a41478f8cde94735eddf011~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1138&s=232490&e=jpg&b=050505)

  


> Demo 地址：https://codepen.io/airen/full/VwJOBNp

  


你只需要调整 `box-shadow` 阴影扩展半径的值和阴影数量，就可以创建出你面要多重边框的变体。

  


使用 `box-shadow` 创建多重边框是简便，但随着阴影列表值的增加，性能也将受到一定的影响，尤其是动画化`box-shadow` 时。因此，在考虑使用 `box-shadow` 构建多重边框时，需要观察它对 Web 性能的影响！

  


### 使用伪元素创建多重边框

  


CSS 中创建多重边框的另一种方法是结合伪元素，并利用元素的 `border` 、`padding` 、`outline` 以及布局边框来创建多重边框。注意，这里没有提 `box-shadow` ，并不是说不能使用 `box-shadow` ，那是因为，使用 `box-shadow` 的话，就没有必要使用这些复杂的组合来创建多重边框。换句话说，如果你担心多重阴影会影响到 Web 的渲染性能，那么这个方法来替代 `box-shadow` 是一个不错的选择。

  


```CSS
.box {
    --b: round(0.5em, 1px);
    --c: #f36;
    --c1: color-mix(in srgb, var(--c), #000 0%);
    --c2: color-mix(in srgb, var(--c), #000 15%);
    --c3: color-mix(in srgb, var(--c), #000 30%);
    --c4: color-mix(in srgb, var(--c), #000 45%);
    --c5: color-mix(in srgb, var(--c), #000 60%);
    --c6: color-mix(in srgb, var(--c), #000 75%);
    
    position: relative;
    border: var(--b) solid var(--c2);
    padding: var(--b);
    outline: var(--b) solid var(--c3);
    background:
        linear-gradient(hsl(from var(--c) 120 s 20%), hsl(from var(--c) 120 s 20%)) content-box,
        linear-gradient(var(--c1), var(--c1)) padding-box;
    
    &::before {
        content: "";
        position: absolute;
        inset: calc(-4 * var(--b));
        padding: inherit;
        background:
            linear-gradient(#0000, #0000) content-box,
            linear-gradient(var(--c4), var(--c4)) padding-box;
        z-index: -1;
        border: var(--b) solid var(--c5);
        outline: var(--b) solid var(--c6);
    }
}
```

在这个示例中，我们只借用了一个 `::before` 元素，你可以根据自己的需要再借用一个 `::after` 元素，创建更多重的边框。

当然，在某些特殊的场景下，我们可以使用 CSS 渐变作为边框图片，比如下面这个示例，使用 `repeating-radial-gradient` 创建的渐变作为边框图片：

    border-image:
        repeating-radial-gradient(

}
