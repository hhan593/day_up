用户头像（Avatar）是 Web 上常见的 UI 组件之一，其中不乏带有镂空效果。使用 CSS 或 SVG 给用户头像添加镂空效果有很多种方法，但每种方法都有其优缺点。在这节课中，我将和大家一起来探索这些解决方案。

  


为了能更好的阅读和理解接下来的内容，你需要具备以下基础知识：

  


-   [CSS 渐变](https://juejin.cn/book/7199571709102391328/section/7199845781149810727)
-   [CSS 剪切](https://juejin.cn/book/7223230325122400288/section/7259668885224456252#heading-1)：`clip-path`
-   [CSS 遮罩](https://juejin.cn/book/7223230325122400288/section/7259668885224456252#heading-9)：`mask`
-   [CSS 三角函数](https://juejin.cn/book/7223230325122400288/section/7242216512176521277)
-   [SVG 路径](https://juejin.cn/book/7341630791099383835/section/7368317966821425190)： `<path>`
-   [SVG 剪切](https://juejin.cn/book/7341630791099383835/section/7362748816769941540) ：`<clipPath>`
-   [SVG 遮罩](https://juejin.cn/book/7341630791099383835/section/7366549423813296165)：`<mask>`

## 镂空的用户头像

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/9e0fc19954f544f0a58a78b94e0fa9a1~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1193&s=758890&e=jpg&b=060606)

  


上图展示了两种在 Web 上常见的用户头像镂空的效果。左侧展示的用户头像有一个状态指示器，为了放置这个状态指示器，头像被裁剪了部分；右侧是展示的用户头像镶嵌在一卡片上，卡片为了能放置用户头像，它也被裁剪了部分。

  


据我的观察，大部分 Web 开发者在创建这类具有内凹口的 UI 效果时，往往会借助 CSS 的 `border` 、`outline` 或 `box-shadow` 等属性给元素添加一个“边框”效果，然后通过绝对定位将其层叠在目标元素上。例如：

  


```HTML
<div class="card">
    <figure class="avatar cut-out">
        <img src="http://i.pravatar.cc/500?img=2" alt="" />
    </figure>
    <div class="content">
        <h3>Tony Geary</h3>
        <p>Expert CSS Hacker</p>
    </div>
</div>

<div class="card ">
    <figure class="avatar punch-out">
        <img src="http://i.pravatar.cc/500?img=6" alt="" />
    </figure>
    <div class="content">
        <p>Lorem ipsum...</p>
    </div>
</div> 
```

  


```CSS
.card {
    position: relative;

    &:has(.punch-out) {
        padding-bottom: 3rem;
    }
}

.cut-out {
    position: relative;

    &::after {
        content: "";
        position: absolute;
        width: 1rem;
        aspect-ratio: 1;
        border-radius: 50%;
        background: #52d55a;
        border: 0.125em solid #fff;
        right: -0.25em;
        bottom: 0.5em;
    }
}

.punch-out {
    position: absolute;
    top: calc(100% - 2rem);
    right: 2rem;
    border: 0.5em solid #000;
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/ece7079b860947069277d6ebd1d01943~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=911&s=219075&e=jpg&b=060606)

  


乍一看，效果似乎还能接受，但我们的组件不仅仅一个状态。当你在悬浮状态（`:hover`）或在暗黑模式下，如果不调整边框的颜色，那么效果看起来就会很奇怪。以左侧的带有指示器的用户头像为例：

  


-   悬浮状态：卡片背景会改变，白色边框看起来会很奇怪
-   暗黑模式下：白色边框会与黑色背景冲突

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/cade904e29314041b18dd4ad8e72e74f~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1714&s=599374&e=jpg&b=111111)

  


当然，我们可以通过 CSS 自定义属性使指示器边框颜色在悬停和暗黑模式下与卡片背景保持一致来避免这种现象发生：

  


```CSS
:root {
    --card-bg: #fff;
    --card-color: #000;
    --card-hover: color-mix(in srgb, var(--card-bg), #000 20%);

    @media (prefers-color-scheme: dark) {
        --card-bg: #454545;
        --card-color: #fff;
    }
}

.card {
    background: var(--card-bg);
    color: var(--card-color);

    &:hover {
        background: var(--card-hover);

        .cut-out::after {
            border-color: var(--card-hover);
        }
    }
}

.cut-out {
    position: relative;

    &::after {
        background: #52d55a;
        border: 0.125em solid var(--card-bg);
    }
}
```

> Demo 地址：https://codepen.io/airen/full/OJKPeYL

  


我们可以使用同样的方式来处理右则用户头像的边框效果，在这个示例中它需要与 `.card` 的父元素 `body` 的 `background-color` 保持一致。虽然这种方式允许你创建一个“类似”镂空的效果，但并不是最好的解决方案。

  


事实上，我们有着更多的更好的解决方案，例如：

  


-   CSS 遮罩
-   SVG 裁剪
-   SVG 遮罩

## 创建带有状态指示器的用户头像

  


我们先来看第一个示例，创建带有状态指示器的用户头像。

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/6441ab94e06f498cb6a8b22edcb9df26~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1714&s=373467&e=jpg&b=090909)

  


### 方案一：CSS 遮罩

  


你是否还记得之前介绍的《[CSS 图形：切割元素角落](https://juejin.cn/book/7395119173656903715/section/7400975207551139875)》一课中的[圆形切割](https://juejin.cn/book/7395119173656903715/section/7400975207551139875#heading-1)，通过组合使用 [CSS 遮罩 `mask`](https://juejin.cn/book/7223230325122400288/section/7259668885224456252#heading-9) 和径向渐变 `radial-gradient()` ，可以使用户头像带有圆形的镂空效果。

  


接下来，让我们来看看如何操作。

  


通过使用径向渐变 `radial-gradient()` ，我们可以绘制一个圆形，然后将剩余的空间填充为另一种颜色。如下图所示：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/23d35f45d91145fc997182f12e8ce4ef~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1262&s=237106&e=jpg&b=ffffff)

  


```CSS
img {
    --w: round(6rem, 1px); /* 控制用户头像大小 */
    --s: round(0.875em, 1px); /* 控制状态指示器大小 */
    --g: round(0.125em, 1px); /* 控制状态指示器位置 */
    --bg: #898ae3;
    --_g: calc(var(--s) + var(--g));
    --b: 4px;

    background: radial-gradient(
        circle var(--s) at right var(--_g) bottom var(--_g),
        #feba0b var(--s),
        #898ae3 0
    );
}
```

  


接下来，我们需要将圆的颜色改变透明色 `#0000` ，并为元素 `.avatar` 添加 `border-radius` ：

  


```CSS
img {
    --w: round(6rem, 1px); /* 控制用户头像大小 */
    --s: round(0.875em, 1px); /* 控制状态指示器大小 */
    --g: round(0.125em, 1px); /* 控制状态指示器位置 */
    --bg: #898ae3;
    --_g: calc(var(--s) + var(--g));
    --b: 4px;

    background: radial-gradient(
        circle var(--s) at right var(--_g) bottom var(--_g),
        #0000 var(--s),
        var(--bg) 0
    );
}
```

  


注意，你可能看到了，`radial-gradient()` 用作于元素的 `background` ，并没有用于 `mask` 。这样做是为了让大家更易于理解 `radial-gradient()` 是如何创建遮罩图的。你现在看到的效果大致如下：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/93574130386e4d77aa80b524addf3546~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2375&h=1262&s=280925&e=jpg&b=8f8fe3)

  


现在，你只需要将上面代码中的 `background` 替换为 `mask` 即可，这样就可以实现用户头像镂空的效果：

  


```CSS
img {
    --w: round(6rem, 1px); /* 控制用户头像大小 */
    --s: round(0.875em, 1px); /* 控制状态指示器大小 */
    --g: round(0.125em, 1px); /* 控制状态指示器位置 */
    --bg: #898ae3;
    --_g: calc(var(--s) + var(--g));
    --b: 4px;

    mask: radial-gradient(
        circle var(--s) at right var(--_g) bottom var(--_g),
        #0000 var(--s),
        var(--bg) 0
    );
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/c3bc8fa7892546b38d27531a400fec57~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=3588&h=1262&s=1047709&e=jpg&b=fcf8f8)

  


你可能已经注意到了，`radial-gradient()` 绘制的透明圆圈的位置是通过 `--s` 和 `--g` 计算获得，并且由 `top` 、`right` 、`bottom` 和 `left` 关键词来决定其位置：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/d06451b6038e453aa0b8cdb98366c51d~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1262&s=341206&e=jpg&b=ffffff)

  


注意，示例中 `radial-gradient()` 中的 `at` 关键词之后的 `right var(--_g) bottom var(--_g)`，它有点类似于 `background-position` 中的 `right` 和 `bottom` 关键词。具体的意思是，`circle` 的圆心位置，距右侧边缘的距离等于 `--_g` ，同时距底部边缘的距离等于 `--_g` ，其中 `--_g` 变量的值等于 `--s + --g` ，即 `calc(var(--s) + var(--g))` 。

  


到目前为止，我们仅仅使用 `mask` 和 `radial-gradient()` 使用户头像具有一个镂空的效果，但状态指示器还是需要依赖伪元素 `::before` 或 `::after` 来构建：

  


```CSS
.element::after {
     content: "";
     position: absolute;
     width: calc((var(--s) - var(--b)) * 2);
     aspect-ratio: 1;
     border-radius: 50%;
     color: #52d55a;
     background: currentColor padding-box;
     border: var(--b) solid #0000;
     right: var(--g);
     bottom: var(--g);
     z-index: 2;
}
```

  


关键之处需要将状态指示器 `::after` 定位与 `radial-gradient()` 创建的圆相匹配。完整的 CSS 代码如下：

  


```CSS
.avatar {
    --w: round(6rem, 1px); /* 控制用户头像大小 */
    --s: round(0.875em, 1px); /* 控制状态指示器大小 */
    --g: round(0.125em, 1px); /* 控制状态指示器位置 */
    --bg: #898ae3;
    --_g: calc(var(--s) + var(--g));
    --b: 4px;

    width: var(--w);
    aspect-ratio: 1;
    border-radius: 50%;
    position: relative;

    img {
        display: block;
        width: 100%;
        height: 100%;
        border-radius: inherit;
        mask: radial-gradient(
            circle var(--s) at right var(--_g) bottom var(--_g),
            #0000 var(--s),
            var(--bg) 0
        );
    }

    &::after {
        content: "";
        position: absolute;
        width: calc((var(--s) - var(--b)) * 2);
        aspect-ratio: 1;
        border-radius: 50%;
        color: #52d55a;
        background: currentColor padding-box;
        border: var(--b) solid #0000;
        right: var(--g);
        bottom: var(--g);
        z-index: 2;
    }
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/a88adaf59d04412bb32e6fd2e4acdcd0~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1100&h=478&s=671219&e=gif&f=255&b=030303)

  


> Demo 地址：https://codepen.io/airen/full/WNVQrmp

  


这效果几乎接近我们期望的效果。如果你是跟着我一起敲入了上面的代码，你会发现我们需要不断的调整 `--g` 的大小，才能使 `radial-gradient()` 绘制的圆在元素所以圆周上。其实，在现代 CSS 中有一种更好的方式，可以轻易将 `radial-gradient()` 绘制的圆位于元素的圆周上，那就是借助 [CSS 的三角函数](https://juejin.cn/book/7223230325122400288/section/7242216512176521277)来设置圆心的位置：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/77159e0066df415ebe1a68bb1d3bd641~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1262&s=441731&e=jpg&b=ffffff)

  


如上图所示，我们可以通过角度 `--a` 和元素的宽度 `--w` 来获取 `radial-gradient()` 渐为绘制的圆的圆心位置：

  


```
posX = 50% + 50% * cos(--a)
posY = 50% + 50% * sin(--a)
```

  


将其放入到 CSS 中，大致如下：

  


```CSS
.avatar {
    --w: round(6rem, 1px); /* 控制用户头像大小 */
    --bg: #898ae3;
    
    --a: 45deg;
    --s: round(0.875em, 1px); /* 控制状态指示器大小 */
    --posX: calc(50% + 50% * cos(var(--a)));
    --posY: calc(50% + 50% * sin(var(--a)));
    --b: 4px;

    width: var(--w);
    aspect-ratio: 1;
    border-radius: 50%;
    position: relative;

    img {
        display: block;
        width: 100%;
        height: 100%;
        border-radius: inherit;
        mask: radial-gradient(
            circle var(--s) at var(--posX) var(--posY),
            #0000 var(--s),
            var(--bg) 0
        );
    }

    &::after {
        content: "";
        position: absolute;
        width: calc((var(--s) - var(--b)) * 2);
        aspect-ratio: 1;
        border-radius: 50%;
        color: #52d55a;
        background: currentColor padding-box;
        border: var(--b) solid #0000;
        top: calc(var(--posY) - var(--s));
        left: calc(var(--posX) - var(--s));
        z-index: 2;
    }
}
```

  


这个时候，你只需要调整 `--a` 的角度值，就可以调整状态指示器的位置：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/ce21e7ec804648e8bde39e49d3587cf3~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1098&h=474&s=1404651&e=gif&f=378&b=030303)

  


> Demo 地址：https://codepen.io/airen/full/MWNayyX

  


正如你所看到的，现在这个方案能满足我们大部分的需求，但状态指示器不得不依赖伪元素 `::after` 来构建。不过，这并不是绝对的，因为我们可以使用现代 CSS 的一些特性，仅在一个元素上就可以构建出带有状态指示器的用户头像。

  


```CSS
@property --s {
    syntax: "<length>";
    initial-value: 0px;
    inherits: true;
}

.avatar {
    --w: 88px; /* 控制用户头像尺寸 */
    --s: 10px; /* 控制状态指示器尺寸 */
    --g: 1px;  /* 控制状态指示器与图片之间的间距*/
    --bg: #09acef; /* 控制状态指示器颜色 */
    box-sizing: content-box;
    width: var(--w);
    aspect-ratio: 1;
    padding: 0 calc(var(--w) / sqrt(2)) calc(var(--w) / sqrt(2)) 0;
    margin: 0 calc(-1 * var(--w) / sqrt(2)) calc(-1 * var(--w) / sqrt(2)) 0;
    outline: var(--s) solid var(--bg);
    outline-offset: calc(-1px * infinity);
    mask: 
        radial-gradient(#000 calc(var(--s) - 1px), #0000 var(--s)),
        radial-gradient(#000 70%, #0000 71%) content-box subtract,
        radial-gradient(
            #000 calc(sqrt(2) * var(--s) + var(--g)),
            #0000 calc(sqrt(2) * var(--s) + var(--g) + 1px)
        );
    clip-path: circle(calc(var(--w) / 2 + var(--s))) content-box;
    cursor: pointer;
    transition: --s 0.5s;
    
    .card:hover & {
        --s: 12px;
    }
}  
```

  


你看到的效果如下：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/c9bdc9888f084ac3bb393f646d19a44d~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1204&h=520&s=258734&e=gif&f=108&b=030303)

  


> Demo 地址：https://codepen.io/airen/full/YzmyqLG

  


请注意，状态指示器的颜色是由 `outline-color` 来控制的。与之前示例类似，它的镂空效果依旧是 `mask` 实现，只不过要更复杂一些，我们使用了多个径向渐变，[并由 `mask-composite` 对遮罩图进行合成操作](https://juejin.cn/book/7223230325122400288/section/7259668885224456252#heading-16)。另外，在这个示例中圆形效果并不是由 `border-radius` 实现的，而是[使用 `clip-path` 的 `circle()` 裁剪出来的](https://juejin.cn/book/7223230325122400288/section/7259668885224456252#heading-1)。

  


你可能已经发现了，状态指示器刚好位于图片的对角线上，并且它的方向将由 `padding` 和 `margin` 来决定。这也造成了状态指示器只能位于元素的对角线上，你可以像下面这样来调整状态指示器的位置：

  


```CSS
@property --s {
    syntax: "<length>";
    initial-value: 0px;
    inherits: true;
}

.avatar {
    --w: 88px;
    --s: 10px;
    --g: 1px;
    --bg: #09acef;
    
    box-sizing: content-box;
    width: var(--w);
    aspect-ratio: 1;
    
    --p: calc(var(--w) / sqrt(2));
    --m: calc(-1 * var(--w) / sqrt(2));

    /* 右上角 */
    padding: var(--p) 0 0 var(--p);
    margin: var(--m) 0 0 var(--m);

    outline: var(--s) solid var(--bg);
    outline-offset: calc(-1px * infinity);
    mask: 
        radial-gradient(#000 calc(var(--s) - 1px), #0000 var(--s)),
        radial-gradient(#000 70%, #0000 71%) content-box subtract,
        radial-gradient(
            #000 calc(sqrt(2) * var(--s) + var(--g)),
            #0000 calc(sqrt(2) * var(--s) + var(--g) + 1px)
        );
    clip-path: circle(calc(var(--w) / 2 + var(--s))) content-box;
    cursor: pointer;
    transition: --s 0.5s;

    .card:hover & {
        --s: 12px;
    }

    /* 右上角 */
    .card:nth-child(2) & {
        padding: var(--p) var(--p) 0 0;
        margin: var(--m) var(--m) 0 0;
        --bg: #f36;
    }

    /* 右下角*/
    .card:nth-child(3) & {
        padding: 0 var(--p) var(--p) 0;
        margin: 0 var(--m) var(--m) 0;
        --bg: #f50ace;
    }

    /* 左下角*/
    .card:nth-child(4) & {
        padding: 0 0 var(--p) var(--p);
        margin: 0 0 var(--m) var(--m);
        --bg: #cef90c;
    }
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/352e416869bc4a1293d4a1acf5d612cb~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=914&h=504&s=1442844&e=gif&f=276&b=040404)

  


> Demo 地址：https://codepen.io/airen/full/NWQGNVB

  


正如你所看到的，这个方案虽然不需要依赖伪元素 `::before` 或 `::after` 来制作状态指示器，但它的代码相对比较复杂。除此之外，要调整状态指示器的位置比较困难。还有一点，要制作仅带边框的状态指示器，它就显得先天不足了。

  


### 方案二：SVG 裁剪

  


这个方案使用了 SVG 和 CSS 的结合。首先，我们需要创建一个路径并将其导出为 SVG。[你可以使用诸如 Figma 这样的图形编辑软件绘制出需要的路径，并导出 SVG](https://juejin.cn/book/7341630791099383835/section/7343645889896022050)：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/48116aadb3ab46e9bc38e5b1fc98fde9~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1570&h=764&s=5785558&e=gif&f=527&b=f3f3f3)

  


你可以使用 [SVGOMG 工具](https://jakearchibald.github.io/svgomg/)对导出的 [SVG 代码进行优化](https://juejin.cn/book/7341630791099383835/section/7368114202180845605)：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/b2e63c8ac40f40159fa3b8e799f71b04~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=3344&h=1800&s=365412&e=jpg&b=f6ba43)

  


> URL:https://jakearchibald.github.io/svgomg/

  


```XML
<svg viewBox="0 0 1189 1189" fill="none">
    <path fill="#FEBA0B" fill-rule="evenodd" d="M1067.06 955.267C1143.56 855.219 1189 730.167 1189 594.5 1189 266.167 922.833 0 594.5 0S0 266.167 0 594.5 266.167 1189 594.5 1189c137.195 0 263.536-46.47 364.139-124.54C945.834 1050.71 938 1032.27 938 1012c0-42.526 34.474-77 77-77 20.07 0 38.35 7.682 52.06 20.267Z" clip-rule="evenodd"/>
</svg>
```

  


接下来，我们需要复制路径 `<path>` 的 `d` 属性的值，并将其转换为相对单位。默认情况下，SVG 路径点是绝对值，这意味着当宽度和高度发生变化时，它们可能会拉伸。为了解决这个问题，[我们可以使用一个在线工具来完成这项转换](https://yoksel.github.io/relative-clip-path/)：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/234542edb1ea438487d355a3a22b1d0a~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=3340&h=1678&s=384609&e=jpg&b=ffffff)

  


> URL: https://yoksel.github.io/relative-clip-path/

  


然后，我们可以将转换为后的路径作为 [SVG 的裁剪路径 `<clipPath>`](https://juejin.cn/book/7341630791099383835/section/7362748816769941540) ，并将 [SVG 内联到 HTML 中](https://juejin.cn/book/7341630791099383835/section/7344089098363076620)：

  


```XML
<svg class="sr-only">
    <clipPath id="circle" clipPathUnits="objectBoundingBox">
        <path d="M0.897,0.803 C0.962,0.719,1,0.614,1,0.5 C1,0.224,0.776,0,0.5,0 S0,0.224,0,0.5 S0.224,1,0.5,1 C0.615,1,0.722,0.961,0.806,0.895 C0.795,0.884,0.789,0.868,0.789,0.851 C0.789,0.815,0.818,0.786,0.854,0.786 C0.871,0.786,0.886,0.793,0.897,0.803" />
    </clipPath>
</svg>
```

  


`clipPathUnits` 属性的值为 `objectBoundingBox`，这意味着路径内的值是相对于应用 `clip-path` 的元素的边界框的相对值。

  


```CSS
.avatar {
    clip-path: url("#circle");
}
```

  


这样，用户头像就具有一个镂空的效果：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/b757f94ea16449d4bea2292c74227a7d~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2330&h=1262&s=851273&e=jpg&b=fbf5f5)

  


到目前为止，结合 SVG 裁剪路径 `<clipPath>` 和 CSS 的 `clip-path` 使用户头像具有圆形镂空的效果，但对于状态指示器，依旧需要使用伪元素 `::before` 或 `::after` 来创建。不过，它的定位和大小相对而言会比较麻烦一些，需要使用浏览器开发者工具不断的调试，才能获得一个较佳的位置。

  


有一个小技巧是，在创建 SVG 裁剪路径时，记录小圆圈的关键参数，例如它的圆心位置与起点位置（`x` 轴与大圆交叉点）的夹角。然后通过 CSS 的三角函数来计算出小圆的圆心位置，从而获得状态指示器 `left` 和 `right` 的偏移量。

  


```CSS
.avatar {
    width: 120px;
    aspect-ratio: 1;
    border-radius: 50%;
    position: relative;
    
    img {
      clip-path: url(#circle);
    }
    
    &::after {
        content: "";
        position: absolute;
        background: lime;
        width: 10px;
        aspect-ratio: 1;
        border-radius: 50%;
        z-index: 2;
        top: calc(50% + 50% * sin(45deg) - 5px);
        left: calc(50% + 50% * cos(45deg) - 5px);
    }
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/4ae00b765d594db0896ff69d2509bed9~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1134&h=514&s=394275&e=gif&f=143&b=040404)

  


> Demo 地址：https://codepen.io/airen/full/eYqpzwB

  


虽然效果看上去还不错，但这种方案灵活度相对比较弱，比如你要调整状态指示器位置时，你不得不重新调整 SVG 的裁剪路径。除此之外，当状态指示器具有不同状态时，例如状态指示器不是一个纯色的圆，而是一个仅有边框的圆，那么这种方案就很难实现。

  


### 方案三：SVG 遮罩

  


这种方案与上一种方案类似，只不过是将 [SVG 遮罩](https://juejin.cn/book/7341630791099383835/section/7366549423813296165#heading-5)（`<mask>`）和 [CSS 遮罩 `mask`](https://juejin.cn/book/7223230325122400288/section/7259668885224456252#heading-9) 相结合。同样的，你需要使用设计软件获得遮罩图形的路径，并将其转换为相对单位。然后将相应的路径 `<path>` 嵌套在 `<mask>` 中。

  


```XML
<svg class="sr-only">
    <mask id="circle" maskUnits="objectBoundingBox" maskContentUnits="objectBoundingBox">
        <path d="M0.897,0.803 C0.962,0.719,1,0.614,1,0.5 C1,0.224,0.776,0,0.5,0 S0,0.224,0,0.5 S0.224,1,0.5,1 C0.615,1,0.722,0.961,0.806,0.895 C0.795,0.884,0.789,0.868,0.789,0.851 C0.789,0.815,0.818,0.786,0.854,0.786 C0.871,0.786,0.886,0.793,0.897,0.803" fill="#fff"/>
    </mask>
</svg>
```

  


注意，也是需要将 SVG 内联到 HTML 中。

  


接下来，在元素上使用 CSS 的 `mask` 来引用 SVG 的 `<mask>` 定义的遮罩图：

  


```CSS
.avatar {
    mask: url("#circle");
}
```

  


然后使用 `::before` 或 `::after` 来创建状态指示器：

  


```CSS
.avatar {
    width: 120px;
    aspect-ratio: 1;
    border-radius: 50%;
    position: relative;
    
    img {
        mask: url(#circle);
    }
    
    &::after {
        content: "";
        position: absolute;
        background: lime;
        width: 10px;
        aspect-ratio: 1;
        border-radius: 50%;
        z-index: 2;
        top: calc(50% + 50% * sin(45deg) - 5px);
        left: calc(50% + 50% * cos(45deg) - 5px);
    }
}
```

  


最终获得的效果与 SVG 裁剪是相似的：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/2cf5aeccc65b406885ef98b4e76529da~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1134&h=514&s=394275&e=gif&f=143&b=040404)

  


> Demo 地址：https://codepen.io/airen/full/XWvmjEZ

  


效果几乎一样，所存在的缺陷也几乎相同。不过，我们完全可以使用 SVG 来创建带有状态指示器的用户头像，并将它们内联到 HTML 中。

  


```HTML
<div class="card">
    <svg viewBox="0 0 100 100" class="avatar">
        <defs>
            <mask id="circle">
                <circle cx="50" cy="50" r="50" fill="white" />
                <circle cx="86%" cy="86%" r="18" fill="black" />
            </mask>
        </defs>
        <g>
            <image mask="url(#circle)" x="0" y="0" height="100%" width="100%" href="http://i.pravatar.cc/500?img=2" />
            <circle cx="86%" cy="86%" r="14" fill="lime" />
        </g>
    </svg>

    <div class="content">
        <h3>Tony Geary</h3>
        <p>Expert CSS Hacker</p>
    </div>
</div>
```

  


这样做的最大好处是，我们不必再担心状态指示器的位置不好调整。

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/0c7eca9463584419bbedaede1418a83a~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1194&s=302781&e=jpg&b=171717)

  


> Demo 地址：https://codepen.io/airen/full/YzmyGdm

  


这是一个简单易用的解决方案，但前提是[你需要对 SVG 有所了解](https://s.juejin.cn/ds/iBJkVLRb/)，否则会略感吃力！

  


> 温馨提示：如果你从示接触过 SVG 方面的知识，但又对其感到好奇，想一探其究竟，那么可以考虑看看我的另一个小册《[深入浅出 SVG](https://s.juejin.cn/ds/iBJkVLRb/)》！

  


这几种解决方案相比较，我个人更趋向于内联 SVG ，在 `<mask>` 中嵌套 `<circle>` 元素创建遮罩图形，并组合 `<image>` 和 `<circle>` 创建带有状态指示器的用户头像。这个方案在所有浏览器上都能正常工作，并且可以根据需要调整状态指示器位置，或禁用遮罩！

  


## 创建用户头像内嵌卡片的效果

  


接下来我们来看第二个案例。使用现代 CSS 实现类似下图的效果：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/8f4ed38033634d128e1b67ab5bc948ed~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1194&s=533889&e=jpg&b=000000)

  


不难发现，卡片有一个内凹圆形，并且在这个内凹圆形上镶嵌了一个圆形的用户头像。就这个效果而言，创建带有内凹圆形背景的卡片才具挑战力。

  


如果你阅读过小册的《[切割元素角落](https://juejin.cn/book/7395119173656903715/section/7400975207551139875)》和《[创建花式边框效果](https://juejin.cn/book/7395119173656903715/section/7403197889905688585#heading-1)》两节课的内容，那么对于你来说就不再具有任何挑战力。在现代 CSS 中，我们只需要好好利用 CSS 的遮罩和径向渐变，就可以很容易获得带有内凹形状：

  


```CSS
.card {
    --r: 20px; /* 控制圆角部分 */
    --s: 40px; /* 控制切口的大小 */
    --a: 20deg; /* 控制曲率的深度 */

    &:nth-child(1) {
        --_m: 100% / calc(2 * var(--r)) var(--r) no-repeat radial-gradient(50% 100% at top, #000 calc(100% - 1px), #0000);
        --_d: (var(--s) + var(--r)) * cos(var(--a));
        
        mask: 
            calc(50% + var(--_d)) var(--_m), calc(50% - var(--_d)) var(--_m),
            radial-gradient(var(--s) at 50% calc(100% + sin(var(--a)) * var(--s)),#0000 100%,#000 calc(100% + 1px))  0 calc(var(--r) * (sin(var(--a)) - 1)) no-repeat,
            linear-gradient(90deg,#000 calc(50% - var(--_d)),#0000 0 calc(50% + var(--_d)),#000 0);
    }
    
    &:nth-child(3) {
        --_m: 0 / calc(2 * var(--r)) var(--r) no-repeat radial-gradient(50% 100% at bottom, #000 calc(100% - 1px), #0000);
        --_d: (var(--s) + var(--r)) * cos(var(--a));
      
        mask: 
            calc(50% + var(--_d)) var(--_m), calc(50% - var(--_d)) var(--_m),
            radial-gradient(var(--s) at 50% calc(-1 * sin(var(--a)) * var(--s)),#0000 100%,#000 calc(100% + 1px)) 0 calc(var(--r) * (1 - sin(var(--a)))) no-repeat,
            linear-gradient(90deg,#000 calc(50% - var(--_d)),#0000 0 calc(50% + var(--_d)),#000 0);
    }
    
    &:nth-child(2) {
        --_m: / var(--r) calc(2 * var(--r)) no-repeat radial-gradient(100% 50% at left, #000 calc(100% - 1px), #0000);
        --_d: (var(--s) + var(--r)) * cos(var(--a));
        
        mask: 
            100% calc(50% + var(--_d)) var(--_m), 100% calc(50% - var(--_d)) var(--_m),
            radial-gradient(var(--s) at calc(100% + sin(var(--a)) * var(--s)) 50%,#0000 100%,#000 calc(100% + 1px)) calc(var(--r) * (sin(var(--a)) - 1)) 0 no-repeat,
            linear-gradient(#000 calc(50% - var(--_d)),#0000 0 calc(50% + var(--_d)),#000 0);
    }
    
    &:nth-child(4) {
        --_m: / var(--r) calc(2 * var(--r)) no-repeat radial-gradient(100% 50% at right, #000 calc(100% - 1px), #0000);
        --_d: (var(--s) + var(--r)) * cos(var(--a));
        
        mask: 
            0 calc(50% + var(--_d)) var(--_m), 0 calc(50% - var(--_d)) var(--_m),
            radial-gradient(var(--s) at calc(-1 * sin(var(--a)) * var(--s)) 50%,#0000 100%,#000 calc(100% + 1px)) calc(var(--r) * (1 - sin(var(--a)))) 0 no-repeat,
            linear-gradient(#000 calc(50% - var(--_d)),#0000 0 calc(50% + var(--_d)),#000 0);
    }
}
```

> Demo 地址：https://codepen.io/airen/full/JjgYbLj

  


按照同样的方式，以内凹曲线方式来切割元素的顶角：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/d9e7fda8a17c433d87a3d89ce5be2a42~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1375&s=169023&e=jpg&b=202020)

  


> Demo 地址：https://codepen.io/airen/full/dyxYOgX

  


我在这里就不对上面这两个示例的代码进行阐述了，如果你阅读或理解上面的代码有一定的难度，请回过头阅读[圆形切割](https://juejin.cn/book/7395119173656903715/section/7400975207551139875#heading-1)和[曲线切割](https://juejin.cn/book/7395119173656903715/section/7400975207551139875#heading-2)这两部分内容，在这里对上面示例的代码有详细阐述！

  


知道如何创建这些曲线形状之后，我们就可以来创建我们所需要的卡片效果了。首先从 HTML 结构开始：

  


```HTML
<div class="card ">
    <figure class="avatar punch-out">
        <img src="http://i.pravatar.cc/500?img=6" alt="" />
    </figure>
    <div class="content">
        <p>Lorem ipsum dolor sit...</p>
    </div>
</div>
```

  


首先给 `.card` 创建内凹圆形，但我们不能直接在 `.card` 上应用 CSS 的 `mask` ，否则溢出卡片的内容都将被隐藏，比如我们这个示例的用户头像：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/f598a2d813694bbfa9190c8597bbbe91~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1039&s=188963&e=jpg&b=050505)

  


因此，我们需要将 `mask` 放置在一个伪元素 `::before` 或 `::after` 上来避免这种现象：

  


```CSS
@property --a {
    syntax: "<angle>";
    inherits: true;
    initial-value: 0deg;
}

.card {
    --r: 30px;
    --s: 80px;
    --a: 20deg;

    position: relative;
    z-index: 0;
    cursor: pointer;
    transition: 0.4s --a;
    padding: 0 1rem 1rem;
    
    &:hover {
        --a: 35deg;
    }

    &::before {
        content: "";
        position: absolute;
        z-index: -1;
        inset: calc(var(--s) / 2 + var(--r) / 4) 0 0 0;
        border-radius: 1em;
        background: var(--card-bg);
        --_m: 0 / calc(2 * var(--r)) calc(2 * var(--r)) no-repeat radial-gradient(50% 50%, #000 calc(100% - 1px), #0000);
        --_d: (var(--s) + var(--r)) * cos(var(--a));
        mask: 
            calc(50% + var(--_d)) var(--_m), calc(50% - var(--_d)) var(--_m),
            radial-gradient(var(--s) at 50% calc(-1 * sin(var(--a)) * var(--s)),#0000 100%,#000 calc(100% + 1px))  0 calc(var(--r) * (1 - sin(var(--a)))) no-repeat,
            linear-gradient(90deg,#000 calc(50% - var(--_d)),#0000 0 calc(50% + var(--_d)),#000 0);
    }

    .avatar {
        height: calc(2 * var(--s));
        aspect-ratio: 1;
        scale: 0.9;
        translate: 0 calc(-1 * sin(var(--a)) * var(--s) - sin(var(--a)) * var(--r));
        border-radius: 50%;
    }
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/a98e32ea9b7c48e591853ac334127ea3~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1202&h=556&s=2041517&e=gif&f=192&b=040404)

  


> Demo 地址：https://codepen.io/airen/full/MWNavRr

  


现在，你只需要按照之前的策略调整伪元素的 `::before` 的 `mask` 以及 `inset` 的值，就可以获得不同变体：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/de5883c600164f3e94ca6f72682fef82~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1218&h=678&s=2042868&e=gif&f=67&b=040404)

  


> Demo 地址：https://codepen.io/airen/full/VwovMzv

  


注意，上面所展示的示例，我们创建的内凹形状都是水平居中的，你可以尝试着调整 `mask` 中 `radial-gradient()` 获得不同位置的内凹形状。

  


在这里我们只展示了 CSS `mask` 创建内凹形状，其实你也可以像之前的案例一样，使用 SVG 的 `<mask>` 与 CSS 的 `mask` 相互结合，也可以使用内联的 SVG 来创建。要是你感兴趣的话，不妨一试，检验一下自己是否可以使用 SVG 来实现相似的效果！

  


## 小结

  


在这节课中，我们以创建镂空用户头像为例，向大家展示了多种不同实现方案。在这里我想说的是，CSS 就是这么的强大，同一种效果，它有多种方法可以实现，尤其是与 SVG 相互结合，使 Web 开发者拥有了更多的解决方案。当然，每种方案都有其利弊，我们在实际使用的时候，应该根据具体的生产环境来选择最为合适的解决方案。
