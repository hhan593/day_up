## 渐进式模糊的探索与思考

  


通常情况下，Web 开发者会添加一个模糊滤镜和叠加半透明纯色背景来制作模糊效果。例如：

  


```CSS
header {
    background: rgb(0 0 0 / 0.4);
    backdrop-filter: blur(5px);
}
```

渐进式模糊并不是什么新鲜事。在 [Photoshop](https://community.adobe.com/t5/photoshop-ecosystem-discussions/how-can-i-create-a-gradient-blur/td-p/1554916) 和 [Apple](https://support.apple.com/sk-sk/guide/motion/motn169fcc4b/5.4.4/mac/10.14) 论坛上就能找到一些关于渐变模糊（Gradient Blur）的讨论。

此外，也有 Web 开发者通过 SVG 的[渐变](https://juejin.cn/book/7341630791099383835/section/7354948936039137289)（`<linearGradient>`）、[遮罩](https://juejin.cn/book/7341630791099383835/section/7366549423813296165)（`<mask>`）和[滤镜](https://juejin.cn/book/7341630791099383835/section/7366549423746187273)中的[高斯模糊](https://juejin.cn/book/7341630791099383835/section/7368318391733452850)（`<feGaussianBlur>`）来创建渐变模糊的效果。然而，在 Web 上使用 CSS 创建渐进式模糊效果的案例很少见。

## 创建渐进式模糊效果

  


正如课程开头所述，通常情况下，Web 开发者会使用 `backdrop-filter` 来创建模糊效果（即“磨砂玻璃”效果）：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/c541862e0f0242bfb883fd3e851ccfec~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1220&h=746&s=9274150&e=gif&f=160&b=292929)

  


然而，大多数 Web 开发者在使用 `backdrop-filter` 创建磨砂玻璃效果时，会忽视一些事情。问题的根源在于：`backdrop-filter` 算法只考虑位于元素正后方的像素。对于像亮度（`brightness()`）或色相旋转（`hue-rotate()`）这样的滤镜，这很有道理。但对于模糊滤镜（`blur()`），我们实际上希望它能考虑近元素的像素。

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/15b3f626a4ef4af093891cc38063a095~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1230&h=624&s=8197373&e=gif&f=221&b=111418)

  


上图展示了调整前后的两种效果的差异。

  


默认情况下，高斯模糊算法应用于元素背后的所有像素。这意味着，如果有一个大而多彩的元素靠近该元素，它将不会产生任何效果。

  


然而，现实中的磨砂玻璃并不是这样工作的。光线从物体上反射，然后透过玻璃。当模糊算法包括附近的内容时，效果看起来会更好。

  


如果我们希望模糊效果考虑到附近的元素，那么就需要扩展该元素，使其覆盖这些元素。然后，使用遮罩去除多余的部分，使其在视觉上与最初预期的大小一致：

  


```HTML
<div class="heading">
    <div class="backdrop"></div>
</div>
<img src="figure.pnng" alt="" />
```

  


```CSS
.heading {
    position: relative;
    background: rgb(0 0 0 / .25);

    .backdrop {
        position: absolute;
        inset: 0;
        height: 200%;
        backdrop-filter: blur(16px);
        mask-image: linear-gradient(
            to bottom,
            black 0% 50%,
            transparent 50% 100%
        );
    }
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/7eede37b1fcf409c974a88462cca8c53~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1216&h=750&s=2717375&e=gif&f=51&b=2a2a2a)

  


> Demo 地址：https://codepen.io/airen/full/mydmPZR

  


让我们一步步来看。首先，我们有一个带有背景模糊效果的区域（`.heading`）:

  


```HTML
<div class="heading">
</div>
<img src="figure.pnng" alt="" />
```

  


```CSS
.heading {
    background: rgb(0 0 0 / .25);
    backdrop-filter: blur(16px);
}
```

  


因为图片元素 `img` 不在 `.heading` 区域的底部，所以它没有被模糊算法考虑到，因此在 `.heading` 区域上看不到那种柔和的颜色光晕：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/77d2aa6335d74c78a35279e38fb6a37a~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1220&h=756&s=5312275&e=gif&f=95&b=2a2a2a)

  


> Demo 地址：https://codepen.io/airen/full/mydmROw

  


我们对 `.heading` 区域进行扩展，使其至少覆盖到 `img` 的一部分。因此，我在 `.heading` 中嵌套了一个子元素 `.backdrop` （也可以考虑使用伪元素 `::before` 或 `::after`），并且将 `backdrop-filter` 移到 `.backdrop` 上，同时将其高度设置为父元素的两倍。

  


```HTML
<div class="heading">
    <div class="backdrop"></div>
</div>
<img src="figure.pnng" alt="" />
```

  


```CSS
.heading {
    position: relative;
    background: rgb(0 0 0 / .25);

    .backdrop {
        position: absolute;
        inset: 0;
        height: 200%;
        backdrop-filter: blur(16px);
    }
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/a03770c57617433fabcc5ab9ce4f350a~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1220&h=758&s=3018206&e=gif&f=50&b=2a2a2a)

  


> Demo 地址：https://codepen.io/airen/full/LEYyxLZ

  


正如你所看到的，模糊区域变大了（`.backdrop`），覆盖了图片的大部分，并正确地进行了模糊处理。但是，我们实际上不想看到多余的模糊区域（不想它比 `.heading` 区域高），我们需要对它进行裁剪，使其高度与父元素 `.heading` 相同。

  


你可能会想到使用 `overflow: hidden` 来处理:

  


```CSS
.heading {
    position: relative;
    background: rgb(0 0 0 / .25);
    overflow: hidden;
    
    .backdrop {
        position: absolute;
        inset: 0;
        height: 200%;
        backdrop-filter: blur(16px);
    }
}
```

  


或者使用 `clip-path` 对它进行裁剪：

  


```CSS
.heading {
    position: relative;
    background: rgb(0 0 0 / .25);
    clip-path: inset(0);
    
    .backdrop {
        position: absolute;
        inset: 0;
        height: 200%;
        backdrop-filter: blur(16px);
    }
}
```

  


不幸运的是，这两种方案在 Chrome 中无法得到想要的效果，在 Firefox 和 Safari 中可以得到我们所要的效果。

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/0a2887575063452b9c7d02a75b2af124~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1360&h=616&s=10481494&e=gif&f=84&b=292929)

  


上图中左侧是 Chrome 浏览器的效果（看不到光晕效果），右侧是 Safari 的效果（可以看到光晕效果）！

  


在 Chrome 浏览器中，`overflow` 或 `clip-path` 裁剪是在滤镜应用之前进行的，因此当模糊算法执行时，内容已经被隐藏。

  


庆幸的是，我们可以使用 CSS 遮罩（`mask`）。在所有浏览器中，遮罩算法发生在滤镜之后。

  


```CSS
.heading {
    position: relative;
    background: rgb(0 0 0 / .25);

    .backdrop {
        position: absolute;
        inset: 0;
        height: 200%;
        backdrop-filter: blur(16px);
        mask-image: linear-gradient(
            to bottom,
            black 0% 50%,
            transparent 50% 100%
        );
    }
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/9a38602e7dc14afdba86b9a56151e458~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1216&h=750&s=2717375&e=gif&f=51&b=2a2a2a)

  


> Demo 地址：https://codepen.io/airen/full/mydmPZR

  


[遮罩 `mask` 也是 CSS 的一个重要特性](https://juejin.cn/book/7223230325122400288/section/7259668885224456252#heading-9)，我们可以通过 `mask` 指定元素的哪些部分应该是透明的：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/a0f9b8493746496da13bfb8e417ae57c~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1280&h=796&s=102706&e=jpg&b=565874)

  


> Demo 地址：https://codepen.io/airen/full/ExGZRJV

  


遮罩图片除了支持透明的格式图像之外，还可以使用渐变作为遮罩。例如，我们可以将图像从不透明渐变到透明：

  


```CSS
img {
    mask-image: linear-gradient(
        to right,
        white 30%,
        transparent
    );
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/b492522aaccf4a25927f74d6d4aabdaa~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2256&h=1136&s=429711&e=jpg&b=5b6a6a)

  


同样的原理，在我们示例中，使用渐变创建一个遮罩，使得 `.backdrop` 超出 `.heading` 区域完全透明：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/94b59c94cf6e441fae4b710c47ab071f~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=652&s=223355&e=jpg&b=242424)

  


相关的 CSS 代码如下：

  


```CSS
.backdrop {
    height: 200%;
    mask-image: linear-gradient(
        to bottom,
        black 0% 50%,
        transparent 50% 100%
    );
}
```

  


你可能已经发现了，示例中的遮罩看起来不像渐变，对吧？通常情况下，大家会认为渐变是颜色平滑地从一个过渡到另一个。例如：

  


```CSS
.backdrop {
    height: 200%;
    mask-image: linear-gradient(
        to bottom,
        black,
        transparent
    );
}
```

  


但在这个示例中，我们需要的正是这样的效果。我们的渐变从 `0%` 到 `50%` 是纯黑色，然后从 `50%` 到 `100%` 就瞬间变成透明。

  


为什么是 `50%` ？简单的解释一下，我们将 `.backdrop` 的 `height` 设置为 `200%` ，因此它的高度是它父容器 `.heading` 的两倍。`mask-image` 渐变中的百分比是相对于当前元素的大小计算的。假设，`.heading` 元素的高度是 `4rem` ，那么 `.backdrop` 的高度会是 `8rem` （父容器高度的 `200%` ，即两倍）。然后，我们的遮罩会显示该元素的前 `50%` （从 `0` 到 `4rem`），其余部分（从 `4rem` 到 `8rem`）将被隐藏。

  


到目前为止，我们所实现的还是单个模糊效果，只是这个模糊效果更接近于真实的“磨砂玻璃”效果。在继续往下之前，我们还有一些小细节需要修复。

  


首先是指针事件。由于 `.backdrop` 元素比较大，它会覆盖附近的元素，使得附近元素变得不可点击和不可选中。比如，我们把上面示例中的图片替换成一行文本，我们无法选中文本：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/25d2c8d52b454bae8bb2967c444e320c~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1218&h=748&s=217495&e=gif&f=101&b=2a2a2a)

  


> Demo 地址：https://codepen.io/airen/full/LEYyxLZ

  


幸运的是，这个问题很容易解决：

  


```CSS
.backdrop {
    position: absolute;
    inset: 0;
    height: 200%;
    backdrop-filter: blur(16px);
    mask-image: linear-gradient(
        to bottom,
        black 0% 50%,
        transparent 50% 100%
    );
    pointer-events: none;
}
```

  


`pointer-events` 属性允许我们指定在处理点击或触摸事件时应该忽略某个元素。`mask-image` 使得模糊层（`.backdrop`）不可见，`pointer-events: none` 使 `.backdrop` 变得“无形”。这也是 `.backdrop` 必须是子元素的另一个原因。我们不希望 `.heading` 本身忽略点击，因为它通常会包含导航链接。我们希望特别针对“磨砂玻璃”元素。

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/e9e446e8a540485ea873ff42ee0d3126~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1222&h=750&s=601983&e=gif&f=150&b=2a2a2a)

  


> Demo 地址：https://codepen.io/airen/full/NPWgopx

  


另一个是顶部闪烁的问题。通过将磨砂玻璃背景扩展到头部下方，我们可以确保模糊算法即使在该元素还没到达头部就已经开始考虑它。但当页面内容超出视口顶部时，它会有一个奇怪的闪烁：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/d3582cbc2f0d4e919e0ad6292f4f0a9e~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1350&h=616&s=14891241&e=gif&f=184&b=14171c)

  


这和之前的问题一样。高斯模糊算法只考虑直接在它下面的像素。例如当图片被滚动出视口时，这些数据不再被模糊算法考虑，从而导致那些不自然的颜色闪烁。

  


不幸运的是，我们不能在这里重用之前的解决方案。据我所知，即使元素正确层叠，视口外的元素也永远不会被 `backdrop-filter` 考虑。到目前为止，我还没有想到或找到一个比较好的解决方案。如果你碰到类似的现象，或者已经有相关的解决方案，请在评论中与我们一起分享！

  


到了这里，我想你你应该掌握了如何使用 `backdrop-filter` 和 `mask` 来创建“更真实”的磨砂玻璃效果了。但我还想在这里跟大家分享另一个技巧：通过添加第二个具有不同滤镜设置的模糊元素，来创造出 3D 玻璃的假象：

> Demo 地址：https://codepen.io/airen/full/raNwPyG

  


这个效果我们使用了两个伪元素 `::before` 和 `::after` ，它们具有自己的 `backdrop-filter` ，其中 `::before` 与之前示例中的 `.backdrop` 是一样的，`::after` 用来创建底部边框，并且通过 `--thickness` 来控制它的厚度，`--radius` 控制模糊效果：

  


```CSS
.heading {
    position: relative;
  
    &::before,
    &::after {
        content: "";
        position: absolute;
        inset: 0;
    }

    &::before {
        height: 200%;
        background: hsla(0, 0%, 100%, 0.1);
        backdrop-filter: blur(16px);
        mask-image: linear-gradient(180deg, #000 0, #000 50%, transparent 0);
    }

    &::after {
        height: 100%;
        background: hsla(0, 0%, 100%, 0.2);
        backdrop-filter: blur(var(--radius, 8px)) brightness(1.2);
        translate: 0 100%;
        mask-image: linear-gradient(
            to bottom,
            #000 0,
            #000 var(--thickness, 3px),
            transparent var(--thickness, 3px)
        );
    }
}
```

  


注意，我们还给 `::after` 的 `backdrop-filter` 添加一个额外的亮度滤镜（`brightness()`），使底部边缘更加突出。

  


事实上，上面这个示例已经应用了两个模糊，离我们要的渐进式模糊效果更进一步了。接下来，我们先来看一个简单的渐进式模糊效果。

  


```HTML
<div class="card">
    <figure>
        <img src="https://picsum.photos/id/103/800/600" alt="">
        <div class="blur">
           <span style="--i: 1;"></span>
           <span style="--i: 2;"></span>
           <span style="--i: 3;"></span>
           <span style="--i: 4;"></span>
           <span style="--i: 5;"></span>
           <span style="--i: 6;"></span>
           <span style="--i: 7;"></span>
           <span style="--i: 8;"></span>
        </div>
    </figure>
    <h3>A small river named Duden</h3>
    <p>Far far away, behind the word mountains, A small river named Duden flows by their place.</p>
</div>
```

  


我们在卡片缩略图上添加了一个 `.blur` 的 `div` ，并且内部嵌套了多个 `span` 。我们将在每个 `span` 上应用不同的 `backdrop-filter` 和 `mask` ，用来创建多层模糊效果，这些模糊效果合并在一起，将产生一个渐进式的模糊效果：

  


```CSS
figure {
    position: relative;
    border-radius: inherit;

    .blur {
        position: absolute;
        inset: 30% 0 0;
        border-radius: inherit;
    }

    span {
        position: absolute;
        inset: 0;
        z-index: var(--i);
        pointer-events: none;
        backdrop-filter: blur(calc(var(--i) * 0.6px));
        border-radius: inherit;
      
        &:nth-child(1) {
            mask-image: linear-gradient(to bottom, rgb(0 0 0 / 0) 0%, rgb(0 0 0 / 1) 12.5%, rgb(0 0 0 / 1) 25%, rgb(0 0 0 / 0) 37.5%);
        }
      
        &:nth-child(2) {
            mask-image: linear-gradient(to bottom, rgb(0 0 0 / 0) 12.5%, rgb(0 0 0 / 1) 25%, rgb(0 0 0 / 1) 37.5%, rgb(0 0 0 / 0) 50%);
        }
      
        &:nth-child(3) {
            mask-image: linear-gradient(to bottom, rgb(0 0 0 / 0) 25%, rgb(0 0 0 / 1) 37.5%, rgb(0 0 0 / 1) 50%, rgb(0 0 0 / 0) 62.5%);
        }
      
        &:nth-child(4) {
            mask-image: linear-gradient(to bottom, rgb(0 0 0 / 0) 37.5%, rgb(0 0 0 / 1) 50%, rgb(0 0 0 / 1) 62.5%, rgb(0 0 0 / 0) 75%);
        }
      
        &:nth-child(5) {
            mask-image: linear-gradient(to bottom, rgb(0 0 0 / 0) 50%, rgb(0 0 0 / 1) 62.5%, rgb(0 0 0 / 1) 75%, rgb(0 0 0 / 0) 87.5%);
        }
      
        &:nth-child(6) {
            mask-image: linear-gradient(to bottom, rgb(0 0 0 / 0) 62.5%, rgb(0 0 0 / 1) 75%, rgb(0 0 0 / 1) 87.5%, rgb(0 0 0 / 0) 100%);
        }
      
        &:nth-child(7) {
            mask-image: linear-gradient(to bottom, rgb(0 0 0 / 0) 75%, rgb(0 0 0 / 1) 87.5%, rgb(0 0 0 / 1) 100%);
        }
      
        &:nth-child(8) {
            mask-image: linear-gradient(to bottom, rgb(0 0 0 / 0) 87.5%, rgb(0 0 0 / 1) 100%);
        }
    }
}
```

我们还可以借助 CSS 的 `sin()` 或 `pow()` 函数来控制每一层的模糊程度。例如下面这个示例，我们使用 `sin()` 函数调整模糊程度：

  


```HTML
<div class="container">
    <img src="https://picsum.photos/id/103/800/600" alt="">
    <div class="blur" style="--layers: 5;">
        <div style="--i: 1;"></div>
        <div style="--i: 2;"></div>
        <div style="--i: 3;"></div>
        <div style="--i: 4;"></div>
        <div style="--i: 5;"></div>
    </div>
</div>
```

  


对应的 CSS 代码如下：

  


```CSS
.container {
    --radw: 115;
    --radh: 45;
    --radx: 50;
    --rady: 85;
    --gradstart: 20;
    --powb: 1.5;
    --pows: 1.75;
    --sinb: 30;
    --sins: 45;
    --rotate: 90;
    --size: 60vmin;
    --padding: 25;
    --radius: 0;
}

.blur {
    pointer-events: none;
    position: absolute;
    inset: 0;
    z-index: 2;

    div {
        position: inherit;
        pointer-events: inherit;
        inset: inherit;
        --blur: calc(
            sin(((var(--layers) - var(--i)) / var(--layers)) * 90deg) * var(--sinb)
        );
        --stop: calc(sin(((var(--i)) / var(--layers)) * 90deg) * var(--sins));

        &::before,
        &::after {
            content: "";
            position: absolute;
        }
        
        &::before {
            inset: 0;
            backdrop-filter: blur(calc(var(--blur) * 1px));
    
            mask: radial-gradient(
                calc(var(--radw) * 1%) calc(var(--radh) * 1%) at calc(var(--radx) * 1%) calc(var(--rady) * 1%),
                #000 calc(var(--gradstart) * 1%),
                #0000 calc((var(--gradstart) + var(--stop)) * 1%)
            );
        }
        
        &::after {
            height: calc(
                ((var(--gradstart) + var(--stop)) / (100 / var(--radh))) * 1%
            );
            width: calc(
                ((var(--gradstart) + var(--stop)) / (100 / var(--radw))) * 1%
            );
            border-radius: 50%;
            left: calc(var(--radx) * 1%);
            top: calc(var(--rady) * 1%);
            translate: -50% -50%;
        }
    }
}
```
