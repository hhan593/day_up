在提到创建或绘制形状时，很多人首先想到的是 [SVG](https://s.juejin.cn/ds/iMaUT59f/)，难免会问，绘制形状为什么不使用 SVG 呢？其实，如果可以的话，就使用 SVG，因为 [SVG 提供了很多绘制图形的元素](https://juejin.cn/book/7341630791099383835/section/7345813971552698406)（例如 `<rect>` 、`<circle>` 、`<polygon>` 和 `<path>` 等），而且它们可以很好的与 [SMIL 动画](https://juejin.cn/book/7341630791099383835/section/7368318466233172005) 或 [CSS 以及 JavaScript 相互结合](https://juejin.cn/book/7341630791099383835/section/7351339840161447945)，创建带有动效的形状。与 CSS 创建形状相比，SVG 更为灵活和强大，只不过，它采用的是 XML 语法，与我们熟悉的 Web 语言略有不同，你需要花一定的时间来学习和了解 SVG ，才能在 Web 中用好 SVG。

如果你想创建 CSS 形状，`clip-path` 属性 —— 以及它的 `polygon()` 函数，将是我们在创建 CSS 形状时常常使用的工具。诸如 [Clippy 工具](https://bennettfeely.com/clippy/)所示，使用 `clip-path` 的 `polygon()` 函数可以创建出各式各样的形状：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/9809a286e8c647759c8eaaaa05f9bda7~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1128&h=506&s=3728635&e=gif&f=136&b=f5f5ee)

  


> URL：https://bennettfeely.com/clippy/

  


注意，[Clippy 工具](https://bennettfeely.com/clippy/)所提供的是 `clip-path: polygon()` 创建 CSS 形状的最基础的用法。事实上，在使用它来创建 CSS 形状时，还有很多技巧，这些技巧可以使你在创建 CSS 形状时变得更加简单，并且创建的形状更灵活，更易于复用。

  


### 理解坐标

  


使用 `clip-path: polygon()` 创建形状，首先要理解不同点的坐标。最简单的方法是像下面这样绘制四个角点：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/e9ee37906b354a3e91358251f8e011f0~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=985&s=191068&e=jpg&b=f2f2f2)

  


左上角是原点，坐标为 `(0,0)` ，右下角是 `(100%,100%)` 。`x` 值从左到右增加，`y` 值从上到下增加。

  


如果你不经常使用 `polygon()` ，很容易在坐标中迷失。你可能会把原点当作左下角，或者搞混 `x` 轴和 `y` 轴。为了避免这种错误，始终在纸上绘制上述图形，并从以下多边形开始：

  


```CSS
.element {
    clip-path: polygon(0 0, 100% 0, 100% 100%, 0 100%);
}
```

  


它将简单地绘制一个适合元素边界的矩形。

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/5ba319bad9684c47939dc1027ce26174~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=3326&h=1784&s=361694&e=jpg&b=f5f5ef)

  


从视觉上看它不会做任何事情，但它是绘制 CSS 形状的起点。虽然听起来可能很奇怪，但这是我多年来的做法。我在纸上绘制坐标点（现在我可以在脑海中完成），从这个多边形开始，然后我就准备好绘制形状了。

  


我们来看两个简单的示例：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/70d9a65fc452401f97359c9f0481dee4~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=985&s=134939&e=jpg&b=ffffff)

  


第一个形状是个梯形，它需要四个点，而我们的初始多边形已经包含了四个点，因此我们不会添加新点。我们只需要更新现有点即可。根据形状，我们可以清楚地看到，需要更新左上角和右上角两个点，更准确地说是它们的 `x` 值：

  


```CSS
.element {
    clip-path: polygon(
        /* x   y*/
           0   0 , /* 左上角 --> 20%  0 */
         100%  0 , /* 右下角 --> 80%  0 */
         100%  100%,
           0   100% 
    )
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/35bec08119c2401894d6cd8a09dda55d~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1273&s=623856&e=jpg&b=fffbfb)

  


你可以尝试更新不同的值，即使是随机值，也能看到多边形是如何变化的。一旦确定了需要更新的点，可以考虑更精确的值来获得最终形状。对于初学者而言，从第一次尝试中获得最终形状并不容易，总是会有一些试错过程。当然，尝试找出要更新的点也是学习的一部分。

  


现在，我们知道需要更新哪些点了，我们可以定义一个 CSS 变量来控制形状：

  


```CSS
.trapezoid {
    --s: 20%;
    clip-path: polygon(var(--s) 0,calc(100% - var(--s)) 0,100% 100%,0 100%);
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/793ef257f095415b9ad50e03e186ad5a~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1017&s=143054&e=jpg)

  


> Demo 地址：https://codepen.io/airen/full/qBzaVVx

  


对于第二个形状，它比梯形稍微复杂一点，它需要添加更多的点（需要六个点）。因此，我们必须添加两个新点。不要关注新点的坐标，首先，尝试在初始多边形中找到它们的位置。

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/bf7fdd2020a540f68604b388dba57d49~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=985&s=246927&e=jpg&b=ffffff)

  


一个点（`(x2,y2)`）需要位于左上角（`(0,0)`）和左下角（`(0, 100%)`）之间，另一个点（`(x1,y1)`）位于右上角（`(100%,0)`）和右下角（`(100%, 100%)`）之间。

  


```CSS
.element {
    clip-path: 
        polygon(
            /* X      Y */
               0      0,      /* 左上角 */
               100%   0,      /* 右上角 */
               x1     y1,     /* 位于右上角与右下角之间 */
               100%   100%,   /* 右下角 */
               0      100%,   /* 左下角 */
               x2     y2      /* 位于左上角与左下角之间 */
        );
}
```

  


现在，我们可以研究并查看如何更新不同的值。新的点位于形状的中间，所以它们的 `y` 值将等于 `50%`。一个点在左边，另一个在右边，所以我们可以将它们的 `x` 值分别设置为 `0` 和 `100%`。

  


```CSS
.element {
    clip-path: 
        polygon(
            /* X      Y */
               0      0,      /* 左上角 */
               100%   0,      /* 右上角 */
               100%     50%,     /* 位于右上角与右下角之间 */
               100%   100%,   /* 右下角 */
               0      100%,   /* 左下角 */
               0     50%      /* 位于左上角与左下角之间 */
        );
}
```

  


在这个步骤中，我们已经正确放置了新的点，并且它们的 `y` 值也已经正确定义了。现在，我们需要更新它们的 `x` 值，并且更新其他点的坐标。同样的，可以通过 CSS 的变量 `--s` 来控制它们的 `x` 值：

  


```CSS
.ribbon {
    --s: 20%;
    clip-path: polygon(
        0 0,                      
        calc(100% - var(--s)) 0, 
        100% 50%,
        calc(100% - var(--s)) 100%,
        0 100%,
        var(--s) 50%
    );
    
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/b0de939dc6d9411eb678edbc7a0ab7af~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1017&s=144896&e=jpg&b=050505)

  


> Demo 地址：https://codepen.io/airen/full/MWMjraE

  


你可能会觉得这些步骤对于简单的形状来说太长，但它们会帮助你简化创建形状的过程。如果你对 `clip-path` 和 `polygon()` 还不熟悉，我强烈建议你在创建 CSS 形状时花时间去做。不要尝试一下子找到所有点及其值。尽可能的一次更新一个值，并查看多边形的变化。经过大量的练习，你将能够跳过大多数步骤，并迅速绘制复杂的形状。

  


### 分析形状的对称性

  


我们绘制的大多数 CSS 形状都有一定的对称性，这可以帮助我们计算一些值。例如之前的梯形，它围绕通过梯形中间的垂直轴对称：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/8fd5c71374174fe98e111619abdc4708~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=985&s=151527&e=jpg&b=ffffff)

  


正如上图所示，整个梯形有两组点：左侧点（`A` 和 `B` ）和右侧点（`A'` 和 `B'`），其中 `A'` 是 `A` 的对称点，`B'` 是 `B` 的对称点。每一组对称点具有相同的 `y` 值，例如 `A` 和 `A'` 点的 `y` 值相同，`B` 和 `B'` 点的 `y` 值相同。与此同时，每两个具有相同 `y` 值的点的 `x` 值总和等于 `100%` 。例如，`A` 和 `A'` 点的 `x` 值总和等于 `100%` ，`B` 和 `B'` 点的 `x` 值总和同样也会等于 `100%` 。

  


这意味着，你只需要知道一半点的坐标值（例如 `A` 和 `B` 点），就可以轻松找出剩余的点（例如 `A'` 和 `B'`）。这也是调试你定义的值的好方法。

  


以下面这个多边形为例：

  


```CSS
.element {
    clip-path: 
        polygon(
            /* X    Y */
               20%  0,  
               75%  0,  
               100% 100%,
               0    100%
    );
}
```

  


上面的代码将不会绘制出一个完美的梯形。因为左上角的 `x` 值是 `20%` ，而右上角的 `x` 值是 `75%` ，它们的和是 `95%` ，而不是 `100%` ，所以绘制出来的形状是不对称的。你需要更改其中一个值或两个值，以获得正确的结果。

  


我们也可以使用 CSS 自定义属性来控制，例如 `var(--s)` （假设是 `A` 点的 `x` 值）和 `calc(100% - var(--s))` （它是 `A'` 点的 `x` 值）。如此一来，无论 `--s` 的值是多少，它们的和始终是 `100%` ：

  


```CSS
.trapezoid {
    --s: 20%;
    clip-path: polygon(var(--s) 0, calc(100% - var(--s)) 0, 100% 100%, 0 100%);
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/e433f752b112496db81eb2687afd13df~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1248&h=612&s=1560997&e=gif&f=221&b=050505)

  


> Demo 地址：https://codepen.io/airen/full/MWMjrBX

  


我们再来看一个更复杂的形状：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/13305738db6e459fbaf3adccf1769781~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=985&s=104017&e=jpg&b=ffffff)

  


绘制这个缎带形状一共需要 `14` 个点，但它与梯形的对称性类似，因此你只需要关注七个点（例如上图中虚线左侧的七个点），就可以轻松找出其他点（虚线右侧的七个点）：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/9280b8e9dc0e47bdb5d3358e8b789d6a~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=985&s=165377&e=jpg&b=ffffff)

  


这个技巧在使用 `clip-path` 时非常重要，理解了这一点（形状的对称性），你在绘制图形时可以减少一半的工作量。要是你发现了更多的对称性，那么工作量还可以继续减半。

  


上面所展示的都是围绕形状垂直轴的对称性。实际上，我们也可以有围绕形状水平轴的对称性，在这种情况下，我们需要考虑 `y` 值，而不是 `x` 值，即两个对称点的 `y` 值之和也需要等于 `100%` 。

  


我们甚至可以在两个轴上都有对称性，如下图所示的星爆形状：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/ee908f2488564beabf5117b4850a4490~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=985&s=204312&e=jpg&b=ffffff)

  


正如你所看到的，绘制上图我们需要 `22` 个，并且该星爆图形具有水平和垂直对称性。我们可以将不同的点分为四组，我们只需要关注一个组，即只需找出五个点。实际上，只需要四个点，因为我们可以删除一个容易找到的点，这个点位于对称轴上方。显然，找到四个点比找 `22` 个点要容易得多！

  


需要注意，就星爆图而言，角数不同时，所需要点数会有所不同。但不管怎么变，都同时具有水平和垂直对称性。

  


```CSS
.starburst {
    clip-path: polygon(
        /* 第一组 */
        100%      50%,
        74.15%    56.47%,
        93.3%     75%,
        67.68%    67.68%,
        75%       93.3%,
        56.47%    74.15%,
        
        /* 第二组 */
        50%       100%,
        43.53%    74.15%,
        25%       93.3%,
        32.32%    67.68%,
        6.7%      75%,
        25.85%    56.47%,
        
        /* 第三组 */
        0%        50%,
        25.85%    43.53%,
        6.7%      25%,
        32.32%    32.32%,
        25%       6.7%,
        43.53%    25.85%,
        
        /* 第四组 */
        50%       0%,
        56.47%    25.85%,
        75%       6.7%,
        67.68%    32.32%,
        93.3%     25%,
        74.15%    43.53%
    );
}
```

### 从一个形状创建不同的变体

  


我们以提示框（`Tooltips` UI 组件）为例：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/f1f70d6a8b7b40b7a2e77a21dd275426~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=759&s=355077&e=jpg&b=ffffff)

  


通常情况下，Web 开发者会以相同过程为每个形状编写 CSS 代码，这意味着你需要做四次相同的工作。这样操作并没有错，因为这四个形状是不同的，但我们可以通过使用一个形状的代码来创建其他三个形状，从而避免额外的工作。这是一个很好的技巧，可以用更少的工作量获得更多的形状。

  


首先，我将定义三种变换，稍后将它们应用于任何我们想要的形状。

  


#### 切换坐标轴

  


第一个变换是切换每个点的 `x` 和 `y` 值。假设你使用 `polygon()` 函数像下面这样绘制了一个多边形：

  


```CSS
.element {
    clip-path: polygon(
        x1     y1,
        x2     y2,
        x3     y3,
        ...    ...,
        xn     yn
    )
}
```

  


然后将其更改变：

  


```CSS
.element {
    clip-path: polygon(
        y1   x1,
        y2   x2,
        y3   x3,
        ...  ...,
        yn   xn
    )
}
```

  


很简单，就是每个点的 `x` 和 `y` 坐标互换了。暂时，不需要了解这样做的原因和功能，你就把它当作是一种自动化过程即可。稍后会通过实例帮助大家理解这样做的作用。

  


#### 反转 `X` 值

  


对于这个变换，我们只更新点的 `x` 值。例如：

  


```CSS
.element {
    clip-path: polygon(
        x1    y1,
        x2    y2,
        x3    y3,
        ...   ...,
        xn    yn
    )
}

/* 反转 x 值 */
.element {
    clip-path: polygon(
        calc(100% - x1)   y1,
        calc(100% - x2)   y2,
        calc(100% - x3)   y3,
        ...               ...,
        calc(100% - xn)   yn
    )
}
```

  


每个 `x` 值的更新方式是使旧值和新值在垂直轴上对称。记住，对称点的总和需要等于 `100%`，这就是公式为 `100% - xi` 的原因。在 CSS 中使用 `calc()` 函数来描述这个公式，即 `calc(100% - xi)` 。

  


#### 反转 `Y` 值

  


与反转 `x` 值相同，但这次是处理 `y` 值：

  


```CSS
.element {
    clip-path: polygon(
        x1    y1,
        x2    y2,
        x3    y3,
        ...   ...,
        xn    yn
    )
}

/* 反转 Y 值 */
.element {
    clip-path: polygon(
        x1    calc(100% - y1),
        x2    calc(100% - y2),
        x3    calc(100% - y3),
        ...   ...,
        xn    calc(100% - yn)
    )
}
```

  


每个 `y` 值的更新方式是使旧值和新值在水平轴上对称。记住，对称点的总和需要等于 `100%`，这就是公式为 `100% - yi` 的原因。在 CSS 中使用 `calc()` 函数来描述这个公式，即 `calc(100% - yi)` 。

  


现在，如果你对现有形状应用其中一种变换，几乎不费吹灰之力就可以得到另一个变体。你可能会问，那具体要使用哪种变换，对吧？答案是：“尝试一下！”将这种变换应用于实际案例中，这样可以更好的帮助你理解它。

  


我们以提示框 UI 效果为例，你只需要编写第一个提示框形状的代码，接着尝试使用变换获得其他变体：

  


```CSS
.tooltip {
    --b: 2em;
    --h: 1em;
    --p: 50%;
    clip-path: polygon(
        0 0,
        0 100%,
        100% 100%,
        100% 0,
        min(100%, var(--p) + var(--b) / 2) 0,
        var(--p) calc(-1 * var(--h)),
        max(0%, var(--p) - var(--b) / 2) 0
    );
    border-image: fill 0 // var(--h) conic-gradient(#90face 0 0);

    &:nth-child(2) {
        clip-path: polygon(
            0 100%,
            0 0,
            100% 0,
            100% 100%,
            min(100%, var(--p) + var(--b) / 2) 100%,
            var(--p) calc(100% + var(--h)),
            max(0%, var(--p) - var(--b) / 2) 100%
        );
    }
    
    &:nth-child(3) {
        clip-path: polygon(
            100% 100%,
            0 100%,
            0 0,
            100% 0,
            100% max(0%, var(--p) - var(--b) / 2),
            calc(100% + var(--h)) var(--p),
            100% min(100%, var(--p) + var(--b) / 2)

}
```

### 反转形状

  


反转形状也被称为“切割形状”（Cut-Out Shapes），视觉上看上去像是一个镂空的效果，这种 UI 效果在 Web 上很常见：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/9f9e1a599ca04e30a29de9aaf79d5478~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1137&s=199938&e=jpg)

  


上图中左侧渐变填充的闪电形状是使用 `polygon()` 函数创建的，右侧矩形缕空的闪电形状是左侧形状的反转版本（切割形状）。这个想法是从一个矩形中切出左边的形状来得到右边的形状。

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/3b7895ee306a4235b61da27d0f58c219~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1137&s=136989&e=jpg&b=f6f6f6)

  


接下来，我们来探讨如何反转使用 `polygon()` 创建的形状。

  


假设你使用 `clip-path: polygon()` 绘制了一个多边形：

  


```CSS
.element {
    clip-path: polygon(
        x0   y0, 
        x1   y1, 
        ...  ...,
        xn   yn
    );
}
```

  


现在，我们要反转上面这个形状！我将会给出创建反转形状的步骤，然后再向大家解释其工作原理。

  


首先，将之前的多边形放入一个 CSS 变量中，例如：

  


```CSS
:root {
    --shape: x0 y0, x1 y1, ...,xn yn;
}
```

  


其次，复制第一个值并将其放在末尾。我们需要重复第一个坐标：

  


```CSS
:root {
    --shape: x0 y0, x1 y1, ...,xn yn, x0 y0;
}
```

  


接下来，我们创建一个与我们要反转的形状相同尺寸的矩形：

  


```CSS
.invert {
    width: 130px;
    aspect-ratio: 0.56;
    clip-path: polygon(0 0, 100% 0, 100% 100%, 0 100%);
}
```

  


然后，重复第一个值以获得以下多边形：

  


```CSS
.invert {
    width: 130px;
    aspect-ratio: 0.56;
    clip-path: polygon(0 0, 100% 0, 100% 100%, 0 100%, 0 0);
}
```

  


最后，我们将 CSS 变量附加到该多边形的末尾：

  


```CSS
.invert {
    --shape: x0 y0, x1 y1, ... xn yn, x0 y0;
    clip-path: polygon(0 0, 100% 0, 100% 100%, 0 100%, 0 0, var(--shape));
}
```

  


完成了！我们的形状已被反转。酷吧？只需这几步，你就能得到切割版形状。我们把 `--shape` 换成一个真实的数据，例如：

  


```CSS
.shape {
    --p: 100% 0, 23% 46%, 46% 44%, 15% 69%, 38% 67%, 0 100%, 76% 57%, 53% 58%, 88% 33%, 60% 37%, 100% 0;
    clip-path: polygon(var(--p));
    
    &.invert {
        clip-path: polygon(0 0,100% 0,100% 100%,0 100%,0 0,var(--p));
    }
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/fa957f409e9d4aa5b0f0151a0c158773~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1089&s=250697&e=jpg&b=050505)

  


> Demo 地址：https://codepen.io/airen/full/xxoEzWZ

  


你可能会好奇它是如何工作的？下图将向你呈现其工作原理：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/2a32024767f240a1a60e0363ef179e62~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=645&s=255030&e=jpg&b=ffffff)

  


我们有两组点：外部点（矩形形状）和内部点（主要形状，即闪电形状）。我们使用外部点来切割矩形形状。从 `(0,0)` 开始，然后回到同一点。在上图中，我故意在点之间添加了一个小间隙，以便大家更好地理解为什么我们需要重复第一个坐标点。

  


之后，我们进入矩形内部，使用内部点来切割主要形状（闪电形状）。与第一个点的逻辑相同，它应该重复两次，以确保我们回到初始点并闭合形状。

  


乍一看，这可能看起来不直观或令人困惑，但尝试用纸和笔跟踪不同的点，你会更好地理解。即使你没有完全掌握这个技巧，你仍然可以轻松地将其应用于任何形状，因为你只需要更新 `--shape` 变量即可。`--shape` 变量主要有两个用途：

  


-   你可以单独使用它来绘制正常的形状
-   与外部点一起使用来获取反转形状（即切割形状）

  


最后，你在实际使用时，并不需要记住很多代码，只需要获取任何形状的代码，并将其放入 `--shape` 中，就完成了！不要忘记在定义变量时将第一个坐标重复到末尾。例如：

  


```CSS
.shape {
    --shape: x0 y0, x1 y1, x2 y2, ..., xn yn, x0, y0;
    clip-path: polygon(var(--shape)); /* 正常形状 */
    
    &.invert {
        clip-path: polygon(0 0,100% 0,100% 100%,0 100%,0 0,var(--shape)); /* 反转形状 */
    }
}
```

  


需要知道的是，在某些情况下上面的代码并不会生效。你可能看不到反转的形状，只会看到初始的矩形。例如下面这个示例：

  


```CSS
.shape {
    --shape: 10% 25%, 35% 25%, 35% 0%, 65% 0%, 65% 25%, 90% 25%, 90% 50%, 65% 50%, 65% 100%, 35% 100%, 35% 50%, 10% 50%,10% 25%;
    clip-path: polygon(var(--shape));
    
    &.invert {
        clip-path: polygon(0 0,100% 0,100% 100%,0 100%, 0 0,var(--shape));
    }
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/9cf5b37b2d92472bb329f6902228f337~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1096&s=146829&e=jpg&b=050505)

  


> Demo 地址：https://codepen.io/airen/full/oNrzMwE

  


不用担心，我们可以在 `polygon()` 函数的开头添加 `evenodd` 来解决这个问题。注意，`polygon()` 函数可以在开头接受一个可选的值为定义绘制形状时使用的填充规则。它有两个值可选：`nonzero` 和 `evenodd` ：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/f06f265a52304056b902f1c2b4cf8a45~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1209&h=449&s=39402&e=jpg&b=fefdfd)

  


你不需要完全理解这些复杂的算法（[要是你想更深入了解它们，你可以点击这里](https://juejin.cn/book/7223230325122400288/section/7259668885224456252#heading-6)）。在我们这个示例中，你只需要确保在 `polygon()` 函数开头添加 `evenodd` ，你就能按前面的规则实现切割形状：

  


```CSS
.shape {
    --shape: 10% 25%, 35% 25%, 35% 0%, 65% 0%, 65% 25%, 90% 25%, 90% 50%, 65% 50%, 65% 100%, 35% 100%, 35% 50%, 10% 50%,10% 25%;
    clip-path: polygon(var(--shape));
    
    &.invert {
        clip-path: polygon(evenodd, 0 0,100% 0,100% 100%,0 100%, 0 0,var(--shape));
    }
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/0acb04c4b3634f3b84787aa0cb8e93a2~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1096&s=165466&e=jpg&b=050505)

  


> Demo 地址：https://codepen.io/airen/full/QWXKBYy

  


现在，这个方法已经完善，无论点的顺序如何都能正常工作。

  


你可能已经发现了，形状紧贴矩形的边框，这在大多数情况下并不理想。也就是说，如果我们能控制形状周围的空间就更好了。

  


为了在形状周围添加一些空间，我们需要像下面这样更新外部点：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/2ebae8d2100849288359fc1f6d36a6d2~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=645&s=196693&e=jpg)

  


所有的外部点将向外移动相同的距离。新的代码如下：

  


```CSS
.invert {
    --s: -20px; /* 控制空间 */
    padding: calc(-1 * var(--s)); /* 添加填充 */
    clip-path: 
        polygon(
            /* 外部点 */
            var(--s) var(--s),
            calc(100% - var(--s)) var(--s),
            calc(100% - var(--s)) calc(100% - var(--s)),
            var(--s) calc(100% - var(--s)),
            var(--s) var(--s),
            /* 内部点 */
            var(--shape)) 
            /* 更新参考框 */
            content-box; 
}
```

  


这样一来，反转形状的最终代码如下：

  


```CSS
.shape {
    --s: -20px; /* 控制空间 */
    /* 内部点 */
    --shape-inner: x0 y0, x1 y1, x2 y2, ..., xn yn, x0 y0; /* 第一个值重复 */
    /* 外部点 */
    --shape-outer: var(--s) var(--s), calc(100% - var(--s)) var(--s), calc(100% - var(--s)) calc(100% - var(--s)), var(--s) calc(100% - var(--s)), var(--s) var(--s);
    /* 大小 */
    width: 50vmin;
    aspect-ratio: 1;
    /* 正常形状 */
    clip-path: polygon(var(--shape)); 
    
    &.invert {
        padding: calc(-1 * var(--s));/* 添加填充 */
        box-sizing: content-box; /* 重置 box-sizing */
        /* 反转形状 */
        clip-path: polygon(evenodd, var(--shape-outer),var(--shape-inner)) content-box; /* 在开头添加 evenodd，结尾添加 content-box */
    }
}
```

### 裁剪形状

  


前面介绍的内容更多的是关于 `clip-path: polygon()` 裁剪形状的理论知识，接下来我们一起来看看，使用 `polygon()` 函数如何裁剪出你自己想要的形状。

  


#### 平行四边形与梯形

  


我们简单的形状开始！

  


众所周知，默认情况之下，Web 上的元素都是一个矩形框，这也是我们最为常见的一种形状，即可矩形或正方形。在此基础上，我们可以轻易地构建出平行四形边、梯形和倾斜矩形等，这些形状有一个共同的特点，即具有一个或两个倾斜边。这些形状都可以使用相同的 CSS 技术来构建。

  


首先，我们通过将四个角点连接起来创建一个基本的矩形：

  


```CSS
.rectangle {
    clip-path: polygon(0 0, 100% 0, 100% 100%, 0 100%);
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/f2ce09c27f914868a628927a43ad6144~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2538&h=1660&s=209403&e=jpg&b=fbfcf7)

  


这段代码不会产生任何效果，因为我们的元素已经是一个矩形了。另外，请注意，我们仅使用了 `0` 和 `100%` 这两个值。

  


接下来，偏移一些值以获得所需的形状。我们可以使用 `--s` 来定义这个偏移值，假设是 `10%` 。如果值为 `0` ，我们将其更新为 `var(--s)` ，如果值为 `100%` ，我们将其更新为 `calc(100% - var(--s))` 。就是这么简单！

  


但我们具体需要更新哪个值呢？这个需要根据具体形状来判断。比如下面这三个形状：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/66e4c2cc7599416b96fcec2699ff039f~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1704&s=1314797&e=jpg&b=fafbf6)

  


对应的代码如下：

  


```CSS
.shape {
    --s: 10%; /* 偏移距离 */
    --shape: 0 0, 100% 0, 100% 100%, 0 100%; /* 矩形 */
    clip-path: polygon(var(--shape));

    /* 平行四边形 */
    &.parallelograms {
        --shape: var(--s) 0,100% 0, calc(100% - var(--s)) 100%, 0 100%;
    }
    
    /* 梯形 */
    &.trapezoid {
        --shape: var(--s) 0, calc(100% - var(--s)) 0, 100% 100%, 0 100%;
    }
    
    /* 倾斜矩形 */
    &.rectangle--skewed {
        --shape: 0 var(--s), 100% 0, 100% 100%, 0 100%;
    }
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/38b7ba4efa1848c9ab94a319ced12fad~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1096&s=168558&e=jpg&b=90f9cf)

  


> Demo 地址：https://codepen.io/airen/full/abgmrXN

  


按类似的方式，你可以获得各种不同的平行四边形，梯形和倾斜矩形。

#### 六边形

  


通常情况下，绘制一个六边形需要六个坐标点。例如：

  


```CSS
.hexagon {
    width: 50vmin;
    aspect-ratio: 0.866; 
    clip-path: polygon(
        0% 25%,
        0% 75%,
        50% 100%, 
        100% 75%, 
        100% 25%, 
        50% 0%
    );
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/405d467ea4054cdbad65bf1e30018b2d~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1096&s=150944&e=jpg&b=050505)

  


> Demo 地址：https://codepen.io/airen/full/ExBNYaJ

  


很简单，对吧？但如果我告诉你，其实还有一种更简单的方法呢？我们可以只用四个点，而不是六个点。

  


`polygon()` 函数一个鲜为人知的技巧是，我们可以设置超出 `[0% 100%]` 范围的点。也就是说，我们可以在元素之外裁剪，这对这种形状以及许多其他形状非常有用。

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/8ee59058f44644c9819df9353b00e912~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1096&s=222211&e=jpg&b=ffffff)

  


我们实际上是在绘制一个菱形，其中两个点被设置在我们尝试制作的六边形的边界之外。其对应的代码如下：

  


```CSS
.hexagon {
    width: 50vmin;
    aspect-ratio: cos(30deg); 
    clip-path: polygon(
        -50% 50%,
        50% 100%,
        150% 50%,
        50% 0
    );
}
```

  


你会注意到，在这里更新了 [`aspect-ratio` 属性](https://juejin.cn/book/7223230325122400288/section/7259316040151236666)的值（元素的宽高比）。使用了一个[三角函数](https://juejin.cn/book/7223230325122400288/section/7242216512176521277) `cos()` 来替代魔法数字 `0.866` 。这个比率的确切值等于 `cos(30deg)` （或 `sin(60deg)`）。而且，`cos(30deg)` 比 `0.866` 更容易记住。

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/65e552dcc5cc4698a3afc25d6ed88f48~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1096&s=171836&e=jpg&b=050505)

  


> Demo 地址：https://codepen.io/airen/full/vYqyBxy

  


注意，[现代 CSS](https://s.juejin.cn/ds/iMHuCoDo/) 已经提供了三角函数相关的功能，如果你从未接触过这方面的知识，请移步阅读《[CSS 的三角函数](https://juejin.cn/book/7223230325122400288/section/7242216512176521277)》！

  


之前我们提到“切换坐标轴”可以从一个形状创建不同的变体。换句话说，我们交换 `x` 和 `y` 轴的值会在垂直和水平轴之间进行切换，这将有助于获得不同的形状。请注意，在切换 `x` 和 `y` 值时，请记得更新 `aspect-ratio` 的值，比如从 `cos(30deg)` 切换为 ` 1  ``/ cos(30deg)` 。因为，我们在交换两个轴，所以新的比例需要等于其倒数，即 `R` （或 `R/1`）变成 `1/R` 。

  


```CSS
.hexagon {
    height: 50vmin;
    aspect-ratio: cos(30deg);
    clip-path: polygon(-50% 50%, 50% 100%, 150% 50%, 50% 0);

    &:nth-child(2) {
        width: 50vmin;
        height: unset;
        aspect-ratio: 1 / cos(30deg);
        clip-path: polygon(50% -50%, 100% 50%, 50% 150%, 0 50%);
    }
}
```

  


我们得到的是六边形的新变体：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/ef6eff8f462344ce811f348d5447b0ba~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1096&s=167729&e=jpg&b=050505)

  


> Demo 地址：https://codepen.io/airen/full/PorbYeE

  


通过这个示例，我们学到了两个使用 `polygon()` 函数创建形状的宝贵经验：

  


-   `polygon()`函数接受超出 `[0% 100%]` 范围的点。这使得我们在某些情况下可以用更少的点来裁剪形状，同时也为创建额外形状提供了可能性
-   切换轴是创建形状变体的有效方法。在六边形的情况下，交换 `x` 轴和 `y` 轴的值会改变六边形的方向

  


#### 八边形

  


八边形与六边形非常相似，它由八个顶点构成，这也是创建八边形的通用方法：

  


```CSS
.octagon {
    width: 50vmin;
    aspect-ratio: 1;
    clip-path: polygon(30% 0%, 70% 0%, 100% 30%, 100% 70%, 70% 100%, 30% 100%, 0% 70%, 0% 30%);
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/a40efa35ca3240459977fa08f4fa955e~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1096&s=148021&e=jpg&b=050505)

  


> Demo 地址：https://codepen.io/airen/full/LYKbYjO

  


然而，我们同样可以使用超出形状边界的坐标来绘制八边形，即使用四个点来定义八边形，就像我们用四个点来定义六边形一样：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/23b5bfeb22e34d719ae0620a25d641d9~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1096&s=219083&e=jpg)

  


我们可以通过 [CSS 的三角函数](https://juejin.cn/book/7223230325122400288/section/7242216512176521277)计算出这个四点的位置，具体的代码如下：

  


```CSS
.octagon {
    width: 50vmin;  
    aspect-ratio: 1;  
    --o: calc(50% * tan(-22.5deg)); 
    clip-path: polygon(
        var(--o) 50%,
        50% var(--o),
        calc(100% - var(--o)) 50%,
        50% calc(100% - var(--o))
    );
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/5e27a775808149119e51de419d6590ec~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1096&s=164190&e=jpg)

  


> Demo 地址：https://codepen.io/airen/full/Porboxv

  


实际上，我们可以进一步优化我们的代码。考虑以下示例：

  


```CSS
.octagon {
    --w: 50vmin;
    width: var(--w);
    aspect-ratio: 1;
    margin: calc(var(--w) * tan(22.5deg) / 2);
    clip-path: polygon(0 50%, 50% 0, 100% 50%, 50% 100%) margin-box;
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/4a36ef16fcb94b148c1487bea8495d42~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1096&s=164190&e=jpg&b=050505)

  


> Demo 地址：https://codepen.io/airen/full/poXNomE

  


最终结果是一样的。在这个优化的代码中：

  


-   `--w` 变量用来定义形状的尺寸
-   `margin` 计算了八边形的外边距，使其在裁剪时具有正确的间距
-   `clip-path`使用了四个点定义八边形的形状，而 `margin-box` 确保裁剪形状的参考区域是 `margin-box` ，而不是默认的 `border-box`，这有助于更精确地控制裁剪区域

  


注意，用于绘制八边形的四个点位于形状的边界之外，并且与这些边距的距离相同。更新之后的代码将这些距离声明在 `margin` 属性上，而不是 `clip-path` 内部，这使得坐标值的定义更加简单。

  


这是我们开始时的 CSS：

  


```CSS
.octagon {
    --o: calc(50% * tan(-22.5deg));
    clip-path: polygon(var(--o) 50%, 50% var(--o), calc(100% - var(--o)) 50%, 50% calc(100% - var(--o)));
}
```

  


优化后的代码即使多了一个属性，也简化了`clip-path`：

  


```CSS
.octagon {
    --w: 50vmin;
    margin: calc(var(--w) * tan(22.5deg) / 2);
    clip-path: polygon(0 50%, 50% 0, 100% 50%, 50% 100%) margin-box;
}
```

  


所有的 `--o` 变量都从 `clip-path` 中移除了，`margin` 属性获得了相同的值。我引入了一个新的变量 `--w`来 设置元素的尺寸，因为我不能依赖百分比值。在这种特定情况下，你会发现元素周围有一些边距，但这个技巧确实可以简化计算。

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/6a3c81377e474048bb34a3cea59aebf3~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=3342&h=1278&s=241626&e=png&b=050505)

  


如果你不希望有额外的边距（`margin`），你可以改用 `padding`，并将相同的 `padding` 值作为负边距。这是另一种保持**多边形简单且适用于图像的技巧**。下面的代码，展示了使用相同 `clip-path` 值创建的不同形状：

  


```CSS
.polygon {
    --w: 200px;
    width: var(--w);
    aspect-ratio: 1;
    object-fit: cover;
    clip-path: polygon(50% 0, 100% 50%, 50% 100%, 0 50%);
    box-sizing: content-box; /* 这个很重要 */
}

.octagon {
    margin: calc(var(--w) * tan(22.5deg) / -2);
    padding: calc(var(--w) * tan(22.5deg) / 2);
}

.hexagon {
    aspect-ratio: cos(30deg);
    margin-inline: calc(var(--w) / -2);
    padding-inline: calc(var(--w) / 2);
}

.hexagon-alt {
    aspect-ratio: 1 / cos(30deg);
    margin-block: calc(var(--w) / -2);
    padding-block: calc(var(--w) / 2);
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/967bef45258e4dbea595728a6a716439~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1096&s=481757&e=jpg)

  


> Demo 地址：https://codepen.io/airen/full/ZEdBYGy

  


特别声明，上面这个示例所展示的技巧只适用于可替换元素，例如 `<img>` 。如果用于其他元素上，需要调整裁剪区域的参考框和 `polygon()` 函数的坐标点！

  


#### 星形

  


相对而言，创建星形要复杂的多，即使你对 `clip-path` 和 `polygon()` 函数很熟悉，也绝非易事。要使用 `polygon()` 裁剪出一个星形（比如五角星），你需要非常精确的值。因此，使用诸如 [Clippy 工具](https://bennettfeely.com/clippy/)获取一个现在的 CSS 代码，或者自己调整代码直到正确的结果。

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/e39b590b79de4ffea297a2cc3eed0aba~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2524&h=1654&s=174294&e=jpg&b=fbfcf6)

  


```CSS
.star {
    clip-path: polygon(50% 0%, 61% 35%, 98% 35%, 68% 57%, 79% 91%, 50% 70%, 21% 91%, 32% 57%, 2% 35%, 39% 35%);
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/752290db946c4e0696653f885894a3e5~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1096&s=158341&e=jpg&b=050505)

  


> Demo 地址：https://codepen.io/airen/full/MWMbKeo

  


正如你所看到的，上面示例是一种通用的裁剪方法，使用了十个点裁剪出一个五角星。从技术上讲，这是没有问题的。但是，我们也可以像之前的六边形或八边形那样，仅使用五个点来裁剪出一个五角星。

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/7c1b88a856864910a39d1745d69e4524~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1096&s=216088&e=jpg&b=ffffff)

  


仅用五个点来裁剪出一个星形，听起来似乎不太可能，但事实上是可以的。诀窍在于 `polygon()` 内点的顺序。如果我们用铅笔在纸上绘制星形，遵循的顺序如下：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/77c4a0cbcb114f37af2c3eb64772eb33~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1096&s=302474&e=jpg&b=ffffff)

  


这是 `clip-path` 和 `polygon()` 函数的另一个隐藏技巧：**我们建立的线条是可以相交**。

  


这正是我们小时候绘制星形的方式——它在CSS中使用`polygon()`同样适用！这是`clip-path`和`polygon()`的另一个隐藏技巧，这也带来了绘制CSS形状的另一个关键课程：我们建立的线条可以相交。即使这是我们都曾用手工制作的图案，这种方法仍然是颠覆性的。

  


以下是如何将这五个点转换为 CSS 代码：

  


```CSS
.star {
    width: 50vmin;  
    aspect-ratio: 1;
    clip-path: polygon(
        50% 0,   /* 1️⃣ */
        calc(50% * (1 + sin(.4turn)))  calc(50% * (1 - cos(.4turn))), /* 2️⃣ */
        calc(50% * (1 - sin(.2turn)))  calc(50% * (1 - cos(.2turn))), /* 3️⃣ */
        calc(50% * (1 + sin(.2turn)))  calc(50% * (1 - cos(.2turn))), /* 4️⃣ */
        calc(50% * (1 - sin(.4turn)))  calc(50% * (1 - cos(.4turn)))  /* 5️⃣ */
    ); 
}
```

  


我再次使用了 **[CSS 的三角函数](https://juejin.cn/book/7223230325122400288/section/7242216512176521277)** 来提高准确性，而不依赖于魔法数字，但即使我们计算出这些值，代码也仍然优于传统的十个点方法：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/9e24946a373349a3a02f3b9ec542125f~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1096&s=186003&e=jpg&b=050505)

  


> Demo 地址：https://codepen.io/airen/full/zYVorLJ

  


你可能发现了，`polygon()` 裁剪出来的星形（五角星）是一个对称的形状。星形上的第二个点和第五个点具有相同的 `y` 值。第三个和第四个点亦是如此。

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/011790cbeb964f76a44f08ce6551e5b1~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1096&s=183159&e=jpg)

  


另外，`x` 值与中心的距离相同（`79% - 50% = 50% - 21%`）。将这些值加起来，我们会看到总和等于 `100%` （`79% + 21% = 100%`）。

  


之前我们提到过，对于对称性形状，它存在重复的值。这将减少计算或查找不同值的工作量。正如上面的代码所示，我们从十个点减少到五个点。现在，只需要记住三个点，其余两个点可以根据对称性特性推导出来。这个示例中的星形（五角星）是围绕中间的垂直轴对称，因此：

  


-   对称点（2️⃣ 的对称点是 5️⃣，3️⃣ 的对称点是 4️⃣ ）的 `y` 值相等
-   对称点 `x` 值的和等于 `100%`

  


即：

  


```
50% 0    /* 1️⃣ */
79% 90%  /* 2️⃣  --> (100% - 79%) = 21% 90%    5️⃣  */
2% 35%   /* 3️⃣  --> (100% - 2%)  = 98% 35%    4️⃣  */
```

  


这样我们重新得到绘制五角星的 CSS 代码：

  


```CSS
.star {
    width: 50vmin;  
    aspect-ratio: 1;
    clip-path: polygon(
        50% 0,    /* 1️⃣ */
        79% 90%,  /* 2️⃣ */
        2% 35%,   /* 3️⃣ */
        98% 35%,  /* 4️⃣ */
        21% 90%   /* 5️⃣ */
    );
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/08ddfa9831e848dc8deaacc9a4d847c9~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1096&s=213328&e=jpg&b=050505)

  


> Demo 地址：https://codepen.io/airen/full/oNrYbJK

  


正如你所看到的，三种不同的方式裁剪出来的五角星效果是一样的。回到之前的六边形和八边形，它们也是对称性图形，可以使用同样的方式来绘制。如果你感兴趣的话，可以尝试使用对称性裁剪出六边形和八边形，就当作是一个小作业。

  


## 在 CSS 中使用遮罩创建形状

  


前面我们讨论的是如何使用 `clip-path` 和 `polygon()` 函数裁剪多边形。接下来，我们将讨论圆形和曲线形状，并介绍在创建 CSS 形状时最常用的另一个属性：[CSS `mask` 属性](https://juejin.cn/book/7223230325122400288/section/7259668885224456252#heading-9)。

  


与之前的部分一样，我们将创建一些形状，同时强调你需要知道的主要技巧。不要忘记，我们的目标并不是学习如何创建特定的形状，而是学习那些允许你创建任何形状的技巧。

  


### 圆形与孔洞

  


淡到 CSS `mask` 属性时，[渐变](https://juejin.cn/book/7199571709102391328/section/7199845781149810727)肯定会出现。例如，我们可以用径向渐变作为遮罩层，在元素上挖出一个孔洞：

  


```CSS
.hole {
    mask: radial-gradient(50px, #0000 98%, #000);
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/b77ea020db3c470e8e6572b32d56533a~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1096&s=142058&e=jpg)

  


> Demo 地址：https://codepen.io/airen/full/zYVoNez

  


你可能会问，为什么不使用背景呢？CSS 的 `mask` 属性提供了更多的灵活性，比如使用任何我们想要的颜色，并可以将效果应用于各种元素，例如 `<img>` 。当然，如果颜色和灵活性不是大问题，那么你也可以选择背景属性来创建也洞，而不是使用遮罩。

  


这两者最大的差异是，背景绘制的孔洞是一个实心的，它法穿透元素自身看到其底部的内容，而遮罩则不同，它可以穿透元素自身，看到底部的内容。另外，如果你想在图片上挖一个孔洞，那么背景属性就要棘手得多了，而遮罩则不会。

  


下面是 `mask` 在 `<div>` 和 `<img>` 上的效果：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/55cb77f52246482396f44560991fd8af~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1084&h=568&s=2490176&e=gif&f=119&b=8d3d3f)

  


> Demo 地址：https://codepen.io/airen/full/LYKbxam

  


在这里，我想强调一下，在 CSS 中创建形状的另一个重要教训：**在使用 CSS 的** **`mask`** **属性时，我们在渐变中使用的颜色实际上是无关紧要的**。

  


我们关注的仅仅是颜色值的 Alpha 通道，因为透明度决定了从元素中挖出去的部分（“遮罩”的地方），从而在中心形成圆形孔洞。渐变的不透明颜色（实色）保留了元素其余部分的可见性。这就是为什么你经常会看到我使用黑色值（例如，`#000`）来表示可见部分，而使用透明色（例如， `#0000`）来表示不可见部分。这其实是 CSS 的 `mask` 的基本特性，如果你对 CSS 遮罩不太了解的话，建议你移步阅读《[CSS 的 Clipping 和 Masking](https://juejin.cn/book/7223230325122400288/section/7259668885224456252)》一文！

  


注意渐变中的实色（纯色）停止点。颜色之间的平滑过渡会导致模糊的边缘，如果我们去掉过渡并且颜色之间突然变化，将会得到平滑、锐利的边缘。但还不完全！我倾向于保持一个非常小的过渡（`98%` 而不是 `100%`）以避免锯齿边缘。这种现象在渐变中非常常见，因此[你需要对渐变有一定的了解](https://juejin.cn/book/7199571709102391328/section/7199845781149810727#heading-16)，才能更好的避免这种现象出现！

### 圆弧

  


这是另一个 CSS 渐变非常适合用来遮罩形状的例子。你可能见过这种类型的形状无数次，因为它是圆形进度条的常见模式。

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/723f9dd1726e4f339df9eefe8eed80a6~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1090&s=794658&e=jpg&b=f7ebde)

  


这一次，我们将介绍另一个技术 —— “[遮罩合成](https://juejin.cn/book/7223230325122400288/section/7259668885224456252#heading-16)”。我们将使用两个渐变来作为遮罩层，即有多个遮罩，有点类似于多背景的应用。然后，使用 `mask-composite` 对多个遮罩层进行合成处理。

  


下图展示了渐变配置和每个遮罩层之间的组合过程：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/79ed3c27df2246e98e153adcee4b6c64~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1090&s=255832&e=jpg&b=fefefe)

  


首先使用 `radial-gradient()` 创建一个完整的圆形，然后使用 [`conic-gradient()`](https://juejin.cn/book/7223230325122400288/section/7259668771856941111) 创建下方的形状。在这两个渐变之间，我们执行一个“交集”合成（`mask-composite: intersect`），以获得未闭合的圆形。接着，我们在遮罩中添加两个额外的径向渐变，以获得未闭合圆形上的圆点。这次我们使用的是默认合成，即 `mask-composite: add` 。

  


CSS 渐变并不是新鲜事物，因为我们经常在 `background` 属性中使用它们，但“合成”是一个新概念。这是一个非常实用的概念，可以解锁很多可能性。

  


具体的 CSS 代码如下：

  


```CSS
.arc {
    --b: 40px; /* 控制进度条粗细 */
    --a: 250deg; /* 进度条进度 */
    width: 50vmin;
    aspect-ratio: 1;
    background: linear-gradient(#2196F3, #8a9b0f);
    --_g:/var(--b) var(--b) radial-gradient(50% 50%,#000 97%,#0000 99%) no-repeat;
    mask:
       top var(--_g),
       calc(50% + 50% * sin(var(--a))) 
       calc(50% - 50% * cos(var(--a))) var(--_g),
       conic-gradient(#000 var(--a),#0000 0) intersect, 
       radial-gradient(50% 50%, #0000 calc(100% - var(--b) - 1px), #000 calc(100% - var(--b)) 99%,#0000);
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/84bc66a4a239485d88ace56024d033a5~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1090&s=211331&e=jpg)

  


> Demo 地址：https://codepen.io/airen/full/qBzqjKV

  


即使代码看起来有些复杂，但使用 [CSS 自定义属性](https://juejin.cn/book/7223230325122400288/section/7249357815410589733)使得调整变得更容易。这是我在创建的大多数形状中使用的重要 CSS 技巧。很多形状需要复杂的公式和大量的渐变，但最终你只需要调整几个变量（CSS 自定义属性）来控制形状。因此，我们不必花太多时间解释数学表达式。我希望专注于技巧和技术，因为 CSS 概念才是重要的。记住，你可以随时获取数学公式，CSS 如何使用它才是关键。

  


注意，我们可以使用不同的渐变组合来实现相同的效果。这很奇怪，因为语法看起来完全不同。下面这个代码片段实现了相同的视觉效果：

  


```CSS
.arc {
    --b: 40px; /* 控制进度条粗细 */
    --a: 250deg; /* 控制进度条进度 */
    width: 50vmin;
    aspect-ratio: 1;
    background: linear-gradient(#2196f3, #8a9b0f);
    padding: var(--b);
    box-sizing: border-box;
    border-radius: 50%;
    --_g: / var(--b) var(--b) no-repeat radial-gradient(50% 50%, #000 97%, #0000 99%);
    mask: 
        top var(--_g),
        calc(50% + 50% * sin(var(--a))) calc(50% - 50% * cos(var(--a))) var(--_g),
        linear-gradient(#0000 0 0) content-box intersect,
        conic-gradient(#000 var(--a), #0000 0);
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/be727dbddffb4d879332fd62419505e2~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1090&s=258280&e=jpg&b=7b669e)

  


> Demo 地址：https://codepen.io/airen/full/xxoRrQX

  


上面代码中添加了 `border-radius` 以使元素变成一个圆形，并添加了等于圆环粗细的内边距（`padding: var(--b)`）。然后，在遮罩中的渐变，使用了一个包含单一透明颜色的 `linear-gradient()` 替换了 `radial-gradient()` ，这个颜色覆盖了元素的内容区域。

  


注意，在这两个示例中，`mask` 都应用了四个渐变，但通过 `--a` 和 `--b` 变量简化了整体渐变。这也是实现相同效果的另一种有效方法。

  


我们还可以使用相同的方式来生成带有虚线边框的圆形：

  


```CSS
.dashed {
    --n: 16; /* 控制虚线数量 */
    --t: 8px; /* 控制虚线粗细 */
    --d: 4deg; /* 控制虚线之间间距 */
    --c: repeating-conic-gradient(
      pink 0 calc(360deg / var(--n)),
      lime 0 calc(2 * 360deg / var(--n))
    ); /* 控制虚线颜色 */

    width: 50vmin;
    aspect-ratio: 1;
    position: relative;

    &::before {
        content: "";
        position: absolute;
        inset: 0;
        border-radius: 50%;
        padding: var(--t);
        background: var(--c);
        mask: 
            linear-gradient(#0000 0 0) content-box intersect,
            repeating-conic-gradient(from calc(var(--d) / 2), #000 0 calc(360deg / var(--n) - var(--d)),#0000 0 calc(360deg / var(--n)));
    }
}
```

  


这次我们在遮罩（`mask`）中结合了两个渐变。其中一个是从黑色到透明的重复圆锥渐变（`repeating-conic-gradient()`），另一个是透明的线性渐变（`linear-gradient()`），用于覆盖元素的内容区域，而 `mask-composite` 属性设置为 `intersect`。

> Demo 地址：https://codepen.io/airen/full/mdZOwgv

### 圆角 Tab 选项卡

  


时不时的会有开发者来咨询，带有圆角的 Tab 选项卡应该如何实现？

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/751846b10c6f4735937d07b038eb2778~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1116&s=162171&e=jpg&b=dedbf1)

  


传统的实现方式是使用伪元素来创建形状：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/080df2123b084b67b6d33c640afcb625~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=617&s=76123&e=jpg&b=ffffff)

  


但它会缺少相应的灵活性和可扩展性，而且它还具有一定的局限性，例如，你可能要随时根据不同的皮肤来调整颜色。相反，我们使用 CSS 遮罩，可以以更少的代码获得更完美的形状，例如带有渐变的圆角选项卡形状：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/16299c9e39144f5494202ee0577ee887~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=850&s=128963&e=jpg&b=ffffff)

  


这个图形效果，并不是顶部圆角难以现实，而是底部部分向内弯曲的部分（内凹圆角）更难实现。使用 CSS 遮罩实现该形状的技术与之前图形相同：使用多个渐变作为遮罩层，并将其组合在一起，只显示我们想要的部分。具体过程如上图所示：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/b47b7a84f5714f479b1eaf4a09320bfa~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=850&s=313457&e=jpg&b=fffefe)

  


我们首先在元素周周围添加一个边框（`border`），但不包括底部边框，并在左上角和右上角应用 `border-radius` ：

  


```CSS
.tab {
    --r: 40px; /* 半径大小 */

    border: var(--r) solid #0000; /* 透明黑色 */
    border-bottom: 0;
    border-radius: calc(2 * var(--r)) calc(2 * var(--r)) 0 0;
}
```

  


接下来，我们添加第一层遮罩。我们只想显示填充区域（`padding-box`），即上图中突出显示的红色区域。

  


```CSS
.tab {
    mask: linear-gradient(#000 0 0) padding-box;
}
```

  


让我们再添加两个径向渐变，来显示那些底部的弧形。

  


```CSS
.tab {
    mask:
        radial-gradient(100% 100% at 0 0, #0000 98%, #000) 0 100% / var(--r) var(--r),
        radial-gradient(100% 100% at 100% 0, #0000 98%, #000) 100% 100% / var(--r) var(--r),
        linear-gradient(#000 0 0) padding-box;
    mask-repeat: no-repeat;
}
```

  


显示用于创建形状内弧的径向渐变。

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/5e05404283154f1a8e3131f992005dfc~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=850&s=179989&e=jpg)

  


这是完整代码的样子：

  


```CSS
.tab {
    --r: 40px; /* 半径大小 */
    border: var(--r) solid #0000; /* 透明黑色 */
    border-bottom: 0;
    mask: 
        radial-gradient(100% 100% at 0 0, #0000 98%, #000) 0 100% / var(--r) var(--r),
        radial-gradient(100% 100% at 100% 0, #0000 98%, #000) 100% 100% / var(--r) var(--r),
        linear-gradient(#000 0 0) padding-box;
    mask-repeat: no-repeat;
    border-radius: calc(2 * var(--r)) calc(2 * var(--r)) 0 0;
}
```

  


像往常一样，控制形状只需要一个变量。让我们稍微关注一下 `border-radius`：

  


```CSS
.tab {
    border-radius: calc(2 * var(--r)) calc(2 * var(--r)) 0 0;
}
```

  


注意，形状的圆角顶部边缘等于半径（`--r`）值的两倍。如果你想知道为什么需要进行计算，这是因为我们有一个透明的边框在外面，需要将半径加倍。前面图中突出显示的蓝色区域的半径等于 `2 * R`，而同一图中突出显示的红色区域的半径等于 `2 * R - R`，即 `R`。

  


实际上，我们可以优化代码，只需要两个渐变，一个径向渐变和一个锥形渐变。

  


```CSS
.tab {
    --r: 40px; /* 半径大小 */
    border-inline: var(--r) solid #0000;
    border-radius: calc(2 * var(--r)) calc(2 * var(--r)) 0 0 / var(--r);
    mask: 
        radial-gradient(var(--r) at var(--r) 0,#0000 98%,#000 101%) calc(-1 * var(--r)) 100% / 100% var(--r) repeat-x,
        conic-gradient(#000 0 0) padding-box;
}
```

## 小结

  


希望你通过这节课的学习之后，对 CSS 形状有了不同的看法。我们讨论了几种形状，但实际上有成百上千种形状，因为你会看到它们在配置上是多么灵活，可以变换出多种变化。

  


归根结底，所有形状都是使用不同的 CSS 概念的某种组合，例如裁剪、遮罩、组合、渐变、CSS变量等等。更不用说一些隐藏的技巧，比如与 `polygon()` 函数相关的技巧：

  


-   它接受超出 `[0% 100%]` 范围的点
