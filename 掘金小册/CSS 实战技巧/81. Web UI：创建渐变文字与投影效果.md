## CSS 创建带有描边和阴影的渐变文字

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/6dbf6c7362cd40c7b95359013eadab7d~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=652&s=122646&e=jpg&b=ffffff)

  


我们以上图中的文本效果为例。文本同时具有三种效果：**渐变、描边和阴影**！

  


### 创建渐变文本

  


在 CSS 中，我们可以通过 `background-clip: text` 将背景裁剪到文本来实现渐变文本效果。例如：

  


```CSS
h1 {
    --c-start: #fff000;
    --c-end: #ff3600;
    background: linear-gradient(var(--c-start), var(--c-end));
    background-clip: text;
    color: transparent;
}
```

  


上面代码创建了一个渐变背景，然后将背景裁剪到文本上。接着，我们通过 `color: transparent` 隐藏了实际文本内容：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/52b81ad84bc847b0aa2ded9500636d27~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2484&h=886&s=193289&e=jpg&b=546161)

  


> Demo 地址：https://codepen.io/airen/full/vEYyoGw

  


另外一种方式是，我们可以将 `background-clip: text` 和 `text-fill-color` 结合使用：

  


```CSS
h1 {
    --c-start: #fff000;
    --c-end: #ff3600;
    background: linear-gradient(var(--c-start), var(--c-end));
    background-clip: text;
    -webkit-text-fill-color: transparent;
}
```

  


上面的代码，`text-fill-color: transparent` 替代了 `color: transparent` 。需要注意的是，`text-fill-color` 是一个非标准属性，在实际使用的时候需要添加 `-webkit-` 前缀！

### 创建描边文本

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/3879a95d69ab465ca8f13060b2e11905~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1280&h=720&s=14323&e=png&b=000119)

  


通常情况下，我们可以使用 `text-stroke` 给文本描边。例如：

  


```CSS
h1 {
    -webkit-text-stroke: 2px red;
}
```

  


它与 `text-fill-color` 相似，在实际使用的时候需要添加 `-webkit-` 前缀。`text-stroke` 是一个简写属性，它包括 `text-stroke-width` （设置描边的厚度）和 `text-stroke-color` （设置描边的颜色）。

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/daf1a63decf0447eaaf510c44abcf8d7~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2280&h=838&s=162120&e=jpg&b=556262)

  


> Demo 地址：https://codepen.io/airen/full/QwWGeOv

  


如果你希望创建一个镂空的文本描边效果，那么可以结合 `text-fill-color: transparent` 或 `color: transparent` 来实现：

  


```CSS
h1 {
    -webkit-text-stroke: 2px red;
    -webkit-text-fill-color: transparent;
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/39e108fbbe394d0baebe4adc43eb5754~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2316&h=924&s=190054&e=jpg&b=556262)

在使用 `text-stroke` 给文本描边时，我们还可通过 `paint-order` 属性来改变填充的描边的绘制顺序。`paint-order` 属性的默认值是 `normal` ，它定义了填充（`fill`）先绘制，然后是描边（`stroke`），最后是标记（`markers`）。但你可以重新定义绘制顺序，比如：

  


```CSS
h1 {
    -webkit-text-stroke: 3px red;
    paint-order: stroke;
}
```

  


上面代码中的 `paint-order` 会告诉浏览器，先描边，然后填充。你可以尝试在下面示例中切换 `paint-order` 的值，查看 `stroke` 和 `fill` 之间的差异：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/25f8e71d755b4005a24368a3f688887e~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1092&h=396&s=327124&e=gif&f=129&b=495958)

```CSS
 h1 {
    -webkit-text-stroke: 3px red;
    -webkit-text-fill-color: transparent;
    --c-start: #fff000;
    --c-end: #ff3600;
    background: linear-gradient(var(--c-start), var(--c-end));
    background-clip: text;
```

我们还可以使用 `background-clip:text` 给文本添加渐变的描边效果。创建渐变描边文本效果时，`text-fill-color` 或 `color` 不能为 `transparent` ，否则将会获得一个渐变填充的文本，文本不会有描边效果：

```CSS
    -webkit-text-fill-color: #fff;
    paint-order: fill;
}
```

> Demo 地址：https://codepen.io/airen/full/azbpdQP

  


效果看上去不是很完美，但已经相当不错了。不过，使用 `text-shdow` 给文本添加描边效果有些缺点存在。

  


首先，如果我们需要修改描边的厚度或颜色，我们需要在多个地方进行更改，这可能会很麻烦。庆幸的是，我们可以借助 **[CSS 自定义属性](https://juejin.cn/book/7223230325122400288/section/7249357815410589733)** 的特性来改善它。比如，我们可以使用 `--stroke-color` 变量存储描边的颜色，`--stroke-width` 变量存储描边的厚度。然后，在 `text-shadow` 中使用这些变量。这样一来，如果需要修改描边的颜色或厚度，只需要在一个地方进行更改：

  


```CSS
h1 {
    --stroke-color: red;
    --stroke-width: 1px;
    text-shadow:
        var(--stroke-width) 0 0 var(--stroke-color),
        calc(var(--stroke-width) * -1) 0 0 var(--stroke-color),
        0 var(--stroke-width) 0 var(--stroke-color),
        0 calc(var(--stroke-width) * -1) 0 var(--stroke-color);
    color: #fff;
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/d075ff1a08c64c0a8886bdacd6f43e24~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1138&h=428&s=3449256&e=gif&f=277&b=4b5b5a)

  


> Demo 地址：https://codepen.io/airen/full/qEBRZqE

  


在示例中，`text-shadow` 属性使用了 `calc()` 函数，使得我们可以将文本描边厚度（`--stroke-width`）乘以 `1` ，从而处理上方和左侧阴影的方向。

  


另一个缺点是，描边厚度 `--stroke-width` 变大时，文本的角落会出现问题：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/79a537124e6d419dbb78fe88a23f23e3~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2314&h=878&s=204377&e=jpg&b=556262)

  


那么，我们应该如何解决这个问题呢？答案很简单：**添加更多的阴影**！

  


如果我们想要为文本添加更多的阴影，就需要弄清楚如何将这些阴影在文本周围偏移，以覆盖文本轮廓的所有空隙。直觉告诉我们，应该将阴影均匀地分布在一个圆形上，圆的半径等于文本描边的厚度。我们可以通过下面的公式来计算阴影的偏移量：

  


```
x = r × cos(α)
y = r × sin(α)
```

  


其中，`x` 和 `y` 是偏移值，`r` 是圆的半径（实际上是我们要偏移的量，相当于文本描边的厚度），`α` 是我们希望将圆分成的角度。我们可以根据需要添加多少个阴影来为文本描边分配 `α` 的值。例如，如果我们添加 `8` 个阴影，那么我们将 `2π` （完整的一个圆）除以 `8` ，每个角度是 `π/4` 。然后，我们将 `α` 的值分配为 `0` ，`π/4` ，`π/2` ，直到完成整个圆，就可以得到完美对齐的 `8` 个阴影偏移。

  


随着我们添加更多的阴影，较大的文本厚度的描边效果会变得更平滑。下面这个示例，我们使用了 `16` 个阴影给文本添加平滑的描边效果：

  


```CSS
h1 {
    --stroke-color: red;
    --stroke-width: 1px;
    --step: 16;

    text-shadow: 
        calc(var(--stroke-width) * cos(calc(360deg / var(--step)) * 0)) calc(var(--stroke-width) * sin(calc(360deg / var(--step)) * 0)) 0 var(--stroke-color),
        calc(var(--stroke-width) * cos(calc(360deg / var(--step)) * 1)) calc(var(--stroke-width) * sin(calc(360deg / var(--step)) * 1)) 0 var(--stroke-color),
        calc(var(--stroke-width) * cos(calc(360deg / var(--step)) * 2)) calc(var(--stroke-width) * sin(calc(360deg / var(--step)) * 2)) 0 var(--stroke-color),
        calc(var(--stroke-width) * cos(calc(360deg / var(--step)) * 3)) calc(var(--stroke-width) * sin(calc(360deg / var(--step)) * 3)) 0 var(--stroke-color),
        calc(var(--stroke-width) * cos(calc(360deg / var(--step)) * 4)) calc(var(--stroke-width) * sin(calc(360deg / var(--step)) * 4)) 0 var(--stroke-color),
        calc(var(--stroke-width) * cos(calc(360deg / var(--step)) * 5)) calc(var(--stroke-width) * sin(calc(360deg / var(--step)) * 5)) 0 var(--stroke-color),
        calc(var(--stroke-width) * cos(calc(360deg / var(--step)) * 6)) calc(var(--stroke-width) * sin(calc(360deg / var(--step)) * 6)) 0 var(--stroke-color),
        calc(var(--stroke-width) * cos(calc(360deg / var(--step)) * 7)) calc(var(--stroke-width) * sin(calc(360deg / var(--step)) * 7)) 0 var(--stroke-color),
        calc(var(--stroke-width) * cos(calc(360deg / var(--step)) * 8)) calc(var(--stroke-width) * sin(calc(360deg / var(--step)) * 8)) 0 var(--stroke-color),
        calc(var(--stroke-width) * cos(calc(360deg / var(--step)) * 9)) calc(var(--stroke-width) * sin(calc(360deg / var(--step)) * 9)) 0 var(--stroke-color),
        calc(var(--stroke-width) * cos(calc(360deg / var(--step)) * 10)) calc(var(--stroke-width) * sin(calc(360deg / var(--step)) * 10)) 0 var(--stroke-color),
        calc(var(--stroke-width) * cos(calc(360deg / var(--step)) * 11)) calc(var(--stroke-width) * sin(calc(360deg / var(--step)) * 11)) 0 var(--stroke-color),
        calc(var(--stroke-width) * cos(calc(360deg / var(--step)) * 12)) calc(var(--stroke-width) * sin(calc(360deg / var(--step)) * 12)) 0 var(--stroke-color),
        calc(var(--stroke-width) * cos(calc(360deg / var(--step)) * 13)) calc(var(--stroke-width) * sin(calc(360deg / var(--step)) * 13)) 0 var(--stroke-color),
        calc(var(--stroke-width) * cos(calc(360deg / var(--step)) * 14)) calc(var(--stroke-width) * sin(calc(360deg / var(--step)) * 14)) 0 var(--stroke-color),
        calc(var(--stroke-width) * cos(calc(360deg / var(--step)) * 15)) calc(var(--stroke-width) * sin(calc(360deg / var(--step)) * 15)) 0 var(--stroke-color);
    paint-order: fill;
}
```

  


上面代码中[使用了 CSS 的三角函数 sin() 和 cos() 来计算阴影的偏移](https://juejin.cn/book/7223230325122400288/section/7242216512176521277)，并且根据阴影个数 `--step` 计算出 `α` 的值。

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/a3714610f12141c98db301032f76e57a~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1156&h=420&s=865970&e=gif&f=148&b=495958)

  


> Demo 地址：https://codepen.io/airen/full/wBvJaNy

  


为了方便，我们可以通过 JavaScript 根据阴影个数来调整 `text-shadow` ：

  


```JavaScript
const ele = document.querySelector("h1");
const colorInput = document.querySelector("#stroke-color");
const widthInput = document.querySelector("#stroke-width");
const stepInput = document.querySelector("#step");
const widthOutput = document.querySelector('output[for="stroke-width"]');
const stepOutput = document.querySelector('output[for="step"]');

const updateCSSVariables = () =>{
    ele.style.setProperty("--stroke-color", colorInput.value);
    ele.style.setProperty("--stroke-width", `${widthInput.value}px`);
    ele.style.setProperty("--step", stepInput.value);

    widthOutput.textContent = `${widthInput.value}px`;
    stepOutput.textContent = stepInput.value;

    updateTextShadow();
}

const updateTextShadow = () => {
    const step = parseInt(getComputedStyle(ele).getPropertyValue("--step"));
    let shadows = [];

    for (let i = 0; i < step; i++) {
        const angle = (i * 360) / step;
        ele.style.setProperty(`--angle-${i}`, `${angle}deg`);
        shadows.push(`
          calc(var(--stroke-width) * cos(var(--angle-${i}))) 
          calc(var(--stroke-width) * sin(var(--angle-${i}))) 
          0 var(--stroke-color)`);
    }

    ele.style.setProperty("--text-shadow", shadows.join(", "));
}

colorInput.addEventListener("input", updateCSSVariables);
widthInput.addEventListener("input", updateCSSVariables);
stepInput.addEventListener("input", updateCSSVariables);

updateCSSVariables();
```

> Demo 地址：https://codepen.io/airen/full/pvoejeX

  


使用这种方法为文本描边的唯一缺点是，当 `radius` 值较大时，最终效果看起来有点粗糙。这是因为膨胀算法试图将字母向各个方向扩展。这意味着它可能不会渲染一个真正的轮廓，轮廓应该在所有方向上保持一致的大小。

现在我们知道如何使用 CSS 创建渐变文本、给文本描边和添加阴影效果了。有些同学会认为，将这三种效果结合在一起，就可以创建带有描边和阴影的渐变文字效果。真的是这样吗？

  


答案是否定的——直接将这三种效果结合在一起，并不能实现我们想要的文本效果：**带有描边和阴影的渐变文字**！为什么呢？接下来，我将逐步带领大家一探其究竟！

  


首先，我们按照之前创建渐变文本的方法，使用 `background-clip: text` 和 `text-fill-color: transparent` （或 `color: transparent`）创建一个渐变文本：

  


```CSS
h1 {
    --g: linear-gradient(#fff000, #ff3600);
    background: var(--g);
    background-clip: text;
    -webkit-text-fill-color: transparent;
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/be201806d0f04598b35c8bbe6d1d6acb~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2276&h=822&s=159557&e=jpg&b=566262)

  


然后，我们使用 `text-shadow` 给文本添加阴影：

  


```CSS
h1 {
    --g: linear-gradient(#fff000, #ff3600);
    background: var(--g);
    background-clip: text;
    -webkit-text-fill-color: transparent;
    text-shadow: 1px 0 10px #000;
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/f5488187bd874a66a5fd093c5191bd38~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2230&h=828&s=148344&e=jpg&b=586464)

  


正如你所看到的，效果不太好，因为 `text-fill-color: transprent` （或 `color: transparent`）使得文本本身变得透明，而它渲染的阴影实际上是在我们设置的背景之上。这意味着，我们不能直接通过 `text-shadow` 给渐变文本添加阴影效果。

  


庆幸的是，我们可以使用 `filter` 的 `drop-shadow()` 函数给文本添加阴影（即投影）效果，这种方式创建的阴影会位于我们设置的背景之下：

  


```CSS
h1 {
    --g: linear-gradient(#fff000, #ff3600);
    background: var(--g);
    background-clip: text;
    -webkit-text-fill-color: transparent;
    filter: drop-shadow(4px 4px 1px #000);
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/ed1f7d73748f4e8fa8d4cc96cec47d38~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2312&h=836&s=174400&e=jpg&b=576464)

  


现在文本同时拥有渐变和阴影（投影）效果了。接下来，就是在此基础上给文本添加描边效果。

  


之前提到过，我们可以通过给 `text-shadow` 设置多个值给文本添加描边效果，或者使用 `text-stroke` 给文本添加描边效果。在带有渐变文本的情景中，使用 `text-shadow` 是无法给文本添加描边效果的。但是，我们可以给 `filter` 同时设置多个 `drop-shadow()` 函数（有点类似于给 `text-shadow` 设置多个值），从而给文本添加描边效果：

  


```CSS
h1 {
    --g: linear-gradient(#fff000, #ff3600);
    background: var(--g);
    background-clip: text;
    -webkit-text-fill-color: transparent;
    filter: 
        drop-shadow(-4px 4px 1px #000) 
        drop-shadow(3px 2px 1px #000)  
        drop-shadow(-1px -2px 1px #000) 
        drop-shadow(2px 6px 3px #000);
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/53004d0a665249a6a9e63f30e86161c7~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2434&h=1054&s=232831&e=jpg&b=576464)

  


> Demo 地址：https://codepen.io/airen/full/qEBradx

  


需要知道的是，`drop-shadow()` 是以指数形式添加的，即 `(2^（阴影数） – 1)`。比如：

  


-   一个阴影等于 `(2^1 – 1)`，即一个阴影被渲染出来；
-   两人阴影等于 `(2^2 – 1)`，即三个阴影被渲染出来；
-   三个阴影等于 `(2^3 – 1)`，即七个阴影被渲染出来。

  


以此类推。其中一个阴影指的是一个 `drop-shadow()`，两个阴影是指用空格分开的两个 `drop-shadow()`，如下面代码所示：

  


```CSS
/* 渲染出一个阴影 (2^1 – 1) */
.one-drop-shadow {
    filter: drop-shadow(10px 10px 0 red) 
}

/* 渲染出三个阴影 (2^2 – 1) */
.two-drop-shadow {
    filter: 
        drop-shadow(10px 10px 0 red) 
        drop-shadow(20px 20px 0 orange)
}

/* 渲染出七个阴影 (2^3 – 1) */
.two-drop-shadow {
    filter: 
        drop-shadow(10px 10px 0 red) 
        drop-shadow(20px 20px 0 orange)
        drop-shadow(30px 30px 0 blue)
}
```

  


这意味着，使用多个 `drop-shadow()` 给文本添加描边效果会使阴影层级变多，影响到阴影的实际效果。基于这个原因，我个人推荐使用 `text-stroke` 给文本添加描边效果：

  


```CSS
h1 {
    --g: linear-gradient(#fff000, #ff3600);
    background: var(--g);
    background-clip: text;
    -webkit-text-fill-color: transparent;
    filter: drop-shadow(2px 6px 3px #000);
    -webkit-text-stroke: 2px #000;
    paint-order: stroke;
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/c220fcad90924a1abce38ad5039c8126~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2254&h=922&s=186565&e=jpg&b=556262)

  


> Demo 地址：https://codepen.io/airen/full/QwWpKOE

  


正如你所看到的，我们使用 CSS 创建了一个带有描边和阴影的渐变文本效果！

  


当然，也有人推荐使用伪元素 `::after` 或 `::before` 和 `text-shadow` 结合来创建带有描边和阴影的渐变文字效果。

![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/110e35bb10134465bf573ba09da3e910~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2488&h=958&s=193480&e=jpg&b=566363)

## SVG 创建带有描边和阴影的渐变文字

  


上面我们主要介绍了如何使用 CSS 创建带有描边和阴影的渐变文字效果。事实上，我们还可以使用一些 SVG 特性来创建相似的效果：

  


-   使用 [SVG 的 `<linearGradient>`](https://juejin.cn/book/7341630791099383835/section/7354948936039137289) 给[文本元素 `<text>` 或 `<tspan>` ](https://juejin.cn/book/7341630791099383835/section/7346773005114507304)填充渐变背景，创建渐变文本
-   使用 [SVG 滤镜](https://juejin.cn/book/7341630791099383835/section/7366549423746187273)中的 `<feDropShadow>` 给渐变文本[描边](https://juejin.cn/book/7341630791099383835/section/7368318146262138889)和[添加阴影](https://juejin.cn/book/7341630791099383835/section/7368318391733452850)

  


这个方法虽然复杂，但你可以轻松地实现所需要的效果：

  


```HTML
<svg class="text">
    <defs>
        <linearGradient id="gradient" x1="0%" y1="0%" x2="0%" y2="100%">
            <stop offset="0%" stop-color="#fff000" />
            <stop offset="100%" stop-color="#ff3600" />
        </linearGradient>
    </defs>
    <filter id="shadow">
        <feDropShadow dx="-4" dy="4" stdDeviation="1" flood-color="black" />
        <feDropShadow dx="3" dy="2" stdDeviation="2" flood-color="black" />
        <feDropShadow dx="-1" dy="-2" stdDeviation="1" flood-color="black" />
    </filter>
    <text x="10" y="50" font-family="Rubik Mono One" font-size="40" fill="url(#gradient)">
        <tspan filter="url(#shadow)">Ends in 7 Days</tspan>
    </text>
</svg>
```
