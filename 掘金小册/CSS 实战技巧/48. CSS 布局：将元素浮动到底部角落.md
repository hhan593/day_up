随着更多的[现代 Web 布局技术](https://s.juejin.cn/ds/iUdGMhFK/)的出现（例如 **[CSS Flexbox](https://juejin.cn/book/7161370789680250917/section/7161621092560273439)** 和 **[CSS Grid](https://juejin.cn/book/7161370789680250917/section/7161372229123440648)** 布局技术等）。Web 开发者认为使用这些现代 Web 布局技术可以梭哈所有布局。而且很多 Web 开发者可能遗忘了 **[CSS 的浮动布局](https://juejin.cn/book/7161370789680250917/section/7161370789768347685#heading-6)**（`float`）技术，甚至认为 `float` 可以退出历史的舞台了。

  


事实上呢？并非如此，Web 上的一些布局效果不得不依赖于 CSS 的浮动布局。例如，[文本围绕着不规则的形状排版](https://juejin.cn/book/7161370789680250917/section/7161625566062313503)时，离开了浮动是很难实现的：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/7efd493769ef4d1c9e6fa19b07c0c6dc~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=3040&h=1145&s=2780787&e=png&a=1&b=1b1b1b)

![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/fedf7005f7064212a08a8a8ad1021c49~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1290&s=1490625&e=jpg&b=fbf9f9)

  


我想有着多年前端开发经验的你，要实现诸如此类的布局效果应该有点棘手，对吧？就拿上图中第一个卡片为例，我就经常会被问及到，它应该如何实现？直到现在，很多 Web 开发者不得不借助 JavaScript 能力来实现它。

## CSS 浮动：`float`

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/9d3a5d8e731840259a054d425e81169a~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1280&h=720&s=59462&e=jpg&b=fffefe)

  


我们这节课的主题是“**将元素浮动到容器的底部角落**”。既然如此，我们先来回忆一下久违的 `float` 。在 CSS 中，我们可以使用 `float` 属性，将元素浮动到左侧（`left`）或右侧（`right`）。它们俗称为左浮动和右浮动。

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/7c6060f43bfe4b21b49330fc26a237e0~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1728&h=1080&s=417739&e=jpg&b=ffffff)

  


浮动的使用就是这么简单，其主要功能就是让文本围绕着浮动元素排列。只不过，早期还没有那么多布局技术的时候，也用它来构建 Web 布局。现如今，它重新回归到初衷，尤其是与 CSS 的 `shape-outside` 或 `shape-inside` 组合在一起的时候，[它能允许我们让文本围绕着不规则的形状排列](https://juejin.cn/book/7161370789680250917/section/7161625566062313503)：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/4283b5f2f3d94c61a818280a2e8e104d~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1802&h=1268&s=1332945&e=png&b=bfc7d0)

  


但是，使用浮动 `float` 无法直接将一个元素紧贴容器的底部角落：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/4453ce502fc24d61b6ddcdaf78a34e78~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1200&h=600&s=18178&e=webp&b=ffffff)

当然，`float` 也没有 `cneter` 值。因此，也无法直接使浮动元素位于文本内容的中间：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/53537e6d1e2640c48f7d69817feffd6d~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=3328&h=1788&s=1099734&e=jpg&b=fefefe)

  


庆幸的是，我们使用现代 CSS 的一些 CSS 特性，仅需要几行 CSS 就能实现期望的效果！

  


## 将元素浮动到底部角落

  


我们从 HTML 结构开始！需要一个容器元素来包含所有内容，例如：

  


```HTML
<div class="wrapper">
    <div class="box">
        <figure class="float">
            <img src="https://picsum.photos/id/163/200/200" alt="" />
        </figure>
        <p>Lorem ipsum dolor ...</p>
    </div>
</div>    
```

  


按照常规的方式，图片（即 `.float` 元素）只能浮动到容器顶部左侧或右侧：

  


```CSS
.wrapper {
    display: flex;
    
    .float {
        float: inline-end; 
        margin-inline-start: 1rem;
    }
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/aa6c81eecd144bd4a5d7a9edf1e46c7f~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1432&h=708&s=9129284&e=gif&f=129&b=557080)

  


注意，上面的代码中使用的是 `inline-end` 逻辑值，如果书写模式是 `ltr` 的话，它相当于 `right` ；如果书写模式是 `rtl` ，它相当于 `left` 。如果你对这方面知识感兴趣的话，可以移步阅读下面这几节课的内容：

  


-   [CSS 的逻辑属性和逻辑值](https://juejin.cn/book/7223230325122400288/section/7254820664442355773)
-   [Web 中的向左向右：Flexbox 和 Grid 布局中的 LTR 与 RTL](https://juejin.cn/book/7161370789680250917/section/7161625525763440647)
-   [Web 中的向左向右：Web 布局中 LTR 切换到 RTL 常见错误](https://juejin.cn/book/7161370789680250917/section/7161625415935590436)

  


接下来，为了使浮动元素 `.float` 能够浮动到容器右下角，我们给浮动元素添加下面这几行 CSS：

  


```CSS
.wrapper {
    display: flex;
    
    .float {
        float: inline-end; 
        margin-inline-start: 1rem;
        height: 100%; /* 这个很重要 */
        display: flex;
        align-items: flex-end;
        shape-outside: inset(calc(100% - 200px) 0 0);
    }
}
```

  


现在，你会发现，图片位于容器的右下角：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/4a3c17c024b74f299b8ee830959149a7~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1476&h=612&s=16250852&e=gif&f=197&b=557080)

  


> Demo 地址：https://codepen.io/airen/full/JoPRojO

  


你可能很好奇，为什么这样做，就可以将图片浮动到元素的底部角落。请接着往下阅读，我将逐步向你解开其中的奥秘。

  


首先，我们将 `.wrapper` 容器定义为一个 Flexbox 容器，它会使其子元素 `.box` 填满整个容器，因为 `.box` 是 `.wrapper` 的唯一子元素，Flexbox容器的 `align-items: stretch` 会拉伸 `.box` ：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/4b460a13939a45e58a8f7fceb19e0c2d~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2712&h=1226&s=508820&e=jpg&b=667e8a)

  


你会发现，我们在 `.box` 元素上没有设置任何样式，它的存在只是为了给浮动元素 `.float` 的 `height: 100%` 提供一个“**参考高度**”。这是因为，只有父容器有明确的高度（`height`）时，百分比的高度值才是安全的。这意味着，要使 `.float` 元素的 `height: 100%` 安全使用，就需要使 `.box` 元素有一个明确的高度值。你可能又纳闷了，代码中并没有给 `.box` 设置 `height` 值呀？这是因为，我们这里使用了 Flexbox 布局，当 Flex 项目（在这里就是 `.box` 元素）被拉伸时，将会根据该项目的已使用尺寸重新布局其内容，从而使百分比尺寸的子元素可以被正确解析。简单地说，默认被拉伸的 Flex 项目有确定的尺寸，这让它的子元素 `.box` 的 `height: 100%` 有相应的参考高度，可以安全使用。在我们这个示例中，`height` 为 `100%` 的 `.float` 元素，其高度等同于 `.box` 元素的高度，也相应的等同于 `.wrapper` 容器的高度：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/96c039b9515547df8756a809098ce75e~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2766&h=1162&s=501033&e=jpg&b=667e8a)

  


正如你所看到的，我们的 `.float` 元素（红色虚线框）会占据文本内容旁边的整个高度（绿色的虚线框），这是基于上面提到的高度计算出来的结果。

  


同样的，我们将 `.float` 元素定义为一个 Flexbox 容器，并且将 `align-items` 设置为 `flex-end` ，将图片推到容器的底部。这段代码的核心是**利用 CSS Flexbox 和** **`float`** **的结合，巧妙地实现了内容自适应高度和图片的底部对齐效果**。

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/8d79cfb453b54fca9b87baa1e933983e~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2688&h=1224&s=486443&e=jpg&b=667e8a)

  


仅仅这样还是不够的，文本并不会围绕着浮动元素 `.float` 排列。为了改变这一现象，我们将用到另一个核心技巧：**使用** **`shape-outside`** **属性，使文本围绕着形状排列**。只有这样，在视觉上才能使文本围绕着浮元素排列：

  


```CSS
.float {
    shape-outside: inset(calc(100% - 200px) 0 0);
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/d98266c82eca43978aa02be5923bbdbd~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2614&h=1128&s=464086&e=jpg&b=667e8a)

  


简单解释一下。CSS 的 `shape-outside` 属性有点类似于 `clip-path` 属性，通过诸如 `inset()` 、`circle()` 、`ellipse()` 和 `polygon()` 等函数，可以获得一个非矩形的形状，相邻的行内内容会围绕这个形状排列。默认情况下，行内内容围绕元素的外边距框（`margin-box`）排列。换句话说，`shape-outside` 决定了内容如何围绕元素的外边距框流动。

  


> 温馨提示，有关于 `shape-outside` 更详细的介绍，可以移步阅读《[创建不规则 Web 布局](https://juejin.cn/book/7161370789680250917/section/7161625566062313503)》一课！

  


在我们这个示例中，使用了 `inset()` 函数将浮动元素定义为一个矩形，即创建一个从图片顶部开始的内嵌矩形。其中：

  


-   **`100% - X`** 表示顶部的起始位置，
-   **`X`** 是图片的高度，
-   **`100%`** 是 `.float` 元素的高度。

  


这使得文本可以在图片顶部的空白区域中环绕排列。这种方式具有响应性，同时可以通过调整 `float` 属性轻松切换图片的左右位置。

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/ec1fc7b46b97408eaa4b901f8f78dea1~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=3322&h=1182&s=615031&e=jpg&b=667e8a)

  


> Demo 地址：https://codepen.io/airen/full/MYgjgeG

  


需要知道的是，这种方案有一个较为明显的缺陷，那就是**需要知道浮动元素（** **`.float`** **）的高度，否则** **`inset()`** **不好定义形状**！

另外，你也可以把上面示例中的 CSS Flexbox 布局替换成 CSS Grid 布局，可以达到相同的效果：

  


```CSS
.wrapper {
    display: grid;
    
    .float {
        float: inline-start;
        margin-inline-end: 1rem;
        height: 100%;
        display: grid;
        align-items: end;
        shape-outside: inset(calc(100% - 200px) 0 0);
    }
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/a05c3f8c7cd542d1928ca4623f2ada3b~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1666&h=698&s=7652310&e=gif&f=126&b=557080)

  


> Demo 地址：https://codepen.io/airen/full/xbKEwmd

  


你可能发现了，上面示例中我用了一个 `<figure>` 元素来包裹 `<img>` 元素，其实也可以不用任何元素来包裹 `<img>` ：

  


```HTML
<div class="wrapper">
    <div class="box">
        <img src="https://picsum.photos/id/163/200/200" alt="" class="float" />
        <p>Lorem ipsum dolor ...</p>
    </div>
</div>  
```

  


对应的 CSS 变为：

  


```CSS
.wrapper {
    display: flex;

    .float {
        float: inline-start;
        margin-inline-end: 1rem;
        height: 100%;
        width: 200px; 
        object-fit: contain;
        object-position: bottom;
        display: flex;
        align-items: flex-end;
        shape-outside: inset(calc(100% - 200px) 0 0);
    }
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/af2e5b9c76f14b4bb14b4bdcbf8e2263~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1666&h=698&s=7652310&e=gif&f=126&b=557080)

  


> Demo 地址：https://codepen.io/airen/full/bNbwEqG

  


移除图片外部的容器之后，我们需要对图片做一些额外的处理。在这里使用了 `object-fit 和 object-position` 属性：

  


-   `object-fit: contain` 确保图片按比例缩放以适应指定的宽高
-   `object-position: bottom` 将图片对齐底部（它有点类似于 `background-position: bottom`）

  


但要注意，使用这种方法时，我们需要同时知道图片宽高，而不像之前的案例，只需要知道图片的高度即可。

  


在这个基础上，我们还可以将两张图片浮动到两个底角，甚至还可以将图片放置在四个角：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/6b31d72940914806b23f1661b6c28b54~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=3336&h=1618&s=1209956&e=jpg&b=677e8a)

  


> Demo 地址：https://codepen.io/airen/full/RNbGZyO

  


两张图片分别浮动到容器的左下角和右下角并不复杂，只需要简单的将 `.float` 使用两次，一个左浮动，一个右浮动：

  


```CSS
.float {
    float: inline-start;
    margin-inline-end: 1rem;
    height: 100%;
    width: 200px;
    object-fit: contain;
    object-position: bottom;
    display: flex;
    align-items: flex-end;
    shape-outside: inset(calc(100% - 200px) 0 0);
    
    ~ .float {
        float: inline-end;
        margin-inline-end: 0;
        margin-inline-start: 1rem;
    }
}
```

  


至于四个角落都有图片的效果，实现起来稍微复杂一点。其中顶部两张图片，我们可以使用传统的浮动方式来实现，底部的两张图片继续使用这节课所介绍的技巧。不过，底部两张图片我们需要通过 `margin` 来调整其位置：

  


```CSS
img {
    float: inline-start;
    margin-inline-end: 1rem;
      
    ~ img {
        float: inline-end;
        margin-inline-start:1rem; 
    }
}

.float {
    float: inline-start;
    height: 100%;
    width: 200px;
    object-fit: contain;
    object-position: bottom;
    display: flex;
    align-items: flex-end;
    shape-outside: inset(calc(100% - 200px) 0 0);
    
    margin-inline-end: calc(1rem + .1px);
    margin-inline-start: calc(-200px - 1rem);
      
    ~ .float {
        float: inline-end;
        margin-inline-end: calc(-200px - 1rem);
        margin-inline-start: calc(1rem + .1px);
    }
}
```

  


注意，容器顶部浮动的 `<img>` 元素，并没有添加 `float` 类名。

  


## 进一步扩展这个概念：浮动居中

  


现在，你已经知道如何将元素浮动到容器的每个角落了。接下来，我们可以进一步扩展这个概念。比如将浮动元素浮动到容器一侧的中间位置。

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/8e698e0a2ed84a3bbbbd9fe1834c4c87~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=3340&h=1082&s=877731&e=jpg&b=677e8a)

  


> Demo 地址：https://codepen.io/airen/full/wBwzrGo

  


对应的 CSS 代码如下：

  


```CSS
.wrapper {
    display: flex;
}

.float {
    /* 浮动到右上角 */
    .wrapper:nth-child(1) & {
        float: inline-end;
        margin-inline-start: 1rem;
    }

    /* 浮动到右侧中间 */
    .wrapper:nth-child(2) & {
        float: inline-end;
        margin-inline-start: 1rem;
        height: 100%;
        display: flex;
        flex-direction: column;
        justify-content: center;
        shape-outside: inset(calc(50% - 75px) 0);
    }

    /* 浮动到右下角 */
    .wrapper:nth-child(3) & {
        float: inline-end;
        margin-inline-start: 1rem;
        height: 100%;
        display: flex;
        align-items: flex-end;
        shape-outside: inset(calc(100% - 150px) 0 0);
    }
}
```

  


图片浮动到容器的右上角和右下角对应的代码比较好理解，我们之前已经介绍过了。我们主要来看图片浮动到容器右侧中间（垂直方向的中间）位置对应的 CSS 代码。首先，我们改变了 Flexbox 容器的主轴方向，变成了 `column` 排列，同时通过 `justify-content: center` 将其固定在容器的中间位置。此外，要使文本也能围绕着图片底部排列，我们需要调整 `shape-outside` 属性的值，从 `inset(calc(100% - X) 0 0)` 改变 ` inset(calc(50% - X  ``/ 2``) 0)` ，这样矩形会调整到图片的垂直居中位置。

  


在单侧居中浮动的基础上稍微改良一下，我们就可以获得浮动居中的效果：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/92992ef8d5c94af29248520f8f6c9a44~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1400&h=884&s=12371790&e=gif&f=145&b=557080)

  


> Demo 地址：https://codepen.io/airen/full/QwLKOGJ

  


由于 `float` 并没有 `center` 这样的特性，因此要实现这个布局效果，还是需要使用一些 Hack 手段的。我们通过可以将两个侧边居中的效果合并起来，模拟出整个元素浮动居中的效果。这也意味着，实现这种效果，对 HTML 结构有一定的要求，需要将文本内容分成两列：

  


```HTML
<div class="wrapper">
    <div class="columns">
        <!-- 左侧列 -->
        <div class="column">
            <!-- 浮动元素，浮动到右侧 -->
            <div class="outside">
                <figure>
                    <img src="https://picsum.photos/id/163/800/800" alt="">

            </div>

        </div>

        </div>
    </div>
</div>
```

  


我们可以通过 Flexbox 或 Grid 布局，将 `.column` 排列到一行，即实现两列布局：

```CSS
    gap: var(--gap);
}
```

接着，我们使用前面示例中的 CSS 技巧，将第一列的浮动元素 `.outside` 浮动到右侧，第二列的浮动元素 `.outside` 浮动到左侧。与此同时，使用 `shape-outside` 定义所需要的形状，我这里使用 `circle()` 定义了一个圆形。其中最为核心的是，这两个形状在 `Y` 轴的位置保持一致，这样才能拼出完整的圆形，当然你也可以使用任意你喜欢的形状：

  


```CSS
.outside {
    position: relative;
    inline-size: calc(var(--size) * 1px);
    block-size: calc(100% + (var(--size) * 1px));

        float: inline-end;

        shape-outside: circle(calc(var(--size) * 1px) at 0 calc(var(--y) * 1%));

}
```

```CSS
figure {
```

## 改变形状

  


[稍微了解 shape-outside 或 shape-inside 的开发者都知道](https://juejin.cn/book/7161370789680250917/section/7161625566062313503#heading-9)，它们可以通过 `inset()` （矩形）、`circle()` （圆形）、`ellipse()` （椭圆形）和 `polygon()` （多边形）来定义形状，也知道可以通过 `url()` 引用外部图像来定义形状。

  


在这里，我想告诉大家的是，我们还可以使用渐变来定义形状。这使得 `shape-outside` 定义不同形状变得更为灵活，因为 CSS 的渐变常常会被用于创建不同的形状，这在之前的课程中有过很多这方面的案例。

  


下面这个示例演示了 `shape-outside` 使用 `linear-gradient()` （定义矩形或三角形）、`radial-gradient()` （定义圆形）和 `conic-gradient()` （定义三角形）等渐变函数定义的形状：

  


```CSS
.wrapper {
    display: flex;
}

.float {
    --w: 150px;
    float: inline-end;
    height: 100%;
    margin-inline-start: 1rem;
    display: flex;
    flex-direction: column;
    justify-content: flex-end;
    shape-outside: var(--shape);
    shape-margin: 1rem;

    .wrapper:nth-child(1) & {
        --shape: linear-gradient(0deg, #fff var(--w), #0000 0);
    }
    
    .wrapper:nth-child(2) & {
        --shape: linear-gradient(-45deg,#000 calc(cos(45deg) * var(--w)),#0000 0);
    }
    
    .wrapper:nth-child(3) & {
        --shape: radial-gradient(var(--w) at bottom right, #000 98%, #0000);
    }
    
    .wrapper:nth-child(4) & {
        justify-content: center;
        --shape: conic-gradient(from 60deg at left, #fff 60deg, transparent 0);
    }
}

img {
    .wrapper:nth-child(2) & {
        clip-path: polygon(100% 0, 100% 100%, 0 100%);
    }

    .wrapper:nth-child(3) & {
        border-radius: 500px 0 0 0;
    }
    
    .wrapper:nth-child(4) & {
        clip-path: polygon(0 50%, 100% 0, 100% 100%);
    }
}
```

## 案例：实现多行文本的展开与折叠

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/5dddb0523b03403f93c58c862b4f95a6~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1290&s=636348&e=jpg&b=607d8b)

  


这个效果核心有三点：

  


-   1️⃣：按钮位于指示符后面
-   2️⃣：超出指定行数，文本被截断，并在末尾添加省略号指示符 `...`
-   3️⃣：点击按钮展示所有文本

  


这三点对应的解决方案：

  


-   1️⃣：使用之前介绍的 `float` 和 `shape-outside` ，将按钮元素浮动到容器的右下角
-   2️⃣：使用 `line-clamp` 来截断文本，并在文本末尾添加 `...` 指示符
-   3️⃣：使用 `:has()` 选择器或 `~` 选择器，控制 `input` （复选框 `checkbox`）的 `:checked` 状态，从而达到控制文本展开与折叠的切换

  


当然，在创建这个效果过程中，还将会涉及到一些其他的 CSS 技巧，比如 `lh` 单位的应用。我们先从它的 HTML 结构开始：

  


```HTML
<div class="card">
    <figure>
        <img src="https://picsum.photos/id/163/300/300" alt="">
    </figure>
    <h3>Float an Element to the Bottom Corner</h3>
    <div class="content">
        <div class="content__inner">
            <div class="action">
                <button>Read More</button>
            </div>
            <p>Earum, deserunt quaerat.</p>
        </div>
    </div>
</div>
```

  


注意，上面代码并没有引入 `<input type="checkbox" />` 。在不考虑交互行为的前提之下，暂时还用不到它。

  


整个卡片我使用了 CSS Grid 来创建布局，并且按照之前所介绍的技巧（`float` 和 `shape-outside` 相结合），将按钮浮动到容器 `.content` 的右下角：

  


```CSS
.card {
    display: grid;
    grid-template-columns: 200px minmax(0, 1fr);
    grid-template-rows: auto minmax(0, 1fr);
    gap: 1rem;
    grid-template-areas:
        "thumb title"
        "thumb content";
    align-content: start;
    align-items: start;
    
    figure {
        grid-area: thumb;
    }

    h3 {
        grid-area: title;
    }

    .content {
        grid-area: content;
    }
}

.content {
    display: flex;

    .action {
        float: inline-end;
        margin-inline-start: 1rem;
        height: 100%; /* 这个很重要 */
        shape-outside: inset(6lh 0 0);
        display: flex;
        align-items: flex-end;
    }
}
```

  


请注意，上面的代码中，我在 `inset()` 函数中使用了一个 `6lh` 替代了之前的 `100% - X` （其中 `X` 是浮动元素的高度）。由于我们不太好明确的确定浮动元素的高度，所以我做了一个反向的思考，即 `inset()` 创建矩形时，它距离顶部的距离与我们所要展示的文本行数来决定。假设我们的文本最多展示 `5` 行，每行的高度是 `1lh` ，按理说，我们应该使用的是 `5lh` 。这个想法并没有任何问题，如果设置为 `5lh` ，它就相当于 `100% - 2lh` （示例中的按钮高度是通过 `line-height: 2` 来控制的）。但是，这样做会产生一些我也无法解释清楚的效果——**浮动元素顶部有更多的留白**：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/30761cb2e3b843e4a041b4d131ea81cc~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=511&s=475527&e=jpg&b=faf6f6)

  


再次强调一下，在实际使用过程中，`inset()` 函数的顶部值应该根据 `line-clamp` 属性的值来变化，如果 `line-clamp` 的值为 `n` ，那么 `inset()` 函数的顶部值就是 `n+1` 个 `lh` 。

  


接下来，就是截断文本，这对于 CSS 来说，非常容易。下面的代码即可实现：

  


```CSS
.content__inner {
    display: -webkit-box;
    -webkit-line-clamp: 5;
    -webkit-box-orient: vertical;
    overflow: hidden;
}
```

  


但是，这里有一个小技巧。按理来说，我们应该是对 `<p>` 元素的文本进行截断，为什么要在它的父容器上做文本截断呢？主要原因是，如果我们在 `<p>` 元素上截断文本，其中 `display: -webkit-box` 或者 `overflow: hidden` 中的任何一个样式规则都会触发清除浮动的机制，这会造成文本无法围绕着 `shape-outside` 创建的形状排列，在视觉上就无法实现文本围绕着按钮排列的效果。

  


到目前为止，我们通过下面的 CSS 代码使我们实现了一个不带交互的效果——**按钮放置在截断文本指示符之后**：

  


```CSS
.content {
    display: flex;

    .action {
        float: inline-end;
        margin-inline-start: 1rem;
        height: 100%; /* 这个很重要 */
        shape-outside: inset(6lh 0 0);
        display: flex;
        align-items: flex-end;
    }

    .content__inner {
        display: -webkit-box;
        -webkit-line-clamp: 5;
        -webkit-box-orient: vertical;
        overflow: hidden;
    }
}
```

  


这个效果也是 Web 开发者在平时开发中时常碰到的一个令人感到痛苦的需求：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/ca7eb1b02ebb43f295441c4df62ba631~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1011&s=328436&e=jpg&b=607d8b)

  


> Demo 地下：https://codepen.io/airen/full/EaYgLbb

  


如果这个“Read More”并不是一个真正的按钮，只它是一个跳转链接，那么到这就完成了你所需要的 UI 效果。要是，你希望截断的文本与按钮有一个交互效果——文本具有折叠和展开效果，那么还需要接着处理。

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/687cf9484c114e4ca5bd8e766908f335~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1212&h=738&s=2045987&e=gif&f=130&b=557080)

  


以往实现上图这个效果，或多或少都需要借助 JavaScript 来实现，甚至直到今天仍然有不少 Web 开发者是通过 JavaScript 来实现。接下来，我就告诉大家一种纯 CSS 实现的方案。首先，我们在上面示例的基础上引入 `type` 为 `checkbox` 的 `input` ，并且将 `button` 替换为 `label` ，并且将`label` 的 `for` 属性值设置为`input` 的 `id` 值，将两者绑定在一起：

  


```HTML
<div class="card">
    <input type="checkbox" class="sr-only" id="toggle" name="toggle"  />
    <figure>
        <img src="https://picsum.photos/id/163/300/300" alt="">
    </figure>
    <h3>Float an Element to the Bottom Corner</h3>
    <div class="content">
        <div class="content__inner">
            <label class="action" for="toggle">
                <span class="button"></span>
            </label>
            <p>Earum, deserunt quaerat id sapiente.</p>
        </div>
    </div>
</div>
```

  


注意，`input` 复选框在视觉上并不可见，因此我使用下面的 CSS 代码在视觉上将其隐藏，但屏幕阅读器还是能识别到它：

  


```CSS
.sr-only {
    position: absolute;
    width: 1px;
    height: 1px;
    padding: 0;
    margin: -1px;
    overflow: hidden;
    clip: rect(0, 0, 0, 0);
    white-space: nowrap;
    border-width: 0;
}
```

  


另外，“按钮”的文本需要在“展开”和“收起”之间来回切换，所以我借助了 `.button` 的伪元素 `::before` 来实现。复选框未选中时，`::before` 的 `content` 属性的值为`展开` ，反之则是 `收起` 。

  


现在，我们只需要在上一个示例的基础上，添加部分 CSS 来控制“展开”状态的样式—— `input:checked` 状态下，截断文本、浮动元素和按钮的样式处理。 CSS 中有两种方式可以处理，第一种是更为传统的方式，使用 `~` 选择器来处理相关元素的样式；第二种方式更现代一些，就是通过 `:has()` 选择器来进行判断。

  


需要知道的是，使用第一种方式对 HTML 结构有较强的要求，`input` 必须放置在要被样式化的元素之前。因此，你会发现，我将 `input` 放置了 `.content` 元素之前。

  


刚才提到过了，`input:checked` 选中状态，我们要做三件事：

  


-   1️⃣：文本不再被截断，重置 `.content__inner` 元素的 `line-clamp` 来实现
-   2️⃣：浮动元素（按钮）浮动到容器右一角，我们需要调整 `inset()` 函数的值，因为在展开状态下，并不知道文本内容有多少行。所以，将 `6lh` 重置为 `100% - X`
-   3️⃣：将伪元素的 `content` 设置为 `收起`

  


对应的 CSS 代码如下：

  


```CSS
#toggle:checked ~ .content {
    .content__inner {
        -webkit-line-clamp: unset;
        display: block;
    }
    
    .action {
        shape-outside: inset(calc(100% - 1lh) 0 0);
    }
    
    .button::before {
        content: "收起";
    }
}
```

  


这样就完成了你所需要的效果：

使用 `:has()` 对于 HTML 结构的要求没有那么强，我们可以将 `input` 和 `label` 标签放置在一起：

```HTML
<div class="card">
```

## 小结

  


在这节课中，我们探讨了 `float` 和 `shape-outside` 将元素浮动到底部角落（也可以是中间）的布局效果。事实上，这种方法不仅可以将矩形放置在容器的角落，还可以放置任意形状在任意位置，并且代码结构非常相似。我们只需要调整 `shape-outside` 属性定义的形状即可。即使是你构建一个响应式 Web 布局，课程中所介绍的 CSS 技巧也能满足。
