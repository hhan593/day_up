图像是 Web 中的重要元素之一。引人注目的图像配合炫酷的效果，是给用户留下深刻印象的关键。虽然有很多方法可以为图像添加样式，但这节课，我将介绍如何使用 CSS 图像添加炫酷的悬停效果，而且这些效果都是基于单个 `img` 元素实现。

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/a1c3394737734d3f84dd63a9f802d32e~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1336&h=702&s=6115935&e=gif&f=131&b=fdf7f6)

  


> Demo 地址：https://codepen.io/airen/full/emmXrbO

  


接下来，我将通过真实的用例向大家展示，如何使用 CSS 的特性（比如 `outline` 、`clip-path` 、`mask` 、渐变以及 CSS 自定义属性等）基于单独的 `img` 元素，为图像创建奇妙的悬停效果。

  


## 覆盖层效果

  


我们先从最简单的覆盖层效果开始。将鼠标悬停在图像上，它有一个漂亮的过渡效果，图像上方的暗色覆盖层会被移除：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/1bf087e6472342a59e75303ed8d8a681~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1334&h=674&s=3890795&e=gif&f=127&b=fef9f8)

  


> Demo 地址：https://codepen.io/airen/full/Qwwoxzb

  


以往要实现这个效果，我们一般是在图像上覆盖一个元素或伪元素来实现。现在，我们只需要在一个 `<img>` 元素上应用下面这段 CSS 代码就可以实现：

  


```CSS
img {
    --c: #f0a;
    --s: 50vmin; /* 图片宽度 */
    --o: calc(var(--s) / 2); /* 图片宽度的一半 */
    
    width: var(--s);
    aspect-ratio: 1;
    cursor: pointer;
    transition: 0.6s;
    
    outline: var(--o) solid color-mix(in oklch, var(--c), #0000 50%);
    outline-offset: calc(-1 * var(--o));

    &:hover {
        --o: 0vmin;
    }
}
```

  


你可能会问，为什么 `outline` 会产生这样的效果？我用下图来解释这个技巧背后的工作原理：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/154e19751f55448aa72c0dfeb81da7f4~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=5203&h=2489&s=1227588&e=jpg&b=fcfcfc)

  


一开始，我们使用 `outline` 给图片添加了一个半透明颜色的轮廓，该轮廓的厚度为图片宽度的一半，这个时候你将得到像上图中步骤 1️⃣ 所示的效果。接着，给 `outline-offset` 设置一个负值，轮廓会向内收缩，并覆盖在图像上（如上图中步骤 2️⃣ 所示）。当 `outline` 的宽度和偏移量使用相同的数值时，就会得到上图中步骤 3️⃣ 所示的覆盖效果。

  


在代码中，我们通过 `--s` 和 `--o` 来定义图片的宽度和轮廓的厚度以及偏移量。注意，我们需要将轮廓向内收缩，因此要使用 `calc()` 将 `--o` 乘以 `-1` ，变成负值，只有这样才是向内收缩。当 `--o` 变量设置为 `--s` 一半时，轮廓覆盖在图片上，因此你能看到一个带有透明度的紫色层覆盖在图像上；当 `--o` 变量设置为 `0` 时，`outline` 就会隐藏，图像也就没有覆盖层了。因此，在鼠标悬停在图像上时，将 `--o` 设置为 `0` ，以实现淡出的效果。

  


覆盖层淡出有一个平滑的过渡效果，这是因为我们对所有属性进行了过渡处理（`transition: .6s`），`transition` 没有显式指定具体过渡属性时，它的值为 `all` ，表示所有可过渡的属性。

  


在现代浏览器中，这个效果同样适用于圆角图像，我们只需要给 `img` 的 `border-radius` 属性指定一个值即可：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/9e3816fe88b64fcc889e4fff7799fc96~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1334&h=638&s=8478029&e=gif&f=125&b=fff9f8)

  


> Demo 地址：https://codepen.io/airen/full/JoozBba

  


不知道你是否注意到了，在上面这个示例中，我们显式指定了图片的尺寸。但在一些场景中，我们可能无法给图片指定具体的尺寸（比如创建一个响应式 Web 设计的时候）。这意味着，`--o` 变量无法与图片的尺寸关联起来。这样的情景中，我们应该如何实现这个悬停效果呢？

  


很简单，我们可以将 `--o` 变量的值设置为 `50cqi` （一个较大值），这里选择 `50cqi` ，是因为它可以相对于容器查询的宽度进行计算，当然你也可以设置为其他的一个较大值，比如 `50vw` 之类。

  


```CSS
img {
    --c: #f0a;
    --o: 50cqi;
    width: 100%;
    aspect-ratio: 1;
    cursor: pointer;
    transition: 0.6s;

    outline: var(--o) solid color-mix(in oklch, var(--c), #0000 50%);
    outline-offset: calc(-1 * var(--o));
}
```

这个时候，你得到了一个类似下图所示的效果：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/a037b8c9694e4483b7e3dfa4243b5a3f~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2286&h=1368&s=1150885&e=png&b=ec8cbf)

  


`outline` 不仅覆盖了图像，还覆盖了其他元素，这是我们不想看到的结果。庆幸的是，我们可以使用 `clip-path` 的 `inset()` 对图像进行裁剪，将溢出图像边缘的轮廓裁剪掉：

  


```CSS
img {
    --c: #f0a;
    --o: 50cqi;
    width: 100%;
    aspect-ratio: 1;
    cursor: pointer;
    transition: 0.6s;

    outline: var(--o) solid color-mix(in oklch, var(--c), #0000 50%);
    outline-offset: calc(-1 * var(--o));
    clip-path: inset(0);
    
    &:hover {
        --o: 0;
    }
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/3722e4718bf54e4bb2ece87989ea1acf~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1328&h=674&s=8930874&e=gif&f=109&b=fff8f6)

  


正如你所看到的，矩形图像效果已经很完美了，带有圆角的效果还需进一步处理。我们可以在 `inset()` 添加 `round` 关键词，并指定圆角半径与图像的 `border-radius` 相等，就可以解决这个问题。同样地，为了更易于管理，我们引入 `--r` 变量，来指定图片的圆角半径以及裁剪区域的圆角半径：

  


```CSS
img {
    --c: #f0a;
    --o: 50cqi;
    --r: 50%;
    width: 100%;
    aspect-ratio: 1;
    cursor: pointer;
    transition: 0.6s;

    outline: var(--o) solid color-mix(in oklch, var(--c), #0000 50%);
    outline-offset: calc(-1 * var(--o));
            clip-path: inset(0);
    
    &:nth-child(2) {
        border-radius: var(--r);
        clip-path: inset(0 round var(--r));
    }

    &:hover {
        --o: 0vmin;
    }
}
```

  


这样你就得到了一个完美的覆盖层悬停效果：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/97e1a04af49240698836bd6d85245dc1~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1332&h=640&s=4426988&e=gif&f=97&b=fff9f8)

  


> Demo 地址：https://codepen.io/airen/full/oggVMeV

  


温馨提示一下，`cqi` 和 `vw` 是 CSS 中的相对单位，前者相对于查询容器的宽度计算，后者相对于浏览器视窗宽度计算，如果你想进一步了解这方面的知识，请移步阅读《[现代 CSS 中的相对单位](https://juejin.cn/book/7223230325122400288/section/7249357892611440700)》。另外，`inset()` 是 `clip-path` 属性中的一个值，你可以通过阅读《[CSS 的 Clipping 和 Masking](https://juejin.cn/book/7223230325122400288/section/7259668885224456252)》更进一步的了解它。

  


## 揭示式效果

  


应用上面示例所提到的技巧，我们可以创建一个揭示式悬念效果。这次我们使用纯色（你也可以像之前一样使用半透明颜色）来遮挡图片，并给图片添加圆角，然后在悬停时实现一个圆形提示效果：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/021ef46b72a64aa49180b6e03e61ca17~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1336&h=642&s=1200169&e=gif&f=141&b=fffbfa)

  


> Demo 地址：https://codepen.io/airen/full/xbbBJoe

  


很酷吧！这个悬停效果依赖两个 CSS 技巧。首先使用 `outline` 属性来创建覆盖层，然后通过动画改变 `outline-offset` 来使其消失：

  


```CSS
img {
    --c: #f0aa;
    --o: 40vmin;
    --r: 1em;
    width: 100%;
    aspect-ratio: 1;
    cursor: pointer;
    transition: 0.6s;
    outline: var(--o) solid var(--c);
    outline-offset: calc(-.5 * var(--o));
    border-radius: 50%;
    
    &:hover {
        --o: 0;
    }
}
```

  


注意，示例中使用了 `border-radius: 50%` 来创建圆形图片，这样使得轮廓（`outline`）也变成圆形，并在悬停时产生圆形动画效果:

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/705e396806414688895f753deb6ddd4c~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1334&h=672&s=3776745&e=gif&f=111&b=fff8f6)

  


不难看出，由于轮廓（`outline`）太大了，整个效果看起来有点难看，但别担心，当我们引入第二个技巧——`clip-path` 时，这个问题就会解决。使用 `clip-path` 可以隐藏不需要的轮廓部分，同时定义图片的圆角半径。别忘了，核心思想是让图片有一个特定的半径，而不是简单的变成圆形图片：

  


```CSS
 img {
    --c: #f0a;
    --o: 50vmin;
    --r: 1em;
    width: 100%;
    aspect-ratio: 1;
    cursor: pointer;
    transition: 0.6s;

    outline: var(--o) solid var(--c);
    outline-offset: calc(-0.5 * var(--o));
    clip-path: inset(calc(50% - 50% * cos(45deg)) round var(--r));
    border-radius: 50%;
    
    &:nth-child(2) {
        --r: 3em;
    }
    
    &:nth-child(3) {
        --r: 50%;
    }

    &:hover {
        --o: 0vmin;
    }
}
```

  


关键是 `inset()` 函数的值，其中 `round var(--r)` 比较好理解，大家好奇的是，`calc(50% - 50% * cos(45deg))` 是怎么来的？下图可以很好的解释它是怎么来的：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/c051a70c574c409a8e5749f4fa308b27~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=5203&h=3643&s=3447707&e=jpg&b=fbfbfb)

  


我们从一个未使用 `clip-path` 的圆角图片开始（虚线框表示图片的实际区域）。接着，我们应用 `clip-path` 来移除圆角部分。换句话说，我们从图片中提取一个内接正方形。最终，我们只会看到红色矩形的部分。这个红色矩形是通过 `inset()` 函数裁剪出来的，该函数接受 `top` 、`right` 、`bottom` 和 `left` 四个值（它的使用有点类似于 `margin` 或 `padding`）。在这个示例中，距离图像边缘的距离都是相同的，所以我们只要给 `inset()` 设置一个值就可以。这个值我们可以根据 **[三角函数](https://juejin.cn/book/7223230325122400288/section/7242216512176521277)** 计算出来，如上图中步骤 2️⃣ 所示。

  


最后，我们使用 `round` 关键词来为裁剪区域添加圆角，从而得到我们想要的圆角效果。

  


使用 `clip-path` 还能隐藏图片外部的轮廓线（`outline`），这样整体效果就非常完美。唯一的缺点是，这种方法会隐藏掉图片的一部分内容，因此可能并不适用于所有图片。

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/23185390c59c4d6abeb32e72e3903b72~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1334&h=520&s=760709&e=gif&f=101&b=fff9f8)

  


> Demo 地址：https://codepen.io/airen/full/xbbBJoe

  


这里所展示的仅是 CSS 创建揭示式悬停动画效果的其中一种技巧，我将在下一节课中，会专门与大家一起探讨其他创建揭示式悬停动画效果的 CSS 技巧！

  


## 图像边框效果

  


接下来，我们来为图像添加精美的边框装饰，而且我们还会为这些边框添加动画，打造奇妙的悬停效果。先来看第一个效果，当鼠标悬停在图像上时，一个正方形边框从图像后面滑出，覆盖在图像上。

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/3cfdba6dda1e4b469d17e35d78bca4a4~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1330&h=516&s=9942339&e=gif&f=147&b=f9f0ed)

  


> Demo 地址：https://codepen.io/airen/full/NPPJELp

  


以下是实现这个效果的 CSS 代码：

  


```CSS
img {
    --c: #f0a;   /* 边框颜色 */
    --b: 10px;   /* 边框厚度 */
    --o: 20px;   /* 偏移量 */
    --gap: 5px;  /* 间距 */

    width: 100%;
    aspect-ratio: 1;
    object-fit: cover;
    object-position: center;

    --g: var(--c) var(--b), #0000 0 calc(100% - var(--b)), var(--c) 0;
    --_o: calc(3 * var(--o));
    padding: 
        calc(var(--gap) + var(--b)) 
        calc(var(--gap) + var(--b) + var(--o))
        calc(var(--gap) + var(--b) + var(--o)) 
        calc(var(--gap) + var(--b));
    background-image: 
        linear-gradient(var(--g)),
        linear-gradient(90deg, var(--g));
    background-size: calc(100% - var(--_o)) calc(100% - var(--_o));
    background-repeat: no-repeat;
    transition: 0.5s;
    cursor: pointer;


    &:nth-child(1) {
        background-position: var(--_o) var(--_o);
        &:hover {
            background-position: 0% 0%;
        }
    }

    &:nth-child(2) {
        padding: 
            calc(var(--gap) + var(--b)) 
            calc(var(--gap) + var(--b))
            calc(var(--gap) + var(--b) + var(--o))
            calc(var(--gap) + var(--b) + var(--o));
        background-position: calc(100% - var(--_o)) var(--_o);

        &:hover {
            background-position: 100% 0%;
        }
    }

    &:nth-child(3) {
        padding: 
            calc(var(--gap) + var(--b) + var(--o))
            calc(var(--gap) + var(--b) + var(--o)) 
            calc(var(--gap) + var(--b))
            calc(var(--gap) + var(--b));
        background-position: var(--_o) calc(100% - var(--_o));
        
        &:hover {
            background-position: 0% 100%;
        }
    }

    &:nth-child(4) {
        padding: 
            calc(var(--gap) + var(--b) + var(--o))
            calc(var(--gap) + var(--b)) 
            calc(var(--gap) + var(--b))
            calc(var(--gap) + var(--b) + var(--o));
        background-position: calc(100% -var(--_o)) calc(100% - var(--_o));

        &:hover {
            background-position: 100% 100%;
        }
    }

    &:hover {
        background-size: calc(100% - var(--o)) calc(100% - var(--o));
    }
}
```

  


这段代码看起来可能有些复杂，但不用太担心，我将会一步步向你解释它的工作原理。我们以第一个效果为例。首先我们定义了几个 CSS 变量，用来控制边框的颜色，厚度、偏移量以及悬停时图片与边框之间的间距：

  


```CSS
img {
    --c: #f0a; /* 边框颜色 */
    --b: 10px; /* 边框厚度 */
    --o: 20px; /* 边框偏移量 */
    --gap: 5px;/* 悬停时图片与边框之间的间距 */
}
```

  


接着给图处设置一个内距（`padding`），使图片内容向里收缩，留一些空间：

  


```CSS
img {
    --c: #f0a; /* 边框颜色 */
    --b: 10px; /* 边框厚度 */
    --o: 20px; /* 边框偏移量 */
    --gap: 5px;/* 悬停时图片与边框之间的间距 */
    
    padding: 
        calc(var(--gap) + var(--b))             /* top */
        calc(var(--gap) + var(--b) + var(--o))  /* right */
        calc(var(--gap) + var(--b) + var(--o))  /* bottom */               
        calc(var(--gap) + var(--b));            /* left */  
}
```

  


注意，`padding` 的值是通过 `--o` 、`--b` 和 `--gap` 计算出来的。现在看到的效果如下图所示，图片与虚线之间的距离就是 `padding` 所产生的：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/4c2da1c6595b45098d71a43da55f17e6~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2286&h=988&s=189983&e=png&b=fffbfa)

  


然后通过两个线性渐变来绘制边框：

  


```CSS
img {
    --c: #f0a; /* 边框颜色 */
    --b: 10px; /* 边框厚度 */
    --o: 20px; /* 边框偏移量 */
    --gap: 5px;/* 悬停时图片与边框之间的间距 */
    
    padding: 
        calc(var(--gap) + var(--b))             /* top */
        calc(var(--gap) + var(--b) + var(--o))  /* right */
        calc(var(--gap) + var(--b) + var(--o))  /* bottom */               
        calc(var(--gap) + var(--b));            /* left */  
        
    --g: var(--c) var(--b), #0000 0 calc(100% - var(--b)), var(--c) 0;   
    background-image: 
        linear-gradient(var(--g)),        /* 绘制上下边框 */
        linear-gradient(90deg, var(--g)); /* 绘制左右边框 */
    background-repeat: no-repeat;     
}
```

  


其中第一个 `linear-gradient` 绘制上下边框，第二个 `linear-gradient` 绘制左右边框，两者合在一起就组合成一个矩形边框：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/50ed4d548ab743be8d65f4f0de3fbd81~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2286&h=988&s=927776&e=png&b=fcf4f2)

  


接着我们通过 `background-size` 来调整渐变图片的尺寸：

  


```CSS
img {
    --c: #f0a; /* 边框颜色 */
    --b: 10px; /* 边框厚度 */
    --o: 20px; /* 边框偏移量 */
    --gap: 5px;/* 悬停时图片与边框之间的间距 */
    
    padding: 
        calc(var(--gap) + var(--b))             /* top */
        calc(var(--gap) + var(--b) + var(--o))  /* right */
```

悬停时，你会看到圆的四个象限从红色区域移动到透明区域。最初边框被隐藏在红色圆圈之下，当它们移动到中心时，我们就能看到它们了。这就是遮罩的作用：**红色区域是“不可见”的部分，渐变会从那里显现出来**。

  


现在，我们把调试 `mask` 的代码替换出来，并应用到 `mask` 属性上：

  


```CSS
img {
    outline: 1px dashed;
    border-radius: 50%;
    
    padding: calc(var(--b) + var(--gap));
    border: var(--o) solid #faca; /* 这里暂时设置了一个半透明颜色 */
    
    --g: #0000 calc(99% - var(--b)), var(--c) calc(100% - var(--b)) 99%, #0000;
    --s: var(--o); 
    --r: 100% 100% at;
    --m: radial-gradient(
        closest-side at 50% 50%,
        #000 calc(100% - var(--o)),
        #0000 calc(100% - var(--b))
    );
    
    background: 
        /* 右下角 */
        radial-gradient(var(--r) 0 0, var(--g)) calc(100% + var(--s)) calc(100% + var(--s)) / 50% 50% no-repeat,
        /* 左下角 */
        radial-gradient(var(--r) 100% 0, var(--g)) calc(0% - var(--s)) calc(100% + var(--s)) / 50% 50% no-repeat,
        /* 右上角 */
        radial-gradient(var(--r) 0 100%, var(--g)) calc(100% + var(--s)) calc(0% - var(--s)) / 50% 50% no-repeat,
        /* 左上角 */
        radial-gradient(var(--r) 100% 100%, var(--g)) calc(0% - var(--s)) calc(0% - var(--s)) / 50% 50% no-repeat;
    mask: var(--m);    
    transition: .5s;
    
    &:hover {
        --s: 0%; /* 记得带上单位，否则动画会失效 */
    }    
}
```

我们将结合 `outline` 和 `mask` 来实现这个效果：

  


```CSS
img {
    --c: #f0a;
    --b: 10px;
    --gap: 20px;
    --s: 50px;
    --w: 300px;
    
    width: var(--w);
    aspect-ratio: 1;
    object-fit: cover;
    object-position: center;
    transition: 0.5s;
    cursor: pointer;

    padding: calc(var(--gap) + var(--b));
    outline: var(--b) solid var(--c);
    outline-offset: calc(-1 * var(--b));
    mask: 
        conic-gradient(at var(--s) var(--s), #0000 75%, #000 0) 0 0 / calc(100% - var(--s)) calc(100% - var(--s)),
        linear-gradient(#000 0 0) content-box;
        
    &:hover {
        outline-offset: calc(-1 * var(--gap));
    }
}
```

  


原理也之前的案例是相似的，通过 `padding` 将图片内容向内收缩，并使用 `outline` 绘制轮廓，然后通过 `outline-offset` 将轮廓向内收缩。同时在悬停时动画改变 `outline-offset` 来达到下面这样的效果：

## 分割图片效果

  


这部分着重介绍如何使用 CSS 遮罩（`mask`）来创建更奇妙的悬停效果——分割图像！CSS 遮罩允许我们使用图片或渐变来隐藏或裁剪元素的某些部分。我们还可以对不同的遮罩进行动画处理，从而实现更奇妙的悬停效果。

  


> 温馨提示，如果你对 CSS 遮罩（`mask`）不太了解的话，建议你移步阅读《[CSS 的 Clipping 和 Masking](https://juejin.cn/book/7223230325122400288/section/7259668885224456252)》！

  


让我们从第一个效果开始——悬停时，将两个半圆组合成一个完整的圆形：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/d0dae19bfa584bc599234d9e1d539c2a~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1334&h=522&s=1493170&e=gif&f=93&b=fcf6f5)

  


> Demo 地址：https://codepen.io/airen/full/GggLJBw

  


CSS 代码如下：

  


```CSS
img {
    --w: 300px;
    width: var(--w);
    aspect-ratio: 1;
    object-fit: cover;
    object-position: center;
    transition: 0.5s;
    cursor: pointer;

    --m: 
        radial-gradient(circle farthest-side at right, #000 99%, #0000) 0 100%/46% 92% no-repeat,
        radial-gradient(circle farthest-side at left, #000 99%, #0000) 100% 0/46% 92% no-repeat;
    mask: var(--m);
    
    &:hover {
        mask-position: 7.5% 50%, 92.5% 50%;
    }
}
```

  


在这里通过两个径向渐变定义了 `mask` 属性，并在悬停时移动它们（调整 `mask-position`）。你会注意到每个遮罩层的高度小于 `100%` （这里设置的是 `92%`），宽度小于 `50%` （这里设置的是 `46%`），这使得它们初始彼此相距较远。

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/231b0ab0dca4460e9bdbfac5a9a883ba~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1136&h=842&s=912522&e=gif&f=99&b=fffbfb)

  


> Demo 地址：https://codepen.io/airen/full/GggLJwP

  


如上图所示，黑色区域是可见，透明区域不可见。悬停时，两个半圆会移向中心，形成一个完整的圆形。利用上述技术，只需要更改渐变的配置，就能实现许多不同的悬停效果：
