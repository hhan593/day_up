## 创建渐变阴影

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/a53581229e814625af2f4bf7dc109a8d~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1280&h=720&s=45226&e=jpg&b=fdfcfc)

  


从其名称上不能理解，“渐变阴影”包含两个特性，其一是阴影，其二是渐变。这两者对于任何一位 Web 开发者来说，都不是事。我想你能熟练地使用 CSS 相关特性创建出自己需要的阴影和渐变效果：

  


-   使用 `text-shadow` 、`box-shadow` 和 `filter` 的 `drop-shadow()` 创建出符合需求的阴影
-   使用 `linear-gradient()` 、`repeating-linear-gradient()` 、`radial-gradient()` 、`repeating-radial-gradient()` 、`conic-gradient()` 和 `repeating-conic-gradient()` 创建出想要的渐变效果

  


一旦要创建渐变阴影，对于部分 Web 开发者而言还是非常棘手的，因为 CSS 的 阴影特性只接受 `<color>` 值，并不能直接将 `<image>` 值（CSS 渐变属于 `<image>`）作为阴影的颜色。

  


换句话说，如果你有一个渐变阴影的需求，那么只能通过各种 CSS 技巧来模拟一种近似渐变的阴影。在这里，我们以元素框的渐变阴影为例：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/b3636b47f8234b34b4f36bc4724de85e~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=892&s=280972&e=jpg&b=222222)

  


我们先从最常见的情景开始。假设你有一个自带背景的元素 `.box` ：

  


```HTML
<div class="box">Gradient Shadow</div>
```

  


```CSS
.box {
    --w: clamp(23ch, 50vw, 46ch);
    --c: #ff90ca;
    --a: 180deg;
    --g: conic-gradient(from var(--a) in oklch longer hue, var(--c) 0 0);
    
    width: var(--w);
    aspect-ratio: 16 / 9;
    display: grid;
    place-content: center;
    font-size: clamp(1.125rem, 3vw + .875rem, 1.375rem);
    
    background: var(--g);
    color: color-contrast(var(--c) vs #fff, #000 to AA);
}
```

  


上面代码没有什么复杂之处，我们给元素设置了一个渐变背景，并且文本颜色会根据 `--c` 的值从 `#fff` 和 `#000` 中选择能达到 `AA` 级别的一个颜色。

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/bd0ffc26a6be458ba5e05ae93fa319f0~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=892&s=249088&e=jpg&b=050505)

  


接下来，我们要为它添加渐变阴影。注意，在这个示例中，我们不需要考虑透明度的问题，因为元素自身填充了一个渐变背景。

  


基于这种情景，解决方案是使用一个定义了渐变背景的伪元素 `::before` 或 `::after` ，并且将它放置在元素的底部，同时使用 `filter` 中的 `blur()` 函数，使带有渐变的伪元素变得模糊，从而模拟出渐变阴影效果：

  


```CSS
.box {
    --s: round(.25em, 1px); /* 控制阴影的扩展半径, 类似 box-shadow 中的 spread */
    --r: round(1em, 1px); /* 控制模糊半径, 类似 box-shadow 中的 blur  */
    --xOffset: 0; /* X轴偏移量，类似 box-shadow 中的 x-offset */
    --yOffset: 0; /* Y轴偏移量，类似 box-shadow 中的 y-offset */
    --g: conic-gradient(from var(--a) in oklch longer hue, var(--c) 0 0); /* 控制渐变，类似 box-shadow 中的 color */
    position: relative;
    
    &::after {
        content: "";
        position: absolute;
        inset: calc(-1 * var(--s)); /* 需要一个负值 */
        z-index: -1; /* 放置在元素的底部 */
        background: var(--g); /* 定义你需要的渐变 */
        filter: blur(var(--r));
        translate: var(--xOffset) var(--yOffset); /* 控制阴影偏移位置 */
    }
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/49e7bae3e9884526afc99bc4245ad53c~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1003&s=336573&e=jpg&b=060606)

  


正如你所看到的，`.box` 的伪元素 `::after` 承载了渐变阴影的一切。在这里，我们使用 CSS 的自定义属性来模拟 `box-shadow` 的主要参数（除关键词 `inset` ）：

![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/49d086ed520544e2860ce42c077621fd~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1620&h=850&s=263862&e=jpg&b=1d2637)

  


-   `--xOffset` 对应 `box-shadow` 的 `x-offset` ，即 `x` 轴的偏移。它将作为 `translate` 的 `x` 轴的值，也可以作为 `translate()` 函数的 `x` 轴的值
-   `--yOffset` 对应 `box-shadow` 的 `y-offset` ，即 `y` 轴的偏移。它将作为 `translate` 的 `y` 轴的值，也可以作为 `translate()` 函数的 `y` 轴的值
-   `--r` 对应 `box-shadow` 的 `blur` ，控制阴影的模糊半径。它将作为 `filter` 的 `blur()` 函数的值
-   `--s` 对应 `box-shadow` 的 `spread` ，控制阴影扩展半径。它将作为 `inset` 属性的值，但必须是一个负值，可以使用 `calc(-1 * var(--s))` 将其转换为负值。如果是正值，阴影就不是扩展了，而是收缩，它将不可见。因为整个伪元素位于元素的底部
-   `--g` 对应 `box-shadow` 的 `color` ，控制渐变效果，你可以根据需要来选择 CSS 渐变中的任一渐变函数

  


需要知道的是，`inset` 必须是 `position` 属性的值不是 `static` 才能生效，通常采用 `absolute`。另外，如果 `.box` 元素采用的是 CSS Grid 布局，[其伪元素 `::after` 也可以通过 `grid-area: 1 / 1 / -1 / -1` 使其填满 `.box` 元素](https://juejin.cn/book/7161370789680250917/section/7161623932439625758)，但伪元素的宽高需要设置为 `calc(100% + 2 * var(--s))` ，不能再由 `inset` 来控制阴影扩展距离。这是 CSS Grid 布局中的一个小技巧，我们后面有一节课会专门与大家探讨这方面的技术！

  


另外，阴影位置的偏移是通过 [CSS 变换中的 `translate`](https://juejin.cn/book/7223230325122400288/section/7259668493158023205) 来控制的，[你也可以使用 `transform` 的 `translate()` 函数](https://juejin.cn/book/7288940354408022074/section/7295240572736897064#heading-4)，它们效果是等效。而“阴影”的模糊度是由 [CSS 滤镜（`filter`）的 `blur()` 函数](https://juejin.cn/book/7223230325122400288/section/7259669043622690853#heading-3)控制。

  


现在，你只需要调整自定义属性的值，就可以获得不同的渐变阴影效果：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/97825cd230564f09be188ea0f097dcba~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1052&h=674&s=18496762&e=gif&f=231&b=050505)

  


> Demo 地址：https://codepen.io/airen/full/OJKLPJm

  


这个方案简单且易用，但仍然有一些与 `z-index: -1` 声明相关的缺点。有时候，可能因为其他的 CSS 样式使 `z-index: -1` 并没有如你预期的那样，将伪元素 `::after` 放置在 `.box` 元素的底部，从而影响了最终的结果。万一你碰到类似的问题，首先需要检查的是 `z-index` 为什么失效的原因。如果你不太知道 `z-index` 失效的原因，那么建议你仔细阅读一下《[`z-index` 失效与修复](https://juejin.cn/book/7199571709102391328/section/7204402059225858052)》一文，你将会找到大部分修复方案。例如，使用 3D 变换来替代 `z-index: -1` 就是一个不错的修复方案：

  


```CSS
.box {
    /* CSS ... */
    
    position: relative;
    transform-style: preserve-3d;
    
    &::after {
        /* z-index: -1; */
        translate: var(--xOffset) var(--yOffset) -1px; /* 或者 transform: translate(var(--xOffset), var(--yOffset), -1px) */
    }
}
```

  


上面的代码不再使用 `z-index: -1` ，而是沿着 `z` 轴进行负方向的平移。注意，别忘了在 `.box` 元素上使用 `transform-style: preserve-3d`，否则[3D变换不会生效](https://juejin.cn/book/7288940354408022074/section/7295240572736897064#heading-8)。

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/d93188001e454d1989468afa9e2a33e9~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1003&s=336573&e=jpg&b=060606)

  


> Demo 地址：https://codepen.io/airen/full/ExqYVgR

  


这个解决方案在大多数需要渐变阴影的情况下可能已经足够好，只要你不需要支持透明背景即可。但我们在实际开发的过程中，`.box` 元素有可能是完全透明的或部分透明的。例如：

  


```CSS
.box {
    --w: clamp(23ch, 50vw, 46ch);
    --c: #ff90ca;
    --a: 180deg;
    --g: conic-gradient(from var(--a) in oklch longer hue, var(--c) 0 0);
    --s: round(.25em, 1px); /* 控制阴影的扩展半径, 类似 box-shadow 中的 spread */
    --r: round(1em, 1px); /* 控制模糊半径, 类似 box-shadow 中的 blur  */
    --xOffset: 0; /* X轴偏移量，类似 box-shadow 中的 x-offset */
    --yOffset: 0; /* Y轴偏移量，类似 box-shadow 中的 y-offset */
    
    
    width: var(--w);
    aspect-ratio: 16 / 9;
    display: grid;
    place-content: center;
    place-self: center;
    font-size: clamp(1.125rem, 3vw + .875rem, 1.375rem);
    background: repeating-linear-gradient(45deg, #ccc 0 30px, #0000 30px 60px);
    color: color-contrast(var(--c) vs #fff, #000 to AA);
    
    position: relative;
    transform-style: preserve-3d;
    
    &::after {
        content: "";
        position: absolute;
        inset: calc(-1 * var(--s)); /* 需要一个负值 */
        /* z-index: -1;  放置在元素的底部 */
        background: var(--g); /* 定义你需要的渐变 */
        filter: blur(var(--r));
        translate: var(--xOffset) var(--yOffset) -1px;
        transition: all .2s;
    }
 }
```

  


`.box` 元素是使用 `repeating-linear-gradient()` 创建的斜纹条纹理背景，它从 `#ccc` 过渡到完全透明 `#0000` ，它将会漏出位于底部的伪元素 `::after` 。你将看到像下图这样的一个效果：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/4053a373f95f4bb1b92b019e99e69249~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1003&s=497724&e=jpg&b=060606)

  


> Demo 地址：https://codepen.io/airen/full/OJKLyxN

  


这并不是我们所期望的效果。

  


接下来，我们就要开始新的挑战，即如何给透明元素设置渐变阴影效果。我们从上面这个示例开始，并且从一个偏移量和扩展距离都为 `0` 的阴影开始：

  


```CSS
.box {
    --s: 0;
    --xOffset: 0;
    --yOffset: 0;
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/d4c91d4534f347b3bbe40616797ec4a2~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1003&s=490793&e=jpg&b=060606)

  


接下来，我们要做的是找到一种方法来裁剪掉或隐藏元素区域内的所有内容，同时保留外面的部分。如下图所示，我们需要将下图中虚线内的区域隐藏起来：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/6688d9fa934f402294653f7e8749eccc~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1003&s=621878&e=jpg&b=060606)

  


注意，我们要裁剪或隐藏的是 `::after` 中的区域，而不是主元素 `.box` 的区域！

  


CSS 提供了两种裁剪区域的方法，即 [CSS 的剪切 `clip-path` 和遮罩 `mask`](https://juejin.cn/book/7223230325122400288/section/7259668885224456252) ，但在这里将使用 `clip-path` 来实现。你可能会疑惑，`clip-path` 如何能够在元素内部进行裁剪呢？不知道你是否还有印象，在小册之前的课程《[现代 CSS 创建形状指南](https://juejin.cn/book/7395119173656903715/section/7395172754053890063)》中，介绍 `clip-path` 属性时，提到过一个小技巧，即“[反转形状](https://juejin.cn/book/7395119173656903715/section/7395172754053890063#heading-9)”（也被称为“切割形状”），视觉上看上去像是一个镂空的效果：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/a8b590bad91043a79e6d679a3178146e~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=645&s=196693&e=jpg&b=ffffff)

  


实际上，`clip-path` 本身无法直接做到这一点，但我们可以通过“反转形状”的技巧（一个特殊的多边形）来实现。我用下图来向大家说明这个特殊的多边形，便于大家易于理解：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/81de91888ac84e48ae3c61ad1af30d1f~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1596&s=678829&e=jpg&b=050505)

  


如上图所示，`B` 是一个很大的值，这样我们就可以获得 `clip-path` 的 `polygon()` 函数中的点坐标：

  


```CSS
.box::after {
    clip-path: polygon(
        -100vmax 100vmax,
        100vmax  -100vmax,
        100vmax 100vmax,
        -100vmax 100vmax,
        -100vmax -100vmax,
        0% 0%,
        0 100%,
        100% 100%,
        100% 0,
        0% 0%
    )
}
```

  


我们可以把 `polygon()` 中的坐标分外部和内部两个部分：

  


```CSS
.box::after {
    --shape-outer: -100vmax 100vmax,100vmax  -100vmax,100vmax 100vmax,-100vmax 100vmax, -100vmax -100vmax;
    --shape-inner: 0% 0%, 0 100%, 100% 100%,100% 0,0% 0%;
    
    clip-path: polygon(var(--shape-outer), var(--shape-inner));
}
```

  


简单的解释一下 `clip-path` 的 `polygon()` 函数中的坐标。上图中青绿色区域（最大的那个矩形）是应用 `clip-path` 后可见的部分。实际上，我们只会在该区域内看到渐变阴影。正如你所见，我们定义了四个点 ：`(-B,-B)` 、`(-B,B)` 、`(B,B)` 和 `(-B,B)` ，其中 `B` 是一个很大的值，例如 `100vmax` ，你也可以使用 `calc(infinity * 1px)` 来表示（它表示的是一个无穷大的值）。这样做的目的是确保我们有足够的空间来展示阴影，因为我们并不知道阴影的扩展半径有多大。除此之外，还有四个点是伪元素的顶点：`(0%, 0%)` 、`(0% 100%)` 、`(100%,100%)` 和 `(100% 0%)` 。

  


你可能看到上图的箭头方向了，它们表示定义多边形的路径。我们从 `(-B,-B)` 开始，直到到达 `(0%,0%)` 。注意，内外裁剪的路径方向刚好相反。其实，也可以使用相同的方向来裁剪多边形，但需要在 `polygon()` 函数中应用 `fill-rule` 规则，它可以接受 `nonzero` 和 `evenodd` 值，用于定义裁剪形状时使用的填充规则。要是你对该规则的计算原理不太了解，可以按我们这个示例，内外裁剪方向相反。

  


另外，`clip-path` 的 `polygon()` 函数裁剪需要的多边形，总共需要十个点，而不是八个点，因为有两个点（`(-B,-B)` 和 `(0%,0%)`）在路径中重复了两次。重复 `(-B,-B)` 点是我们要该点过渡到 `(0%,0%)` 点，重复 `(0%,0%)` 点，相当于我们要关闭最终的裁剪路径。下图可以帮助你更好的理解这一点：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/771ae917e88f4112b773b174cc809f5d~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=645&s=255030&e=jpg&b=ffffff)

  


现在，我们把定义好的 `clip-path` 应用到 `.box` 的伪元素 `::after` 上，因为渐变阴影是通过它来制作的：

  


```CSS
.box {
    --w: clamp(23ch, 50vw, 46ch);
    --c: #ff90ca;
    --a: 180deg;
    --g: conic-gradient(from var(--a) in oklch longer hue, var(--c) 0 0);
    --s: 0px; /* 控制阴影的扩展半径, 类似 box-shadow 中的 spread */
    --r: round(1em, 1px); /* 控制模糊半径, 类似 box-shadow 中的 blur  */
    --xOffset: 0; /* X轴偏移量，类似 box-shadow 中的 x-offset */
    --yOffset: 0; /* Y轴偏移量，类似 box-shadow 中的 y-offset */
    
    --b: calc(infinity * 1px); /* B，一个无穷大的值 */
    --_b: calc(infinity * -1px); /* -B, 这里负值表示坐标方向*/
    --shape-outer: var(--_b) var(--_b), var(--b) var(--_b), var(--b) var(--b), var(--_b) var(--b), var(--_b) var(--_b); /* 外坐标点 */
    --shape-inner: 0% 0%, 0% 100%, 100% 100%, 100% 0%, 0% 0%;
    
    /* 省略基本样式 */
    position: relative;
    transform-style: preserve-3d;
    
    &::after {
        content: "";
        position: absolute;
        inset: calc(-1 * var(--s)); /* 需要一个负值 */
        /* z-index: -1;  放置在元素的底部 */
        background: var(--g); /* 定义你需要的渐变 */
        filter: blur(var(--r));
        translate: var(--xOffset) var(--yOffset) -1px; 
        clip-path: polygon(var(--shape-outer), var(--shape-inner));
    }
}
```

  


这个时候，你会看到位于 `.box` 底部的背景（它的父容器的背景），而不是绘制渐变阴影的 `::after` ：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/a387aba0af3647fba6c8b6639b53f711~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1003&s=361481&e=jpg&b=060606)

  


> Demo 地址：https://codepen.io/airen/full/eYqOpKK

  


效果看上去似乎还不错，对吧！但别忘了，上面这个示例我们把阴影的扩展半径 `--s` ，偏移位置 `--xOffset` 和 `--yOffset` 都设置为 `0` 。这意味着，上面这个效果是一个特殊情况，阴影的偏移量和扩展距离都等于 `0` 。

  


现在，让我们定义扩展距离，例如 `--s` 的值不是 `0` ，而是 `0.5em` 。这个时候，你看到的效果如下图所示：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/15392f114d4641959fb6dedef0e97b7d~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1003&s=517680&e=jpg&b=060606)

  


很明显，现在伪元素比主元素大，因为我们是使用负的 `inset` 来实现阴影扩展距离的。这意味着，我们要重新调整 `--shape-inner` 的坐标点。请记住，我们始终需要裁剪掉主元素内部的部分，因此 `--shape-inner` 的坐标点需要重新调整，我将所有的 `0%` 值增加了 `--s` ，即 `calc(0% + var(--s))`，并将 `100%` 的值减少了 `--s` ，即 `calc(100% - var(--s))` ：

  


```CSS
.box {
    --shape-inner:
        calc(0% + var(--s)) calc(0% + var(--s)),
        calc(0% + var(--s)) calc(100% - var(--s)),
        calc(100% - var(--s)) calc(100% - var(--s)),
        calc(100% - var(--s)) calc(0% + var(--s)),
        calc(0% + var(--s)) calc(0% + var(--s));
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/dd0603267fe7481dbb97811370951615~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1003&s=536373&e=jpg&b=060606)

  


> Demo 地址：https://codepen.io/airen/full/mdNbWEO

  


最后，我们来开启阴影偏移的值，即 `--xOffset` 和 `--yOffset` 不等于 `0` ：

  


```CSS
.box {
    --offset: 1em;
    --xOffset: var(--offset);
    --yOffset: calc(-1 * var(--offset));
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/f3452969ed2a448886e12abea81ead32~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=875&s=529220&e=jpg&b=060606)

  


注意，渐变阴影的偏移是通过 `translate` 平移伪元素 `::after` 来实现的，所以你看到的阴影会错位。我们需要再次调整 `--shape-inner` 的坐标，并将各个点沿相反方向移动：

  


```CSS
.box {
    --shape-inner:
        calc(0% + var(--s) - var(--xOffset)) calc(0% + var(--s) - var(--yOffset)),
        calc(0% + var(--s) - var(--xOffset)) calc(100% - var(--s) - var(--yOffset)),
        calc(100% - var(--s) - var(--xOffset)) calc(100% - var(--s) - var(--yOffset)),
        calc(100% - var(--s) - var(--xOffset)) calc(0% + var(--s) - var(--yOffset)),
        calc(0% + var(--s) - var(--xOffset)) calc(0% + var(--s) - var(--yOffset));
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/8d9f734f128c4ddb8e17ae141cdc8a01~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1059&s=550839&e=jpg&b=060606)

  


> Demo 地址：https://codepen.io/airen/full/QWeLpdr

  


现在你看到的效果几乎是完美的。你可能又会疑惑了，怎么还是“几乎”呢？别忘了，伪元素 `::after` 的参考框是内边距框（`padding-box`）。也就是说，如果你给主元素添加边框，就会出现重叠。

  


```CSS
.box {
    border: 1em solid;
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/723e8c18209e4972b31af25b3b1c4cc8~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1299&s=687480&e=jpg&b=060606)

  


在这种情况之下，我们需要调整 `inset` 属性的值：

  


```CSS
.box {
    border: 1em solid;
    
    &::after {
        inset: calc(-1 * var(--s) - 1em);
    }
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/e51cea10ee104d9788ec74640b9ee06c~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1299&s=800095&e=jpg&b=080808)

  


> Demo 地址：https://codepen.io/airen/full/BaXBWmJ

  


我认为这是更适合的做法，扩展距离将更加准确，因为它是从边框框（`border-box`）开始计算的，而不是内边框（`padding-box`）。不过，你需要根据主元素的边框厚度（`border-width`）来调整 `inset` 属性的值。

  


你可能已经发现了，在这之前我们所展示的案例都没有圆角。试问一下，如果主元素 `.box-shadow` 设置了 `border-radius` ，渐变阴影又将会变成什么样呢？

  


了解 `box-shadow` 的开发者应该知道，它定义的阴影能自动匹配元素的 `border-radius` 。例如下面这个示例：

  


```CSS
.box {
    box-shadow:
        inset 0 0 50px #fff,      
        inset 20px 0 80px #f0f,   
        inset -20px 0 80px #0ff,  
        inset 20px 0 300px #f0f,  
        inset -20px 0 300px #0ff, 
        0 0 50px #fff,            
        -10px 0 80px #f0f,        
        10px 0 80px #0ff;         
    border-radius: var(--radius, 50%);
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/0be5f7166d8b4eb2ba8c744d0ed7380e~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1210&h=774&s=15937096&e=gif&f=161&b=030303)

  


> Demo 地址：https://codepen.io/airen/full/GRVKEmR

  


正如你所看到的，`box-shadow` 创建的阴影会随带有圆角（`border-radius`）的主元素变化，阴影在视觉上也具有圆角效果。这是 [CSS 的 `border-radius`](https://juejin.cn/book/7199571709102391328/section/7199845563389444099) 和 [`box-shadow` 的基本特性](https://juejin.cn/book/7199571709102391328/section/7199844993455325216#heading-4)。

  


现在，我们创建的渐变阴影并不是直接使用 `box-shadow` 创建的，而是通过伪元素 `::after` 来模拟的。因此，我们有必要考虑主元素 `.box` 设置 `border-radius` 时阴影的效果。

  


```CSS
.box {
    --radius: round(.125em, 1px);
    border-radius: var(--radius);
}
```

  


你可能会认为，这个应该很简单，只需要在 `::after` 上继承主元素的 `border-radius` 属性就可以完成：

  


```CSS
.box {
    --radius: round(.125em, 1px);
    border-radius: var(--radius);
    
    &::after {
        border-radius: inherit;
    }
}
```

  


不得不说，这是一个好主意。主元素为非透明时，这样做可以达到预期的效果（下图左侧）；但主元素为透明时，情况就不同了，伪元素内部角是直角，并露出底部元素（下图右侧）：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/3fe629ba391e46b388aff5898bbeaad1~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=842&s=424159&e=jpg)

  


> Demo 地址：https://codepen.io/airen/full/wvVwepW

  


这意味着，直接在伪元素 `::after` 继承主元素的 `border-radius` 属性的值，并不能解决主元素为透明时的阴影圆角效果。这是因为，`clip-path` 的 `polygon()` 函数到目前为止无法处理曲线。这也是我们无法切掉主元素内部区域的主要原因（上图右侧四个顶点露出的白色部分）。

  


庆幸的是，[我们可以使用 CSS 的遮罩 `mask` 来解决](https://juejin.cn/book/7223230325122400288/section/7259668885224456252#heading-9)。只不过，需要在 `.box` 元素中嵌套一个 HTML 元素：

  


```HTML
<div class="box">
    Gradient Shadow
    <span class="mask"></span>
</div>    
```

  


如果你害怕 `span.mask` 被其他 CSS 样式污染，你也可以使用一个自定义的 HTML 标签元素，例如：

  


```HTML
<div class="box">
    Gradient Shadow
    <shadow />
</div>    
```

  


不用担心 `<shadow />` 无效，现代 Web 浏览器可以很好的解析它。

  


接下来，我们把重点放在 CSS 上。首先，对 `<shadow />` 设置为绝对定位，并故意创建一个溢出：

  


```CSS
.box {
    --radius: round(2em, 1px); /* 圆角半径 */
    --bw: 0.25em; /* border-width */
    
    position: relative;
    transform-style: preserve-3d;
    background: repeating-linear-gradient(45deg, #ccc 0 30px, #0000 30px 60px);
    border: var(--bw) solid;
    border-radius: var(--radius);
    
    shadow {
        position: absolute;
        inset: -150px;
        border: 150px solid #0000;
        border-radius: calc(150px + var(--radius));
        translate: 0 0 -1px;
        outline: 1px dashed; /* debug code */
    }
}
```

  


上面代码很简单，但 `shadow` 中的一些数字（例如 `150px`）可能会令你感到困惑。请先不用太关注这些具体的数字，我想随着逐步深入，你将会明白这背后的逻辑。下图中虚线框对应的就是 `shadow` ：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/964fd31623be4a2d9edf4a145920fe0c~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1253&s=351365&e=jpg&b=ffffff)

  


接下来，我们使用 `shadow::after` 伪元素创建渐变阴影：

  


```CSS
.box {
    --g: conic-gradient(from 90deg at 40% -25%,#ffd700, #f79d03,#ee6907,#e6390a,#de0d0d,#d61039,#cf1261, #c71585,#cf1261,#d61039,#de0d0d,#ee6907, #f79d03,#ffd700,#ffd700, #ffd700);
    --s: round(0.5em, 1px); /* 控制阴影的扩展半径, 类似 box-shadow 中的 spread */
    --r: round(0.25em, 1px); /* 控制模糊半径, 类似 box-shadow 中的 blur  */
    --xOffset: 0; /* X轴偏移量，类似 box-shadow 中的 x-offset */
    --yOffset: 0; /* Y轴偏移量，类似 box-shadow 中的 y-offset */
    --b: calc(infinity * 1px); /* B，一个无穷大的值 */
    --_b: calc(infinity * -1px); /* -B, 这里负值表示坐标方向*/
    --radius: round(2em, 1px); /* 圆角半径 */
    --bw: 0.25em; /* border-width */
    
    border: var(--bw) solid;
    border-radius: var(--radius);
    position: relative;
    transform-style: preserve-3d;

    shadow {
        position: absolute;
        inset: -150px;
        border: 150px solid #0000;
        border-radius: calc(150px + var(--radius));
        outline: 1px dashed;
        translate: 0 0 -1px;
      
        &::after {
            content: "";
            position: absolute;
            inset: calc(-1 * var(--s));
            background: var(--g);
            filter: blur(var(--r));
            translate: var(--xOffset) var(--yOffset);
            border-radius: var(--radius);
        }
    }
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/bf9bc06db4db40d7b8d2c004f5af3d57~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1253&s=625491&e=jpg&b=ffffff)

  


正如你所见，`shadow::after` 中创建渐变阴影的代码与之前所有示例是相同的。唯一的区别是 3D 变换定义在 `<shadow>` 元素上，而不是 `shadow::after` 上。

  


万事俱备，只欠东风。我们接下来要使用 `mask` 来隐藏 `shadow::after` 的部分区域，即对应主元素尺寸的区域，因为我们目标只是希望阴影可见。

  


```CSS
shadow {
    mask: linear-gradient(#000 0 0) content-box, linear-gradient(#000 0 0);
    mask-composite: exclude;
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/f32cb71622bb44d7a4cf097c95381c56~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1253&s=282241&e=jpg&b=ffffff)

  


请注意，我们使用边框和 `inset` 的组合来定义该区域。这使我们能够保持额外元素的内边距框与主元素相同，从而避免伪元素需要额外的计算。使用额外元素 `<shadow>` 带来的另一个好处是，该元素是固定的，只有伪元素在移动。这将使我们能够轻松的定义遮罩。

  


还有一点，[为了使内半径与主元素的圆角相匹配](https://juejin.cn/book/7395119173656903715/section/7401099842379841545#heading-1)。示例中定义了一个大的边框（`150px`）和一个等于大边框加主元素半径的圆角。外部的 `border-radius` 是 `calc(150px + var(--radius))` ，内部的 `border-radius` 是 `var(--radius)` ：

  


```CSS
shadow {
    border: 150px solid #0000;
    border-radius: calc(150px + var(--radius));
    
    &::after {
        border-radius: var(--radius);
    }
}
```

  


除此之外，为了确保边框可见，为此，我定义了两个遮罩层——一个覆盖内容区域，另一个覆盖边框区域，然后通过 `mask-composite` 对这两个遮罩层进行差集（`exclude`）操作。这也是我们这个技巧中最为关键的部分。不知道你是否还记得，[在之前创建渐变边框的时候，我们也应用了相似的技术](https://juejin.cn/book/7395119173656903715/section/7413371655377911835#heading-5)。

  


最后还有一点需要解决的是，我们需要将边框的宽度加到 `<shadow>` 元素的 `inset` 中：

  


```CSS
shadow {
    position: absolute;
    inset:calc(-150px - var(--bw)); /* --bw 主元素的边框厚度 */
    border: 150px solid #0000;
    border-radius: calc(150px + var(--radius));
    translate: 0 0 -1px;
      
    mask: linear-gradient(#000 0 0) content-box, linear-gradient(#000 0 0);
    mask-composite: exclude;
}
```

  


现在你看到的效果如下：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/e99d3110fb5a42cba5e81ff97babd3d9~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1253&s=281439&e=jpg&b=ffffff)

  


> Demo 地址：https://codepen.io/airen/full/QWeLMvx

  


这样，你就创建了一个带有圆角渐变阴影的效果。最终的 CSS 代码如下：

  


```CSS
.box {
    --g: conic-gradient(from 90deg at 40% -25%,#ffd700, #f79d03,#ee6907,#e6390a,#de0d0d,#d61039,#cf1261, #c71585,#cf1261,#d61039,#de0d0d,#ee6907, #f79d03,#ffd700,#ffd700, #ffd700);
    --s: round(0.5em,1px); /* 控制阴影的扩展半径, 类似 box-shadow 中的 spread */
    --r: round(0.25em, 1px); /* 控制模糊半径, 类似 box-shadow 中的 blur  */
    --xOffset: 0; /* X轴偏移量，类似 box-shadow 中的 x-offset */
    --yOffset: 0; /* Y轴偏移量，类似 box-shadow 中的 y-offset */
    --b: calc(infinity * 1px); /* B，一个无穷大的值 */
    --_b: calc(infinity * -1px); /* -B, 这里负值表示坐标方向*/

    --radius: round(2em, 1px); /* 圆角半径 */
    --bw: 0.25em; /* border-width */

    background: repeating-linear-gradient(45deg, #ccc 0 30px, #0000 30px 60px);
    color: var(--c);
    width: 80vmin;
    aspect-ratio: 16 / 9;
    display: grid;
    place-content: center;
    font-size: clamp(1.125rem, 3vw + 0.875rem, 1.375rem);
    
    border: var(--bw) solid #666;
    border-radius: var(--radius);
    position: relative;
    transform-style: preserve-3d;

    shadow {
        position: absolute;
        inset:calc(-150px - var(--bw));
        border: 150px solid #0000;
        border-radius: calc(150px + var(--radius));
        translate: 0 0 -1px;
        mask: linear-gradient(#000 0 0) content-box, linear-gradient(#000 0 0);
        mask-composite: exclude;
        
        &::after {
            content: "";
            position: absolute;
            inset: calc(-1 * var(--s));
            background: var(--g);
            filter: blur(var(--r));
            translate: var(--xOffset) var(--yOffset);
            border-radius: var(--radius);
        }
    }
}
```

  


这个技巧的最大缺点就是我们为边框设置了一个较大值，比如示例中的 `150px` 。这个值应该足够大，以容纳阴影，但又不能太大，以免出现溢出或滚动条问题。注意，这个 `150px` 并不是一个经验值，你可以根据实际需要调整它，只要确保一点，有足够的空间可以容纳阴影的显示。你也可以使用一个 CSS 自定义属性来表示，比如 `--border` ，这样易于调整它的值：

  


```CSS
shadow {
    --border: 150px;
    inset:calc(-1 * var(--border) - var(--bw));
    border: var(--border) solid #0000;
    border-radius: calc(var(--border) + var(--radius));
}
```

  


上面示例所展示的是所有圆角的半径都相等。如果你有需要，可以为每个角应用不同的半径：

  


```CSS
.box {
    --radius-tl: calc(0.5 * var(--radius));
    --radius-tr: var(--radius);
    --radius-br: calc(1.5 * var(--radius));
    --radius-bl: calc(2 * var(--radius));
    border-radius: var(--radius-tl) var(--radius-tr) var(--radius-br) var(--radius-bl);
      
    shadow {
        border-radius: 
            calc(var(--border) + var(--radius-tl)) 
            calc(var(--border) + var(--radius-tr))
            calc(var(--border) + var(--radius-br))
            calc(var(--border) + var(--radius-bl));
        
        &::after {
            border-radius: var(--radius-tl) var(--radius-tr) var(--radius-br) var(--radius-bl);
        }
    }
 }
```

![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/4b9aa2a6b5094a2ba4bfe9285006fd78~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1019&s=139163&e=jpg&b=04082f)

  


这个效果其实一点不复杂，卡片的左上角和右下角有一个变形的渐变边框和带有渐变效果的阴影。[我们小册之前专门有一节课向大家介绍了如何制作渐变边框](https://juejin.cn/book/7395119173656903715/section/7413371655377911835)，将制作渐变边框的技巧与这节课的渐变阴影技巧相结合，很易就能实现目标效果。

  


简单地说，我们将利用 `::before` 和 `::after` 来制作渐变边框和渐变阴影，其中一个用于创建渐变边框，另一个用于创建渐变阴影，具体是哪一个，看你喜好。

  


我们先从它的 HTML 结构开始：

  


```HTML
<div class="card"></div>
```

  


现在，我们可以开始为它添加样式。我们将为 `.card` 定义尺寸，同时将它设置为网格容器，居中所有内容，使其看起来更好。我们还会将其 `position` 设置为 `relative` ，这样可以更轻松地对其子元素使用 `absolute` 定位。另外，与之前示例一样，我在这里选择将 `transform-style` 设置为 `preserve-3d` ，便于将伪元素 `::before` 和 `::after` 放置到 `.card` 元素后面：

  


```CSS
.card {
    display: grid;
    place-content: center;
    padding: 1rem;
    font-size: clamp(1.25rem, 3vw + .675rem, 1.5rem);
    text-align: center;
    text-wrap: balance;
    width: fit-content;
    aspect-ratio: 3 / 4;
    line-height: 1.685;
    
    --g:linear-gradient(-45deg, #687bdd 0%, #221e22 50%, #e90e45 100%);
    --s: round(.5em, 1px);
    --r: round(5em, 1px);
    
    --primary: #030a2e;
    --color: color-contrast(var(--primary) vs #111, #fff to AA);
    background: var(--primary);
    color: var(--color);
    position: relative;
    transform-style: preserve-3d;
    
    &::before,
    &::after {
        content: "";
        position: absolute;
        inset: calc(-1 * var(--s));
        translate: 0 0 -1px;
    }
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/2442e96bcfff42039aa3137bce27b72a~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1100&s=340021&e=jpg&b=03082e)

  


上图中红色和绿色虚线框分别代表的是 `.card` 的 `::before` 和 `::after` 两伪元素。接下来，我们要在这两个伪元素上分别创建渐变边框和渐变阴影。我选择了在 `::before` 上创建渐变边框，在 `::after` 创建渐变阴影：

  


```CSS
.box {
    --g: linear-gradient(-45deg, #687bdd 0%, #221e22 50%, #e90e45 100%); /* 渐变效果 */
    --s: round(0.5em, 1px); /* 渐变边框厚度和渐变阴景扩展距离*/
    --r: round(5em, 1px); /* 渐变阴影模糊半径 */
    
    /* CSS ...*/
    position: relative;
    transform-style: preserve-3d;
    
    &::before,
    &::after {
        content: "";
        position: absolute;
        inset: calc(-1 * var(--s)); /* 它需要是一个负值 */
        translate: 0 0 -1px; /* 等同于 z-index: -1 */
        background: var(--g);
    }

    &::after {
        filter: blur(var(--r));
    }
}
```

  


不难发现，`::before` 和 `::after` 几乎相同，唯一区别就是 `::after` 设置了 `filter` 的 `blur()` ，模拟渐变阴影的模糊程度。

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/01f0e31ba9a943a7977b3c3c7e7bc2e5~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1100&s=272289&e=jpg&b=050930)

  


现在，渐变边框和渐变阴影效果都有了。但离我们最终要的效果还差一点——倾斜的边框和阴影效果。实现这个效果很简单，通过 CSS 变换中的 `skew()` 就可以实现：

  


```CSS
.card {
    /* CSS ... */
    
    &::before,
    &::after {
        /* CSS ... */
        translate: 0 0 -1px;
        transform: skewX(3deg) skewY(6deg);
    }
}
```

  


你可能发现了，代码中同时有 `translate` 和 `transform` 属性，但它们并不会相互覆盖，只会相互叠加，其中 `translate` 是现代 CSS 中为变换新增的特性——单个变换，它等同于 `transform` 中的 `translate()` 或 `translate3d()` 。要是你对这方面特性感兴趣，[可以点击获取更多这方面的信息](https://juejin.cn/book/7223230325122400288/section/7259668493158023205)。

  


现在你看到的效果如下：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/fc0103655df742d7b32b5a7d538d6885~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1100&s=274302&e=jpg&b=050930)

  


现在，你完全可以调整 `--s` 、`--r` 和 `--g` 几个 CSS 变量来获得你想要的卡片效果。甚至稍微调整一下，就可以使渐变边框和渐变阴影动起来。例如，我在这里借助 [CSS 的 `@property` 规则](https://juejin.cn/book/7223230325122400288/section/7258870477462962236)和 [CSS 的 `transition`](https://juejin.cn/book/7288940354408022074/section/7292735608995184678) 特性 ，在卡片悬浮状态和获得焦点状态动画化渐变的效果：

  


```CSS
@property --a {
    syntax: "<angle>";
    initial-value: 0deg;
    inherits: false;
}

.card {
    display: grid;
    place-content: center;
    padding: 1rem;
    font-size: clamp(1.25rem, 3vw + 0.675rem, 1.5rem);
    text-align: center;
    text-wrap: balance;
    width: fit-content;
    aspect-ratio: 3 / 4;
    line-height: 1.685;

    --a: -45deg;
    --g: linear-gradient(var(--a), #687bdd 0%, #221e22 50%, #e90e45 100%);
    --s: round(0.5em, 1px);
    --r: round(5em, 1px);
    --primary: #030a2e;
    --color: color-contrast(var(--primary) vs #111, #fff to AA);
    background: var(--primary);
    color: var(--color);
    position: relative;
    transform-style: preserve-3d;
    transition: --a .2s ease-in-out; /* 动画化 --a */
    
    &::before,
    &::after {
        content: "";
        position: absolute;
        inset: calc(-1 * var(--s));
        translate: 0 0 -1px;
        transform: skewX(3deg) skewY(6deg);
        background: var(--g);
    }

    &::after {
        filter: blur(var(--r));
    }
    
    &:is(:hover, :focus-visible) {
        --a: 135deg;
    }
}
```

### 创建光晕效果

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/9ffc8922bc5d447a89ed9f82a63784b1~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1606&s=739040&e=jpg&b=000000)

  


Web 网站或应用上在展示产品时，为了突出产品效果，往往会在产品图片底部添加一些光晕效果，如上图所示。仔细观察，上图中的光晕效果并不是单纯的带有模糊度的纯色。因此，我们无法直接单纯的使用 `box-shadow` 或 `drop-shadow()` 来实现。

  


我猜，可能部分同学已经想到了相应的方案。如果你还没想到，不要紧！接着往下阅读，你一下子就会明白其背后原理。

  


事实上，实现上图光晕效果，其原理非常简单，甚至与创建渐变阴影有点类似。它主要包含两个技巧，其一使用渐变来创建多彩的光（即光的颜色），其二就是利用 CSS 遮罩 `mask` 特性隐藏部分背景。注意，它有一个过渡的效果，这对于熟悉 CSS 遮罩 `mask` 的开发者来说，一点都不是问题：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/32e6d62ba4624d339518436688ec380d~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=2124&s=735184&e=jpg&b=03082f)

  


具体的代码如下：

  


```HTML
<div class="card">
    <figure>
        <img src="figure.png" alt="" />
    </figure>
    <div class="card__content"></div>
</div>
```

  


我将使用 CSS Flexbox 来构建卡片的布局，并在它的伪元素 `::after` 上应用 CSS 渐变和遮罩来创建产品图片底部的光晕效果：

  


```CSS
.card {
    padding: 1rem 2rem;
    border-radius: 0.5em;
    display: flex;
    gap: 3rem;
    align-items: center;
    width: clamp(320px, 60vw, 760px);
    aspect-ratio: 16 / 9;
    position: relative;
    border: 1px solid #eceff133;
    

    &::after {
        content: "";
        position: absolute;
        inset: 0;
        z-index: -1;
        border-radius: inherit;
        background: linear-gradient(135deg,aqua 12.9254386713%,deeppink 37.9254386713%);
        mask: radial-gradient(circle at 22% 50%, black 0%, #0000 30%);
    }
    
    &:nth-child(2n) {
        flex-direction: row-reverse;
      
        &::after {
            background: 
                radial-gradient(circle at 75% 50%,rgb(0 255 255 / 0.5),#0000 25%),
                radial-gradient(circle at 75% 50%,rgb(0 255 255 / 0.5),#0000 30%),
                radial-gradient(circle at 75% 50%,rgb(0 255 255 / 0.5),#0000 35%),
                linear-gradient(-45deg, aqua 12.9254386713%, deeppink 37.9254386713%);
            mask: radial-gradient(circle at 78% 50%, black 0%, transparent 30%);
            background-blend-mode: color-dodge;
        }
    }
}
```

![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/3c37730c5ebb4e479920893b011ebfa2~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1414&s=673890&e=jpg&b=e6e4e5)

  


> Demo 地址：https://codepen.io/airen/full/BaXBPep

![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/1a36b8d010f143e288408e514f85f0f9~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1414&s=940448&e=jpg&b=e6e1e4)

### 给图像添加发光效果

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/3bce01f4ef164c08ab13ee017d8a44d0~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1414&s=1096629&e=jpg&b=061726)

  


当我看到上图的效果，第一反应是：“哇，这个基于图像本身发光效果真酷”！另一反应是：“你肯定碰到过类似的需求”。

  


接下来，我们来使用 CSS 实现一个类似这样的效果：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/db4f9d3ce17148f8a2b8cf497d610a92~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=889&s=312121&e=jpg&b=041b28)

  


首先创建一个简单的 HTML：

  


```HTML
<div class="card">
    <div class="content">
        <img src="olympic-mascot.png" alt="Olympic Mascot" />
    </div>
    <h3>Olympic mascot</h3>
</div>  
```

  


当你看到这个效果，我想你首先会想到使用 `fliter` 的 `drop-shadow()` 函数给 `<img>` 创建投影效果。但它与 `box-shadow` 是有点类似，你只能给它设置一个纯色。它也无法在不同图像之间复用，因为每张图像的单一颜色会有所不同。此外，单色发光在像上图右侧这样多彩的图像上效果并不好看。

  


比较好的解决方案是使用与 `img` 图像源作为 `.content` 元素的背景，这也是为什么在 HTML 中使用 `div` 元素来包裹 `img` 元素的原因之一：

  


```CSS
.content {
    width: 200px;
    aspect-ratio: 1;
    background: var(--url)center / contain no-repeat;
    
    img {
        display: block;
        width: 100%;
        height: 100%;
        box-sizing: content-box;
        object-fit: contain;
        object-position: center;
    }
}
```

  


这里有一个小技巧，为了使 `.content` 中的背景图和 `img` 在层叠时不发生错位的现象，我在这里使用 `background-position` 和 `background-size: contain` 将背景图设置在元素的正中间。同时使用 `object-fit` 和 `object-position` 使 `img` 也位于其容器中间。在这里，`object-fit` 与 `background-size` ，`object-position` 与 `background-position` 达到相同的功能，只不过服务对象不同，一个是为 `img` 服务，一个是为 `background-image` 服务。有关于这方面更详细的介绍，[可以点击这里进行了解](https://juejin.cn/book/7199571709102391328/section/7199845888997457959#heading-11)！

  


接下来，我们需要对其进行模糊处理，使图像具有发光光效果，而不仅仅是双重图像。[我通过在 `img` 元素上使用 `backdrop-filter` 实现了这一点](https://juejin.cn/book/7223230325122400288/section/7259669043622690853#heading-48)。它允许一个元素对其后面的元素应用滤镜。因此，将 `backdrop-filter` 应用于 `<img>` 上是对其底下的 `.content` 的背景图像应用滤镜：

  


```CSS
.content img {
    backdrop-filter: blur(15px) saturate(3.5);
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/ff4370b372c2436a85aa78552375c6b0~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=889&s=302111&e=jpg&b=041b28)

  


这效果几乎就是我们所期望的效果。然而，你可以看到效果存在一个小瑕疵：`.content` 裁剪了模糊背景，产生了奇怪的硬边缘。为了解决这个问题，我们可以使用一个小技巧，使 `img` 元素的边框盒子变大而不移动元素本身：

  


```CSS
.content img {
    backdrop-filter: blur(15px) saturate(3.5);
    padding: 20px;
    margin: -20px;
    box-sizing: content-box;
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/efc4ce8792ce4cca9327f70cd89b3f2f~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1158&h=486&s=1176238&e=gif&f=95&b=031822)

  


要做到这一点，还需确保元素的 `box-sizing` 为 `content-box` ，而不是 `border-box` 。

  


到这一步，其实你已经使图像具有发光效果了，但你可以在此基础上[添加一些处理图像特效的样式](https://juejin.cn/book/7223230325122400288/section/7259669043622690853)，使其效果更细腻一点，比如我在 `.content` 新增了 `mix-blend-mode: difference` ，使其与父容器图层进行差异化处理。并且在 `img` 上新增了一个投影滤镜效果 `filter: drop-shadow(-3px -6px 11px #0ace)` 。与此同时，为了进一步的避免元素周边有硬化效果，我还使用了一个 `mask` ，对元素周边做虚化处理。最终的 CSS 代码如下：

```CSS
        height: 100%;

}

}
```

看到上图，你可能会认为我们接下来要创建一个毛玻璃的 UI 效果。你说的对，但也不全对。因为我们接下来要创建一个质感十足的光晕背景，但制作这个光晕背景将会用到制作毛玻璃 UI （磨砂效果）效果的技巧。例如下图这个效果：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/31c763f7c4444237aef57343b1915909~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1194&s=394453&e=jpg&b=8b9aa0)

  


你千万别想着使用 CSS 的渐变能实现类似上图这样的光晕背景。至少到目前为止，我自己无法仅通过 CSS 渐变可以实现它。但是我们换过一下思路，就能使用 CSS 轻易实现上图所示的背景效果。

  


我们从 HTML 结构开始：

  


```HTML
<div class="glow">
    <div class="content">
        Glowing Gradient Glassmorphism
    </div>
    <div class="glow__bgs">
        <span></span>
    </div>
</div>
```

  


这个结构很简单，但我还是要简单的解释一下。`.glow` 容器包裹了内容 `.content` 和制作背景的元素 `.glow__bgs` 。如果你需要更复杂的光晕背景，可以在 `.glow__bgs` 添加更多的 `<span>` 元素。

  


我在这里使用 CSS Grid 对它进行布局，将 `.content` 和 `.glow__bgs` 层叠在同一个区域，并且使用 `z-index` 将 `.content` 放置在 `z` 轴的顶层，因为我们要确保内容不被 `.glow__bgs` 遮盖。

  


```CSS
.glow {
    position: fixed;
    inset: 0;
    
    display: grid;
    
    > * {
      grid-area: 1 / 1 / -1 / -1;
    }
    
    .content {
        z-index: 2;
    }
    
    .glow__bgs {
        z-index: 1;
    }
}
```

  


接着使用绝对定位，将 `.glow__bgs` 自身和它的伪元素 `span` 的伪元素 `::before` 和 `::after` 填充 `.glow__bgs` 。

  


```CSS
.glow__bgs {
    position: relative;
      
    > * {
        position: absolute;
        inset: 0;
    }
      
    &::before,
    &::after,
    *::before,
    *::after {
        content: "";
        position: absolute;
        inset: 0;
    }
}
```

  


注意，这样你一下子就多出四个元素来，这样做主要是为了避免使用更多的空标签。

  


有了这个基础之后，就可以进入第一个关键环节，我们在 `.glow__bgs` 和（或） `span` 以及它们的子元素上设置背景颜色或渐变效果，[然后使用 `clip-path` 裁剪出你想要的图形形状](https://juejin.cn/book/7223230325122400288/section/7259668885224456252)：

  


```CSS
.glow__bgs {
    
    &::before {
        background: orange;
        clip-path: polygon(20% 20%,60% 15%, 80% 40%,30% 20%);
        z-index: 2;
    }
      
    &::after {
        background: lime;
        clip-path: circle(10% at 70% 65%);
        z-index: 2;
    }
      
    span {
        background: repeating-radial-gradient(circle at center,#90acef 0% 5%,#ffd 5% 10%);
        clip-path: polygon(0% 15%, 15% 15%, 15% 0%, 85% 0%, 85% 15%, 100% 15%, 100% 85%, 85% 85%, 85% 100%, 15% 100%, 15% 85%, 0% 85%);
        z-index: 1;
        &::before {
            background: red;
            clip-path: ellipse(20% 10% at 30% 50%);
        }
        
        &::after {
            background: pink;
            clip-path: polygon(60% 30%,50% 80%,30% 50%)
        }
    }
} 
```

有点遗憾的是，目前 CSS 还没有具备随机生成颜色和图形的功能，但 CSS 处理器，比如 Sass 、LESS 是具备这方面的能力的。不过，这不是我们所关注的重点。我们关注的重点是，CSS 怎么就让这些具有不同颜色的形状变成了光晕效果。

```CSS

    position: absolute;
    inset: 0;
    backdrop-filter: blur(100px);
    background-color: rgb(10 10 10 / .25);
    z-index: -1;
}
```

```JavaScript
const applyOverlayMask = (e) => {

};
```

```JavaScript
cards.forEach((card) => {
    card.addEventListener('mousemove', (event) => {
        const target = event.currentTarget; // 当前触发事件的 card
        const rect = target.getBoundingClientRect(); // 获取该 card 的位置和大小

        // 计算鼠标在容器中的偏移量
        let offsetX = event.clientX - rect.left;
        let offsetY = event.clientY - rect.top;

        // 将偏移量转换为百分比
        let percentX = (Math.min(Math.max(offsetX / rect.width, 0), 1) * 100).toFixed(2);
        let percentY = (Math.min(Math.max(offsetY / rect.height, 0), 1) * 100).toFixed(2);


        // 将百分比值设置为当前 card 的 CSS 自定义变量 --x 和 --y
        card.style.setProperty('--x', `${percentX}%`);
        card.style.setProperty('--y', `${percentY}%`);
    });

    card.addEventListener('mouseout', () => {
        // 鼠标移出时可以执行一些操作，比如重置变量等
        console.log('Mouse left the card');
    });
});
```
