## 创意裁剪图片画廊

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/abc0a18c28e545b99023fafb772a8867~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1029&s=614606&e=jpg&b=607d8b)

  


我们跳过 HTML，直接进入 CSS。我们使用 `grid-template-columns` 和 `grid-auto-rows` 创建一个 `3 x 3` 的网格：

  


```CSS
.gallery {
    --g: 10px;
    --s: 60vmin;
    width: var(--s);
    aspect-ratio: 1;
    
    display: grid;
    gap: var(--g);
    grid-template-columns: repeat(3, minmax(0,1fr));
    grid-auto-rows: minmax(0, 1fr);
    
    img {
        width: 100%;
        height: 100%;
        object-fit: cover;
        object-position: center;
    }
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/1eeb94cf5ee04c7e970f85f64b681bbe~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2742&h=1780&s=484630&e=jpg&b=687f89)

  


你会发现，上面的代码并没有按照我们的预期——创建一个 `3 x 3` 的网格，而是一个 `3 x 2` 的网格，这是因为我们只使用 `grid-template-columns` 指定了网格的列数，但并没有使用 `grid-template-rows` 指定网格的行数。当然，你也可以将 `grid-template-rows` 属性的值设置为 `repeat(3, minmax(0, 1fr))` 来指定网格行数。这样就创建了一个 `3 x 3` 的显示网格。

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/db41b38637f94291aa47ecf9e3a13c6c~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2760&h=1796&s=463897&e=jpg&b=687f88)

  


如果你对 CSS Grid 布局技术不熟悉的话，可能就会有疑惑了，`3 x 3` 的网格如何创建呢？其实很简单，当我们使用 `grid-area` 将网格项目放置在显式网格范围外时，浏览器就会自动创建一个隐式网格。我在小册的《[CSS 布局：隐式网格和自动放置](https://juejin.cn/book/7395119173656903715/section/7434843979486593062)》课程中详细阐述过这方面的知识。

  


在指定网格项目（图片）位置之前，我们先简单的分析一下我们的目标效果。不难发现：

  


-   第一张和第四张图片只占了一个网格单元格区域，并且分别放置在网格的第一个单元格和最后一个单元格
-   第二张和第三张图片占了四个网格单元格区域，分别位于网格右上角和左下角

  


我们可以使用网格线的数字索引号将网格项目放置到我们指定的网格区域范围内：

  


```CSS
.gallery {
    /* CSS... */
    
    img {
        /* CSS... */
        
        &:nth-child(1) {
            grid-area: 1 / 1 / 2 / 2;
        }
        
        &:nth-child(2) {
            grid-area: 1 / 2 / 3 / 4;
        }
        
        &:nth-child(3) {
            grid-area: 2 / 1 / 4 / 3;
        }
        
        &:nth-child(4) {
            grid-area: 3 / 3 / 4 / 4;
        }
    }
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/f25171e40bb447b78dcb6b3ad177c39f~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2742&h=1778&s=622413&e=jpg&b=657e8a)

  


正如你所看到的，即使我们没有使用 `grid-template-rows` 属性指定网格行数。但是，`grid-area` 放置第三个和第四个网格项目时溢出了显式网格区域，此时浏览器就自动添加了相应的行数来满足放置网格项目的需要。

  


你可能已经注意到了，网格行数自动增加的同时，网格行高也发生了变化。这是因为我们使用了 `grid-auto-rows` 指定了网格行轨道的尺寸，在这个示例中，它将会均分网格容器的剩余空间（网格容器高度减去所行间距）。

  


需要知道的是，我们还可以使用 `span` 关键词（注意，这是 CSS Grid 布局中独有的关键词，它不是 HTML 中的 `<span>`）来合并单元格。例如，第二张图片和第三张图片，它们分别合并了两列和两行。我们可以将上面的代码调整为：

  


```CSS
.gallery {
    /* CSS... */
    
    img {
        /* CSS... */
        
        &:nth-child(1) {
            grid-area: 1 / 1 / 2 / 2;
        }
        
        &:nth-child(2) {
            grid-area: 1 / 2 / span 2 / span 2;
        }
        
        &:nth-child(3) {
            grid-area: 2 / 1 / span 2 / span 2;
        }
        
        &:nth-child(4) {
            grid-area: 3 / 3 / 4 / 4;
        }
    }
}
```

  


它所获得的效果是一致的：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/0cf680dd43e0428783dfdf08d9395470~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2754&h=1736&s=629581&e=jpg&b=667e8a)

  


甚至只需要将第二张和第三张图片显式地放置到网格区域，从而让第一张和第四张图片自动放置在它们的周围。换句话说，在这个示例中，第一张和第四张图片不需要显式指定 `grid-area` ，借助网格自动放置网格项目的能力，可以使布局达到与之前一样的结果。

  


```CSS
.gallery {
    /* CSS... */
    
    img {
        /* CSS... */

        &:nth-child(2) {
            grid-area: 1 / 2 / span 2 / span 2;
        }
        
        &:nth-child(3) {
            grid-area: 2 / 1 / span 2 / span 2;
        }
    }
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/4c8a3e1e515040e7ace0f82fa8161924~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2760&h=1734&s=637107&e=jpg&b=667f8a)

  


到目前为止，使用 CSS Grid 构建出我们所需要的布局。但第二张和第三张图片正好重叠在中间。这并不是错误！使用 `grid-area` 放置指定这两张图片在网格中的位置时，正好它们占用了同一个单元格。这也是我们所期望的一个效果。

  


接下来，我们就可以使用 `clip-path` 对它们进行裁剪，从而获得最终的效果：

  


```CSS
.gallery {
    /* CSS... */
    
    img {
        /* CSS... */

        &:nth-child(2) {
            grid-area: 1 / 2 / span 2 / span 2;
            clip-path: polygon(0 0, 100% 0, 100% 100%, calc(50% + var(--g) / 4) 100%, 0 calc(50% - var(--g) / 4));
        }
        
        &:nth-child(3) {
            grid-area: 2 / 1 / span 2 / span 2;
            clip-path: polygon(0 0, calc(50% - var(--g) / 4) 0, 100% calc(50% + var(--g) / 4), 100% 100%, 0 100%);
        }
    }
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/c2d1dfb8230e4200aeb28a27b654b087~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1029&s=1245381&e=jpg&b=677d8a)

  


最终的 CSS 代码如下：

  


```CSS
.grid {
    --s: 60vmin;
    --g: 10px;
    width: 60vmin;
    aspect-ratio: 1;

    display: grid;
    grid-template-columns: repeat(3, minmax(0, 1fr));
    grid-auto-rows: minmax(0, 1fr);
    gap: var(--g);

    img {
        width: 100%;
        height: 100%;
        object-fit: cover;
        object-position: center;

        &:nth-child(2) {
            grid-area: 1 / 2 / span 2 / span 2;
            clip-path: polygon(
                0 0,
                100% 0,
                100% 100%,
                calc(50% + var(--g) / 4) 100%,
                0 calc(50% - var(--g) / 4)
            );
        }

        &:nth-child(3) {
            grid-area: 2 / 1 / span 2 / span 2;
            clip-path: polygon(
                0 0,
                calc(50% - var(--g) / 4) 0,
                100% calc(50% + var(--g) / 4),
                100% 100%,
                0 100%
            );
        }
    }
}
```

## 图片分割特效

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/e487772d2bee47a283749886d6c51768~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1140&h=594&s=4419757&e=gif&f=133&b=35afa7)

  


这个示例同样会用到之前学到的[角度切割技巧](https://juejin.cn/book/7395119173656903715/section/7400975207551139875#heading-0)，不过我们使用角度切割技巧来制作一个鼠标悬浮时揭示完整的图片。

  


实现这个效果与之前案例的 HTML 略有不同，我们在网格容器中嵌套了两张一样的图片：

  


```HTML
<div class="gallery">
    <img src="https://picsum.photos/id/163/800/800" alt="" />
    <img src="https://picsum.photos/id/163/800/800" alt="" />
</div>    
```

  


我们使用 CSS Grid 将两张图片完全重叠在一起：

  


```CSS
.gallery {
    --s: 400px;
    display: grid;
    
    img {
        width: var(--s);
        aspect-ratio: 1;
        object-fit: cover;
        object-position: center;
        grid-area: 1 / 1 / -1 / -1/
    }
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/ee49dbc657a1474780d6733e2adfe6d2~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2752&h=1752&s=415017&e=jpg&b=667e8a)

  


在 CSS 中，我们可以使用 `clip-path` 或 `mask` 对图片进行分割。不管使用哪种方式，实现鼠标悬浮效果的原理是一致的。仔细观察，我们会发现图片有三种不同的状态：

  


-   1️⃣：当鼠标没有悬浮在图片上时，每张图片的一半是可见的
-   2️⃣：当鼠标悬浮在第一张图片上时，图片会显示得更完整，但保留一个小角被裁剪
-   3️⃣：当鼠标悬浮在第二张图片上时，第一张图片只剩下一个小三角形可见

  


下图展示了第一张图片在悬浮状态的三种裁剪效果：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/598ef5bd012640479d8e297c333813da~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=2185&s=1928911&e=jpg&b=607d8b)

  


在 CSS 中，使用 `clip-path` 和 `mask` 都可以裁剪出这三种结果。但是在悬浮状态使每种状态有一个平滑的过渡效果，还是有一些细节需要注意的。

  


-   虽然 `clip-path` 可以被动画化，但 `polygon()` 函数在裁剪形状时的坐标点数量和顺序要相同，否则无法获得平滑过渡的动效效果
-   `mask` 是通过遮罩形状（`mask-image`）裁剪形状，但 `mask-image` 属性不可以被动画化，因此我们需要借助其他的技巧来实现

  


我们先以 `clip-path` 为例。从目标结果来看，第 1️⃣ 和第3️⃣ 状态都是一个三角形，只有第2️⃣状态是一个多边形。但是，要动画化 `clip-path` 就必须保证 `polygon()` 函数的坐标点数量和顺序是一致的。这意味着，我们只能通过三个点来裁剪出第2️⃣状态的形状。你可能会问，三个点怎么才能裁剪出所需要的三角形形状呢？

  


大家是否还有印象，我在小册的《[现代 CSS 创建形状指南](https://juejin.cn/book/7395119173656903715/section/7395172754053890063)》一课中，在介绍一种 CSS 技巧，`polygon()` 只需要用到四个坐标点，就可以裁剪[六边形](https://juejin.cn/book/7395119173656903715/section/7395172754053890063#heading-12)和[八边形形状](https://juejin.cn/book/7395119173656903715/section/7395172754053890063#heading-13)：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/0c524b77baf84267b46710b6d698c323~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1096&s=222211&e=jpg&b=ffffff)

  


简单地说，`clip-path` 可以接受超出 `[0%,100%]` 范围的值，这允许我们创建“溢出”形状。使用这个技巧，`polygon()` 函数只需要三个点（而不是五个点）就可以裁剪出第 2️⃣ 状态的形状：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/e7851185d56f4a4a80e6090a812b20b3~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=2547&s=1251381&e=jpg&b=607d8b)

  


我们可以采用同样的原理对第二张图进行裁剪，使它们都具有第2️⃣ 状态的形状：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/c501cc256eb1453f8752639eea20fe35~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1336&h=822&s=4850950&e=gif&f=146&b=557080)

  


如此一来，悬浮状态中的三个形状结果都有了，而且满足了动画化 `clip-path` 的需求。

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/c2f8a1ed121f4adc845a8bdf159acccf~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=4854&h=2185&s=2263422&e=jpg&b=607d8b)

  


现在，我们可以使用下面的 CSS 代码来裁剪出所需要的形状：

  


```CSS
.gallery {
    /* CSS... */
    
    img {
        /* CSS... */
        
        &:first-child {
            clip-path: polygon(
                0 0, 
                calc(100% + var(--p)) 0 , 
                0 calc(100% + var(--p))
            );
        }
      
        &:last-child {
            clip-path: polygon(
                100% 100%, 
                100% calc(0% - var(--p)), 
                calc(0% - var(--p)) 100%
            )
        }
    }
}
```

  


上面的代码中，我们引入了 `--p` 变量。它可以用来优化我们的 CSS 代码。当我们添加悬浮过渡时，只需要更新这个变量的值，而不是更新整个 `clip-path` 。稍后你可以看到它的真实作用。

  


如果你使用浏览器实时查看上面的代码的结果，你会发现，虽然 `clip-path` 将图片裁剪成三角形了，但两张图片之间并没有任何间距。

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/5e1edf460dee4ff5be3837147d763994~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2556&h=1788&s=442725&e=jpg&b=667e8a)

  


这是因为，我们没有给 `--p` 指定任何值，因此上面的代码相当于：

  


```CSS
.gallery {
    /* CSS... */
    
    img {
        /* CSS... */
        
        &:first-child {
            clip-path: polygon(
                0 0, 
                100% 0 , 
                0 100%
            );
        }
      
        &:last-child {
            clip-path: polygon(
                100% 100%, 
                100% 0%, 
                0% 100%
            )
        }
    }
}
```

  


裁剪之后的两个三角形吻合在一起，重新拼接成一张图片：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/0f7be3efc3784e8bb1ace523bf45e1b9~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1246&h=776&s=14742046&e=gif&f=108&b=547080)

  


现在，我们可以在 `img` 上添加 `transition` ，然后在不同状态下更新 `--p` 变量的值，来实现最终效果：

  


```CSS
.grid {
    --s: 400px;
    --g: 10px;
    display: grid;
    place-content: center;

    img {
        width: var(--s);
        aspect-ratio: 1;
        object-fit: cover;
        object-position: center;
        grid-area: 1 / 1 / -1 / -1;
        cursor: pointer;
        --p: calc(-1 * var(--g));
        transition: 0.4s 0.1s;

        &:first-child {
            clip-path: polygon(
                0 0,
                calc(100% + var(--p)) 0,
                0 calc(100% + var(--p))
            );
        }

        &:last-child {
            clip-path: polygon(
                100% 100%,
                100% calc(0% - var(--p)),
                calc(0% - var(--p)) 100%
            );
        }
    }

      &:hover {
          img:last-child,
          img:first-child:hover {
              --p: calc(50% - var(--g));
          }
    
          img:first-child,
          img:first-child:hover + img {
              --p: calc(-50% - var(--g));
          }
      }
}
```

  


最终效果如下：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/8351aedba08c470da552bb75ae5a911c~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1118&h=696&s=6000002&e=gif&f=127&b=557080)

  


> Demo 地址：https://codepen.io/airen/full/xbKbdea

  


接下来，我们使用 `mask` 来创建一个类似的效果。刚才提到过，CSS 无法动画 `mask-image` ，但我们可以在创建遮罩图的时候引入一个 CSS 变量，比如 `--pos` ，并且[使用 `@property` 来注册这个变量 `--pos`](https://juejin.cn/book/7223230325122400288/section/7258870477462962236) 。如此一来，我们可以动画化 `--pos` 变量，从而实现悬浮的过渡动画效果：

  


```CSS
@property --pos {
    syntax: "<percentage>";
    inherits: true;
    initial-value: 0%;
}

.grid {
    --s: 400px;
    --g: 10px;
    display: grid;
    place-content: center;
    grid-template-columns: auto;
    grid-auto-flow: column;
    overflow: hidden;
    font-size: 0;

    img {
        width: var(--s);
        aspect-ratio: 1;
        object-fit: cover;
        object-position: center;
        grid-area: 1 / 1 / -1 / -1;
        cursor: pointer;
        transition: --pos 0.4s 0.1s;
      
        --a: -45deg;
        --pos: 50%;
        --m: linear-gradient(
            var(--a),
            #0000 calc(var(--pos) + (1 - sqrt(2) / 2) * var(--g) + 1em),
            #000 0
        );
        
        mask: var(--m);

        &:first-child {
            --a: -45deg;
            font-size: calc(var(--g) / 2);
        }

        &:last-child {
            --a: 135deg;
            font-size: calc(var(--g) / 2);
        }
    }

    &:hover {
        img:first-child {
            --pos: 75%;
        }

        img:last-child {
            --pos: 25%;
        }
    }
    
    &:nth-child(2):hover {
        img:first-child {
            --pos: 25%;
        }

        img:last-child {
            --pos: 75%;
        }
    }
}
```

  


遗憾的是，我们并无法像 `clip-path` 方案那样同时对图片的两个方向提供揭示动效。因为 `clip-path` 属性裁剪掉的区域，它同时也会使相关的交互行为丢失，比如点击和悬浮行为。但 `mask` 不一样，它只是隐藏不可见的区域，但不会丢失相关的交互行为。因此，我们始终只能拿到一张图片的悬浮状态。

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/9f94c79a678a45d9af6356bf789086cf~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1276&h=612&s=11631602&e=gif&f=124&b=547080)

  


> Demo 地址：https://codepen.io/airen/full/raBaGVW

  


## 超级英雄布局：CSS 分层面板

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/1e863fe376064ba68382b8cce473b3f2~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1051&h=760&s=72962&e=webp&b=f4efeb)

  


这种布局通常被称为超级英雄布局（Superhero Layout）和漫画布局（Comics Layout）。就是通过使用重叠的面板为漫画叙事带来了深度、动作感和细节。而这种分层布局在 Web 设计中并不新鲜，但 CSS Grid 为我们提供了一种比绝对定位更简单、更可预测的实现方式。

  


我们来看一个稍微简单一点的示例，整个效果是一种艺术风格，通过小块单独的部分来创造装饰性设计。但它也可以由其他较小的图像组成的复合图像。具体效果如下图所示：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/bb7640374f9a4b2f9055bd2e940d2d66~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1177&s=748703&e=jpg&b=627c89)

  


创建上图所示的效果，其核心就两点：

  


-   使用 CSS Grid 布局技术，将图片（网格项目）层叠在一起，有可能是部分重叠
-   使用 `clip-path` 裁剪出所需要的形状，在这个示例中，主要是三角形和菱形

  


相比而言，使用 `clip-path` 裁剪出所需要的形状要更简单，棘手的是定义网格。接下来，我们一步一步以上图为例，看看网格如何来定义？

  


我们再次想象每张图片是一个矩形，然后根据看到的布局定义网格。我们从两张图片开始：

  


-   这两张图片都是正方形
-   第一张图片的尺寸是第二张的一半

  


如此一来，我们可以先定义一个 `2 x 2` 的网格：

  


```CSS
.grid {
    display: grid;
    grid-template-columns: 1fr 2fr;
    grid-template-rows: 1fr 2fr;
}
```

  


默认情况之下，第一张图片会自动放置在第一个网格单元格中，第二张图片会自动放置在网格单元格的第二个单元格中：

  


```CSS
.grid {
    display: grid;
    grid-template-columns: 1fr 2fr;
    grid-template-rows: 1fr 2fr;
    
    img {
        width: 100%;
        aspect-ratio: 1;
        object-fit: cover;
        object-position: center;
    }
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/d35882b85638410ab73ac898c8c40f0d~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2788&h=1742&s=673139&e=jpg&b=67828a)

  


使用 `clip-path` 将这两张图片裁剪成所需要的三角形形状：

  


```CSS
.grid {
    /* CSS... */
    
    img {
        /* CSS... */
        clip-path: polygon(var(--shape));
        
        &:nth-child(1) {
            --shape: 0% 0%, 100% 0%, 0% 100%;
        }
        
        &:nth-child(2) {
            --shape: 0% 0%, 100% 0%, 100% 100%;
        }
    }
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/c8b6dab4416543bb931fdf2f8310fd50~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2762&h=1746&s=415083&e=jpg&b=67838a)

  


现在，我们在布局中增加一张图片：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/8a56888564f54165b99bf84acf81f48b~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1177&s=738368&e=jpg&b=627d8a)

  


新增的图片使布局变得复杂了些。我们需要画一些线来帮助我们重新定义网格：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/67fbbb9b53fa4661ae418b3cdae7daa1~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1177&s=625409&e=jpg&b=677d8a)

  


从一个 `2 x 2` （两行两列）的网格调整为 `3 x 4` 的网格（三行四列）。这些布局有些不对称，我们需要花一点心思来确认网格轨道的尺寸。之前只有两第图片的时候，第 `2` 列的宽度是第 `1` 列的两倍。仔细观察，最初的两列分成了四列，为了保持列网格 `1` 和列网格线 `3` 之间的距离与列网格线 `3` 和列网格线 `5` 之间的距离保持与最初的比例一致（后者是前者的两倍）。我们可以把列轨道的尺寸重新定义为 `1fr 3fr 3fr 5fr` ：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/ea085009fad84ecd984a89fa957f6dc4~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1177&s=749840&e=jpg&b=677d8a)

  


```CSS
/* 调整前 */
.grid {
    display: grid;
    grid-template-columns: 1fr 2fr;
}

/* 调整后 */
.grid {
    display: grid;
    grid-template-columns: 1fr 3fr 3fr 5fr;
}
```

  


我们可以按照类似的方式来调整行网格轨道的尺寸：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/e3dcb865618f48b09aa89bcc34b6353a~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1177&s=817751&e=jpg&b=677d8a)

  


调整后的网格如下：

  


```CSS
.grid {
    display: grid;
    grid-template-columns: 1fr 3fr 3fr 5fr;
    grid-template-rows: 4fr 2fr 2fr;
}
```

  


由于网格重新定义了，我们需要调整相应网格项目的位置：

  


```CSS
.grid {
    display: grid;
    grid-template-columns: 1fr 3fr 3fr 5fr;
    grid-template-rows: 4fr 2fr 2fr;
    
    img {
        /* CSS... */
        &:nth-child(1) {
            grid-area: 1 / 1 / 2 / 3;
            --shape: 0% 0%, 100% 0%, 0% 100%;
        }
    
        &:nth-child(2) {
            grid-area: 1 / 3 / 4 / 5;
            --shape: 0% 0%, 100% 0%, 100% 100%;
        }
          
        &:nth-child(3) {
            grid-area: 1 / 2 / 2 / 4;
            --shape: 50% 0%, 100% 50%, 50% 100%, 0% 50%;
        }
    }
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/73eb8937cec44958a55b9dea0d8401e8~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2788&h=1782&s=584224&e=jpg&b=6b7d8a)

  


接下来，可以按照类似的方法来调整网格的定义，因为布局需要更多的行和列才能容纳所有图片：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/c25a826887184d55837de2a91504b7c0~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1177&s=1014070&e=jpg&b=677d8a)

.grid {
    display: grid;

    img {

        clip-path: polygon(var(--shape));
        
        &:nth-child(1) {
            grid-area: 1 / 1 / 3 / 4;
            --shape: 0% 0%, 100% 0%, 0% 100%;
        }

        &:nth-child(2) {

            --shape: 0% 0%, 100% 0%, 100% 100%;
        }

        &:nth-child(3) {
            grid-area: 1 / 2 / 4 / 5;
            --shape: 50% 0%, 100% 50%, 50% 100%, 0% 50%;
        }

        &:nth-child(4) {

        }

        &:nth-child(5) {

如果你觉得使用这种方式来定义网格太过于繁琐，那么还可以考虑使用 CSS Grid 的隐式网格来创建。

  


```CSS
.grid {
    display: grid;
    grid-template-columns: repeat(12, minmax(0, 1fr));
    grid-auto-rows: minmax(0, 1fr);
    width: min(800px, 100%);
    margin: 50px auto;
    aspect-ratio: 3 / 2;
    
    img {
        width: 100%;
        height: 100%;
        object-fit: cover;
        object-position: center;
        clip-path: polygon(var(--shape));

        &:nth-child(1) {
            grid-area: 1 / 1 / 5 / 5;
            --shape: 0% 0%, 100% 0%, 0% 100%;
        }

        &:nth-child(2) {
            grid-area: 1 / 5 / 9 / 13;
            --shape: 0% 0%, 100% 0%, 100% 100%;
        }

        &:nth-child(3) {
            grid-area: 1 / 2 / 7 / 8;
            --shape: 50% 0%, 100% 50%, 50% 100%, 0% 50%;
        }

        &:nth-child(4) {
            grid-area: 4 / 1 / 9 / 5;
            --shape: 25% 0, 100% 60%, 50% 100%, 0 100%, 0 20%;
        }

        &:nth-child(5) {
            grid-area: 4 / 3 / 9 / 13;
            --shape: 50% 0, 100% 100%, 0 100%;
        }
    }
}
```

  


上面的代码中使用 `grid-template-columns` 定义了一个 `12` 列的网格，并且使用 `grid-auto-row` 指定每行的行高为 `1fr` 。最后根据图片的位置调整 `grid-area` 属性的值。

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/f04ceab8ccd04bb5b289750081e18477~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2780&h=1758&s=892183&e=jpg&b=667d89)
