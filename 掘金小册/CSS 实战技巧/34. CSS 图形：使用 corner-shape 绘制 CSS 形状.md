## `corner-shape` 简介

  


`corner-shape` 是一个全新的 CSS 属性，用于自定义**元素角的形状**。它是 `border-radius` 的增强版——让你不仅能做出圆角（`round`），还可以轻松实现内凹角（`scoop`）、斜角（`bevel`）、切角（`notch`）和方圆形（`squircle`）等各种效果。

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/fc93c1bde764406d960e076bfc0d0664~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2552&h=1158&s=1195434&e=png&b=faf5f5)

  


这意味着，要使 `corner-shape` 生效，其必要条件是：**元素显式设置了** **`border-radius`** 。换句话说，当你设置 `border-radius` 时，默认会得到圆角效果，而 `corner-shape` 允许你将这些圆角变成其他形状。因为它就是彡来改变“角落”的“形状”的。

  


`corner-shape` 属性的语法规则非常简单：

  


```CSS
.shape {
    corner-shape: round | scoop | bevel | notch | square | squircle | <superellipse()>
}
```

  


-   `round` ：是 `corner-shape` 的默认值。角的形状为外凸的四分之一圆弧；等价于 `superellipse(1)`
-   `scoop` ：角的形状为内凹的四分之一椭圆；等价于 `superellipse(-1)`
-   `bevel` ：角的形状为一条直的对角线，既不凸也不凹；等价于 `superellipse(0)`
-   `notch` ：角的形状为内凹的 `90` 度切角；等价于 `superellipse(-infinity)`
-   `square` ： 角的形状为标准的 `90` 度直角；等价于 `superellipse(infinity)`
-   `squircle` ：角的形状为类圆角矩形（介于圆和方之间的平滑过渡曲线）；等价于 `superellipse(2)`
-   `superellipse(K)`： 角的形状为四分之一超椭圆，其中参数 `K` 控制曲线的弯曲程度，公式基于指数 `2K`

  


在使用 `corner-shape` 属性时，除了可以使用 `round` 、`scoop` 、`bevel` 、`notch` 、`square` 和 `squircle` 等关键词来定义角落形状之外，还可以使用 `superellipse(K)` 这个函数，通过调整参数 `K` ，定义出各种不同的角落形状，甚至更多变体。规范中提供了一张有关于 `superellipse()` 函数的示意图，展示了当参数 `K` 变化时角落形状是如何改变的：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/c21810dbed874e7cb56d409c8a5009c0~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2714&h=1726&s=226906&e=jpg&b=ffffff)

  


上图展示了将右上角设置为 `superellipse()` 不同值时的形状对比：从 `infinity`（直角）到 `1`（圆角），再到 `0`（斜角），`-1`（内凹圆），一直到 `-infinity`（尖锐内凹切角）。

  


使用 `corner-shape` 有一些事项需要注意：

  


-   如果未设置 `border-radius`（或设为 `0`），那么角的区域大小为零，`corner-shape` 将不生效
-   `corner-shape` 不会改变 `border-radius` 的裁剪行为，只是将角的形状变成不同风格
-   该属性定义的是**边框外边缘的形状**，边框的内边缘会跟随外边缘的路径，但这条路径不一定是超椭圆；通常保持接近恒定的距离（除非两个边交汇的边框宽度不同，此时会线性变化）
-   `corner-shape` 也会影响 `box-shadow` 的绘制方式以及元素超出部分的裁剪路径。不过这两者**不会严格跟随** **`corner-shape`** **的外轮廓**，而是以坐标轴对齐的方式对其进行缩放处理

  


来看一个简单的示例：

  


```CSS
img {
    width: 200px;
    aspect-ratio: 1;
    object-fit: cover;
    object-position: center;
    
    border-radius: calc(100%/(2 + sqrt(2))); /* 这个是必须的 */
    
    &:nth-child(1) {
        corner-shape: round;
    }
    
    &:nth-child(2) {
        corner-shape: scoop;
    }
    
    &:nth-child(3) {
        corner-shape: bevel;
    }
    
    &:nth-child(4) {
        corner-shape: notch;
    }
    
    &:nth-child(5) {
        corner-shape: squircle;
    }
    
    &:nth-child(6) {
        corner-shape: superellipse(calc(cos(30deg) - 1));
    }
}
```

## 如何使用 `corner-shape` 绘制形状

  


接下来，我们来看看 `corner-shape` 能绘制出哪些形状。

  


### 绘制八边形

  


我们先从八边形形状开始。在这里，我们将 `border-radius` 的值设置为 `--r` ，并且指定 `corner-shape` 属性的值为 `bevel` ：

  


```CSS
img {
    --r: 0%;
    --s: 300px;
    width: var(--s);
    aspect-ratio: 1;
    object-fit: cover;
    object-position: center;
    
    border-radius: var(--r);
    corner-shape: bevel;
}
```

  


请尝试着拖动示例中的滑块，动态调整变量 `--r` 的值（它的值范围是 `0% ~ 100%`），请留意这个过程中，`img` 的变化：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/6ecbf7dbcce44f8f9eb9f87cfc98e928~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1492&h=574&s=481501&e=gif&f=149&b=ffffff)

  


> Demo 地址：https://codepen.io/airen/full/YPXJxey

  


仔细观察，并不难发现，随着 `--r` 值的改变，图片的形状也会随之改变：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/db9b07f76b9d49a8abcad59c2f6744ff~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1816&s=1856481&e=jpg&b=ffffff)

  


-   当 `--r` 的值为 `0%` 时，`corner-shape: bevel` 并没有生效，图片不会有任何形状的变化
-   当 `--r` 的值为 `12%` 时（只要大于 `0`），`corner-shape: bevel` 绘制了一个八边形（不等边）
-   当 `--r` 的值为 `29%` 时，`corner-shape: bevel` 绘制了一个正八边形，其中 `29%` 是一个特殊值（`calc(100% / (2 + sqrt(2)))`），它刚好切出正八边形的角
-   当 `--r` 的值为 `50%` 时（只要大于 `50%`），`corner-shape: bevel` 维持绘制一个菱形

  


也就是说，当 `corner-shape` 的值为 `bevel` 时，且 `border-radius` 的值在大于 `0%` 和小于 `50%` 之间时，将会绘制出一个八边形。如果找到一个合适的 `border-radius` 值，就能获得一个正八边形（八边形的边长相等）。这个合适的值，我们可以通过数学和几何知识来获得：

  


```CSS
img {
    border-radius: calc(100% / (2 + sqrt(2)));
} 
```

  


这个公式计算出来的值大约是 `29.2893%` 。注意，这里使用百分比是关键，因为这样图形可以响应式缩放——当然，别忘了加上 `aspect-ratio: 1` 来保持正方形比例。

  


### 绘制菱形

  


在八边形的基础上，如果我们增加 `border-radius` 的值，当它达到 `50%` 时，就会变成一个菱形（对角线连接的形状）。事实上，只要 `border-radius` 的值大于或等于 `50%` ，`corner-shape: bevel` 绘制出来的就是一个菱形：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/e035cf408a5d4a7db3299a2e75d59049~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2552&h=1158&s=493018&e=png&b=fefdfd)

  


### 绘制五边形

  


使用 `corner-shape: bevel` 和 `border-radius` 结合，天然就能得到一个八边形，因为我们对元素四个角落分别斜切一刀，就获得了一个八边形。但是，要通过它们绘制五边形还是需要做一些额外的处理。

  


在使用 `corner-shape` 和 `border-radius` 绘制五边形之前，我们先回忆一下 `clip-path` 的 `polygon()` 是如何裁剪出一个五边形：

  


```CSS
img {
    clip-path: polygon(50% 0%, 100% 38%, 82% 100%, 18% 100%, 0% 38%);
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/946fbbd12326438e8a29f0dfa081fde1~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1074&s=407833&e=jpg&b=fbfcf7)

  


你可能会问—— `corner-shape:bevel` 如何才能绘制出五边形？**[关键因素是 `border-radius`](https://juejin.cn/book/7199571709102391328/section/7199845563389444099)**。 熟悉 `border-radius` 的同学都知道，它可以接受多个不同的值，最多的时候是八个值：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/faebacad6aec4126a491348907fc457d~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1665&h=763&s=393998&e=jpg&b=101010)

  


`corner-shape` 的前提条件就是四个角落是圆角，它们也可以是椭圆角。这意味着，我们根据 `polygon()` 每个点的坐标，就可以快速获取到每个角的圆角半径。然后将这些值应用到 `border-radius` 中。下图展示了整个过程：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/f1e804385982430bae390bffe98ba213~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2520&h=2540&s=2166661&e=jpg&b=fbfcf7)

  


将上图转换为 CSS 代码：

  


```CSS
img {
    --s: 300px;
    width: var(--s);
    aspect-ratio: 1;
    object-fit: cover;
    object-position: center;
    
    border-radius: 50% 50% 18% 18% / 38% 38% 62% 62%;
    corner-shape: bevel;
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/4ce4d9e0739d442ab063372eae3d3d5d~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2552&h=1158&s=273799&e=png&b=fdfcfc)

  


> Demo 地址：https://codepen.io/airen/full/OPVBBwo

  


我们可以引入三个 CSS 变量来控制 `border-radius`，如此一来，只需要调整它们的值，就可以获得五边形的变体：

  


```CSS
/* 引入变量前 */
img {
    border-radius: 50% 50% 18% 18% / 38% 38% 62% 62%;
}

/* 引入变量后 */
img {
    --r1: 50%;
    --r2: 18%;
    --r3: 38%;
    
    border-radius: var(--r1) calc(100% - var(--r1)) var(--r2) var(--r2) / var(--r3) var(--r3) calc(100% - var(--r3)) calc(100% - var(--r3)); 
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/c2f08cbfab424b379997ed9c12e0d37c~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1464&h=570&s=4183719&e=gif&f=209&b=fbf8f8)

  


> Demo 地址：https://codepen.io/airen/full/vEOVbMR

  


这就好比调整了 `clip-path` 的 `polygon()` 中的坐标值一样：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/0e1660897f4543d5a8fc07b6d3192c8c~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1266&h=732&s=8285941&e=gif&f=517&b=f8fbf5)

  


另外一种方式，我们可以根据三角函数来计算五边形在圆圈上点的位置，然后将这些位置对应的值应用于 `border-radius` 中。在《[CSS 图形：创建正多边形和星爆形状](https://juejin.cn/book/7395119173656903715/section/7397319151691235328#heading-1)》课程中我们详细阐述了如何使用三角函数与 `polygon()` 绘制一个正六边形，我们也可以使用同样的原理来绘制正五边形：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/953f21f1f82f4af2b7876ec6aa5bab11~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1754&s=684561&e=jpg&b=f7f6f4)

  


我们可以根据下面的公式来计算每个点的坐标值（`P1 ~ P5`）：

  


```CSS
Px = r + r * cos(angle + (i * 360deg) / sides) = 50% + 50% * cos(angle + (i * 360deg) / sides);
Py = r + r * sin(angle + (i * 360deg) / sides) = 50% + 50% * sin(angle + (i * 360deg) / sides);
```

  


为了方便，你可以使用 CSS 的变量来存储这些点的位置：

  


```CSS
img {
    --w: 300px;
    width: var(--w);
    aspect-ratio: 1;
    object-fit: cover;
    object-position: center;

    --a: -18deg;
    --s: 5;
    --p1x: calc(50% + 50% * cos(var(--a) + (1 * 360deg) / var(--s)));
    --p1y: calc(50% + 50% * sin(var(--a) + (1 * 360deg) / var(--s)));
    --p2x: calc(50% + 50% * cos(var(--a) + (2 * 360deg) / var(--s)));
    --p2y: calc(50% + 50% * sin(var(--a) + (2 * 360deg) / var(--s)));
    --p3x: calc(50% + 50% * cos(var(--a) + (3 * 360deg) / var(--s)));
    --p3y: calc(50% + 50% * sin(var(--a) + (3 * 360deg) / var(--s)));
    --p4x: calc(50% + 50% * cos(var(--a) + (4 * 360deg) / var(--s)));
    --p4y: calc(50% + 50% * sin(var(--a) + (4 * 360deg) / var(--s)));
    --p5x: calc(50% + 50% * cos(var(--a) + (5 * 360deg) / var(--s)));
    --p5y: calc(50% + 50% * sin(var(--a) + (5 * 360deg) / var(--s)));

    --r_lt_x: var(--p4x);
    --r_rt_x: var(--p4x);
    --r_rb_x: var(--p2x);
    --r_lb_x: var(--p2x);
    --r_lt_y: var(--p3y);
    --r_rt_y: var(--p3y);
    --r_rb_y: calc(100% - var(--p3y));
    --r_lb_y: calc(100% - var(--p3y));
    
    border-radius: 
        var(--r_lt_x) var(--r_rt_x) var(--r_rb_x) var(--r_lb_x) /
        var(--r_lt_y) var(--r_rt_y) var(--r_rb_y) var(--r_lb_y);
    corner-shape: bevel;
}
```

  


我们采用的是 **正五边形顶点排布在单位圆上**，通过计算两个水平坐标与两个垂直坐标的差值来近似表达宽高比：

  


```CSS
img {
    /* 结果约为 1.05146 */
    aspect-ratio: calc((cos(18deg) - cos(162deg)) / (sin(90deg) - sin(234deg)));
}
```

  


正五边形内接单位圆，从 `-90deg`（顶点朝上）开始，每隔 `72deg` 放一个点，为了覆盖最大水平跨度：取 **最左（** **`P3`** **）和最右（** **`P5`** **）的点;** 为了覆盖最大垂直跨度：取最上（`P4`） 和最下（`P2` 或 `P1`）：

  


```CSS
img {
    aspect-ratio: calc(
        (cos(var(--a)) - cos(var(--a) + 3 * 360deg / var(--s))) / (sin(var(--a) + 1 * 360deg / var(--s)) - sin(var(--a) + 4 * 360deg / var(--s)))
    );
}
```

  


看上去有点复杂，如果你不想了解其背后的原理，你可以记住它的高宽比大约为 `1.05146` ，这样我们就可以通过 `corner-shape: bevel` 和 `border-radius` 绘制出一个正五边形：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/b312f03616014d61a0308ced6012dc31~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2552&h=1158&s=448051&e=png&b=ffffff)

  


> Demo 地址：https://codepen.io/airen/full/azORXgp

  


使用这种方法绘制五边形时，如果五边形点顶位置发生变化，那么 `border-radius` 的值也需要做出相应的调整，否则无法获得所需要的图形。

  


### 绘制六边形

  


我们可以采用绘制五边形的方式来绘制六边形，唯一的区别就是 `border-radius` 属性的值不同——只需要给 `border-radius` 的 `x` 轴和 `y` 轴设置不同的半径值，就可以快速获得一个六边形。

  


我们先从菱形（`border-radius` 的值为 `50%`）开始，然后减少它的水平（`x`）或垂直（`y`）圆角半径：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/93830ede5b7e4994820a657b5fc5b5ea~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=2121&s=1328575&e=jpg&b=fefefe)

  


转换为 CSS 代码，大致如下：

  


```CSS
img {
    --w: 300px;
    width: var(--w);
    aspect-ratio: 1;
    object-fit: cover;
    object-position: center;

    border-radius: var(--rx, 50%) / var(--ry, 25%);
    corner-shape: bevel;
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/bfd17eb6600148129746125afb6bd0b1~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1490&h=526&s=5233530&e=gif&f=284&b=fefdfd)

  


> Demo 地址：https://codepen.io/airen/full/azORxva

  


这应该是绘制六边形最简单的方式了。需要注意的是，使用这种方式绘制六边形，我们需要确保 `--rx` 和 `--ry` 其中一个是 `50%` 的值，因为当它们都小于 `50%` 时，会得到一个八边形；当等于或大于 `50%` 时会得到一个菱形。另外，只要它们当中的一个值为 `0%` 时，我们得到的是一个矩形。

  


上面示例中的图片宽高比是 `1` ，我们可以调整它的值，使六边形的的比例看起来更正确：

  


```CSS
img {
    border-radius: 50% / 25%;
    corner-shape: bevel;
    aspect-ratio: cos(30deg);
}

/* 或者 */
img {
    border-radius: 25% / 50%;
    corner-shape: bevel;
    aspect-ratio: 1/cos(30deg);
}
```

  


-   `cos(36deg) ≈ 0.866` ，表示图片的宽高比为 `0.866` ，即宽是高的 `86.6%`
-   `1 / cos(30deg) ≈ 1.1547` ，表示图片的宽高比是 `1.1547` ，即宽是高的 `115.4%`

  


这两种写法可以控制六边形的朝向是平头还是尖头：

  


-   `cos(30deg)`：六边形是尖头，适合垂直平铺
-   `1 / cos(30deg)` ：六边形是平头，适合水平平铺

  


具体的效果如下：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/897d18595f844bce9c67d91f081f1a03~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2552&h=1158&s=961011&e=png&b=fbf8f8)

  


> Demo 地址：https://codepen.io/airen/full/EajdJWO

### 绘制三角形

  


沿用相同的逻辑，我们可以创建大多数常见的形状，其中也包括三角形。我们依赖可以从菱形开始，然后调整水平或垂直的圆角半径，比如像下面这样：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/e7ade9ecd233420e93004813c0ebcc59~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=2121&s=892618&e=jpg&b=ffffff)

  


乍一看这个例子可能会有点难理解，因为它不像之前的图形那样四个角都有斜线。但别忘了，`border-radius` 可以设为 `0`，这样就可以禁用某个角的圆角效果。

  


```CSS
img {
    --s: 300px;
    width: var(--s);
    object-fit: cover;
    object-position: center;
    border-radius: 50% / 100% 100% 0% 0%;
    corner-shape: bevel;
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/5fd19f849c5d458ca30abfc3d6a50302~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2552&h=1158&s=367743&e=png&b=ffffff)

  


> Demo 地址：https://codepen.io/airen/full/MYwzYaE

  


从这个基础出发，我们可以轻松地通过不同的圆角组合和 `aspect-ratio` 值，创建各种类型的三角形：

  


```CSS
img {
    width: 100%;
    object-fit: cover;
    object-position: center;
    border-radius: var(--radius, 50%);
    aspect-ratio: var(--ratio, 1);
    corner-shape: bevel;

    &:nth-child(1) {
        --radius: 50% / 100% 100% 0% 0%;
        --ratio: 1 / cos(30deg);
    }

    &:nth-child(2) {
        --radius: 50% / 0% 0% 100% 100%;
        --ratio: 1 / cos(30deg);
    }

    &:nth-child(3) {
        --radius: 0% 100% 100% 0% / 50%;
        --ratio: cos(30deg);
    }

    &:nth-child(4) {
        --radius: 100% 0% 0% 100% / 50%;
        --ratio: cos(30deg);
    }

    &:nth-child(5) {
        --radius: 0% 100% 0% 0%;
    }

    &:nth-child(6) {
        --radius: 100% 0% 0% 0%;
    }

    &:nth-child(7) {
        --radius: 0% 0% 0% 100%;
    }

    &:nth-child(8) {
        --radius: 0% 0% 100% 0%;
    }
}
```

### 斜边

  


我们之前创建的所有图形都依赖于百分比，但别忘了，`border-radius` 也支持使用具体长度单位（比如 `px` 、`em` 和 `rem` 等）。这意味着，我们可以制作尺寸可变但形状保持不变的元素。比如下面这个带有斜边的示例，不管元素宽度如何变化，斜边都会保持相同的大小：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/b630d962063d4b9ab62076fe7da2a4bc~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1051&s=667039&e=jpg&b=ffffff)

  


代码非常简单，如下所示：

  


```CSS
img {
    display: block;
    width: 100%;
    height: 100%;
    object-fit: cover;
    object-position: center;
    
    border-top-right-radius: 5rem 100%;
    corner-shape: bevel;
}
```

  


在这种情况下，不需要使用简写的 `border-radius` ，因为我们只关心右上角的圆角。至于这个值，我认为它已经很直观了。需要注意的是，当使用单个方向的长写属性（比如 `border-top-right-radius`）时，不需要用 `/` 来分隔水平和垂直的半径值。

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/47ef571964e948a99eecc9b256e3167d~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1268&h=568&s=16138378&e=gif&f=178&b=ffffff)

## 为什么需要 `corner-shape`

  


正如你所看到的，结合 `corner-shape` 和 `border-radius` ，我们可以以最简便的方式创建出 **[三角形](https://juejin.cn/book/7395119173656903715/section/7397319555476881460)**、**[多边形](https://juejin.cn/book/7395119173656903715/section/7397319151691235328)**（比如五边形、六边形，尤其是八边形）、**[角度切割](https://juejin.cn/book/7395119173656903715/section/7400975207551139875#heading-0)**和**[圆角切割](https://juejin.cn/book/7395119173656903715/section/7400975207551139875#heading-1)** 等形状。你可能会问：这种方法比我们之前所介绍的方法（比如 `clip-path` 和 `mask` 绘制图形）更好吗？

  


答案是肯定的。首先，这种语法比 `clip-path` 或 `mask` 都要更简单，还可以提高代码的可读性，因为我们需要处理的数值更少。但最重要的优势是：我们可以给这些形状添加边框：

  


```CSS
img {
    width: 200px;
    aspect-ratio: 1;
    object-fit: cover;
    object-position: center;
    border-radius: calc(100%/(2 + sqrt(2)));
    border: .5em solid #0000;
    background: linear-gradient(-45deg, #000, #4158d0, #c850c0, #ffcc70, #f50, #fff) border-box;
    
    &:nth-child(1) {
        corner-shape: round;
    }
    
    &:nth-child(2) {
        corner-shape: scoop;
    }
    
    &:nth-child(3) {
        corner-shape: bevel;
    }
    
    &:nth-child(4) {
        corner-shape: notch;
    }
    
    &:nth-child(5) {
        corner-shape: squircle;
    }
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/97272470c6a94bce8d3078b98521938b~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2552&h=1158&s=1216050&e=png&b=fdfbfb)

  


> Demo 地址：https://codepen.io/airen/full/qEdJrVe

  


在此之前，要给自定义形状添加边框，简单就是 Web 开发者的噩梦，而 `corner-shape` 让这变得轻而易举！

  


这其实也很合理，因为默认情况下，当我们使用 `border-radius` 时，边框（`border`）和其他装饰（比如 `box-shadow` 或 `outline`）都会跟随圆角走。即使我们改变了角的形状，这种行为依然成立。需要注意的是，到目前为止，如果元素应用了 `box-shadow` 或 `outline` 还会存在一些瑕疵，但我相信这些瑕疵最终都能被解决掉。有兴趣的同学不妨一试！

  


## 小结

  


在这节课中，我们所展示的大部分案例都是围绕着 `corner-shape: bevel` 和 `border-radius` 展开的。正如你所看到的，通过它们可以轻松创建许多不同的形状。我们所需要做的只是玩转大家熟悉的 `border-radius` 属性。更别提我们还能够为这些形状添加边框和阴影效果，这相比使用 `clip-path` 或 `mask` 创建形状来说，真的是一个巨大的突破！

  


有一点需要知道的是，我们还可以给 `corner-shape` 设置多个不同的值（为每个角落设置不同的形状），并且结合 `border-radius` 创建出更为复杂的形状。比如：

  


```CSS
img {
    corner-shape: notch bevel bevel notch;
    border-radius: 60% 40% 40% 60% / 25% 50% 50% 25%;
}
```

  


在这里，我们结合了 `bevel` 和 `notch` 创建了一个箭头形状：
