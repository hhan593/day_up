## 准备工作

  


接下来，我将向大家展示使用现代 CSS 技巧来创建多种不同形状的丝带，并且代码尽量简洁。事实上，创建丝带形状所需要的 HTML 结构非常简单：

  


```HTML
<div class="ribbon">CSS Ribbon</div>
```

  


我们只需要一个元素就能创建出各种丝带形状。另外，我们的目标是提供可轻松控制丝带形状的 CSS 代码。你可以调整文本、调整屏幕大小，改变字体属性，丝带形状始终会完美地适应其内容。

  


## 从最简单的丝带形状开始

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/d63eadd05da04251836042dd6d7320a1~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1316&s=296348&e=jpg&b=ffffff)

  


这应该是丝带形状中最简单的一种（如上图所示）。我们以上图中的第一个丝带形状为例，元素右侧有一个凹口，我们把这个凹口的大小用 CSS 自定义属性来定义，比如 `--c` 。其次，这个丝带形状是一个围绕着水平轴上下对称的形状。

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/398d964d3cfb4846bbe7ac81ab199b0e~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=844&s=126518&e=jpg&b=ffffff)

  


现在，我们可以使用 `clip-path` 的 `polygon()` 函数对元素进行裁剪，并且裁剪出我们所需要的丝带形状：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/a5748232be964f888327f00f3f5bd162~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=844&s=217444&e=jpg&b=ffffff)

  


对应的 CSS 如下：

  


```CSS
.ribbon {
    --bg: #FA6900;
    --color: color-contrast(var(--bg) vs #fff, #aefeaf, #333);
    --c: 30px; /* 丝带凹口大小 */
    --shape: 0 0, 100% 0, calc(100% - var(--c)) 50%, 100% 100%, 0 100%;
    clip-path: polygon(var(--shape));
    
    /* 基本样式 */
    background: var(--bg);
    color: var(--color);
    line-height: 2;
    padding-inline: .5em;
    border-right: var(--c) solid #0000;
    width: fit-content;
}
```

  


这个时候你只需要调整 `--bg` 和 `--c` 两个 CSS 变量的值，你就可以获得不同的丝带形状：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/b34c2f044a1c4d919b8330e1fc9a591e~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1142&h=618&s=3352004&e=gif&f=258&b=465767)

  


> Demo 地址：https://codepen.io/airen/full/LYKWvWO

  


注意，这里有几个小技巧：

  


-   设置了一个以丝带凹口大小一样的透明右边框（`border-right: var(--c) solid #ccc`），它有效的防止了 `clip-path` 切到文本
-   [元素的宽度设置一个内在尺寸 `width: fit-content`](https://juejin.cn/book/7161370789680250917/section/7161625601713897508) ，这意味着盒子会使用可用的空间，但永远不会超过 `max-width` 。这个不是必须的，你可根据实际情况来决定
-   [文本颜色使用 `color-contrast()` 函数](https://juejin.cn/book/7223230325122400288/section/7239644346935410720)，它将根据背景颜色（`--bg`）的变化来选择合适的文本颜色（会从 `#fff` 、 `#aefeaf`和 `#333` 中选择最佳值）

有趣的是，有些丝带形状还可以使用 [CSS 的锥形渐变 `conic-gradient`](https://juejin.cn/book/7223230325122400288/section/7259668771856941111) 和边框图像 `border-image` 来实现。下图展示这种方法的实现过程：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/eaecd444a2ba4dd9a940cdac292087aa~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=2514&s=715303&e=jpg&b=ffffff)

  


正如你所看到的，如果仅仅是绘制类似上图这样简单的丝带形状，我们使用 [CSS 的 `coinc-gradient`](https://juejin.cn/book/7223230325122400288/section/7259668771856941111) 就可以实现：

  


```CSS
.ribbon {
    --bg: #fa6900;
    --color: color-contrast(var(--bg) vs #fff, #aefeaf, #333);

    /* 基本样式 */
    color: var(--color);
    line-height: 2;
    width: fit-content;
    font-size: 1.5rem;
    background: conic-gradient(from 45deg at calc(100% - .5lh),#0000 25%, var(--bg) 0);
    padding-left: 0.5em;
    padding-right: calc(0.5em + .5lh);
}
```

  


调整代码中 `coinc-gradient` 的开始角度（紧跟`from` 关键词后值）和圆心位置（紧跟 `at` 关键词后的值）以及透明颜色（`#0000`）在渐变轴上的位置，可以创建不同丝带变体：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/95df714fa6244f6c9795bf0d866e6ba2~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1612&h=764&s=4986263&e=gif&f=182&b=455466)

  


> Demo 地址：https://codepen.io/airen/full/bGPWELP

  


注意，如果丝带形状带有两个凹口，则需要使用两个 `coinc-gradient`，并且要使用 `background-size` 调整背景尺寸和`background-position` 调整位置：

  


```CSS
.ribbon {
    background: 
        /* 凹口在右侧 */
        conic-gradient(from 45deg at calc(100% - .5lh),#0000 25%,var(--bg) 0),
        /* 凹口在左侧 */
        conic-gradient(from 225deg at .5lh,#0000 25%,var(--bg) 0);
      background-size: calc(100% - .5lh) 100%; /* 背景图片尺寸*/
      background-position:100% 0, 0 0; /* 第一个居右对齐，第二个居左对齐 */
      background-repeat: no-repeat;
      padding-inline: calc(0.5em + .5lh);
}
```

  


你可能已经发现了，在这个示例中，我们将透明边框的样式取消掉了，改为了 `padding-left` 或 `padding-right` ，需要知道的是，丝带形状凹口位于元素的哪一侧，则相应的 `padding` 内需要加上凹口的尺寸，这里是 `0.5lh` 。因此，你会发现示例中，`padding-left` 或 `padding-right` 的值为 `calc(0.5em + 0.5lh)` 。

  


当然，你也可以继续使用一个透明的边框，只不过透明边框的尺寸是 `0.5lh` 。如此一来，元素的左右内距可以设置为相同的值，例如 `0.5em` 。不过，要是这样设置元素的基础样式，就需要将 `background-origin` 和 `backround-clip` 属性的值设置为 `border-box` :

  


```CSS
/* 单侧凹口 */
.ribbon {
    --bg: #fa6900;
    --color: color-contrast(var(--bg) vs #fff, #aefeaf, #333);
    --b: 0.5lh; /* 边框大小，等于丝带凹口尺寸*/

    /* 基本样式 */
    color: var(--color);
    line-height: 2;
    padding-inline: 0.5em;
    width: fit-content;
    
    /* 丝带 */
    background-image: conic-gradient(from 45deg at calc(100% - .5lh),#0000 25%,var(--bg) 0);
    background-origin: border-box;
    background-clip: border-box;
    background-repeat: no-repeat;
    border-right: var(--b) solid #0000;
}

/* 双侧凹口 */
.ribbon {
    --bg: #fa6900;
    --color: color-contrast(var(--bg) vs #fff, #aefeaf, #333);
    --b: 0.5lh; /* 边框大小，等于丝带凹口尺寸*/

    /* 基本样式 */
    color: var(--color);
    line-height: 2;
    padding-inline: 0.5em;
    width: fit-content;
    
    /* 丝带 */
    background-image: 
        conic-gradient(from 45deg at calc(100% - .5lh),#0000 25%,var(--bg) 0),
        conic-gradient(from 225deg at .5lh,#0000 25%,var(--bg) 0);
    background-size: calc(100% - .5lh) 100%;
    background-position:100% 0, 0 0;
    background-origin: border-box;
    background-clip: border-box;
    background-repeat: no-repeat;
    border-inline: var(--b) solid #0000;
}
```

**温馨提示**：CSS 的锥形渐变 `conic-gradient` 是一个非常强大的特性，它除了用于创建渐变背景之外，还可以用于创建不同的形状。如果你对它了解的不多，个人强烈建议你移步阅读《[现代 CSS](https://s.juejin.cn/ds/iMcXTVf2/)》小册中的《[CSS 的锥形渐变](https://juejin.cn/book/7223230325122400288/section/7259668771856941111)》！

  


除此之外，我们还应用到了一个新单位，即 `lh` 。你可能会想知道 `lh` 单位是怎么回事。它是 [CSS 字体相对单位中的一个](https://juejin.cn/book/7223230325122400288/section/7249357892611440700)，相对于元素的 `line-height` 值计算。由于我们使用的是单行文本，`line-height` 属性控制了元素的高度，所以元素的高度等于 `1lh` ，在这个示例中，它等于 `2 × 1.5 × 16 = 48px` （因为 `line-height` 的值为 `2` ，`font-size` 的值是 `1.5rem` ， `<html>` 元素的 `font-size` 是默认值 `16px`） ，这非常有用。另外，丝带的凹口是一个等腰三角形，因此其大小等于元素高度的一半，即 `0.5lh` 。

  


我想你还有另一个困惑？既然 `conic-gradient` 就可以创建出丝带形状，为什么还要使用 `border-image` 呢？这个是有原委，因为我们将丝带形状应用于 Web 的元素上时，它可能要延伸到浏览视窗的边缘，例如下图的效果：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/4e2c834aef9c470884bfbb5468aa8394~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1246&s=752232&e=jpg&b=516371)

  


> Demo 地址：https://codepen.io/airen/full/QWXvyXy

  


正如之前的步骤图所示：

  


-   1️⃣：使用 `coinc-gradient` 创建一个丝带形状的图片
-   2️⃣：将 `conic-gradient` 创建的图片作为 `border-image-source` 的值

  


为了达到想要的效果，左侧无限延伸到浏览器视窗的左边缘，或右侧无限延伸到浏览器视窗的右边缘，我们将使用 `border-image-outset` 来调整相应侧的延伸值，将该值设置为一个较大的值，比如 `100vw` 。具体的代码如下所示：

  


```CSS
.ribbon {
    --bg: #fa6900;
    --color: color-contrast(var(--bg) vs #fff, #aefeaf, #333);

    /* 基本样式 */
    color: var(--color);
    line-height: 2;
    width: fit-content;
    transition: all 0.2s;

    /* 左侧无限延伸 */
    &.ribbon--start {
        border-image: conic-gradient(from 45deg at calc(100% - 0.5lh),#0000 25%,var(--bg) 0) fill 0 //0 0 0 100vw;
        padding-right: calc(0.5em + 0.5lh);
        justify-self: start;
    }

    /* 右侧无限延伸 */
    &.ribbon--end {
        border-image: conic-gradient(from 225deg at 0.5lh, #0000 25%, var(--bg) 0) fill 0 //0 100vw 0 0;
        padding-left: calc(0.5em + 0.5lh);
        justify-self: end;
    }
}
```

  


注意，示例中的 `border-image` 属性使用了简写方式，因此你会看到代码中有两个反斜杠（`//`），例如 `fill 0 //0 100vw 0 0` 。这是 `border-image` 的一种简写方式：

  


```CSS
.element {
    /* source | slice | width | outset | repeat */
    border-image: url("/images/border.png") 27 23 / 50px 30px / 1rem round space;
}

/* 等同于 */
.element {
    border-image-source: url("/images/border.png");
    border-image-slice: 27 23;
    border-image-width: 50px 30px;
    border-image-outset: 1rem;
    border-image-repeat: round space;
}
```

  


回到我们示例中的 `border-image`:

  


```CSS
.ribbon {
    border-image: conic-gradient(from 45deg at calc(100% - 0.5lh),#0000 25%,var(--bg) 0) fill 0 //0 0 0 100vw;
}

/* 等同于 */
.ribbon {
    border-image-source: conic-gradient(from 45deg at calc(100% - 0.5lh),#0000 25%,var(--bg) 0);
    border-image-slice: fill 0;
    border-image-width: 1; /* 初始值 */
    border-image-outset: 0 0 0 100vw;
    border-image-repeat: stretch; /* 初始值 */
}
```

  


如果你觉得代码中的两个反斜杠不好理解，你可以将属性的值设置为初始值：

  


```CSS
.ribbon {
    border-image: conic-gradient(from 45deg at calc(100% - 0.5lh),#0000 25%,var(--bg) 0) fill 0 //0 0 0 100vw;
    
    /* 等同于 */
    border-image: conic-gradient(from 45deg at calc(100% - 0.5lh),#0000 25%,var(--bg) 0) fill 0 / 1 /0 0 0 100vw stretch;
}
```

  


**切记，千万不能直接删除另一个反斜杠，否则会造成** **`border-image`** **语法错误，浏览器将会直接忽略** **`border-image`** **的设置**。

```CSS
.ribbon {
    --d: 0.5em;

    --color: color-contrast(var(--bg) vs #fff, #aefeaf, #333);
```

```CSS
    &:nth-child(2) {
      background: 
          radial-gradient(0.2em 50% at left, #000a, #0000) border-box,
          conic-gradient(from 45deg at calc(100% - var(--c)),#0000 25%,var(--bg) 0) padding-box;
      padding-left: 0.5em;
      padding-right: calc(var(--c) + 0.5em);
    }
}
```

## 侧面丝带形状

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/b513ff7b7a4e4b898856211992bd45e4~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=830&s=336244&e=jpg&b=ececec)

  


在课程开头的时候，我们提到过，丝带形状常见的方式之一就是放置在卡片的角落上，并且该丝带在侧面会有一个深色三角形或半圆形状。我们把这种丝带形状称为“侧面丝带形状”（如上图所示）。

  


接下来，我们来一起探讨这种丝带形状又应该如何实现。实现方案很多，但这里我们只探讨现代 CSS 技巧，并且仅在一个元素上实现。例如下图这个侧面丝带形状：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/c16fbb05da9b40f887cd1a07dedeb01d~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=830&s=176210&e=jpg&b=ffffff)

  


与之前方式一样，将丝带形状凹口的距离定义为 `c` （即 `--c`），控制折叠的部分（上图深蓝色）的尺寸为 `f` （即 `--f`）。

  


```CSS
.ribbon {
    --c: 0.5lh; /* 控制凹口尺寸 */
    --f: 10px;  /* 控制折叠部分尺寸 */
}
```

  


从上图中不难发现，侧带丝带形状的折叠部分的颜色要比丝带的背景色更深。在这里我们可以借助 [CSS 的 `color-mix()` 函数基于丝色的背景颜色来控制折叠区域的背景颜色](https://juejin.cn/book/7223230325122400288/section/7237288025221234744)：

  


```CSS
.ribbon {
    --c: 0.5lh; /* 控制凹口尺寸 */
    --f: 30px;  /* 控制折叠部分尺寸 */
    --bg: #3FB8AF; /* 控制丝带背景色 */
    --fc: color-mix(in srgb, var(--bg), #000 35%); /* 控制丝带折叠区域的背景颜色 */
    --color: color-contrast(var(--bg) vs #fff, #aefeaf, #333); /* 控制文本颜色 */
}
```

  


接下来就是使用 CSS 创建该丝带形状了，整个过程如下图所示：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/b0001379c8d34231bc538320b7c48562~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=2197&s=868897&e=jpg&b=ffffff)

  


上图展示了两种方案，它们不同之处就是控制丝带形状和折叠区域大小与颜色有所不同，左侧给元素添加了一个底边框，边框的颜色为 `--fc` ，厚度为 `--f` ；右侧的折叠区是一个背景图片。具本的代码如下：

  


```CSS
.ribbon {
    --c: 0.5lh; /* 控制凹口尺寸 */
    --f: 10px;  /* 控制折叠部分尺寸 */
    --bg: #3FB8AF; /* 控制丝带背景色 */
    --fc: color-mix(in srgb, var(--bg), #000 35%); /* 控制丝带折叠区域的背景颜色 */
    --color: color-contrast(var(--bg) vs #fff, #aefeaf, #333); /* 控制文本颜色 */
    
    /* 基本样式 */
    position: absolute;
    z-index: 2;
    line-height: 2;
    width: fit-content;
    font-size: 1.5rem;
    color: var(--color);
    top: 1.5rem;
    
    /* 方案一：折叠区使用的是 border */  
    &.ribbon--right {
        right: calc(-1 * var(--f)); /* 位于右侧 */
        padding-right: .5em;
        padding-left: calc(.5em + var(--c));
        background: var(--bg);
        border-bottom: var(--f) solid var(--fc); /* 折叠区 */
        
        /* 凹口在左侧，折叠区在右侧 */
        clip-path: polygon(
            0 0, 
            100% 0,
            100% calc(100% - var(--f)),
            calc(100% - var(--f)) 100%,
            calc(100% - var(--f)) calc(100% - var(--f)),
            0 calc(100% - var(--f)),
            var(--c) calc((100% - var(--f)) / 2) 
        );
    }
    
    /* 方案二：折叠区使用的是 background-image */
    &.ribbon--left {
        left: calc(-1 * var(--f)); /* 位于左侧 */
        padding-left: .5em;
        padding-right: calc(.5em + var(--c));
        background: 
            var(--bg) /* 丝带主背景 */
            linear-gradient(var(--fc),var(--fc)) left bottom / 100% var(--f) no-repeat; /* 折叠区 */
        height: calc(1lh + var(--f)); /* 整个丝带高度为 1lh 加 --f */
        clip-path: polygon(
            0 0,
            100% 0,
            calc(100% - var(--c)) .5lh,
            100% calc(100% - var(--f)),
            var(--f) calc(100% - var(--f)),
            var(--f) 100%,
            0 calc(100% - var(--f))
        )
    }
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/6d5d167b1ed84bc38e0d2666e8b3dd8f~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1176&h=682&s=10173086&e=gif&f=259&b=f4f4f4)

  


> Demo 地址：https://codepen.io/airen/full/PormQrG

  


请注意方案二，整个元素的高度应该是 `calc(1lh + var(--f))`，另外需要控制好 `linear-gradient` （它创建了折叠区）的位置和尺寸（使用 `backround-position` 和 `background-size`）来控制。

  


按照类似的方式，你就可以创建出不同变体的侧面丝带形状。同样的，给大家留一个小作业，如果丝带形状如果使用锥形渐变（`conic-gradient`）来创建，那么应该如何编写 CSS 呢？另一个挑战是，使用你所掌握的现代 CSS 特性，创建下图所示的侧面丝带形状：

## 角落丝带形状

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/468bf68ae5664e7185a7c08ff8022d3e~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1056&s=429674&e=jpg&b=eae8e8)

  


角落丝带形状对于 Web 开发者来说，应该不会感到陌生，因为它常出现在 Github 的“Fork me”按钮上。

  


这种丝带形状比之前所介绍的丝带形状要简单一些，因为我们不需要考虑丝带的凹口方向与大小，只需要通过一个变量 `--f` 来控制折叠区的大小和形状。

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/7bce67c9b988421495211c8d827feb10~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=2589&s=820217&e=jpg&b=ffffff)

  


我们首先定义背景颜色（`--bg`），并在底部设置一个厚度为 `--f` 的边框，而且它的颜色是 `--bg` 和 `35%` 的黑色 `#000` 的混合色（通过 [CSS 的 `color-mix()`](https://juejin.cn/book/7223230325122400288/section/7237288025221234744) 可以很容易得到），以获得与主色不同的色调，并模拟丝带形状的折叠部分。与此同时，[使用 `color-contrast()` 函数基于背景色 --bg 来设置文本颜色](https://juejin.cn/book/7223230325122400288/section/7239644346935410720)：

  


```CSS
.ribbon {
    --f: 10px; /* 控制折叠部分尺寸 */
    --bg: #3fb8af; /* 控制丝带背景色 */
    --fc: color-mix(in srgb, var(--bg),#000 35%); /* 控制丝带折叠区域的背景颜色 */
    --color: color-contrast(var(--bg) vs #fff,#aefeaf,#333); /* 控制文本颜色 */

    /* 基本样式 */
    position: absolute;
    z-index: 2;
    line-height: 2;
    width: fit-content;
    font-size: 1.5rem;
    color: var(--color);
    padding-inline: 1lh;
    background: var(--bg); /* 丝带主色 */
    border-bottom: var(--f) solid var(--fc); /* 丝带折叠部分 */
}
```

  


然后根据上图中所示，给 `clip-path` 的 `polygon()` 应用正确的坐标点：

  


```CSS
.ribbon {
    clip-path: polygon(
        100% calc(100% - var(--f)), /* P1 */
        100% 100%, /* P2 */
        calc(100% - var(--f)) calc(100% - var(--f)), /* P3 */
        var(--f) calc(100% - var(--f)), /* P4 */
        0 100%, /* P5 */
        0 calc(100% - var(--f)), /* P6 */
        999px calc(100% - var(--f) - 999px), /* P7 */
        calc(100% - 999px) calc(100% - var(--f) - 999px) /* P8 */
    );
}
```

  


大多数点应该很容易理解，但有两个稍微复杂的点：`999px calc(100% - var(--f) - 999px)` 和 `calc(100% - 999px) calc(100% - var(--f) - 999px)`。因为，我们必须在每侧剪切一个等腰三角形的形状，但要做到这一点，我们需要知道元素的高度。由于我们并不希望显式给元素设置一个固定的尺寸，所以无法获得这个信息。

  


庆幸的是，有一个小技巧，那就是依赖一个较大值，比如代码中的 `999px` ，该值应大于我们可能遇到的任何高度。注意，如果丝带文本内容仅有一行，你可以考虑使用 `1lh` 来替代 `999px` 。

  


到这一步，我们只是使用 CSS 创建出了丝带形状，但要将丝带形状放置到具体的位置，还需要使用 CSS 的变换特性，对丝带形状进行旋转和位移。

  


```CSS
/* 位于左上角 */
.ribbon--top-left {
    transform-origin: 100% 100%;
    translate: calc((cos(45deg) - 1) * 100%) -100%;
    rotate: -45deg;
    left: 0;
    top: 0;
}

/* 位于右上角 */
.ribbon--top-right {
    top: 0;
    right: 0;
    transform-origin: 0 100%;
    translate: calc((1 - cos(45deg)) * 100%) -100%;
    rotate: 45deg;
}

/* 位于右下角 */
.ribbon--bottom-right {
    bottom: 0;
    right: 0;
    transform-origin: 100% 100%;
    rotate: 135deg;
    translate: -70% 0%;
    translate: calc(-1 * cos(45deg) * 100%) 0%;
}

/* 位于左上角 */
.ribbon--bottom-left {
    left: 0;
    bottom: 0;
    transform-origin: 0% 100%;
    rotate: 225deg;
    translate: calc(cos(45deg) * 100%) 0%;
}
```

## 全幅丝带形状

  


应用在 Web 上的一些丝带形状，它有可能会横跨整个元素，例如卡片或浏览器视窗宽度。我们把这类丝带形状称为“全幅丝带形状”：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/bcddf65b03e242389f059e1ed53c6c81~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=973&s=395225&e=jpg&b=ececec)

  


我们以下图这个全幅丝带为例，看看这类丝带形状又应该如何实现：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/1500bbb1095a41979b68ca6892e4e699~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1056&s=159287&e=jpg&b=ffffff)

  


与之前创建丝带形状方式一样，我们先使用 CSS 自定义属性定义控制丝带形状的关键变量：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/f56688aebd4a4383bc46109af10ddb24~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1056&s=240939&e=jpg&b=38353e)

  


```CSS
.ribbon {
    /* 控制丝带形状的变量 */
    --s: 50px; /* 控制丝带大小 */
    --d: 20px; /* 控制深度 */
    --c: 25px; /* 控制凹口 */
    
    /* 控制丝带颜色 */
    --bg: #BD4144; /* 主背景色 */
    --dc: color-mix(in srgb,var(--bg),#000 45%); /* 控制丝带深度区域（折叠）颜色 */
    --sc: color-mix(in srgb,var(--bg),#000 25%); /* 控制丝带颜色 */
    --color: color-contrast(var(--bg) vs #fff,#aefeaf,#333); /* 控制文本颜色 */
}
```

  


接着，我们使用 CSS 渐变来创建形状：

  


```CSS
.ribbon {
    background: 
        conic-gradient(at left var(--s) bottom var(--d),#0000 25%,var(--dc) 0 37.5%,var(--sc) 0)0 /50% 100% no-repeat,
        conic-gradient(at right var(--s) bottom var(--d),var(--sc) 62.5%,var(--dc) 0 75%,#0000 0) 100%/50% 100% no-repeat;
    background-color: var(--bg);
}
```

  


每个渐变（`conic-gradient`）覆盖一半区域，并由三种不同的颜色组成。一种是透明色（`#0000`），另两种混合色 `--sc` 和 `--dc` (主色 `--bg` 和 `#000` 按照不同的比例混合得到的)。通过在渐变后面定义背景颜色，我们就为丝带创建了不同的色调，你只需要更新主色 `--bg` 就可以为丝带渲染不同的颜色。

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/639c65868f1d49feaae868990ad297a4~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1032&h=666&s=5119967&e=gif&f=253&b=f5f5f5)

  


最后一步是应用 `clip-path` 。我使用下图来说明 `polygon()` 函数坐标点：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/312782ced4ea45d395f655d0e2721647~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=2818&s=953730&e=jpg&b=38353e)

  


乍一看有点复杂，因为所需的坐标点数量很多。但你只需要按照上图所示，将每一个点的坐标按顺序填入 `polygon()` 函数，就会变得很容易。而且，我们要创建的丝带形状是一个围绕元素中心的 `Y` 轴对称（左右对称）。[按照之前的课程所说](https://juejin.cn/book/7395119173656903715/section/7395172754053890063)，`Y` 轴对称的形状的对称点，其 `x` 轴的总和是 `100%` ，`y` 轴的值相等。如此一来，我们只需要关注 `7` 个点，这比之前的 `14` 个点少一半。以 `P1` 点为例，它的坐标是 `(0, d)` ，与其对称的是 `P6` 点，所以 `P6` 的坐标点是 `(100% - 0, d) = (100%, d)` 。依此类推，我们就可以快速获得创建丝带形状所有点的坐标：

  


```
P1  = (0, d)            👉  P6  = (100% - 0, d) = (100%, d)
P2  = (s, d)            👉  P5  = (100% - s, d)
P3  = (s, 0)            👉  P4  = (100% - s, 0)
P14 = (c, 50% + d / 2)  👉  P7  = (100% - c, 50% + d / 2)
P13 = (0, 100%)         👉  P8  = (100% - 0, 100%) = (100%, 100%)
P12 = (s + d, 100%)     👉  P9  = (100% - (s + d), 100%) = (100% - s - d, 100%)
P11 = (s + d, 100% - d) 👉  P10 = (100% - (s + d), 100% - d) = (100% - s -d, 100% - d)
```

  


对应的 CSS 代码如下：

  


```CSS
.ribbon{
    clip-path: polygon(
        0 var(--d), /* P1 */
        var(--s) var(--d), /* P2 */
        var(--s) 0, /* P3 */
        calc(100% - var(--s)) 0, /* P4 */
        calc(100% - var(--s)) var(--d), /* P5 */
        100% var(--d), /* P6 */
        calc(100% - var(--c)) calc(50% + var(--d) / 2), /* P7 */
        100% 100%, /* P8 */
        calc(100% - var(--s) - var(--d)) 100%, /* P9 */
        calc(100% - var(--s) - var(--d)) calc(100% - var(--d)), /* P10 */
        calc(var(--s) + var(--d)) calc(100% - var(--d)), /* P11 */
        calc(var(--s) + var(--d)) 100%, /* P12 */
        0 100%, /* P13 */
        var(--c) calc(50% + var(--d) / 2) /* P14 */
    )
}
```

  


完整的 CSS 代码如下所示：

  


```CSS
.ribbon {
    /* 控制丝带形状的变量 */
    --s: 50px; /* 控制丝带大小 */
    --d: 20px; /* 控制深度 */
    --c: 25px; /* 控制凹口 */

    /* 控制丝带颜色 */
    --bg: #BD4144; /* 主背景色 */
    --dc: color-mix(in srgb, var(--bg),#000 45%); /* 控制丝带深度区域（折叠）颜色 */
    --sc: color-mix(in srgb, var(--bg), #000 25%); /* 控制丝带颜色 */
    --color: color-contrast(var(--bg) vs #fff,#aefeaf,#333); /* 控制文本颜色 */

    /* 基本样式 */
    position: absolute;
    z-index: 2;
    line-height: 2;
    text-align: center;
    font-size: 1.5rem;

    top: 1.5rem;
    left: calc(-1 * (var(--s) + var(--d)));
    right: calc(-1 * (var(--s) + var(--d)));
    color: var(--color);
    padding: 0 calc(var(--s) + 0.5em) var(--d);
    background: 
        conic-gradient(at left var(--s) bottom var(--d),#0000 25%,var(--dc) 0 37.5%,var(--sc) 0) 0 /50% 100% no-repeat,
        conic-gradient(at right var(--s) bottom var(--d),var(--sc) 62.5%,var(--dc) 0 75%,#0000 0) 100%/50% 100% no-repeat;
    background-color: var(--bg);
    clip-path: polygon(
        0 var(--d),
        var(--s) var(--d),
        var(--s) 0,
        calc(100% - var(--s)) 0,
        calc(100% - var(--s)) var(--d),
        100% var(--d),
        calc(100% - var(--c)) calc(50% + var(--d) / 2),
        100% 100%,
        calc(100% - var(--s) - var(--d)) 100%,
        calc(100% - var(--s) - var(--d)) calc(100% - var(--d)),
        calc(var(--s) + var(--d)) calc(100% - var(--d)),

        0 100%,
        var(--c) calc(50% + var(--d) / 2)
    );
}
```

## 折叠丝带形状

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/4660bbca6d69410991e0fdcf43ad754a~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=948&s=435034&e=jpg&b=fbfbfb)

  


我们把类似上图所示的丝带形状称为“折叠丝带”。它有一个明显的特征，丝带至少一侧有一个折边。接下来，我们以下面这个折叠丝带形状为例，使用 CSS 应该如何实现：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/3c7d533a9b474fbdace1707f719504e5~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1056&s=144714&e=jpg&b=ffffff)

  


同样的，首先定义折叠丝带形状所需要 CSS 自定义属性：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/1796f49eeeb04c94842fe90e91fe6744~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1056&s=231374&e=jpg&b=ffffff)

```CSS
.ribbon {
    --s: 50px; /* 控制折叠区 */

}
```

  


与之前的侧面丝带形状相似，我们使用底部边框（`border-bottom`）来创建折叠丝带形状的折叠区，与此同时要考虑丝带形状的凹口区域的大小。与此同时，我们添加一些内距（`padding`）避免文本被裁剪：

  


```CSS
.ribbon {
    --s: 50px; /* 控制折叠区 */
    --c: 25px; /* 控制凹口 */

    /* 控制丝带颜色 */
    --bg: #62ABA8; /* 主背景色 */
    --dc: color-mix(in srgb,var(--bg),#000 45%); /* 控制丝带深度区域（折叠）颜色 */
    --sc: color-mix(in srgb, var(--bg), #000 25%); /* 控制丝带颜色 */

    padding-left: calc(1lh + .5em);
    padding-right: calc(var(--c) + .5em);
    border-bottom: var(--s) solid var(--sc);
    background: var(--bg);
    width: fit-content;
    font-size: 1.5rem;
    line-height: 2;
    color: var(--color);
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/802110fa6643461b80338bc41b05db05~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1144&h=610&s=4015650&e=gif&f=227&b=f3f3f3)

  


然后，只需要按照下图的方式，将正确的坐标点应用于 `polygon()` ，就可以创建想要的折叠丝形状：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/7e86e5d01b434d10b317cd6e099e4604~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1056&s=304117&e=jpg&b=ffffff)

```CSS
.ribbon {
    clip-path: polygon(

        1lh calc(100% - var(--s)), /* P5 */
        1lh calc(100% - var(--c)), /* P6 */
        0.5lh 100%, /* P7 */
        0 calc(100% - var(--c)), /* P8 */
        0 calc(100% - var(--s)) /* P9 */
    );
}
```

## 旋转丝带形状

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/fb470c48a38c424499c87186a4c49f48~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1256&s=536403&e=jpg&b=f5f5f5)

  


要是你觉得之前所介绍的几种丝带形状太过简单的话，那么接下来要创建的丝带形状，你肯定会感兴趣。因为创建它们要比之前所创建的丝带形状复杂一些。我们先来看旋转丝带，以下图这个丝带形状为例：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/8a9dce73953e4bd39344c467d882704c~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1056&s=219675&e=jpg&b=ffffff)

  


和之前创建的丝带形状一样，除了使用 [CSS 自定义属性](https://juejin.cn/book/7223230325122400288/section/7249357815410589733)和 `calc()` 之外，还将使用 [CSS 的三角函数](https://juejin.cn/book/7223230325122400288/section/7242216512176521277) 来计算 `polygon()` 函数坐标点。为了能让大家更易于理解这个丝带形状背后的逻辑，让我们旋转它并保持文本在一条直线上。

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/749e26470e5a43829bbfceef3965fc98~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1056&s=292455&e=jpg&b=ffffff)

  


我增加了一些透明度以便看到主元素后面的部分。我们将使用伪元素 `::before` 和 `::after` 来创建它们（如上图所示），上图中蓝色虚线部分是元素（`.ribbon`）区域。这个丝带形状由两个变量来控制其形状：

  


```CSS
.ribbon {
    --c: 30px;  /* 控制凹口大小 */
    --a: 15deg; /* 控制旋转角度 */
}
```

  


其中 `--a` 将控制主元素旋转以及其他很多事情。另外，我们使用之前的方式来控制丝带的颜色：

  


```CSS
.ribbon {
    --c: 25px; /* 控制凹口 */
    --a: 15deg; /* 控制旋转角度 */

    /* 控制丝带颜色 */
    --bg: #C73225; /* 主背景色 */
    --sc: color-mix(in srgb, var(--bg), #000 25%); /* 控制丝带颜色 */
    --color: color-contrast(var(--bg) vs #fff,#aefeaf,#333); /* 控制文本颜色 */
}
```

  


让我们从主元素开始。从图中可以看到，我们需要从每一侧切掉它，所以你可能会很自然的想到使用 `clip-path` ，但这次不是。我们将依靠渐变来创建丝带主体的形状（看上去像个平形四边形）：

  


```CSS
.ribbon {
    background: linear-gradient(
        calc(90deg - var(--a)),
        #0000 calc(1lh * tan(var(--a))),
        var(--bg) 0 calc(100% - 1lh * tan(var(--a))),
        #0000 0
    );
}
```

  


注意，这里用到了 CSS 在角函数的计算：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/9a9e035ce9ec4b12b022dc26e1f74bfd~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1966&s=585564&e=jpg&b=ffffff)

  


`a` 是我们定义的角度变量（即 `--a`）。考虑到这一点，渐变需要有一个等于 `90deg - a` 的角度，并且透明色（`#0000`）应从 `0` 开始，到 `d` 结束。通过三角函数的计算，`d` 等于 `1lh * sin(a)`。如果我们在另一侧应用相同的逻辑，就得了下面这个代码：

  


```CSS
.ribbon {
    background:linear-gradient(
        calc(90deg - var(--a)),
        #0000 0% calc(1lh * sin(var(--a))),
        var(--bg) calc(1lh * sin(var(--a))) calc(100% - 1lh * sin(var(--a))),
        #0000 calc(100% - 1lh * sin(var(--a))) 100%
  );
}
```

  


我们通过去掉隐含的 `0%` 和 `100%` 进行了优化，当有两个连续的颜色停止值相等时，可以将第二个替换为 `0`。因此，你看到的渐变代码就变成下面这样：

  


```CSS
.ribbon {
    background: linear-gradient(
        calc(90deg - var(--a)),
        #0000 calc(1lh * sin(var(--a))),
        var(--bg) 0 calc(100% - 1lh * sin(var(--a))),
        #0000 0
    );
}
```

  


添加一些基础样式，你现在看到的 `.ribbon` 元素的效果如下图所示：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/fe03f47c5dad45b9986cbc5a5bd3ae36~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1056&s=127888&e=jpg&b=f4f4f4)

  


接下来，我们将注意力转移到伪元素上。这里同样需要用到三角函数，来确定伪元素的尺寸：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/0af62d791dfe4efdb761a01e2c4febef~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1335&s=468138&e=jpg&b=ffffff)

  


如上图所示，我们知道主元素的高度是 `1lh` ，根据三角函数，我们可以快速获得伪元素元素的高度 `H` ，它等于 `1lh ÷ cos(a)`，即 ` calc(1lh  ``/ cos(var(--a))``)` 。对于伪元素的宽度 `W` ，它等于 `(100% - x) × cos(a)` ，其中 `100%` 是 `.ribbon` 元素的宽度，`x` 是`.ribbon` 切口的大小（对应的是渐变中透明部分的大小）。从上一步中，我们可以获得 `x` 等于 `1lh × tan(a)` ，那么伪元素的 `W` 等于 `(100% - x) × cos(a) = (100% - (1lh × tan(a))) × cos(a)`，即 `calc((100% - 1lh * tan(var(--a))) - cos(var(--a)))` ：

  


```CSS
.ribbon {
    position: relative;
    
    &::before,
    &::after {
        content: "";
        position: absolute;
        width: calc((100% - 1lh * tan(var(--a))) * cos(var(--a)));
        height: calc(1lh / cos(var(--a)));
    }
}
```

  


确定尺寸后，我们应正确放置每个伪元素，旋转它，并使用 `clip-path` 裁剪它。最终的 CSS 代码如下：

  


```CSS
.ribbon {
    --c: 25px; /* 控制凹口 */
    --a: 15deg; /* 控制旋转角度 */

    /* 控制丝带颜色 */
    --bg: #c73225; /* 主背景色 */
    --sc: color-mix(in srgb, var(--bg), #000 25%); /* 控制丝带颜色(伪元素的背景颜色) */
    --color: color-contrast(var(--bg) vs #fff,#aefeaf,#333); /* 控制文本颜色 */

    width: fit-content;
    font-size: 1.5rem;
    line-height: 2;
    padding-inline: calc(1lh + 0.5em);
    color: var(--color);
    position: relative;
    rotate: var(--a); /* 旋转整个元素 */
    background: linear-gradient(calc(90deg - var(--a)),#0000 calc(1lh * sin(var(--a))),var(--bg) 0 calc(100% - 1lh * sin(var(--a))),#0000 0);

    &::before,
    &::after {
        content: "";
        position: absolute;
        background: var(--sc);
        height: calc(1lh / cos(var(--a)));
        width: calc((100% - 1lh * tan(var(--a))) * cos(var(--a)));
        rotate: calc(-1 * var(--a));
        translate:0 0 -1px; /* 类似于 z-index: -1 */
    }

    &::before {
        left: 0;
        top: 0;
        transform-origin: top left;
        clip-path: polygon(0 0,100% 0,calc(100% - var(--c)) 50%,100% 100%,calc(1lh / sin(var(--a))) 100%);
    }

    &::after {
        right: 0;
        bottom: 0;
        transform-origin: bottom right;
        clip-path: polygon(0 0,calc(100% - 1lh/sin(var(--a))) 0,100% 100%,0 100%,var(--c) 50%);
    }
}
```

  


`clip-path` 的 `polygon()` 函数的值应该很容易理解，比较难于理解的是三角函数相关的计算。当然，如果你不想了解这些数学计算，也不要紧，你只需要调整代码中的变量，就可以获得你想要的旋转丝带形状：

## 多行丝带形状

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/03a3f66fb1cb4ddf9787a8d162c7d673~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1035&s=747256&e=jpg&b=fafafa)

  


前面创建的丝带形状都有一个共同特点，那就是文本内容是单行的。接下来，我们来探讨多行丝带形状的创建，相比之前的这些丝带形状，它要更复杂一些。

  


首先，我们在 `.ribbon` 元素中添加断行标签 `<br />` ，整丝带的文本强制断行显式：

  


```HTML
<div class="ribbon">
    A ribbon <br />
    title taking <br />
    multi-line <br /> 
    of text
</div>
```

  


假设，我们要创建类似下图这个多行丝带形状：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/7c1ea4a863bd46ea81a691997daf16a8~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1056&s=302182&e=jpg&b=ffffff)

  


同样具有挑战的是，仅依赖一个元素，并且可以容纳任意行数文本（多行文本）的丝带形状，它还是具有响应式。

  


如果你仔细观察这个丝带形状，你会发现丝带形状中有一个部分在整个形状中不断重复（下图蓝色虚线框中的部分）：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/c2939e3f89bf4c92a764ef546588e97b~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2800&h=1056&s=262472&e=jpg&b=ffffff)

  


你可能已经发现了，上图中丝带形状末端并没有带任何凹口。这是因为，我想先从这个通用设计开始，然后逐步添加额外的装饰，最终实现想要的响应式多行丝带形状。

  


看到这样的设计需求，我首先想到的是 CSS 渐变，因为它非常适合创建可重复的图案。实际上，我们所讨论的就是为文本元素添加背景，每一行文本都会有背景，并且根据文本的行数重复。因此，渐变的高度需要与一行文本的高度相同。在 CSS 中，我们使用 `1lh` 可以轻易满足这个需求。

  


让我们来处理第一个渐变，它是一个矩形形状，位于文本后面，覆盖了部分行，并在行之间留白:

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/2a24b17d231f4e1eafc0d94be0356914~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1056&s=523064&e=jpg&b=ffffff)

  


正如你所看到的，这个线性渐变是从丝带主色（例如 `--bg`）到透明色 `#0000` ，其中主色（文本行的背景色）的高度为 `70%` ，透明色（文本行之间的间距）高度则为 `30%` ：

  


```CSS
.ribbon {
    --bg: #C73226; /* 主背景色 */
    --sc: color-mix(in srgb, var(--bg), #000 25%); /* 控制丝带颜色 */
    --color: color-contrast(var(--bg) vs #fff,#aefeaf,#333); /* 控制文本颜色 */

    width: fit-content;
    font-size: 1.5rem;
    line-height: 2;
    text-align: center;
    color: var(--color);
    padding-inline: calc(var(--c) + .5em);
    background-image: linear-gradient(var(--bg) 70%, #0000 0);
    background-position: 0 .15lh;
    background-size: 100% 1lh;
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/67933841f39f408d89f31d0d23ae98d0~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=951&s=150134&e=jpg&b=f4f4f4)

  


上面的代码并不复杂，对吧？我们在 `.ribbon` 元素上设置了线性渐变（`linear-gradient`），这个渐变从 `--bg` 过渡到 `#0000` （默认从上往下，即 `to bottom`）。使用 `background-position` 控制渐变背景的起始位置，并使用 `background-size` 控制渐变背景的尺寸。

  


注意，`background-position-y` 的值为 `0.15lh` ，它等于行间距的一半（在这个示例中，行间距是 `30%` ，也就是 `0.3lh` ，它的一半就是 `0.15lh`）。另外，我们的渐变可以是 `#0000` 到 `--bg` ，再到 `#0000` 。这个渐变可以简化为两个颜色并应用偏移量会更高效，更易读。

  


接下来，我们需要第二个渐变，用于丝带形状的倾斜部分。这个渐变位于第一个渐变的后面。下图通过在前面丝带的颜色中添加一点透明度来更好的展示它们之间的关系：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/100ab76442f4486796c29a33d1372527~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2800&h=1690&s=588379&e=jpg&b=ffffff)

  


如果使用 `linear-gradient` 来描述的话，它大概像下面这样：

  


```CSS
linear-gradient(to bottom right, #0000 50%, var(--sc) 0 X, #0000 0);
```

  


这一次，我们使用 `to bottom right` 设置了渐变的方向。与此同时，颜色从对角线（`50%`）开始，而不是其默认的 `0%` ，并应该在一个我们用 `X` 作为占位符的值处停止。这个值有点复杂，所以我用下图来阐述我们在做什么：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/b62dfae0ff8341ff8750c098af47a97d~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2800&h=1690&s=708992&e=jpg&b=ffffff)

  


上图中，绿色箭头展示了线性渐变的方向，我们可以看到渐变颜色的停止点分别为 `50%` 、`X` 和 `100%` 。我们可以应用一些几何知识计算出 `X` 的值

  


```
(X - 50%) / (100% - 50%) = 70% / 100% 
X = 85%
```

  


我们可以把计算出来的 `X` 应用到第二个线性渐变中：

  


```CSS
.ribbon {
    background-image: 
      linear-gradient(var(--bg) 70%, #0000 0),
      linear-gradient(to bottom left, #0000 50%, var(--sc) 0 85%, #0000 0);
    background-position: 0 .15lh;
    background-size: 100% 1lh;
}
```

  


这个时候，你看到的效果如下：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/5eae55bc1e2044f8a0ee5715fdfb1b2a~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=951&s=166025&e=jpg&b=f4f4f4)

  


这已经完成了设计的主要部分！接下来，我们可以开始制作丝带形状。

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/e798402f6f284c55bbb7cec1ea5af2ba~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2800&h=2486&s=1306022&e=jpg&b=ffffff)

  


我们只需要使用 `clip-path` 按上图所示的坐标对 `.ribbon` 进行裁剪，就可以获得我们想要的多行丝带形状：

  


```CSS
.ribbon {
    --c: 25px; /* 控制凹口 */
    /* 控制丝带颜色 */
    --bg: #c73226; /* 主背景色 */
    --sc: color-mix(in srgb, var(--bg), #000 25%); /* 控制丝带颜色 */
    --color: color-contrast(var(--bg) vs #fff,#aefeaf,#333); /* 控制文本颜色 */

    width: fit-content;
    font-size: 1.5rem;
    line-height: 2;
    text-align: center;
    color: var(--color);
    place-self: center;
    text-transform: capitalize;
    padding-inline: calc(var(--c) + 0.5em);
    background-image: linear-gradient(var(--bg) 70%, #0000 0),
      linear-gradient(to bottom left, #0000 50%, var(--sc) 0 85%, #0000 0);
    background-position: 0 0.15lh;
    background-size: 100% 1lh;

    clip-path: polygon(
        0 0.15lh, /* P1 */
        100% 0.15lh, /* P2 */
        calc(100% - var(--c)) 0.5lh, /* P3 */
        100% 0.85lh, /* P4 */
        100% calc(100% - 0.15lh), /* P5 */
        0 calc(100% - 0.15lh), /* P6 */
        var(--c) calc(100% - 0.5lh), /* P7 */
        0 calc(100% - 0.85lh) /* P8 */
    );
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/85d2a3a8405c43f887066f7d73480275~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1174&h=672&s=3089630&e=gif&f=240&b=f3f3f3)

  


> Demo 地址：https://codepen.io/airen/full/gONRxEQ

## 曲线丝带形状

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/842a72f0f8dd4450ad3525c54d87584f~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1772&s=1436071&e=jpg&b=292929)

  


有的时候，Web 上的丝带形状很可能是带有曲线风格的，如上图所示。对于此类丝带形状，仅仅使用 CSS 是很难实现的。例如下图这个效果：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/e8eda0c43d034ba3be4122a4e38512d9~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1280&h=736&s=34635&e=jpg&b=ffffff)

  


庆幸的是，[将 SVG 的 `<path>` 和 `<textPath>` 结合起来使用](https://juejin.cn/book/7341630791099383835/section/7346773005114507304)，就可以轻易实现该效果。

  


```XML
<svg xmlns="http://www.w3.org/2000/svg" viewBox="0 0 691.3 201.8" class="text">
    <defs>
        <!-- 放置文本所需的路径 -->
        <path id="text-path" d="M108.7 124.1c44.2 29 101 38.4 152 24.4 37.5-10.3 70.7-32.1 106.5-47.3 41.9-17.8 88.1-26.4 133.2-20.8 31.7 4 62.7 15.3 88.7 33.6" fill="none" stroke="#333" stroke-width="2" stroke-dasharray="8" stroke-dashoffset="4" stroke-miterlimit="6" />
    </defs>
    <g id="banner"><!-- 丝带 --></g>

    <!-- 需要放置的文本 -->
    <text>
        <textPath href="#text-path" startOffset="50%">Coca-Cola is Cool!</textPath>
    </text>
</svg>

<svg xmlns="http://www.w3.org/2000/svg" viewBox="0 0 691.3 201.8" class="text">
    <defs>
        <!-- 放置文本所需的路径 -->
        <path id="text-path-02" d="M116.7 96.2c28.3-8.8 57.6-15.2 87.1-19.9 49.7-7.8 89.1-7.7 141.9-7.7 52.8 0 92.2-.1 141.9 7.7 30.2 4.7 60.1 11.3 88.9 20.4" fill="none" stroke="#333" stroke-width="2" stroke-dasharray="8" stroke-dashoffset="4" stroke-miterlimit="6" />
    </defs>
    <g id="banner-02"><!-- 丝带 --></g>

    <!-- 需要放置的文本 -->
    <text>
        <textPath href="#text-path-02" startOffset="50%">Coca-Cola is Cool!</textPath>
    </text>
</svg>
```

  


使用 CSS 给 `textPath` 添加一点样式：

  


```CSS
textPath {
    font-family: "Leckerli One", cursive;
    fill: oklch(0.35 0.07 194.63);
    font-size: 325%;
    font-weight: 800;
    letter-spacing: 0.02em;
    text-anchor: middle;
    text-shadow: 0 1px 0 rgba(255, 255, 255, 0.8);
}
```
