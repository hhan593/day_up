心形形状（❤️）通常以 Icon 图标的方式用于 Web，它常用于点赞、收藏按钮上。Web 开发者可以通过多种不同的方式创建“心形”。除了古老的图片方式之外，还有一些现代方式，比如字体图标，[SVG 图标](https://juejin.cn/book/7341630791099383835/section/7351368000697532427)等。当然，有的 Web 开发者也会使用纯 CSS 来绘制心形形状。这也是我们今天这节课所要探讨的主题！

  


使用 CSS 创建心形形状，有很多种方式。其中最早的方式使用伪元素加 [`border-radius`](https://juejin.cn/book/7199571709102391328/section/7199845563389444099) 和[旋转](https://juejin.cn/book/7223230325122400288/section/7259668493158023205)的方法。但在这节课中，我将给大家分享一些创建心形形状的现代 CSS 技巧。通过这节课的学习，你不仅能掌握构建一个心形的 CSS 技巧，你还能将图片转换成心形形状。

-   CSS 的 `border-image`

## Web 上哪些地方会使用心形形状

“心形”形状在 Web 上可以用于多种场景，包括：

## 现代 CSS：创建心形形状

  


随着现代 CSS 特性的出现，使用 CSS 创建心形形状的方法越来越多，而且不再需要依赖额外的元素（包括伪元素）、圆角和旋转。

  


### 使用 CSS 渐变

  


让我们直接进入第一个方法，在这里我们将通过两个径向渐变（`radial-gradient()`）和一个[锥形渐变](https://juejin.cn/book/7223230325122400288/section/7259668771856941111)（`conic-gradient()`）来创建一个心形。下图可以帮助你理解每个渐变所起的效果：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/b98776704724444f8ecb7596e9c698f0~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1281&s=447352&e=jpg&b=ffffff)

  


正如你所看到的，每个 `radial-gradient()` 将占据元素顶部的四分之一来创建心形形状的圆形部分，`coinc-gradient()` 将创建底部的三角形部分。其中困难的部分是圆心位置以及渐变颜色停止点的位置。我们可以通过试验和调整得出的最佳位置，以确保得到一个优雅的心形形状：

  


```CSS
.heart {
    --size: 50vmin;
    --bg: red;
    width: var(--size);
    aspect-ratio:1;
    background: 
        /* 1️⃣ */
        radial-gradient(circle at 60% 65%, var(--bg) 64%, #0000 65%) no-repeat top left/50% 50%, 
        /* 2️⃣ */
        radial-gradient(circle at 40% 65%, var(--bg) 64%, #0000 65%) no-repeat top right/50% 50%,
        /* 3️⃣ */
        conic-gradient(from -45deg at 50% 84.9%,var(--bg) 90deg,#0000 0) no-repeat bottom/100% 50.1% ;
}
```

  


注意，你不需要调整渐变中的那些百分比数值，你只需要调整元素的尺寸 `--size` 来控制心形的大小（我们使用 `aspect-ratio: 1` 定义了一个正方形元素），并调整渐变颜色 `--bg` 即可。

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/155a25f2eb0748ba93bebdd2c7c8aa91~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1260&h=776&s=2151549&e=gif&f=259&b=040404)

  


> Demo 地址：https://codepen.io/airen/full/KKjWqxY

  


值得注意的是，同样基于两个 `radial-gradient()` 和一个 `conic-gradient()` ，但可以使用不同的语法和更少的代码来实现相同的效果：

  


```CSS
.heart {
    --bg: red;
    --size: 50vmin;
    width: var(--size);
    aspect-ratio: 1;
    background:
        /* 1️⃣ */
        radial-gradient(at 70% 31%, var(--bg) 29.9%, #0000 30%),
        /* 2️⃣ */
        radial-gradient(at 30% 31%, var(--bg) 29.9%, #0000 30%),
        /* 3️⃣ */
        conic-gradient(from -45deg at 50% 84.9%, var(--bg) 90deg, #0000 0)  bottom/100% 50.1% no-repeat;
}
```

### 使用 `clip-path` + 渐变

  


使用上一种方法（两个 `radial-gradient` 和一个 `conic-gradient`）创建心形形状，有可能会遇到 `conic-gradient` 创建的渐变边缘会有锯齿。[虽然对渐变中的百分比值进行微调，有可能避免这种现象出现](https://juejin.cn/book/7199571709102391328/section/7199845781149810727#heading-16)。但是，在这里我想向大家分享另一种方法。

  


使用线性渐变（`linear-gradient`）来替代锥形渐变（`conic-gradient`），在心形底部绘制一个矩形。然后，利用 `clip-path` 属性的 `polygon()` 函数坐标可以超出 `[0%,100%]` 的特性将整个元素裁剪成一个三角形，从而实现心形形状：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/56a3562d8f4f4c88b01977b1ef755b95~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1281&s=406220&e=jpg&b=ffffff)

  


具体的 CSS 代码如下：

  


```CSS
.heart {
    --size: 50vmin;
    --bg: #ff0000;
    width: var(--size);
    aspect-ratio:1;
    place-self: center;
    transition: all .2s;
    background:
        /* 1️⃣ */
        radial-gradient(at 70% 31%, var(--bg) 29.9%, #0000 30%),
        /* 2️⃣ */
        radial-gradient(at 30% 31%, var(--bg) 29.9%, #0000 30%),
        /* 3️⃣ */
        linear-gradient(var(--bg) 0 0) bottom/100% 50% no-repeat;
    clip-path: polygon(-43% 0, 50% 91%, 143% 0);
}
```

  


同样的，你只需要调整 `--size` 和 `--bg` 可以获得不同心形形状的变体：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/ace36cefaf244ab5bb5b2885495d261d~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1220&h=770&s=4564166&e=gif&f=299&b=080808)

  


> Demo 地址：https://codepen.io/airen/full/NWZpaea

  


### 使用 `border-image`

  


CSS 的 `border-image` 属性非常强大，它允许使用图像或渐变作为元素的边框。这意味着它提供了比传统边框（`border`）更灵活和复杂的样式。通过 `border-image` 属性，我们可以将应用于边框的图片切割成一个九宫格，使其具有强大的响应式：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/a3fc7e4b42cc42c885ed9ccb332a697d~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1832&s=776180&e=jpg&b=fff8f7)

  


```CSS
.box {
    border: 30px solid transparent;
    border-image-source: url("./path/to/nature.png");
    border-image-slice: 70;
    border-image-width: 30px;
}
```

  


简单的解释一下：

  


-   **`border-image-source`**：定义用于边框的图像或渐变
-   **`border-image-slice`**：定义图像的哪一部分将用于边框
-   **`border-image-width`**：定义边框的宽度

  


对应的效果如下图所示：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/f95eeec162fd420d89651774a592861e~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1216&h=562&s=3894883&e=gif&f=157&b=fffafa)

  


除此之外，`border-image` 属性还有另外两个子属性：

  


-   **`border-image-outset`**：定义边框图像超出元素边框盒的距离
-   **`border-image-repeat`**：定义如何重复图像以填充边框，它的值有 `stretch`、`repeat` 和 `round` ，有点类似于 `background-repeat`

  


我们可以利用 `border-image` 特性来创建心形形状。如果使用 `border-image` 的话，我们仅用一个径向渐变 `radial-gradient` 就可以。注意，我们将 `radial-gradient` 绘制的圆作为 `border-image-source` 的值。

  


```CSS
.heart {
    --bg: red;
    --size: 50vmin;
    width: var(--size);
    aspect-ratio: 1;
    border-image: radial-gradient(var(--bg) 69%, #0000 70%) 84.5%/50%;
}
```

  


我们需要形状覆盖整个元素，因此宽度（`border-image-width`）需要尽可能的大。其中 `50%` 是最小值，但我们可以设置比 `50%` 更大的值，只不过结果是相同的。需要注意的是，小于 `50%` 时将会产生一个间隙。例如，如果你将 `border-image-width` 设置为 `45%` ，中间会有一个隙间等于 `10%` （`100% - 2 * 45% = 10%` ）。

  


在使用 `border-image` 时，最棘手的是 `border-image-slice` 属性值的设置。当你将其设置为 `100%` 时，每个角上都会有一个圆；设置为 `50%` 时，它又会给你一个覆盖整个元素的完整圆。也就是说，我们需要不断的调式才能获得最终所需要的形状。经过多次测试，这个最佳值大约是 `84.5%` 。

  


最后要提醒大家的是，我们最终还是要使用 `clip-path` 对元素进行裁剪，才能得到心形形状：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/20881216cfdc4e87ba8b981a7491a412~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1832&s=708710&e=jpg&b=ffffff)

  


完整的 CSS 代码如下：

  


```CSS
.heart {
    --bg: red;
    --size: 50vmin;
    width: var(--size);
    aspect-ratio: 1;
    border-image: radial-gradient(var(--bg) 69%, #0000 70%) 84.5%/50%;
    clip-path: polygon(-42% 0, 50% 91%, 142% 0);
}
```

  


同样的，你只需要调整 `--size` 和 `--bg` 的值，就可以获得不同的心形变体：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/c96be3dca8f4439e9dfae8956ff317c6~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1100&h=770&s=4737732&e=gif&f=303&b=080808)

  


> Demo 地址：https://codepen.io/airen/full/VwJpyZr

  


### 使用 `mask` + 渐变

  


前面这几种创建心形形状的方法比较适用于创建类似“心形”的图标（❤️），但要像下图这样的，将内容（比如文字或图片）放入心形形状中，就显得比较鸡肋了：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/c82a8c2182ac49099bc6da84b3049f49~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1336&s=1630301&e=jpg&b=faf7f7)

  


不过，我们可以使用 CSS 的遮罩，并重用前面的代码来实现所要的效果。如果之前使用的是背景方法（例如，CSS 渐变，`clip-path` 加渐变）将依赖于 `mask` ；如果之前使用的是边框的方法（例如 `border-image`）将依赖于 `mask-border` 。

  


> 温馨提示，到目前为止，[支持 `mask-border` 属性的现代浏览器仅有 Safari](https://caniuse.com/?search=mask-border)！

  


我们可以通过[ CSS 的条件判断 `@supports`](https://juejin.cn/book/7223230325122400288/section/7258822669154877480) 对 `mask-border` 做降级处理：

  


```CSS
.heart {
    display: block;
    --size: 50vmin;
    width: var(--size);
    aspect-ratio: 1;
    place-self: center;
    object-fit: cover;
    object-position: center;
    transition: all 0.2s;
    --_m: radial-gradient(#000 69%, #0000 70%) 84.5%/50%;
    -webkit-mask-box-image: var(--_m);
    mask-border: var(--_m);
    clip-path: polygon(-42% 0, 50% 91%, 142% 0);
}

@supports not (-webkit-mask-box-image: var(--_m)) {
    .heart {
        --_m: 
            radial-gradient(at 70% 31%, #000 29%, #0000 30%),
            radial-gradient(at 30% 31%, #000 29%, #0000 30%),
            linear-gradient(#000 0 0) bottom/100% 50% no-repeat;
        mask: var(--_m);
    }
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/7a250238918b40a483eb68e218df4a1e~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1204&h=748&s=7082993&e=gif&f=184&b=050505)

  


> https://codepen.io/airen/full/XWLMZrV

  


在此基础上，稍微添加一点点代码，就可以实现一个很酷的悬停效果：

  


```CSS
.heart {
    outline: 100vmax solid #ff3e60;
    outline-offset: -100vmax;

    &:hover {
        outline-color: #0000;
    }
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/33192d0658844aca8f065e3e5a850584~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1198&h=778&s=2848735&e=gif&f=151&b=060606)

  


> Demo 地址：https://codepen.io/airen/full/KKjWQdW

```CSS
.heart {

    width: var(--size);

    object-position: center;

    clip-path: path("M47.6 300.4L228.3 469.1c7.5 7 17.4 10.9 27.7 10.9s20.2-3.9 27.7-10.9L464.4 300.4c30.4-28.3 47.6-68 47.6-109.5v-5.8c0-69.9-50.5-129.5-119.4-141C347 36.5 300.6 51.4 268 84L256 96 244 84c-32.6-32.6-79-47.5-124.6-39.9C50.5 55.6 0 115.2 0 185.1v5.8c0 41.5 17.2 81.2 47.6 109.5z");
}
```

正如你所看到的，如果将 [SVG 和 CSS 结合起来](https://juejin.cn/book/7341630791099383835/section/7351339840161447945)，你可以创建出非常完美的心形形状。不过，关于 SVG 这部分内容超出来这节课的范畴，如果你对这方面的知识感兴趣的话，请阅读我的小册《[深入浅出 SVG](https://s.juejin.cn/ds/iMWPkMQw/)》中的《[SVG 剪切](https://juejin.cn/book/7341630791099383835/section/7362748816769941540)》和《[SVG 遮罩](https://juejin.cn/book/7341630791099383835/section/7366549423813296165)》，你将会对这方面有一个更全面的认识！
