## 创建返回顶部按钮

  


正如你所看到的，现在“返回顶部”按钮一直固定在页面的右下角，这并不是我们所需要的效果，我们想要的是按钮随着页面的滚动淡入淡出。这个我们稍后来解决！

  


第二个问题时，我们点击“返回顶部”按钮，虽然页面如期返回顶部，但没有平滑的滚动效果。我们先来解决这个问题。我们可以将根元素 `html` 的 `scroll-behavior` 设置为 `smooth` ，这样页面在返回顶部时就会有一个平滑滚动的效果：

  


```CSS
@media screen and (prefers-reduced-motion: no-preference) {
    html {
        scroll-behavior: smooth;
    }
}
```

  


注意，我们把它放在了一个[媒体查询](https://juejin.cn/book/7223230325122400288/section/7257368158451793935)条件中， `prefers-reduced-motion` 媒体查询用于检测用户的系统是否被开启了动画减弱功能，如果开启了用户看不到平滑滚动的效果，这是动画效果对于部分群体并不太友好！

  


现在，你再点击“返回顶部”按钮，页面就会有一个平滑的滚动效果：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/45a838333be24536b56f7eb65c1b5bf3~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1194&h=894&s=19961766&e=gif&f=75&b=4b5b5a)

  


> Demo 地址：https://codepen.io/airen/full/EaaxxgR

  


接下来，我们来解决第一个问题——“返回顶部”按钮不是一直固定在页面右下角，它需要一个淡入淡出的效果。实现这个效果，我们将会用到两种 CSS 技术：

  


-   使用 **[CSS Grid](https://juejin.cn/book/7161370789680250917/section/7161372229123440648)** 来创建布局
-   使用 `position: sticky` 来让按钮固定在屏幕右下角

  


我们的布局将是一个两列的网格——一列用于内容（`main`），另一列用于“返回顶部”按钮（`a`）。为此，我们在 `body` 中添加以下 CSS ：

  


```CSS
body {
    display: grid;
    grid-template-columns: repeat(2, auto);
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/b050cdb5684e478ea86dd6ea9717766c~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2844&h=1246&s=638973&e=jpg&b=5b6e6f)

  


“返回顶部”按钮并没有如期的放置在页面的右一角，我们可以在 `a` 元素上设置 `place-self: end` 将它放置在底部：

  


```CSS
.back-to-top {
    place-self: end;
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/3d5acd9fc61a46bba258fb6472c724bd~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2844&h=1790&s=785794&e=jpg&b=61676c)

  


现在，给 `.back-to-top` （返回顶部按钮）引入以下 CSS 代码：

  


```CSS
.back-to-top {
    position: sticky;
    bottom: 135px;
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/f7eae43575f64e539cfc9a87e63045b4~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1420&h=774&s=18607602&e=gif&f=73&b=4b5b5a)

  


由于我们使用了 `sticky` 行为，按钮被固定在屏幕的右下角。现在，我们需要控制这个行为，使按钮最初是隐藏的。为此，我们将使用 `margin-top` ，并设置其值为屏幕高度（`100dvh`）。

  


```CSS
.back-to-top {
    margin-top: 100dvh;
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/b98be2f38b494573a8766d7d58403093~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1410&h=680&s=15200100&e=gif&f=62&b=4b5b5a)

  


看到了吗？“返回顶部”按钮只有在滚动页面时才会出现，然后保持固定。这离我们的目标越来越近了！

  


我们可以做得更好一点，添加一个偏移量，这个时候 `margin-top` 将变成：

  


```CSS
.back-to-top {
    --offset: 100px;
    margin-top: calc(100dvh + var(--offset));
}
```

  


上面的代码意味着：“页面向下滚动 `100px` 后，才会显示返回顶部的按钮”。也就是说，我们在屏幕高度（`100dvh` ）的基础上添加了一个偏移量 `--offset` ，来控制按钮何时出现：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/bccd7ac5c2dc4186bbfad27d4e6ca69f~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1416&h=630&s=14268125&e=gif&f=62&b=4b5b5a)

  


最后的修饰是将按钮所在列的宽度设置为 `0` ，将 `body` 元素的 `grid-template-columns` 从 `repeat(2, auto)` 改为 `auto 0px` 即可。这种方式会将按钮的空间缩小到 `0` ，按钮会试图通过添加换行（如果有文本内容的话）来适应这个空间。为了解决这个问题，我们只需要添加 `white-space: nowrap`来禁用换行。

  


最终的 CSS 代码如下：

  


```CSS
body {
    display: grid;
    grid-template-columns: auto 0px;
}

.back-to-top {
    position: sticky;
    bottom: 135px;
    --offset: 100px;
    margin-top: calc(100dvh + var(--offset));
    z-index: 15;
    text-decoration: none;
    width: 80px;
    aspect-ratio: 1;
    border: 4px solid #000;
    border-radius: 20px;
    background-color: #fff;
    cursor: pointer;
    display: grid;
    place-content: center;
    translate: 0 120px;
    color: #000;
    place-self: end;
    white-space: nowrap;

    svg {
        grid-area: 1 / 1 / -1 / -1;
        display: block;
        width: 36px;
        height: 36px;
        rotate: 73deg;
        translate: 4px 4px;
        transition: all 0.3s 0.3s ease;
        position: relative;
        z-index: 2;
    }

    &:after {
        content: "";
        position: absolute;
        inset: 0px -4px -8px -4px;
        z-index: -1;
        border-radius: 0 0 25px 25px;
        border-bottom: 10px solid #000;
    }
}

@media screen and (prefers-reduced-motion: no-preference) {
    html {
        scroll-behavior: smooth;
    }
}
```

  


这样我们就创建了第一个纯 CSS 的“返回顶部”按钮，并带有滑动效果：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/cf5e3a54e8c741ec9c4a005ed81ac1ed~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1416&h=630&s=14268125&e=gif&f=62&b=4b5b5a)


  


如果你滚动页面，你会注意到“返回顶部”按钮会从蓝色（`blue`）区域移动到红色（`red`）区域。蓝色区域等于屏幕的高度（`100dvh` ）：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/af464a4e0a374b14aaf09162770aefac~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1396&h=592&s=10121382&e=gif&f=44&b=566156)

  


可以想象，蓝色区域是 `.back-to-top__container` 的“不可见”部分，而红色区域是“可见”部分。如果我们将相同的渐变应用于 `mask` ，并使用透明和不透明的颜色组合：

  


```CSS
.back-to-top__container {
    mask: linear-gradient(#0000 100dvh, #000 0);
}
```

  


现在，按钮只有在滚动时才会出现：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/a7b607e660994e6993e2702c120e5e24~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1392&h=602&s=20680018&e=gif&f=121&b=4b5b5a)

  


为了获得渐隐效果，我们需要增加两种颜色之间的距离。让我们引入一个 CSS 变量，并像下面这样更新 `mask` ：

  


```CSS
.back-to-top__container {
    --fade: 100px;
    mask: linear-gradient(
        #0000 100dvh,
        #000 calc(100dvh + var(--fade))
    );
}
```

  


透明颜色 `#0000` 在 `100dvh` 处结束，而不透明颜色 `#000` 从 `100dvh + 100px` 开始。我们在透明和不透明之间有 `100px` （即 `--fade`）的渐隐。这样就得到了以下效果：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/a12d33993aec45f8b2ff1205464463b7~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1388&h=574&s=16636479&e=gif&f=95&b=4b5b5a)

  


我们快完成了！“返回顶部”按钮正在按照预期渐隐。我们只是缺少了偏移变量来控制渐隐效果发生的时机。为此，我们像之前的效果一样引入 `--offset` 偏移变量，并且将 `.back-to-top__container` 的列宽设置为 `0` 。最终的 CSS 代码如下：

  


```CSS
.back-to-top__container {
    display: grid;
    align-items: end;
    --offset: 100px;
    --fade: 100px;
    mask: linear-gradient(
      #0000 calc(100dvh + var(--offset)),
      #000 calc(100dvh + var(--offset) + var(--fade))
    );
}
```

  


最后，调整 `body` 的 `grid-template-columns` 的值为 `auto 0px` ，然后调整返回按钮的位置。最终 CSS 如下：

  


```CSS
body {
    display: grid;
    grid-template-columns: auto 0px;
}

.back-to-top__container {
    display: grid;
    align-items: end;
    --offset: 100px;

        #0000 calc(100dvh + var(--offset)),

.back-to-top {

```

上面这个示例，虽然“返回顶部”按钮有了渐隐效果，但不是淡入淡出的动画效果。接下来，我们再来创建另一个“返回顶部”按钮效果。它将拥有淡入淡出的动画效果，而且我们整个布局不再依赖于 CSS Grid 和 `position: sticky` 来实现。整个效果依赖于 **[CSS 滚动驱动动画](https://juejin.cn/book/7223230325122400288/section/7259272255786450981)** 来实现。

  


在这里，我们使用固定定位将“返回顶部”按钮固定在屏幕的右下角：

  


```CSS
.back-to-top {
    position: fixed;
    bottom: 1rem;
    right: 1rem;
}
```

  


然后引入 `@keyframes` 定义的动画：

  


```CSS
@keyframes revealScroller {
    0%,
    10% {
        opacity: 0;
        translate: 0 100%;
    }
    18%,
    100% {
        opacity: 1;
        translate:0 0;
    }
} 
```

  


为了使用它，我们像平常一样创建动画，但将 `animation-timeline` 属性设置为 `scroll()`。

  


```CSS
.back-to-top {
    animation: revealScroller 0.2s ease-out;
    animation-timeline: scroll();
    position: fixed;
    bottom: 1rem;
    right: 1rem;
}
```

  


这个时候“返回顶部”按钮的淡入淡出完全由页面的滚动来控制：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/2c9f802d72dc460eb32dd5223f4da06f~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1306&h=510&s=12548026&e=gif&f=69&b=4a5a59)

  


> Demo 地址：https://codepen.io/airen/full/oggNXdd

  


最后来，我们来给“返回顶部”按钮添加指示器效果（实现类似于 theneedledrop.com 网站上的“返回顶部”按钮效果）。

  


首先，我们在上面示例的基础上，给“返回顶部”按钮设置一个渐变背景：

  


```CSS
.back-to-top {
    --yellow: oklch(85.33% 0.185 92.12);
    --white: oklch(100% 0 0);
    background-image: linear-gradient(var(--yellow), var(--white));
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/d86b11d9fb0048feb7613ed43c42f089~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2822&h=1326&s=721495&e=jpg&b=566363)

  


为了避免一种颜色渐变到另一种颜色，我们添加颜色停止点：

  


```CSS
.back-to-top {
    background-image: linear-gradient(var(--yellow) 0% 50%, var(--white) 50% 100%);
}
```

  


我将第一个颜色从 `0%` 开始，到 `50%` 停止，第二个颜色从 `50%` 开始，到 `100%` 结束。效果如下：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/41eba3036b0846d088e9beb45fb4767d~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2824&h=1102&s=551464&e=jpg&b=556262)

  


实际上，我们希望它从 `0%` 开始，逐渐过渡到 `100%` 。

  


接下来，我们需要将颜色停止点放在自定义属性中，因为我们希望对它进行动画处理：

  


```CSS
.back-to-top {
    background-image: linear-gradient(var(--yellow) 0% var(--progress), var(--white) var(--progress) 100%);
}
```

  


传统上，动画会在文档时间线上运行。计时器从页面加载时的 `0` 开始，随着时间推移向前跳动。在此之前，这是我们唯一可以访问的动画时间线。随着 CSS 滚动驱动动画的出现，我们有了两个额外的时间线。其中一个与滚动容器在某个轴上的滚动位置相关联。为了使用它，我们像平常一样创建动画，但将 `animation-timeline` 属性设置为 `scroll()`。

  


```CSS
 @keyframes revealScroller {
    0%,
    10% {
        opacity: 0;
         translate: 0 100%;
    }
    18%,
    100% {
        opacity: 1;
        translate: 0 0;
    }
}

@keyframes progress {
    0% {
        --progress: 0%;
    }
    100% {
        --progress: 100%;
    }
}

.back-to-top {
    animation: revealScroller 0.2s ease-out, progress linear;
    animation-timeline: scroll();

    --yellow: oklch(85.33% 0.185 92.12);
    --white: oklch(100% 0 0);
    background-image: linear-gradient(
        var(--yellow) 0% var(--progress),
        var(--white) var(--progress) 100%
    );

    position: fixed;
    bottom: 1rem;
    right: 1rem;
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/46fba2156f6d4bdca79203809d462cab~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1406&h=608&s=18628250&e=gif&f=76&b=4b5b5a)

  


到现在为止，我们仅看到了淡入淡出的动画效果（`revealScroller`），但并没有看到 `progress` 相关的动画效果。这是因为 `--progress` 是一个字符串，我们无法对字符串进行动画化。庆幸的是，[我们可以通过 `@property` 来注册 `--progress`](https://juejin.cn/book/7395119173656903715/section/7446278656840138762) ，并指定它的类型为 `<length-percentage>` 。比如：

  


```CSS
@property --progress {
    syntax: '<length-percentage>';
    inherits: false;
    initial-value: 0%;
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/e579fbdb20ce4acaad23a4642d3b84f0~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1404&h=506&s=13601994&e=gif&f=66&b=4a5a59)

  


现在，你已经拥有一个带指示器的“返回顶部”按钮的效果。但离最终效果还有一点点距离，我们需要把箭头转向上面。我们注册另一个自定义属性，并在 `2%` 时改变它的值。我们将该属性用于旋转 SVG ：

  


```CSS
@property --angle {
    syntax: '<angle>';
    inherits: true;
    initial-value: 0deg;
}

@keyframes progress {
    0% {
        --progress: 0%;
        --angle: 74deg;
    }
    2% {
        --angle: 0deg;
    }
    100% {
        --progress: 100%;
    }
}

.back-to-top svg {
      rotate: var(--angle);
}
```

  


最终的效果如下：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/bb6dc22722a54c419d63e51f954b57b4~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1404&h=588&s=20485031&e=gif&f=95&b=4b5b5a)
