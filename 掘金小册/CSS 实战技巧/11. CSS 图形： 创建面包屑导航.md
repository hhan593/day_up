面包屑导航（`Breadcrumb`）组件是一种用于显示用户当前在网站或应用程序中的位置的 UI 组件。它通常以一系列的链接或文本显示，呈现出用户从首页或上层页面逐步导航到当前页面的路径。这种组件得名于[童话故事中的“面包屑”](https://en.wikipedia.org/wiki/Hansel_and_Gretel)，因为它类似于在用户的浏览路径上留下的线索，帮助用户知道他们是如何到达当前页面的。简单地说，它通过展示用户当前所在页面及其层级关系，帮助用户理解他们在网站或应用程序中的位置，并提供快速返回上一层级或首页的方式。

  


面包屑导航组件广泛用于结构复杂的网站，如电商网站、博客、文档库、分类信息网站等，尤其适合那些具有多层次内容结构的网站。与此同时，它也会以各种不同的 UI 方式在 Web 上呈现，例如，基于文本链接的、超越简单文本链接的、多步骤流程的和带有子导航的面包屑等：

## 从 HTML 结构开始

  


通常情况之下，面包屑组件（`Breadcrumb`）也可以看作是一个导航系统，因此遵循一些准则来正确定义 HTML 以确保其可访问性是很重要的。接下来的演示，我都采用类似下面这样的 HTML 结构：

  


```HTML
<!-- 不带图标 -->
<nav class="breadcrumb" aria-label="Breadcrumb">
    <ol>
        <li>
            <a href="#">1st Level</a>
        </li>
        <li>
            <a href="#">2nd Level</a>   
        </li>
        <li>
            <a href="#">3rd Level</a>   
        </li>
        <!-- 依此类推，在此省略 -->
        <li>
            <a href="#" aria-current="page">Location</a>   
        </li>   
    </ol>          
</nav>

<!-- 带图标 -->
<nav class="breadcrumb" aria-label="Breadcrumb">
    <ol>
        <li>
            <a href="#"><svg></svg>1st Level</a>
        </li>
        <li>
            <a href="#"><svg></svg>2nd Level</a>   
        </li>
        <li>
            <a href="#"><svg></svg>3rd Level</a>   
        </li>
        <!-- 依此类推，在此省略 -->
        <li>
            <a href="#" aria-current="page"><svg></svg>Location</a>   
        </li>   
    </ol>          
</nav>
```

  


我想，这样的 HTML 代码对于你来说，应该无需做过多的阐述。需要注意的是，使用带有 `aria-label="Breadcrumb"` 的 `<nav>` 元素，明确定义我们的导航是为面包屑导航，而不是网站的主导航。

  


此外，最后一个链接使用 `aria-current="page"` 表明它是当前页面。也有人认为，当前页是不需要跳转的，因此，也有 Web 开发者将最后一个链接替换为其他 HTML 的内联元素，例如 `<span>` 。其实，HTML 规范在这一点上也并没有明确的定义。你可以根据自己的喜好来做选择！

  


> 注意，代码中 `aria-label` 和 `aria-current` 属性都隶属于 ARIA 知识体系，它们的主要作用就是用于提高 Web 的可访问性，[你可以在这里找到相关的解释](https://www.w3.org/TR/html-aria/)。

  


## 创建基础样式

  


接下来，我们给面包屑组件添加一些基础样式，使其好看一点。

  


首先，我们要做的就是覆盖 `ol` 元素上的默认样式，并考虑将所有列表项 `li` 排成一行。我们可以使用 [CSS Flexbox 或 Grid](https://s.juejin.cn/ds/irscbqu4/) 来完成这个任务。在这里，我选择了 [CSS Grid 来布局](https://juejin.cn/book/7161370789680250917/section/7161624041885958151)：

  


```CSS
.breadcrumb {
    --columns: 4;
    --gap: 1px;
    width: 100%;

    ol {
        display: grid;
        gap: var(--gap);
        grid-template-columns: repeat(var(--columns), max-content);
        list-style: none outside none;
    }
    
    svg {
        display: block;
        width: 1em;
        height: 1em;
    }
}
```

  


注意，`svg` 元素的 `width` 和 `height` 设置 `1em` ，是便于你根据 `font-size` 来调整 SVG 图标的尺寸。另外，这里使用[内在尺寸 `max-content`](https://juejin.cn/book/7161370789680250917/section/7161625601713897508) 来根据列表项的内容大小调整列宽。

  


接着，我们来处理元素的尺寸。为了确保元素的尺寸设置正确，我们将定义容器元素 `.breadcrumb` 的 `font-size` ，然后在后代元素上使用 `em` 单位：

  


```CSS
.breadcrumb {
    --fs: 1rem;
    font-size: var(--fs);
    
    a {
        display: inline-flex;
        align-items: center;
        font-size: 1.5em;
        line-height: 2;
        padding-inline: .5em;
        gap: 0.375rem;
        text-decoration: none;
    }
}
```

  


这种技巧特别有用，由于我们使用 `em` 来计算面包屑元素的大小，因此可以通过重新定义 `.breadcrumb` 元素的 `font-size` 来调整所有元素的大小：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/fd35c0dc21c8447d857b1cf8966a1d3a~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1008&h=560&s=803583&e=gif&f=27&b=fdfdfd)

  


> 温馨提示，这种技巧在 CSS 中非常普遍，它是 [CSS 相对单位中的字体相对单位的基本特性](https://juejin.cn/book/7223230325122400288/section/7249357892611440700)。在 **[现代 CSS](https://s.juejin.cn/ds/irsj8er5/)** 中，还有很多与之类似的单位，如果你感兴趣的话，请移步阅读《[CSS 字体相对单位](https://juejin.cn/book/7223230325122400288/section/7249357892611440700)》！

  


设置完大小之后，我们还需要给面包屑组件设置背景颜色和文本颜色等。在设置颜色的过程中，我们可以考虑文本颜色和背景颜色的对比度，使其更易于阅读，这样做能给用户带来更好的体验。

  


```CSS
.breadcrumb {
    --bg: #3598DB;
    --color: color-contrast(var(--bg) vs #fff, #D2F2FB, #333);
    
    a {
        background: var(--bg);
        color: var(--color);
    }
}
```

  


我们通过 `color-contrast() 函数`，使 `--color` 的值能够根据 `--bg` 的颜色在 `#fff` 、`#d2f2fb` 和 `#333` 三个值列表中选择最佳的一个值。

  


需要知道的是，通常情况之下，面包屑的当前项（`aria-current="page"`），即最后一级，它的背景颜色会和其他颜色略有不同，比如高亮显示，或者设置一个不同的背景颜色和文本颜色。我们可以考虑使用 `color-mix() 函数`或[相对颜色](https://juejin.cn/book/7223230325122400288/section/7231515598088306720#heading-14)来处理，我选择了相对颜色：

  


```CSS
.breadcrumb {
    --primary: #3598DB;
    --bg: var(--primary);
    --color: color-contrast(var(--primary) vs #fff, #D2F2FB, #333);
    
    a {
        background: var(--bg);
        color: var(--color);
        
        &[aria-current] {
            --bg: hsl(from var(--primary) 5 s l);
            --color: color-contrast(hsl(from var(--primary) 5 s l) vs #fff, #D2F2FB, #333);
        }
    }
}
```

  


另外，还使用了属性选择器 `[aria-current]` 来匹配最后一个（或当前页）。

  


最后，给链接的 `:hover` 和 `:focus-visible` 定义不同的颜色。这一点也很重要，[尤其是 `:focus-visible`](https://juejin.cn/book/7223230325122400288/section/7226969870444232742)，可以为焦点状态设置样式，这样可以简化键盘导航并确保可访问性。

  


```CSS
.breadcrumb {
    --columns: 4;
    --gap: 1px;
    --fs: 1rem;
    --primary: #3598DB;
    --bg: var(--primary);
    --color: color-contrast(var(--primary) vs #fff, #D2F2FB, #333);
    
    font-size: var(--fs);
    
    ol {
        display: grid;
        gap: var(--gap);
        grid-template-columns: repeat(var(--columns), max-content);
        list-style: none outside none;
    }
    
    a {
        display: inline-flex;
        align-items: center;
        font-size: 1.5em;
        line-height: 2;
        padding-inline: .5em;
        gap: 0.375rem;
        text-decoration: none;
        background: var(--bg);
        color: var(--color);
        transition: all .3s;
      
        &[aria-current] {
            --bg:hsl(from var(--primary) 5 s l);
            --color: color-contrast(hsl(from var(--primary) 5 s l) vs #fff, #D2F2FB, #333);
        
            &:is(:hover,:focus-visible) {
                --bg: hsl(from var(--primary) 5 s 65%);
                --color: color-contrast(hsl(from var(--primary) 5 s 65%) vs #fff, #D2F2FB, #333);
            }
        }
      
        &:is(:hover,:focus-visible) {
            --bg: hsl(from var(--primary) h s 65%);
            --color: color-contrast(hsl(from var(--primary) h s 65%) vs #fff, #D2F2FB, #333);
        }
    }
    
    svg {
        display: block;
        width: 1em;
        height: 1em;
    }
}
```

  


注意，上面代码中还使用了 `:is() 选择器`，在这里主要用来优化代码！

  


这个时候，你能看到一个最普通的面包屑效果：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/678a05fd204243d4b5d8f154c5cf1946~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1100&h=470&s=586658&e=gif&f=172&b=030303)

## 创建丝带状面包屑

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/5c9ba0ee09864e75ac6ebb973b54c3c4~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1600&h=1000&s=24477&e=png&b=ffffff)

  


我们先从最常见的丝带形状的面包屑效果开始！

  


对于丝带形状的面包屑，传统的做法是通过伪元素 `::before` 和（或 ）`::after` 来创建两端的形状，比如三角形：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/3c448c13b9e94341bd2d68ea4b41b16c~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=561&s=280183&e=jpg&b=fdfdfd)

  


现在，我们完全可以使用更现代的方法来创建它。比如，[使用 `clip-path` 的 `polygon()` 函数裁剪出所需要的丝带形状](https://juejin.cn/book/7395119173656903715/section/7399654560878788658)：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/ae9291bbe00941acbdfa9e1a6e6e3953~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=2246&s=1378929&e=jpg&b=ffffff)

  


正如你所看到的，多边形的形状（丝带形状）由一个变量 `s` 控制。我们可以使用 CSS 的自定义属性来定义它，例如 `--s` ：

  


```CSS
.ribbon {
    --s: 20px;
    line-height: 2; 
    padding-inline: .3em calc(var(--s) + .3em);
    clip-path: polygon(
        0 0,                          /* P1 */
        calc(100% - var(--s)) 0,1     /* P2 */
        00% 50%,                      /* P3 */   
        calc(100% - var(--s)) 100%,   /* P4 */
        0 100%                        /* P5 */
    );
    background: #64908A;
    width: fit-content;
    
    &.demo-2 {
        clip-path: polygon(
            0 0,                       /* P1 */
            calc(100% - var(--s)) 0,   /* P2 */
            100% 50%,                  /* P3 */ 
            calc(100% - var(--s)) 100%,/* P4 */
            0 100%,                    /* P5 */
            var(--s) 50%               /* P6 */
        );
    }
    
    &.demo-3 {
        clip-path: polygon(
            0 0,         /* P1 */
            100% 0,      /* P2 */
            100% 100%,   /* P3 */
            0 100%,      /* P4 */
            var(--s) 50% /* P5 */
       );
    }
}
```

  


假设我们把第二种丝带形状用于面包屑中：

  


```CSS
.breadcrumb {
    --s: 20px;
    a {
        clip-path: polygon(0 0,calc(100% - var(--s)) 0,100% 50%,calc(100% - var(--s)) 100%,0 100%,  var(--s) 50%);
    }
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/bbbf29af96104d31a0b56b185d3b2968~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=745&s=161237&e=jpg&b=060606)

  


这已经很接近目标效果了，但仍需要修正一些问题。`clip-path` 在裁剪出我们所需要的丝带过程中，文本也被裁剪掉了。为了解决这个问题，我们需要根据 `--s` 增加链接的内距（`padding`）。我们可以像下面这样给 `a` 元素添加`padding` ：

  


```CSS
.breadcrumb {
    /* 省略其他 CSS */
    
    a {
        /* 省略其他 CSS */
        padding: 0.5lh calc(var(--s) + 0.5lh);
    }
}
```

  


在 `padding` 设置的过程中，我们使用了 `lh` 单位，它是相当于元素的 `line-height` 计算的，这里把块轴方向的内距设置为 `0.5lh` ，内联轴方向的内距设置为 `var(--s) + 0.5lh` ，包括了丝带形状凹口（或凸出）的距离。这个时候，你看到的效果如下：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/0157bbb49d18404bbf4b4326f522041b~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=745&s=175573&e=jpg&b=060606)

  


正如你所看到的，整个效果越来越接近我们的目标效果了。但还有一个地方需要我们继续修正，那就是链接元素之间的间距。虽然我们之前定义的间距（`gap`）没有变化，但使用 `clip-path` 裁剪出我们所需要的形状时，使元素之间的间距在视觉上变得更大了。为了减少间距，我们可以考虑使用一些负边距（`margin`），例如：

  


```CSS
li:not(:last-child) a{
    margin-right: calc(var(--s) / -2);
}
    
li:not(:first-child) a{
    margin-left: calc(var(--s) / -2);
}
```

  


注意，为了避免不必要的溢出，我使用了 `:not()` 选择器禁用了第一个元素的左侧负边距和最后一个元素右侧的负边距：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/5bef8417509a4adda42309ff487d5e05~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=745&s=165887&e=jpg&b=060606)

```CSS

```

### 添加渐变

  


上面展示的示例，面包屑每个列表项的背景颜色都是纯色。如果你希望每个列表项是一个渐变色，那么很简单，我们只需要将 `a` 元素的 `background` 设置为一个渐变色：

  


```CSS
.breadcrumb {
    --columns: 4;
    --gap: 1px;
    --fs: 1rem;
```

### 调整列表项之间的间距

  


有的时候，你可能期望面包屑的列表项之间要有一点间距。类似下图这样的效果：

我们可以通过不同的方式来实现：除第一个列表项的 `a` 元素上设置 `margin-left` 或者除最后一个列表项的 `a` 元素上设置 `margin-right` ，具体的值根据你的喜好来设置：

```CSS
li:not(:last-child) a {

li:not(:first-child) a {
    margin-left: calc(var(--s) / -2);
```

### 调整丝带形状

```CSS
li:first-child a {
    clip-path: polygon(0 0, calc(100% - var(--s)) 0,  100% 50%, calc(100% - var(--s)) 100%, 0 100%);

}
```

### 添加边框

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/e6325185396c4b5e95de4fd14b453d57~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=929&s=307327&e=jpg&b=050505)

  


由于我们在 `a` 元素上应用了 `clip-path` 属性，因此无法直接在 `a` 元素上设置 `border` ，否则你只能看到上下边框，左右边框被裁剪掉了。但有一种技巧可以绕开它，那就是在 `ol` 元素上设置边框，并且调整列表项之间的间距与边框大小一样：

  


```CSS
.breadcrumb {
    --r: .5em;
    --b: .125em;
    
    ol {
        font-size: 1.5em;
        border-radius: var(--r);
        overflow: hidden;
        border: var(--b) solid currentColor;
        background: currentColor;
    }
    
    li:not(:last-child) a {
        margin-right: calc(-1 * var(--s) + var(--b));
    }
}
```

  


需要注意的一点是，`ol` 和 `a` 元素的 `font-size` 要保持一致。在我们这个示例中，可以将 `a` 元素上的 `font-size` 移动 `ol` 元素上来，因为 `font-size` 是一个继承属性。

## 创建圆形切口状面包屑

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/7c40990be8e743c4856856ac45385dca~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=929&s=276206&e=jpg&b=446b7e)

  


接下来，我们再来看看类似上图这样的圆形切口的面包屑 UI 效果如何实现？

  


仔细观察，上图中的每个元素类似一个胶囊形状，[只需要将元素的 `border-radius` 属性设置一个很大的值](https://juejin.cn/book/7199571709102391328/section/7199845563389444099#heading-3)，例如 `999rem` ，就可以完成！

  


```CSS
.breadcrumb {
    a {
        border-radius: 999rem;
    }
}
```

  


另外，将带有边框效果的元素相互叠加，使其看上去产生带有圆形切口的效果：

  


```CSS
.breadcrumb {
    a {
         padding: 0.25em 0.5lh;
         border-radius: 9999rem;
         border: var(--b) solid color-mix(in oklch, var(--primary), #000 70%);
    }
    
    li {
        &:not(:first-child) {
            padding-left: 1.5lh;
        }
        
        &:not(:last-child) {
            margin-right: -3lh;
        }
    }
}
```

  


这个时候，你将得到一个类似下图这样的结果：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/bc4999c9f5a047faa0ea726146edfce6~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=929&s=261642&e=jpg&b=446b7e)

  


不难发现，它们的层叠顺序与我们期望的不一样。因此，我们需要改变它们在 `z` 轴的层级，即调整每个元素的 `z-index` 值，它的值从大往小变。我们可能通过 CSS 的自定义属性来完成：

  


```CSS
.breadcrumb {
    li {
        z-index: var(--index);
    }
}
```

  


注意，这个 `--index` 的值以内联的方式进行设置，并根据元素在 HTML 文档中出现的顺序，从大往小变化：

  


```HTML
<nav class="breadcrumb" aria-label="Breadcrumb">
    <ol>
        <li style="--index: 4;"></li>
        <li style="--index: 3;"></li>
        <li style="--index: 2;"></li>
        <li style="--index: 1;"></li>
    </ol>
</nav>        
```

不难发现，整个效果是通过 `border` 和层叠来模拟的，这使得整个效果是有其身自的局限性，比如，如果我们只需要相邻之处有圆形切口的效果，那么上面这种方法就显得比较鸡肋了。不管你如何调整 `--b` 的值（边框的厚度），都难达到期望的效果：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/4420af794707489eb94ae5c845c530ad~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=929&s=255500&e=jpg&b=446b7e)

  


不过，我们可以使用现代 CSS 来避免这个现象。我们可以利用之前《[CSS 图形：切割元素角落](https://juejin.cn/book/7395119173656903715/section/7400975207551139875)》课程中的[创建圆形切口的技巧](https://juejin.cn/book/7395119173656903715/section/7400975207551139875#heading-1)来实现。简单地说，[使用 `mask` 和 `radial-gradient` 来创建内凹的圆形切口](https://juejin.cn/book/7223230325122400288/section/7259668885224456252)，与此同时使用 `border-radius` 来实同外凸的圆形，两者相互衔接，就实现了我们所要的效果：

  


```CSS
.breadcrumb {

    --gap: 1px;
    --fs: 1rem;
    --primary: #3598db;
    --color: color-contrast(var(--primary) vs #fff, #d2f2fb, #333);

    font-size: var(--fs);
    width: 100%;

    ol {
        display: grid;
        gap: var(--gap);
        grid-template-columns: repeat(var(--columns), max-content);
        list-style: none outside none;
        font-size: 1.5em;
    }

    a {
        --color: color-contrast(hsl(from var(--primary) 65 s 30%) vs #fff,#d2f2fb,#333);
        display: inline-flex;
        align-items: center;
        line-height: 2;
        padding: 0 0.5lh;
        gap: 0.375rem;
        text-decoration: none;
        color: var(--color);
        transition: all 0.3s;

        background: linear-gradient(to right,var(--primary),hsl(from var(--primary) 65 s 30%)) fixed;
        border-radius: 0 999rem 999rem 0;

        &:hover {
            background: linear-gradient(to right,var(--primary),hsl(from var(--primary) 25 s 30%)) fixed;
        }
    }

    li {
        &:first-child a {
            border-radius: 999rem;
        }
        
        &:not(:first-child) a {
            padding-left: 1lh;
            mask: radial-gradient(circle 0.5lh at 0 50%,#0000 calc(0.5lh + 0.125em),#000 0) 0 0;
        }
        
        &:not(:last-child) a {
            margin-right: calc(-0.5lh + 0.0625em);
        }
    }

    svg {
        display: block;
        width: 1em;
        height: 1em;
    }
}
```

## 创建平行四边形面包屑

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/121478b7f78b48deaa03a51c9ff12959~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1442&h=530&s=1295233&e=gif&f=170&b=375d70)

  


这是我们在这节课中要创建的最后一个面包屑 UI 效果。面包屑的每个元素的 UI 效果是一个平行四边形。

  


其实，这个效果对于大部分 Web 开发者来说，没有太大的挑战力，因为使用 CSS 的变换中的 `skew` 就可以完成。只不过，有一个小细节需要注意，`skew` 对元素扭曲之后，需要在其子元素进行一个反向的扭曲，这样才能使其子元素回归到初始状态。

  


在这里为了不增加额外的标签元素，把最初设置在 `a` 链接上的颜色移动到 `li` ，并且在 `li` 设置 `skew(-35deg)` ，同时在 `a` 元素上设置 `skew(35deg)`。最后就是一些细节上的调试，除最后一个元素之外的 `border-right` 为 `none`；除最后一个元素之外的 `margin-right` 设置与边框厚度一样的负值。等等。

  


具体的代码如下：

  


```CSS
.breadcrumb {
    --columns: 4;
    --gap: 1px;
    --fs: 1rem;
    --primary: #90acef;
    --color: color-contrast(var(--primary) vs #fff, #d2f2fb, #333);

    font-size: var(--fs);
    width: 100%;

    ol {
        display: grid;
        grid-template-columns: repeat(var(--columns), max-content);
        list-style: none outside none;
        font-size: 1.5em;
    }
    
    li {
        --bg: var(--primary);
        cursor: pointer;
        background: var(--bg);
        transform: skew(-35deg);
        color: var(--color);
        border: .125em solid currentColor;
      
        &:not(:last-child) {
            border-right: none;
            
            &:hover {
                margin-right: -.125em;
            }
        }
        
        &:first-child{
            border-radius: .25em 0 0 .25em;
        }
      
        &:last-child{
            border-radius: 0 .25em .25em 0;
        }
      
        &:hover {
            --bg: hsl(from var(--primary) h 20% 50%);
            --color: color-contrast(hsl(from var(--primary) h 20% 50%) vs #fff,#d2f2fb,#333);
            border: .125em solid currentColor;
            z-index: 2;
        }
    }

    a {
        display: inline-flex;
        align-items: center;
        line-height: 2;
        padding: 0 0.5lh;
        gap: 0.375rem;
        text-decoration: none;
        color: var(--color);
        transition: all 0.3s;
        transform: skew(35deg);
    }

      svg {
          display: block;
          width: 1em;
          height: 1em;
      }
}
```

除了使用 CSS 变换对元素进行扭曲之外，还可以使用 `clip-path` 裁剪出一个平行四边形。这个与制作丝带形状类似，我们新增一个 `--s` 变量，用于控制切口的大小：

  


```CSS
.breadcrumb {
    --columns: 4;
    --gap: 1px;
    --fs: 1rem;
    --s: 0.5em;
    --primary: #90acef;
    --bg: var(--primary);
    --color: color-contrast(var(--primary) vs #fff, #d2f2fb, #333);

    font-size: var(--fs);
    width: 100%;

    ol {
        display: grid;
        gap: var(--gap);
        grid-template-columns: repeat(var(--columns), max-content);
        list-style: none outside none;
        font-size: 1.5em;
    }

    a {
        display: inline-flex;
        align-items: center;
        line-height: 2;
        padding: 0 calc(var(--s) + .5lh);
        gap: 0.375rem;
        text-decoration: none;
        color: var(--color);
        transition: all 0.3s;
        cursor: pointer;
        background: var(--bg);
        clip-path: polygon(var(--s) 0, 100% 0, calc(100% - var(--s)) 100%, 0 100%);

        &:hover {
            --bg: hsl(from var(--primary) h 20% 50%);
            --color: color-contrast(hsl(from var(--primary) h 20% 50%) vs #fff,#d2f2fb,#333);
        }
    }
    
    li:not(:last-child) a {
        margin-right: calc(-1 * var(--s));
    }

    svg {
        display: block;
        width: 1em;
        height: 1em;
    }
}
```

## 小结

  


这节课，主要向大家展示了如何使用现代 CSS 技术来创建一个外观漂亮，并且可扩展的面包屑组件。正如你所看到的，我们使用了非常简单的代码和一些 `clip-path` 技巧来实现丝带形状，平行四边形形状；使用 `mask` 技巧来实现圆形切口等。你所需要做的就是调整一些数值来定制你的面包屑组件 UI。别忘了，这个组件的可扩展性很强，你只需调整 CSS 自定义属性的值，就可以获得面包屑的变体。
