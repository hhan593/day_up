## 内凹曲线形状简介

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/e911e1637cd840ac97ac2145e41d726b~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=3645&h=1530&s=63254&e=webp&b=fee9ca)

  


我们这里所说的“内凹曲线形状”有很多个称谓，比如：

  


-   **倒角 (Inverted Radius)** ：在元素的**角落处去除一个圆形或弧形部分**，与常规的“圆角”相反。[比如我们之前所介绍的圆角切割](https://juejin.cn/book/7395119173656903715/section/7400975207551139875#heading-1)
-   **凹槽 (Notch)** ：在元素的**侧边而不是角落**挖出一个圆形或弧形的切口，[例如我们常见的优惠卷或邮票 UI 风格](https://juejin.cn/book/7395119173656903715/section/7403197889905688585#heading-1)
-   **圆切 (Rounded Cut)** ：圆切形状与倒角相似，但弧线部分**更圆润、更光滑**，过渡更加自然。[这与之前所介绍的曲线切割很相似](https://juejin.cn/book/7395119173656903715/section/7400975207551139875#heading-2)
-   **钟形曲线 (Bell Curve)** ：钟形曲线是**符合钟形分布的曲线形状**，通常从一侧平滑地凸起再回落，[比如我们之前所介绍的波浪形状边框](https://juejin.cn/book/7395119173656903715/section/7403197889905688585#heading-3)

  


虽然它们的叫法略有差异，但所做的事情是相似的，都是使用 CSS 来创建内凹曲线形状。如下图所示：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/6936c6654b5f43c1b495f4d22f56db80~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2274&h=1218&s=1678439&e=jpg&b=0e0e0e)

  


不得不面对的一个现实是，使用 CSS 实现它们具有一定的挑战性。大多数时候，Web 开发者不得不借助多个元素或伪元素，通过堆叠模拟曲线效果。这种方法虽然可以实现目标，但往往缺乏灵活性，而且 CSS 代码中到处都是一些“魔法数字”，这些魔法数字使得大家难以理解，也不易于维护。

  


接下来，我将尽可能的通过真实案例，向大家展示如何使用**单个元素**和**灵活的 CSS 代码**来创建这些内凹曲线形状。

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/4cdb417cff804c8ca87566992f1157ce~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2010&h=798&s=305250&e=jpg&b=fefdfd)

  


## 创建内凹曲线圆角形状

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/1e62b81e888b4d4b9694ea630977a235~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=932&s=334662&e=jpg&b=9ee1ae)

  


我们先来创建内凹曲线圆角形状。如上图所示，图片角落切出一个圆滑的内凹圆角形状。

  


在 CSS 中，要裁剪出一个圆角角落很简单，如果你有阅读过《[CSS 图形：切割元素角落](https://juejin.cn/book/7395119173656903715/section/7400975207551139875)》一课，那么你应该掌握了[如何实现元素角落带有一个内凹圆角的效果](https://juejin.cn/book/7395119173656903715/section/7400975207551139875#heading-1)。在这里，我们重温一下这个 CSS 技巧。

  


我们只需要在 `mask` 中应用一个径向渐变 `radial-gradient()` 即可：

  


```CSS
img {
    --r: 40px; /* 控制内凹圆角大小 */
    mask: radial-gradient(var(--r) at top right, #0000 98%, #000);
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/f637de16622d4e6eb33f4571516b5741~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1037&s=209738&e=jpg&b=9ee1ae)

  


> Demo 地址：https://codepen.io/airen/full/vYopVBR

  


就这么简单！不过，你需要对[径向渐变 `radial-gradient()`](https://juejin.cn/book/7199571709102391328/section/7199845781149810727) 和[遮罩 mask](https://juejin.cn/book/7223230325122400288/section/7259668885224456252#heading-9) 的工作原理有所了解！如果你不熟悉 `mask` 和渐变的使用，我建议你先阅读相关基础文章，理解如何裁剪角落。如果你想深入了解，请查看小册之前的课程《[CSS 图形：切割元素角落](https://juejin.cn/book/7395119173656903715/section/7400975207551139875)》！

  


接下来，我们将用类似的技术来创建带有平滑曲线的效果的内凹圆角，但相关的配置会更复杂一些。

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/d428fb9d63e64861858b1af0fee5a29c~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2750&h=3319&s=1260421&e=jpg&b=ffffff)

  


首先，我们使用 `radial-gradient()` 裁剪出右上角（上图中 1️⃣ 所示形状），但这次会从上方和右侧做一些偏移：

  


```CSS
.inverted-radius {
    background: radial-gradient(
        40px at 100% 0,
        #0000 99%,#376367 101%
    ) right 25px top 25px no-repeat;
}
```

  


注意，这里使用 `background` 是为了更好的帮助大家理解 `radial-gradient()` 绘制的图形，最后我们需要将 `background` 替换为 `mask` 。

  


接着，我们引入锥形渐变 `conic-gradient()` 来创建第二个形状（上图中 2️⃣ 所示形状），用于填充因偏移产生的空白区域，同时保留一些空间呈现曲线：

  


```CSS
.inverted-radius {
    background: 
        conic-gradient(at calc(100% - 90px) 90px,#0000 25%,red 0),
        radial-gradient(40px at 100% 0,#0000 99%,#376367 101%) right 25px top 25px no-repeat;
}
```

  


最后，我们再用两个径向渐变 `radial-gradient()` 创建两个圆（上图中 3️⃣ 所示形状），用于填充剩余部分，从而模拟出完整的曲线效果：

  


```CSS
.inverted-radius {
    background:
        radial-gradient(blue 70%,#0000 72%)  right 65px top 0 / 50px 50px no-repeat,
        radial-gradient(blue 70%,#0000 72%)  right 65px / 50px 50px no-repeat,
        conic-gradient(at calc(100% - 90px) 90px,#0000 25%,red 0),
        radial-gradient(40px at 100% 0,#0000 99%,#376367 101%) right 25px top 25px no-repeat;
}
```

  


为了清晰起见，我使用了三种颜色表示渐变的拼接。如果换成统一的颜色，四个渐变拼起来的图形看起来像下图这样：

  


```CSS
.inverted-radius {
    background:
        radial-gradient(#000 70%,#0000 72%)  right 65px top 0 / 50px 50px no-repeat,
        radial-gradient(#000 70%,#0000 72%)  right 65px / 50px 50px no-repeat,
        conic-gradient(at calc(100% - 90px) 90px,#0000 25%,#000 0),
        radial-gradient(40px at 100% 0,#0000 99%,#000 101%) right 25px top 25px no-repeat;
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/e6293da02e884d4b908983ff9cc1aeb8~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1037&s=177478&e=jpg&b=ffffff)

  


现在，把这几个渐变绘制的图形作为遮罩图，有颜色的部分将是元素可见的区域，其余部分则为透明区域，该区域对应的部分将会被隐藏。现在，你只需要将 `background` 替换为 `mask` ：

  


```CSS
img {
    mask:
        radial-gradient(#000 70%,#0000 72%)  right 65px top 0 / 50px 50px no-repeat,
        radial-gradient(#000 70%,#0000 72%)  right 65px / 50px 50px no-repeat,
        conic-gradient(at calc(100% - 90px) 90px,#0000 25%,#000 0),
        radial-gradient(40px at 100% 0,#0000 99%,#000 101%) right 25px top 25px no-repeat;
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/b2b89e2ab6c6437b8f44a82130ee64ae~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1037&s=276876&e=jpg&b=9ee0b0)

  


> Demo 地址：https://codepen.io/airen/full/GRVyzLb

  


你可能会对代码中的一些数字感到好奇，可能会问，渐变中这些数字是怎么来的？接下来，我将使用两个 [CSS 自定义属性](https://juejin.cn/book/7223230325122400288/section/7249357815410589733)来替代这些数值。这两个自定义属性可以更好的帮助我们控制内凹圆角形状：

  


-   `--r` ：曲线的小圆半径
-   `--s` ：裁剪的尺寸

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/3dfd1fdb8f46485ea7b093f95475f574~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1037&s=250444&e=jpg&b=ffffff)

  


首先来看第一个径向渐变：

  


```CSS
.inverted-radius {
    --r: 25px; /* 控制小圆半径 */
    --s: 40px; /* 控制裁剪大小 */
    background: radial-gradient(
        var(--s) at 100% 0,
        #0000 99%,#376367 101%
    ) right var(--r) top var(--r) no-repeat;
}
```

  


上面代码中的 `radial-gradient()` 会在元素的右上角顶点（`at 100% 0`）绘制一个半径为 `--s` 的透明圆，元素的其他区域为 `#376367` 。它绘制的渐变会向下偏移 `--r` ，同时向左偏移 `--r` ：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/6da9b448b31e43028812481c7678efe0~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1174&h=626&s=177053&e=gif&f=113&b=ffffff)

  


第一个 `radial-gradient()` 绘制了一个右上角带有内凹圆角（大小为 `--s` ，相当于半径为 `--s` 的圆的四分之一），并且该渐变距元素顶部 `--r` （即 `background-position-y` 等于 `var(--r)` ，使用 `top var(--r)` 来描述，更易于理解），同时距离元素右侧边缘 `--r` （即 `background-position-x` 等于负的 `var(--r)` ，我们可以使用 `right var(--r)` 来描述，它等同于 `left calc(-1 * var(--r))`）。

  


接着在这个渐变之上叠加一个锥形渐变 `conic-gradient()` ：

  


```CSS
.inverted-radius {
    --r: 25px; /* 控制小圆半径 */
    --s: 40px; /* 控制裁剪大小 */
    
    background: 
        conic-gradient(
            at right calc(var(--s) + 2 * var(--r)) top calc(var(--s) + 2 * var(--r)),
            #0000 25%,red 0),
        radial-gradient(
            var(--s) at 100% 0,
            #0000 99%,#376367 101%
        ) right var(--r) top var(--r) no-repeat;
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/a70679598cc148738d076bcf3be8045f~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1037&s=234448&e=jpg&b=ffffff)

  


`conic-gradient()` 的圆心位于距离元素的顶部和右侧的 `s + 2r` 位置，并且 `0% ~ 25%` （即 `0deg ~ 90deg`）之间是透明区域（`#0000`），其他区域是红色（`red`）。如果你对锥形渐变感兴趣的话，[可以点击这里获取更多关于它的信息](https://juejin.cn/book/7223230325122400288/section/7259668771856941111)！

  


最后，再使用两个 `radial-gradient()` 绘制两个小圆，小圆的半径为 `--r` ，第一个小圆距元素顶部 `0` ，距元素右侧 `s+r` ；第二个小圆距元素右侧 `0` ，距元素顶部 `s+r` ：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/8d4519edba4e487ca62bb8b84c49a937~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1037&s=260470&e=jpg&b=ffffff)

  


```CSS
.inverted-radius {
    --r: 25px; /* 控制小圆半径 */
    --s: 40px; /* 控制裁剪大小 */

    background: 
        radial-gradient(blue 70%,#0000 72%) right calc(var(--s) + var(--r)) top 0 / calc(2*var(--r)) calc(2*var(--r)) no-repeat,
        radial-gradient(blue 70%,#0000 72%) right 0 top calc(var(--s) + var(--r)) / calc(2*var(--r)) calc(2*var(--r)) no-repeat,
        conic-gradient(at right calc(var(--s) + 2 * var(--r)) top calc(var(--s) + 2*var(--r)), #0000 0% 25%,red 0),
        radial-gradient(var(--s) at 100% 0,#0000 99%,#376367 101%) right var(--r) top var(--r) no-repeat;
}
```

  


如果你纠结 `radial-gradient()` 中渐变颜色停止点位置的魔法数字 `70%` 和 `72%` 不易于理解，那么可以考虑换过一种方式来创建这两个小圆：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/dd6fda68ed77486692cb944459e52267~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1224&h=390&s=14644&e=webp&a=1&b=fffee1)

  


即，通过小圆的尺寸和圆心坐标位置来绘制小圆。在我们这个示例中，小圆的尺寸是 `2r` ，第一个小圆的圆心的 `Y` 轴位置距离元素顶部是 `r` （即 `top var(--r)`），`X` 轴位置距离元素右侧边缘是 `s + 2r` （即 `right calc(var(--s) + 2 * var(--r))`）；第二个小圆的圆心的 `Y` 轴位置距离元素顶部是 `s + 2r` （即 `calc(var(--s) + 2 * var(--r))`），`X` 轴位置距离元素右侧边缘是 `r` （即 `right var(--r)`）。对应的 CSS 代码如下：

  


```CSS
.inverted-radius {
    --r: 25px; /* 控制小圆半径 */
    --s: 40px; /* 控制裁剪大小 */
    
    background: 
        radial-gradient(calc(2 * var(--r)) circle at top var(--r) right calc(var(--s) + 2 * var(--r)), blue 50%,#0000 51%)  no-repeat,
        radial-gradient(calc(2 * var(--r)) circle at top calc(var(--s) + 2 * var(--r)) right var(--r), blue 50%,#0000 51%)  no-repeat,
        conic-gradient(at right calc(var(--s) + 2 * var(--r)) top calc(var(--s) + 2*var(--r)), #0000 0% 25%,red 0),
        radial-gradient(var(--s) at 100% 0,#0000 99%,#376367 101%)  right var(--r) top var(--r) no-repeat;
}
```

  


同样的，把 `background` 替换成 `mask` 就可以使元素具有一个内凹圆角形状：

  


```CSS
.inverted-radius {
    --r: 25px; /* 控制小圆半径 */
    --s: 40px; /* 控制裁剪大小 */
    
    mask: 
        radial-gradient(calc(2 * var(--r)) circle at top var(--r) right calc(var(--s) + 2 * var(--r)), blue 50%,#0000 51%)  no-repeat,
        radial-gradient(calc(2 * var(--r)) circle at top calc(var(--s) + 2 * var(--r)) right var(--r), blue 50%,#0000 51%)  no-repeat,
        conic-gradient(at right calc(var(--s) + 2 * var(--r)) top calc(var(--s) + 2*var(--r)), #0000 0% 25%,red 0),
        radial-gradient(var(--s) at 100% 0,#0000 99%,#376367 101%)  right var(--r) top var(--r) no-repeat;
}
```

  


现在，你只需要调整 `--r` 和 `--s` 的值，就可以获得具有不同尺寸的内凹曲线圆角形状：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/e90c4bdaf4204bac82341f5b55e543e5~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1202&h=736&s=4116452&e=gif&f=259&b=94d9a4)

  


> Demo 地址：https://codepen.io/airen/full/abeEeog

  


现在，我们只需要调整径向渐变的圆心位置以及锥形渐变的起始角度和圆心位置，我们就可以在元素的不同方向创建内凹曲线圆角：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/7da5a9b03a7246808cdf344a4b3278d6~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1856&s=787055&e=jpg&b=ffffff)

  


  


我们把上图转换为 CSS 代码：

  


```CSS
img {
    --r: 25px; /* 控制小圆半径 */
    --s: 40px; /* 控制裁剪大小 */
    
    display: block;
    width: 40vmin;
    aspect-ratio: 1;
    border-radius: 25px;
    transition: all 0.2s;
}

.top-right {
    mask: 
        radial-gradient(calc(2 * var(--r)) circle at top var(--r) right calc(var(--s) + 2 * var(--r)),blue 50%,#0000 51%) no-repeat,
        radial-gradient(calc(2 * var(--r)) circle at top calc(var(--s) + 2 * var(--r)) right var(--r),blue 50%,#0000 51%) no-repeat,
        conic-gradient(from 0deg at right calc(var(--s) + 2 * var(--r)) top calc(var(--s) + 2 * var(--r)),#0000 0% 25%,red 0) no-repeat,
        radial-gradient(var(--s) at 100% 0, #0000 99%, #376367 101%) right var(--r) top var(--r) no-repeat;
}

.top-left {
    mask: 
        radial-gradient(calc(2 * var(--r)) circle at top var(--r) left calc(var(--s) + 2 * var(--r)),blue 50%,#0000 51%) no-repeat,
        radial-gradient(calc(2 * var(--r)) circle at top calc(var(--s) + 2 * var(--r)) left var(--r),blue 50%,#0000 51%) no-repeat,
        conic-gradient(from 0deg at left calc(var(--s) + 2 * var(--r)) top calc(var(--s) + 2 * var(--r)),red 0% 75%,#0000 0),
        radial-gradient(var(--s) at 0 0, #0000 99%, #376367 101%) left var(--r) top var(--r) no-repeat;
}

.bottom-right {
    mask: 
        radial-gradient(calc(2 * var(--r)) circle at bottom var(--r) right calc(var(--s) + 2 * var(--r)),blue 50%,#0000 51%) no-repeat,
        radial-gradient(calc(2 * var(--r)) circle at bottom calc(var(--s) + 2 * var(--r)) right var(--r),blue 50%,#0000 51%) no-repeat,
        conic-gradient(from 90deg at right calc(var(--s) + 2 * var(--r)) bottom calc(var(--s) + 2 * var(--r)),#0000 0% 25%,red 0) no-repeat,
```

## 创建内**凹曲线形状**

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/7be5ea1acfee42dea58834f0db0c9931~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=827&s=236944&e=jpg&b=ffffff)

  


接下来看另一种形状——内凹曲线形状不是位于元素的顶角，而是位于元素的侧面。

  


与之前的形状类似，只需要一个简单的径向渐变就可以给元素添加一个圆形的凹口：

  


```CSS
.notch {
    --r: 3rem;
    mask: radial-gradient(var(--r) at top, #0000 98%, #000);
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/18f6aa6f217e4fdc97da49f8a50cc9ed~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1376&h=862&s=3656659&e=gif&f=125&b=030303)

  


> Demo 地址：https://codepen.io/airen/full/PoMQmYB

  


但要让这些内凹口形状带有曲线效果，就需要**更多的渐变配置**。我们以顶部内凹曲线形状为例：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/32bcca36e5d942cfb5f8013b5b2c1fc7~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=4096&h=1073&s=981522&e=jpg&b=ffffff)

  


同样使用四个渐变来创建内凹曲线形状：

  


-   首先使用一个 `radial-gradient()` 创建内凹圆口（上图 1️⃣ 所示）
-   然后再使用一个 `linear-gradient()` 创建一个线性渐变，用于填充顶部偏移出来的空间（上图 2️⃣ 所示）
-   最后使用两个 `radial-gradient()` 创建两个小圆，用于创建内凹口顶部的连接，使其具有平滑曲线效果（上图 3️⃣ 所示）

  


它们拼接在一起，就获得了我们想要的效果——顶部带有一个内凹曲线形状。

  


和之前的案例类似，我们可以使用 `--r` 和 `--s` 变量来控制内凹曲线的大小：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/66929a9817c747438afbc8aa4bd994e9~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=2798&s=402935&e=jpg&b=ffffff)

  


```CSS
.notch {
    --r: 20px;  /* 控制小圆半径 */
    --s: 40px;  /* 控制凹口尺寸 */
    
    mask: 
        radial-gradient(calc(var(--r) * 2) circle at top var(--r) left calc(50% - var(--s) - var(--r)), #000 50%, #0000 51%),
        radial-gradient(calc(var(--r) * 2) circle at top var(--r) right calc(50% - var(--s) - var(--r)), #000 50%, #0000 51%),
        linear-gradient(90deg, #000 0 calc(50% - var(--s)  - var(--r)), #0000  calc(50% - var(--s)  - var(--r)) calc(50% + var(--s) + var(--r)), #000 0),
        radial-gradient(var(--s) at top, #0000 98%, #000) top var(--r) left 0 no-repeat;
}
```

除此之外，我们还可以引入另一个变量 `--a` ，用来控制曲线的深度：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/384d63b21acf402b9ea566daa4b7f6cd~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=3918&h=1349&s=768134&e=jpg&b=ffffff)

  


这个形状是由三个圆组成，它们排列在一起（大圆用于切割内凹口，两个小圆用于曲线的弧度）。我们可以通过调整这些圆的排列来增加对形状的控制。如上图所示，在左侧（我们之前的做法），三个圆的圆心都在同一水平位置（水平对齐），但在右侧，大圆向上偏移了一点。虽然这三个圆仍然相邻，但圆心不再完全对齐。我们可以把大圆和其中一个小圆的圆心连接成一条直线，圆心在同一位置连接的直线与圆心不在同一位置连接的直线会有一个夹角，我们把这个夹角定义为一个新的变量： `--a` 。

  


这样一来，我们就可以根据[三角函数来计算大圆的偏移量](https://juejin.cn/book/7223230325122400288/section/7242216512176521277)，具体的 CSS 如下所示：

  


```CSS
.notch {
    --r: 20px;  /* 控制小圆半径 */
    --s: 40px;  /* 控制凹口尺寸 */
    --pos: 50%; /* 控制位置 */
    --a: 30deg; /* 控制曲率深度 */
    
    --_d:(var(--s) + var(--r))*cos(var(--a));
    
    mask:
        radial-gradient(calc(2 * var(--r)) at left var(--pos) top var(--r),#000 50%,#0000 51%) left calc(var(--pos) + var(--_d)) top 0 no-repeat,
        radial-gradient(calc(2 * var(--r)) at left var(--pos) top var(--r),#000 50%,#0000 51%) left calc(var(--pos) - var(--_d)) top 0 no-repeat,
        radial-gradient(var(--s) at left var(--pos) top calc(-1*sin(var(--a)) * var(--s)), #0000 100%,#000 calc(100% + 1px)) 0 calc(var(--r)*(1 - sin(var(--a)))) no-repeat,
        linear-gradient(90deg,#000 calc(var(--pos) - var(--_d)),#0000 0 calc(var(--pos) + var(--_d)),#000 0);
}
```

这个示例的 HTML 很简单：

  


```HTML
<div class="avatar">
    <img src="http://i.pravatar.cc/200?img=2" alt="">
    <h2>Rooms</h2>
</div>
```

  


相应的 CSS 代码如下：

  


```CSS
@property --a {
    syntax: "<angle>";
    inherits: true;
    initial-value: 0deg;
}

.avatar {
    --r: 30px;
    --s: 80px;
    --a: 20deg;
    width: 100vw;
    position: relative;
    z-index: 0;
    cursor: pointer;
    transition: 0.4s --a;
    color: #fff;
    font-family: system-ui, sans-serif;
    text-align: center;
    
    &:hover {
        --a: 35deg;
    }
    
    img {
        display: block;
        height: calc(2 * var(--s));
        aspect-ratio: 1;
        scale: 0.9;
        translate: 0 calc(-1 * sin(var(--a)) * var(--s) - sin(var(--a)) * var(--r));
        border-radius: 50%;
        background: #d95b43;
        margin: auto;
    }
    
    &::before {
        content: "";
        position: absolute;
        z-index: -1;
        inset: calc(var(--s) / 2 + var(--r) / 4) 0 calc(var(--s) / -3) 0;
        background: linear-gradient(45deg, #00a0b0, #88a65e);
        --_m: 0 / calc(2 * var(--r)) calc(2 * var(--r)) no-repeat radial-gradient(50% 50%, #000 calc(100% - 1px), #0000);
        --_d: (var(--s) + var(--r)) * cos(var(--a));
        mask: 
            calc(50% + var(--_d)) var(--_m), 
            calc(50% - var(--_d)) var(--_m),
            radial-gradient(var(--s) at 50% calc(-1 * sin(var(--a)) * var(--s)),#0000 100%, #000 calc(100% + 1px)) 0 calc(var(--r) * (1 - sin(var(--a)))) no-repeat,
           linear-gradient(90deg,#000 calc(50% - var(--_d)),#0000 0 calc(50% + var(--_d)), #000 0);
    }

    h2 {
        margin-block: calc(var(--s) / -3) 0;
        font-size: 3rem;
    }
}
```

  


注意，在这个示例中，曲线凹口应用在伪元素 `::before` ，因为直接应用在 `.avatar` 上，会造成用户头像被隐藏。当然，你也可以将曲线凹口应用在 `h2` 元素，并且通过 CSS 的锚点来定位用户头像。要是你感兴趣的话，可以尝试着用这种方案来实现。另外，示例中的动画效果是通过 `transition` 来动画化变量 `--a` 实现的，因此你需要使用 `@property` 来注册 `--a` 变量。有关于这方面更详细的介绍，[可以点击这里获取](https://juejin.cn/book/7223230325122400288/section/7258870477462962236)！

  


你可能注意到了，示例中的渐变写法与我们之前的案例略有不同，但最实达到的目的是相似的。你可以尝试着理解上面的 CSS 代码。比较简单的方法是，你先把 `mask` 换成 `background` ，然后分别在不同的元素上应用渐变，你会豁然开朗！还有就是，你可以在示例中引入 `--pos` 变量，改变默认的位置！有关于这方面，就不在这里阐述了，感兴趣的可以自己尝试一下，权当是挑战自己的 CSS 吧！
