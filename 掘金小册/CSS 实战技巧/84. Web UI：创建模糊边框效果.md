## 创建模糊边框效果

  


我们从创建透明边框开始。在 CSS 中，我们有很多种方式可以给一个元素添加透明边框。在这里，我给伪元素 `::after` 设置了一个带有透明度的边框，并将它覆盖在 `img` 元素上（也可以是其他元素）：

  


```HTML
<figure>
    <img src="https://picsum.photos/800/600" alt="" />
</figure>
```

  


```CSS
figure {
    --b: 1.5rem;
    display: block;
    width: 400px;
    aspect-ratio: 4 / 3;
    position: relative;
    
    &::after {
        content: "";
        position: absolute;
        inset: 0;
        border: var(--b) solid rgb(0 0 0 /.3);
        pointer-events: none;
        z-index: 2;
    }
    
    img {
        display: block;
        width: 100%;
        height: 100%;
        object-fit: cover;
        object-position: center;
    }
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/beacda777d2c4705a06d8e24326f7a45~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2522&h=1012&s=144325&e=jpg&b=ffffff)

  


现在，图片带有一个厚度为 `--b` 的透明边框。你可能会认为，我们只需要在伪元素上设置一定的模糊度，就可以获得一个带有模糊效果的边框。比如：

  


```CSS
.figure::after {
    filter: blur(10px);
}

/* 或者 */
.figure::after {
    backdrop-filter: blur(10px);
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/12fb5436555e47b3ba215c93aabadde3~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=798&s=379719&e=jpg&b=fefefe)

  


但不管哪一种方式，离我们所期望的效果都有点距离。

  


这个时候，我以为将 `backdrop-filter` 与 `clip-path` 结合起来就可以实现：

  


-   使用 `backdrop-filter` 在 `img` 元素上覆盖一个“磨砂玻璃”层
-   然后使用 `clip-path` 裁剪掉“边框”内的部分，仅留下边框区域的“模糊效果”

  


对应的 CSS 代码如下：

  


```CSS
figure {
    --b: 1.5rem;
    display: block;
    width: 400px;
    aspect-ratio: 4 / 3;
    position: relative;

    &::after {
        content: "";
        position: absolute;
        inset: 0;
        border: var(--b) solid #0000;
        pointer-events: none;
        z-index: 2;
        background: linear-gradient(
            to bottom,
            rgb(0 0 0 / 0.3),
            rgb(0 0 0 / 0.3)
        );

        backdrop-filter: blur(10px);
        --shape: 
            var(--b) var(--b), 
            calc(100% - var(--b)) var(--b),
            calc(100% - var(--b)) calc(100% - var(--b)),
            var(--b) calc(100% - var(--b)), 
            var(--b) var(--b);
        clip-path: polygon(
            evenodd,
            0 0,
            100% 0,
            100% 100%,
            0 100%,
            0 0,
            var(--shape)
        );
    }

    img {
        display: block;
        width: 100%;
        height: 100%;
        object-fit: cover;
        object-position: center;
    }
}
```

  


但结果告诉我，这个方法行不通：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/33f29d8093034e2c83bbc5c586c2b2c0~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2522&h=1016&s=155875&e=jpg&b=fefdfd)

  


注意，示例中的 [`clip-path` 应用到了反转形状相关的技巧](https://juejin.cn/book/7395119173656903715/section/7395172754053890063#heading-9)，这个技巧在之前的课程中有相关介绍。这里就不再重复阐述了！

  


虽然我们使用了 `clip-path` 裁剪掉了中间区域，只留了边框区域，但高斯模糊算法还是将底部伪元素底部的区域进行了模糊计算。出现这种现象，主要还是因为浏览器对 `clip-path` 和 `backdrop-filter` 执行顺序造成的。这意味着，我们即使使用 `clip-path` 也无法实现我们所期望的效果。

  


庆幸的是，我们可以借助 CSS 的 `mask` 来隐藏中间区域，只留出边框区域，从而实现带有模糊效果的边框：

  


```CSS
figure {
    --b: 1.5rem;
    display: block;
    width: 400px;
    aspect-ratio: 4 / 3;
    position: relative;

    &::after {
        content: "";
        position: absolute;
        inset: 0;
        border: var(--b) solid #0000;
        pointer-events: none;
        z-index: 2;
        background: linear-gradient(
            to bottom,
            rgb(0 0 0 / 0.3),
            rgb(0 0 0 / 0.3)
        );

        backdrop-filter: blur(10px);
        --shape: 
            var(--b) var(--b), 
            calc(100% - var(--b)) var(--b),
            calc(100% - var(--b)) calc(100% - var(--b)),
            var(--b) calc(100% - var(--b)), var(--b) var(--b);
        clip-path: polygon(
            evenodd,
            0 0,
            100% 0,
            100% 100%,
            0 100%,
            0 0,
            var(--shape)
        );
        --fill: linear-gradient(#000, #000);
        mask: var(--fill) padding-box exclude, var(--fill);
    }

    img {
        display: block;
        width: 100%;
        height: 100%;
        object-fit: cover;
        object-position: center;
    }
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/365df3eb1ca74ef0a6495733daafb026~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2486&h=986&s=230506&e=jpg&b=ffffff)

  


> Demo 地址：https://codepen.io/airen/full/bNGrKEX

  


既然用到了 CSS 的 `mask` 特性，那么我们可以选择更高级的方法来实现模糊边框效果。通过 `backdrop-filter: blur()` 叠加遮罩层（`mask-image`）来实现渐晕效果（模糊边框效果）。遮罩由六个渐变组合而成：

  


-   四个径向渐变，分别用于四个角
-   两个线性渐变

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/cfbe5804146a4a5c8beeabfadfbf2871~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2560&h=1350&s=11832&e=webp&b=414141)

  


注意，上图中黑色区域表示遮罩层中可见的部分。具体的代码如下：

  


```HTML
<div class="container">
    <img class="image" src="https://picsum.photos/800/600" alt="" />
    <div class="blur-vignette"></div>
</div>
```

  


你也可以使用伪元素 `::before` 或 `::after` 来替代上面代码中的 `.blur-vignette` 。关键的 CSS 代码如下所示：

  


```CSS
:root {
    --radius: 32px;
    --inset: 16px;
    --transition: 12px;
    --blur: 11px;
}

.container {
    position: relative;
    width: 300px;
    aspect-ratio: 1;
    border-radius: var(--radius);
    overflow: hidden;
    place-self: center;

    img {
        display: block;
        width: 100%;
        height: 100%;
        object-fit: cover;
        border-radius: var(--radius);
    }
}

.blur-vignette {
    position: absolute;
    inset: 0;
    border-radius: var(--radius);
    backdrop-filter: blur(var(--blur));
    --r: max(var(--transition), calc(var(--radius) - var(--inset)));
    --corner-size: calc(var(--r) + var(--inset)) calc(var(--r) + var(--inset));
    --corner-gradient: transparent 0px, transparent calc(var(--r) - var(--transition)), black var(--r);
    --fill-gradient: black, black var(--inset), transparent calc(var(--inset) + var(--transition)),  transparent calc(100% - var(--transition) - var(--inset)), black calc(100% - var(--inset));
    --fill-narrow-size: calc(100% - (var(--inset) + var(--r)) * 2);
    --fill-farther-position: calc(var(--inset) + var(--r));
    mask-image: 
        linear-gradient(to right, var(--fill-gradient)),
        linear-gradient(to bottom, var(--fill-gradient)),
        radial-gradient(at bottom right, var(--corner-gradient)),
        radial-gradient(at bottom left, var(--corner-gradient)),
        radial-gradient(at top left, var(--corner-gradient)),
        radial-gradient(at top right, var(--corner-gradient));
    mask-size: 
        100% var(--fill-narrow-size),
        var(--fill-narrow-size) 100%, var(--corner-size), var(--corner-size),
        var(--corner-size), var(--corner-size);
    mask-position: 
        0 var(--fill-farther-position),
        var(--fill-farther-position) 0, 
        0 0, 
        100% 0, 
        100% 100%, 
        0 100%;
    mask-repeat: no-repeat;
}
```

之前我们提到过，直接在伪元素上应用 `filter` 创建的模糊边框效果并不能达到我们所期望的效果。事实上，我们可以通过一些小技巧来使模糊边框效果更佳。只不过，这种技巧有一点绕。首先，我们将 `img` 替换成 `background-image` ，并且将它应用到一个容器上，比如 `div` ：

  


```CSS
:root {
    --url: url('https://picsum.photos/id/309/800/600');
    --b: 1.5rem;
}

.figure {
    width: 400px;
    aspect-ratio: 4 / 3;
    position: relative;
    border: solid var(--b) rgb(0 0 0 / .35);
    background: var(--url) no-repeat center / cover;
    background-origin: border-box;
    background-clip: padding-box;
}
```

  


注意上面代码中的 `background-origin` 和 `background-clip` ，它使背景图片位于 `padding-box` ，边框区域（`border-box`）中不会有任何背景。

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/9414d9b5cc8a4f31ad2ccbb0f9c58430~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1912&h=852&s=656813&e=png&b=fcf8f7)

  


接下来，我们通过伪元素 `::after` 和 `filter` 结合起来创建模糊边框。其中的关键之处是，伪元素 `::after` 会继承 `.figure` 元素的背景，并且将伪元素放置在 `.figure` 元素的底部：

  


```CSS
:root {
    --url: url("https://picsum.photos/id/309/800/600");
    --b: 1.5rem;
    --blur: 16px;
}

.figure {
    width: 400px;
    aspect-ratio: 4 / 3;
    position: relative;
    border: solid var(--b) rgb(0 0 0 / 0.35);
    background: var(--url) no-repeat center / cover;
    background-origin: border-box;
    background-clip: padding-box;
    

    &::after {
        content: "";
        position: absolute;
        z-index: -1;
        inset: calc(-1 * var(--b));
        border: inherit;
        border-color: transparent;
        background: inherit;
        background-clip: border-box;
        filter: blur(var(--blur));
    }
}
```

  


温馨提示，上面代码中应用到了 CSS 中的各种盒子框概念，如果你对这方面不太熟悉的话，请移步阅读之前的课程《[CSS 中的各种 `*-box`](https://juejin.cn/book/7395119173656903715/section/7445257036046991410)》！

  


这个时候，我们获得了一个模糊效果的边框：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/a1a3ef571c994b70aaac9dc99f359c7e~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1912&h=852&s=831065&e=png&b=faf5f3)

  


  


不难发现，应用在伪元素 `::after` 上的 `filter:blur()` 有一个向外的发散效果，这些发散出去的模糊效果并不是我们想要的。你可能会想到，在 `.figure` 元素上使用 `overflow: hidden` 将溢出的模糊效果裁剪掉：

  


```CSS
.figure {
    overflow: hidden;
}
```

  


不幸的是，`overflow: hidden` 会创建一个新的上下文，与伪元素的 `z-index:-1` 会产生冲突。使得伪元素不可见：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/69a6095c3fc04ae681210e03bf395b92~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1912&h=852&s=656814&e=png&b=fcf8f7)

  


> 温馨提示，如果你想更深入的了解 `overflow` 和 `z-index` 的背后的工作原理，请移步阅读《[溢出常见问题与排查](https://juejin.cn/book/7199571709102391328/section/7213705145954074679)》和《[`z-index` 失效与修复](https://juejin.cn/book/7199571709102391328/section/7204402059225858052)》！

  


碰到类似这种现象，我们可以通过下面的方式来修复：

  


```CSS
.figure {
    width: 400px;
    aspect-ratio: 4 / 3;
    position: relative;
    padding: var(--b);
    background: var(--url) no-repeat center / cover;
    background-origin: padding-box;
    background-clip: content-box;
    overflow: hidden;
    
    &::after {
        content: "";
        position: absolute;
        z-index: -1;
        inset: 0;
        background: inherit;
        background-clip: padding-box;
        filter: blur(var(--blur));
    }
}
```

  


上面代码通过 `padding-box` 来模拟 `border-box` 区域（即边框的厚度），然后相应的调整 `background-origin` 和 `background-clip` 属性的值。这样我们就能获得更完美的模糊边框效果：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/96ca59233ab74a3b8167d59a0f87cb42~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1912&h=852&s=748389&e=png&b=faf5f3)

使用 `clip-path` 进行裁剪时，我们直接添加 `padding` 不会受到任何影响。但如果使用 `overflow: hidden` 来裁剪溢出的模糊时，我们需要做一些调整。因为我们是通过 `padding` 来模糊边框厚度的。我们可以像下面这样来调整 CSS 代码：

  


```CSS
:root {
    --url: url("https://picsum.photos/id/309/800/600");
    --b: 1.5rem;
    --blur: 16px;
}

.figure {
    text-align: center;
    color: #fff;
    font: 600 1.5em/ 1.375 segoe script, comic sans ms, cursive;
    text-shadow: 1px 1px #000;
    width: 400px;
    padding: calc(1em + var(--b));
    position: relative;
    background: var(--url) no-repeat center / cover;
    background-origin: padding-box;
    background-clip: content-box;
    overflow: hidden;

    &::before,
    &::after {
        content: "";
        position: absolute;
        z-index: -1;
        inset: 0;
        border: var(--b) solid rgb(0 0 0 / .3);
        background: inherit;
        background-origin: border-box;
        background-clip: padding-box;
    }
    
    &::before {
        border-color: transparent;
        background-clip: border-box;
        filter: blur(var(--blur));
    }
}
```

  


相比而言，要比使用 `clip-path` 更为复杂，但最终效果是相似的：

使用 `filter` 与 `backdrop-filter` 类似， 同样可以结合其他类似的滤镜，创建一些更丰富的边框效果：

  


```CSS
.figure {
    &:nth-child(1)::after {
        filter: blur(var(--blur));
    }
    
    &:nth-child(2)::after {
        filter: blur(var(--blur)) sepia(100%);
    }
    
    &:nth-child(3)::after {
        filter: blur(var(--blur)) grayscale(100%);
    }
    
    &:nth-child(4)::after {
        filter: blur(var(--blur)) contrast(120%) saturate(50%) sepia(40%);
    }
}
```

## 小结

  


课程中展示了三种制作模糊边框效果的方法，相比而言，如果需要创建出 Apple Vision Prol 平台的模糊渐晕效果，则需要通过 `backdrop-filter` 叠加遮罩层（`mask-image`）来实现。这种方法创建的模糊效果有点类似上一节课介绍的[渐进式模糊效果](https://juejin.cn/book/7395119173656903715/section/7445547336421687347)。在实际应用的过程中，大家应该选择更为适合自己项目的方案来创建模糊边框效果。
