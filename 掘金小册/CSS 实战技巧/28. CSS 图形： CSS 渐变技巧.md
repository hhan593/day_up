Web 开发者对于 CSS 渐变而言，一定不会感到陌生，不需要再重复介绍它们是什么以及如何使用。小册之前的课程中几乎每一节都有应用到 CSS 渐变，例如创建 CSS 图案、UI 形状、渐变边框、渐变阴影、渐变文本，甚至是 CSS 加载动画。然而，渐变的语法相当复杂，如果不留心，很容易变得难以理解。

  


在这节课中，我们不会讨论复杂的 CSS 渐变。相反，我们将保持简单，我会带你了解使用**一个**渐变就能实现的各种惊人效果。在这个过程中，你会发现渐变在基础层面上很简单，但如果我们挖掘其潜力，它们也是非常强大的。

  


## CSS 渐变简介

  


CSS 渐变允许颜色在元素中平滑过渡。除了基本的线性渐变（`linear-gradient()`）、径向渐变（`radial-gradient()`）和锥形渐变（`conic-gradient()`）之外，还支持重复渐变，例如 `repeating-linear-gradient()` 、`repeating-radial-gradient()` 和 `repeating-conic-gradient()` ，让渐变效果在元素中不断重复。这为创建条纹、网格等图案提供了强大的工具。

  


以下是每种渐变的基本用法：

  


```CSS
/* 线性渐变 */
.gradient {
    /* 水平方向渐变 */
    background-image: linear-gradient(to right, red, yellow);

    /* 垂直方向渐变 */
    background-image: linear-gradient(to bottom, blue, green);

    /* 斜角 45° 的渐变 */
    background-image: linear-gradient(45deg, pink, purple);
}

/* 径向渐变 */
.gradient {
    /* 圆形径向渐变 */
    background-image: radial-gradient(circle, red, yellow, green);

    /* 从右上角扩展的径向渐变 */
    background-image: radial-gradient(circle at top right, blue, purple);
}

/* 锥形渐变 */
.gradient {
    background-image: conic-gradient(red, yellow, green, blue);
}

/* 重复线性渐变 */
.repeating-gradient {
    background-image: repeating-linear-gradient(45deg, red, yellow 10%, green 20%);
}

/* 重复径向渐变 */
.repeating-gradient {
    background-image: repeating-radial-gradient(circle, blue, white 10px, blue 20px);
}

/* 重复锥形渐变 */
.repeating-gradient {
    background-image: repeating-conic-gradient(red 0deg 30deg,yellow 30deg 60deg);
}
```

  


需要知道的是，CSS 渐变不仅限于用于 `background-image` ，它们还可以用于 `mask-image` 和 `border-image` 等属性。它们为 Web 设计师和开发者提供了丰富的创作可能，可以创造出丰富多样的视觉效果，提升页面的设计美感和用户体验。

  


接下来，我们一起来看看只使用一个渐变能为我们做些什么？

## 网格线

  


接下来，我们来看单个 CSS 渐变的另一个案例，创建网格线，如下图所示：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/6eaab49ac14148688e9d09592ec2cdee~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1030&s=1065856&e=jpg&b=ffffff)

  


你可能会说它也是“CSS 图案”中的一种，这个说法是对得！但我们可以通过添加控制线条粗细和单元格总数等变量使其更加灵活。换句话说，让我们使用单个 CSS 渐变来创建一个网格：

  


```CSS
.grid {
    --n: 9; /* 行数 */
    --m: 9; /* 列数 */
    --s: 60px; /* 控制网格大小 */
    --t: 2px; /* 厚度 */
    --c: #09aefe; /* 控制网格线颜色 */

    width: calc(var(--m) * var(--s) + var(--t));
    height: calc(var(--n) * var(--s) + var(--t));
    
    background: conic-gradient(from 90deg at var(--t) var(--t),#0000 25%,var(--c) 0) 0 0 / var(--s) var(--s);
  }
```

  


上面的代码创建了一个 `9 x 9` 的网格（九行九列），并且每个单元格的大小是 `60px` ，网格线的厚度是 `2px` ：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/569e461b128a4b939098f2f6dc50abc9~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1030&s=285454&e=jpg&b=050505)

  


> Demo 地址：https://codepen.io/airen/full/abeLvRd

  


我们来分解一下其制作原理。

  


首先，我们使用锥形渐变创建网格的最小单元：

  


```CSS
.grid {
    width: 200px;
    aspect-ratio: 1;
    outline: 2px dashed;
    background: conic-gradient(from 90deg at 10px 10px, #0000 25%, #09aefe 0);
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/6fbffcea4e244a528932d463053afc28~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1030&s=230292&e=jpg&b=060606)

  


`conic-gradient()` 创建了一条水平线和一条垂直线。渐变的大小（网格单元格）由变量 `--s` 控制，因此我们通过计算获得网格的宽度和高度，以便获得我们想要的数量线条，从而创建网格图案。你可能会好奇，在计算网格宽度和高度时，为什么要加上 `var(--t)` ？如果没有它，整个网格图案会缺少右侧和底部的线条，这在我们使用渐变情况下是合乎逻辑的。要解决这个问题，我们需要再重复一次，但不使用完整的大小。因此，我们在计算宽度和高度的时候，只添加网格线的厚度，留出足够的空间使渐变再多一次平铺，从而获得缺失的网格线。

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/954dfcf990be4075bf7baf078dd175d3~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1322&s=385594&e=jpg&b=050505)

  


注意上图中淡蓝色的线也是渐变绘制的，但没有更多的空间来呈现，你可以将其视作为被裁剪掉的部分。

  


上面所展示的是，我们已知网格行数和列数的情况，但很多时候我们并不知道网格的行数或列数。换句话说，我们希望网格的列数和行数能根据容器的可用空间而变化。针对于这种情景，我们可以去掉 `--m` 变量（列数）和 `--n` 变量（行数），像下面这样来定义元素的宽和高：

  


```CSS
.grid {
    --s: 80px;  
    --t: 4px;   
    --c: #09aefe;
    background: conic-gradient(from 90deg at var(--t) var(--t),#0000 25%,var(--c) 0)  0 0/var(--s) var(--s);
  
    width: calc(round(down,100%,var(--s)) + var(--t));
    height: calc(round(down,100%,var(--s)) + var(--t));
}
```

  


我们使用了 `round()` 函数告诉浏览器将元素调整为全宽，并将值四舍五入为 `--s` 的倍数。换句话说，浏览器会为我们找到合适的倍数！

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/abd216b44fec498b9841cc9616194433~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1190&h=638&s=9727382&e=gif&f=419&b=030303)

  


> Demo 地址：https://codepen.io/airen/full/wvVrKZY

  


在未来，我们也可以使用 `calc-size()` 函数来实现这一点：

  


```CSS
.grid {
    --s: 80px;  
    --t: 4px;   
    --c: #09aefe;
    background: conic-gradient(from 90deg at var(--t) var(--t),#0000 25%,var(--c) 0)  0 0/var(--s) var(--s);
    
    width: calc-size(auto,round(down,size,var(--s)) + var(--t));
    height: calc(round(down,100%,var(--s)) + var(--t));
}
```

  


使用 `calc-size()` 基本上与上一个示例相同，但我们将 `auto` 视为宽度值，而不是使用 `100%` 。

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/08e12d4b617a46db99906af898294b50~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1190&h=638&s=9727382&e=gif&f=419&b=030303)

## 高清渐变：彩虹渐变

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/0e2fa055f63a4469ad944c3406e85fd1~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1010&s=444283&e=jpg&b=292929)

  


众所周知，CSS 渐变的另一个功能就是允许我们给元素添加平滑的渐变背景。比如下面这个彩虹渐变：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/09df4dcec2494a69b8e419e5533f4340~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=929&h=200&s=1556&e=webp&b=00fff7)

  


我想，大部分 Web 开发者首先会考虑使用线性渐变 `linear-gradient()` 将彩虹色（红 `red` 、橙 `orange` 、黄 `yellow` 、绿 `green` 、蓝色 `blue` 、靛 `indigo` 和紫 `violet`）连接在一起：

  


```CSS
.rainbow-linear {
    --roygbiv: red, orange, yellow, green, blue, indigo, violet, red;
    background: linear-gradient(90deg, var(--roygbiv));
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/fc4cd79a674049d9be5f5bc69085828a~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1124&s=823193&e=jpg&b=fba803)

  


这种做法并没有错，但不能完全达到目的。此外，你需要不断调整颜色的停止点，直到你得到完美的效果。

  


其实，在[现代 CSS](https://s.juejin.cn/ds/iSUUuM6U/) 中，有一个更简单的解决方案，即[颜色插值](https://juejin.cn/book/7223230325122400288/section/7233227753909125178#heading-31)。我们只需要使用一个颜色就能实现这个效果：

  


```CSS
.rainbow-linear {
    background: linear-gradient(90deg in hsl longer hue, red 0 0);
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/0ade6b8d3a8d4ae0b3e4a0bfe3491387~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1124&s=923679&e=jpg&b=ee01ff)

  


> Demo 地址：https://codepen.io/airen/full/YzmEwJv

  


  


其实有一个更简单的解决方案。我们只需使用一个颜色就能实现这个效果！

  


```CSS
background: linear-gradient(90deg in hsl longer hue, red 0 0);
```

  


你可能对这种新的颜色语法感到好奇，甚至还会觉得很奇怪。在这里简单的解释一下，[不过有关于这方面更详细的介绍，可以点击这里获取](https://juejin.cn/book/7223230325122400288/section/7233227753909125178#heading-31)。

  


我们先从最简单的渐变开始，例如：

  


```CSS
body {
    background: linear-gradient(90deg, red, red); 
}

/* 或者 */
body {
    background: linear-gradient(90deg, red 0 0); 
}
```

  


上面的代码将会创建一个从红色到红色的渐变，就是一纯红色的背景。当我们设置为 `in hsl` 时，渐变颜色使用的是 [HSL 颜色空间](https://juejin.cn/book/7223230325122400288/section/7231515598088306720#heading-5)，而不是默认的 [RGB 颜色空间](https://juejin.cn/book/7223230325122400288/section/7231515598088306720#heading-4)。

  


```CSS
body {
    background: linear-gradient(90deg in hsl, red 0 0); 
}
```

  


不过，在视觉上，它依旧还是一个纯红色：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/6b0de8c82abd4b859b71158ab9d17565~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1085&s=766379&e=jpg&b=ff0101)

  


接下来的 `longer hue` 才是颜色插值中有趣的部分。当我们的颜色处于 HSL 颜色空间时，色相通道的值是一个角度值，例如 `0deg` （`red` 等同于 `hsl(0deg 100% 50%)`）。你可以将 HSL 颜色空间视为一个圆，其中角度定义了颜色在该圆中的位置：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/14de6de049b84a0da5978ca3ba64b1c3~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1536&h=670&s=252330&e=webp&b=fefdfd)

  


由于这是一个圆，我们可以使用“短路径”（`shorter hue`）或“长路径”（`longer hue` ）在两个点之间移动。

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/b70e3f3f8d5f4e4990bf4cf4aa78cad1~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=389&h=388&s=28900&e=webp&b=f8efeb)

如果我们考虑相同的点（在我们的例子中是红色），那么“短路径”仅包含红色，而“长路径”则会穿过整个颜色空间，包含所有颜色。

  


```CSS
body {
    background: linear-gradient(90deg in hsl longer hue, red 0 0); 
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/deba840e618249ca8cfd64a0b0208aed~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1085&s=1043691&e=jpg&b=ff3c01)

  


> Demo 地址：https://codepen.io/airen/full/poMdydP

  


除了“短路径”（`shorter hue`）或“长路径”（`longer hue` ）之外，还有增加（`increasing hue` ）或减少（`decreasing hue`） 相色的插值：

  


```CSS
:root {
    --standard-hue:background: linear-gradient(to right in hsl, hsl(140deg 100% 75%), hsl(240deg 100% 75%));
    --shorter-hue: linear-gradient(to right in hsl shorter hue, hsl(140deg 100% 75%), hsl(240deg 100% 75%));
    --longer-hue: linear-gradient(to right in hsl longer hue, hsl(140deg 100% 75%), hsl(240deg 100% 75%));
    --increasing-hue: linear-gradient(to right in hsl increasing hue, hsl(140deg 100% 75%), hsl(240deg 100% 75%));
    --decreasing-hue: linear-gradient(to right in hsl decreasing hue, hsl(140deg 100% 75%), hsl(240deg 100% 75%));
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/08c3bed2b62b49edb99a98d439c5df90~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1085&s=888075&e=jpg&b=7ff1fe)

  


> Demo 地址：https://codepen.io/airen/full/MWNOyqO

  


由于其独特的形状和颜色排列方式，每个颜色空间的最终渐变效果都会有所不同。比如下面示例中的“从蓝到白”的渐变，每个颜色空间都会以不同的方式处理它。

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/9f00f14a4b9546d1991b776865b8c9c9~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2394&h=5500&s=6733430&e=png&b=5d597a)

## 有趣技巧：创建颜色数组

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/53bc7b3e5b7e484582405d8c4dc98a78~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1416&h=1012&s=74746&e=jpg&b=17181a)

  


你是否还有印象，之前在介绍 [CSS 创建进度条](https://juejin.cn/book/7395119173656903715/section/7405199788926304308)时，展示了一个根据 `<progress>` 的 `value` 的值动态调整进度条的滑块的颜色：

  


```CSS
progress[value] {
    --w: 50vw;
    --track-color: linear-gradient(to bottom, #141414, #343434);
    --progress-color: 
        linear-gradient(#fff8, #fff0),
        repeating-linear-gradient(135deg, #0003 0 10px, #0000 0 20px),
        /* if < 30% "red" */ 
        linear-gradient(red 0 0) 0 / calc(var(--w) * 0.3 - 100%) 1px,
        /* if < 60% "orange" */ 
        linear-gradient(orange 0 0) 0 / calc(var(--w) * 0.6 - 100%) 1px,
        /* else "green" */ 
        green;
    inline-size: var(--w);
    mask:
        radial-gradient(circle closest-side,#000 98%,#0000 102%) 0/12.5% 100%,
        linear-gradient(90deg,#000 50%, #0000 0) 8%/25% 100%;
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/8cdba331ccb14a738b5ca29c2965269f~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1028&s=272745&e=jpg&b=246677)

  


> Demo 地址：https://codepen.io/airen/full/XWLqQjj

  


其实，我们可以使用另一个小技巧来实现。这个技巧就是从一个颜色“数组”中索引遍历出目标颜色。更准确地说，我们可以使用 CSS 创建一个“颜色数组”，通过改变一个索引变量从数组中选择所需的颜色。我知道，这听起来可能有些不可思议，但上面的示例已经告诉我们，CSS 是可以做到的。

  


接下来，我们就来详细的阐述这个技巧背后的工作原理。

  


我们先从一个简单的用例开始，在 `--colors` 变量中定义两个颜色：

  


```CSS
:root {
    --colors: #000, #fff;
}
```

  


在这个例子中，我们将使用 [CSS 的 `color-mix()` 函数](https://juejin.cn/book/7223230325122400288/section/7237288025221234744)，不过我们这里不会使用它的默认功能——也就是混合颜色，而是使用它的另一个不为人知小技巧：**根据参数列表返回其中一个颜色**。

  


```CSS
:root {
    --colors: #000, #fff; /* 定义一个颜色数组 */
    --i: 0;               /* 定义索引变量 */
  
    --_color:    color-mix(in hsl, var(--colors) calc(var(--i) * 100%));
    --_bg-color: color-mix(in hsl, calc(var(--i) * 100%) var(--colors));
}

body {
    color: var(--_color);
    background: var(--_bg-color);
}
```

  


到目前为止，我们所做的只是将颜色数组赋值给 `--colors` 变量，然后使用索引变量 `--i` 来选择颜色。索引从 `0` 开始，所以这里只有 `0` 或 `1`，就像布尔检查一样。代码可能看起来有点复杂，但如果我们将变量替换成具体的值，就会变得清晰。例如，当 `i=0` 时：

  


```CSS
:root {
    --_color: color-mix(in hsl, #000, #fff 0%);
    --_bg-color: color-mix(in hsl, 0% #000,#fff);
}
```

  


上面的代码的意思是：

  


-   `--_color` 是由 `100%` 的 `#000` 和 `0%` 的 `#fff` 混合而成，结果是 `#000`
-   `--_bg-color` 是由 `0%` 的 `#000` 和 `100%` 的 `#fff` 混合而成，结果是 `#fff`

  


类似地，当 `i=1` 时：

  


```CSS
:root {
    --_color: color-mix(in hsl, #000, #fff 100%);
    --_bg-color: color-mix(in hsl, 100% #000,#fff);
}
```

  


-   `--_color` 是由 `0%` 的 `#000` 和 `100%` 的 `#fff` 混合而成，结果是 `#fff`
-   `--_bg-color` 是由 `100%` 的 `#000` 和 `0%` 的 `#fff` 混合而成，结果是 `#000`

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/d9a91a63a35b4e3e88e4202bb2d3508a~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1192&h=618&s=688466&e=gif&f=219&b=018b7d)

  


> Demo 地址：https://codepen.io/airen/full/wvVPEJE

  


想想看，我们仅通过一个简单的 CSS 小技巧，就实现了两种颜色之间的切换。这种技巧非常适用于为你的网站添加暗黑模式的场景。你可以将暗黑模式的两种颜色定义在同一个变量中，然后在[媒体查询的 `prefers-color-scheme` 条件](https://juejin.cn/book/7223230325122400288/section/7257368158451793935)中更新 `--i` 变量（索引值）的值，就可以在暗色和亮色模式之间切换。

  


```CSS
:root {
    --colors: #000, #fff;  /* 定义颜色数组 */
    --i: 0;                /* 索引值 */
  
    --_color:    color-mix(in hsl, var(--colors) calc(var(--i) * 100%)); /* i = 0 时，结果是 #000 */
    --_bg-color: color-mix(in hsl, calc(var(--i)*100%) var(--colors));   /* i = 0 时，结果是 #fff */

    @media (prefers-color-scheme: dark) {
        --i: 1; /* i = 1 时，--_color 值为 #fff, --_bg-color 值为 #000 */
    }
}

body {
    color: var(--_color);
    background: var(--_bg-color);
}  
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/6a4fc722c9f542f7aa65cfcda26baa7d~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1182&h=610&s=287900&e=gif&f=169&b=ffffff)

  


> Demo 地址：https://codepen.io/airen/full/NWQwLbE

  


注意，你可能对 `color-mix()` 函数中 `in hsl` 感到好奇。其实它指的是混合颜色在哪个颜色空间进行混合，但在这个示例中，它实际上没有什么作用，因为我们并没有真正进行颜色混合。因此，你可以使用任何颜色空间，例如 `in srgb` (sRGB)、`in hsl` （HSL）、`in oklch` （OKLCH）、`in oklab` （OKLAB）等等。无论选择哪种颜色空间，都别忘了写上它，因为这是 `color-mix()` 函数必需的参数。如果想了解更多关于 `color-mix()` 函数相关的知识，[可以点击这里获取](https://juejin.cn/book/7223230325122400288/section/7237288025221234744)！

  


接下来，我们将 `--colors` 变量设置更多的颜色值：

  


```CSS
:root {
    --colors: red, orange, yellow, green, blue, indigo, violet;
    --i: 0;
}
```

  


与此同时，我们新增另一个变量 `--n` ，用它来定义数组中的颜色数量（“颜色数组”的长度）：

  


```CSS
:root {
    --colors: red, orange, yellow, green, blue, indigo, violet;
    --i: 0;
    --n: 7;
}
```

  


索引变量 `--i` 和之前类似，但现在它可以从 `0` 到 `n-1` 之间选择所需的颜色。

  


接着，我们使用 `linear-gradient()` 来替代之前的 `color-mix()` 。我们可以像下面这样将变量 `--colors` 应用于 `linear-gradient()` ：

  


```CSS
:root {
    --colors: red, orange, yellow, green, blue, indigo, violet;
    --i: 0;
    --n: 7;
}

body {
    background: linear-gradient(90deg, var(--colors));
}
```

渐变的定义是颜色之间的平滑过渡，但在渐变过程中，数组中的实际颜色会至少显示几个像素，而它们之间会有颜色混合的效果。例如：

  


-   左侧是红色
-   右侧是紫色
-   中间是其他过渡的颜色

```CSS
:root {
    --colors: red, orange, yellow, green, blue, indigo, violet;
    --i: 0;
    --n: 7;
}

body {
```

> Demo 地址：https://codepen.io/airen/full/jOgaXMd

  


这样设置后，渐变会被无限拉伸，相当于只显示数组中的一个颜色。根据这个思路，我们可以通过渐变来根据索引从颜色数组中提取特定颜色。通过这种方法，我们不仅能实现简单的颜色过渡，还能在需要时从数组中挑出指定的颜色。

  


也就是说，上面这个示例，我们从 `--colors` 数组定义的渐变颜色中选取了红色。如果想切换颜色，只需调整渐变的位置（`background-position`）即可。

  


默认情况之下，渐变位于元素的左上角（`background-position: 0 0`），所以我们只能看到左侧的颜色，即可红色。现在，我们可以使用索引变量 `--i` 来调整渐变的位置，从数组中获取想要的颜色。

  


```CSS
body {
    background-position: calc(var(--i) * 100% / (var(--n) - 1)) 0%;
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/0033efab9a814cfa893eeb00eca8bcd9~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2000&h=1532&s=452019&e=jpg&b=ffffff)

  


完整的 CSS 代码如下：

  


```CSS
:root {
    --colors: red, orange, yellow, green, blue, indigo, violet;
    --i: 0;
    --n: 7;
}

body {
    background: linear-gradient(90deg, var(--colors)) no-repeat;
    background-size: calc(1px * infinity) 100%;
    background-position: calc(var(--i) * 100% / (var(--n) - 1)) 0%;
}
```

  


或者简写成：

  


```CSS
:root {
    --colors: red, orange, yellow, green, blue, indigo, violet;
    --i: 0;
    --n: 7;
}

body {
    --g:linear-gradient(90deg, var(--colors)) no-repeat;
    background: var(--g) calc(var(--i) * 100% / (var(--n) - 1)) 0% / calc(1px * infinity) 100%;
}
```

很酷，对吧！我们在 CSS 中实现了一个颜色数组，而且代码并不复杂。只需要一个简单的背景技巧和几个变量就能完成。

  


我想你已经掌握了这个小技巧，并且可以将它应用到实际项目中。不过，我们还可以对该技巧进一步优化。之前，我提到过，索引变量 `--i` 的值被限制在 `0` 到 `n-1` 之间，这样的设置是合理的。超出这个范围的任何值都可能会导致没有颜色显示。那么，如果我们取消这个限制，让索引变量 `--i` 可以接受任意值，是不是更完美一些。

  


简单地说，我们可以使用 CSS 的 `mod()` 函数（“取模运算”）将 `--i` 变量的任意值转换到 `[0, n-1]` 范围。例如：

  


-   当 `--i = 50` 时，`mod(var(--i), var(--n))` 等同于 `50 mod 7 = 1` ，对应 `--colors` 中的第二个颜色值
-   当 `--i = -10` 时， `mod(var(--i), var(--n))` 等同于 `-10 mod 7 = 4` ，对应 `--colors` 中的第五个颜色值

  


因此，我们可以将前面的渐变代码调整为：

  


```CSS
:root {
    --colors: red, orange, yellow, green, blue, indigo, violet;
    --i: 0;
    --n: 7;
}

body {
    --_i: mod(var(i), var(--n));
    --g:linear-gradient(90deg, var(--colors)) no-repeat;
    background: var(--g) calc(var(--_i) * 100% / (var(--n) - 1)) 0% / calc(1px * infinity) 100%;
}
```
