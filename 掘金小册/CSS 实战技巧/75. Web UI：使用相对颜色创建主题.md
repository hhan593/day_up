## 调整颜色的痛点

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/5f868b043b734293bcedda92d008394d~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1792&h=1024&s=840428&e=webp&b=dddfc0)

  


一直以来，CSS 是不具备动态调整颜色的能力。因此，很多 Web 开发者会借助 CSS 处理器（比如 Sass）的颜色函数 `saturate()` 、`desaturate()` 、`lighten()` 、`darken()` 和 `adjust-hue()` 等，动态调整颜色的饱和度、亮度、暗度和色调。直到 [CSS 自定义属性](https://juejin.cn/book/7223230325122400288/section/7249357815410589733)的出现，有些 Web 开发者也会借助 CSS 自定义属性的特性，来模拟 Sass 的一些颜色函数功能：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/e45bcc7d3f714f1fa6a585efb3407831~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=4052&h=1976&s=2028910&e=jpg&b=faf4f1)

  


我们以修改颜色的不透明度为例，在相对颜色语法出现之前，通常需要为颜色的各个通道（通常是 HSL）创建多个自定义属性，然后将它们组合成最终颜色和变体颜色。这意味着要管理大量的颜色参数，容易变得繁琐且难以维护：

  


```CSS
:root {
    --brand-hue: 300deg;
    --brand-saturation: 75%;
    --brand-lightness: 50%;
    --brand-hsl:
        var(--brand-hue)
        var(--brand-saturation)
        var(--brand-lightness);
    --brand-color: hsl(var(--brand-hsl));
    --brand-color-variant: hsl(var(--brand-hsl) / 50%);
}
```

  


正如上面代码所示，我们用了一大堆的变量，仅仅是为了获得一个透明度为 `50%` 的颜色变体。

  


因此，Web 开发者一直以来希望 CSS 具备动态调整颜色的能力。庆幸的是，近几年，CSS 正在高速发展，颜色模块为 Web 开发者提供了很多原生的能力，比如我们之前课程中提到的 [`color-mix()`](https://juejin.cn/book/7223230325122400288/section/7237288025221234744) （混合颜色）和 [`color-contrast()`](https://juejin.cn/book/7223230325122400288/section/7239644346935410720) （颜色对比度）以及 CSS 相对颜色。尤其是 CSS 相对颜色特性，使得我们可以使用任何颜色空间或语法定义品牌颜色，并且仅用一行代码就能创建半透明的变体颜色，代码更简洁，意图也更清晰：

  


```CSS
:root {
    --brand-color: hsl(300deg 75% 50%);
    --brand-color-variant: hsl(from var(--brand-color) h s l / 50%);
}
```

  


除此之外，CSS 相对颜色允许我们动态调整颜色的饱和度、亮度、色调等，甚至还可以获得反色，补色以及调整对比度等。这些能力完全可以替代 Sass 处理器中的颜色函数，使得我们在管理颜色变得更简洁和清晰！

  


## 相对颜色语法

  


CSS 相对颜色特性允许从一个颜色派生出另一个颜色。这个基色称为原始颜色，它出现在 `from` 关键词之后。浏览器会将这个原始颜色转换并拆分成各个组成部分，并将它们作为变量提供，以便在新的颜色定义中使用。

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/8c7558d5ee1840618d0987bff1cc4662~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1075&h=712&s=16173&e=png&b=212529)

  


上图展示的是相对颜色的语法规则，可以看到：

  


-   `green` 颜色作为原始颜色
-   `rgb()` 函数内部的 `r` 、`g` 、`b` 和 `alpha` 变量分别提取 `green` 的通道值，即 `r=0` （红色通道）、`g=128` （绿色通道）、`b=0` （蓝色通道）和 `alpha=100%` （透明度通道）

  


这个示意图展示了如何将 `green` 拆分为 `r` 、`g` 、`b` 和 `alpha` 变量，然后将它们直接用于 `rgb()` 颜色定义中。在这个示例中，变量被原样放回 `rgb()` ，因此颜色依然是绿色。

  


### `from` 关键词

  


`from` 关键词的作用是指定原始颜色，它出现在颜色值定义的最前面。下面的代码展示了如何在 `rgb()` 函数中加入 `from green` ：

  


```CSS
.ele {
    color: green;
    color: rgb(0 128 0);
    color: rgb(from green r g b); /* 结果 👉 rgb(0 128 0) */
}
```

当 `from` 关键词出现在颜色函数（比如 `rgb()`）的第一个参数位置时，它会将该颜色定义转换为相对颜色。在 `from` 关键词之后，CSS 需要提供一个颜色，它将作为新颜色的基础。

  


简单来说，`from` 关键词的作用是将颜色转换为 `r` 、`g` 和 `b` 通道值，供新的颜色使用：

  


```CSS
.ele {
    color: rgb(from green r g b);
}
```

  


在 `rgb(from green r g b)` 这样的语法中，颜色转换过程一目了然，这使得自定义属性与相对颜色语法非常契合。换句话说，你可以使用自定义属性，而不必关心原始颜色的具体格式：

  


```CSS
.ele {
    --brand-color: green;
    color: rgb(from var(--brand-color) r g b);
}
```

  


需要知道的是，相对颜色语法允许你选择想要的[颜色空间](https://juejin.cn/book/7223230325122400288/section/7231515598088306720)：

  


```CSS
:root {
    --brand-color: green;
    
    /* 相对颜色语法 👉 传统颜色空间 */
    --brand-rgb: rgb(from var(--brand-color) r g b);   
    --brand-hsl: hsl(from var(--brand-color) h s l);   
    --brand-hwb: hwb(from var(--brand-color) h w b);  
    
    /* 相对颜色语法 👉 新的颜色空间 */
    --brand-lab: lab(from var(--brand-color) l a b);   
    --brand-lch: lch(from var(--brand-color) l c h);   
    --brand-oklab: oklab(from var(--brand-color) l a b);   
    --brand-oklch: oklch(from var(--brand-color) l c h);
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/6d8f0da70f444a6aaaa4ca645e146021~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2968&h=960&s=190841&e=jpg&b=387e23)

  


> Demo 地址：https://codepen.io/airen/full/EaxYPOJ

  


正如你所看到的，虽然使用了不同的颜色空间，但最终所呈现的颜色都是 `green` 。

  


到目前为止，`from` 之后的颜色（`green`）会被转换为指定的颜色空间，输入和输出的格式可以不同。这也意味着你可以灵活选择颜色空间，根据不同任务的需求，使用最合适的颜色空间进行调整。

  


有趣的是，CSS 相对颜色语法的变量可以自由混合、匹配、重复或省略，顺序也不必固定：

  


```CSS
:root {
    --brand-color: green;
    
    --brand-variant-1: rgb(from var(--brand-color) g g g); /* 👉 灰色 */
    --brand-variant-2: rgb(from var(--brand-color) b r g); /* 👉 蓝色 */
    --brand-variant-3: rgb(from var(--brand-color) 0 r 0); /* 👉 黑色*/
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/d191d35dce5749d5850ab3efb4daa59e~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2800&h=872&s=123907&e=jpg&b=f2f3f7)

  


> Demo 地址：https://codepen.io/airen/full/jEONqOK

  


甚至还可以使用 `calc()` 、[`min() 、max() 或 clamp()` 函数](https://juejin.cn/book/7223230325122400288/section/7241401565653762108)来修改变量，从原始色的基础上动态获得不同的颜色：

  


```CSS
:root {
    --brand-color: green;
    --brand-variant-1: lch(from var(--brand-color) l c calc(h * 2));
    --brand-variant-2: lch(from var(--brand-color) min(.5, 1) c h);
    --brand-variant-3: lch(from var(--brand-color) l max(1, 2) h);
    --brand-variant-4: lch(from var(--brand-color) l c clamp(100, l * 2, 300));
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/989f7ac6865540d28dffb8dc6c64cdca~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2808&h=1026&s=165652&e=jpg&b=f2f3f7)

  


> Demo 地址：https://codepen.io/airen/full/ByaBKNm

  


这在 **HSL、HWB、LCH 和 OKLCH 这类圆柱形颜色空间** 中特别实用。

  


另外，CSS 相对颜色语法还提供了 `alpha` 变量（透明通道），这是一个可选值，主要用于调整颜色的透明度：

  


```CSS
:root {
    --brand-color: green;
    --brand-variant-1: oklch(from var(--brand-color) l c h / 1); /* 👉 alpha=100%  */
    --brand-variant-2: oklch(from var(--brand-color) l c h / .9); /* 👉 alpha=90%  */
    --brand-variant-3: oklch(from var(--brand-color) l c h / .8); /* 👉 alpha=80%  */
    --brand-variant-4: oklch(from var(--brand-color) l c h / .7); /* 👉 alpha=70%  */
    --brand-variant-5: oklch(from var(--brand-color) l c h / .6); /* 👉 alpha=60%  */
    --brand-variant-6: oklch(from var(--brand-color) l c h / .5); /* 👉 alpha=50%  */
    --brand-variant-7: oklch(from var(--brand-color) l c h / .4); /* 👉 alpha=40%  */
    --brand-variant-8: oklch(from var(--brand-color) l c h / .3); /* 👉 alpha=30%  */
    --brand-variant-9: oklch(from var(--brand-color) l c h / .2); /* 👉 alpha=20%  */
    --brand-variant-10: oklch(from var(--brand-color) l c h / .1); /* 👉 alpha=10%  */
    --brand-variant-10: oklch(from var(--brand-color) l c h / 0); /* 👉 alpha=0%  */
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/b54530908039432a9602e47bfc1872f5~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2774&h=1104&s=608921&e=jpg&b=eae3da)

  


> Demo 地址：https://codepen.io/airen/full/xbxKVVb

  


这种方式可以直接为颜色设定不透明度，而且不需要依赖于原始值的调整。尤其是在基于颜色关键词（比如 `green`）或十六进制颜色（比如 `#008000`）调整透明度时，非常便利。这也使得你多了一种调整颜色透明度的方式。

  


这就是 CSS 相对颜色语法的规则与基本应用。这里对它做一个简单的总结：

  


-   `from` 关键词可以让你从现有颜色派生新颜色
-   你可以自由选择颜色空间，让颜色转换更符合需求
-   颜色通道变量可以随意组合、调整、重复，甚至还可以使用 `calc()` 、`min()` 、`max()` 和 `clamp()` 等函数计算新颜色
-   `alpha` 变量可以独立控制颜色的透明通道

  


CSS 的相对颜色使得我们在控制颜色时变得更加直观、灵活和强大，为现代 CSS 设计提供了更高效的解决方案！

  


## 相对颜色的基础用例

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/59b448003f744eb993ecb67e697009ae~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1792&h=1024&s=274478&e=webp&b=27afd1)

  


接下来，我将通过一些简单的示例，进一步向大家展示 CSS 相对颜色特性的基础应用。通过这些用例可以轻松的掌握如何通过 CSS 相对语法来调整颜色。

  


### 使颜色变亮

  


以往我们都是通过 CSS 处理器（比如 Sass）的 `lighten()` 函数来调整颜色的亮度。

  


```SCSS
$btn-bkg: purple;

.element {
    background: $btn-bkg;
    color: white;
    
    &:hover {
        background: lighten($btn-bkg, 10%)
    }
}
```

  


现在，我们可以直接在 CSS 中使用相对颜色语法来调整颜色的亮度。其中，简单且暴力的方式是直接将颜色亮度（`l` 通道）指定为你所需要的值：

  


```CSS
.element {
    --brand-color: green;
    background: oklch(from var(--brand-color) 70% c h);
}
```

  


上面的代码，新颜色的亮度直接被指定为 `70%` ，而不是基于原始颜色进行相对调整。这样，绿色会直接变得更亮，而色相（`h`）和色度（`c`）则保持不变：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/dbe568e09cb74593be4950126cc910bd~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2768&h=1102&s=188305&e=jpg&b=fff5ed)

  


> Demo 地址：https://codepen.io/airen/full/QwWLNmz

  


在调整颜色亮度时，OKLCH、OKLAB、XYZ 或 sRGB 颜色空间能提供最可预测的效果。这也是我喜欢使用 `oklch()` 函数的原因之一。

  


另外一种方式是，我们可以按比例调整颜色的亮度，这种方式更接近于 Sass 的 `lighten()` 函数功能。比如：

  


```CSS
.element {
    --brand-color: green;
    background: oklch(from var(--brand-color) calc(l * 1.25) c h); /* 👉 l = .52 × 1.25 = 0.65 */
}
```

  


上面的代码，首先会将 `green` （即 `--brand-color`）转换为 OKLCH 颜色空间，然后通过 `calc(l * 1.25)` 增加亮度通道，使其比原始颜色的亮度增加了 `25%` ，新颜色的亮度值大约为 `0.65` ，因为 `green` 颜色的亮度值为 `0.52` （ `green = oklch(0.52 0.18 142.49)`）。 这样做，会使绿色整体变亮：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/0d9b05e289af4e0594541bf0c3003fd2~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2772&h=612&s=100784&e=jpg&b=488f32)

  


> Demo 地址：https://codepen.io/airen/full/wBvwGYY

  


注意，在使用 `calc()` 调整亮度值时，可以使用任意的四则运算，例如 `oklch(from var(--brand-color) calc(l + .13) c h)` 。

  


### 使颜色变暗

  


与增加亮度相似，我们可以使用相对颜色语法使颜色变暗，实现类似 Sass 的 `darken()` 函数功能：

  


```CSS
.element {
    --brand-color: green;
    
    background: oklch(var(--brand-color) calc(l - .25) c h); /* 👉 l = .52 - .25 = 0.27 */
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/95036c9cbd15459c8845b2163544490f~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2858&h=672&s=104818&e=jpg&b=ffffff)

  


> Demo 地址：https://codepen.io/airen/full/MYWgyRY

  


### 增加颜色饱和度

  


同样的，我们使用 CSS 的相对颜色语法可以实现 Sass 的 `saturate()` 函数功能，即增加颜色饱和度：

  


```CSS
.element {
    --brand-color: green;
    
    background: hsl(from var(--brand-color) h calc(s * 1.5) l); /* 👉 s = 1 × 1.5 = 1.5 */
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/294b0bf531b74c7dbc3e15993039254b~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2810&h=656&s=99652&e=jpg&b=3e911f)

  


> Demo 地址：https://codepen.io/airen/full/XJWrNwd

  


上面示例，通过 `calc(s * 1.5)` 增加了颜色饱和度（`s` 通道），使其提高 `50%` ，会使颜色更加更加鲜艳、更具视觉冲击力。同样的，也可以将颜色饱和度设置为一个特定值：

  


```CSS
.element {
    --brand-color: green;
    background: hsl(from var(--brand-color) h 150% l);
}
```

  


### 减少颜色饱和度

  


与上面示例相反，我们使用 CSS 相对颜色语法来实现 Sass 的 `desaturate()` 函数功能，即减少颜色的饱和度：

  


```CSS
.element {
    --brand-color: green;
    background: hsl(from var(--brand-color) h calc(s / 2) l); /* 👉 s = 1 ÷ 2 = 0.5 */
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/3a2c23afc7d64edaa2fe57f39f69c697~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2822&h=664&s=103500&e=jpg&b=ffffff)

  


> Demo 地址：https://codepen.io/airen/full/jEONyOg

  


上面示例，颜色的饱和度减少了一半（即 `50%`），颜色会变得更灰、更柔和。需要知道的是，这种方式相对地减少了颜色的饱和度，使其更向灰色靠近，但不会改变颜色的其他属性！

  


### 增强颜色色度

  


这种效果类似于增加颜色饱和度，但有几个不同之处。首先，它是色度（Chroma）变化，而不是饱和度（Saturation）变化。这是因为高清颜色空间（HDR）并不使用饱和度，而是使用色度，允许 Web 开发者将颜色的鲜艳度提升到超出饱和度所能达到的范围。

  


```CSS
.element {
    --brand-color: green;
    background: oklch(from var(--brand-color) l calc(c + .1) h); /* 👉 l = .18 + .1 = 0.28 */
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/8985f74fa2484a608acea3d8499d77b6~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2828&h=656&s=105431&e=jpg&b=ffffff)

  


> Demo 地址：https://codepen.io/airen/full/vEYBgLj

  


这种方式会使颜色变得更加鲜艳，比简单的饱和度提升效果更强。

  


### 调整颜色的不透明度

  


前面我们展示了一个调整颜色不透明度的案例，直接指定颜色的不透明度通道的值。例如：

  


```CSS
.element {
    --brand-color: green;
    
    background: oklch(from var(--brand-color) l c h / 50%);
}
```

  


其实，我们也可以按比例调整颜色的不透明度：

  


```CSS
.element {
    --brand-color: green;
    
    background: oklch(from var(--brand-color) l c h / calc(alpha / 2)); /* 👉 alpha = 1 ÷ 2 = 50% */
}
```

  


代码中的 ` calc(alpha  ``/ 2``)` 将不透明度减半：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/21f1087701e2422fb41c23c18d8e5f7f~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2824&h=682&s=243709&e=jpg&b=eee7e0)

  


> Demo 地址：https://codepen.io/airen/full/JojPEEr

  


这种方式使得我们控制颜色的透明度变得尤其灵活，浏览器就帮我们计算好了，我们不需要去关注原始颜色的透明度是多少。换句话说，如果原始颜色自身就带有一定透明度，那么它会在原始颜色透明度的基础上进行计算：

  


```CSS
.element {
    --brand-color: hsl(300 50% 50% / 75%);
    
    background: oklch(from var(--brand-color) l c h / calc(alpha * .8));/* 👉 alpha = .75 × .8 = .6 */
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/e55fd9af5b4d415ab2f2a4bc6d22669a~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2778&h=612&s=321758&e=jpg&b=ffffff)

  


> Demo 地址：https://codepen.io/airen/full/jEONyGP

  


上图中原始颜色的透明度是 `.75` ，中间颜色（`oklch(from var(--brand-color) l c h / 50%)`）的透明度是 `50%` ，因为直接为新颜色指定了透明度，最右侧颜色（`oklch(from var(--brand-color) l c h / calc(alpha * .8))`）的透明度 `.6` ，因为初始颜色透明度是 `.75` ，`calc(alpha * .8)` 会相对于初始颜色的透明进行计算 `alpha = .75 × .8 = .6` ：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/0dce9a171c8441f287a4530bb54f7da3~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1576&h=692&s=6976496&e=gif&f=246&b=ffffff)

  


这种方式与 Sass 的 `opacify()` 、`fade-in()` 、`transparentize()` 和 `fade-out()` 函数功能相似：

  


-   `opacify()` 和 `fade-in()` 会创建一个更不透明的颜色；CSS 相对颜色语法可以将 `alpha` 加上一个值，比如 `calc(alpha + .4)` ，在原始颜色的透明度上增加 `40%`
-   `transparentize()` 和 `fade-out()` 会创建一个更透明的颜色；CSS 相对颜色语法可以将 `alpha` 减去一个值，比如 `calc(alpha - .4)` ，在原始颜色的透明度上减去 `40%`

  


### 颜色反转

  


颜色反转是颜色库中常见的颜色调整功能。实现颜色反转的一种方式是将颜色转换为 RGB，然后从 `255` 减去每个通道的值。

  


```CSS
.element {
    --brand-color: green;
    background: rgb(from var(--brand-color) calc(255 - r) calc(255 - g) calc(255 - b));
}
```

  


这种方式通过反转颜色每个通道的值，将颜色从 `green` 反转为其对立色：

### 互补色

  


如果你的目标不是颜色反转，而是取补色，那么色相旋转可能是你要找的解决方案。选择一个提供色相的颜色空间（比如 HSL），然后使用 `calc()` 来旋转色相，旋转的角度根据你需要的结果调整。要找到一个颜色的补色，只需要旋转半圈，即将色相（`h` 通道）增加或减少 `180` 度：

  


```CSS
.element {
    --brand-color: green;
    background: hsl(from var(--brand-color) calc(h + 180) s l);
}
```

### 增加颜色对比度

  


这时所说的颜色对比度指的是元素背景颜色和文本颜色之间的对比度，颜色对比度是创建可访问性 Web 页面的重要指标之一。现代 CSS 的 `color-contrast()` 函数使我们创建可访问性 Web 页面（提供符合 WCAG 标准的颜色对比度）变得非常容易。我曾在《[颜色对比度：`color-contrast()`](https://juejin.cn/book/7223230325122400288/section/7239644346935410720)》和《[CSS 排版：颜色和对比度](https://juejin.cn/book/7395119173656903715/section/7445257290149527589)》课程中专门探讨了这方面的知识。

  


除此之外，我们也可以使用 CSS 相对颜色语法来增加颜色对比度。我们可以考虑使用 `L*` （Lstar），它是实现可访问性颜色对比度的一种方法。它采用来自 LCH 和 OKLCH 颜色空间的近似感知均匀的亮度（`l`）通道，并通过 `calc()` 进行计算。

  


如果你想要不同程度的对比度，`L*` 亮度差异通常为：

  


-   低对比度：约 `40`
-   中等对比度：约 `50`
-   高对比度：约 `60`

  


这有助于在不同的设计需求下，确保文本和背景颜色之间的可读性和视觉效果。下面这个示例展示了如何增强深色和浅色的对比度：

  


```CSS
:root {
    --c-dark: darkred;
    --c-light: lightpink;
}

/* 增强深色的对比度 */
.dark {
    background: var(--c-dark);
    color: oklch(from var(--c-dark) calc(l + .6) c h); /* 高对比度 */
}

/* 增强浅色对比度 */
.light {
    background: var(--c-light);
    color: oklch(from var(--c-light) calc(l - .6) c h); /* 高对比度 */
}
```

  


由于 `.dark` 元素的背景颜色 `darkred` 是深色（亮度值较低），因此将 `60%` （即 `.6`）加到亮色通道（`l`）。这种技术用于找到与原色相同色相但具有良好对比度的亮色文本颜色，以便在深色背景上保持可读性。`.light` 元素相似，由于原始颜色 `lightpink` 是浅色（亮度值较高），因此在亮度通道上减少 `60%` ，这使得深色文本在浅色背景上具有更好的对比度：

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/ffec7e3c25dd4074b74281549189a992~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=2784&h=796&s=72850&e=jpg&b=f5bbc3)

  


> Demo 地址：https://codepen.io/airen/full/emYOyJy

  


以上这些简单示例是关于 CSS 相对颜色语法的基础功能。它基本上能替代 Sass 处理器中大部分的颜色函数功能。

  


## 案例一：创建调色板

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/bf04beb84377463daa47f18f2759e909~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1600&h=800&s=25803&e=png&b=fffcfc)

相对颜色语法在创建调色板方面非常有效。由于有许多颜色空间可供选择，它特别有用且强大。接下来，我们来看看如何使用 CSS 相对颜色语法创建调色板，你可以尝试修改 `--base-color` ，看看这些调色板有多动态。非常有趣！

  


### 单色调色板

  


创建单色调色板的方式是从同一个色相出发，变化亮度和暗度。中间的颜色是调色板的源颜色，左右两侧分别是更亮和更暗的颜色变体：

  


```CSS
:root {
    --base-color: green;
    
    --c-0: oklch(from var(--base-color) calc(l + 0.5) c h); /* 最亮*/
    --c-1: oklch(from var(--base-color) calc(l + 0.45) c h); 
    --c-2: oklch(from var(--base-color) calc(l + 0.4) c h); 
    --c-3: oklch(from var(--base-color) calc(l + 0.35) c h); 
    --c-4: oklch(from var(--base-color) calc(l + 0.3) c h);
    --c-5: oklch(from var(--base-color) calc(l + 0.25) c h);
    --c-6: oklch(from var(--base-color) calc(l + 0.2) c h);
    --c-7: oklch(from var(--base-color) calc(l + 0.15) c h);
    --c-8: oklch(from var(--base-color) calc(l + 0.1) c h);
    --c-9: oklch(from var(--base-color) calc(l + 0.05) c h);
    --c-10: oklch(from var(--base-color) l c h); /* 与源颜色一致 */
    --c-11: oklch(from var(--base-color) calc(l - 0.05) c h);
    --c-12: oklch(from var(--base-color) calc(l - 0.1) c h);
    --c-13: oklch(from var(--base-color) calc(l - 0.15) c h);
    --c-14: oklch(from var(--base-color) calc(l - 0.2) c h);
    --c-15: oklch(from var(--base-color) calc(l - 0.25) c h);
    --c-16: oklch(from var(--base-color) calc(l - 0.3) c h);
    --c-17: oklch(from var(--base-color) calc(l - 0.35) c h); 
    --c-18: oklch(from var(--base-color) calc(l - 0.4) c h); 
    --c-19: oklch(from var(--base-color) calc(l - 0.45) c h); 
    --c-20: oklch(from var(--base-color) calc(l - 0.5) c h); /* 最暗*/
}
```

  


![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/23e0de09a73945f7bc7c6b661ff58fb6~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1368&h=572&s=7238027&e=gif&f=98&b=ecf0f4)

在这个例子中，`--base-color` 是源颜色，其他颜色通过对亮度（`l` 通道）进行调整（增亮或加暗）来创建，从而形成一个单色调色板。

  


### 相似色调色板

  


由于在 OKLCH 和 HSL 中旋转色相（`h`）非常简单，因此创建一个相似色调色板非常容易。，只需要根据喜欢的结果旋转色相，并更改基础颜色，浏览器就会自动构建新的调色板：

  


```CSS
:root {
    --base-color: green;
    --primary: var(--base-color);
    --secondary: oklch(from var(--base-color) l c calc(h - 45));
    --tertiary: oklch(from var(--base-color) l c calc(h + 45));
}
```

### 三分色调色板

  


与互补色类似，三分色调色板是一个基于一个基础色的对立但和谐的色相旋转。补色位于色轮的对立面，就像一条穿过色轮中间的直线，而三分色调色板则像三角形的三条线，找到从基础色旋转 `120` 度的两种颜色，通过旋转色相 `120` 度来实现：

  


```CSS
:root {
    --base-color: green;
    
    --c-0: oklch(from var(--base-color) l c calc(h - 120));
    --c-1: var(--base-color); /* 或 oklch(from var(--base-colo) l c h)*/
    --c-2: oklch(from var(--base-color) l c calc(h + 120));
}
```

四分色调色板由四种颜色组成，这些颜色均匀地分布在色轮上，形成一个没有明显主导色值的调色板。你可以把它看作是两对补色。若使用得当，它可以非常有意义。

### 带有轻微色相旋转的单色调色板

  


单色调色板可能会显得有些单调，我们可以在改变亮度时，为每个新颜色添加轻微或较大的色相旋转。下面这个示例在每个色板上将亮度降低 `10%` ，并同时将色相旋转 `10` 度，形成一个看似渐变的色彩过渡：

我们通过 `data-type` 数据属性来区分 `Alert` 不同的状态，每个 `Alert` 组件整体上并没有太大的差异，最明显的就是颜色上的差异。因此，我们通过 `data-type` 为每种状态定义不同的颜色：

正如你所看到的，到目前为止，`Alert` 的所有状态都是一样的。在继续往下之前，我们仔细观察目标 UI 效果，不难发现，`Alert` 每个状态的背景颜色、文本颜色和边框颜色都不一样。最为关键的是，同一个 `Alert` 的背景颜色、文本颜色和边框颜色都是由 `--color` 来决定的。

  


你可能已经想到了，我们使用 CSS 相对颜色语法给 `Alert` 组件的每个状态着色：

  


```CSS
aside {
    border-color: oklch(from var(--color) l c h / 25%);
    background: oklch(from var(--color) calc(l * 1) c h / 20%);
    color: oklch(from var(--color) calc(l * 1) c h);
}

aside::selection {
    background: oklch(from var(--color) calc(l * 1.1) c h);
    color: oklch(from var(--color) 1 c h);
}
```

  


通过前面内容的学习，理解上面的代码应该不是问题：

  


-   边框颜色（`border-color`），很简单，它只是在 `--color` 的基础上将透明度指定为 `25%`
-   背景颜色（`background-color`），同时调整了 `l` 通道的值（事实上并没有什么变化，因 `calc(l * 1)` 并不会使 `l` 通道的值发生任何改变），与此同时，透明度也指定为 `20%`
-   文本颜色（`color`）其实就是 `--color`

  


事实上，你也可以将代码调整为：

  


```CSS
aside {
    border-color: oklch(from var(--color) l c h / 25%);
    background: oklch(from var(--color) l c h / 20%);
    color: oklch(from var(--color) l c h);
}

aside::selection {
    background: oklch(from var(--color) calc(l * 1.1) c h);
    color: oklch(from var(--color) 1 c h);
}
```

不再需要单独调整边框颜色、文本颜色和背景颜色。

  


如果你阅读了原案例，你会发现原案例中还使用了 [`color-mix()` 函数](https://juejin.cn/book/7223230325122400288/section/7237288025221234744)。我想作者的出发点是考虑了 CSS 相对颜色语法的兼容性，权当是一种降级处理。这也意味着，`color-mix()` 函数应用得好，也能帮助我们快速构建组件的皮肤（主题）颜色：

  


```CSS
aside {
    border-color: color-mix(in oklch, var(--color) 50%, oklch(100% 0 0 / 15%));
    background: color-mix(in oklch, var(--color) 25%, oklch(100% 0 0 / 50%));
    color: color-mix(in oklch, var(--color) 100%, oklch(100% 0 0));
}

aside::selection {
    background: color-mix(in oklch, var(--color) 90%, oklch(100% 0 0));
    color: color-mix(in oklch, var(--color) 10%, oklch(100% 0 0));
}
```
