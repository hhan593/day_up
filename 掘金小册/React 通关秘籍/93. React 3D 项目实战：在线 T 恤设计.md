前面我们学了 React 如何结合 Three.js 开发 3D 项目。

有两种方式：

- 使用原生 Three.js api，绘制出来的 canvas 挂到某个 dom 上。
- 使用 react-three-fiber 渲染器，组件化开发 3D 场景

这节我们来做一个 React 的 3D 实战项目：在线 T 恤设计。

就是用户可以自己设计 T 恤的图案、文字、预览效果。

改下 App.tsx

```javascript
import { OrbitControls } from '@react-three/drei'
import { Canvas, useLoader } from '@react-three/fiber'
import { Suspense } from 'react';
import { DRACOLoader, GLTFLoader } from 'three/examples/jsm/Addons.js';

function App() {
  return <Canvas camera={{
    position: [0, 500, 500]
  }} style={{
      width: window.innerWidth,
      height: window.innerHeight
  }}>
    <ambientLight/>
    <axesHelper args={[1000]}/>
    <directionalLight position={[500, 400, 300]}/>
    <OrbitControls/>
    <Suspense fallback={null}>
      <Tshirt/>
    </Suspense>
  </Canvas>
}

function Tshirt() {
  const gltf = useLoader(GLTFLoader, 'tshirt.glb')

  gltf.scene.scale.setScalar(1000);

  return <primitive object={gltf.scene}/>
}

export default App
```
创建 3D 场景，加载一个 T 恤模型。

scale.setScalear 是 x、y、z 都放大这些倍数。

draco 是一个压缩 3D 模型的方案，谷歌推出的。

https://github.com/google/draco


![image.png](https://p6-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/a9613e8eb51148d688f6cd8c00edf64b~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=802&h=634&s=217932&e=png&b=fffefe)

所以如果模型压缩过，那我们可肯定要解压缩才能用。


![image.png](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/592235a5f42e44de91be16419adae13a~tplv-k3u1fbpfcp-jj-mark:0:0:0:0:q75.image#?w=1900&h=606&s=131764&e=png&b=1f1f1f)

```javascript
const gltf = useLoader(GLTFLoader, 'tshirt.glb', (loader) => {
    const dracoLoader = new DRACOLoader()
    dracoLoader.setDecoderPath( 'https://www.gstatic.com/draco/versioned/decoders/1.5.6/' );
    loader.setDRACOLoader(dracoLoader)
  })
```
指定 GltfLoader 用网上的一个 draco 库来解压缩。

```javascript
function Background() {
  const { scene } = useThree();
  useTexture('./sky.png', (tex) => {
    tex.colorSpace = SRGBColorSpace;
    scene.background = tex;
  });
  return null;
}
```
用 useTexture 加载图片作为背景，这里要设置颜色空间，不然色彩会有差别。
