 在 Git 中删除分支有以下命令:

1. **删除本地分支**

```
git branch -d 分支名
```

如果该分支已经合并到其他分支,可以直接删除。如果该分支还未合并,可以使用 `-D` 强制删除:

```
git branch -D 分支名
```

2. **删除远程分支**

```
git push origin --delete 远程分支名
```

或者使用简写形式:

```
git push origin :远程分支名
```

3. **删除已经合并到主干的远程分支**

```
git remote prune origin
```

该命令会删除远程已经被合并到主干的分支。

4. **删除多个分支**

使用管道符号可以一次性删除多个分支:

```
git branch | grep '分支名称' | xargs git branch -D
```

注意替换 `'分支名称'` 为你要删除的分支名称的通配符。

5. **删除本地和远程分支**

```
# 删除本地分支
git branch -D 本地分支名

# 删除远程分支
git push origin --delete 远程分支名
```

总之删除分支需要注意当前分支是否已经合并,以及是否会丢失重要提交记录。如果有疑虑,可以先备份相关分支。
