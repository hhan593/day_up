 ### 伪类选择器是用于选择元素的特殊状态或位置的选择器。常用的伪类选择器包括:

1. **链接状态伪类**
   - `:link` - 选择未访问过的链接
   - `:visited` - 选择已访问过的链接
   - `:hover` - 选择鼠标悬停的元素
   - `:active` - 选择被用户激活(点击)的元素

2. **UI状态伪类**
   - `:focus` - 选择当前获取焦点的元素
   - `:enabled` - 选择启用状态的元素
   - `:disabled` - 选择禁用状态的元素
   - `:checked` - 选择被选中的单选按钮或复选框
   - `:indeterminate` - 选择处于不确定状态的进度条、复选框等

3. **结构伪类**
   - `:first-child` - 选择父元素的第一个子元素
   - `:last-child` - 选择父元素的最后一个子元素
   - `:nth-child(n)` - 选择父元素的第 n 个子元素
   - `:nth-last-child(n)` - 选择从后往前数的第 n 个子元素
   - `:only-child` - 选择父元素仅有的一个子元素

4. **其他伪类**
   - `:not(selector)` - 选择除去指定元素的所有元素
   - `:target` - 选择当前活动的目标元素(URL片段指向的元素)
   - `:lang(language)` - 选择指定语言的元素

伪类选择器为选择元素的特定状态或位置提供了便利,可以根据需求灵活运用。同时,也可以将伪类选择器与其他选择器组合使用,以实现更加精确和复杂的选择条件。


### CSS的 `:nth-child(n)` 伪类选择器允许你基于其在父元素中的位置来选择元素。然而，这个选择器通常用于选择一个特定的元素。如果你想选择多个元素，你需要为每个元素指定一个条件，或者使用更复杂的表达式。

但是，你不能直接在 `:nth-child(n)` 中列出多个值。相反，你需要使用 `an+b` 公式来匹配多个元素。这里的 `a` 和 `b` 是你定义的数字，`n` 是从0开始的计数器。

例如，如果你想选择第一个、第三个和第五个元素，你可以使用逗号 `,` 来组合多个 `:nth-child()` 表达式：


```css
li:nth-child(1),
li:nth-child(3),
li:nth-child(5) {
    color: red;
}
```
在这个例子中，所有列表项中的第一个、第三个和第五个元素都会变成红色。

然而，如果你想要一个更简洁的方式，你可以尝试使用更复杂的 `an+b` 公式。例如，如果你想要选择索引为1、3、5、7等的元素（即所有奇数索引的元素），你可以使用 `:nth-child(2n+1)`：


```css
li:nth-child(2n+1) {
    color: red;
}
```
在这个例子中，所有列表项中的奇数索引元素都会变成红色。

请注意，`:nth-child()` 是基于1的索引，而不是基于0的。所以 `:nth-child(1)` 选择的是第一个元素，`:nth-child(2)` 选择的是第二个元素，依此类推。



### :nth-child-of-type

`:nth-child-of-type()` 是一个 CSS 伪类选择器，它允许你根据元素类型及其在父元素中的位置来选择特定的元素。与 `:nth-child()` 不同，`:nth-child-of-type()` 仅计算与指定类型匹配的子元素。

例如，假设你有一个包含多种元素的父元素，你只想选择其中的特定类型的元素，并且这些元素要基于它们的类型顺序。在这种情况下，`:nth-child-of-type()` 就非常有用。

以下是一些使用 `:nth-child-of-type()` 的例子：

### 选择特定类型的第一个元素

如果你有一个包含多个 `p` 元素的 `div`，并且你想选择第一个 `p` 元素，你可以这样做：

```css
div p:nth-child-of-type(1) {
    color: red;
}
```

### 选择特定类型的奇数元素

如果你想选择所有奇数位置的 `p` 元素，你可以使用 `2n+1` 公式：

```css
div p:nth-child-of-type(2n+1) {
    color: red;
}
```

### 选择特定类型的特定位置的元素

如果你想选择第二个和第四个 `p` 元素，你可以这样做：

```css
div p:nth-child-of-type(2),
div p:nth-child-of-type(4) {
    color: red;
}
```

### 选择特定类型的最后两个元素

选择特定类型的最后两个元素稍微复杂一些，因为你需要知道这种类型的元素总共有多少个。但是，如果你知道有至少两个这种类型的元素，并且你想要选择最后两个，你可以使用 `:nth-last-child-of-type()` 伪类选择器，它是 `:nth-child-of-type()` 的逆序版本：

```css
div p:nth-last-child-of-type(1),
div p:nth-last-child-of-type(2) {
    color: red;
}
```

这将选择 `div` 中最后两个 `p` 元素。

请注意，`:nth-child-of-type()` 和 `:nth-last-child-of-type()` 伪类选择器都是基于1的索引，而不是基于0的。所以 `:nth-child-of-type(1)` 选择的是第一个匹配类型的元素，`:nth-child-of-type(2)` 选择的是第二个，依此类推。

这些选择器提供了一种强大的方式来选择特定类型的元素，并基于它们在父元素中的位置应用样式。它们特别适用于当父元素包含多种类型的子元素，并且你只想针对其中一种类型的子元素应用样式时。
