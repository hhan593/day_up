### 18、getQueryselector和GetClassName的区别？

`querySelector` 和 `getElementsByClassName` 是 JavaScript 中用于选择 DOM（文档对象模型）元素的两种不同方法，它们各自具有独特的特点和用途：

1. **`querySelector`**:
   - **选择第一个匹配元素**：`querySelector` 方法返回文档中与指定选择器或选择器组匹配的第一个 `Element` 对象。如果没有找到匹配的元素，则返回 `null`。
   - **CSS选择器**：此方法接受任何有效的 CSS 选择器（如 `#id`, `.class`, `tag`, `[attribute=value]` 等）作为参数。
   - **更多的灵活性**：由于它能使用复杂的 CSS 选择器，`querySelector` 提供了更多的灵活性。

2. **`getElementsByClassName`**:
   - **选择所有匹配的元素**：`getElementsByClassName` 方法返回一个 `HTMLCollection` 对象，其中包含文档中所有具有指定类名的元素。
   - **基于类名选择**：它仅接受一个或多个类名作为参数，类名之间用空格分隔。
   - **实时的 HTMLCollection**：返回的 `HTMLCollection` 是实时的，意味着如果文档中的元素更改，该集合也会相应更新。

总结来说，`querySelector` 提供了更大的灵活性，允许使用复杂的 CSS 选择器来获取单个元素，而 `getElementsByClassName` 则专门用于根据类名选择一组元素。`querySelector` 通常用于更精确地定位元素，而 `getElementsByClassName` 更适用于处理具有相同类名的多个元素。
