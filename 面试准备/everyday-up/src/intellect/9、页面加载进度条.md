获取页面加载或特定资源加载的进度，通常依赖于具体场景。以下是几种常见场景和对应的解决方案：

### 1. 使用 XMLHttpRequest (XHR) 加载资源

当你使用`XMLHttpRequest`（XHR）来异步加载资源（如 AJAX 请求）时，可以监听`progress`事件来获取加载进度：

```javascript
var xhr = new XMLHttpRequest();

xhr.open("GET", "/my/resource/url", true);

xhr.onprogress = function (e) {
  if (e.lengthComputable) {
    var percentComplete = (e.loaded / e.total) * 100;
    console.log(percentComplete + "%");
  }
};

xhr.onload = function () {
  if (this.status == 200) {
    // 处理成功结果
  }
};

xhr.send();
```

### 2. Fetch API

对于支持 Fetch API 的现代浏览器，获取加载进度稍微复杂一些，因为 Fetch API 本身不直接支持进度事件。不过，你可以通过读取 Response 的`body`属性（一个`ReadableStream`对象）来间接实现：

```javascript
fetch("/my/resource/url")
  .then((response) => {
    const contentLength = response.headers.get("Content-Length");
    if (!contentLength) {
      // Content-Length未知时的处理逻辑
      return;
    }
    const totalBytes = parseInt(contentLength, 10);
    let loadedBytes = 0;

    const reader = response.body.getReader();
    const stream = new ReadableStream({
      start(controller) {
        function read() {
          reader
            .read()
            .then(({ done, value }) => {
              if (done) {
                controller.close();
                return;
              }
              loadedBytes += value.byteLength;
              console.log(`Received ${loadedBytes} of ${totalBytes}`);
              // 处理接收到的数据chunk，例如更新进度条
              controller.enqueue(value);
              read();
            })
            .catch((error) => {
              console.error("Stream reading failed:", error);
              controller.error(error);
            });
        }
        read();
      },
    });

    return new Response(stream, {
      headers: { "Content-Type": "application/octet-stream" },
    });
  })
  .then((response) => {
    // 最终处理响应数据
  })
  .catch((error) => console.error("Fetch failed:", error));
```

### 3. 文件上传进度

对于文件上传，使用 XHR 或 Fetch 时，可以通过监听上传事件（`xhr.upload.onprogress` 或者通过 Fetch API 的 `body` 使用 `ReadableStream`）来获取进度信息：

```javascript
var xhr = new XMLHttpRequest();
xhr.open("POST", "/upload/url", true);

xhr.upload.onprogress = function (e) {
  if (e.lengthComputable) {
    var percentComplete = (e.loaded / e.total) * 100;
    console.log(percentComplete + "%");
  }
};

xhr.onload = function () {
  if (this.status == 200) {
    // 上传成功
  }
};

xhr.send(formData); // formData是需要上传的数据
```

### 注意点

- 不是所有的资源加载都能准确获取到进度信息。例如，使用`<img>`标签直接加载图片时，浏览器并不提供原生的进度信息。
- 对于不支持`Content-Length`头部的响应（例如，使用`Transfer-Encoding: chunked`的响应），无法预先知道总大小，因此无法计算精确的进度百分比。
- 确保考虑跨域资源共享（CORS）的限制，因为这可能影响到能否读取响应头部等信息。

根据你的具体需求（如资源加载、文件上传等），选择合适的方法来实现进度监控，可以显著提升用户体验。
