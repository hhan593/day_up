### 27、css 样式的优先级？
[css样式优先级](https://juejin.cn/post/6844903709772611592?searchId=20240306155845F9B2352E6FB59B0E3999)
- 1. 将优先级分为四类  (A   B   C   D)  根据各类选择器个数进行由左到右这个进行比较。
- 2. 内联样式 优先级最高                       A类 
- 3. id 选择器、                              B类
- 4. 类选择器 、 属性选择器、伪类选择器 优先级   C类
- 5. 标签选择器 、 伪元素选择器                 D类
