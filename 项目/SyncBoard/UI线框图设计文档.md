**左侧工具栏按钮（从上到下）：**

| 图标 | 工具 | 快捷键 |
|------|------|--------|
| ▶ | 选择 (Select) | V |
| 🔲 | 矩形 (Rectangle) | R |
| ○ | 椭圆 (Ellipse) | O |
| △ | 三角形 (Triangle) | T |
| ◇ | 菱形 (Diamond) | D |
| ─ | 箭头/连线 (Arrow) | A |
| ✏ | 画笔 (Freehand) | P |
| 🅰 | 文字 (Text) | X |
| 🖼 | 图片上传 (Image) | I |
| 🔍 | 框选 (Lasso) | L |

**右侧属性面板（选中元素时显示）：**
- 填充颜色选择器
- 描边颜色 + 宽度 + 样式 (实线/虚线)
- 透明度滑块
- 字体大小 + 粗体/斜体
- 对齐方式 (选中多元素时)
- 图层操作 (上移/下移/置顶/置底)

| 断点 | 宽度 | 适配策略 |

## 13. Pencil 制作指南

### 推荐使用的 Pencil 内置模板集

| 模板集 | 用途 |
|--------|------|
| Common Shapes | 基础矩形、圆形、线条 |
| Desktop - GTK Widgets | 按钮、输入框、下拉菜单、滑块 |
| Web UI | 浏览器框架、导航栏、卡片 |
| Misc | 图标、箭头、标注 |
| Android / iOS | 移动端适配原型 (Phase 4) |

### 制作步骤

1. **创建项目**：在 Pencil 中新建项目，设置画布尺寸 1440x900 (桌面)
2. **创建页面**：为每个界面创建独立 Page（登录、注册、仪表盘、画布、设置）
3. **绘制组件**：使用内置模板集拖拽基础控件
4. **添加链接**：使用 Pencil 的 "Link To" 功能串联页面跳转
5. **导出**：导出为 PNG 或 HTML 交互原型
