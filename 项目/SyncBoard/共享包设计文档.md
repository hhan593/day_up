# SyncBoard 共享包设计文档

## 1. 概述

`@syncboard/shared` 是 SyncBoard 单体仓库中的公共包，为前端 (web) 和后端 (server) 提供统一的类型定义、常量和工具函数，确保前后端数据结构和通信协议的一致性。

---

## 2. 包配置

```json
{
  "name": "@syncboard/shared",
  "version": "0.0.1",
  "main": "src/index.ts",
  "types": "src/index.ts",
  "scripts": {
    "build": "tsc",
    "clean": "rm -rf dist"
  }
}
```

- **引用方式**: `"@syncboard/shared": "workspace:*"` (pnpm workspace 协议)
- **编译目标**: ES2020, ESNext 模块
- **输出**: 声明文件 + Source Maps

## 6. 工具函数

### 6.1 共享验证 (utils/validation.ts)

```typescript
/** 邮箱格式验证 */
export function isValidEmail(email: string): boolean {
  return /^[^\s@]+@[^\s@]+\.[^\s@]+$/.test(email);
}

/** 用户名规则: 3-20位，字母数字下划线 */
export function isValidUsername(username: string): boolean {
  return /^[a-zA-Z0-9_]{3,20}$/.test(username);
}

/** 密码强度: 至少8位，含字母和数字 */
export function isValidPassword(password: string): boolean {
  return password.length >= 8 && /[a-zA-Z]/.test(password) && /\d/.test(password);
}
```

## 8. 前后端使用方式

```typescript
// 前端 (apps/web) 使用
import { User, Room, WS_EVENTS, DEFAULT_ELEMENT_STYLE } from '@syncboard/shared';

// 后端 (apps/server) 使用
import { LoginDto, JwtPayload, WS_EVENTS } from '@syncboard/shared';
```

通过共享包，前后端始终使用相同的类型定义和常量，避免接口不一致问题。
