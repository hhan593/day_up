# 需求 (SwarmNote V2)

> 基于 SwarmNote 本地优先、P2P 同步（libp2p）和零冲突合并（yjs/CRDT）的产品愿景，重构桌面端 UI 设计规范。
> 
> **核心原则：** 将硬核的分布式技术隐藏在幕后，通过极简的界面、流畅的动效赋予用户极致的掌控感和安全感。

## 设计风格 (Design Tokens)

- **主题基调：** 暗黑极客风（Dark Mode 优先）。主背景使用深邃灰（`#0A0A0A` 到 `#171717`），避免纯黑造成的视觉疲劳。
    
- **品牌高亮：** 琥珀色（Amber `#F59E0B`）。用于表示集群（Swarm）状态、活跃链接和交互焦点。
    
- **窗口形态：** 无边框窗口（Frameless Window），支持系统级模糊/毛玻璃材质（如 macOS 的 Vibrancy 或 Windows 11 的 Mica）。
    
- **组件规范：** 面向键盘重度用户，所有主要操作均可通过快捷键或 `⌘K` 命令面板完成。

**左侧边栏 (Left Sidebar - 260px)：**

- **顶部：** 当前知识库切换下拉菜单 + 伪搜索框（点击唤起 ⌘K 面板）。
    
- **文件树：** 支持无限层级。悬停时显示 `...` 操作按钮。选中文件按 `Enter` 直接内联重命名。
    
- **底部 Swarm 面板：** 极其重要的 P2P 状态中枢（详见 P1）。折叠时仅占用 `56px` 高度。
    

**核心编辑区 (Main Editor - Flex 居中)：**

- **顶部标题栏：** 显示路径面包屑。如果同一局域网内有其他设备打开了此文档，右侧会显示其设备 Icon（如 📱）。
    
- **编辑器内容区：** 基于 CodeMirror 6，所见即所得。正文最大宽度限制 `720px`，两侧留白。
    
- **协同指示器：** 其他设备正在输入时，显示带有设备名称的绝对定位“远程光标”。
    

**右侧上下文面板 (Right Panel - 280px，可折叠)：**

- **大纲 & 链接：** 自动提取 Markdown H1-H6 生成目录，显示双向链接。
    
- **历史时光机 (History)：** 核心差异化功能。不提供传统的版本列表，而是提供一条基于 yjs 的**垂直时间轴**。

### P1：网络连接与底层感知（MVP 必须）

#### 3. Swarm 状态面板 (左侧边栏底部)

将复杂的 mDNS 发现和 libp2p 连接状态可视化，赋予用户掌控感。

**状态 A：折叠态（日常无感知）**

- 🟢 绿色常亮：已连接且数据一致。
    
- 🟡 琥珀色呼吸灯：正在后台交换 CRDT 数据。
    
- ⚪️ 灰色：当前仅本地离线运行。
    

**状态 B：展开态（点击后向上抽屉式展开）**

Plaintext

```
┌──────────────────────────────────────────┐
│ 🟢 Swarm 网络                          ▼ │
│                                          │
│ 💻 Mac Studio (本机)                     │
│ 192.168.1.100 · 在线                     │
│                                          │
│ 我的 Swarm (2)                [➕ 配对] │
│ 🟢 📱 iPhone 15 Pro (P2P 直连)           │
│ 🟡 💻 ThinkPad X1 (正在同步...)          │
│                                          │
│ ⚡ 局域网新发现 (1)                     │
│ ⚪️ 📱 iPad Air (请求交换公钥)            │
│    [ 忽略 ]        [ 允许同步 ]          │
└──────────────────────────────────────────┘
```

- **自动发现动效：** 当 mDNS 嗅探到新设备时，“局域网新发现”模块会自下而上滑出，并伴随琥珀色 (`border-amber-500`) 的脉冲边框发光动效。
    

#### 4. WAN 广域网配对弹窗 (Magic Wormhole 机制)

当设备不在同一 Wi-Fi 下时，摒弃输入 IP 的反人类操作。

- 点击“➕ 配对”后，生成 3 个随机中文词语（如：`橘子-勇敢-海洋`）和二维码。
    
- 另一台设备输入这 3 个词语，即可穿透 NAT 建立 P2P 隧道。

### P2：特色协同机制（Phase 2）

#### 5. 无感冲突合并 (Silent Merge)

在 SwarmNote 中，**不存在传统的“文件冲突警告弹窗”**。

- **交互逻辑：** 当离线设备重新连网，底层 `yjs` 瞬间完成合并计算。
    
- **UI 呈现：** 编辑器中，刚刚从其他设备合并进来的文本段落，背景色会泛起**微弱的绿色高亮（持续 2 秒）**，随后像水波纹一样淡出。
    
- **兜底机制：** 如果用户不满意合并结果，直接在右侧“历史时光机”点击上一个节点，即可查看 Diff 或一键恢复。
    

#### 6. 时光机 Diff 模式

点击右侧时间轴的某个历史节点（如“昨天 22:00”）触发。

- **UI 变化：** 顶部标题栏变色，显示横幅 `⏳ 您正在查看 [昨天 22:00] 的历史版本`。
    
- **中间区域：** 编辑器变为**只读模式**，展示红绿背景的文本 Diff 对比。
    
- **操作项：** 提供 `[ 恢复到此版本 ]`、`[ 仅复制选中段落 ]`、`[ 退出时光机 ]`。

### P3：Onboarding 欢迎流程

#### 7. 零摩擦启动沙盒 (Sandbox)

绝对不允许强迫用户注册账号或阅读冗长的教程。

- **首次打开：** 0 秒进入主界面，自动生成一个名为 `.swarmnote` 的本地默认存储库。
    
- **预置笔记内容：** 工作区仅有一篇预置笔记《👋 欢迎来到纯粹的本地网络》。
    
- **互动式引导：**
    
    - 文档内写明：“请拿出手机，连上同一个 Wi-Fi 并打开 SwarmNote。”
        
    - 当用户照做后，桌面端左下角 Swarm 面板**自动闪烁弹出**新设备请求。
        
    - 文档底部提供一个“破坏测试区”，鼓励用户删掉一段文字，然后通过右侧“历史”面板找回，完成 Aha Moment 体验。

## 交互细节与状态机

### 文件与编辑操作

|**操作**|**触发方式**|**UI 行为反馈**|
|---|---|---|
|新建文档|`⌘N` 或 侧边栏 `+`|瞬间创建，光标自动聚焦标题，无需等待保存。|
|重命名|选中文件 `Enter`|文件名变为 `<input>` 框，文本全选，失焦即保存。|
|块级拖拽|悬停段落左侧|出现 `⋮⋮` 手柄，可上下拖拽整个 Markdown 块。|
|快速命令|空行输入 `/`|弹出就地小菜单，快速插入代码块、表格、待办。|

### P2P 与同步状态机 (核心)

|**状态**|**视觉标识 (Swarm面板)**|**说明 (底层映射)**|
|---|---|---|
|**稳态**|🟢 绿色常亮|设备间 state vector 完全一致，无可同步的 update。|
|**合并中**|🟡 琥珀色呼吸灯|yjs 正在接收并合并远端的二进制 update 包。|
|**发现新节点**|🟡 琥珀色脉冲圆点|mDNS 广播收到响应，等待用户确认交换公钥。|
|**离线孤岛**|⚪️ 灰色无动效|网卡断开或未发现节点，数据全量写入本地 SQLite。|

### 全局快捷键

|**快捷键 (Mac)**|**功能**|**快捷键 (Win/Linux)**|
|---|---|---|
|`⌘ + K`|唤起全局命令/搜索面板|`Ctrl + K`|
|`⌘ + \`|切换左侧边栏显隐 (沉浸模式)|`Ctrl + \`|
|`⌘ + Shift + H`|展开右侧历史时光机|`Ctrl + Shift + H`|
|`Enter`|(焦点在文件树时) 重命名|`F2`|

## 与原有设计稿的代差升级说明

为确保研发团队明确新版 UI 的重构方向，特列出与旧版 `.pen` 设计稿的根本性差异：

|**维度**|**废弃旧版设计**|**SwarmNote V2 新版设计**|
|---|---|---|
|**编辑器内核**|BlockNote (类似 Notion)|**CodeMirror 6** (更极客，完美契合 yjs/CRDT)|
|**布局风格**|传统顶部工具栏 + 扁平列表|**无边框窗口 + 三栏布局** (类似 Obsidian)|
|**网络感知**|传统的云盘式进度条/同步按钮|**底部的雷达式 Swarm 面板** (强调 P2P 拓扑)|
|**冲突处理**|弹窗：保留本地还是远端？|**0 弹窗**。静默 CRDT 合并 + 高亮提示 + 历史时间轴|
|**账号与启动**|选择个人资料、设定云端|**零摩擦**。打开即用，自动沙盒，mDNS 自动碰一碰配对|
