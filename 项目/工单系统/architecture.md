## 路由架构

### 路由类型

| 类型 | 文件 | 说明 |
|------|------|------|
| 常量路由 | `router/index.js` | 无需权限的路由（登录、404等） |
| 动态路由 | `router/index.js` | 需要权限验证的路由 |
| 权限路由 | 后端返回 | 根据用户权限动态生成 |

### 路由守卫流程

```
进入路由
    │
    ▼
┌─────────────┐
│ permission.js│ (全局守卫)
│ 检查 Token   │
└─────────────┘
    │
    ├── 无 Token ──▶ 白名单? ──▶ 登录页
    │
    ▼
┌─────────────┐
│ 获取用户信息 │
│ getInfo()   │
└─────────────┘
    │
    ▼
┌─────────────┐
│ 生成动态路由 │
│ generateRoutes│
└─────────────┘
    │
    ▼
  进入页面
```

## 权限控制

### 权限类型

1. **菜单权限** - 控制侧边栏菜单显示
2. **按钮权限** - 控制页面按钮显示（`v-hasPermi`）
3. **路由权限** - 控制页面访问（`permissions` / `roles`）

### 权限格式

- **权限标识**: `module:function:action`（如 `system:user:add`）
- **超级权限**: `*:*:*`（所有权限）
- **超级角色**: `admin`
