## 🚀 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产环境
npm run build:prod
```

## 📋 项目信息

- **项目名称**: sjht (世纪恒通客服工单系统)
- **版本**: 3.8.4
- **技术栈**: Vue 3.2.41 + Vite 2.9.14 + Element Plus 2.1.8
- **状态管理**: Pinia 2.0.14
- **路由**: Vue Router 4.0.14
